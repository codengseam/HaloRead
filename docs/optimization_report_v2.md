# 资治通鉴二十章 第二轮优化总结报告

## 一、本轮优化目标

用户反馈：文章可读性高、故事感强、连续性好，但**丰富性和完整性不能少**。要求：
1. 讲人物和讲背景模块多添加史料故事，让人物和背景更丰满
2. 至少原文中的全部故事和二十四史里的故事得添加上
3. 其他书上的故事也可以加
4. 可读性强的同时，注意人物丰满和道理理解深度
5. 启用多Agent并行执行，多专家团优化和打分

## 二、优化执行过程

### 阶段一：规则与提示词更新

**RULES.md 更新：**

- §3 讲背景：新增"史料丰富性"（用故事说明制度）、"制度故事化"、"纵向脉络"、"横向并置扩展"

### 阶段二：4批并行Agent优化20章

启动4个并行子代理，每个负责5章：
- Agent 1：第1-5章（三家分晋→张仪连横）
- Agent 2：第6-10章（胡服骑射→纸上谈兵）
- Agent 3：第11-15章（窃符救赵→大泽乡起义）
- Agent 4：第16-20章（破釜沉舟→鸟尽弓藏）

每篇扩充内容：
- 讲人物：每人物补充2-3个外部史料故事（来自二十四史、《左传》《国语》《战国策》《世说新语》等）
- 讲背景：补充制度故事化、纵向脉络、横向并置
- 字数从2000-3000字扩充到3500-5000字

### 阶段三：质量检测与修复

20章全部通过 quality.py 五项检测：
- ✅ structure（五段结构完整）
- ✅ ai_tone（无AI套路句）
- ✅ modern_jargon（无现代术语）
- ✅ mixed_language（无中英混杂）
- ✅ sublimation_quota（升华配额合规）

修复的共性问题：
- "缩影"→"一个典型"（商鞅变法）
- "这说明"→直接陈述（孙庞斗智）
- "可见"→直接陈述（窃符救赵、奇货可居）
- "他不是X，是Y"→"他并非X"（荆轲刺秦）

### 阶段四：4个并行专家团打分

启动4个并行打分Agent，每个负责5章，按A/B/C三维度打分。

## 四、优化亮点

### 1. 人物丰满度大幅提升

每篇主要人物都补充了2-3个外部史料故事：

- **三家分晋**：智伯（蓝台之宴+五贤+伐郑骂赵）、赵襄子（婢妾出身+灭代铜斗舞+其姐自刺）、张孟谈（功成身退+退隐后被请回）
- **商鞅变法**：商鞅（公叔痤荐鞅+景监推荐+诈俘公子卬+作法自毙）、公子虔（两度受刑+闭门八年）、赵良（百里奚对比）
- **窃符救赵**：信陵君（下交侯嬴→盗符杀将→留赵十年→酒色自毁）、侯嬴（自刭双重动机）、朱亥（三桩事+瞪虎撞柱）
- **荆轲刺秦**：荆轲（深沉好书+剑术不算顶尖+临死找台阶）、高渐离（击筑送别→铅块藏筑→终身不复近诸侯）、太子丹（赵国旧交→急躁催荆轲→被父斩献敌）
- **鸟尽弓藏**：韩信（由齐王徙楚王+多多益善+长乐宫被杀）、彭越（昌邑起兵杀迟到者+被吕后骗杀）、英布（骊山刑徒+见彭越之肉起兵）

### 2. 背景丰富度大幅提升

每篇背景都补充了制度故事化、纵向脉络、横向并置：

- **三家分晋**：六卿演变+范氏中行氏之乱+田氏代齐对照+礼崩乐坏具体表现
- **商鞅变法**：李悝《法经》+吴起变法楚国+申不害重术+宗法制瓦解
- **纸上谈兵**：秦赵国力对比+长平地理+反间计历史（信陵君、范增对照）+纸上谈兵典故成形
- **鸿门宴**：关中地理+鸿门位置+楚军内部主战主和派+分封制与郡县制之争
- **鸟尽弓藏**：郡国并行制+异姓王与同姓王+功臣集团与皇权+秦亡教训+白马之盟影响

### 3. 人物关系网清晰

## 五、共性问题与改进方向

### 1. 问道悟道段套路句式

部分篇章问道悟道开头使用"这出戏，藏着两条让人开眼的规律""这事，历史上有几处例子"等固定句式，有AI套路感。

**改进方向**：将此类句式加入quality.py检测，各篇变换句式。

### 2. "是X的Y"判断句式重复

部分篇章高频使用"是项羽霸业的起点""是势所必然""是韩信的劫/屈/死"等判断句式。

**改进方向**：减少判断句式，改用叙事性表达。

### 3. 讲背景"再说"串联

部分篇章讲背景多用"再说"串联段落，过渡词单一。

**改进方向**：丰富过渡方式，用内容自然衔接。

### 4. 部分篇章与续篇重复

渑池之会与完璧归赵在蔺相如、秦昭王部分有重复；张仪连横与苏秦合纵在纵横家群体、马王堆帛书部分有重复。

**改进方向**：续篇应聚焦本事件独特内容，避免与前篇重复。
