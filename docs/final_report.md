## 二、LoopAgent 循环机制

### 测评框架（3个并行测评Agent）

- **Agent A：结构与规则符合度（满分30）** — 五段结构/结语质量/引用规范/不编造名家/frontmatter/不堆古文
- **Agent B：史料深度与洞察（满分40）** — 史实准确/故事戏剧性/人物复杂面/背景补齐/名家点评精准/悟道深刻
- **Agent C：语言风格与AI味（满分30）** — 白话口语化/节奏感/无AI套路句/无中英混杂/不每段升华

**总分100。单章≥85为合格，<85触发优化。**

### 循环流程

```
生成一章 → 3个Agent并行测评 → 定位短板 → 更新RULES.md/quality.py/prompts/ → 修复本章 → 沉淀到loop_log.md → 进入下一章
```

## 四、LoopAgent 发现的规律性问题

### 1. "不是X而是Y"句式最顽固
- 第2章首次发现，第3章仍出现8处，直到强化为"每篇≤1处且须口语化"才控制住
- 后续章节仍需持续检测

### 2. 现代术语不断变种
- 第1章：博弈论/坐标系/放大器
- 第2章：方法论/资产/润滑剂/精算师
- 第4章：天使投资/社会流动/常设机构/纸面实力
- 第5章：智商掉线/话术
- 第6章：（已基本控制）
- 黑名单需持续扩充

### 3. 段尾升华反复出现
- 讲人物小传段尾定性升华最常见
- 问道悟道4个小节每节都来对仗金句
- 需严格执行升华配额

### 4. 模板过渡句变种多
- "还有一层背景须交代"→"再说一层背景"→"这道理在历史上反复应验"→"这道理X也懂"
- 每发现一种新变种就加入黑名单

### 5. 史实核验要点
- 典故归属（一鸣惊人/纸上谈兵成形年代）
- 出处准确性（荀子《非十二子》非《非相》）
- 年份准确性（商鞅俘公子卬前340非前341）
- 史料层累（马王堆帛书对苏秦年代的修订）
- 演义虚构≠史实（诸葛亮骂死王朗/空城计）
- 人物隐情（苏秦死间身份）
