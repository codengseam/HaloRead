### 关键纠正说明

1. **电池型号「BP-A12」错误**：用户清单中标注的 BP-A12 并非 90D 电池。90D 官方电池为 **LP-E6N**（1865mAh，7.2V，约 79g，CIPA 1300 张）。LP-E6（老款 1800mAh）与 LP-E6NH（2130mAh，EOS R6 用）均物理兼容，但 LP-E6N 为 Canon 指定的 90D OEM 电池。来源：https://www.apotelyt.com/camera-power/canon-90d-battery
2. **适马 30mm f/1.4 DC DN Contemporary 无 EF 卡口**：该镜头仅提供 L-Mount / Canon EF-M / Canon RF / FUJIFILM X / Micro 4/3 / Nikon Z / Sony E 七种无反卡口，**无法直接用于 90D**（90D 为 EF 卡口单反）。替代方案：适马老款 Sigma 30mm f/1.4 DC HSM Art（EF 卡口，已停产）或 Canon EF 35mm f/2 IS USM。来源：https://www.sigma-global.com/en/lenses/c016_30_14/

## 调研报告小结

### 关键发现摘要

1. **机身规格已多源确认**：90D 的传感器（32.5MP APS-C）、DIGIC 8、45 点全十字对焦、5481 点 DPAF、ISO 100-25600（扩展 51200）、4K 30p 无裁切、LP-E6N 电池、701g 机身等核心参数，已通过 Canon 中国/欧洲/德国/俄罗斯/土耳其官网 + B&H + DPReview + cameralabs 多源交叉验证，可作为专栏生成的高置信参数基石。

2. **两处用户清单错误已纠正**：
   - **电池型号「BP-A12」错误**：实际为 LP-E6N（1865mAh，7.2V，79g，CIPA 1300 张）。
   - **适马 30mm f/1.4 DC DN Contemporary 镜头无 EF 卡口**：仅提供 L-Mount / EF-M / RF / Fuji X / M43 / Nikon Z / Sony E 七种无反卡口，无法直接用于 90D。提供替代方案：适马老款 30mm f/1.4 DC HSM Art（EF 卡口，已停产）或 Canon EF 35mm f/2 IS USM。

3. **连拍速度需精确区分 OVF vs Live View**：OVF 10fps（One-Shot/AI Servo AF）、Live View 11fps（One-Shot 锁定对焦）/7fps（Servo AF）。专栏中描述连拍时务必注明模式。

4. **追焦命中率有实测支撑**：DPReview 论坛用户实测小型鸟类命中率 90%+，中型鸟类接近 100%；综合测评运动场景 87% 合焦率。专栏引用「追焦可靠」时建议同时给出场景条件（光线、主体大小、ISO）。

5. **配件参数齐全**：神牛 V860III-C（GN60，HSS 1/8000s，回电 0.01-1.5s）、NiSi V7 系统（True Color CPL + ND8/64/1000/32000 + GND8 Soft/Medium/Reverse）、Peak Design Slide Lite（104g，承重 200lb）+ Capture v3 均已确认官方规格。
