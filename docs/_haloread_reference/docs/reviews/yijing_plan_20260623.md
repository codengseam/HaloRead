# 易经课 章节方案（v2 · 已采纳专家团评审意见）

## 核心目标
1. 讲透每一卦的「神」——性质与精神气质，不被具体场景带偏
2. 用真实历史人物/事件贴卦贴爻，落到地上
3. 引用孔子《易传》及历代名家，行文口语化有温度
4. 简单占卜解卦，能上手不迷信
5. 偏小说风格、阅读性强，让人想往下读，同时学到核心知识
6. 厉害典故用「原文＋注释」方式讲，内容必须有据可依

## 单章结构（采用 rules.md 标准五段+结语，顺序不可调换）
1. **讲事情**：卦象本义 + 所贴历史故事 + 卦辞爻辞白话解读（合并）
2. **讲人物**：该卦所贴历史人物的人性画像（用本书之外史料佐证）
3. **讲背景**：卦的时代背景/为何此卦如此取象 + 讲性质（讲神）
4. **讲道理**：孔子《易传》+ 历代注家对该卦的直接注解
5. **问道悟道**：由该卦提炼的本质规律（含现代启示，须有史料支撑，不写鸡汤）
6. **结语**：一句话总结核心智慧
文末附「参考来源」

## 生成约束（P0/P1 已落实）
1. **结构**：严格五段+结语，不另造结构
2. **命名**：纯中文 `{章节}_{事件}.md`
3. **史实**：诸葛亮用《前出师表》原文（须注《三国志》出处）；赵简子已替换；勾践"卧薪"为后起成语须注；文王演易为传统说法须注《系辞》疑问语气
4. **层累交代**：河图洛书数图为宋代陈抟刘牧所传；先天后天八卦为宋代邵雍所立；太极图为周敦颐所传——概念与图须区分
5. **名家引用**：每卦至少 2 位古代易学家有明确该卦注解；当代学者仅作补充，不充 P0 名额
6. **跨章重复控制**：同一人物/事件只在一卦展开，他卦一笔带过
7. **顺序声明**：易经按卦序排列（非历史时间线），此为易经内在逻辑
8. **引用克制**：古文≤20字、行内引用每千字≤3处、删内联跳转、文末参考来源
9. **语言**：口语化，禁 AI 套话（我们可以看到/这告诉我们/综上所述等），禁现代术语硬套（博弈论/底层逻辑等）
