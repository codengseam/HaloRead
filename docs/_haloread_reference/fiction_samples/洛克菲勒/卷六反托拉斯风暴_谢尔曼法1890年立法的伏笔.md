# 谢尔曼法：1890年立法的伏笔

## 一、入戏

一八九零年七月二日，华盛顿，白宫。

正午的日头晒得草坪发白，东厅里却凉得发冷。本杰明·哈里森总统坐在橡木桌后，接过侍从递来的钢笔，在一份刚由国会送来的羊皮卷上签下名字。墨迹未干，他放下笔，抬头看了一眼窗外——宾夕法尼亚大道上，有零星几个举着标语的人在喊口号，标语上写着"打倒托拉斯"。

哈里森揉了揉太阳穴。这份叫《谢尔曼反托拉斯法》的东西，从今天起，就是美利坚的法律了。

## 二、破题

要解决的问题只有一个——这部新法，会不会头一个拿标准石油开刀？

同一天，纽约，百老汇二十六号九楼。洛克菲勒桌上摊着当天的《纽约时报》，头版正是哈里森签字的消息。他翻开膝上账本A，在新的一页写下两行字：一八九零，立法；一八九五，不动；一九一零，动手。

金手指在这一页换上了最冷的一档——它算的不再是成本和运费，而是"政治气候"。账本A第一次替洛克菲勒算起了时间。

## 三、推进

### 一、托拉斯惹出的民怨

这部法的来由，要倒回十年前。

一八八二年标准石油托拉斯成立之后，美国商界像发现了新大陆。糖业托拉斯、威士忌托拉斯、棉花油托拉斯、铅业托拉斯、绳索托拉斯——一夜之间冒出几十家。每一家都学着标准石油的样儿，把几十家公司捏成一团，把价格捏在一个人手里。

老百姓最先尝到滋味的是糖。一八八七年糖业托拉斯一成立，食糖零售价半年涨了两成。家庭主妇在杂货店里骂街，报纸编辑在社论里跳脚，牧师在布道台上说"这是魔鬼的营生"。一八八八年大选年，两党候选人都在台上喊"管管托拉斯"——可谁也说不清怎么管。

俄亥俄州的参议员约翰·谢尔曼，是头一个真动手的人。谢尔曼这一年六十七岁，在参议院坐了三十年，做过财政部长，是共和党里的老面。他这辈子跟钱打了半辈子交道，看得明白：各州的公司法管不了跨州的托拉斯，要管，只能联邦出面。

一八八八年十二月，谢尔曼把第一版反托拉斯法案送进参议院。他在台上说了一句话，后来被报纸反复引——"如果我们不能控制这些结合，它们就会控制我们。"参议院鼓掌通过，可法案到了众议院，被对方党派搁死了。

谢尔曼不甘心。一八九零年他重起炉灶，把法案改短、改软，去掉几处最刺眼的话，重新送进国会。这回两边都点了头。六月二十日参议院通过，二十天里众议院也通过，七月二日，哈里森签字。美利坚合众国第一部反托拉斯法，就这么成了。

### 二、法里头写了什么

法案不长，拢共八条。可真正要命的是头两条。

第一条说，凡是"限制州际贸易或对外贸易的合同、合并、合谋"，统统非法。第二条说，凡是"垄断、企图垄断、与他人合谋垄断"任一州际或对外贸易的人，都算犯罪。

洛克菲勒看完报纸，把这八条逐字抄进账本A。抄完，他盯着一二两条看了许久。他知道，按这两条字面去抠，标准石油托拉斯从出生那天起就犯法——一八八二年那纸协议，九个人托管四十家公司，控制全美九成炼油，这要不算"限制贸易"和"垄断"，天下就没有垄断了。

可洛克菲勒也看出了破绽。法案没说"什么是垄断"，没说"什么叫限制贸易"，没说由谁来起诉、由谁来判、判了怎么执行。一句话，刀是铸出来了，可谁来挥、怎么挥，全是空的。

他把账本A翻到新的一页，写下：法有了，刀没手。

### 三、刀没手的那十年

往后的事实，跟账本A算的分毫不差。

一八九零年到一九零一年，整整十一年，联邦政府拿谢尔曼法起诉的案子拢共十八件。其中大半是工会案——政府拿这条反垄断的法，反过来收拾罢工的工人。真正拿来告大企业的，一只手数得过来。一八九二年，政府告过糖业托拉斯，最高法院判决政府败诉，理由是"糖是制造业，不是贸易"。一八九五年，政府告一家铁路巨头，又是最高法院判决败诉。

洛克菲勒把这些判决一条条剪下来，贴进账本A。他看得明白：不是法不管用，是华盛顿没人想用。哈里森下台了，克利夫兰回来了，麦金莱上来了——这几位总统，谁也不愿得罪华尔街。标准石油这些年往两党送的竞选款，从来没断过线。

一八九零年代末，弗拉格勒有一次问他："约翰，谢尔曼那法，到底会不会落到我们头上？"洛克菲勒合上账本A，慢慢说："亨利，华盛顿现在没有一个人想动我们。可法律这东西，一旦立了，就摆在那儿。今天没人用，不等于明天没人用。"

弗拉格勒追问："那我们该怎么办？"

洛克菲勒没答。他把账本A翻到那页写着"一九一零，动手"的字，用手指压住，半天不松。

## 四、高潮

一八九零年七月四日，独立纪念日。纽约满城放烟花，可百老汇二十六号九楼的灯，照旧亮到最晚。

屋里只剩洛克菲勒和弗拉格勒。桌上摊着账本A，翻到刚写的那两行——"一八九五，不动；一九一零，动手"。

弗拉格勒盯着那两行字，看了半天，抬头问："约翰，你怎么算出来的？"

洛克菲勒拿起铅笔，在"一八九五"旁边点了一下。"亨利，你看华盛顿那帮人。哈里森签这法，是为大选年糊弄选民。下一任不管是克利夫兰还是谁，都得哄着华尔街。一八九五之前，没人会动这法。"他又在"一九一零"旁边点了一下，"可法律立了，就有人会用。民怨在攒，托拉斯在长，等到有一天华盛顿来一个真想干事的人——这把刀就落下来。我算着，二十年。"

弗拉格勒沉默半晌，问："那这二十年，我们做什么？"

洛克菲勒把账本A合上，放回膝头。"做两件事。"他竖起一根手指，"第一，把这法研究透。多德那边，专门组一队人，盯着每一桩反托拉斯案子，每一份判决。我们得比华盛顿更懂这把刀。"他竖起第二根手指，"第二，把托拉斯的结构，再换一层皮。这法针对的是'限制贸易的合同'和'合并'——那我们就把合同拆开，把合并藏深。让它表面上看，哪儿都不犯法。"

弗拉格勒点头，又问："万一二十年后真动手呢？"

洛克菲勒看着窗外烟花的光，缓缓说："那我们就让法庭去吵。这法没说清什么叫垄断——只要没说清，我们就有得打。真正的战场，不在华盛顿，在最高法院那九个老头身上。"

那一刻，账本A上"一九一零"三个字，像一颗埋进土里的种子。弗拉格勒不知道，洛克菲勒自己也不知道——二十年后，正是最高法院那九个老头，亲手把这颗种子刨了出来。

## 五、余味

**OPC 翻译桥**：

谢尔曼法是监管的尾部风险——一八九零年埋下，一九一一年引爆，整整二十一年的伏笔。OPC 启示：业务模式踩在法律灰区时，必须预设"被禁"的 Plan B。今天起，请审视你的业务——有没有监管风险？数据合规、税务、平台规则、行业准入，哪一项改一条规矩就能让你关门？列三个 Plan B：业务模式被禁怎么办？平台封号怎么办？税法修改怎么办？每个 Plan B 写清楚触发条件、切换路径、备用主体。多数 OPC 死于"赌监管不会变"，活下来的人，都在"预设 Plan B"。监管这把刀，铸出来就摆在那儿，今天没人挥，不等于明天没人挥。

---

那夜散了之后，洛克菲勒把账本A锁回保险柜。窗外烟花散尽，百老汇重归安静。他站在窗前，望着远处哈德逊河上的一点灯火，对身后的弗拉格勒说了一句话——

"亨利，这部法律现在不管用。可二十年后，会管用。我们得在它管用之前，把该做的事做完。"

弗拉格勒应了一声，没接话。他不知道，二十年后的一九一一年，正是这部一八九零年的法，把标准石油撕成了三十四块。
