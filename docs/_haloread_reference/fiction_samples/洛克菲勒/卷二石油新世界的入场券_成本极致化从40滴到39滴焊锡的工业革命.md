# 成本极致化：从40滴到39滴焊锡的工业革命

## 一、入戏

1865年春，克利夫兰Excelsior Works炼油厂车间。

洛克菲勒蹲在一台焊锡机旁边，半个钟头没动。焊工老比尔每封一只油桶，手腕一抖，焊锡落下来——四十滴。

"再封一只。"洛克菲勒说。

老比尔又封一只。四十滴，一滴不多，一滴不少。

"约翰先生，"老比尔擦汗，"你蹲这干嘛？四十滴，是师父传下来的规矩，多少年了。"

洛克菲勒没抬头。账本A摊在膝盖上，铅笔在纸上慢慢画圈。

"四十滴。"他低声说，"我猜，三十九滴就够。"

## 二、破题

同行看一只油桶，看到的是装油的容器。洛克菲勒看一只油桶，看到的是三十八个成本节点——桶板、桶箍、桶底、焊缝、焊锡、人工、损耗、搬运、仓储、利息，每一个节点都是一滴锡、一分钱。

这一回，账本A从"算账"升级成"重塑工业流程"——金手指第一次伸进车间的机器里。

## 三、推进

### 三十九滴

据传，洛克菲勒花了一整个礼拜，蹲在焊锡机旁。

他先让老比尔封四十只桶，每只四十滴——一只不漏。这是基线。

然后他让老比尔减一滴，封四十只桶，每只三十九滴——也不漏。

老比尔嘀咕："少一滴，能省几个钱？"

"一桶省四分之一美分。"洛克菲勒头也不抬，"一年装三十万桶，省七十五块。装得越多，省得越多。"

老比尔不信。洛克菲勒让他再减一滴——三十八滴。这回，四十只桶里有七只漏了。

"加回去。"洛克菲勒说。

于是定在三十九滴。

这件事后来被写进许多传记，被反复转述，成了"标准石油式成本控制"的图腾。史学界对细节有争议——有人说洛克菲勒亲自试验，有人说他是听工人随口一提就拍板。但有一点没争议：从那以后，Excelsior Works的每一只油桶，都按三十九滴封。

账本A上多了一行——

**焊锡：39滴/桶。省0.0025美元/桶。**

这是账本A第一次反过来指挥机器。

### 把森林搬进炼油厂

焊锡只是开头。

账本A翻到"桶"那一节，洛克菲勒把一只油桶拆成零件，逐项往下算——桶板、桶箍、桶底、桶盖、焊缝、油漆、人工。每一项他都问同一个问题：这一项，能不能自己干？

桶板最贵。克利夫兰的木桶厂把桶板卖给炼油厂，一桶板要一毛八。洛克菲勒算过——买原木自己锯，一桶板只要九分。

1866年，他干脆买下一片橡木林，自办伐木场，自锯桶板。木桶厂老板骂他"不务正业"，他没接话，只在账本A上记下：**桶板：自营，省0.09美元/桶。**

接着是副产品。同行炼油，只取煤油，剩下的全当废料倒掉。洛克菲勒不倒。他把釜里出来的东西一样一样分开——上层的轻油，提炼成汽油，卖给干洗店；中层的重油，做成润滑油，卖给机械厂；凝固的部分，做成石蜡，卖给蜡烛厂；最底下的黑油渣，做成沥青，卖给修路的。

一桶原油进来，五样东西出去。同行一桶赚七块，洛克菲勒一桶赚十块多——多出来的三块，全是别人倒掉的"废料"。

账本A上又多了一行——

**副产品回收：+3.1美元/桶。**

### 火烧出来的系统

1866年秋，厂子又着了一次火。

这回烧得更狠，三台蒸馏釜报废两台，半个车间塌了。安德鲁斯急得直跺脚，新来的工人吓得跑了一半。

火灭当晚，洛克菲勒没回家。他蹲在废墟边，账本A摊在膝盖上，一页一页记——起火点、烧毁设备、停工天数、修复成本、保险赔付。

第二天一早，他干了两件事。

第一件，重新画车间的布局。原来蒸馏釜挨着木桶仓库，一着火就连锁烧。他把蒸馏区、灌装区、仓储区彻底分开，中间砌防火墙，地面铺石板。账本A里多了一份"防火流程图"——每个釜配沙桶，每个工人都背得出灭火车间的路线。

第二件，去找保险公司。

"卡特先生，"洛克菲勒把账本A摊在保险公司老板桌上，"我要续保，保费要降两成。"

"降？"卡特笑，"你刚着过火。"

"正因为刚着过火，才该降。"洛克菲勒翻到防火流程图那一页，"你看——分区、防火墙、石板地、沙桶、灭火车间。这套东西，克利夫兰没有第二家有。下次着火的概率，比去年低一半。你降两成保费，我少付的钱够我再砌一道防火墙。这是个良性循环。"

卡特翻了翻账本A，半天没说话。

最后保费降了一成五。

## 四、高潮

1867年初，克利夫兰商会年宴。

酒过三巡，克利夫兰第二大炼油厂的老板布朗站起来，端着酒杯，冲洛克菲勒那桌一笑。

"我说约翰，"他嗓门很大，全场都听得见，"听说你天天蹲车间数焊锡？一滴锡、一美分的，跟我们这些做大生意的，到底不在一个段位。"

满桌哄笑。

布朗接着说："炼油这行，靠的是油源、是销路、是胆量。你倒好，趴在账本上抠一美分。一美分能抠出个帝国来？"

洛克菲勒没急着回话。他从内袋里抽出账本A，慢慢翻到焊锡那一页，搁在桌上。

"布朗先生，"他声音不高，"我给你算一笔。"

"一只桶，三十九滴焊锡，比四十滴省四分之一美分。"

"我厂子现在一年装三十万桶，省七十五块。"

"可这不是我要说的。"洛克菲勒翻了一页，"按我账本A上的规划，三年内，我一年要装到三百万桶。到那天——"

他抬眼，看着布朗。

"——一年光焊锡这一项，省七千五百块。"

满桌安静下来。

洛克菲勒接着说："布朗先生，你那家厂，去年净利润多少？"

布朗脸涨红了，没答。在场的人都知道——他那家厂，去年净赚不到七千。

"你的全部利润，"洛克菲勒把账本A合上，"都没我这一滴锡省得多。"

他把账本A收回内袋，端起酒杯，朝布朗那桌遥遥一举，一饮而尽。

整桌人没人再笑。

布朗站也不是，坐也不是，酒杯举在半空，半天放不下。

## 五、余味

**OPC 翻译桥**：

洛克菲勒把焊锡从四十滴减到三十九滴——每桶省一美分的四分之一，按他账本A上的规划，三年后一年能省七千五百块（相当于今天的十五万美元）。账本A没教他怎么炼油，教他的是把一只油桶拆成三十八个成本节点，挨个问"这一滴能不能减"。

OPC 启示：找出你的"第四十滴焊锡"。今天起，列出你工作流里所有"重复 N 次"的动作——发邮件、回客服、写报告、做报价单。每个重复动作里，都藏着一滴多余的"焊锡"。是手动抄同一份表格？是每次都重写开头？是客服回复里多打的五个字？自动化掉它，复利就是你的利润。多数 OPC 死于"没时间优化"，活下来的人，都把"省一美分"当战略——不是抠门，是把别人懒得算的账，算成一整条护城河。

---

年宴过去半个月，一个雪后的早晨，洛克菲勒在金斯伯里街的办公室里收到一封信。

信封上的字迹陌生，落款是佛罗里达——一个他没去过的地方。

他拆开，只看了两行，眉头就动了一下。

信的开头写着：

> 敬启者，亨利·弗拉格勒，谨就克利夫兰至产油区铁路运费一事，冒昧致函……

洛克菲勒把信放在账本A上，盯着那个名字看了很久。

亨利·弗拉格勒。

他把信折好，贴身收起，没让安德鲁斯看见。
