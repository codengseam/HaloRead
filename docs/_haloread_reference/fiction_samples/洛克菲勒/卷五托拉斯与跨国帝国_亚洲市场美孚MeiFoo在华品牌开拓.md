# 亚洲市场：美孚 Mei Foo 在华品牌开拓

## 一、入戏

一八九零年某夜，上海外滩，一座靠江的仓库门口。

雨刚停，黄浦江上的雾还没散。一个穿长衫的中国伙计，正指挥几个苦力，把一箱箱木箱从马车上卸下来。木箱上印着一行洋文——Standard Oil。仓库里头，一个美国青年拆开一只箱子，从里头捧出一盏黄铜座子、玻璃罩子的小煤油灯，凑到油灯下端详。他对身边的人说："灯到了。下一步，是把它们送进中国农民的屋里。"

那青年叫莫尔斯，标准石油派驻中国的第一任代理。

## 二、破题

要解决的问题只有一个——中国农村用什么照明，标准石油就得让它们改用什么。

那一年，中国四亿人口，九成住在乡下，点灯用的是菜油、豆油、蜡烛。煤油，乡里人大多没听过。要让几亿人换一种灯、换一种油，靠一句"煤油比菜油亮"是没用的——他们没钱买灯，也没人教他们点。

金手指在这一夜第一次跨国落地。账本A上落下三行字：一盏煤油灯成本零点五美元；一个农民三年买煤油花五美元；送灯一盏，换油三年，回报十倍。

洛克菲勒对弗拉格勒说："亨利，我们把灯白送给他们，油我们来卖。"

## 三、推进

### 一、注册"美孚"

一八九零年前后，标准石油的中国代理正式注册了一个中文商标——美孚。

"美"字取美观、美好之意；"孚"字取信服、可信之意。两个字凑在一起，是"美观可信"四字的缩写，又带着一股稳当的旧式商号味儿。从此，标准石油卖到中国的煤油，桶上都印着这两个字；那盏黄铜座子的小煤油灯，也就被中国人叫成了"美孚灯"。

名字定下，代理网也开始铺。标准石油不直接进村——它在上海、汉口、天津几个口岸设了办事处，再找中国本地的买办、商号做二级代理，一层层往下铺。到了县里，再由本地杂货铺、油盐店代卖。这套法子，是标准石油在美国玩熟了的分销网，搬到中国，只多了一层"洋行—买办—商号—店铺"的翻译。

莫尔斯后来在汇报里写了一句：中国这地方，省份比欧洲国家还多，方言比欧洲语言还杂，可点灯这件事，全中国都一样——亮，就行。

### 二、白送一盏灯

灯造出来了，代理网铺下了，可农民不买。

道理很简单——一盏美孚灯卖到乡下，要花掉一个农民小半个月的工钱。没灯，煤油卖给谁？这是死结。

洛克菲勒的解法，是账本A上那三行字。一八九零年代初，他下令：免费送灯。不是送一两盏，是几百万盏。

送灯的法子很讲究。代理们带着美孚灯下乡，先找村子里说话算数的人——族长、塾师、货郎——送给他们一盏，点上，让全村来看。一盏美孚灯的亮度，顶得上七八盏菜油灯，整个屋子照得通亮。村里人一看，眼睛就挪不开了。

接下来是更绝的一手——送灯的同时，附一小瓶煤油，够点十几个晚上。等那瓶油点完了，农民已经习惯了亮堂堂的灯下生活，再回去点那盏昏黄的菜油灯，谁都受不了。这时候，代理再上门卖煤油，一分钱不让，一桶桶往外走。

后来有人算过，一八九零年代那几年，标准石油在中国白送出去的美孚灯，少说也有几百万盏。按零点五美元一盏的成本，光送灯就烧掉了一两百万美元。可账本A上那行字没骗人——三年下来，一个农民买煤油花的钱，是那盏灯成本的十倍。

### 三、一盏灯换一片市场

灯送出去，油卖进来。

美孚煤油桶从上海口岸上岸，分装进小罐，由代理商一船船运进长江、运进珠江、运进运河。最远的地方，挑夫挑着煤油罐翻山越岭，把油送到云贵山里的小村子。一八九零年代中期，连云南山里的苗寨，都点上了美孚灯。

这套生意跑起来之后，巴库煤油在中国几乎看不见影子——巴库到中国，要绕大半个地球，运费比标准石油从美国出港还贵，根本进不来。诺贝尔兄弟和罗斯柴尔德在欧洲能跟标准石油掰手腕，可到了中国沿海、长江沿岸，他们连一桶油都送不到。

一八九零年代到一九一零年代，美孚灯在中国农村普及开来。后人估算，那二十年间，标准石油占了中国煤油市场一半以上的份额。半个中国的夜晚，是被美孚灯照亮的。

## 四、高潮

一八九四年某天，纽约，百老汇二十六号，九楼。

洛克菲勒把一份中国代理的电报翻来覆去看了三遍。电报上是当月销量——美孚煤油在华销量，第一次超过了在欧洲的销量。

弗拉格勒推门进来："约翰，你叫我来？"

"亨利，你看这个。"洛克菲勒把电报递过去。

弗拉格勒看完，抬起头："亚洲打开了吗？"

洛克菲勒没急着答。他从公文包里抽出账本A，翻到那一页跨国比较表，指着底下新添的两行——一八九四年在华销量，超欧洲；一八九四年在华市场份额，过半。

"打开了，"洛克菲勒说，"中国这一仗，我们用一盏灯换了一片市场。巴库够不着的地方，我们够得着。"

弗拉格勒笑了："那亚洲这一仗，赢了？"

洛克菲勒合上账本，没笑。他指着地图上欧洲那一块："赢了一半。亚洲打开了，可巴库还没死。"

"什么意思？"

"巴库在欧洲跟我们对耗，耗不动我们，他们就会换法子。"洛克菲勒的手指从欧洲划到美国本土，"我们的对手，开始用铁路运价做文章了。"

## 五、余味

**OPC 翻译桥**：

标准石油在中国白送美孚灯、卖煤油——典型的"硬件免费、耗材收费"。今天的打印机卖得便宜，墨盒卖得贵；剃须刀送你一个，刀片月月买；游戏机亏本卖，靠游戏抽成赚钱。这套打法的内核，是"亏本获客、长期复购"。OPC 启示：找到你的"美孚灯"——一个客户入门的低门槛产品或服务，用它锁住长期耗材、订阅、复购。今天起，请审视你的产品矩阵——有没有"亏本获客加长期复购"的组合？多数 OPC 死在"每个产品都要赚钱"，活下来的人都在"用灯换油"。本地品牌加全球分发，是 OPC 最锋利的杠杆。

---

那天深夜，洛克菲勒独自坐在九楼。账本A摊在桌上，"中国销量"那一行底下，他用红铅笔写了一行小字——铁路运价。

大洋对岸的中国乡村，美孚灯一盏接一盏亮起来。可洛克菲勒的目光，已经从亚洲，转回了美国本土的铁路线。

他知道，那群在巴库耗不动的对手，要换个地方动手了。
