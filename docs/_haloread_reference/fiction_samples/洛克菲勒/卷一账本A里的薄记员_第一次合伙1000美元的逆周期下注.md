# 第一次合伙·1000美元的逆周期下注

## 一、入戏

1859年3月，克利夫兰的春寒还咬人。

十九岁的洛克菲勒站在休伊特-塔特尔公司那间他待了三年半的阁楼办公室里，把私人物品一件件收进一只磨边的皮箱——墨水瓶、镇纸、一支断了笔尖的钢笔，还有一本封皮磨得发亮的黑皮账本。

合上箱盖前，他把那本黑账本单独抽出来，塞进了外套的内袋。

贴着心口的位置。

休伊特在楼梯口叫住他："约翰，真不再想想？"——指的自然是那个被"推迟"的参股邀约。

洛克菲勒回头："想清楚了。我想自己干。"

## 二、破题

他自己干的方式，是和莫里斯·克拉克合伙。

克拉克比他大六岁，英国移民，在休伊特-塔特尔时就是同事，话多、性急、好面子，张口就是"我在利物浦那会儿"。在克拉克眼里，这个十九岁的穷小子合伙人，最让他看不顺眼的，就是那本不离身的黑账本——"斤斤计较，成不了大事。"

合伙还没开张，两人的第一次冲突，就在码头上炸了。

那天他俩去外滩码头验一批待运的谷物。克拉克捏了一把麦粒，闻了闻，拍板："这批要了，成色不错。"

洛克菲勒没说话，蹲下来从桶底抽出几粒，又翻开账本A对照上月的损耗记录。

"莫里斯，这批的水分比上批高两成。"他指着桶壁上一道水印，"按这个水分，到纽约损耗会吃掉一成半的利。"

克拉克脸一沉："我看了二十年谷物，你说我看走眼？"

"您没看走眼。您看的是成色，我看的是到岸成本。"

克拉克被噎得说不出话。最后这批货压了价才要——码头货主自己心里有数，没敢争。

回公司的路上，克拉克一句没说话。洛克菲勒知道，这事儿没完。

## 三、推进

### 一千美元的来路

合伙总资本两千美元，每人出一千。

洛克菲勒这三年半在休伊特-塔特尔攒下八百。差额两百是小事，真正要他张嘴的，是另外那笔——他得向父亲借钱。

父亲威廉·洛克菲勒是个走街串巷的游医，卖"包治百病"的药水，常年不着家，钱包倒是常年鼓着。

"借钱可以。"威廉坐在旅馆的破椅子上，翘着二郎腿，"利息一成。"

洛克菲勒的指节在桌沿上捏白了。

"市面上一成是公道价。"威廉剔着牙，"儿子，做生意归做生意。"

洛克菲勒没争。他在账本A上写下一行：**借威廉·洛克菲勒一千美元，年息十厘，到期还本付息。** 把借条推过去。

父亲签了字，又从裤兜里数出十张崭新的绿背钞票，一张张摊在桌上，像在数自己的胜利。

走出旅馆那晚，欧几里大道的风刮得人睁不开眼。洛克菲勒在路灯下翻开账本A，在一千美元那笔下面，又添了一行小字：

**此款来源存疑。父亲收入不稳。**

他不敢问这钱干不干净，也不敢想。这笔钱像一根刺，埋进他合伙创业的根里，也埋进卷一往后的每一页。

### 秋天的反向算式

1859年秋，南北局势一天比一天紧。南边的炮声还没响，谷物的价格先炸了。

克利夫兰的农产品市场上，麦子一周跌了三成，猪肉无人问津，连果农都开始把烂苹果往河里倒。

克拉克一进办公室就把帽子往桌上一摔："抛！全部抛！再不抛就烂在仓里了！"

洛克菲勒没抬头，还在账本A上写写画画。

"你聋了？"克拉克走过来，伸手要把账本合上，"我说话你听见没有？"

"听见了。"洛克菲勒把账本转过来，推到克拉克面前，"你先看这个。"

那一页上，他用红墨水列了一张表：

- 若战争爆发（概率六成）：谷物三个月内涨四成，猪肉涨五成。
- 若战争避免（概率四成）：谷物再跌一成，猪肉持平。
- 期望收益：正向。

"你这是哪门子歪理？"克拉克盯着那张表，眉头拧成一团，"南边还没打起来，你就押战争？"

"我不押战争。"洛克菲勒把铅笔搁下，"我押概率。"

"你一个十九岁的毛头小子，懂什么概率？"

"我懂账本。"

克拉克冷笑："账本上的小气，做不了大生意。"

办公室里安静了三秒。

洛克菲勒抬起头，眼睛冷得像井水："莫里斯，我是合伙人，不是你的伙计。这笔货，我主张囤。"

克拉克被这眼神噎了一下。合伙协议上白纸黑字——重大决策两人签字。他甩了甩手："行。你囤。赔了算你的。"

两个月后，南卡罗来纳的局势把北方每户人家的恐慌推到顶点。粮价应声飞涨，麦子一周涨了两成，猪肉涨了三成。

克拉克那段时间每次走进办公室，脸上的表情都像吞了苍蝇。

### 红利下的暗痕

到1859年年底，克拉克-洛克菲勒公司这一年净赚了四千多美元——分到洛克菲勒手上，是两千出头。

是他三年半工资的两倍。

克拉克那阵子态度明显软了。他开始主动把账目推给洛克菲勒管，自己去跑外勤、谈客户。有天晚上，他甚至破天荒请洛克菲勒喝了一杯威士忌。

"约翰，"克拉克举着杯子，"你这小子，算账是真有一套。"

洛克菲勒没碰杯子。

"莫里斯，"他说，"账不是算给人看的，是算给自己看的。"

克拉克愣了一下，哈哈大笑："你这人，无趣。"

洛克菲勒没笑。他知道，克拉克嘴上的"无趣"，底下藏着更深的东西——这人不服输，更不服人。今天能因为赚钱夸你，明天就能因为赔钱踩你。

裂痕没合上，只是被一时的红利盖住了。

## 四、高潮

腊月二十三，公司分红的前一夜。

办公室里烧着壁炉，克拉克、洛克菲勒，还有两个跑腿的伙计围在桌前对账。克拉克念数，伙计记，洛克菲勒核对账本A。

一切顺利，直到洛克菲勒的铅笔停在某一页上。

"等一下。"他说。

克拉克抬头："怎么了？"

"十一月十四日，茶水费，四十七美元。"洛克菲勒把账本A翻过去，指着克拉克记的那一栏，"你这一笔，记在了'招待客户'里。"

"本来就是招待客户的。"

"那天没有客户。"洛克菲勒的声音不高，每个字钉得很死，"我查了当天的出货单，没有一笔对应的外勤。莫里斯，这四十七美元，是你请朋友喝酒。"

克拉克的脸"唰"地红了。

办公室里鸦雀无声。两个伙计低着头，大气不敢出。

"约翰·洛克菲勒，"克拉克把椅子往后一推，站了起来，"你当着伙计的面，跟我算四十七美元的茶水费？"

"账就是账。"洛克菲勒没站起来，"四十七美元，要么你补上，要么从你分红里扣。"

"你——"

克拉克的拳头在桌沿上捏了又捏，最终没砸下去。他从钱夹里数出四十七美元，一张张拍在桌上。

"莫里斯·克拉克，补茶水费四十七美元。"洛克菲勒在账本A上写下这一行，签了自己的名字，"麻烦你也签一个。"

克拉克抓起笔，在名字栏上划了几个字，笔尖差点把纸戳破。

他摔门而出。

## 五、余味

**OPC 翻译桥**：

洛克菲勒十九岁合伙创业，启动资金拼凑自积蓄和父亲的高息借贷——合伙第一天就埋下了"控制权之争"的雷。今天那些结伴开公司的年轻人，多半也是这样：兄弟情义一上来，散伙条款没人提。等真闹翻那天，才发现连"谁有权签字"都没写清楚。

OPC 的启示很冷，但很值钱：**合伙的第一天就要想清楚退出条款，控制权比股权比例更值钱。**

今天起，如果你有合伙人，去签一份"散伙协议"——

1. **触发条件**：分歧无法调和、合伙人离职、合伙人死亡或丧失行为能力，写清什么情况下可以散伙。
2. **估值方法**：约定按市盈率、净资产、或第三方评估，三种里挑一种，白纸黑字。
3. **退出方式**：现金一次性、分期支付、或强制回购，写清时间表和违约金。

多数 OPC 死于"先做兄弟后算账"。散伙协议比合伙协议更重要——洛克菲勒用四十七美元的茶水费买到的这个教训，今天依然值钱。

---

克拉克摔门出去之后，办公室里只剩壁炉里柴火爆裂的声音。

洛克菲勒合上账本A，正要吹灯，忽然听见走廊尽头传来克拉克压低的一句话，像是对自己说的，又像是对着空气骂的：

"总有一天……"

后面的话被风吞了。

洛克菲勒站在原地，没动。

他知道，合伙的雷，已经埋下了。
