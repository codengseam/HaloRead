# 形象管理课 · 顺序质检报告

## 一、质检方法与覆盖

本报告对 `/workspace/output/形象管理课/` 下 45 篇正文进行"顺序"维度的独立质检。

**质检方法**：
1. 用 LS 列出 45 篇正文，确认篇数与预编排表一致（3+4+9+9+8+4+4+3+1=45 ✓）。
2. 用 Grep 扫描全部 45 篇 frontmatter 的 `title / chapter / event / sort / chapter_sort` 字段，核验与文件名章节前缀、与预编排表的三重一致性。
3. 用 Read 深读 16 篇样本（每章 1-2 篇，覆盖全部 9 个章节组），评估单章叙事顺序（入戏→破题→方法论→避坑→践行的递进是否清晰、方法论是否条理分明、有无中段过长背景打断）。
4. 检查"形/健""术/场"两组易越界章节的边界是否清晰。

**抽样深读清单（16 篇）**：
- 开篇：为什么学形象管理、怎么用这份专栏、形象是非语言沟通（3/3 全抽）
- 道：形象的价值放大原理、形象的边界与变通
- 形：体型诊断、个人风格、气味管理（含与"术"组边界篇）
- 术：西装体系、衣橱规划（首末两端）
- 场：正式场合、职场日常
- 健：体态矫正（与"形"组边界篇）、减脂与饮食
- 避坑：常见穿搭雷区
- 践行：90天形象升级路径

**评分标准**：
- 叙事顺序（8 分）：五段式递进完整度 + 方法论条理性 + 无中段打断
- 章节排序（7 分）：chapter/sort/chapter_sort 字段正确性 + 章内递进逻辑
- 合格线：总分 ≥ 11/15

## 五、章节边界检查

### 5.1 形 / 健 边界（诊断 vs 执行）

- **形_体态评估**（形组 sort=6）：讲"怎么判断自己属于哪种体态问题"——靠墙站立测试、侧面间隙观察、双肩等高检查。纯诊断，不教动作。
- **健_体态矫正**（健组 sort=1）：开篇明确定位"诊断自己属于哪种体态问题，是「形」组的体态评估章节的任务，这里默认你已经知道自己要矫正哪一项"，直接进入矫正动作（圆肩/头前伸/骨盆前倾/高低肩）。

**边界判定**：清晰。形=诊断，健=执行，两章通过显式交叉引用衔接，无内容重复或越界。健组 sort=1 紧接形组 sort=6，排序上形成"评估→矫正"的自然过渡。

### 5.2 术 / 场 边界（通用 vs 场景）

- **术_西装体系**（术组 sort=1）：开篇明确定位"这一篇讲通用规范，不讲某场婚礼该穿什么、某次面试该穿什么——那是'场'组的事"。
- **术_衣橱规划**（术组 sort=9）：明确"不讲'出差怎么打包'——那是场景问题，归'场'组"。
- **场_正式场合**（场组 sort=5）：聚焦 Black Tie/White Tie/国服/婚礼葬礼等具体场景的着装规范应用。
- **场_职场日常**（场组 sort=1）：聚焦职场三档正式度的日常应用。

**边界判定**：清晰。术=通用单品工具与规范，场=具体场景下的应用决策。术组篇章多次显式声明"场景应用归场组"，场组篇章默认读者已掌握术组单品知识，无越界。

### 5.3 其他边界抽查

- **形/术边界**：形_个人风格明确"具体的单品选择与搭配执行，留给后续章节"；术_西装体系开篇即要求读者已懂"肩型与体型"（形组内容）。衔接清晰。
- **避坑/术边界**：避坑_常见穿搭雷区明确"不讲某一条领带该怎么打——那些是单章已经讲透的'术'。这一章专门拆解跨场景、跨单品、反复发作的系统性穿搭误区"。定位清晰，无重复。
- **践行/全专栏边界**：践行_90天形象升级路径开篇定位"把前面所有内容压缩成可执行的日常清单"，三阶段（诊断期/构建期/精进期）显式引用形、术、场、健各组成果，作为全专栏的执行收束。定位清晰。
