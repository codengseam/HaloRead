# 第11章 三十而立——私学开张

## 讲事情

鲁昭公二十年，公元前522年，曲阜城里出了件不大不小的事——孔丘三十岁了。

三十岁这年，孔丘没做官，没发财，也没显赫的功名。可他自己心里有数。后来他在《论语·为政》里回忆这一辈子："吾十有五而志于学，三十而立，四十而不惑，五十而知天命，六十而耳顺，七十而从心所欲不逾矩。"——十五岁立志向学，三十岁立住了。立什么？立学问，立人格，立一辈子要走的路。

这"立"字，不是凭空来的。

孔丘祖上本是宋国贵族，往上数六代是宋湣公的直系，可传到他这一辈，早败落了。父亲叔梁纥在鲁国做过陬邑大夫，是一员猛将，偪阳之战力举城门，名震诸侯。可孔丘三岁上父亲就没了，母亲颜徵在带着他搬回曲阜阙里，孤儿寡母，日子紧巴。少年孔丘没什么显贵可依，"吾少也贱，故多能鄙事"——《论语·子罕》——我小时候出身低贱，所以会干很多粗活。

他干过什么粗活？《孟子·万章下》记着两笔：孔子尝为委吏矣，曰"会计当而已矣"——做管仓库的委吏，账目算得清清楚楚就行；尝为乘田矣，曰"牛羊茁壮长而已矣"——做管畜牧的乘田，牛羊长得壮实就行。《史记·孔子世家》也记了一笔："孔子贫且贱。及长，尝为季氏史，料量平；尝为司职吏而畜蕃息。"委吏、乘田，都是当时最小的吏，贵族子弟不屑于干。孔丘干，而且干得踏实。账目算得平，牛羊喂得肥。这不是大事，可这见出一个人的性子——干什么像什么。

到了三十岁，孔丘的学问已经成了。他通礼、乐、射、御、书、数六艺，熟诗书，明礼制，在鲁国有了名气。可他没去求个一官半职，反倒做了件当时没人做过的事——在自己家里设教授徒。

地点就在阙里老宅。院子不大，几间草屋，一棵老槐。树下摆几张几案，案上堆着竹简。天一亮，院子里就有人来。有穿绸的，有穿布的，有比孔丘小不了几岁的，也有十几岁的半大孩子。这些人凑一块，只为听孔丘讲诗书礼乐。

孔丘收学生，规矩只有一条："自行束脩以上，吾未尝无诲焉"——《论语·述而》。束脩是十条干肉，是最薄的见面礼。只要自己备得十条干肉来，我没有不教的。十条干肉，穷人家也凑得出。这一条规矩立下来，门槛就低到地下去了——这跟当时"学在官府"的老规矩，完全是两个世界。

这事传出去，鲁国上下都有些议论。有人觉得孔丘是异想天开——平民百姓也配学礼乐？有人觉得孔丘是沽名钓誉——收几个徒弟能成什么气候？可有一人不这么看。这人就是齐国的国君齐景公。

齐景公这年跟相国晏婴一起来鲁国。一来是外交，二来是听说鲁国有个叫孔丘的，学问了得，想见见。一见面，齐景公就抛出一个大问题："昔秦穆公国小处辟，其霸何也？"——《史记·孔子世家》——从前秦穆公，国家小、地方偏，怎么就称霸了呢？

这问题不好答。秦穆公是春秋五霸之一，可秦国地处西陲，在中原诸侯眼里是蛮夷。齐景公问这话，有他的心思——齐国是大国，齐桓公是第一霸主，齐景公想重振霸业，拿秦穆公说事，是想探探孔丘的见识。

孔丘不慌不忙，答得干脆："秦，国虽小，其志大；处虽辟，行中正。"——《史记·孔子世家》——秦国虽小，志向大；地方虽偏，行事正。这话听着平常，可下一句就见分量了："身举五羖，爵之大夫，起累绁之中，与语三日而授之以政。以此取之，虽王可也，其霸小矣。"

五羖是五张黑羊皮。秦穆公用五张羊皮把百里奚从奴隶堆里赎回来，跟他谈了三天话，就把国政交给他。孔丘说：秦穆公敢这么用人，凭这取人，就算称王都够，称霸还算小的。

齐景公听了，"景公说"——《史记·孔子世家》——高兴，服了。

这一答，答出了孔丘的见识。齐景公问的是"霸"，孔丘答的是"人"。在孔丘看来，秦穆公称霸的根子不在地广兵强，在于敢用百里奚——敢从奴隶堆里拔人才。这话暗合孔丘自己办私学的心思——英雄不问出处，人才不论贵贱。

齐景公走后，孔丘的名声更响了。来求学的人越来越多。院子里坐不下，就搬到院外大树下；树下坐不下，就凑钱在阙里附近筑起一方土坛，坛上杏树成荫——后来人管这叫"杏坛"。

孔丘讲课，不照本宣科。他讲《诗》，讲《书》，讲礼，讲乐，可他讲的不只是字句，是道理，是做人。弟子们记下来，后来编成一部书，叫《论语》。《史记·孔子世家》记："孔子以诗书礼乐教，弟子盖三千焉，身通六艺者七十有二人"——孔子用诗、书、礼、乐教人，弟子大概三千，精通六艺的有七十二人。

三千弟子，七十二贤人。这规模，在春秋时是头一份。

最早来拜师的几个弟子里，有个叫颜路的，鲁国人，比孔丘小六岁。颜路没什么大名气，可他后来生了个儿子，叫颜回——那是后话。还有个叫曾点的，性子散淡，有一回孔丘让弟子们各言其志，曾点说："暮春者，春服既成，冠者五六人，童子六七人，浴乎沂，风乎舞雩，咏而归。"——《论语·先进》——暮春时节，穿上春服，五六个成年人、六七个孩子，在沂水里洗洗澡，在舞雩台上吹吹风，唱着歌回家。孔丘听了，喟然叹曰："吾与点也！"——我赞同曾点啊。曾点这种性子，孔丘也收，也喜欢。

最特别的一个，是子路。子路姓仲名由，字子路，也叫季路，鲁国卞人，比孔丘小九岁。子路初来的时候，"性鄙，好勇力，志伉直"——《史记·仲尼弟子列传》——性子粗，好勇斗狠，脾气直。他来时头上插着雄鸡毛，腰间挂着野猪皮，一副市井豪强打扮，进门就冲孔丘嚷嚷。换别的先生，早把他轰出去了。孔丘不急不恼，慢慢跟他讲礼，讲修身，讲君子之道。子路听着听着，气焰矮了下去。后来子路换了儒服，托人引荐，正式拜入孔丘门下——《史记·仲尼弟子列传》。这一拜，子路跟了孔丘一辈子。

三十岁这年，孔丘立住了。立的不是官位，不是财富，是一条前人没走过的路——把知识从官府里拿出来，给所有想学的人。这条路一走，就走出了中国教育史的新章。

## 讲人物

**孔子：三十而立的"立"**

三十而立，立的是什么？《论语·为政》里孔丘自己说的"三十而立"，历来解说不一。可放在孔丘三十岁这年的处境看，立的是三样东西。

一是立学问。孔丘十五岁志于学，十五年到三十岁，学问打下了底子。他通六艺，熟诗书，明礼制，能跟齐景公论秦穆公称霸，对答如流。这不是临时抱佛脚能来的，是十几年的功夫。

二是立人格。孔丘少时贫贱，干过委吏、乘田这些小吏的活。可他没因为贫贱就自暴自弃，也没因为干粗活就放下学问。"吾少也贱，故多能鄙事"——这话里没有怨，没有愧，是坦荡。三十岁这年，他不求官，不攀贵，在自己家里设教——这是立人格。一个人立不立，不在别人怎么看，在自己怎么活。

三是立事业。三十岁设教授徒，是孔丘一辈子事业的起点。他后来周游列国、删诗书、定礼乐、作春秋，根子都在这条事业上——教育。教育不是他的退路，是他的正路。他一辈子没做大官，可他做了件更大的事——教出了三千弟子，七十二贤人。

【杂记/传说】《庄子·渔父》里记了一则故事，说孔丘六十岁那年，在缁帷之林游学，有个渔父跟他论道，孔丘恭恭敬敬受教。这故事未必可靠，可透出孔丘一辈子好学的性子——六十岁还向渔父请教。三十而立，立的不是"满了"，是"立住了继续学"。

**早期弟子：各色人等**

孔丘最早收的几个弟子，形形色色。

颜路，颜回的父亲，比孔丘小六岁。颜路本事平平，可他生了个好儿子。后来颜回拜师，成了孔丘最得意的弟子。颜路这一拜，拜出了两代师徒缘。

曾点，曾参的父亲，性子散淡，不慕荣利。他想的不是治国平天下，是暮春时节跟朋友孩子洗澡唱歌。孔丘偏偏最欣赏他——"吾与点也"。孔丘欣赏的不是散淡本身，是曾点那份不勉强、不造作的真性情。

子路，最特别的一个。子路初见孔丘，是来砸场子的，结果被孔丘折服。子路这个人，一辈子改不掉粗豪的性子，可他对孔丘忠心耿耿，做了孔丘一辈子的贴身护卫。孔丘后来说"自吾得由，恶言不闻于耳"——《史记·仲尼弟子列传》——自从有了子路，没人敢骂我。子路这一拜，是市井豪强变成了孔门弟子，是粗人开始走君子的路。

孔丘收这些弟子，不挑门第，不挑性子，挑的是有没有求学的心。这就是三十岁这年，孔丘立下的另一个规矩——有教无类。这规矩后来成了中国教育的一块碑。

## 讲背景

孔丘三十岁办私学，不是凭空冒出来的。这事能成，有几层背景。

第一层，鲁国的文化底子。鲁国是周公的封地。周公制礼作乐，鲁国是周文化在东方的重镇。"周礼尽在鲁矣"——《左传·昭公二年》——周礼都在鲁国。晋国韩起出使鲁国，看完鲁国的典籍，感叹"周礼尽在鲁矣，吾乃今知周公之德与周之所以王也"。鲁国保存的礼乐典籍，是诸侯国里最全的。孔丘生在鲁国，长在鲁国，从小耳濡目染，这是他学问的根。换一个国家，未必出得了孔丘。

第二层，官学衰败。西周那会儿，"学在官府"——学校设在官府里，能进去的都是贵族子弟。天子设的叫辟雍，诸侯设的叫泮宫。可到了春秋，周王室衰微，诸侯争霸，官学撑不住了。典籍流散，掌管典籍的官员四散。知识从贵族手里漏出来，民间有了求学的需求，可没人教。这是个空档。

第三层，士阶层崛起。春秋时诸侯争霸，打仗要将军，治国要谋臣。可贵族子弟未必有本事，国君得从下层里挑人。士这个阶层，介于大夫和庶人之间，成了各国抢手的人才。士要做事，就得有本事；要有本事，就得有人教。官学垮了，谁来教？私学应运而生。

孔丘不是第一个办私学的人。据《左传》记载，郑国的邓析早孔丘一百多年就办过私学，教人诉讼，"学讼者不可胜数"——《左传·定公九年》注。可邓析教的是讼术，是技术；孔丘教的是道，是修身齐家治国平天下的学问。孔丘把私学从"术"提升到"道"，这是孔丘的开创。

还有一层，孔丘自己的准备。孔丘三十岁前，学问已经成了。他通六艺，熟诗书，明礼制。他做过委吏、乘田，知道民间疾苦。他十五岁志于学，十五年的功夫下来，到了三十岁，是该立的时候了。齐景公来问，他能答；弟子来学，他能教。这不是运气，是功夫到了。

时势造英雄，英雄也造时势。官学衰败、士阶层崛起、鲁国文化底子厚，这是时势；孔丘学问已成、胆识够、肯放下身段设教授徒，这是英雄。两边凑一块，私学就开张了。

## 讲道理

**司马迁论孔子：高山仰止**

司马迁在《史记·孔子世家》末尾写了一段著名的赞语："《诗》有之：'高山仰止，景行行止。'虽不能至，然心乡往之。余读孔氏书，想见其为人。适鲁，观仲尼庙堂车服礼器，诸生以时习礼其家，余祗回留之不能去云。天下君王至于贤人众矣，当时则荣，没则已焉。孔子布衣，传十余世，学者宗之。自天子王侯，中国言六艺者折中于夫子，可谓至圣矣！"——《史记·孔子世家》。

司马迁这段话，把孔丘的地位说透了。天下的君王贤人多了，活着的时候荣华，死了就完了。可孔丘是个布衣——平民，传了十几代，学者还尊他为宗。天下讲六艺的，都以孔丘为标准。这叫"至圣"。

这段话的分量，在于"布衣"二字。孔丘没做过大官，没立过军功，他靠什么传十余世？靠学问，靠教育，靠三千弟子七十二贤人。三十岁办私学，是这条路的起点。

**朱熹论"三十而立"**

朱熹《论语集注》在"三十而立"下注："有以自立，则守之固而无所事志矣。"——有所自立，就守得牢固，不用再靠志向支撑了。朱熹把"立"讲成"自立"——不是靠外力，是靠自己立住。十五岁志于学，是立了志向；三十岁而立，是志向变成了立身之本。

朱熹这一注，点出"立"的本质——从"志"到"立"，是从想做到做到了。孔丘三十岁，学问立住了，人格立住了，事业立住了。这不是想出来的，是干出来的。

## 问道悟道

**"立"的本质：从依赖到自立**

三十而立，立的是什么？往深里看，是从依赖到自立的转折。

人一辈子，有几个转折。十五岁志于学，是知道自己要什么——这是方向的觉醒。三十岁而立，是能靠自己立住——这是能力的成熟。四十岁不惑，是明白什么对什么错——这是判断的清明。五十岁知天命，是知道自己的边界——这是位置的认清。

孔丘三十岁这年，从"志于学"到"立"，是从"想学"到"能教"的跨越。想学是求人，能教是自立。孔丘不再只是个学生，他成了先生。这一步跨过去，他就不是跟着别人走的人了，是能带别人走的人。

这事古今皆然。诸葛亮二十七岁出山，之前在隆中躬耕读书——那是"志于学"；出山后辅佐刘备，三分天下——那是"立"。王阳明三十七岁龙场悟道，之前格竹子格到病倒——那是求索；悟道后提出"知行合一"——那是立。人都有从"志"到"立"的转折，这个转折一来，人就稳了。

**私学的开创：把知识从特权变成权利**

孔丘办私学，往深里看，是把知识从特权变成权利。

孔丘以前，知识是贵族的专利。典籍在官府，礼乐在官府，教育在官府。平民百姓，连认字的门都没有。知识垄断在少数人手里，社会就僵化——贵族永远是贵族，平民永远是平民。

孔丘把这条线划开了。十条干肉，谁都能来学。这一划，开启了中国文化的一个根本传统——知识不属于某一阶层，属于所有求学者。这事在西方，要晚得多。古希腊的斯巴达，教育是公民的特权，奴隶没份。雅典的柏拉图办学院，收的也是贵族子弟。欧洲中世纪，知识垄断在教会手里。直到文艺复兴、宗教改革，印刷术普及，知识才慢慢流向民间。孔丘在公元前五世纪就做的事，欧洲要到公元十五世纪以后才大规模做。这一前一后，差了两千年。

**教育的力量：一个人改变一个文明**

孔丘一辈子没做过大官，没发过大财，没立过大功。可孔丘是圣人——为什么？因为教育。

孔丘教出三千弟子，七十二贤人。这些弟子把他的学问传到各国，传给弟子的弟子。孔丘死后，儒家分成八派，可都尊孔丘为宗。曾子传子思，子思传孟子，一代传一代，传了两千多年。中国文化能绵延不断，靠的就是这条民间的传承链。

这事有反证。古埃及的象形文字，掌握在祭司手里，是少数人的专利。祭司阶层一衰落，象形文字就失传了，后人是靠罗塞塔石碑才重新破译的。知识垄断在少数人手里，文明就脆弱；知识散在民间，文明就坚韧。孔丘把知识散到民间，是中国文化能绵延两千多年的一个根本原因。

一个人，靠教育，改变了一个文明。这事从三十岁那年开始。

## 结语

三十岁这年，孔丘没做官，没发财，只是在阙里老宅摆了几张几案，收了几个弟子——可这一摆，摆出了中国两千五百年的师道。
