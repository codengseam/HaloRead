# 第13章 孔门三杰——颜回、子路、子贡

## 讲事情

孔门弟子三千，贤人七十二，可要挑几个最出彩的，绕不开这三位——颜回、子路、子贡。

这三人，是孔丘门下的"三杰"。颜回内敛深沉，子路粗豪直率，子贡机敏善辩。三种性子，三种人品，三种路数。孔丘对这三人的教法也不一样——一人一方，因材施教。这三人的故事，得一个一个说。

**先说颜回——陋巷里的乐天派**

颜回，字子渊，鲁国人，少孔丘三十岁——《史记·仲尼弟子列传》。颜回拜师的时候年纪不大，可孔丘一见他就看出了不一样。

颜回家穷。穷到什么程度？"一箪食，一瓢饮，在陋巷"——《论语·雍也》——一筐饭，一瓢水，住在破巷子里。别人都替他愁得慌，"人不堪其忧"——别人都受不了那份苦。可颜回自己呢？"回也不改其乐"——颜回照样乐呵呵的。孔丘见了，连叹两声："贤哉，回也！贤哉，回也！"——好样的颜回！好样的颜回！——《论语·雍也》。

这两声叹，叹出孔丘对颜回的欣赏。颜回的乐，不是没心没肺的乐，是求道的乐。穷是穷，可颜回心里有东西——有学问，有大道，有孔丘教的仁义礼智。心里有这些，外头的苦就不算什么。这就是"安贫乐道"。

颜回的好学，孔丘心里有数。《论语·雍也》记孔丘的话："有颜回者好学，不迁怒，不贰过。不幸短命死矣。今也则亡，未闻好学者也。"——有个叫颜回的好学，不把怒气撒到别人身上，同样的错不犯第二回。颜回的好学，不是死读书，是修身——不迁怒是修心，不贰过是修行。孔丘把"好学"两个字给了颜回，旁人没份。

颜回的性子，跟旁人不一样。孔丘说过："吾与回言终日，不违如愚。退而省其私，亦足以发，回也不愚。"——《论语·为政》——我跟颜回讲一天，他一句反驳都没有，像个傻子。可他退下去自己琢磨，照样能发挥，颜回不傻。

颜回不是没想法，是先听进去，再慢慢消化。这种性子，看着木讷，其实是深沉。子路是听了就说，子贡是听了就问，颜回是听了就悟。三种性子，三种学法。

有一回，孔丘想试试弟子们的悟性，问子贡："女与回也孰愈？"——你跟颜回谁强？子贡实话实说："赐也何敢望回？回也闻一以知十，赐也闻一以知二。"——《论语·公冶长》——我哪敢跟颜回比？颜回听说一个能推出十个，我听说一个只能推出两个。孔丘听了，点头："弗如也，吾与女弗如也。"——是不如，我跟你都不如他。

子贡是孔门里最聪明的人之一，连子贡都自认不如颜回。颜回的悟性，可见一斑。

**再说子路——粗人成君子**

子路的故事，得从初见孔丘说起。

子路姓仲名由，字子路，鲁国卞人，少孔丘九岁——《史记·仲尼弟子列传》。子路初来的时候，"性鄙，好勇力，志伉直，冠雄鸡，佩豭豚，陵暴孔子"——性子粗，好勇斗狠，脾气直，头上插着雄鸡毛，腰间挂着野猪皮，冲着孔丘嚷嚷。孔丘不急不恼，"设礼稍诱子路"——慢慢用礼来引导他。子路听着听着，气焰矮了下去。后来"子路后儒服委质，因门人请为弟子"——子路换了儒服，备了见面礼，托门人引荐，正式拜入孔丘门下——《史记·仲尼弟子列传》。

这一拜，子路跟了孔丘一辈子。

子路的性子，一辈子改不掉——直、勇、莽。孔丘说过"由也好勇过我，无所取材"——《论语·公冶长》——子路好勇超过我，可没地方用。可孔丘离不开子路。子路是孔丘的保镖，孔丘说"自吾得由，恶言不闻于耳"——《史记·仲尼弟子列传》——自从有了子路，没人敢骂我。

子路的直，有时候直得让孔丘哭笑不得。有一回孔丘感叹："道不行，乘桴浮于海。从我者其由与？"——《论语·公冶长》——我的道行不通了，我坐木筏子漂到海上去。跟我走的，大概是子路吧？子路听了，高兴坏了——老师带我！孔丘接着说："由也好勇过我，无所取材。"——子路好勇超过我，可没地方用。这话是夸也是损——子路的忠心可嘉，可子路的勇用错了地方。

还有一回，孔丘对颜回说："用之则行，舍之则藏，惟我与尔有是夫！"——《论语·述而》——用我就干，不用我就藏，只有我跟你能这样吧？子路在旁边听了，醋意上来，问："子行三军，则谁与？"——老师您要是带三军打仗，带谁？孔丘不客气："暴虎冯河，死而无悔者，吾不与也。必也临事而惧，好谋而成者也。"——赤手空拳打老虎，没船硬过河，死了都不悔的，我不带。我要带的是临事而惧、好谋而成的人。——《论语·述而》。

这话是直接敲打子路。子路好勇，勇到敢"暴虎冯河"——赤手打虎、徒步过河。孔丘要的不是这种莽勇，是"临事而惧，好谋而成"——遇事敬畏，靠谋划成事。子路听了，估计脸红一阵白一阵。可子路不记仇，第二天照样跟在孔丘身边。

子路就是这样的人——粗是粗，可粗里有真。他对孔丘的忠心，是孔门里最直的。

**最后说子贡——商贾出身的瑚琏**

子贡，姓端木名赐，卫国人，少孔丘三十一岁——《史记·仲尼弟子列传》。子贡跟颜回、子路不一样——他是商人出身，"利口巧辞"——嘴皮子利索，脑子活。子贡拜师前就会做生意，后来成了孔门首富。

子贡拜师后，有一回问孔丘一个要命的问题："赐也何如？"——《论语·公冶长》——老师，您看我是个什么样的人？孔丘答："女，器也。"——你呀，是个器。子贡追问："何器也？"——什么器？孔丘说："瑚琏也。"——瑚琏。

瑚琏是什么？是宗庙里盛黍稷的玉器，祭祀用的贵重礼器。孔丘是说：子贡啊，你还没到"不器"的境界，可你也是器里头最贵重的那一种——像瑚琏一样，是国之重器。

孔丘说过"君子不器"——《论语·为政》——君子不能只是一件器具。可这会儿他对子贡说"女，器也"——你是个器。这是夸还是贬？是夸。子贡还没到"不器"的境界，可子贡是器里头最贵重的那种——瑚琏，国之重器。这话里有期许，也有遗憾——期许子贡是国之重器，遗憾子贡还没到"不器"的君子境界。

子贡的聪明，孔丘心里有数。《论语·雍也》记孔丘的话："赐也达，于从政乎何有？"——子贡通达事理，从政对他有什么难的？子贡的"达"，是通达——能看透事理，能应变。这种本事，从政绰绰有余。

子贡的好学，也跟旁人不一样。有一回子贡问孔丘："贫而无谄，富而无骄，何如？"——《论语·学而》——穷但不巴结人，富但不骄傲，怎么样？孔丘说："可也。未若贫而乐，富而好礼者也。"——还行。可不如穷还能乐、富还好礼的人。子贡接着说："《诗》云：'如切如磋，如琢如磨'，其斯之谓与？"——《诗经》里说"像切像磋，像琢像磨"，说的就是这个吧？

孔丘听了，高兴："赐也，始可与言《诗》已矣，告诸往而知来者。"——子贡啊，可以跟你谈《诗》了，告诉你过去的你能推知未来的。——《论语·学而》。

"告诸往而知来者"——这是夸子贡的悟性。子贡能举一反三，能从一句话推到另一层意思。这种悟性，颜回有，子贡也有，可路子不一样——颜回是"闻一以知十"，深沉内敛；子贡是"告诸往而知来者"，机敏外发。

**三人性格迥异**

颜回、子路、子贡，三人性格迥异。

颜回内敛深沉，话不多，可悟性极高。孔丘讲一天，他一句不反驳，回去自己琢磨，照样能发挥。颜回的乐，是求道的乐；颜回的穷，是求道的穷。颜回是孔丘最得意的弟子。

子路粗豪直率，话多，性子急，好勇斗狠。孔丘说什么他第一个反应，可常常反应错了。子路的忠心是直的，子路的勇是莽的，子路的人是真的。子路是孔丘的贴身护卫。

子贡机敏善辩，嘴皮子利索，脑子活，会做生意。子贡能举一反三，能跟孔丘论《诗》。子贡的聪明是外发的，子贡的本事是全面的。子贡是孔门里的全才。

三种性子，三种人品。孔丘对三人的教法也不一样。

## 讲人物

**颜回：内敛的悟道者**

颜回的性子，是内敛深沉。他不张扬，不显摆，可心里有数。孔丘说"吾与回言终日，不违如愚"——跟颜回讲一天，他一句反驳都没有，像个傻子。可颜回不是傻，是先听进去，再慢慢消化。"退而省其私，亦足以发"——退下去自己琢磨，照样能发挥。

颜回的悟性，连子贡都服气。子贡自认"闻一以知二"，颜回"闻一以知十"——差了五倍。颜回的悟，不是小聪明，是大智慧。小聪明是反应快，大智慧是消化深。颜回属于后者。

颜回的安贫乐道，不是装样子。颜回穷是真穷，"一箪食一瓢饮在陋巷"，别人都受不了。可颜回"不改其乐"——照样乐。这乐不是没心没肺，是心里有大道。心里有大道，外头的苦就不算什么。这是颜回的境界。

孔丘对颜回，是欣赏，是期许，也是心疼。颜回穷，孔丘不嫌；颜回木讷，孔丘不急。孔丘看颜回，看的是骨子里的东西——不迁怒、不贰过、好学、安贫乐道。这些东西，比口才、比勇武、比钱财都珍贵。

**子路：粗豪的真君子**

子路的性子，是粗豪直率。他不像颜回那样内敛，也不像子贡那样机敏。子路是直来直去，想到什么说什么。

子路的直，有时候直得冒犯孔丘。有一回孔丘去见南子——卫灵公的夫人，名声不好。子路不高兴，"子见南子，子路不说"——《论语·雍也》。孔丘急得发誓："予所否者，天厌之！天厌之！"——我要是做了不对的，老天厌弃我！老天厌弃我！——《论语·雍也》。子路敢给老师甩脸子，孔丘还得跟他发誓。这师徒俩的关系，是孔门里最不讲究礼数的一对。

可子路的直，是真。子路不装，不演，不奉承。孔丘说什么，子路听了就信，信了就做。子路的忠心，是孔门里最直的。

孔丘对子路，是敲打，是引导，也是依赖。子路好勇，孔丘就常压他——"由也好勇过我，无所取材"；子路莽撞，孔丘就提醒他——"暴虎冯河，死而无悔者，吾不与也"。可孔丘离不开子路——"自吾得由，恶言不闻于耳"。子路是孔丘的护卫，也是孔丘的镜子。子路的直，照出孔丘的真。

**子贡：机敏的全才**

子贡的性子，是机敏善辩。他不像颜回那样深沉，也不像子路那样粗豪。子贡是脑子活，嘴皮子利索，会做生意，会论《诗》，会问问题。

子贡的聪明，是外发的。颜回的聪明是内敛的——"闻一以知十"，可不说出来；子贡的聪明是外发的——"告诸往而知来者"，马上就接上来。子贡能举一反三，能跟孔丘论《诗》，能问出"贫而无谄富而无骄何如"这种问题。这种悟性，是机敏的悟性。

子贡的本事，是全面的。他能言善辩——是言语科的高材生；他能做生意——是孔门首富；他能从政——孔丘说他"于从政乎何有"。可子贡还没到"不器"的境界，他是"器"——不过是"瑚琏"，国之重器。

孔丘对子贡，是期许，是点拨，也是警醒。子贡太聪明，聪明到什么都能干，可什么都干得不够"深"。孔丘说他"器也"——是器，不是"不器"的君子。这话里有期许——子贡是瑚琏；也有遗憾——子贡还没到"不器"。孔丘点子贡，是盼他别只做"器"，要做"不器"的君子。

## 讲背景

要懂这三大弟子，得先懂他们出身的差异。

颜回是鲁国人，出身贫寒。颜回的父亲颜路，也是孔丘的弟子，比孔丘小六岁。颜家穷，可颜家是士阶层——没落的小贵族。颜回的穷，是士的穷，不是农民的穷。士的穷，是有学问的穷，是有底子的穷。颜回能安贫乐道，是因为他心里有大道——这大道是士的传承，是孔丘教的仁义礼智。

子路是鲁国卞人，出身市井。子路"性鄙，好勇力"——性子粗，好勇斗狠。子路不是士，是市井豪强。子路的勇，是市井的勇，不是贵族的勇。子路拜师前，是个粗人；拜师后，是个粗人开始走君子的路。子路的转变，是从市井到士的跨越。

子贡是卫国人，出身商贾。子贡"利口巧辞"——嘴皮子利索。子贡不是士，是商人。在春秋那会儿，商人地位不高——"士农工商"，商排在最后。可子贡会做生意，有钱，有脑子。子贡拜师，是从商贾到士的跨越。

三人出身不同——颜回是没落士人，子路是市井豪强，子贡是商贾。三种出身，在春秋那会儿都是"边缘人"——不是当权的世袭贵族，是下层往上走的人。孔丘收这三人，是"有教无类"的典型——不论出身，只论求学的心。

三人拜师的时代背景，是春秋末期的社会流动。当时官学垮了，士阶层崛起，下层往上走的路子打开了。颜回、子路、子贡都是走这条路的人——他们想学本事，想往上走，可官学没了，只能找私学。孔丘的私学，是他们的出路。

三人拜师后，孔丘的私学也进入了鼎盛期。颜回的悟性、子路的忠勇、子贡的才辩——这三人成了孔丘门下的台柱子。后来孔丘周游列国，这三人一直跟着；孔丘讲学，这三人是最出彩的。这三人的早期交往，是孔门最精彩的一段。

## 讲道理

**朱熹论颜回"不改其乐"**

朱熹《论语集注》在"贤哉回也"下注："颜子之贫如此，而处之泰然，不以害其乐，故夫子再言贤哉回也，以深叹美之。"——颜回穷成这样，可处之泰然，不因穷害了乐，所以孔丘连叹两声"贤哉回也"，深深赞叹。——朱熹《论语集注·雍也》。

朱熹这一注，点出颜回的"乐"不是一般的乐，是"处之泰然"——穷归穷，心里不乱。这乐是内心的定，不是外头的顺。朱熹又引程颐的话："颜子之乐，非乐箪瓢陋巷也，不以贫窭累其心而改其所乐也。"——颜回的乐，不是乐在箪瓢陋巷，是不让穷累了自己的心，不改自己的所乐。程颐这话更进一层——颜回乐的不是穷，是道；穷累不着他的心，所以他的乐不改。

**朱熹论子路"好勇"**

朱熹《论语集注》在"由也好勇过我无所取材"下注："夫子善其勇，而病其暴。"——孔丘赞他的勇，可病他的莽。朱熹点出孔丘对子路的两面态度——赞他的忠勇，病他的莽撞。子路的勇是双刃剑，用对了是忠，用错了是莽。孔丘敲打子路，是要他把勇用在正处。

**朱熹论子贡"瑚琏"**

朱熹《论语集注》在"女器也""瑚琏也"下注："器者，有用之成器。瑚琏，宗庙中黍稷之器。夏曰瑚，殷曰琏，周曰簠簋。言子贡虽未至于不器，其亦器之贵者矣。"——器是有用的成器。瑚琏是宗庙里盛黍稷的礼器，夏代叫瑚，殷代叫琏，周代叫簠簋。子贡虽然没到"不器"的境界，可也是器里头最贵重的。——朱熹《论语集注·公冶长》。

朱熹这一注，把孔丘的意思讲透了。"君子不器"是最高标准——君子要通达，不能只是一件器具。可子贡还没到这境界，他是"器"——可这"器"是瑚琏，是国之重器。朱熹点出"虽未至于不器"——孔丘不是贬子贡，是期许他还能再进一步。

## 问道悟道

**因材施教：一人一方**

孔丘教颜回、子路、子贡，是三种法子。

颜回内敛深沉，孔丘就跟他讲大道——"用之则行，舍之则藏，惟我与尔有是夫"——这种话只跟颜回说，因为颜回能接住。子路粗豪直率，孔丘就常敲打他——"暴虎冯河，死而无悔者，吾不与也"——这种话是对子路的药，治他的莽。子贡机敏善辩，孔丘就点他——"女，器也"——这种话是子贡的镜子，照出他的境界。

这就是因材施教。一种病一种药，一个人一方子。颜回的病是"太深"，孔丘就多鼓励；子路的病是"太莽"，孔丘就多敲打；子贡的病是"太聪明"，孔丘就多点拨。三种病，三种药，一个老师，三个弟子。

这事古今皆然。扁鹊看病，齐桓侯不信，扁鹊走了；虢太子假死，扁鹊一针救活——同是扁鹊，看病看人下药。诸葛亮教刘禅，写《出师表》苦口婆心；教姜维，授兵法倾囊相授——同是诸葛亮，教法因人而异。王阳明讲学，对钝根人说"存天理灭人欲"，对利根人说"致良知"——同是王阳明，说法看人。

教育的本质，不是把一个模子套在所有人头上，是看什么人用什么法子。这就是因材施教。

**三种人格的典范**

颜回、子路、子贡，是三种人格的典范。

颜回是"内圣"的典范——安贫乐道，不迁怒不贰过，闻一以知十。颜回的道是向内的，是修心，是悟道。颜回走的是"仁"的路——心里有仁，外头的苦就不算什么。

子路是"行义"的典范——粗豪直率，好勇斗狠，可忠心耿耿。子路的道是向外的，是行动，是担当。子路走的是"勇"的路——吃了人家的饭，不躲人家的难。

子贡是"达才"的典范——机敏善辩，会做生意，会论《诗》。子贡的道是通达的，是应变，是办事。子贡走的是"智"的路——告诸往而知来者，能举一反三。

仁、勇、智——这三种人格，是孔门的三根柱子。颜回的仁，子路的勇，子贡的智——凑在一起，就是一个完整的君子。孔丘教这三人，不是要把他们教成一样的人，是要让他们各成其才。颜回成仁，子路成勇，子贡成智——三人各走各的路，可都从孔丘这里出发。

这事古今皆然。刘备手下，关羽是义，张飞是勇，诸葛亮是智——三种人格，凑成蜀汉的根基。唐太宗手下，房玄龄善谋，杜如晦善断，魏徵敢谏——三种人格，凑成贞观之治。一个团队，需要不同的人格——全是颜回不行，全是子路不行，全是子贡也不行。三种人格凑在一起，才是完整的。

**师徒缘分：一生的相互成就**

孔丘与颜回、子路、子贡的师徒缘，是一辈子的缘。颜回跟了孔丘一辈子，子路跟了孔丘一辈子，子贡也跟了孔丘一辈子。这三人，是孔丘门下最亲近的弟子。

师徒缘分，不是教与学的关系，是相互成就。孔丘教这三人学问，这三人也成就了孔丘。没有颜回，孔丘的"仁"没人接住；没有子路，孔丘没人护卫；没有子贡，孔丘的学问传不出去。这三人，是孔丘的弟子，也是孔丘的知音。

这事古今皆然。苏格拉底教了柏拉图，柏拉图教了亚里士多德——师徒三代，成就了希腊哲学。达摩传慧可，慧可传僧璨——一代传一代，传出了禅宗。师徒缘分，是文化传承的命脉。老师把学问、境界、人格传给学生，学生再传给学生的学生——这条线不断，文化就不断。

孔丘与颜回、子路、子贡的师徒缘，是中华文化里最动人的一段。这段缘分从杏坛开始，一直延续到孔丘去世，延续到弟子们各奔东西，延续到儒家分成八派——可都尊孔丘为宗。这条线，从杏坛出发，传了两千多年。

## 结语

颜回的仁、子路的勇、子贡的智——三种人格从杏坛出发，可这条路不止于讲学，孔丘的道要行于天下，下一步，是鲁国的政坛。
