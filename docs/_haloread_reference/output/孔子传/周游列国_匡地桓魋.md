# 第19章 匡地被围与桓魋之难

## 讲事情

孔子第一次离卫，是被人挤走的。卫灵公派兵监视，孔子待不下去，带着弟子往陈国方向走。可这一走，差点把命走没了。

事情出在匡地。

匡地，在今河南长垣一带，是卫国跟郑国交界的地方。孔子一行路过匡，给孔子驾车的弟子颜刻，拿马鞭子指着城墙的一个缺口说："昔者吾入此，由彼缺也。"——从前我跟阳虎进这个城，就是从这个缺口进的。——《史记·孔子世家》

这话本是无心，可听者有意。匡人一听"阳虎"两个字，眼睛都红了。

阳虎是谁？阳虎是鲁国季氏的家臣，曾经专权鲁国，带兵侵略过匡地，对匡人烧杀抢掠，匡人恨他入骨。巧的是，孔子长得跟阳虎有几分像。——《史记·孔子世家》

匡人一合计：那个驾车的人说跟阳虎来过，车上这位又长得像阳虎——准是阳虎！于是匡人一拥而上，把孔子一行围了起来。

这一围，就是五天。

弟子们慌了。颜回在混乱中走散，后来才找回来。孔子见到颜回，说："吾以汝为死矣。"——我以为你死了。颜回说："子在，回何敢死！"——老师还在，回怎么敢死！——《史记·孔子世家》

颜回这话，是情急之语，也是真心话——老师还在，弟子不敢死。可这话里头，也透着一份紧张：五天了，匡人没放人的意思，谁也不知道接下来会怎样。

子路更急。子路是个能打的人，他想冲出去跟匡人拼命。孔子拦住了他。

孔子拦住子路，说了一段流传两千五百年的话："文王既没，文不在兹乎？天之将丧斯文也，后死者不得与于斯文也；天之未丧斯文也，匡人其如予何？"——文王死后，文化不就在我身上吗？天若要灭这文化，我死后人得不到；天若不灭，匡人能把我怎么样？——《论语·子罕》

这段话，是孔子的底气，也是孔子的使命。

"文王既没，文不在兹乎"——文王死了，文化在我这里。这话听着狂，可不狂。周文王演周易，周公制礼乐，这套文化传到孔子这一代，孔子是接棒的人。孔子觉得自己接的不是一个人的棒，是一代代传下来的文化之棒。这棒在他手里，他不能丢，也不会丢。

"天之将丧斯文也，后死者不得与于斯文也"——天若要灭这文化，我死后人就得不到。这话是反推——我现在还活着，文化还在我身上，说明天没要灭。天若要灭，我早就死了，文化也早就断了。

"天之未丧斯文也，匡人其如予何"——天若不灭，匡人能把我怎么样？这话是结论——我的命在天不在匡人，匡人围得住我的人，围不住天意。

孔子说完这话，该干什么还干什么。他拿出琴来，弹琴唱歌。——《史记·孔子世家》记载，孔子"弦歌不辍"。匡人围着他，他弹琴；弟子们害怕，他弹琴。这琴声里，有镇定，有自信，也有一种"我命在天"的从容。

围了五天，匡人终于弄清楚了——这位不是阳虎，是孔丘。匡人散了。

孔子一行脱险，继续往陈国走。

可这一路上，孔子的麻烦没断。在陈国待了三年，没遇上明主。后来又离开陈国，路过宋国。

在宋国，又出事了。

孔子到了宋国，跟弟子们在一棵大树下习礼——演练礼仪。这本是孔子常做的事，走到哪儿练到哪儿。可这一练，练出了祸事。

宋国有个司马，叫桓魋（tuí），又叫向魋。桓魋是宋景公的宠臣，宋桓族的首领，权势熏天。桓魋看孔子在宋国习礼，收弟子，搞影响力，心里不痛快——孔子在宋国待下去，自己的权势必受影响。桓魋起了杀心。

桓魋没直接动手，先派人去把那棵大树拔了。——《史记·孔子世家》

拔树，是警告，也是威胁——你孔子在我宋国练礼，我让你连个练礼的地方都没有。再不走，下一步拔的就是你的脑袋。

弟子们害怕，催孔子快走。孔子说了一段跟匡地类似的话："天生德于予，桓魋其如予何？"——天生德在我身上，桓魋能把我怎么样？——《论语·述而》

这话跟"匡人其如予何"是一个路子——我的命在天，不在桓魋。可孔子说完这话，还是走了。而且走得匆忙——《史记》说孔子"微服而行"——换了便服，悄悄走的。

这一细节，是司马迁的神来之笔。孔子嘴上说"桓魋其如予何"，腿底下却换了便服溜了。这不是矛盾，这是孔子——自信归自信，该避还是要避。天意归天意，人事归人事。孔子不是莽夫，他知道自己使命未完，不能死在桓魋手里。可他也不能不说那句"天生德于予"——这话是说给弟子听的，也是说给自己听的。

孔子离开宋国，去了郑国。在郑国，孔子跟弟子走散了，一个人站在东门，被郑国人讥为"累累如丧家之狗"。孔子听完，欣然笑曰"然哉！然哉！"。——这一段，详见卷六相关章节，此处不赘。

## 讲人物

**孔子：以天自任的圣人**

孔子在匡地被围，说"文王既没，文不在兹乎"；在宋国遇桓魋，说"天生德于予，桓魋其如予何"。这两段话，是孔子一辈子最"狂"的话。

可这狂，不是真狂，是有底气的狂。孔子的底气来自哪儿？来自他对自己使命的认知。孔子觉得自己接的是文王、周公的棒，这棒不能在他手里断。这棒若断了，文化就断了。所以天不会让他死——天若让他死，等于天要灭这文化，可天不会灭这文化。

这份认知，是孔子的"天命观"。孔子五十而知天命——五十岁知道自己的天命是什么。天命不是算命，是知道自己该干什么、能干什么。孔子的天命，就是传这文化。这天命在他身上，他就不怕——匡人围不住，桓魋杀不了。

可孔子不是莽夫。嘴上说"桓魋其如予何"，腿底下换便服走人。这份"嘴硬腿软"，不是虚伪，是智慧——天意归天意，人事归人事。孔子知道自己的命在天，可他也知道天意要通过人事来实现。该避要避，该走走，可走的时候，话还是要说。这份"走归走，话照说"，是孔子的真功夫。

【杂记/传说】《庄子·秋水》里有一段孔子困于宋的寓言化描写，说孔子"削迹于卫，伐树于宋，穷于商周，围于陈蔡"——在卫被削迹，在宋被伐树，在商周被困，在陈蔡被围。《庄子》是把孔子当反面教材，说孔子"知穷之有命，知通之有时，临大难而不惧者，圣人之勇也"——知道穷有命，知道通有时，临大难不惧，是圣人之勇。《庄子》这话，表面是赞，实是讽——庄子觉得孔子这套"知天命"是装的，真圣人该是庄子那样的逍遥。可庄子的讽，反而衬出孔子的真——孔子是真信天命，不是装的。

**子路：想拼命的勇者**

匡地被围，子路想冲出去跟匡人拼命。这是子路的性格——遇到事，先打。

子路这个人，勇是真勇，可勇得直，勇得莽。孔子评价子路"好勇过我，无所取材"——勇超过我，可惜没处用。——《论语·公冶长》。孔子又说"由也好勇过我，无所取材"——子路好勇过我，可惜没处用。孔子这话，是赞也是叹——赞子路的勇，叹子路的勇没处用。

子路的勇，在匡地差点闯祸。若真冲出去，匡人一拥而上，孔子一行可能就交代了。孔子拦住子路，是拦住了一场祸。可子路这份勇，也是孔子需要的——后来孔子周游列国，子路一路跟着，遇到危险子路在前。子路是孔子的"护卫"，也是孔子的"刹车"——子路敢顶撞孔子，让孔子不至于一意孤行。

子路后来在卫国死，也是死在这份勇上。卫国内乱，子路明知必死还去救主，被人剁成肉酱。子路的死，是子路性格的必然——该上的时候不上，就不是子路了。

**颜回：不敢死的弟子**

匡地被围，颜回走散了。后来找回来，孔子说"吾以汝为死矣"，颜回说"子在，回何敢死"。

颜回这话，听着是情急之语，实是真心话。颜回是孔子最得意的弟子，颜回心里，老师比自己的命重要。老师还在，弟子不敢死——这份"不敢死"，不是怕死，是敬师。

颜回这个人，比孔子小三十岁，是孔子弟子中最聪明的一个。孔子说颜回"不迁怒，不贰过"——不迁怒于人，不重犯同样的错。——《论语·雍也》。又说"吾与回言终日，不违，如愚。退而省其私，亦足以发，回也不愚"——我跟颜回说一天，他不反驳，像傻。退下去看他的私行，足以发明我说的，颜回不傻。——《论语·为政》。颜回是那种"听得多说得少"的弟子，可他听进去的，都化成了行动。

颜回"子在，回何敢死"这话，后来成了师生之情的经典。可这话里头，也藏着颜回的命——颜回比孔子早死，孔子哭之恸"天丧予！天丧予！"。颜回没敢在匡地死，可颜回还是死在孔子前头。这份"不敢死"，最终没能让颜回活过老师。

**桓魋：嫉妒贤能的权臣**

桓魋这个人，是宋国的司马，掌兵权。桓魋想杀孔子，不是孔子得罪了他，是嫉妒——孔子在宋国习礼收徒，影响力大，桓魋怕自己的权势被压下去。

桓魋拔树，是恐吓，不是真杀。若真要杀，直接派兵动手就是，何必拔树？桓魋是怕——怕杀了孔子，宋景公不答应，天下人非议。可桓魋又不能不管——孔子在宋国待下去，自己的权势必受影响。所以桓魋选了拔树这一手——既警告了孔子，又没真动手。

桓魋这一手，是权臣的典型做法——既怕又狠，既想除掉对手又不敢明着来。后来桓魋在宋国也没好下场——宋景公后来想除桓魋，桓魋出奔，死在异国。桓魋这一辈子，跟孔子斗了一回，赢了孔子（孔子走了），输了宋国（自己也被赶走了）。这份"赢小输大"，是嫉妒贤能者的通病。

## 讲背景

匡地被围这事，得从阳虎说起。

阳虎是鲁国季氏的家臣，本来是季氏的家奴，后来专权鲁国，连季氏都受他控制。阳虎掌权时，带兵侵略过匡地——匡地当时是卫国的地方，阳虎侵略匡地，是鲁国跟卫国的冲突。阳虎对匡人烧杀抢掠，匡人恨他入骨。

后来阳虎在鲁国失败，逃到齐国，又从齐国逃到晋国，投奔赵简子。阳虎这个人，是有才的——他能从家奴爬到专权鲁国，是有手段的。可他的才，是权谋的才，不是道义的才。孔子评价阳虎"陪臣执国命"——家臣掌国命，是春秋末期礼崩乐坏的典型。——《论语·季氏》

孔子长得像阳虎，是巧合。可这巧合，差点要了孔子的命。匡人恨阳虎，看到"阳虎"来了，一拥而上。这事儿，是孔子代人受过——阳虎造的孽，孔子来背。可这事也说明，春秋末期，列国之间仇怨深重，一地一城都有血债。孔子周游列国，走的就是这些有血债的地方。

再说宋国桓魋之难。宋国是殷商的后裔，周武王灭商后，封微子启于宋，延续殷祀。宋国在春秋时期不算大国，可也不小。宋景公在位时，宋国跟齐国、晋国、楚国都有交往。桓魋是宋桓族的首领，宋景公的宠臣，掌兵权。宋国的政治，跟鲁国类似——也是公室衰弱，卿大夫专权。桓魋就是宋国的"季氏"。

桓魋怕孔子，不是没道理。孔子在鲁国做大司寇，行摄相事，把鲁国治理得井井有条——这种本事，若到了宋国被重用，桓魋的权势必受影响。桓魋拔树，是怕孔子在宋国扎根。可桓魋这一手，也暴露了宋国的政治——宋景公想用孔子，桓魋不让。宋国跟鲁国一样，国君想做点事，卿大夫不让。

再说孔子的"天命观"。孔子说"文王既没，文不在兹乎"，这话不是凭空说的。孔子一辈子学周礼，习周乐，觉得自己接的是文王、周公的棒。这套文化，从文王到周公，从周公到孔子，一代代传下来。孔子觉得自己是这棒的接棒人——这棒若在他手里断，文化就断了。这份认知，是孔子的"天命"。天命不是天让他做什么，是他知道自己的使命是什么。孔子五十而知天命，知的就是这个。

再说"微服而行"。孔子说"桓魋其如予何"，可还是换了便服走。这事儿看着矛盾，其实不矛盾。孔子的"天命观"不是宿命论——不是"天保佑我，我就什么都不怕"，是"天给我使命，我要靠人事来完成"。该避要避，该走走，可走的时候，话还是要说。这份"嘴硬腿软"，是孔子的真功夫——天意归天意，人事归人事，两者不矛盾。

## 讲道理

司马迁在《史记·孔子世家》里记孔子在匡地被围，说"匡人拘孔子益急，弟子惧"。可孔子"弦歌不辍"——弹琴唱歌不停。司马迁这一笔，是赞——赞孔子的镇定。司马迁没直接评论，可他记的细节，比评论更有力。

《论语·子罕》记孔子"文王既没，文不在兹乎"这段话。这段话，是孔子"天命观"最集中的表述。朱熹在《论语集注》里注这段，说："圣人以天自处，非自大也。以道之所在，天之所不亡也。"——圣人以天自处，不是自大。是道所在的地方，天不会让它亡。朱熹这话，是给孔子辩护——孔子不是自大，是知道道在天不在人。

朱熹又说："匡人其如予何者，言不能违天而害我也。"——匡人能把我怎么样，是说不能违天害我。朱熹这话，把孔子的"天命观"说透了——天不让我死，人害不了我。

《论语·述而》记孔子"天生德于予，桓魋其如予何"。朱熹注："孔子自言天生德于我，桓魋必不能害我。然犹微服而过宋，所谓知命者不立乎岩墙之下也。"——孔子自言天生德于我，桓魋必不能害我。可还是换了便服过宋，就是知命者不立于岩墙之下。朱熹这话，把孔子的"嘴硬腿软"说清楚了——知命不是冒险，是知道天意，还要尽人事。孟子说"知命者不立乎岩墙之下"——知命的人不站在要倒的墙下面。——《孟子·尽心上》。朱熹引这话注孔子，是赞——孔子是真知命。

孟子在《孟子·万章下》里论孔子："孔子，圣之时者也。"——孔子是圣之时者。孟子这话，是说孔子能因时制宜——该进则进，该退则退，该避则避。孔子在匡地"弦歌不辍"，在宋国"微服而行"，是该镇定时镇定，该避时避——这就是"圣之时者"。

## 问道悟道

孔子在匡地和宋国的两段遭遇，藏着几条让人开眼的规律。

**使命感是最大的护身符。**

孔子在匡地被围五天，弟子们慌了，孔子弹琴。孔子为什么能镇定？因为他有使命感——他觉得自己接的是文王、周公的棒，这棒不能在他手里断。这份使命感，让他不怕死——不是不怕，是觉得死不了。天给他的使命没完，天不会让他死。

使命感这东西，是最大的护身符。古今中外，有使命感的人，往往能在最危险的时候镇定。苏格拉底被判死刑，朋友安排好逃路，苏格拉底不逃——他守的是法律的道，哪怕这法律判了他死刑。耶稣进耶路撒冷，知道要被钉十字架，还是进了——他守的是使命的道，哪怕这使命要他的命。文天祥在元大都关了三年，宁死不降——他守的是臣节的道，哪怕这臣节要他的命。

这些人，跟孔子一样，都是有使命感的人。使命感让他们不怕死，可也让他们往往死得惨。孔子没死在匡地，没死在宋国，是运气，也是天意——他的使命还没完。可苏格拉底、耶稣、文天祥，使命完了，人就走了。使命感是护身符，可这护身符只在使命未完时管用。使命一完，人就走了。

**天意归天意，人事归人事。**

孔子说"桓魋其如予何"，可还是换了便服走。这事儿看着矛盾，其实不矛盾。孔子的"天命观"不是宿命论——不是"天保佑我，我就什么都不怕"，是"天给我使命，我要靠人事来完成"。

这份"天意归天意，人事归人事"，是儒家最硬的智慧。天意是大方向，人事是小细节。大方向在天，小细节在人。人不能改大方向，可人能改小细节。孔子知道天意不让他死在桓魋手里，可他还是换便服走——因为天意要通过人事来实现。若孔子不换便服，硬着头皮等桓魋来杀，那不是信天命，是找死。

这事古今中外都一样。诸葛亮六出祁山，知道"汉室难复兴"，可还是出——天意归天意，人事归人事。诸葛亮不能改汉室将亡的大方向，可他能尽人事——出祁山，伐中原，能做多少做多少。后来诸葛亮死在五丈原，是人事尽完了。岳飞抗金，知道宋高宗不想真打，可还是打——天意归天意，人事归人事。岳飞不能改宋高宗求和的大方向，可他能尽人事——打仗，收复失地，能做多少做多少。后来岳飞被秦桧害死，是人事尽完了。

孔子、诸葛亮、岳飞，都是"知其不可而为之"的人。他们知道大方向改不了，可小细节能做。这份"做"，不是傻，是尽人事。尽人事不是改天意，是让天意通过自己来实现。

**代人受过是圣人的命。**

孔子长得像阳虎，被匡人围。阳虎造的孽，孔子来背。这事儿，是代人受过。

圣人往往代人受过。苏格拉底被控"腐蚀青年"，可苏格拉底没腐蚀谁——他只是跟人辩论。耶稣被钉十字架，罪名是"犹太人的王"，可耶稣没称王——他只是传道。孔子被匡人围，因为长得像阳虎——可孔子不是阳虎。这些人，都是代人受过。

代人受过，是圣人的命。圣人活在世上，往往被误解，被冤枉，被代人受过。可圣人不变——孔子被围，不恨匡人；苏格拉底被判，不恨雅典人；耶稣被钉，不恨罗马人。圣人的"不恨"，不是装的，是知道——代人受过是命，命改不了，可道还能传。孔子传了道，苏格拉底传了哲学，耶稣传了福音。道传下去了，代人过就没白代。

---

## 结语

圣人嘴上说"其如予何"，腿底下换了便服——天意归天意，人事归人事，这才是真知命。
