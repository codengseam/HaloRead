# 第4章 三岁丧父与曲阜孤儿

## 讲事情

鲁襄公二十四年前后，陬邑大夫府邸里传来一阵哭声。

那哭声不大，却撕心裂肺。

叔梁纥躺在床上，眼睛已经闭上了。这个曾经在偪阳之战中"力举城门"的勇士，这个七十多岁还能生儿子的硬汉，终究没能熬过这一场病。他走的时候，身边那个三岁的男孩还不太懂事，只拽着父亲的衣角，仰头看大人们哭。

那男孩叫孔丘。

颜徵在跪在床前，年纪不过二十出头。她没有嚎啕，只是把脸埋在丈夫的手背上，肩膀一抖一抖。府里的人进进出出，忙着办丧事，没人顾得上这个年轻的妾室和她怀里的孩子。

按礼制，叔梁纥要葬回防山祖茔。防山在鲁国东边，是孔氏一族世代安息的地方。送葬那天，颜徵在抱着孔丘站在府门口，看着灵柩被抬出去，越走越远，直到拐过街角，再也看不见。

她没有跟去。

不是不想去，是不能去。

按当时鲁国的礼俗，年轻寡妇不宜抛头露面参加丈夫葬礼，何况她只是妾室，并非正妻。叔梁纥生前有过正妻，生过九个女儿；后来又娶妾，生了个儿子叫孟皮，是个跛子。颜徵在是最后娶的，孔丘是最小的儿子。这桩婚事本就不合常规——叔梁纥七十出头，颜徵在不到二十，司马迁写"纥与颜氏女野合而生孔子"，"野合"二字，说的就是这年龄悬殊、不合常礼的结合。

如今丈夫一走，颜徵在在府里的处境，一下子就尴尬起来。

她不是正妻，没有主持家事的资格。孟皮虽是跛子，却是庶长子，论宗法，比孔丘更靠近继承人的位置。府里上上下下的眼色，颜徵在看得清清楚楚——这地方，待不下去了。

她做了一个决定：走。

带着孔丘，离开陬邑大夫府，去曲阜城里讨生活。

曲阜是鲁国都城，离陬邑不算太远。颜徵在的娘家颜氏，是曲阜城里的小户人家，算不上贵族豪门，世代以礼乐传家，却没什么势力。颜徵在选的地方叫阙里，在曲阜城东南角，离鲁国宫城不远，是个平民聚居的巷子。住在这里的人，有做小买卖的，有当差的，有替贵族管杂务的，日子都不算宽裕。

颜徵在租了一间小屋，带着孔丘住了下来。

从大夫府邸到阙里小巷，这一步走出去，就再没回头。

日子一下子就紧巴了。从前在陬邑，好歹是大夫之家，吃穿不愁；如今在阙里，柴米油盐都得自己张罗。颜徵在年轻，手脚勤快，纺线、缝补、浆洗，什么活都干。孔丘还小，帮不上忙，就蹲在门口玩泥巴。他有个习惯，跟别的小孩不一样——别的小孩玩泥巴是糊一脸，他玩泥巴，是摆小碗小碟，磕头作揖，模仿大人祭祀的样子。

"这孩子，怎么净玩这些？"邻居看了觉得稀奇。

颜徵在没说话，只是把儿子拉回屋里。

她心里清楚，这孩子早慧。三岁没了爹，话却说得比同龄孩子利索，眼睛看什么都记得住。阙里离鲁国宫城近，逢到宫里有祭祀、朝会，钟鼓之声隔着墙都能听见。孔丘听见乐声就挪不动步，非要扒着门缝看个究竟。颜徵在也不拦他，由着他看。

曲阜是周公的封地。周公制礼作乐，鲁国是诸侯里保存周礼最完整的国家，"周礼尽在鲁矣"。城里隔三差五就有祭祀活动，太庙里钟鼓齐鸣，大夫们冠冕朝服，进退有度。这种环境，对别的孩子是背景音，对孔丘，却是启蒙课。

颜徵在没读过多少书，但她知道一件事：这孩子将来不能跟她一样。

她得让他读书，让他懂礼，让他有出息。可她一个寡妇，拿什么供孩子读书？鲁国那时候，学问都掌握在贵族手里，平民想识字，得有人教，有人带。颜徵在能做的，就是尽量把孔丘往那个圈子里推——阙里离太庙近，离贵族们活动的地方近，孔丘耳濡目染，总能沾点光。

日子就这么一天天过。

孔丘五岁那年，阙里巷口来了个卖祭器的老头。孔丘蹲在摊子前看了一下午，回家就用泥巴捏了一堆俎豆——那是祭祀时盛祭品的礼器。他摆了一地，自己跪在中间，学着大人的样子行礼。

颜徵在从屋里出来，看见这一幕，眼眶就红了。

她没说什么，转身回屋，继续纺线。

这一纺，就是好几年。

## 讲人物

**颜徵在**：二十出头守寡，带着三岁幼子离开大夫府邸，独自撑起一个家。她不是那种哭哭啼啼的弱女子，做得出"离开"这个决定，本身就是一种决断。在陬邑大夫府，她好歹有个名分，吃穿不愁；到了阙里，一切从零开始。她选阙里，不是随便选的——那里离鲁国宫城近，礼乐氛围浓，对孩子成长有利。一个年轻寡妇，能有这份远见，不容易。

她出身颜氏，不是大族，却世代沾染礼乐之风。她教不了孔丘多少学问，但她给了孔丘两样东西：一是清贫中的体面，二是接触礼乐的机会。司马迁写孔子"为儿嬉戏，常陈俎豆，设礼容"，这"常"字背后，是颜徵在日复一日的默许和引导。

【杂记/传说】后世有说法称颜徵在年轻时与叔梁纥"祷于尼丘"而生孔子，此事《史记》有载，但"野合"二字历来争议不断，有说不合礼制，有说野外相会，至今没有定论。颜徵在的身世，史料所载不多，她像一道影子，只在对孔子的叙述里偶尔显形。

**孔丘**：三岁丧父的孩子，往往有两种走向，要么早熟，要么孤僻。孔丘走的是前一条路。他没有父亲可以依靠，却也没有父亲的影子压着，反而长成了一种早慧的沉稳。"为儿嬉戏，常陈俎豆"——别的孩子玩泥巴，他玩祭祀。这不是天赋异禀，是环境使然：阙里离太庙近，他天天看，天天学，礼乐对他来说不是书本上的字，是眼前活生生的场面。

**叔梁纥**：孔子记忆里几乎没有这个父亲。叔梁纥死时孔子才三岁，留下的只有一个"勇士"的名声和一座远在防山的坟。偪阳之战"力举城门"的事迹，孔子后来是听别人说的。这个父亲对孔子的影响，与其说是言传身教，不如说是缺席本身——没有父亲的孩子，会更早学会自己撑着。

## 讲背景

要明白颜徵在为什么"走"，得先明白鲁国那时候的宗法制度。

鲁国是周公的封地，宗法制执行得格外严格。一个大夫之家，正妻生的儿子叫嫡子，妾室生的叫庶子，嫡子继承家业，庶子只能靠边站。叔梁纥的正妻生了九个女儿，没儿子；妾室生了个孟皮，是个跛子；最后才娶了颜徵在，生了孔丘。论嫡庶，孔丘是庶子；论长幼，孟皮在前。叔梁纥一死，这个家的继承权怎么分，是个麻烦事。

更麻烦的是颜徵在的身份。她不是正妻，是妾，而且年纪太轻，比叔梁纥的大女儿还小。丈夫在世时，有丈夫护着，没人敢说什么；丈夫一走，她在府里就成了一个尴尬的存在——论地位，比下人高不了多少；论辈分，又是主子的妾。这种处境，待下去只有受气。

颜徵在选曲阜阙里，还有一层考虑：鲁国是"礼乐之邦"，周公的封地，保存周礼最完整。晋国韩起出使鲁国，曾感叹"周礼尽在鲁矣"。把孔丘放在这样的环境里，耳濡目染，比待在陬邑那个小地方强得多。陬邑虽是叔梁纥的封地，但那是个偏僻小城，没什么文化氛围。曲阜不一样，太庙、泮宫、贵族们的朝会祭祀，全是活的教材。

再说寡妇的处境。春秋时期，寡妇改嫁并不稀奇，但颜徵在没改嫁。她带着孔丘单过，这在当时需要相当的勇气。一个年轻女人，没了丈夫，带着个孩子，在平民巷子里讨生活，要顶住多少闲言碎语，可想而知。她能撑下来，靠的是一股韧劲——这股韧劲，后来也传给了孔丘。

## 讲道理

司马迁在《史记·孔子世家》里写"丘生而叔梁纥死，葬于防山"，又写"孔子疑其父墓处，母讳之也"。一个"讳"字，道尽了颜徵在的难处——她不是不想告诉儿子父亲葬在哪，是按礼制她根本没去过葬礼，无从告知。司马迁把这件事记下来，本身就是一种态度：圣人的出身，并不光鲜。

孟子论圣人，有"天将降大任于是人也，必先苦其心志，劳其筋骨"之语——《孟子·告子下》。这话虽非专指孔子，但用来观照孔子少孤的经历，却再贴切不过。三岁丧父，少年贫贱，正是"苦其心志"的开端。

## 问道悟道

三岁丧父这件事，对孔子一生的影响，远比表面看到的深。

第一层，是身份的悬空。叔梁纥是陬邑大夫，孔丘本是大夫之子，按理该有封地、有家业、有体面的出身。可父亲一死，母亲带他离开大夫府，他就从"大夫之子"变成了"阙里平民"。这身份的跌落，让孔子从小就明白一件事：出身不能靠一辈子，得靠自己。后来他说"吾少也贱，故多能鄙事"，这"贱"字的根子，就埋在三岁丧父这一年。

第二层，是礼乐的启蒙。颜徵在选阙里，无意中给孔子选了一条路。阙里离太庙近，礼乐之声日日入耳，孔子"为儿嬉戏，常陈俎豆"，不是天赋，是环境。这印证了一个道理：一个人早年的环境，往往比天赋更重要。孟母三迁，颜母一迁，异曲同工。

第三层，是寡母的韧性。颜徵在不改嫁，独自养大孔子，这种坚韧，是孔子性格里"知其不可而为之"那股劲的源头。一个在寡母膝下长大的孩子，会比别人更早懂得什么叫"撑着"，什么叫"不认输"。

古今中外，少年孤露而终成大器者，比比皆是。孟子幼年丧父，由母亲抚养成人，孟母三迁择邻，断机教子，与颜徵在携子迁居阙里，何其相似。范仲淹两岁丧父，母亲改嫁朱氏，他少年寄居寺庙苦读，后来写下"先天下之忧而忧"。欧阳修四岁丧父，母亲郑氏以荻画地教他识字，终成一代文宗。这些人有个共同点：少时失怙，却都有一位不肯认命的母亲。

母亲的眼界，往往决定孩子的起点。颜徵在带孔子去阙里，孟母带孟子搬了三次家，范仲淹的母亲改嫁也要供他读书——这些母亲未必读过多少书，却都懂得一个道理：环境塑造人，教育改变命。孔子后来能成为万世师表，起点就在阙里那间小屋，就在颜徵在纺车旁的那盏油灯下。

## 结语

三岁丧父的孩子，后来撑起了两千年的礼乐。
