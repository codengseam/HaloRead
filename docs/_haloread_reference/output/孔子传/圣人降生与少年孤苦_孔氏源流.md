# 第1章 孔氏源流——从宋国殷商后裔到鲁国陬邑

## 讲事情

要讲孔子，得先讲他这个"孔"字从哪儿来。

这得从一场亡国说起。

公元前十一世纪，周武王伐纣，牧野一战，商纣王在鹿台自焚，殷商六百年的天下，一夜崩塌。可周人得了天下，却没把殷人赶尽杀绝。武王封纣王之子武庚于殷墟，让他继续奉祀商朝祖先；又派自己的三个弟弟管叔、蔡叔、霍叔盯着，叫"三监"。谁料武王一死，武庚联合三监反了。周公东征，三年平乱，杀了武庚。

这回周人学乖了——光盯着不行，得找个能服众的殷人来看守殷商遗民。挑来挑去，挑中了一个人。

这人叫微子启。

微子启是商纣王的庶兄，纣王无道时他屡屡劝谏，纣王不听，他干脆抱着祭器跑到周人这边来，算是早早"弃暗投明"。周公觉得这人可用，便封他于宋，让他继承商朝的祭祀，管理殷商遗民。宋国就这么来的，都城在商丘（今河南商丘）。

"微子开（启）者，殷帝乙之首子而帝纣之庶兄也"——《史记·宋微子世家》。周公封微子于宋，是为了"奉其先祀"——让商朝的香火不断。这一封，就封出了一个延续八百年的诸侯国，也封出了后来孔氏这一支血脉。

微子启传了十代，传到了宋湣公。宋湣公有个弟弟叫炀公，湣公死后炀公即位，湣公的儿子鲋祀不服，杀了炀公，自己当了国君，就是宋厉公。这一支往下传，传到宋殇公的时候，朝中有个大臣，叫孔父嘉。

孔父嘉，是孔氏这一支能往上追到的第一个有名有姓的祖先。

他是宋国的大司马，掌兵权。宋殇公在位十年，连年打仗，打了十一仗，国人都怨。孔父嘉是主战派，自然也跟着背骂名。可他忠于殇公，殇公也信他，君臣二人，一个敢打，一个敢撑。

偏偏这时候，出了一个人。

这人叫华督，是宋国的太宰，跟孔父嘉同朝为官，论资历论手腕都不在孔父嘉之下。华督有个心思——他想当国相，可孔父嘉挡在前面，他过不去。

机会来了。

鲁桓公元年（公元前711年）的一天，华督走在宋国都城的街上，迎面过来一辆车。车上坐着一个女人，华督一抬眼，愣住了。

那女人美得不寻常。

华督盯着她看，目送她远去，嘴里喃喃了一句："美而艳。"

这女人是谁？孔父嘉的妻子。

《左传·桓公元年》记这一幕，只有寥寥数字："宋华父督见孔父之妻于路，目逆而送之，曰：'美而艳。'"可这十几个字里，藏着一桩血案的引子。华督这一眼，看上的不只是孔父嘉的妻子，更是孔父嘉的命。

华督是个狠人。他既起了歹意，就不动声色地布局。他知道，孔父嘉是司马，手里有兵，硬来不行；可孔父嘉连年主战，国人怨气大，这怨气，正好可以借。

于是华督在国中散布流言："殇公即位十年，打了十一仗，民不聊生，都是孔父嘉的罪过！他要是不死，宋国没有太平日子！"

这话传了一年。

到了鲁桓公二年（公元前710年）春，华督动手了。

他带着一帮人，直接攻进孔氏府邸，杀了孔父嘉，夺了他的妻子。宋殇公闻讯大怒——孔父嘉是他最信任的大臣，华督擅杀司马，眼里还有没有国君？殇公要治华督的罪。

可华督比他快。殇公还没动手，华督先下手为强，把殇公也杀了。

一杀大臣，二杀国君，华督在宋国一手遮天。他从郑国接回公子冯，立为宋庄公，自己稳坐太宰之位。从此宋国大权，落入华氏之手。

孔父嘉一死，孔氏这一支就塌了半边天。

幸存下来的人，心里清楚：宋国待不下去了。华督掌权，华氏势大，孔父嘉的子孙在宋国朝堂上再无立足之地，弄不好哪天就被斩草除根。

得走。

往哪儿走？宋国北边是晋、郑，那是中原争霸的漩涡；南边是楚，蛮夷之地；东边，是鲁国。

鲁国是个好去处。鲁国是周公的封地，礼乐之邦，最讲规矩；离宋国不远不近，华督的手伸不到那儿；更重要的是，鲁国对殷商遗民不算排斥——周初分封，殷人六族分给鲁国，鲁国本就有殷商遗民聚居。一个落难的殷商后裔，去鲁国讨生活，不算太突兀。

孔父嘉的后人，就这么踏上了东迁的路。

关于这一段，《史记·孔子世家》只记了一句："其先宋人也，曰孔防叔。"——孔子的先祖是宋国人，叫孔防叔。孔防叔是孔父嘉的曾孙（一说玄孙），正是他带着家族迁到了鲁国。至于更细的细节，《史记》没写，但《孔子家语·本姓解》补了一笔：孔父嘉之后，"其五代孙曰木金父，奔鲁，因家焉"。

不管是孔防叔还是木金父，总之孔氏这一支，从宋国迁到了鲁国，从此成了鲁人。

到了鲁国，落脚在陬邑（今山东曲阜东南）。陬邑是个小地方，算不上繁华，可也正因小，才好安身——大地方人多眼杂，小地方没人盯着。孔氏在这里置了几亩薄田，慢慢扎下根。

传了几代，传到叔梁纥这一代。

叔梁纥是个武人，力气大，胆子也大，在鲁国当了个陬邑大夫——不是什么大官，管着一个邑的事务，相当于一个乡镇的镇长。可这个"大夫"的名分，保住了孔氏这一支在鲁国的体面。从宋国落难贵族，到鲁国陬邑大夫，孔氏这一支，算是重新站稳了脚跟。

只是叔梁纥自己也没想到，他这个陬邑大夫，最大的功业不是打仗，不是治邑，而是生了一个儿子。

那个儿子，叫孔丘。

## 讲人物

**微子启**：商纣王的庶兄，殷商灭亡后归周，被封于宋，奉商祀。这个人最难得的，是"识时务"。纣王无道时他劝谏，劝不动就抱着祭器跑——这不是背叛，是清醒。他知道商朝气数已尽，与其陪葬，不如把商朝的香火保下来。周公封他于宋，正是看中他这份清醒。微子启这一支，成了殷商遗民里最体面的一支，既保住了祖宗祭祀，又在新朝有了立足之地。司马迁在《史记·宋微子世家》里写他，笔端是带着敬意的——"微子故能仁贤，乃代武庚"，一个"仁贤"，一个"代"，定下了他的历史位置。

**孔父嘉**：宋国大司马，孔氏这一支的第一个显祖。他是个悲剧人物——忠心耿耿，却死于非命。宋殇公连年打仗，他是主战派，背了国人怨气；华督觊觎他妻子，他浑然不觉。等华督动手时，他连反抗的机会都没有。《左传》记他被杀，只一句"二年春，宋督攻孔氏，杀孔父而取其妻"，冷冰冰的，可这冷冰冰背后，是一个家族的倾覆。孔父嘉之死，逼得孔氏东迁，反而成就了后来的孔子——这是他生前万万想不到的。

**华督**：宋国太宰，孔父嘉的政敌。这人是个典型的"权谋家"——为达目的不择手段。看上人家妻子，就散布流言煽动民怨；杀了大臣不够，连国君也一起杀。一杀大臣，二杀国君，宋国朝堂被他搅得天翻地覆，自己却稳坐钓鱼台。华督这种人，历朝历代都有，他们活得滋润，可他们搅乱的局面，往往要几代人去收拾。孔父嘉死在华督手里，是孔氏的劫，也是孔氏的转——没有这场劫，就没有东迁鲁国，就没有后来的孔子。

**叔梁纥**：孔氏迁鲁后的第三代（一说第四代），陬邑大夫。从宋国落难贵族到鲁国乡镇大夫，孔氏这一支的家道，到叔梁纥这里算是稳住了。叔梁纥是个武人，勇力过人，后来在偪阳之战中"力举城门"救同伴，名留青史。可他最大的心事不是打仗，是没儿子——这事，下一章细说。

## 讲背景

要明白孔氏为什么从宋国迁到鲁国，得先明白春秋初年的政治格局。

周武王灭商后，分封天下。殷商遗民不能全杀，也不能全留，周公的办法是"分而治之"：一部分留在殷墟，由武庚管；一部分迁到洛阳，由周人直接控制；一部分分给诸侯，比如鲁国就分到了"殷民六族"。微子启封于宋，是另一种安排——让商朝王族自己管殷商遗民，奉商祀，算是给商朝一个体面的收场。

宋国的地位很特殊。它是"公爵"，在诸侯里爵位最高（齐、鲁都是侯爵），因为它是"客"——周人以客礼待殷商后裔，给足面子。可爵位高不等于实力强。宋国地处中原四战之地，北有晋、齐，南有楚，东有鲁，西有郑、卫，夹在中间，谁强它就挨谁的打。宋国想称霸，宋襄公试过，泓水一战败给楚国，"仁义之师"成了笑柄。从此宋国就老实了，在大国之间左右逢源，勉强保命。

孔父嘉之死，就发生在宋国这种夹缝求存的大背景下。宋殇公连年打仗，打的是谁？主要是郑国。宋郑两国世仇，打来打去没个完。孔父嘉是主战派，华督是主和派（至少借主和上位），两派相争，华督胜了。这种大臣相争、借民怨上位的事，春秋各国都在演，宋国只是其中一例。

再说为什么奔鲁。鲁国跟宋国，关系不一般。鲁国是周公的封地，宋国是微子的封地，一个是周室功臣之后，一个是殷商王族之后，两国世代通婚。鲁桓公的夫人就是宋国人（文姜，虽然她后来闹出丑闻）。孔氏迁鲁，不算投奔陌生之地，多少有点渊源。再加上鲁国礼乐文化浓厚，对殷商遗民有容纳的传统（鲁国本就有殷民六族），孔氏选鲁国，是经过盘算的。

至于陬邑，那是个小地方，在鲁国东南，离都城曲阜不远。孔氏选这儿落脚，一是小地方好安身，二是离曲阜近，能沾点文化氛围。后来叔梁纥当陬邑大夫，就是在这块地盘上扎下的根。陬邑虽小，却成了孔氏在鲁国的发祥地——孔子就生在这里。

还有一层背景：春秋时期，"士"这个阶层是流动的。一个家族在一国待不下去，迁到另一国重新开始，是常事。孔氏从宋迁鲁，不算孤例。晋国的范氏、中行氏后来逃到齐国，楚国的伍子胥逃到吴国，都是这种"士"的流动。春秋之所以叫"春秋"，就是因为这种流动和重组从未停止。孔氏的东迁，是这大潮里的一朵小浪花，可这朵浪花，后来翻出了天大的动静。

## 讲道理

司马迁在《史记·孔子世家》开篇就交代孔子的来历："孔子生鲁昌平乡陬邑。其先宋人也，曰孔防叔。防叔生伯夏，伯夏生叔梁纥。"司马迁特意把孔子的世系从宋国写起，是要告诉读者：孔子虽是鲁人，根却在殷商。这一笔，为后面写孔子"从周"还是"从殷"埋下了伏笔——孔子后来常说"郁郁乎文哉，吾从周"，可他骨子里流的是殷人的血。司马迁写世家，开篇就立这个根，是有深意的。

《左传》记孔父嘉之死，用的是"宋督攻孔氏，杀孔父而取其妻"——一个"攻"字，一个"取"字，把华督的跋扈写尽了。左氏记事，往往不动声色，可这不动声色里，褒贬自见。华督杀大臣、夺其妻、弑其君，左氏一字未评，却字字是控诉。这就是所谓的"春秋笔法"——不议论，只记事，让事实自己说话。

## 问道悟道

孔氏从宋国迁到鲁国，这件事看似只是一个家族的迁徙，可往深了想，里头藏着几层道理。

第一层，是"血脉的漂流"。一个家族的根，往往不在它最终落脚的地方，而在它出发的地方。孔氏在鲁国住了三代，就成了地道的鲁人；可往上追，他们是宋人；再往上追，他们是殷商王族。孔子后来在宋国娶妻（亓官氏），在宋国寻根，不是没有缘故的——那是他祖上的故土。一个人的身份，不能只看他出生在哪里，还得看他的血脉从哪里来。中国人讲"寻根"，寻的就是这条血脉的来路。

第二层，是"灾难的转化"。孔父嘉之死，对孔氏是灭顶之灾；可正是这场灾，逼着孔氏东迁鲁国，才有了后来的孔子。要是孔父嘉不死，孔氏继续在宋国当贵族，孔子可能就是个宋国的大夫，在宋国朝堂上跟华氏、向氏斗来斗去，历史上多一个政客，少一个圣人。灾难逼人迁徙，迁徙改变命运——这个规律，古今中外反复应验。犹太人被逐出耶路撒冷，散落欧洲，反而孕育了顽强的文化生命力；西罗马灭亡后，罗马典籍靠拜占庭和阿拉伯人保存，才有了后来的文艺复兴。灾难是灭门之祸，也是涅槃之机，全看活下来的人怎么走下一步。

第三层，是"边缘的力量"。孔氏迁到鲁国，是外来户，不是大族，到叔梁纥这一代也只是个陬邑大夫，挤不进鲁国权力的核心。可正是这种"边缘"位置，反而成全了孔子。他要是在权力核心里长大，从小耳濡目染的是权谋倾轧，未必能跳出那个圈子看世界；正因为他在边缘，既沾了鲁国礼乐文化的光，又没被权力漩涡吞没，反而能以一个"局外人"的视角，审视那个礼崩乐坏的时代。边缘，有时候是观察中心最好的位置。

古今中外，这种"边缘出圣人"的事，屡见不鲜。苏格拉底是雅典的石匠之子，不算贵族；耶稣生在伯利恒一个小木匠家，加利利在当时是犹太地的边陲；释迦牟尼虽是王子，可他走出王宫、在边缘悟道，才成了佛。中心的人忙着争权夺利，边缘的人才有闲心仰望星空。孔氏从宋国迁到鲁国，从王族沦为大夫，从中心退到边缘——这一退，退出了一个万世师表。

## 结语

殷商的血脉流落到鲁国陬邑，却在边缘长出了一棵参天大树。
