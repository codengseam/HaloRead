# 人际互惠的尺度

## 帮了三次，第四次拒绝就翻脸

小周在组里是出了名的热心。

同事老李赶项目，让他帮忙跑个数据，他跑了。下周老李又来，说"上次那个口径你再帮我调一下"，他调了。再下周，老李的客户报告写不完，让他帮忙润色一下措辞，他也润了。

第四次，老李又来了。这回是周末，让他帮忙做一个本来该老李自己做的PPT。小周那天约了家人看病，犹豫了一下，说"这次可能帮不上，我有事"。

老李脸一沉："就你忙？前几次不都帮了吗？这次怎么就不行了？"

小周愣住。他帮了三次，没换来一句正式的谢谢，倒换来第四次拒绝时的翻脸。

更让他窝火的是，他后来回想，老李这三次帮忙，从来没还过——没请他吃过一顿饭，没在他忙的时候搭过一次手，连一句"以后你有事找我"都没说过。三次帮忙，在小周这边是人情，在老李那边是"小周好说话，找他准没错"。

小周不是个例。职场里这种事天天发生：你帮别人帮成了习惯，对方把你的帮忙当成了义务；你拒绝一次，对方比你还委屈。

反过来还有一种。小周有次被老李帮过一个小忙——一份文件老李顺手帮他打印了。从那以后，老李动不动就提"上次我帮你打印那份文件"，言下之意是"你欠我的"。一份打印纸的事，被老李念叨了半年，最后小周宁愿自己跑两趟也不愿再让他帮。

帮人帮出怨，被帮帮出债。这就是互惠失了尺度。

## 互惠是硬通货，但过度互惠是绑架

人际互惠是社交的硬通货。你帮我，我帮你，关系就这么转起来。谁都不帮谁的关系，叫陌生人。

但互惠这条法则有个临界点。在临界点之内，互惠是润滑剂，让关系越处越顺；过了临界点，互惠就变成绑架，让关系越处越累。

临界点在哪？在"对等"两个字上。

你帮我一次，我记着，找机会还一次，这叫对等。你帮我一次，我从此把你当恩人供着，你说什么我听什么，这叫被绑架。你帮我一次，我帮你十次还嫌不够，这叫被压榨。

小周被老李翻脸，是因为他帮了三次没让老李还过一次——他主动放弃了"对等"，让老李觉得"找小周帮忙是零成本"。零成本的东西没人珍惜，所以第四次拒绝时老李心里盘算的是"小周变卦了"，根本没往"小周有难处"那想。

老李对小周那份打印纸的反复念叨，是另一种失度——把小恩放大成大恩，用放大了的人情反复索取。一份打印纸值不了几个人情，但老李把它念成了"我帮过你大忙"，这就是把互惠当成了要挟。

互惠的尺度，一句话能讲透：**小恩常还，大恩慎受**。

小恩要常还，是因为小恩不还就会积成大恩，大恩一积，关系就失衡。同事顺手帮你带杯咖啡，你下次顺手帮他带一份早餐，这事就清了。你不还，他记着，下次他帮你带第二杯咖啡时，心里就开始算账了。

大恩要慎受，是因为大恩还不起。还不起的恩，要么变成你心里的债，压着你处处让步；要么变成对方的筹码，动不动拿出来要你"还人情"。无论哪种，关系都拧巴了。

## 侯嬴的自刭：大恩难报

《战国策》里有个故事，把"大恩的沉重"讲到了极致。

战国时秦军围赵，赵国向魏国求救。魏王怕秦，不敢发兵。魏国的信陵君想救赵，却调不动军队——兵符在魏王卧室里，他拿不到。

信陵君的门客侯嬴给他出了个主意：魏王最宠的如姬，她的父亲被人杀了，三年没破案，是信陵君派人替她报了仇。如姬一直想报答，没找到机会。你只要开口，如姬一定会帮你把兵符偷出来。

信陵君照办，如姬果然盗了兵符。信陵君拿到兵符，要带人去接管魏将晋鄙的军队。侯嬴又给他推荐了屠夫朱亥，让他带在身边，万一晋鄙不交兵权，就让朱亥当场击杀。

临行前，侯嬴对信陵君说了一段话：臣本应随您同行，但年老走不动了。我数着您到达晋鄙军营的日子，到那一天，我面向北方自刎，以此送公子。

信陵君走后，侯嬴果然在算定的日子北向自刭。

这事读起来震撼。侯嬴给信陵君出的计策，是灭九族的大罪——盗兵符、杀大将，哪一条都够他死一百回。他帮信陵君帮到这个份上，已经还不清了。

侯嬴选择自刭，这份恩太重是主因。他活着就成了信陵君的负担——信陵君要想着怎么报他、怎么安置他、怎么防他日后变成把柄。他一死，这份恩干干净净地结清，信陵君可以一心去救赵。

侯嬴用死还了这份大恩。这就是大恩的沉重——重到要拿命去平账。

职场里当然没这么极端，但道理是一样的。别人帮了你一个改变命运的大忙——比如关键时刻替你顶了锅、把你引荐进了一个改变职业轨迹的圈子、在你最困难时借了你一笔救命钱——这种恩你很难用一顿饭、一份礼物还清。它会在你心里压很久，也会在对方心里记很久。

这种大恩，要么不轻易受，受了就认账，别想着用小恩小惠糊弄过去。糊弄只会让关系更拧巴——对方觉得你不识抬举，你觉得对方小题大做。

## 互惠三原则：及时还、等价还、不超额还

把"小恩常还、大恩慎受"拆成可操作的步骤，就是互惠三原则。

**第一，及时还。**

人情这东西，越拖越重。同事今天帮你带杯咖啡，你明天就帮他带份早餐，这事就清了。你拖一周，那杯咖啡在他心里就升值了——"我都帮你带咖啡了，你连个谢都没有"。你拖一个月，一杯咖啡能变成"这人不懂事"。

及时还的另一个好处，是让对方知道你"记着"。记着这件事本身，比还的东西更重要。你记着，说明你在乎这段关系；你忘了，对方会觉得你不在乎。

及时还的意思，是在对方还没开始算账之前还，不一定当天就还。一杯咖啡三五天内还，一次小帮忙一两周内还，一次中等帮忙一个月内还。再大的忙，做不到等价还，至少要有一个明确的"我记着，我会找机会还"的表态。

**第二，等价还。**

还的人情要和受的人情大致对等。对方帮你带杯咖啡，你回一顿米其林，对方会有压力——下次他不敢再帮你带咖啡了，因为他还不回来。对方帮你带杯咖啡，你连句谢都没有，对方也会有压力——他觉得你不懂规矩，下次不帮你了。

等价还的目的在让对方舒服。让对方觉得"我帮你这个忙，你认了，也还了，咱俩两清"。两清的关系最轻松，谁也不欠谁，下次还能继续互相帮。斤斤计较反而把关系搞僵。

等价还的尺度，看帮忙的成本。对方花十分钟帮你跑个数据，你回他十分钟的事——帮他校对一份文档、帮他跑个腿、请他喝杯奶茶。对方花一周帮你顶了一个项目，你回他一周的事——在他忙的时候搭一周手、把一个有用的资源介绍给他。

**第三，不超额还。**

这条容易被忽略。很多人觉得"我多还一点，显得我大方"，其实多还一点反而出问题。

你多还，对方会有两种反应。一种是觉得欠了你——他本来帮你一杯咖啡，你回他一顿大餐，他下次想帮你都不敢了，怕还不起。另一种是觉得理所当然——你这次多还了，他下次期待你继续多还，你哪天还少了，他反而觉得你变抠了。

不超额还，是给关系留余地。留余地，关系才能长期转下去。

## 防绑架三招：不贪大恩、不滥施恩、明确边界

光会还还不够，还得防着被绑架。防绑架有三招。

**第一，不贪大恩。**

别人要帮你帮到"改变命运"的份上，先想清楚你还不还得起。还不起的恩，宁可不要。

有人主动提出帮你一个大忙——"我帮你跟领导说说，把你调到那个核心项目去"——你先别急着感激，先想：他为什么要帮我这个忙？我以后拿什么还？如果还不起，这个忙我是不是宁可不受？

不贪大恩谈不上不领情，更多是在保护这段关系。你不受这个大恩，你们还能平等相处；你受了，从此你矮他半头，他说什么你都得听。很多职场里的"人身依附"，就是这么来的。

**第二，不滥施恩。**

帮人也要有节制。你帮一个人帮得太频繁、太主动、太不图回报，对方不会感激你，只会觉得你好用。

小周帮老李三次，问题不在帮，在帮得太顺手、太不设门槛。老李一开口他就帮，老李自然觉得"找小周没成本"。零成本的帮忙，时间一长就变成义务。

帮人之前问自己一句：这个忙我帮了，对方会还吗？会还的忙可以帮，不会还的忙要慎重。有些忙真不该帮——你帮了一个不还人情的人，那不叫帮，叫惯。

**第三，明确边界。**

互惠最怕模糊。你帮我我帮你，帮着帮着就没了边界，最后谁也说不清谁欠谁。

明确边界的方法是"把帮忙说清楚"。你帮同事一个忙，可以顺口说一句"这次我帮你跑这个数据，下次我那个报告你帮我看看"——把帮忙变成一个明确的交换，而不是含糊的"以后再说"。

含糊的"以后再说"最坑人。你心里记着"我帮过他"，他心里忘着"他帮过我"，时间一长，你觉得他白眼狼，他觉得你莫名其妙。把交换说清楚，关系就清爽。

## 西奥迪尼：互惠原理

西奥迪尼在《影响力》里把"互惠"列为六大影响力武器之首。

大意据《影响力》：互惠原理认为，我们应该尽量以类似的方式报答他人为我们所做的一切。这一原理的威力在于，即使是不请自来的恩惠，也能触发我们的亏欠感。

西奥迪尼讲过一个实验：实验员在陌生人面前"无意"递了一瓶可乐，之后请陌生人买他的彩票。接过可乐的人，买的彩票数量是没接过的人的两倍——一瓶可乐的"恩"，换来了真金白银的"还"。

这个原理在职场的应用是：你主动帮过的人，更可能在你需要时帮你。但西奥迪尼也提醒，这条原理会被利用——有人会故意先施小恩，再索取大报。这就是为什么"不贪大恩"那么重要。

互惠原理的另一面是：你施的恩如果让对方觉得"还不清"，他不会感激你，会躲着你。因为亏欠感太重，重到压得他喘不过气，他唯一的解脱方式就是不见你。所以施恩也要有度，别把对方压垮。

## 实操：人情账本

读完这一章，给自己建一本人情账本。

拿出一张纸，分两栏。左边写"我欠别人的人情"，右边写"别人欠我的人情"。

左边那一栏，把最近三个月别人帮过你的事列出来，每一项后面标一个"还了吗"——已还、未还、还不起。已还的划掉，未还的标个还的期限，还不起的标个"如何表态"。

你会发现，你欠的人情比你以为的多。很多人情你没还，原因往往不在不想还，在忘了。忘了就是没还，没还就在对方心里记着。

右边那一栏，把最近三个月你帮过别人的事列出来，每一项后面标一个"对方还了吗"——已还、未还、不指望还。

写完之后，看两栏是否失衡。如果你欠别人的人情远远多于别人欠你的，说明你在关系里是"索取方"，得赶紧还一批，不然你的口碑会越来越差。如果你帮别人的远远多于别人帮你的，说明你是"老好人型施恩者"，得收一收，别再滥施恩。

账本这东西，算计是次要的，主要让你心里有数。心里有数，关系才不会失衡；关系不失衡，互惠才是润滑剂，不是绑架。
