# 沟通中的认知偏差

## 那条被读歪的微信

周三下午，小陈给同组的小林发了条微信："这个数据你那边再核对一下，我这边看着有点对不上。"

小陈觉得自己说得很客气。"再核对一下"，多委婉；"我这边看着有点对不上"，把问题揽到自己这边，没说对方错。他发完就去忙别的了。

十分钟后，小林把电话打过来，语气冲："你什么意思？这数据是你上周发我的，现在让我核对？甩锅甩得挺溜啊。"

小陈懵了。他完全没想过"甩锅"这两个字会跟自己扯上关系。他解释："我就是想让你看一眼，没别的意思。"

小林更来气了："你直接说'我觉得这数据有问题'不就行了？非要说'你核对一下'，这不就是暗示是我的问题？"

两个人在电话里越说越拧，最后不欢而散。

同一天，小陈还有件事。他上午找领导汇报一个方案，领导听完点了点头："嗯，知道了，先这样吧。"小陈心里踏实了，回去就按方案推进。结果一周后方案被卡住，他去找领导，领导说："我当时又没说同意，你推进什么？"

小陈更懵了。领导明明点头了，明明说"先这样吧"，这不是同意是什么？

这两件事，一件是同事把他的好意读成了恶意，一件是他把领导的敷衍读成了同意。表面看是两个人沟通不畅，底下其实是同一件事：认知偏差。

你以为的"说清楚了"，对方根本没懂；你以为的"对方生气"，其实只是累了；你以为的"领导同意"，其实只是敷衍。沟通里的很多误会，话没说好只是表层，脑子先骗了你才是根子。

## 脑子先骗了你

认知偏差跟智商无关，它是大脑的偷懒方式。

大脑每天要处理的信息太多，不可能每条都仔细想。于是它进化出一套捷径：用过去的经验、当下的情绪、自己的视角，去快速判断别人在说什么、想什么。这套捷径大多数时候管用，一旦用到沟通上，就会出问题。

沟通里最常出问题的，有三种偏差。

第一种叫**透明度错觉**。你以为自己的想法像写在玻璃上一样，别人一眼就能看懂。你心里想的是"我请他核对一下"，你以为对方能读出"这只是个核对请求，没有指责"。其实对方只看到"你核对一下"这五个字，再结合自己的情绪和处境，解读出了完全不同的意思。

你的意图在你脑子里是透明的，到了对方脑子里就是一团雾。你以为说清楚了，其实只说了一半，另一半在你脑子里没出口。

第二种叫**投射效应**。你用自己的想法去套别人。你觉得自己不会因为一句"核对一下"生气，就以为别人也不会；你觉得点头就是同意，就以为领导点头也是同意。你拿自己的内心当模板，去印别人，印出来当然走样。

第三种叫**确认偏差**。你一旦对某个人或某件事有了判断，就只看支持这个判断的证据，忽略反对的。你觉得小林爱甩锅，他每句话你都能读出甩锅的味道；你觉得领导支持你，他每个动作你都能解读成支持。证据到了你眼里已经被判断筛过一遍，早就失去了中性。

这三种偏差叠在一起，沟通就成了一场各自解读的游戏。你说A，他听成B，你以为自己说了A，他以为自己听对了B，两个人都觉得自己没错，误会就这么攒下来了。

## 亡斧疑邻

《列子》里有个故事，把确认偏差讲得透透的。

有个人丢了斧头，到处找不着。他怀疑是邻居儿子偷的。于是开始观察那邻居儿子：走路的样子，像偷斧头的；脸色，像偷斧头的；说话，像偷斧头的。一举一动，没一处不像贼。

后来这人去挖地，从自家坑里把斧头翻出来了。第二天再看见邻居儿子，动作神态全变了，没一处像贼。

斧头还是那把斧头，邻居儿子还是那个邻居儿子，变的只是这人的脑子。怀疑他偷的时候，每个动作都是证据；不怀疑了，证据就全没了。

职场里这种事天天在演。你一旦觉得某个同事"针对你"，他每次找你沟通你都读出敌意，他笑一下你觉得是嘲笑，他不笑你觉得是冷暴力。你一旦觉得领导"看重你"，他随便点个头你都当成嘉许，他随口一句"再改改"你都当成栽培。

你沟通的对象其实是你脑子里那个他，跟眼前这个人没多大关系。真正的他说了什么，你早就没在听了。

## 纠偏三招

知道偏差会来，怎么挡？三个动作，平时就练。

**第一招：复述确认。**

对方说完一段话，你用自己的话复述一遍，问"我理解得对吗"。这一招专治透明度错觉。你以为听懂了，一复述就发现只听了个大概；你以为对方懂了，让对方复述一遍就发现他理解的是另一个版本。

复述是用自己的话把对方的意思重新讲一遍，别照着原话念。"你的意思是，这周三之前要交一版，但不要最终版，对吗？"一句话，把时间、内容、要求都校准了。对方如果说"不对，我说的是……"，误会当场就消了，不用等到出事才发现。

这一招反过来也用。你说完一段重要的话，主动问一句："我刚才说的，你怎么理解的？"别嫌啰嗦，重要的事多说这一句，能省后面三天的扯皮。

**第二招：多问一句。**

觉得对方话里有话、语气不对、态度变了，先别急着下判断，先问一句。

小林如果接到小陈那条微信，不直接判定"他在甩锅"，而是回一句"你说的对不上，是指哪几个数？我这边再看看"，事情就完全不一样。一句话把对方的意思问清楚，比自己在脑子里演半天戏管用。

多问一句的关键是"问事实，不问动机"。问"你指哪几个数"是问事实，问"你是不是想甩锅"是问动机。问事实能把沟通拉回地面，问动机只会把对方推到对立面。

**第三招：换位验证。**

下结论前，先换个位置想一下：如果我是他，听到这话会怎么想？如果我是领导，点头是什么意思？

这一招专治投射效应。你拿自己的内心套别人，套之前先停一下，问自己"他和我处境一样吗？他和我掌握的信息一样吗？他和我今天的状态一样吗？"答案多半是不一样。不一样，你的套用就不成立。

换位验证要承认对方有对方的脑子，别当成替对方找借口。他的判断基于他的信息、他的情绪、他的经历，跟你的不一样。承认这一点，你才不会拿自己的心思去硬套别人。

## 系统1的偷懒

卡尼曼在《思考，快与慢》里讲过，人有两套思考系统：系统1快速、自动、凭直觉；系统2缓慢、费力、要理性。

认知偏差就是系统1在偷懒。它凭直觉、凭经验、凭情绪，一秒钟就给你一个判断。这个判断快是快，常常是错的。系统2能纠偏，但系统2懒，不主动上线。

所以纠偏三招的本质，是强行把系统2叫起来。复述确认、多问一句、换位验证，每一步都在逼系统2干活。系统2一上线，系统1的那些自动判断就被压住了，你才能看见对方真正在说什么。

这三招不需要多聪明，需要的是慢下来。慢半拍，问一句，复述一遍，误会就少一半。

## 复盘你最近一次误会

读完这一章，给自己做个功课。

翻一翻你最近一次闹得不愉快的沟通。可能是跟同事的一通电话，可能是跟领导的一次汇报，可能是跟家人的一句微信。

把那次沟通还原成三段：

第一段，**我当时的意图是什么**。要写当时脑子里真正想的那个，别事后合理化。

第二段，**对方接收到的意思是什么**。从对方的反应倒推，他到底听成了什么。

第三段，**中间差在哪**。是透明度错觉（我没说全），是投射效应（我以为他和我一样想），还是确认偏差（我带着旧判断在听）？

把这三段写下来，你会发现大多数误会都不是对方故意使坏，也不是你脑子笨，就是偏差在中间作怪。看清楚是哪个偏差，下次同类场景你就能提前防。

认知偏差没法消灭，它是大脑的出厂设置。但你能学会认出它。认出来了，它就没办法替你说话；认不出来，它会一直在你耳边替你解读别人，你还以为那是你自己的判断。

沟通高手也有偏差，他只是知道自己有偏差，所以多问一句、多复述一遍、多换一次位。慢的那半拍，就是他和普通人的差别。
