# 倾听是最高级的说服

## 两个销售，一个下午

小郑和老吴都是卖企业软件的，同一天去拜访同一个客户。

小郑先到。他打开笔记本，从公司简介开始讲，讲到产品功能、技术架构、行业案例、客户名单、价格方案，一口气讲了四十分钟。客户中途喝了两次水，看了三次手机，眼神飘了几次。小郑没注意，他太想把这单签下来，太怕冷场，所以一直说一直说。

讲完，客户说："行，我们再研究研究，回头联系你。"

小郑出了门，心里有点慌。"再研究研究"这五个字，他听过太多次了，基本等于没戏。

老吴后到。他坐下后第一句话是："张总，在听我介绍之前，能不能先聊聊你们现在最头疼的事？我看看我们的东西能不能帮上忙，帮不上也没关系。"

张总愣了一下，然后开始倒苦水：现在的系统怎么慢、怎么卡、IT怎么抱怨、业务怎么催、老板怎么施压。老吴全程没插话，偶尔点头，偶尔记两笔，偶尔问一句"这个事大概多久了"或"你们之前试过什么办法"。

张总说了二十分钟，停下来，问："你的产品是怎么解决这个的？"

老吴这才开口，而且只讲了十分钟。他没讲公司简介，没讲技术架构，只针对张总刚才说的那几个痛点，讲了对应的解决思路。

临走时，张总说："你下周带技术来一趟，我们细聊。"

老吴出了门，知道这单成了大半。

小郑和老吴的差别不在产品知识，不在话术，在听。小郑一直在说，老吴一直在听。说的人把信息倒出去，听的人把信息收进来。最后成交的，是那个收信息的人。

## 大多数人不是在倾听，是在等自己说话

倾听这件事，听起来简单，做起来极难。

难在哪？难在人的大脑天生是说话模式，不是听话模式。对方一开口，你的脑子就开始转："他这话什么意思？""我该怎么回应？""我有个类似的经历可以分享。""他说得不对，我得反驳。"

你以为你在听，其实你在准备自己的下一句。对方的话只是个信号枪，枪一响，你就开始跑自己的赛道。

这种"伪倾听"有几个典型表现。

一是**抢话头**。对方话还没说完，你就接上"对对对，我也遇到过"。你接的不是他的话，是你自己的话。

二是**急着给建议**。对方刚说完一个困扰，你立刻说"你应该怎么怎么做"。你都没搞清楚他到底想要什么，就开始开药方。

三是**贴标签**。对方说了一通，你心里默默给他分类："哦，他是这种性格的人。""嗯，他在抱怨。"标签一贴，你就不再真的听他了。

四是**走神**。对方说到第三句，你的脑子已经飘到下午的会、晚上的饭、明天的邮件。脸上还挂着微笑，心里早不在了。

这四种表现的共同点是：你把自己放在了主角的位置上，对方只是个配角。可沟通的真相是——谁在听，谁才是主角。说的人是在给你递信息，你怎么接、接不接、用不用，决定这场沟通的走向。

## 倾听是主动的，不是被动的

很多人对倾听有个误解，以为倾听就是闭嘴。其实闭嘴只是倾听的最低配。

真正的倾听是主动的。你在主动做三件事。

**第一，主动收集信息。** 对方说的每一句话里都藏着信息：他的需求、他的顾虑、他的优先级、他和别人的关系、他没说出口的话。你要像侦探一样，从他的措辞、语气、停顿、重复里把这些信息捞出来。

**第二，主动让对方感到被理解。** 人都渴望被理解。你一旦让对方觉得"这个人懂我"，他的防御就卸下一半，话匣子就打开了。被理解的人更愿意听你的，因为他觉得你和他是一边的。

**第三，主动为说服蓄力。** 倾听不是放弃说服，是为说服做准备。你听得越透，越知道从哪个角度切入他能接受。盲目的说服是撞墙，听懂之后的说服是开门。

倾听这三件事，件件都是主动的。被动的那叫"听着"，不叫倾听。

## 韩非子：说服难在懂对方的心

韩非子在《说难》里讲过一句话，把倾听的本质点透了。

大意据《韩非子·说难》：凡说之难，在知所说之心，可以吾说当之。

翻译过来：说服的难处，不在你说得好不好，而在你懂不懂对方的心，能不能用对方能接受的话去对应他的心。

韩非子把说服这件事拆得很细。他说，说服难的不是你有没有口才、有没有道理，而是你能不能摸到对方心里在想什么。对方想要名，你跟他谈利，他听不进去；对方怕风险，你跟他讲机会，他更怕。你得先知道他心里那根弦是什么调，才能弹出他听得进的音。

怎么知道对方心里在想什么？靠听。不是听他嘴上说什么，是听他话里话外透出来的那个真实关切。

韩非子两千多年前就看明白的事，今天大多数职场人还没学会。我们太急着表达，太怕沉默，太想证明自己有想法，结果话说了很多，对方的心一点没摸到。

## 三级倾听：听内容、听情绪、听需求

倾听可以分三级，一级比一级深。

**第一级：听内容。**

这是最基础的。对方说了什么事实、什么数据、什么经过，你听清楚、记下来。听起来简单，其实很多人做不到。开会时你回想刚才那个同事说了什么，常常只能想起个大概。听内容的基本功是专注，是把手机扣过去、把脑子拉回来、把对方的话当回事。

**第二级：听情绪。**

对方说话时的语气、节奏、用词，都在透露情绪。他语速变快了，可能是急了；他突然停顿了，可能是在斟酌；他用了一些极端词，比如"永远""从来""根本"，说明他情绪上来了。听情绪，是听内容背后的那个人。

举个例子。同事说"这个项目反正也没人管"。内容是"项目没人管"，情绪是"我觉得委屈、被忽视"。如果你只听内容，你会接"那我去问问领导"。如果你听出了情绪，你会接"你是不是觉得这个项目你做了很多但没被看见"。后者一出口，对方会觉得你真懂他。

**第三级：听需求。**

需求是情绪底下的那个真实想要。同事抱怨"项目没人管"，需求可能是"我希望我的工作被认可"，也可能是"我希望有人帮我分担"，也可能是"我希望领导明确这个项目的优先级"。同一个抱怨，背后是不同的需求。

听需求靠猜，也靠问。猜是根据对方的处境和情绪推断，问是用开放式问题去验证。"你这么说，是不是希望……？"这种问法既是在确认需求，也是在让对方感到被理解。

三级倾听不是三个阶段，是同时进行的。你在听内容的同时听情绪，在听情绪的同时猜需求。练到后来，这三件事会变成一个本能反应。

## 复述确认法：让对方知道你听见了

光听还不够，得让对方知道你听见了。最简单的办法是复述确认。

对方说完一段话，你用自己的话复述一遍："我理解一下，你的意思是……对吗？"

这个动作有三个好处。第一，你逼自己真的听懂了——如果你复述不出来，说明你刚才没听进去。第二，对方感到被重视——他知道自己说的话被接住了。第三，你给对方一个纠正的机会——如果你理解错了，他会立刻说"不是，我的意思是……"，省得你带着误解往下走。

复述确认有个关键：用你自己的话，不要原样重复。原样重复像鹦鹉学舌，显得敷衍；用自己的话复述，才显得真懂了。

还有个进阶版：复述内容+点出情绪。比如"我理解你的意思是这个项目排期太紧，而且你听起来挺为难的，是不是因为前面那个需求还没收尾？"这一句话把内容、情绪、需求都接住了，对方听完会觉得"这个人真懂我"。

## 柯维：知彼解己

柯维在《高效能人士的七个习惯》里把倾听放在第五个习惯，叫"知彼解己"。

大意据《高效能人士的七个习惯》：知彼解己——先寻求理解对方，再寻求被对方理解。

柯维说，大多数人沟通是为了被理解，不是为了理解对方。所以我们一开口就在讲自己，根本没在听。可真相是，你想被理解，得先让对方觉得你理解了他。人是对等的——你理解我，我才愿意理解你。

知彼解己的顺序不能反。先解己再知彼，是大多数沟通失败的根源。先知彼再解己，是倾听的精髓。

## 下次对话，只听不插嘴

读完这一章，给自己布置一个作业。

下一次和别人对话——不管是和同事开会、和朋友吃饭、和家人聊天——刻意做一件事：只听，不插嘴。

不抢话头，不给建议，不贴标签，不走神。对方说完一段，你用复述确认一下，然后等他继续说。

对话结束后，花两分钟复盘：对方今天真正想说什么？他嘴上讲的是什么，情绪上透出来的是什么，底下的需求是什么？我有没有在哪一刻差点插嘴？那一刻我想说什么？

如果你能坚持做几次，你会发现一个反直觉的事：你越少说，对方越觉得你懂他；你越懂他，他越愿意听你的。

倾听是最高级的说服，因为它不靠你说服对方，靠对方自己说服自己。你只是给他一个被听见的场，他自己会在那个场里把心里的话理顺、把该做的决定做出来。

会说话的人赢一时，会倾听的人赢一世。
