# 三天，够不够

## 后天面试，今天才动

周三晚上十点，小李给我发消息："哥，我后天上午有个面试，今天才开始准备，是不是凉了？"

小李是个工作三年的产品经理，前两年在公司里待得舒服，没怎么面试过。这次被猎头推了一家不错的公司，岗位也对口，他一直觉得"到时候临场发挥就行"，结果临到头了才发现——自我介绍没准备、离职原因没想好、缺点问题没打磨、谈薪心里没底、反向面试不知道问啥。

他慌了。

我问他："你还有两天半，对吧？"

"对，周四一天，周五上午半天。"

"够了。"我说，"但前提是你得按计划来，不能东一榔头西一棒子。"

很多人面对面试准备，要么彻底摆烂觉得"临场发挥就行"，要么焦虑到什么都想准备结果什么都没准备好。三天时间，全面提升是不可能的，但抓大放小、练熟几个核心动作，足够让你从"裸考上阵"变成"有备而来"。

三天速成的本质是抓大放小——让你在面试那四十分钟里，把几个一定会被问到的核心问题答得稳、答得准、答得有钩子。全面提升是以后慢慢练的事，三天干不了。

下面这份三天计划，是我给小李的，也给你。

## 三天怎么排

### 第一天：背熟三段稿

第一天的任务最重，也最关键——把一定会被问到的三个问题写成稿、改顺、背熟。

哪三个？自我介绍、离职原因、缺点。

这三个问题，九成以上的面试都会问，而且问的方式几乎一样。你把这三个答稳了，面试的前十分钟就稳了，心态也稳了。

**自我介绍**：写一份三分钟版。结构是"我是谁加我做过什么（用一条主线串）加我为什么来（埋一个钩子）"。写完对着镜子说三遍，录音听一遍，把所有"嗯""然后""那个"标记出来，删掉。再说到不卡壳为止。

**离职原因**：写一段三十秒能说完的稿。原则是向前看，讲你想去什么方向，别讲你现在公司有什么问题。哪怕你是因为上家公司烂才走的，也别吐槽上家。一句"我希望往XX方向深入，目前公司业务调整后这个方向空间有限"就够了。

离职原因最忌讳的是说前任公司或前任领导的坏话。面试官心里会想：你今天这么说上家，明天离职了也会这么说我们。

**缺点**：写一段带"改进动作"的缺点。原则是说真缺点，但配上你正在怎么改。别用"我太追求完美"这种伪缺点，面试官听了翻白眼。说一个真实的、不影响岗位核心的缺点，然后讲你最近在怎么改。

比如应聘产品经理，可以说"我有时候太关注细节，会拖慢决策。最近我在练'先定大方向再抠细节'，每周给自己设一个决策截止时间。"

第一天结束，你应该有三段背得出来的稿。背到能自然说出来、不卡壳、不啰嗦，背到脱口而出但不带背稿痕迹，而不是背到一字不差。

### 第二天：模拟面试加录音复盘

第二天不能再闷头写了，得开口练。

找一个人陪你模拟——朋友、前同事、甚至对着摄像头录自己都行。让对方用面试官的口吻问你，你完整地答一遍，全程录音。

模拟面试的重点是听你"听起来像不像一个准备过的人"。录音会暴露所有问题：语速太快、声音发抖、答到一半跑题、被追问就卡壳、说到关键数字含糊其辞……

听录音的时候，拿笔标记三类问题：

- **卡壳点**：哪里停顿超过两秒？这些地方是你没想清楚，回去补。
- **啰嗦点**：哪里绕了三圈才说到重点？这些地方砍掉一半。
- **虚词点**：哪里全是形容词没有事实？"我能力很强""我很有团队精神"——换成具体的事和数字。

模拟面试至少做两轮。第一轮暴露问题，第二轮验证改进。两轮之间，把第一轮暴露的问题逐个修。

第二轮模拟完，你应该能做到：自我介绍流畅不带背稿痕迹、常见问题不卡壳、被追问能接住、谈薪不慌。

### 第三天：谈薪、反向面试、跟进话术

第三天收尾，准备三个临场动作的话术。

**谈薪**：提前查清楚岗位市场价（招聘网站、猎头、同行打听），定一个区间。底线是多少、期望是多少、报价是多少，三个数心里要有数。面试官问期望薪资，别报一个死数，报一个区间，下限是你的底线，上限比期望高一截留谈判空间。

谈薪的具体技巧，下面会讲，这里你只要记住：不要先报低价，不要被"你上家多少"框死，报区间不报死数。

**反向面试**：面试官问"你有什么想问我的吗"，千万别说"没有了"。准备三个问题，分别问业务、问团队、问岗位。问业务显示你做过功课，问团队显示你关心协作，问岗位显示你想知道怎么干好。

三个问题示例：

- "这个岗位接下来半年最重要的三件事是什么？"
- "团队目前的人员结构和协作方式大概是什么样的？"
- "如果我入职，前三个月您希望我交付什么？"

这三个问题问完，面试官心里会给你加分——这是个想干活的人，不是来混日子的。

**面试后跟进**：面试结束当天，发一条简短的消息给HR或面试官，感谢对方时间，重申你对岗位的兴趣，附上一句面试中聊到的某个具体点（显示你认真听了）。一条消息，三句话，不要长。

跟进是职业素养。很多人不跟进，你跟进了，就在同等条件的候选人里多了一分。

## 面试沟通三件套

三天时间，你不可能把所有沟通技巧都练熟。但有三件套，必须练到能条件反射。

**第一件：自我介绍框架**

自我介绍要给面试官一个记住你、想问你、想录用你的理由。简历面试官已经看过了，你再念一遍没意义。框架是：信息层（我是谁）加主线层（我做过什么，用一条线串）加钩子层（我为什么来，埋一个亮点）。

三分钟，话筒在你手里。你铺什么钩子，面试官就顺着问什么。把钩子铺在你最想被问的方向上，面试就变成你的主场。

**第二件：STAR答题法**

面试官问"你做过什么有挑战的项目""讲一个你解决过的难题"，别泛泛而谈，用STAR框架。

- **情境**：一句话交代背景。
- **任务**：一句话说你负责什么。
- **行动**：重点讲你做了什么，具体到动作。
- **结果**：用数字或事实收尾。

STAR的精髓在"行动"那一环——很多人讲项目，背景讲了一大堆，自己做了什么一句带过。面试官想听的是你做了什么。项目本身多牛是公司的事，跟你关系不大。

每个你准备要讲的项目，都按STAR写一遍，背熟。被问到的时候，张口就能来。

**第三件：反向面试三问**

面试最后那个"你有什么想问我的"，是你展示思考深度的机会。三个问题：问业务方向、问团队协作、问岗位期待。

问得越具体，越显你做过功课。"你们公司未来怎么发展"这种大问题显得没准备，"你们接下来半年最想突破的业务是哪块"就显得你想过。

## 临场急救三招

准备得再好，临场总会出意外。三个急救招，记住能救命。

**第一招：卡壳了怎么办？**

答到一半突然卡住，脑子一片空白。这时候别硬撑，也别装。坦诚地说："这个问题我想一下，给我十秒钟组织一下语言。"面试官不会扣分，反而觉得你真诚。

十秒钟里，快速回忆STAR结构——情境、任务、行动、结果。从情境开始重新起头，往往就接上了。

最忌讳的是卡壳之后开始胡说，越说越乱，最后自己都不知道在说什么。卡壳停一下，比硬撑胡说要好十倍。

**第二招：紧张到发抖怎么办？**

面试前心跳加速、手心出汗、声音发抖，这是生理反应，跟专不专业没关系。两个动作能缓解：一是深呼吸，吸气四秒、屏气四秒、呼气四秒，做三轮；二是开场前喝一口水，让身体有个停顿动作。

声音发抖的时候，刻意放慢语速。紧张会让人不自觉加快语速，越快越抖，越抖越快，恶性循环。一慢下来，抖就止住了。

还有一招：把面试官想象成你以前的同事，你在跟他讲你做过的一个项目。心态一变，紧张就少一半。

**第三招：被压力面怎么办？**

有些面试官会故意施压——"你这个数据也没什么了不起的""你觉得你这个经验能胜任吗"。这时候别上头，也别怂。

压力面测试的是你在压力下能不能保持理性。面试官故意施压，看你怎么接。应对方法是：先承认对方的角度有道理，再用事实回应。

"您说得对，这个数据在行业里不算最高。我想补充的是，这个提升是在XX约束下做到的，具体过程是……"

承认加补充，不卑不亢。被压力面激怒或者被压力面吓怂，都输了。

## 三天准备清单

把这份清单打印出来，每完成一项打勾。三天后全部打完，你就准备好了。

**第一天清单**

- 写一份三分钟自我介绍稿（信息层加主线层加钩子层）
- 录音听一遍，标记所有虚词和卡壳
- 改到不卡壳，对着镜子说三遍
- 写一段三十秒离职原因稿（向前看，不吐槽上家）
- 写一段带改进动作的缺点稿（真缺点加正在怎么改）
- 三段稿都能脱稿自然说出

**第二天清单**

- 找一个人陪你模拟面试，全程录音
- 听录音，标记卡壳点、啰嗦点、虚词点
- 逐个修改
- 做第二轮模拟，验证改进
- 把要讲的两到三个项目按STAR写一遍，背熟

**第三天清单**

- 查清岗位市场价，定好底线、期望、报价三个数
- 准备三个反向面试问题（业务、团队、岗位）
- 写一条面试后跟进消息模板
- 把临场急救三招在脑子里过一遍
- 整理面试当天的着装、路线、时间

清单全部打勾，你就可以安心睡一觉，第二天去面试了。

## 一句话

三天速成不了沟通高手，但能让你把几个核心动作练到不慌——面试那四十分钟，拼的是你准备得有多扎实。
