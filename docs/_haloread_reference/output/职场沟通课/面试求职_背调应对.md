# 沟通课·背调应对

## 一句"主导"，丢了到手的offer

陈昊拿到那家互联网公司的offer时，高兴得当晚请了几个朋友吃饭。岗位是他想要的，薪资也谈到位了，HR说走完背调流程就发正式offer，让他提供前两家公司的证明人。

陈昊没当回事。他简历上写的是"主导某用户增长项目，日活提升40%"，他觉得这没什么问题，毕竟那个项目他确实参与了不少活儿。证明人填了前公司的一个平级同事和一个组长。

背调公司打电话过去，问了几个问题：这个项目是谁牵头的？陈昊在里面的角色是什么？组长实话实说：项目是总监牵头的，陈昊是执行成员之一，活儿干得还行，但说"主导"不太准确。

背调报告交回HR手里。HR把陈昊的简历和报告对照了一下，"主导"对"执行成员"，这属于简历夸大。offer撤了。

陈昊后来托人打听原因，HR说了一句："能力我们可以培养，诚信出了问题，没法培养。"

陈昊不是能力不行，是栽在了他没当回事的那一关——背调。

## 背调在查什么，又在防什么

很多人对背调有种本能的抵触，觉得这是公司在找茬、在挖老底。这个想法偏了。

换个立场想：一家公司招一个人，要付薪水、要给权限、要让他接触客户和数据，万一简历造假、前科累累，损失的是公司。背调，就是公司在掏钱和给权限之前，做一次风险排查。

它查的东西其实就几类：身份学历真不真，履历日期对不对，岗位和职责夸没夸，离职原因是什么，有没有劳动纠纷，有没有竞业限制。这些信息，绝大多数都能通过你提供的证明人和公开渠道核实。

背调的本质是风险控制。它的目的是确认你说的跟真的对得上，挑刺只是手段，不是目的。对得上的，皆大欢喜；对不上的，公司会怀疑你其他地方也对不上。

所以应对背调的根本很简单：让简历上的每一句话都经得起核实，让证明人说到你的时候愿意讲好话。这两件事做到了，背调就是走个流程。

## 萧何追韩信，与一封推荐信的分量

背调这事的逻辑，两千年前就有人玩明白了。

楚汉相争那会儿，韩信从项羽那边跑出来投奔刘邦，刘邦没看出他的本事，让他当了个管粮草的小官。韩信觉得没意思，连夜跑了。萧何听说之后，来不及报告刘邦，月下亲自去追，追了两天才把韩信劝回来。

追回来还不算。萧何回去跟刘邦说："韩信是国士无双，你要是只想在汉中称王，用不着他；你要是想争天下，除了他没第二个人。"刘邦将信将疑，但信萧何这个人，于是拜韩信为大将。

这件事里有个关键：刘邦并不认识韩信，他凭什么信一个跑过来的人？凭的是萧何的信用。萧何把自己的名声押上去，替韩信做了背书。这就是推荐人的力量。

背调里你提供的证明人，扮演的就是萧何这个角色。面试官不认识你，他打电话给你前领导、前同事，问的其实就是一句话：这个人，你敢不敢替他背书？敢，offer稳了；犹豫，或者说了实话里的不好，offer就悬。

韩信有萧何，是因为萧何亲眼见过他的本事。你的证明人愿意替你说话，前提也是他真觉得你行。所以背调这事，功夫在背调之外——在你平时怎么干活，怎么跟同事相处，怎么离开一家公司。

## 三件事提前做，背调就稳了

背调要从写简历那一刻开始防，临到头才准备就晚了。三件事做到位，基本无忧。

第一件，简历真实。这是底线，也是最重要的一条。简历可以包装，不能造假。包装和造假的区别在哪？包装是把真做过的事讲得更有条理、更突出价值；造假是没做过的事说做过，参与的事说主导。"主导"和"参与"这两个词，坑了不知道多少人。拿不准的时候，用"负责""参与""协助"这种能对得上的词，别往大了说。日期也别错，入职离职月份要对得上社保记录，差一个月都可能被问。

第二件，提前告知前领导。背调前，给打算填为证明人的前领导或前同事打个招呼。一方面是礼貌，让人家有个心理准备；另一方面，你可以侧面了解一下对方对你的真实评价。如果对方语气犹豫，或者关系本来就一般，那就换一个证明人。别把证明人填成一颗定时炸弹。

第三件，备好证明人。一般背调要求提供最近两段经历的证明人，每段两到三人，最好是直接上级、平级同事、下级（如果有）。证明人要选跟你关系不错、且能说清楚你工作内容的人。提前跟每个人确认：方便做证明人吗？背调公司可能会联系你。对方答应了再填。

## 前东家说坏话，怎么办

这是背调里最让人慌的情况：跟前东家闹得不愉快，离职时撕过脸，怕背调的时候被说坏话。

先说个事实：正规背调公司打电话，问的是客观信息——任职时间、岗位、职责、离职原因、是否愿意再合作。它不会问"你觉得这人怎么样"这种开放问题，也不会只听一面之词。前东家如果恶意中伤，背调公司会交叉验证，你也有申诉的机会。

如果真遇到前东家说的情况跟事实不符，你有几条路。第一，向背调公司申诉，提供能证明事实的材料，比如离职证明、劳动合同、工资流水。第二，向HR说明情况，主动讲清楚你跟前东家的矛盾在哪，是什么性质，让对方有个判断。第三，如果前东家的说法涉及诋毁，影响了你入职，可以依法维权。劳动者享有平等就业的权利，以不实信息损害他人就业的，要承担相应责任。具体条款以《劳动合同法》及现行有效法规为准。

但最根本的解法，是离职的时候别把桥烧了。哪怕受了委屈，体面地走，把交接做干净，把话说在明处。职场圈子比你想的小，前东家的一句话，可能在你不知道的地方跟着你很多年。

## 简历自查：每条经历，谁能替你说话

拿出你的简历，做一件事：在每一段工作经历旁边，标注一到两个证明人。标注的时候问自己三个问题。

第一，这个人知道我在这段经历里具体做了什么吗？如果不知道，他做证明人也讲不清楚，换一个。

第二，这个人愿意说我好话吗？如果犹豫，说明关系没到位，别冒险。

第三，这个人的身份合适吗？最好是直接上级或密切合作的同事，HR的证明力有限，亲戚朋友完全无效。

标完之后，你会发现简历上有些经历，你找不到合适的证明人。这往往意味着两种情况：要么这段经历时间太短、接触的人少；要么这段经历里你跟人关系处理得不好。无论哪种，都是你要在背调前想清楚怎么应对的。

还有一条：把简历上每一条经历，跟你能想到的客观证据对应起来——项目文档、邮件记录、汇报材料、公开的产品或案例。背调真问到细节，你能拿出东西来，比任何解释都管用。

## 结语

背调这关过得硬不硬，看的是你过去几年怎么干活、怎么做人，背调那天只是个收尾。简历真实是底线，关系干净是护身符。
