# 一个唱黑脸，一个递台阶

## 两个人一台戏

老周去跟一家供应商谈年度合同。

对方来了两个人——一个是采购总监，姓王，板着脸；一个是采购经理，姓刘，笑呵呵的。

老周刚报完价，王总监就拍桌子："这个价格你也好意思报？比去年高百分之十五，你们是不是不想做了？"

老周心里一紧，正要解释。

刘经理赶紧打圆场："老王你别急，让人家说说嘛。"转头对老周笑："周总，您看今年成本涨了我们也理解，但这个涨幅确实有点大，您再想想，能不能优惠点？"

老周松了口气，觉得刘经理是好人，王总监是坏人。为了不让刘经理难做，他主动降了八个点。

王总监还是板着脸："还是高。"

刘经理又打圆场："老王，周总已经让了这么多了，差不多行了。周总您再让一点，我帮您劝劝老王。"

老周又让了五个点。

回去路上，老周越想越不对——他让了十三个点，比预期多让了一倍。他突然反应过来：王总监和刘经理是一伙的，一个施压一个递台阶，把他绕进去了。

红白脸战术，他中了。

## 红白脸的本质

红白脸是谈判桌上最老的战术之一，但到现在还管用，因为它戳的是人性的弱点。

它的本质是：**一个人施压制造焦虑，一个人给台阶让你感恩让步**。

白脸负责凶。他挑刺、压价、威胁、拍桌子，把气氛搞紧张，让你心里发慌。你被他的气势压住，本能想找出口。

红脸负责软。他和颜悦色、替你说话、给你台阶，让你觉得他是"自己人"。你被他的善意打动，本能想回报他。

一压一拉之间，你的判断力就乱了。你忘了谈的是价格，开始想着"不能让刘经理难做""别让王总监发火""赶紧签了走人"。

你让步，不是因为价格该让，是因为情绪被绕进去了。

这就是红白脸的杀伤力——它不攻击你的逻辑，攻击你的情绪。

明白了这一层，你就知道怎么破——**不接情绪，只谈问题**。

## 赤壁前的朝堂

《三国志》里赤壁之战前的孙吴朝堂，是一场经典的红白脸大戏，也是识破红白脸的教科书。

曹操拿下荆州，写信威胁孙权："今治水军八十万众，方与将军会猎于吴。"

信到了孙权手里，朝堂炸了。

主降派以张昭为首，一通分析：曹操挟天子令诸侯，名正言顺；曹操水陆并进，兵力悬殊；咱们打不过，不如降了保江东。

张昭是文臣领袖，说话有分量。他这一通说，朝堂上大半人都跟着附和。这是白脸——把后果讲得吓人，制造焦虑。

主战派以周瑜、鲁肃为首。鲁肃私下跟孙权说："众人皆可降曹操，惟将军不可降曹操。我降了还能当个郡守，您降了能当什么？"周瑜更是分析曹操的弱点：北方兵不习水战、粮草不继、军心不稳。

这是红脸——给孙权信心，让他觉得"还能打"。

但孙权没被任何一方带跑。

他听完两边的话，没急着表态。他做了一件事——**找决策依据**。

他问周瑜：你说曹操能打，凭什么？

周瑜给了五条具体分析：曹操后方不稳、马超韩遂在关西为患、北方兵不习水战、天寒粮缺、士卒水土不服。

孙权听完，拍板：打。

这一手是识破红白脸的关键。

张昭的白脸、周瑜的红脸，都是情绪层面的拉扯。孙权没在情绪上接，他直接问"凭什么"——把情绪层面的拉扯，拉回到事实层面的判断。

事实清楚了，决策就清楚了。曹操的弱点是事实，张昭的焦虑是情绪。孙权信事实，不信情绪。

红白脸的破法，就在这里——**不被情绪带跑，直接找决策依据**。

## 用红白脸

红白脸不是只能被对方用，你也可以用。但用之前要想清楚分工。

**分工要明确。**

谁唱白脸、谁唱红脸，提前定好。白脸的人要能压得住场子，红脸的人要能圆得回来。两个人配合不默契，反而露馅。

**白脸先上。**

白脸先开口，挑刺、压价、提苛刻条件，把气氛搞紧张。白脸的力度要够，但别过头——过头了对方直接走人，戏就演不下去了。

**红脸收尾。**

白脸施压到一定程度，红脸出来打圆场："老张你别急，让人家说说嘛。"然后转头给对方台阶："您看这样行不行……"

红脸给的台阶，就是你想让对方接受的方案。对方被白脸压过，看到红脸的台阶，本能觉得"这个能接受"——其实这个台阶就是你原本的目标价。

**注意边界。**

红白脸是战术，不是欺骗。白脸可以挑刺压价，但不能辱骂人格；红脸可以打圆场，但不能承诺做不到的条件。战术用过了头，对方识破之后翻脸，反而砸了合作。

## 破红白脸

对方用红白脸，你怎么破？三步。

**第一步：识别。**

红白脸用得再熟，也有破绽。识别的标志有三条：

- 对方两人态度反差大，一个凶一个软。
- 凶的那个挑刺但不走人，软的那个替你说话但不让步。
- 你被绕进去之后，让步总是"为了不让软的那个难做"。

三条中两条，基本就是红白脸。

**第二步：不接情绪。**

识别之后，最关键的一步是不接情绪。

白脸凶你，你别生气，他凶的是策略不是你这个人。红脸软你，你别感动，他软的也是策略不是真帮你。

怎么不接？把对话拉回事实。白脸说"你这价格太离谱"，你别辩解"不离谱"，你说"我们这个价格的依据是市场均价加技术溢价，您可以看这份对比表"。红脸说"您再让一点我帮您劝劝"，你别感动，你说"价格我已经让到目标位了，再让需要换其他条件"。

每句话都把对话拉回事实，不接情绪层面的拉扯。

**第三步：直接找决策人。**

红白脸战术有一个软肋——唱戏的两个人往往都不是最终决策人。白脸施压、红脸递台阶，都是为了让你在他们这一层就范。

破法是直接找决策人。

"两位的意见我听到了，但这个项目最终拍板的是贵司哪位领导？我希望能直接和决策人沟通，省得来回传话耽误时间。"

这一手把红白脸的戏台拆了。决策人只有一个，没法唱红白脸。你直接和决策人谈，对方两人就失去了"中间人"的位置。

当然，对方可能不让见决策人。但即便不见，你提出这个要求本身，就已经让对方知道——你看穿了。

## 一个常见的变体

红白脸不一定要两个人，一个人也能演。

一个人怎么演？**"我得回去请示领导"**。

这是单人版红白脸：谈判桌上他对你客气（红脸），但说"领导那边不好交代"（白脸不在场，但白脸的压力在）。

你提的条件他都说"我个人觉得可以，但领导那边……"——把锅推给不在场的领导，自己装好人。

破法和双人版一样：直接找领导。

"既然您说了不算，那能不能安排我跟领导直接沟通一次？我当面把方案讲清楚，省得您来回传话。"

他要是真说了不算，会去安排；他要是说了算，装不了，就只能自己拍板。

## 实操

复盘你遇到过的一次红白脸，按下面三个问题分析：

1. 当时对方是怎么分工的？谁是白脸谁是红脸？白脸怎么施压，红脸怎么递台阶？把当时的对话回忆出来，写下来。
2. 你当时怎么应对的？是被绕进去让步了，还是识破了？如果被绕进去了，是哪一步被带跑的——是白脸的施压让你慌了，还是红脸的台阶让你感动了？
3. 如果重来一次，你会怎么破？按本章的三步——识别、不接情绪、找决策人——写一个应对方案。

复盘不是为了后悔，是为了下次不重蹈覆辙。红白脸这招太常见，每个商务人都得会破。

## 一句话

红白脸攻击的是情绪不是逻辑——不接情绪、只谈问题、直接找决策人，戏台就塌了。
