# 西奥迪尼的影响力法则

## 两种人，两种待遇

公司里有两类人。

一类是老张。老张找谁办事，谁都不太情愿。他开口就是"这个事你帮我做一下""这个忙你必须帮""这是领导安排的"。道理都对，权限也有，可大家就是不愿意配合。当面答应，背后拖延；嘴上说好，心里骂娘。

另一类是老王。老王找谁办事，大家都挺给面子。他开口是"我这边有个事，想听听你的意见""上次你帮的那个忙，我一直记着""这个事我觉得你最在行"。同样是求人，老王一开口，别人就愿意动。

老张和老王的差别不在权限，不在道理，在影响力。

影响力这东西，看不见摸不着，但决定了一个职场人能调动多少资源、能办成多少事。有人觉得影响力是"地位给的"，有人觉得是"性格带的"，西奥迪尼说——影响力有规律，可以学。

## 卧底三年，找出六把武器

罗伯特·西奥迪尼是亚利桑那州立大学的心理学教授。他年轻的时候干过一件有意思的事——他去"卧底"。

他跑去二手车行当销售，去募捐机构当电话推销员，去保险公司当业务员，去培训机构当话术教练。他不研究理论，他研究"那些让人乖乖配合的话术到底长什么样"。

他卧底了三年，访谈了几百号"合规专家"——就是那些专门让人点头、让人掏钱、让人签字的人。他把这些人的话术拆开、归类、验证，最后总结出六种反复出现的模式。

这六种模式，他叫"影响力的六大武器"。

**互惠、承诺与一致、社会认同、喜好、权威、稀缺。**

西奥迪尼说，这六把武器不是话术技巧，是人脑里的"快捷反应按钮"。人每天要做无数决定，不可能每个都仔细想，于是大脑进化出了六条捷径——看到某些信号，就自动按某个按钮，自动给出某种反应。销售和说服高手，就是会按这些按钮的人。

关键在于：这六把武器，正道用是"让对方愿意配合做对的事"，邪道用才是"操控别人损人利己"。西奥迪尼自己反复强调，影响力的正道是真诚，不是套路。

## 六把武器的正道用法

**第一把：互惠。**

人有一种本能——别人给了你什么，你总觉得欠着，想还回去。这就是互惠。

正道用法：**先帮人，再求人。** 你想找同事帮忙，先在别的事上帮过他；你想让客户听你的，先给他提供点免费的价值。互惠的本质是建立一种"我对你好"的底色，让对方在你开口时愿意配合，而不是冷冰冰的等价交换。

邪道用法是"小恩小惠换大利益"，比如送个小礼物让人签大单。这种短期可能管用，长期会让人反感，觉得被算计。

**第二把：承诺与一致。**

人有一种倾向——一旦做了承诺，就想保持一致，不想自己打自己脸。

正道用法：**小承诺起步。** 你想让同事支持一个大项目，别一上来就让他全押。先让他答应一个小事——比如"帮我看看这个思路对不对"——他答应了，就等于站到了你这个方向上。后面再推进大的，他更愿意保持一致。

这条用在自我管理上也灵。你想养成一个习惯，先承诺一个最小的版本，做到了再加大。一致性会推着你往前走。

**第三把：社会认同。**

人不确定的时候，会看别人怎么做。大家都选的，多半错不了。

正道用法：**用案例说话。** 你推一个方案，与其讲一堆道理，不如说"隔壁部门上个月就是这么做的，效果不错"。案例比道理有说服力，因为案例给了对方一个"别人已经验证过"的安全感。

但要注意——社会认同用过头了就是"从众绑架"。"大家都这么干"不能成为唯一理由，得配上事情本身的道理。

**第四把：喜好。**

人更愿意答应自己喜欢的人。这不用解释。

正道用法：**先建立好感，再谈事情。** 好感从哪来？从真诚的关心、真实的赞美、共同的经历里来。你记住同事孩子的名字、记得他上次提过的爱好、在他难的时候搭把手——这些都是在存"喜好"的款。等你开口求人时，这笔款就能取出来用。

喜好跟讨好是两码事。讨好是无原则地迎合，喜好是真诚地建立连接。区别在于——讨好的人没有自我，喜好的人有自我，只是也关心别人。

**第五把：权威。**

人倾向于服从权威——专家、领导、有头衔的人。

正道用法：**展示专业，而不是亮头衔。** 真正的权威来自"我在这件事上研究得深，我说的有依据"，而不是"我是领导你得听我的"。你在沟通里展示的是你的专业判断、你的数据支撑、你的过往经验，对方自然认你的权威。

邪道用法是"拿鸡毛当令箭"，用职位压人。短期有效，长期损耗信任。

**第六把：稀缺。**

人觉得"少的东西更值钱"。机会越少，越想要。

正道用法：**强调机会的稀缺性，但不说谎。** "这个窗口期就这两周""这个名额我们部门只有一个"——如果这是真的，说出来能让对方更重视。但稀缺不能编造，编出来的稀缺一旦被识破，信任归零。

## 伦理边界：武器是中性的，用法分正邪

讲到这，必须说一句重的。

西奥迪尼这六把武器，本身是中性的，像一把刀。刀能切菜，也能伤人。区别在用刀的人。

正道用，是"让对方愿意配合做对的事"——这件事本身对双方、对组织都有好处，你用影响力让对方更愿意配合，是双赢。

邪道用，是"操控对方做对他不利的事"——这件事只对你有好处，对方做了会吃亏，你用影响力让他看不出吃亏，这是操控。

西奥迪尼自己说过，他写《影响力》的目的不是教人操控，是让人识别操控。你懂了这六把武器，一来能用正道提升自己的沟通效率，二来能识别别人在用什么招对你，不被套路。

职场里，长期主义的人用影响力建立信任，短期主义的人用影响力收割利益。前者越用越顺，后者用几次就没人信了。

这一点，和真诚待人是一脉相承的。影响力是术，真诚是道。术可以学，道不能丢。你用六把武器去让对方愿意配合做对的事，是真诚的延伸；你用六把武器去坑人，迟早翻车。

## 选一把武器，设计一个动作

读完这一章，给自己布置一个作业。

选一个你正在推进的事——一个要找人配合的项目、一个要说服老板的方案、一个要谈下的客户。

想清楚：这件事，用哪把武器最有效？

- 如果对方是"你帮过我我才帮你"的性格，用**互惠**——先在别的事上帮一把。
- 如果对方需要逐步建立信心，用**承诺与一致**——先让他答应一个小事。
- 如果对方怕担风险，用**社会认同**——拿同类案例给他看。
- 如果对方对你还不太熟，用**喜好**——先建立连接，再谈事情。
- 如果对方需要专业判断，用**权威**——把你的依据摆出来。
- 如果对方在犹豫拖延，用**稀缺**——让他知道窗口期有限。

想清楚之后，设计一个具体的沟通动作——什么时候说、怎么说、说什么。一个动作就够，别贪多。

然后去执行，执行完复盘：这把武器管用吗？对方反应如何？下次换一把会不会更好？

影响力靠练出来，不是天生的。练的前提是——你心里有那六把武器，知道什么时候拔哪一把。
