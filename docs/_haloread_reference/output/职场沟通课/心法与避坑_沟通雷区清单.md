# 沟通雷区清单

## 一句话丢掉的晋升

老孙在组里干了五年，资历够、能力够、人缘也行。那年组里有个晋升名额，领导私下跟他透底："这次基本是你，准备一下述职。"

老孙高兴，中午跟几个同事吃饭，多喝了两杯。席间有人聊起另一个组的组长老赵，说老赵最近被客户投诉了。老孙接了一句："老赵那人吧，做事是有一套，但太抠细节，客户受不了。"

这话传到老赵耳朵里，又传到领导耳朵里。领导找老孙谈话："你跟我说老赵不行，那以后你跟老赵的组怎么协作？"

晋升的事黄了。领导没明说是因为这句嘴，但年底的述职名单里没有老孙。老孙后来才知道，领导在评审会上说了一句："老孙业务没问题，但嘴上没把门的，再观察一年。"

一句饭桌上的话，把五年的积累抹掉一半。

更冤的是小林。小林周末加班到晚上十点，回家路上发了条朋友圈："又是被公司榨干的一天，资本家果然没有心。"配了张深夜地铁的照片。

她设了"仅同事可见"——她以为这样安全。结果周一上班，领导把她叫进去："朋友圈我看到了。觉得累可以跟我谈，发朋友圈解决不了问题，还影响团队士气。"

小林懵了——她没加领导微信。后来才知道，同事截了图转发到了一个有领导在的小群。那条朋友圈成了她"负能量、不团结"的标签，跟了她大半年，直到她离职。

职场里有些话，说出口就收不回；有些事，做下去就定标签。这就是雷区。

## 雷区的本质：说出去的话不属于你

很多人对雷区有误解，觉得"我又没说错，凭什么算我头上"。可职场的话，对错不重要，效果才重要。

你说"老赵抠细节"，这话本身不算错——老赵确实抠细节。但这话从你嘴里出去，传到老赵耳朵里，就是"老孙在背后说我坏话"。传到领导耳朵里，就是"老孙不团结同事"。传到评审会，就是"老孙嘴不严"。

同一句话，在你嘴里是事实，在别人耳朵里是立场，在传话人嘴里是武器。你控制不了它怎么传，你只能控制说不说。

雷区的本质是：**说出去的话不属于你**。它属于听的人，属于传的人，属于所有听到它的人。每个人都会按自己的立场重新解读它、加工它、使用它。你以为你在说A，传到最后变成Z，Z算你头上。

所以雷区的关键，不在于"不能说真话"，而在于"有些真话不能在这个场合、对这些人、用这种方式说"。真话该说，但要分场合、分对象、分方式。不分场合的真话，比假话还伤人。

韩非子在《说难》里讲过一句话：夫事以密成，语以泄败。

大意据《韩非子·说难》：事情往往因为保密而成功，因为泄露而失败。

这话两千年前就讲透了。该保密的事说出去就败了——不只是大事，连你"对某人有看法""对公司不满""想跳槽"这种私事，说出去都会败。败掉的不只是事，更是你这个人。

## 十大雷区清单

把职场里最容易踩的雷列出来，十条，每条配一个替代说法。

**一、甩锅话。**

雷区："这不关我事。""这块不是我负责的。""你找别人吧。"

这三句话一出，对方听到的就变成了"我不担责"——根本不会往"这不是我的事"那想。领导听到，给你贴"没有担当"的标签；同事听到，给你贴"不好合作"的标签。

替代说法："这块是XX在跟，我帮你对接一下。""这事我不熟，但我可以帮你问问。"——你不接事可以，但别甩。指条路，比甩锅强一百倍。

**二、情绪话。**

雷区："你怎么这样啊！""你到底懂不懂啊！""气死我了！"

情绪话的特征是"对人不对事"。你说的是情绪，对方听到的是攻击。一攻击，对方就防御，一防御，沟通就死了。

替代说法：把情绪翻译成事实。"这个结果跟我预期的不一样，我们看看哪里出了问题。""我有点担心这个节奏，咱们对一下。"——说事实，不说情绪。事实能讨论，情绪只能对抗。

**三、八卦话。**

雷区："你知道吗，XX跟XX好像……""听说XX要走了。""XX那个事啊，我跟你说……"

八卦是职场最廉价的社交货币，也是最贵的雷。你传一次八卦，听的人心里会想："他今天能跟我说XX的事，明天就能跟别人说我的事。"你以为是亲密，对方觉得你嘴不严。

替代说法：不接八卦。别人跟你说"你知道吗XX……"，你回"是吗，我不太清楚"。不传、不评、不接。几次下来，没人再跟你传八卦，你反而成了"靠谱"的人。

**四、抱怨话。**

雷区："公司怎么这样。""领导真是的。""这活没法干了。"

偶尔抱怨人之常情，但抱怨成了习惯，你就成了"负能量源"。领导最怕的，与其说是员工有意见，不如说是员工有意见不说、到处散。散的负能量会传染，传染了团队就废。

替代说法：把抱怨翻译成建议。"这个流程有点慢，能不能优化一下？""这个安排我有顾虑，跟您聊聊。"——抱怨是问题导向，建议是方案导向。前者让人烦，后者让人敬。

**五、承诺话。**

雷区："这事包在我身上。""没问题，肯定搞定。""我保证。"

轻易承诺是职场大忌。你拍胸脯时痛快，做不到时狼狈。一次做不到，信用扣一分；两次做不到，信用清零；三次做不到，你成了"嘴炮"。

替代说法："我尽力，但有几个环节有风险，到时候再说。""这个我接，但需要XX配合，我去协调。"——承诺留余地，做不到有退路。余地体现的是负责任，并非不担当。

**六、站队话。**

雷区："我跟你一头的。""我支持你，不支持他。""咱们一起对付XX。"

公司里的站队，今天赢的明天可能输。你今天站A，明天A倒了，你跟着倒。你今天说"我不支持B"，明天B上台了，你第一个被收拾。

替代说法：表态对事不对人。"这个方案我有几个顾虑。""这个做法我同意，那个我有不同意见。"——对事可以表态，对人别轻易站队。立场可以模糊，事情必须清楚。

**七、轻蔑话。**

雷区："这都不会？""就你？""你也配？"

轻蔑话是关系毒药。一句轻蔑，记一辈子。你今天轻蔑了同事，他记着；十年后他上去了，你还在原地，他记着。职场是个长跑，别在路上埋雷。

替代说法：把轻蔑翻译成帮助。"这块我熟，我帮你看看。""这个有点绕，我们一起捋一下。"——同样的事，轻蔑让人恨，帮助让人记。

**八、比较话。**

雷区："你看人家XX，再看看你。""XX早就做完了，你怎么还没好？""你要是有XX一半就行了。"

比较话伤人于无形。被比较的人，不会因为比较而奋进，只会因为比较而怨恨。你拿他跟别人比，他恨你；他恨你，就不给你好好干。

替代说法：直接说期望。"这个我希望下周三之前能给我。""这个质量还差一截，再打磨一下。"——说标准，不说别人。标准是客观的，比较是伤人的。

**九、绝对话。**

雷区："从来没人这样做过。""绝对不可能。""永远行不通。"

绝对话把路堵死。你一说"绝对不可能"，万一可能了呢？你一说"从来没人做"，万一有人做了呢？绝对话一被证伪，你的专业度就打折。

替代说法："目前我没见过成功的案例。""按现有条件，难度很大。""我倾向于不做，但可以再评估。"——留余地体现的是承认不确定性，并非不专业。承认不确定，反而显得专业。

**十、隐私话。**

雷区：聊同事的工资、聊领导的私生活、聊别人的家庭状况、聊自己的隐私。

隐私话是最大的雷。聊别人的隐私，是嘴不严；聊自己的隐私，是授人以柄。你今天跟同事聊了你的工资，明天全组都知道了；你今天跟同事聊了领导的私事，后天领导就知道了。

替代说法：不聊。别人聊，你岔开。聊隐私的圈子，迟早出事。出事的时候，参与过的人都有份。

## 三人成虎：传三遍就是事实

《战国策》里有个典故，叫"三人成虎"。

大意据《战国策》：庞葱要陪魏太子去赵国做人质，临走前问魏王："如果一个人说街上有老虎，您信吗？"魏王说不信。"两个人说呢？""半信半疑。""三个人说呢？""信了。"庞葱说："街上显然不会有老虎，但三个人说就有了。赵国离魏国比街上远多了，议论我的人不止三个，希望您明察。"庞葱走后，果然有人说他坏话，魏王后来真不信他了。

这典故讲的是谣言。但放在雷区上也成立——你的一句话，被传三遍，就成了"事实"。

老孙说"老赵抠细节"，传到第三个人耳朵里，变成了"老孙说老赵不行"。小林发朋友圈"被公司榨干"，传到第三个人耳朵里，变成了"小林骂公司是资本家"。

你控制不了传话的人，但你能控制源头。源头在你这里断了，就传不出去。这就是为什么雷区要"不说"——说了肯定有事，不说至少少有事。

## 实操：本周雷区自查

读完这一章，做一次本周雷区自查。

回想你最近一周说过的所有"非业务"的话——饭桌上、茶水间、微信里、电话中。对照十条雷区，看看踩了几条。

每踩一条，写下来：当时说的是什么？在什么场合？对谁说的？传出去会变成什么？

然后写替代说法。如果当时用替代说法，效果会怎样？

这个练习的目的，就是让你对自己的"嘴"多一份觉察。觉察是改变的第一步。你不知道自己踩了雷，下次还会踩；你知道了，下次开口前会停半秒。那半秒，能救你很多事。

写完之后，挑一条最常踩的雷，下周专门盯它。比如你最常说"这不关我事"，下周每次想说之前，强制改成"这块XX在跟，我帮你对接"。一条一条改，比一次改十条管用。

## 一句话

雷区的要害，在于说话之前过一遍脑子。过脑子本身就是成熟的体现。成熟的人也会说，只是他们知道什么话在什么场合能说、什么话在任何场合都不能说。

嘴上没把门的人，能力再强也走不远。把门守住，路才长。
