# 该开口，还是该沉默

## 谈判桌上的两难

老周跟一家供应商谈了三小时，回来一脸疲惫。

我问怎么了。

老周说："对方报了个价，我没接话，想着晾一晾。结果对方也闷着不说话，桌上冷了五分钟，最后对方说'那再考虑考虑'，起身走了。"

"我本来想等对方先松口，结果把单晾没了。"

老周的徒弟小陈上个月也谈过一单，情况正好相反。对方报了个价，小陈急着反驳，噼里啪啦说了一通"我们预算有限""市场行情不是这样""您这个价格没诚意"，结果对方脸色一沉："既然没诚意，那就别谈了。"

老周该开口时没开口，小陈该沉默时没沉默。

两个人都栽在同一个问题上：**不知道什么时候该说话，什么时候该闭嘴。**

这个问题，两千多年前就有人给出了答案。那个人叫鬼谷子。

## 捭是开，阖是闭

《鬼谷子》讲捭阖。

捭，是开。阳，主动，言。
阖，是闭。阴，被动，默。

《鬼谷子·捭阖》里说："捭之者，开也，言也，阳也；阖之者，闭也，默也，阴也。"

[注释：捭是开启、言语、属阳；阖是闭藏、沉默、属阴。]

这话听着玄，翻译成白话就是：沟通有两手，一手是"说"，一手是"不说"。说，是主动出击，把信息放出去，把意图亮出来；不说，是被动收敛，把耳朵打开，把对方的底摸清。

新手以为沟通就是"说"——说得越多越占便宜。老手才知道，"不说"比"说"更难，也更有用。

但捭阖的关键在**交替**。

只会捭不会阖，是话痨，把对方说烦了，把自己说死了；只会阖不会捭，是闷葫芦，对方摸不透你，谈不下去。

高手的本事在切换——该说的时候说到位，该听的时候听到位，因人因时切换。

## 先说一个史料问题

讲鬼谷子之前，得先交代一件事。

鬼谷子这个人，《史记》里提过一笔——苏秦、张仪"俱事鬼谷先生学术"。但鬼谷子到底是谁，长什么样，活了多久，正史里几乎没有可靠记载。后世关于鬼谷子的种种传说，很多是层层叠加的，越说越神。

今本《鬼谷子》这本书，署名鬼谷子，但成书年代历来有争议。有学者认为部分篇章是战国纵横家的辑录，也有学者认为其中掺入了后人的拟托之作。书里的思想是不是鬼谷子本人原话，没办法考证。

所以这一章讲的"鬼谷子说"，更准确的说法是"《鬼谷子》这本书里说"。鬼谷子可能是托名，今本《鬼谷子》成书年代有争议，不能当成确定的史实来叙述。

但书里的智慧是真的。捭阖这套东西，不管谁写的，确实管用。

## 苏秦的前倨后恭

《战国策》和《史记》里都记了苏秦的故事。

苏秦是鬼谷子的弟子（按传统说法），学的是纵横术，也就是捭阖之道。

（注：1973年马王堆帛书《战国纵横家书》出土后，学界对苏秦活动年代有修订，传统史料中部分苏秦事迹实为苏代或年代有误，本节取传统叙事作说明。）

苏秦年轻的时候去秦国游说，没说动，钱花光了，灰溜溜回家。到家的时候，妻子不下织机迎接，嫂子不给他做饭，父母不跟他说话。

苏秦受了这个刺激，发愤读书，"刺股"的典故就出在这里——困了就拿锥子扎大腿，扎醒了接着读。

后来苏秦再去游说，这回说成了，挂六国相印，合纵抗秦，风光无两。

路过老家，他嫂子趴在地上迎接，不敢抬头。苏秦笑问："何前倨而后恭也？"——嫂子你之前那么傲，现在怎么这么恭敬？

嫂子实话实说："见季子位高金多也。"——因为你现在地位高、钱多。

苏秦叹了口气，说了一句千古名言："此一人之身，富贵则亲戚畏惧之，贫贱则轻易之。"——同是我这个人，富贵了亲戚就怕我，贫贱了亲戚就看轻我。

这个故事里藏着捭阖的影子。

苏秦第一次出门，是"捭"得太急——本事没到家，话先放出去了，结果碰一鼻子灰。回家闭门读书那几年，是"阖"——收敛、积累、等时机。第二次出门，该捭的时候捭到位，六国相印到手。

嫂子的态度变化，也是捭阖——苏秦贫贱时，嫂子"阖"着他，不搭理；苏秦显达时，嫂子"捭"着迎，热情得过分。

人情冷暖，本来就是捭阖交替。

## 捭阖四用

捭阖有具体的用法。拆成四条。

**第一，何时捭。**

信息不对称在你这边的时候，捭。你知道对方不知道的事，主动说出来，掌握节奏。

对方犹豫不决的时候，捭。对方卡在某个点上拿不定主意，你给一个明确的方向，推他一把。

需要亮底线的时候，捭。谈了半天谈不到点上，你把底线摆出来，省得互相猜。

**第二，何时阖。**

对方情绪上头的时候，阖。对方在气头上，你说什么都是错，闭嘴让他先冷静。

你不了解对方底牌的时候，阖。不知道对方要什么、能给什么，先听，听完再开口。

对方在试探你的时候，阖。对方抛一句话看你反应，你一接话就露底，闭嘴让他猜。

**第三，捭的尺度。**

捭的尺度是"说到位就停"。

老谈判手有个习惯：说完关键一句，就闭嘴，看着对方。这一句是捭，之后的沉默是阖。让对方消化，让对方回应。很多新手说完关键一句还怕对方没听懂，接着解释、补充、强调，结果把关键句淹没了。

捭的尺度，是"少而准"。一句话说到位，胜过十句话绕圈子。

**第四，阖的时机。**

阖的时机是"听完关键信息就开口"。

对方说话的时候，你在阖。但阖要带着问题听。听到关键信息——对方的底线、对方的顾虑、对方的真实需求——立刻从阖转捭，针对这个信息回应。

阖太久，对方以为你没在听，或者以为你心虚；阖太短，关键信息没听全就开口，容易说错话。

## 谈判桌上的捭阖

回到老周那单。

老周的问题，是"阖"过了头。对方报价后，老周想晾一晾，结果晾成了冷场。对方以为老周没诚意，起身走了。

正确的做法是：对方报价后，先"阖"一下——重复一遍对方的报价，确认自己听清楚了，"您说的是一百二十万，对吧？"这一句是阖中的小捭，确认信息，也给对方一个回应的信号。

然后真正地"阖"——沉默几秒，做出思考的样子。这几秒的作用是给对方心理压力，让他自己琢磨"是不是报高了"。

如果对方主动加一句"这个价格可以再聊"，说明对方心虚了，你转"捭"——"那您看多少能聊？"把球踢回去。

如果对方也闷着不说，你转"捭"——亮一个你的预期，但留余地："我们这边能接受的区间在九十到一百万，您这个一百二，差距有点大，您看能不能往中间靠？"

捭阖交替，节奏在手里。

小陈的问题正好相反，是"捭"过了头。对方一报价，小陈噼里啪啦反驳一通，把对方说急了。

正确的做法是：对方报价后，先"阖"——听完，停一下，问一句"您这个价格是怎么算出来的？"让对方说。对方说的时候，你在听、在算、在找破绽。

听完再"捭"——针对对方算账里的漏洞，精准地回应一句："您刚才说含运费，但我们这边自己提货，运费这块是不是可以减掉？"一针见血，不绕。

## 汇报和面试里的捭阖

捭阖不只用在谈判。

**汇报里。**

领导问你"这个项目进展怎么样"，新手往往"捭"过头——从立项开始讲，讲了十分钟还没讲到重点，领导脸都黑了。

老手先"阖"——快速判断领导想知道什么。领导问进展，可能是想知道"按时不按时"，也可能是想知道"有没有问题"，也可能是想知道"要不要我决策"。判断清楚了再"捭"，一句话点到领导的心上。

拿不准的时候，先反问一句："您是想了解整体进度，还是想看具体哪一块？"这一句是"阖"——把领导的需求摸清楚，再有的放矢地"捭"。

**面试里。**

面试官问"你为什么离开上一家公司"，新手要么"捭"过头——大倒苦水，把上家公司说了一通；要么"阖"过头——支支吾吾，半天说不出一句。

正确的做法是先"阖"——听清楚面试官问的是什么，是想知道你的职业规划，还是在试探你跟前任老板的关系。然后"捭"——一句话点到："我想往XX方向发展，上一家平台有限，所以出来找更大的空间。"说完就停，让面试官接着问。

捭阖交替，不抢话，不闷话。

## 实操

复盘你最近一次沟通——可以是谈判，可以是汇报，可以是跟同事讨论问题。

按下面四个问题想：

1. 这次沟通里，你"捭"了几次，"阖"了几次？是捭多阖少，还是阖多捭少？
2. 该捭的时候你捭了吗？该阖的时候你阖了吗？有没有该闭嘴的时候抢话，或者该开口的时候闷着？
3. 你捭的时候，是说到位就停，还是说了一堆没重点？你阖的时候，是带着问题听，还是发呆？
4. 下次同样的场景，你会在哪个节点切换捭阖？

把答案写下来。写下来你才会发现，自己到底是"老周型"还是"小陈型"。

## 一句话

捭是开，阖是闭——高手的本事在切换，该开口时开口，该沉默时沉默。
