# 为什么职场人必须练沟通

## 一、入职第三个月的震惊

小林是今年刚入职的应届生，名校硕士，专业对口，技术扎实。入职前他以为，工作就是坐在工位上把活儿干好，凭本事吃饭。

入职第三个月，他算了一笔账，把自己吓了一跳。

这两个月，他真正埋头写代码、做方案的时间，加起来不到三成。剩下七成在干嘛？跟产品经理对需求，跟测试扯一个问题算谁的责任，跟前端约接口联调时间，给主管写周报，在群里回复"收到""好的""我看一下"，开会、开会、开会。

他跟同届入职的同学吐槽，发现大家都一样。一个同学说得更直白："我以为我是来写代码的，结果我是来开会的，代码是加班写的。"

小林有点想不通：我寒窗七年学的专业，到头来七成时间在沟通？那我这专业还学它干嘛？

他这个想不通，恰恰是大多数职场新人的盲区。

## 二、沟通是能力的放大器

很多人有个根深蒂固的错觉：专业能力是1，沟通是后面的0，可有可无。甚至有人觉得沟通是"不务正业"，专业不行的人才去搞它。

这个错觉，得反过来理解。

专业能力是你的底子，沟通是放大这个底子的杠杆。一个公式可以这么写：你在职场上的实际价值，等于专业能力乘以沟通杠杆。

专业能力是10，沟通杠杆是1，你的实际价值是10——你做了10分的事，别人知道10分，能用上10分。专业能力是10，沟通杠杆是0.1，你的实际价值是1——你做了10分的事，别人只看到1分，能用上1分，剩下9分白做。专业能力是8，沟通杠杆是2，你的实际价值是16——你做了8分的事，但通过沟通把它讲清楚、推下去、对齐了，最后发挥出16分的效果。

这就是为什么会出现"能力一般但会沟通的人升职了，能力强但不会沟通的人原地踏步"。那个会沟通的人升上去，自有它的道理——他实际交付的价值更大。公司看的，是谁交付的价值更多。专业能力决定你的下限，沟通杠杆决定你的上限。

这个道理放公司层面一样成立。一个公司技术再强、产品再好，要是不会跟市场沟通、不会跟投资人沟通、不会跟员工沟通，照样死。反过来，一个公司能在绝境里翻盘，往往靠的就是掌舵人那几张嘴。

## 三、艾柯卡和国会的那个下午

1978年底，李·艾柯卡接手克莱斯勒汽车公司。那时候的克莱斯勒，离破产就差一口气——亏损严重，库存堆积，现金快见底，市场上没人看好。

艾柯卡是搞汽车的老手，他心里清楚，要救克莱斯勒，光靠内部降本增效来不及了，得找一笔钱撑过去。他想到的唯一出路，是找美国国会要一笔联邦贷款担保——大概十五亿美元。

这个想法在当时近乎天方夜谭。一家私营汽车公司，凭什么让纳税人掏钱救你？国会山上反对声一片，媒体也冷嘲热讽。艾柯卡心里明白，这事成不成，账本说了不算，他那张嘴说了算。

1979年底到1980年初，艾柯卡一次次上国会作证。他没有念稿子，没有打感情牌，他拿数据说话——克莱斯勒倒了，几十万人失业，供应商连锁倒闭，政府最后掏的失业救济和税收损失，比这十五亿多得多。他承诺自己只拿一美元年薪，跟公司共患难。他讲得坦率、有底气，也讲得让人没法不信。

国会最终通过了贷款担保。克莱斯勒拿到钱，缓过这口气。艾柯卡没停在国会，他接着跟员工讲、跟经销商讲、跟消费者讲。他亲自上电视拍广告，那句"如果你能找到一辆更好的车，买它就是了"成了美国广告史上的经典。1983年，克莱斯勒提前七年还清贷款，起死回生。

艾柯卡后来在自己那本畅销自传里写过，他这辈子做过最难的决策不少，可真正救活克莱斯勒的，是他能让国会、能让员工、能让消费者相信克莱斯勒值得救。一个汽车老手的专业能力，是这盘棋的底子；把这盘棋下活的，是他把底子讲出去、推下去、对齐了各方利益的那张嘴。

专业能力是1，沟通是后面的杠杆。艾柯卡这个杠杆，把克莱斯勒从棺材里撬了出来。

## 四、沟通卡着你的五个节点

回到普通职场人身上。沟通不是某一次晋升才用得上的事，它卡着你职业生涯的每一个关键节点。

**第一个节点：入职。**

入职头三个月，你能不能快速搞清楚谁是谁、事怎么流转、规矩是什么，全靠沟通。问对了人，少走两周弯路；问错了人，或者闷头不问，三个月还在原地打转。新人最大的忌讳，往往是不敢问、不会问，倒不在于会不会做。

**第二个节点：转正。**

转正答辩真正考的，是你能不能把做的活讲清楚、讲出价值，做了多少活反在其次。同样三个月的活，有人讲得让领导眼前一亮，有人讲得让领导怀疑"这三个月你到底干了啥"。差的就是沟通。

**第三个节点：晋升。**

晋升看的是影响力。影响力怎么来？靠你做的事被看见、被认可、被扩散。你做了个工具帮全组提效，自己不说，没人知道；跟领导汇报一次、在组里分享一次、写个文档发出来，影响力就起来了。晋升答辩更是沟通的硬仗——你得用半小时讲清你这一年凭什么升。

**第四个节点：跨部门。**

跨部门协作是职场沟通的重灾区。你不归他管，他不归你管，凭什么人家配合你？靠的就是沟通——把共同利益讲出来，把对方的好处讲清楚，把风险提前说在明处。跨部门推不动事，十有八九问题出在沟通没到位，事本身往往没问题。

**第五个节点：离职。**

离职也要沟通，而且要沟通得体面。怎么跟领导提、怎么交接、怎么不撕破脸、怎么留个好人脉，全是沟通。离职沟通搞砸的，行业圈子里传一圈，下次背调就埋了雷。

从进到出，沟通卡着每一个节点。哪一节断了，职业生涯都得卡一下。

## 五、自测：你的沟通短板在哪

说了这么多，落到你自己身上。下面这份自测清单，拿支笔对着打勾，看看你的沟通短板到底在哪一层。

**信息层（每条符合打1分）：**

- 领导问我进度，我能三句话讲清楚做到哪、卡在哪、下一步。
- 我写的邮件和消息，别人看一遍就懂，不用反复追问。
- 接需求时，我会主动问清目标、边界和截止时间，而不是埋头就做。
- 开会发言，我能讲出重点，不绕圈子。

**信任层（每条符合打1分）：**

- 我答应别人的事，基本都能按时做到；做不到会提前说。
- 出了问题，我会主动承认，而不是先想怎么甩锅。
- 同事遇到难题，愿意来找我；我也愿意帮。
- 过去一年，我没因为一次沟通失误，让合作关系明显变差。

**利益层（每条符合打1分）：**

- 跟人谈事情，我会先想对方要什么，再想我要什么。
- 遇到分歧，我能找到双方都能接受的方案，而不是硬顶或硬让。
- 跨部门推事，我能讲清"配合你有什么好处"。
- 谈薪、谈资源时，我敢开口，也讲得出理由。

12分以上，沟通已经是你的强项，这门课帮你更上一层。6到11分，你有基础但有明显短板，重点补弱项。5分以下，别急，这门课从最基础的讲起，跟着练就行。

记下你失分最多的那一层，那就是你接下来最该补的地方。

## 结语

专业能力决定你的下限，沟通杠杆决定你的上限。职场里七成的时间在沟通，这并非公司低效，是因为沟通本来就是职场的主活儿。把这门主活儿练好，你的专业能力才能真正兑现成价值。
