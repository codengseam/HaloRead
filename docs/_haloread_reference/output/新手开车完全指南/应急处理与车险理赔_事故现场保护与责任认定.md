# 出事别慌先拍照再挪车

## 一、入戏：那个挪车挪没了几万块

那年三月，杭州一辆出租车变道蹭到旁边车道的私家车，双方车损都不重，保险杠刮花。出租车司机说「兄弟，堵车了，咱挪到路边再说」，私家车主心想也是，没拍照就跟着挪了。挪到路边后，出租车司机翻脸：「是你追的我尾，你全责。」私家车主拿不出证据，行车记录仪角度没拍到碰撞瞬间，交警到场后只能根据车辆最终位置和受损部位判断——结果认定为双方责任无法查清，各负同等责任。私家车主多赔了 3 万多。

事后私家车主才知道，如果当时先拍照固定现场，碰撞部位、车道线、车牌一应俱全，变道全责跑不掉。挪车之前没拍照，等于把责任认定权拱手让人。

新手在事故现场最大的问题是慌。一撞车脑子空白，要么任由对方摆布，要么急着挪车「不影响交通」。这一篇就把事故现场处置的标准流程讲清楚：有人伤先救人报警，无人员伤亡小事故先拍照再挪车，责任争议不挪车等交警。

## 二、破题：先救人、再拍照、后挪车，顺序不能错

事故现场处置的核心是顺序。顺序错了，轻则责任说不清，重则触犯法律。

第一步永远是判断有没有人伤。有人伤，救人放第一位——拨打 120 急救，同时报警 122。在救人过程中尽量不破坏现场，但如果救人需要移动车辆或物品，标记位置（用粉笔、石块在地上画线，或拍照记录原位置）。法律明文规定，发生人身伤亡事故必须报警，不得私了，不得擅自撤离现场（《道路交通安全法》第七十条）。

第二步是拍照固定现场。无论人伤事故还是车损事故，拍照都是核心动作——照片是你后续责任认定和保险理赔的证据。拍完照、固定证据后，再考虑是否挪车。

第三步是判断能否挪车。仅当「无人伤 + 车辆可自行移动 + 双方对事实责任无争议 + 证件保险齐全」四个条件同时满足时，才能挪车快处。只要有人伤、有争议、涉酒驾无证、撞公共设施、对方逃逸等情形，必须保护现场等交警。

为什么不能直接挪车？因为挪车后现场证据消失，责任认定只能靠双方陈述和最终车辆位置，遇到对方翻脸你就说不清。拍照固定证据后再挪车，既不耽误交通，又保住了责任认定的依据。

责任认定有一套明确的法律原则：追尾后车全责、变道全责、转弯让直行、右转让左转、支路让干路。这些原则有法条支撑，交警按此定责，保险公司按此理赔。下面方法论部分展开讲拍照取证、挪车条件、责任认定三块。

## 三、方法论：拍照取证 + 挪车条件 + 责任认定

### 拍照取证：五张照片不能少

事故现场拍照有标准，五张照片覆盖全部取证需要：

第一张：事故全景。距离事故车辆 5—10 米，拍下两车相对位置、车道线、周围参照物（路口、标线、护栏）。这张照片用来还原事故发生时车辆在车道内的位置。

第二张：车头方向 45 度全景。从车头斜前方 45 度角拍，能同时看到两车的前部、车牌、碰撞部位。这张照片是责任认定的核心证据。

第三张：车尾方向 45 度全景。从车尾斜后方 45 度角拍，对称于第二张。两张 45 度照片构成事故的立体还原。

第四张：碰撞部位特写。直接拍两车接触部位的细节，显示碰撞角度、刮擦方向、漆面痕迹。这张照片用来判断碰撞方式（追尾、侧碰、刮擦）。

第五张：双方车牌特写。把双方车牌拍清楚，作为车辆身份证据。

如果有刹车痕迹、散落物（碎片、玻璃）、交通标志标线，额外补拍。夜间或雨雾天打开闪光灯，确保照片清晰。拍照时不要站在车道中央——人在车道内被追尾是二次事故，比事故本身更危险。

### 挪车条件：四个条件同时满足

拍照固定证据后，判断能否挪车。只有以下四个条件同时满足，才能挪车快处：

条件一：无人员伤亡。有人伤必须报警等交警，不得挪车。

条件二：车辆可自行移动。车辆损坏严重无法行驶，原地等拖车。

条件三：双方对事实及责任无争议。一方认全责或双方认可同等责任，可以挪车。如果对谁全责有争议，原地等交警。

条件四：证件保险齐全。双方驾驶证、行驶证、交强险均有效。无证、无牌、无保险必须报警。

满足四个条件后，把车挪到不妨碍交通的地点（路肩、匝道、停车场），再协商处理或走 12123 快处快撤流程（下一篇详讲）。

### 不能挪车的九种情形

《道路交通安全法》第七十条规定以下情形必须保护现场、立即报警，不得挪车：

1. 造成人员伤亡（含轻微伤）
2. 双方对事实成因有争议
3. 驾驶人酒驾/毒驾/无证
4. 车辆无牌无证无保险
5. 撞公共设施（护栏、路灯、绿化）无物损负责人在场
6. 载运危险物品
7. 车辆无法移动
8. 一方离开现场（逃逸）
9. 故意造成（碰瓷）

这些情形挪车等于破坏现场，可能承担不利后果。

### 责任认定的五条总原则

交警根据当事人行为对事故所起作用及过错严重程度确定责任（《道路交通安全法实施条例》第九十一条）。逃逸的，逃逸方承担全部责任（第九十二条）。常见全责认定原则有五条：

第一条：追尾后车全责。依据《道路交通安全法》第四十三条，后车应与前车保持足以采取紧急制动措施的安全距离。无论前车是急刹还是缓刹，追尾都是后车责任。

第二条：变道全责。依据《道路交通安全法实施条例》第四十四条，变更车道不得影响相关车道内车辆正常行驶。注意：变道后 3 秒内被追尾仍算变道方全责——「变道完成」需车身摆正且保持至少 3 秒稳定行驶。

第三条：转弯让直行。依据《道路交通安全法实施条例》第五十二条，转弯车让直行车先行。无论左转右转，撞了直行车就是转弯车责任。

第四条：右转让左转。相对方向行驶的两辆车同时转弯，右转让左转。因为左转车转弯半径大、占用路口时间长。

第五条：支路让干路。从支路进入干路，让干路车辆先行。

加上无信号灯路口的「右侧先行原则」（无标志标线控制时让右方道路来车先行），这套让行规则要刻进脑子，既能避免事故，也能在事故后判断责任。

### 责任争议：不挪车等交警

双方对责任有争议时，不要勉强协商，更不要挪车。原地保护现场，开双闪、摆三角牌、撤人到安全地带，报警等交警到场处理。交警根据现场痕迹、车辆位置、双方陈述、行车记录仪、监控录像综合判断，出具《道路交通事故认定书》。

对认定有异议的，可以在收到认定书之日起 3 日内向上一级公安交管部门书面申请复核。

## 四、避坑：事故现场的四种致命错误

### 错误一：撞了就挪车，不拍照

挪车后现场证据消失，对方翻脸你拿不出证据，责任说不清。拍照是挪车的前提，五张照片一张不能少。

### 错误二：有人伤也挪车

有人伤必须保护现场、立即报警，不得挪车。挪车破坏现场可能被认定为破坏证据，承担不利责任。救人需要移动车辆时标记原位置。

### 错误三：碰瓷、酒驾、无证也私了

碰瓷的、酒驾的、无证的、无保险的，私了等于被骗或违法。这些情形必须报警，让交警处理。对方催着私了，更要警惕。

### 错误四：在车道内理论

撞车后双方站在车道内理论责任，是二次事故高发场景。先撤离到安全地带，再协商或等交警。车道内的每一秒都是赌命。

## 五、践行：把五张照片练成习惯

事故现场处置，关键时刻能保住你钱袋子和法律权益的是肌肉记忆。建议在车内贴一张纸条，写两句话：

「撞车后：先看人伤 → 拨 120/122 → 拍五张照 → 满足四条件才挪车。」

「有人伤、有争议、酒驾无证、碰瓷，绝不挪车，报警等交警。」

平时上车前确认手机电量充足、行车记录仪正常工作。遇到事故时打开行车记录仪的「事件保存」功能，锁定碰撞前后的视频片段。五张照片拍完再考虑挪车，重要的事说三遍：五张照片拍完再挪车。

最后一句：撞车不可怕，可怕的是撞车后处置不当。先救人、再拍照、后挪车，顺序错一步代价翻倍。把五张照片和四个挪车条件刻进脑子，事故现场不慌不乱。
