# 加油站是爆炸危险区，一根烟能引爆一整车

## 一、入戏：在加油机旁扫码支付，被工作人员吼了一声

第一次自助加油，我在加油机旁掏出手机准备扫码支付，话音未落工作人员冲过来吼了一声：「加油区不能用手机！」我当时一头雾水——平时便利店扫码都没事，怎么加油站就不行？

回家查资料才明白，加油机周边 3 米是爆炸危险区，手机射频电磁辐射可能引燃油气混合气体。应急管理部《加油站作业安全规范》AQ 3010-2022 把这条写得清清楚楚。我那次差点违法还浑然不觉。

后来跑长途多了，发现加油站和服务区的事故新闻不少——有人在加油区抽烟引燃油气、有人在服务区疲劳驾驶出事故、有人长时间占用车位被贴条。这些地方新手以为是「日常停车休息」，其实是有严格作业规范的高风险区域。这一篇讲加油站和服务区的规矩。

## 二、破题：加油站是「爆炸危险作业区」，不是「普通停车场」

新手对加油站最大的误解，是把它当成「加个油的便利店」。其实加油站是「爆炸危险作业区」——汽油在加油过程中会挥发成油气，油气与空气混合形成可燃混合气体，遇到明火、静电、电火花即引爆。加油机周边 3 米是规范划定的爆炸危险区，这个区域内的任何火源、电火花、金属撞击都可能引发爆炸。

所以加油站的规矩，本质上是「消除一切点火源」。应急管理部 AQ 3010-2022《加油站作业安全规范》和 XF/T 3004-2020《汽车加油加气站消防安全管理》把每一条都写成了强制要求——熄火加油、禁烟禁明火、加油区禁手机、不维修车辆、不抛掷金属、雷暴天气停止作业。这些不是「建议」，是「必须」，违反就是违法，出事故就是刑责。

新手最容易踩的三条红线：一是加油不熄火（发动机运转产生电火花、排气管高温，可能引燃油气）；二是加油区用手机（射频辐射引燃油气）；三是加油区吸烟或明火（直接引燃油气）。这三条任何一条引发事故，都是全责 + 刑责。

服务区则是另一套规矩。服务区是「长途驾驶的强制休息点」——疲劳驾驶是高速事故的主要诱因，《道路交通安全法实施条例》规定连续驾驶不超 4 小时、每 2 小时强制休息 20 分钟。服务区不是「想停就停、想走就走」的停车场，是有分区、有规范、有时长限制的休息场所。新手在服务区常犯的错——长时间占车位、不停规定区域、疲劳硬撑不进服务区——都可能引发事故或被处罚。

把这个原理想通，加油站和服务区的规矩就顺了。

## 三、方法论：加油作业规范 + 自助加油静电 + 服务区规范

### 进站限速与停放：5km/h 入站，按指引停放

加油站进站限速 5km/h——这个速度比步行快一点，几乎贴着怠速进。限速的原因是进站时车还在运转，慢速防追尾、防擦碰加油机、防碾压管线。

进站后按工作人员指引或地面箭头停到加油机旁，车头朝外（方便驶离）。停车后立即熄火、拉手刹、关闭车载电源（包括点烟器、USB 充电、行车记录仪停车监控等）。驾乘人员下车到安全区等候（加油作业区外），不要留在车内。

站内禁止随意行驶和掉头——加油站通道狭窄、加油机多、人员走动多，随意行驶容易撞人撞车。需要掉头或换加油机，按工作人员指引走。

### 熄火加油：发动机必须关停

熄火加油是加油站第一红线。原因有三：发动机运转产生电火花（火花塞、点火线圈），可能引燃油气；排气管温度高（200-400℃），油气接触即燃；发动机运转时车辆有震动，可能带动油枪晃动溅油。

正确顺序：停稳 → 挂 P 挡（自动）/ 空挡（手动）→ 拉手刹 → 熄火 → 关闭所有车载电源 → 下车到安全区。加油完毕 → 拧紧油箱盖 → 油枪归位 → 确认无渗漏 → 上车 → 启动发动机 → 驶离。

加油过程中严禁启动车辆、开关车门、用加油枪敲油箱口——这些动作都产生电火花或金属撞击火花，可能引燃油气。加油没结束就启动车辆，是新手常犯的错，工作人员会立刻制止。

### 加油作业区红线：禁烟禁明火禁手机禁维修

加油作业区（加油机周边 3 米爆炸危险区）的四条红线：

禁烟禁明火：加油站全域严禁吸烟、使用打火机、明火作业。吸烟是加油站最严重的违规，油气遇明火瞬间爆燃，2024 年国内仍有因加油区吸烟引发火灾的事故报道。

禁手机：加油作业区严禁使用手机（含通话、扫码支付、拍照）。手机射频电磁辐射在油气环境中可能产生电火花引燃油气。需要手机支付的，到站内便利店或收银区（设有可燃气体声光报警装置的区域）使用，或熄火驶离加油机后使用。

禁维修车辆：不在作业区维修车辆、敲打金属、抛掷金属物品。维修产生的火花、金属撞击火花都可能引燃油气。需要维修的，驶离加油站到维修点。

禁易燃易爆品：不在加油区存放或携带易燃易爆品（如烟花爆竹、汽油桶、酒精）。雷暴、龙卷风、台风天气停止加油卸油作业——雷电感应可引燃油气，规范明确要求恶劣天气停业。

### 加油操作：确认熄火、不触碰油管、不敲油箱口

加油操作（含自助加油和工作人员加油）的规范：

加油前确认熄火、拉手刹、关车载电源。驾乘人员下车到安全区等候。不触碰加油枪油管（防油品溅出）。加油中不启动车、不开关车门、不用加油枪敲油箱口（防电火花和金属撞击火花）。禁止向塑料桶等非导静电容器直接加注汽油——塑料桶不导电，汽油流动产生静电积聚，瞬间放电引燃油气。需要散装汽油的，用金属防爆桶 + 登记身份购买。

加油结束待油枪自动跳枪关闭、归位、油箱盖拧紧，再启动驶离。跳枪后不要继续捏枪强制加油——油量过满会从油箱溢出，遇热膨胀也容易溢出，溢出的汽油是重大火灾隐患。

### 自助加油静电释放：触碰静电释放按键 2 秒

自助加油最大的隐性风险是静电。人体穿着化纤衣物摩擦会积聚静电（几千伏），触碰金属加油枪瞬间释放静电，可能引燃油气。国内已有多起自助加油静电引燃油气的事故。

新版安全规范要求：自助加油前，单手触碰静电释放按键（加油机上的金属圆点或挂式释放球）至少 2 秒，释放人体静电。释放后再取加油枪，全程不要回到车内（回到车内衣物摩擦再次积聚静电）。加油枪插入油箱口后再开始加油，不要在油箱口外开枪试油。

加油过程中不要梳理头发、不要脱穿化纤外套（摩擦再次产生静电）。加油结束先触摸车身金属部分（如车门把手）释放静电，再取下加油枪归位。

### 加油口位置识别：油表箭头指向

新手第一次加油常找不到加油口在左还是在右。看仪表盘油表图标——图标上有一个小三角箭头，箭头指向哪边，加油口就在车身哪边。这个细节新手不知道，开到加油站停错边，加油枪够不着还要重新挪车。

加油口位置因车而异：多数德系车（大众、宝马、奔驰）加油口在右侧；多数日系车（丰田、本田、日产）加油口在左侧；国产车两边都有。油表箭头是最准确的指引。

### 新能源 / 混动车加油补充

插电混动 / 增程车虽然纯电驶入加油站，加油前必须确认：车机关闭动力电池对外供电（部分车型需要切换到「加油模式」或「保养模式」）、不挂挡待驶、关闭所有车载电子设备。

行车记录仪停车监控、车载净化器、USB 快充头等熄火后仍通电的非防爆电子设备，加油前要断电——这些设备在加油作业区内同样有电火花风险。新能源车电池包进水的涉水风险见泥泞涉水篇，本篇只讲加油规范。

### 服务区规范：分区停放 + 限时休息 + 离车锁门

服务区是长途驾驶的强制休息点，规范如下：

分区停放：服务区按车型分区——大车区（货车、客车）、小车区（轿车、运动型多用途汽车）、危化品区（危险品运输车）。按车型停到对应区域，不要乱停。危化品区不要靠近（爆炸 / 泄漏风险），小车停大车区会被大车遮挡视线、剐蹭风险高。

进出场减速：服务区入口和出口限速通常 40km/h，进场减速（避免追尾排队车），出场加速到主线车速再汇入（避免低速汇入被后车追尾）。汇入主线前观察后方来车，打转向灯，确认安全距离再汇入。

疲劳必休息：连续驾驶不超 4 小时，每 2 小时强制休息 20 分钟。午后 13-15 时是疲劳高发时段，夜间行车每 1-2 小时进服务区休息。疲劳时进服务区小睡 15-30 分钟比硬撑安全得多——疲劳驾驶反应迟钝、判断失误，是高速事故主要诱因。

夜间停车：夜间在服务区停车休息，开示廓灯（让其他车看到你的车）或停到指定区域（有照明和监控）。不要长时间占用大车位（货车司机需要休息，长时间占用影响他们）。

离车锁门贵重物品随身：服务区人员复杂，离车必锁门，贵重物品（钱包、手机、笔记本电脑）随身携带，不要留在车内显眼处。长时间停车规范：服务区不是停车场，长时间占用（如过夜）要看服务区规定，部分服务区禁止过夜停车。

## 四、避坑：加油站和服务区 5 个高频错误

第一个错：加油不熄火。发动机运转产生电火花和排气管高温，可能引燃油气。停车后立即熄火、关电源、拉手刹，再加油。

第二个错：加油区用手机。手机射频电磁辐射在油气环境中可能引燃油气，加油作业区禁手机。需要手机支付的到便利店或收银区，或驶离加油机后使用。

第三个错：自助加油不释放静电。人体化纤衣物积聚静电，触碰加油枪瞬间放电引燃油气。自助加油前单手触碰静电释放按键 2 秒，全程不要回车内。

第四个错：服务区疲劳硬撑不进。疲劳驾驶是高速事故主要诱因，每 2 小时强制休息 20 分钟。疲劳时进服务区小睡 15-30 分钟，比硬撑出事故强百倍。

第五个错：服务区乱停占位。不按车型分区停放、长时间占用大车位、危化品区靠近，都可能引发事故或被处罚。按车型停到对应区域，不长时间占用，远离危化品区。

## 五、践行：加油站和服务区操作清单

下次进加油站或服务区，按这几步走：

加油站：进站限速 5km/h，按指引停放车头朝外。停车后立即熄火、拉手刹、关电源、下车到安全区。加油作业区禁烟禁明火禁手机禁维修。自助加油前触碰静电释放按键 2 秒，全程不回车内。加油结束跳枪后不要强制加油，拧紧油箱盖、油枪归位、确认无渗漏再启动驶离。

加油口位置：看仪表盘油表箭头，箭头指向哪边加油口就在哪边。

新能源 / 混动：加油前关闭动力电池对外供电，关闭停车监控等非防爆电子设备。

服务区：按车型分区停放（大车区 / 小车区 / 危化品区），进出场限速 40km/h，疲劳必休息（每 2 小时 20 分钟，午后 13-15 时高发），夜间停车开示廓灯，离车锁门贵重物品随身。

加油站和服务区的核心是「消除一切点火源」和「疲劳强制休息」。把这两条练到形成肌肉记忆，加油和长途休息都不再出事。
