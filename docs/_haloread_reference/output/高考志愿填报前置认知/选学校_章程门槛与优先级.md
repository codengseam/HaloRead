# 选学校：招生章程门槛与优先级法则

招生章程是高校依据《教育法》《高等教育法》和教育部规定制订、经主管部门审核备案后向社会公开发布的招生信息主要载体，**不得擅自更改、必须严格执行**。翻译成人话就是：这是法律性文件，写进章程的每一句话都是硬约束。这一章讲两件事——**怎么把章程读出退档风险、怎么按分数段排优先级**。

## 一、招生章程必读：法律性文件不是摆设

很多家长觉得招生章程是"流程性文件"，扫一眼就过。结果每年都有考生分数高出某校录取线几十分，却被退档——原因就藏在章程里那几行小字：单科差一分、语种不符、体检不合格、选科不匹配。

山东省考试院的提醒很直接：部分高校在编报招生来源计划时，对投档限制条件、退档条件、附加信息做了备注，省考试院公布招生计划时虽一并公布，但仍可能无法涵盖院校招生要求的全部信息，**务必查阅高校招生章程原文**。

所以候选院校清单出来后，下一步不是看学校排名，而是逐字研读每一所院校2026年招生章程的"录取规则""专业特殊要求""身体健康状况要求"三部分。这一步省不掉，省掉就要承担退档代价。

## 二、五类门槛排查清单

把章程读出风险，按下面五类逐条筛查：

### 1. 单科成绩要求

外语类专业常要英语单科，财经类常要数学，汉语言文学常要语文。具体阈值各校不同：

- 上海大学英语专业：英语单科不低于115分；
- 宁波大学阳明创新班：数学、外语不低于120分；
- 贵阳学院：英语专业英语≥105分，汉语言文学语文≥105分，数学与应用数学数学≥90分。

风险点：哪怕总分高出几十分，只要单科差一分，档案依然会被退回。所以查章程时单科要求要单列一栏，逐条核对。

### 2. 语种限制

大多数学校主流专业不限外语语种，但有几类专业多只招英语考生：

- 英语专业本身；
- 中外合作办学项目；
- 国际贸易、外交学等涉外专业；
- 部分理工科专业的双语教学班。

典型案例：吉林大学英语、俄语、日语、朝鲜语、西班牙语专业，**只录外语语种为英语的考生**——也就是说高考选日语、俄语的考生报不了这些专业。还有部分院校要求考生参加英语口试且成绩合格，比如北京外国语大学、贵阳学院英语专业要求口试≥80分。口试由各省教育考试院在高考后统一组织，漏报就等于失去报考资格。

高考选考日语、俄语等小语种的考生，必须避开仅招英语考生的专业，否则直接退档。

### 3. 视力/色觉/身高/体检

依据《普通高等学校招生体检工作指导意见》及各校招生专业体检标准：

| 体检项 | 受限专业典型 | 案例 |
|---|---|---|
| 色盲色弱 | 化学、药学、生物学、医学、美术、地质、采矿 | 西安交大：色弱不予录取医学/药学/护理/化学类，色盲不予录取物理类；贵阳学院：色弱不录生物工程、应用心理学、食品科学与工程、茶学、学前教育、体育教育 |
| 视力 | 飞行技术、航海技术、消防工程、刑事科学技术、空军招飞 | 武汉理工航海技术/轮机工程要求裸视力达标 |
| 身高 | 军校、警校、空乘、旅游管理 | 武汉理工航海技术/轮机工程要求身高≥1.65m |
| 嗅觉 | 食品科学与工程 | "嗅觉迟钝者不予录取" |
| 听力 | 师范类、外语类、音乐类 | — |

**实操建议**：考生手上的体检表就是一份"专业禁入清单"，填报前务必逐条对照目标院校招生章程中的身体条件要求彻底筛查。

### 4. 选科要求

2026年新高考选科门槛全面收紧：

- **理工、农业、医学类超过92%的本科专业**强制要求考生同时选考"物理"和"化学"两门科目；
- **法学、公安类**多要求"历史+政治"；
- 重点师范院校文科专业对选科也有细化要求。

选科不符，即使总分达标也会直接退档，专业报考范围将大幅受限。筛选志愿时必须逐条核对专业选科要求，确保组合完全匹配，容不得任何侥幸心理。

### 5. 特殊类型：政审/面试/体测

军校、武警院校、公安本科、司法警官本科、消防救援院校、公费师范生、免费医学生——均在本科提前批录取，需政审、体检、体测、面试。

这类院校还有两个特殊点：

- **投档模式多为顺序志愿**（梯度志愿），"志愿优先、分数为辅"，第一志愿权重极高，一志愿没录上二志愿录取概率极低；
- **提前批录取后无法退档、不能改志愿**，后续所有批次志愿全部失效。

不想从事定向基层工作、不愿参军的考生，提前批一个都不要随便填，否则录取后无法反悔。

## 三、不符合条件直接剔除

门槛排查的标准流程：

1. **逐字研读招生章程**：重点核对本省《招生计划》与院校《招生章程》中的"录取规则""专业特殊要求""身体健康状况要求"三部分；
2. **建立"否决清单"**：将不符合单科、语种、体检、选科任一条件的院校/专业直接剔除，不纳入候选；
3. **关注"备注栏"**：省考试院公布的招生计划备注栏可能还有性别、文化课总分、民族、合作单位等附加限制；
4. **电话复核**：对章程表述模糊的条款，拨打高校招生组电话咨询并留存记录。

一句话：**有任何一条不符的，直接从候选库删除**，不要抱侥幸心理。每年都有考生想"应该差不多吧"赌一把，结果就是档案被退。

## 四、城市/学校/专业优先级法则

不同分数段，优先级排法完全不同：

### 高分段（前10%）：学校 > 专业

- **名校光环**：985/211带来的保研率、选调资格、校招资源，远胜普通本科；
- **深造通道宽**：保研率高，考研复试时名校出身有加分；
- **选调资格**：很多省份的定向选调只面向双一流院校；
- **策略**：冲名校王牌专业，全部勾选服从调剂防止高分退档。

仅电气、临床、计算机等强壁垒专业，可以放弃偏远名校选择行业强校。

### 中分段（中间50%）：城市 + 专业 + 学校平衡

- **城市逻辑**：一线/新一线的实习、校招、产业资源碾压小城市；
- **专业逻辑**：优先可考证、就业率稳定的实操类专业，纯理论文史类慎选；
- **学校逻辑**：省内头部公办本科 + 行业特色二本（电力、铁道、医药、师范）国企定向校招资源远超普通综合类二本。

这一段最忌讳追校名好听，应用型专业才是性价比之选。

### 中低分段：专业 > 学校

- **就业导向**：工科、护理、计算机应用、学前教育等就业对口性强的方向；
- **能上本科不选专科**：为未来考研、考公保留基础学历优势；
- **保底院校**：必须覆盖至本科线下10分区间，确保有学可上。

这一段家长最容易纠结"是去偏远二本还是城市专科"，答案是优先本科+实用专业，城市其次。

## 五、章程查询入口

两个权威渠道必须用：

1. **教育部"阳光高考"信息平台**：https://gaokao.chsi.com.cn ——全国高校招生章程统一发布入口，可按院校名称检索；
2. **目标院校官网**：进入"招生就业"或"本科招生"栏目，下载当年招生章程PDF原文，对照前面五类门槛逐条筛查。

省考试院官网公布的本省《招生计划》大绿本/大蓝本也要核对一次——某些特殊条款只在备注栏出现。章程原文 + 招生计划备注栏 + 体检表，三份材料交叉核对完，才算把门槛排查做扎实。

章程门槛排查完，下一章讲怎么用一分一段表和录取数据精准定位分数线。
