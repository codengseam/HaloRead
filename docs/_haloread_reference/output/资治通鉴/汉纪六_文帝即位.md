# 资治通鉴·汉纪六·文帝即位

## 讲事情

文帝即位，是功臣集团选出来的皇帝，也是汉朝从乱到治的转折。

诸吕被诛后，长安城里，功臣们要选一个新皇帝。少帝刘弘是吕后立的，皇后是吕禄的女儿，功臣们不放心。功臣们商量：少帝和惠帝的几个儿子，都不是惠帝亲生，不该继承皇位。这话真假难辨，可功臣们要的是换人，名分上的理由，找一个就行。

选谁？这是功臣集团的头等大事。选错了，诸吕之乱白忙一场；选对了，汉朝才能安稳。

有人提议齐王刘襄。刘襄是齐悼惠王刘肥的长子，高祖的长孙。诸吕之乱，齐王首举义兵，西攻吕国，发檄讨吕。论血脉，论功劳，齐王都够格。可大臣们反对。大臣们说：齐王的舅舅驷钧，是个恶人，跟吕氏一个路数。立齐王，等于再来一个吕氏。诸吕之乱的教训，就是外戚专权。立齐王，驷钧必专权，这不行。

又有人提淮南王刘长。刘长是高祖的儿子，可刘长年少，母亲的家族也不善。大臣们也否了。

最后，大臣们选中代王刘恒。刘恒是高祖的第四个儿子，母亲薄姬。薄姬原是魏王豹的妾，魏王豹败后，薄姬被收入汉宫。薄姬入宫一年多，没见过刘邦。后来刘邦听管夫人、赵子儿说起薄姬，才召幸薄姬一次。薄姬说：昨夜梦见一条龙盘在我胸上。刘邦说：这是贵兆，我来成全你。这一夜，薄姬有了身孕，生下刘恒。此后薄姬再没见过刘邦。

薄姬不受宠，反成了她的福气。吕后当政，戚夫人被弄成人彘，诸姬被囚；唯独薄姬，因为不受宠，吕后放她出宫，跟儿子刘恒到代地就藩。代地在北边，紧贴匈奴，苦寒之地。薄姬跟刘恒在代地十几年，小心谨慎，不惹事，不招摇。刘恒在代地，仁孝闻天下。

大臣们选中刘恒，看中的就是三条：高祖血脉，母家薄氏谨良，本人仁孝。功臣们要的，是一个不会报复功臣、不会重用外戚的皇帝。

使者到代国，迎刘恒入京。刘恒接到消息，又惊又疑。诸吕之乱刚平，长安城里刀光剑影，这时候迎他入京，是福是祸？

刘恒问左右。郎中令张武主张怀疑：诸大臣都是高祖时大将，习兵，多谋诈。他们迎大王，不可信。大王称病勿往，观望形势。中尉宋昌主张去：秦失政，诸侯豪杰并起，最后刘氏得天下，天下人心归刘。高祖封子弟为王，刘氏根深蒂固。吕后专权八年，周勃一入北军，全军左袒——人心在刘氏，不在吕氏。如今大臣迎大王，是顺应人心。大王不可疑，疑了反倒坏事。

刘恒听了宋昌，又问母亲薄太后。薄太后也拿不准，让刘恒占卜。卜得卦，卦象大吉。刘恒这才决定入京。

刘恒谨慎，不敢大意。他派舅舅薄昭先去长安，见太尉周勃，探虚实。周勃把诸吕之乱的经过、迎立代王的缘由，一一告诉薄昭。薄昭回报刘恒：大臣是真心迎立，没问题。刘恒这才放心，带宋昌、张武等六人，启程入京。

走到长安城外，功臣们来迎。太尉周勃跪在地上，想跟刘恒说几句私话。宋昌挡住：所言公事，公言之；所言私事，王者无私。——公事当众说，私事王者没有。——《史记·孝文本纪》。周勃只好当众献上天子玺符。刘恒辞让，不肯受，说：到代邸再议。

到了代邸，功臣们再拜，请刘恒即位。刘恒三让，然后受。这"三让"，是古礼，也是姿态——不是我要当皇帝，是你们请我当的。

刘恒入宫，当夜，拜宋昌为卫将军，镇抚南北军；拜张武为郎中令，掌宫殿。这两手，是把长安的兵权和宫城的警卫，从功臣手里接过来。功臣们诛诸吕，握着南北军；刘恒即位第一夜，就把南北军换了自己人。这是刘恒的谨慎——功臣能立他，也能废他；兵权得握在自己手里。

刘恒即位，是为汉文帝。文帝即位，改元。这一年，是公元前180年。

文帝即位后，干了几件大事。

第一件，废连坐。文帝元年十二月，文帝下诏：法者，治之正也。今犯法者已论，而使无罪之父母妻子同产坐之，朕甚不取。其除收孥相坐律。——大意据《史记·孝文本纪》。法是治国的正道，一个人犯法，让无罪的父母妻子连坐，朕不赞成。废除收孥连坐之法。这是文帝的第一德政。

第二件，赐民爵。文帝赐天下民爵一级，女子百户牛酒，大酺五日。这是与民同乐，也是收人心。

第三件，减刑罚。文帝废肉刑，减笞刑，开法制史新篇。

第四件，减赋税。文帝减田租之半，三十税一；减徭役，男子服役由一年一次改为三年一次。这是与民休息。

第五件，以德化民。文帝不好大喜功，不兴兵戈，不修宫室。匈奴入寇，文帝忍；南越王赵佗称帝，文帝派人修赵佗在真定的祖坟，给赵佗的兄弟封官，赵佗感动，去帝号称臣。文帝以德服人，不以力服人。

文帝在位二十三年，公元前157年崩。文帝的儿子景帝即位，继续文帝的政策，史称"文景之治"。文景之治，是中国历史上第一个治世。文帝开其端，景帝继其后，到武帝时，汉朝国力达到顶峰。这一切，都从文帝即位那一年开始。

## 讲人物

**汉文帝刘恒：谨慎成全的仁君**

刘恒这个人，最大的特点是谨慎。诸吕之乱后，功臣迎他入京，他不敢直接去，先派舅舅薄昭探路；到长安城外，周勃要私语，宋昌挡住，他默许；入宫第一夜，先换南北军和郎中令。这一连串，都是谨慎。

刘恒的谨慎，是被环境逼出来的。刘恒八岁封代王，在代地十几年。代地苦寒，紧贴匈奴，又远离长安。吕后当政，诸王动辄得咎——赵王刘如意被毒杀，赵王刘友被饿死，赵王刘恢被逼自杀。刘恒在代地，靠的就是小心谨慎，不惹事，才活下来。这种环境养成的谨慎，到他当皇帝也没丢。

可刘恒的谨慎，不是胆小。该决断时，他能决断。入宫第一夜换南北军，是大决断——功臣刚立他，他就动功臣手里的兵权，这要胆量。可刘恒做得不动声色，功臣们也没话说。刘恒的谨慎，是谋定后动；刘恒的决断，是看准了就做。这两样合起来，是刘恒的本事。

刘恒的仁，也是真的。废连坐，减刑罚，减赋税，都是为百姓。文帝自己穿粗糙的黑帛，不穿锦绣；宠幸的慎夫人，衣服不拖地，帷帐不绣花。文帝想修一座露台，召工匠一算，要百金。文帝说：百金，中人十家之产也。我奉先帝宫室，常恐羞之，何以台为！——百金是十户中等人家的家产，我住先帝的宫室都怕奢侈，还修什么台！——《史记·孝文本纪》。文帝罢修露台。文帝的俭，是真俭；文帝的仁，是真仁。

文帝临终遗诏：朕闻之，盖天下万物之萌生，靡不有死。死者天地之理，物之自然，奚可甚哀！——大意据《史记·孝文本纪》。万物有生有死，死是天理，不必过分哀伤。文帝遗诏，简葬，不发民，不久丧。文帝连死都看得开，这是文帝的达观。

**宋昌：一语定乾坤的郎中**

宋昌是代王的郎中，官不大，可在文帝即位这事上，宋昌一句话定了乾坤。张武主张怀疑，不去；宋昌主张去，说了一大篇道理——秦失政，刘氏得人心；高祖封同姓，根深蒂固；吕后专权，周勃一呼全军左袒，人心在刘。大臣迎大王，是顺应人心，不可疑。

宋昌的话，句句在点子上。宋昌看的是大势——人心在刘氏，这是诸吕之乱证明了的。刘恒要是疑了不去，反倒给功臣们换人的借口。宋昌的明白，是看大势的明白。

宋昌还有一处硬气。到长安城外，周勃要跟刘恒私语，宋昌挡住：所言公事，公言之；所言私事，王者无私。这一挡，是给刘恒立威——皇帝没有私事，一切当众说。宋昌这一挡，让功臣们知道，新皇帝不是好拿捏的。宋昌的硬气，是臣子的硬气——保护主上的威严。

**张武：谨慎过头的郎中令**

张武是代王的郎中令，主张怀疑，不去长安。张武的理由是：诸大臣都是高祖时大将，习兵，多谋诈，不可信。

张武的话，不是没道理。功臣们刚诛诸吕，长安城里刀光剑影，谁知道迎代王是真是假？张武的谨慎，是为刘恒的安全着想。可张武的谨慎，过了头。功臣们要立皇帝，是真心的——诸吕之乱，功臣们冒了身家性命，立个新皇帝安稳天下，是他们的利益所在。张武只看到功臣的"多谋诈"，没看到功臣的利益所在。

刘恒听了宋昌，没听张武，是刘恒的判断。可刘恒也没亏待张武——入宫第一夜，拜张武为郎中令，掌宫殿警卫。刘恒用张武的谨慎，守宫城；用宋昌的明白，镇南北军。刘恒用人，是各取所长。

**薄太后：不争之德成就了文帝**

薄太后是文帝的母亲，魏王豹的妾出身。薄姬不受刘邦宠，反成了她的福气——吕后放她出宫，跟儿子到代地。薄姬在代地，谨小慎微，不惹事，把刘恒养大。

薄太后的好处，是不争。吕后争，弄得诸吕被诛；薄太后不争，反成太后。薄太后不争，是性格，也是智慧。薄太后在代地十几年，没有外戚干政，没有家族跋扈。功臣们选刘恒，看中的就是薄氏谨良。薄太后的不争，成全了刘恒的帝位，也成全了文景之治。

薄太后的不争，还体现在她不干预朝政。文帝即位，薄太后在后宫，不干政。薄太后的弟弟薄昭，文帝封为轵侯，没有大权。后来薄昭杀人，文帝不忍杀舅舅， 让大臣穿丧服去薄昭家哭，薄昭自杀。薄太后没闹。薄太后的不争，是真不争。

## 讲背景

文帝即位这事，得放在诸吕之乱后的政治格局里看。

先说功臣集团为什么要选新帝。诸吕之乱，功臣们诛了吕氏，可少帝还在。少帝是吕后立的，皇后是吕禄的女儿。功臣们怕少帝长大后报复，也怕吕氏余党借少帝之名作乱。功臣们要换皇帝，是为自保，也是为汉朝的安稳。功臣们选新帝，选的是自己的身家——立个温和的、不会报复的皇帝，功臣集团才能安稳。

再说为什么选代王。功臣们选新帝，有三条标准：高祖血脉，母家不恶，本人仁孝。齐王刘襄母家驷钧恶，淮南王刘长年少母家也恶，都不行。代王刘恒是高祖现存最年长的儿子，母亲薄氏谨良，本人仁孝闻天下。功臣们选刘恒，是选一个外戚不会专权的皇帝。诸吕之乱的教训，就是外戚专权。功臣们要的，是一个没有强势外戚的皇帝。薄氏谨良，是刘恒被选中的关键。

再说代地的位置。代地在今山西北部、河北一带，都晋阳（后徙中都）。代地紧贴匈奴，是汉朝的北边。代地苦寒，远离长安，不是富庶之地。刘邦把刘恒封到代地，一是因为代地要人守（挡匈奴），二是因为刘恒不受宠（给个苦地方）。可代地这个位置，反成全了刘恒——远离长安，躲过吕后的清洗；苦寒之地，养成刘恒的谨慎俭朴。代地是刘恒的"龙潜之地"，也是刘恒的修行场。

再说文帝的施政为什么是"与民休息"。汉初，天下刚定，民疲于兵。秦亡的教训，是秦法严酷，民不聊生。汉初的政策，是吸取秦亡教训，与民休息。刘邦的约法三章，萧何的镇关中，曹参的萧规曹随，都是这条路线。文帝继之，废连坐，减刑罚，减赋税，以德化民。文帝的政策，是汉初与民休息路线的延续，也是文景之治的开端。

再说废连坐的意义。连坐是秦法的核心——一人犯法，父母妻子同产连坐。连坐的目的是威慑，可连坐也制造冤狱——无罪的人因亲人犯罪受罚。文帝废连坐，是改秦法，也是行仁政。文帝废连坐，开了中国法制史上的新篇章——刑罚只及犯罪者本人，不株连无辜。这是文帝的第一德政，也是文景之治的法治基础。

再说匈奴与南越的问题。文帝时，匈奴仍强，屡屡南下。文帝忍，不开战，守和亲。南越王赵佗称帝，文帝不兴兵，而是修赵佗在真定的祖坟，给赵佗兄弟封官，赵佗感动去帝号。文帝以德服人，不以力服人。文帝的忍，是国力不够的忍，也是仁君的忍。文帝不开边，不穷兵，把国力留给民生。这是文帝的明白——打不起就不打，把力气用在百姓身上。

## 讲道理

司马光在《资治通鉴》里对文帝有高度评价。大意据臣光曰，文帝除收孥相坐之法，减刑罚，赐民爵，与民休息，仁矣。文帝以德化民，不以力服人，海内晏然。文帝之治，可谓仁政。司马光治史重民本，文帝的与民休息，正合司马光的理想。司马光看文帝，看的是"仁"——文帝的仁，是真仁，不是假仁。

贾谊在《治安策》里论文帝时的形势。大意据贾谊说，当今之势，有可痛哭者一，可流涕者二，可长太息者六。天下初定，诸侯王强，匈奴骄，制度未立。贾谊看文帝时的天下，看的是隐患——诸侯王强（后来七国之乱应验），匈奴骄（后来武帝开战应验），制度未立。贾谊的《治安策》，是给文帝的警钟。文帝听了贾谊的话，部分采纳——众建诸侯（分封诸侯王子弟，削弱诸侯），可没全做。贾谊的忧虑，到景帝时七国之乱，才被证实。贾谊看形势，比文帝看得远。

王夫之在《读通鉴论》里论文帝。大意据王夫之说，文帝之治，在于能忍。匈奴入寇，文帝忍；诸侯王骄，文帝忍；赵佗称帝，文帝忍。文帝之忍，非怯也，乃养民力、待时也。文帝不争一时之功，而求万世之安。王夫之看文帝，看的是"忍"——文帝的忍，是养民力，是待时，不是怯。王夫之也指出，文帝的忍，也有弊端——诸侯王越忍越骄，终于景帝时爆发。文帝的忍，是双刃剑。

## 问道悟道

文帝即位这出戏，藏着两条让人开眼的规律。

**不争，反成大器。**

薄姬不受宠，吕后放她出宫；刘恒在代地，不惹事，不招摇。可最后，皇位落到刘恒头上。受宠的戚夫人，被弄成人彘；受宠的赵王刘如意，被毒杀；受宠的淮南王刘长，后来谋反被废。不争的刘恒，反成文帝。

不争反成大器这事，历史上不止一桩。春秋时，楚庄王即位三年，不理朝政，日夜作乐，下令"敢谏者死"。伍举进谏，说有鸟止南方之阜，三年不翅不飞不鸣，是什么鸟？楚庄王答：三年不翅，将以长羽翼；不飞不鸣，将以观民则。虽不飞，飞将冲天；虽不鸣，鸣将惊人。——大意据《韩非子·喻老》。半年后，楚庄王亲政，罢淫乐，听政，所诛者数百，所进者数百，一鸣惊人，终成霸主。楚庄王的"三年不飞"，跟刘恒的"代地十几年"，是一路的——都是不争，都是养，都是待时而动。

**立威不在杀，在让。**

文帝即位，没杀功臣，没立威于杀戮。文帝立威，靠的是"让"——三让然后受，让功臣们知道皇帝不是抢来的；入宫第一夜换南北军，让功臣们知道兵权不在他们手里。文帝的让，是姿态；文帝的换，是实质。姿态做足，实质握紧，这才是立威。

立威不在杀这事，历史上也有例子。周武王灭商，不杀商的贤人，反而封比干之墓，表商容之闾，释箕子之囚。——大意据《史记·殷本纪》。武王立威，靠的是敬贤，不是杀人。商朝的遗民，见武王敬贤，心服。文帝立威，跟武王一路——都是不杀而服人。杀，是下策；让，是上策。文帝的让，让出了文景之治。

## 结语

一个不受宠的代王，靠谨慎和仁厚，开出了汉朝第一个治世。
