# 资治通鉴·汉纪三十六·班超通西域

## 讲事情

班超通西域，是一个书生扔了笔、带着三十六个人，在西域大漠里搅了三十一年风云的故事。这三十一年，让西域五十余国重新归汉，也让"投笔从戎"四个字成了后世读书人的口头禅。

班超是扶风平陵人，班彪的少子，班固的弟弟，班昭的哥哥。班家是史学世家，父亲班彪写《史记后传》，哥哥班固写《汉书》，妹妹班昭后来续成《汉书》。班超从小在书堆里长大，可班超不爱书。

班超年轻时，家里穷。哥哥班固被召入京校书，班超跟母亲到洛阳，没钱养家，只好替官府抄书，挣点钱糊口。抄书是个苦差事，一天到晚伏案，手不停，眼不歇。有一天，班超抄着抄着，突然把笔一扔，站起来叹气："大丈夫无他志略，犹当效傅介子、张骞立功异域，以取封侯，安能久事笔砚间乎！"——大丈夫就算没别的本事，也该学傅介子、张骞到异域立功，博个封侯，怎能一辈子埋在笔砚里！旁边的人都笑他，班超不理。

公元73年，机会来了。汉明帝派奉车都尉窦固击北匈奴，班超从军，做假司马。班超打仗有胆有谋，窦固赏识，派班超和郭恂出使西域。

班超带着三十六个人，先到鄯善。鄯善王广一开始对班超礼数周到，嘘寒问暖。可过了几天，鄯善王忽然冷淡了，爱答不理。班超一琢磨，断定是北匈奴的使者到了，鄯善王心里有了别的主意。

班超叫来鄯善王派来侍候的人，诈他："匈奴使者来了几天了，现在住哪？"侍胡吓慌了，以为班超知道，一五一十招了。班超把侍胡关起来，召集三十六人喝酒。酒过三巡，班超站起来，说："诸位跟我一起在绝域，想立大功求富贵。现在匈奴使者才来几天，鄯善王礼数就废了。要是鄯善王把我们抓了送给匈奴，我们的骨头就喂了豺狼。怎么办？"

众人都说："现在在危亡之地，死生听司马的。"

班超说了一句千古名言："不入虎穴，不得虎子。"——不进虎穴，抓不到虎崽。班超说：现在的办法，只有趁夜用火攻匈奴使者。他们不知道我们多少人，一定惊乱，我们一举全歼。灭了匈奴使者，鄯善王就吓破了胆，大事可成。

当夜，班超率三十六人出发。班超让十人带着鼓，藏在匈奴使者营后，约好见火起就击鼓呐喊。其余的人持刀弩，埋伏在营门两侧。班超顺风放火，前后鼓噪，匈奴使者营大乱。班超亲手杀三人，部下杀三十余人，其余百余人烧死在营里。

第二天，班超把匈奴使者的首级给鄯善王看。鄯善王吓得叩头，说愿归汉，把儿子送到洛阳做人质。班超回去报告窦固，窦固上报明帝，明帝壮班超之节，升班超为军司马，让他继续经营西域。

班超继续西行。到于阗，于阗王广德刚被匈奴封了，对班超冷淡。于阗信巫，巫师对广德说：神怒了，要班超的马。班超痛快答应，让巫师自己来取。巫师来了，班超一刀砍了巫师的头，送还广德。广德吓坏了，又听说班超在鄯善的事，立刻杀匈奴使者降汉。

到疏勒，疏勒王是龟兹人兜题，是匈奴立的。班超派田虑去见兜题，兜题傲慢。田虑趁兜题不备，把他绑了，班超另立疏勒故王兄子忠为王，疏勒归汉。

班超在西域，靠的就是这股子胆气。可班超不只是莽，还有谋。北匈奴屡次攻西域，班超以少敌多，坚守疏勒。公元78年，班超封定远侯。

公元90年，大月氏（贵霜帝国）派七万兵攻班超。班超兵少，部下慌了。班超说：月氏兵虽多，千里远征，粮草接济不上。我们收好粮食坚守，他们饿几天就退了。果然，大月氏攻不下，粮尽，求和。班超受其降，大月氏岁贡不断。

公元94年，班超发龟兹、鄯善等八国兵七万人，讨焉耆。焉耆王广诈降，班超将计就计，斩焉耆王，传首京师。至此，西域五十余国皆归汉。

公元97年，班超派副使甘英出使大秦（罗马帝国）。甘英走到条支（今波斯湾），临大海欲渡。安息（帕提亚）的船夫吓唬甘英：海上有大风暴，顺风三个月能过，逆风两年也回不来，船上的人思乡落泪的多了。甘英没敢渡海，返回。甘英是第一个走到波斯湾的中国使者，可他没渡过那片海。

班超在西域三十一年，老了，想家。公元100年，班超上书和帝："臣不敢望到酒泉郡，但愿生入玉门关。"——臣不敢奢望到酒泉，只求活着进玉门关。班超的妹妹班昭也上书代兄求归，言辞恳切。和帝动容，公元102年召班超回洛阳。班超回到洛阳，拜射声校尉，同年八月卒，年七十一。

班超走的时候，西域诸国哭着送。班超的儿子班勇后来也经营西域，可再没班超的气象。

## 讲人物

**班超：扔了笔的书生将军**

班超是史学世家的异类。父亲班彪、哥哥班固、妹妹班昭，都是写史的人。班超偏不写史，要去打仗。班超扔笔时，旁边的人都笑他。可班超这一扔，扔出一个定远侯。

班超的胆，是西域那盘棋的本钱。三十六人夜袭匈奴使者营，是班超的成名战。班超的胆不是莽，是算准了——匈奴使者不知道汉军多少人，趁夜火攻，必乱。班超算准了鄯善王的心——杀了匈奴使者，鄯善王没退路，只能降汉。班超这一手，是胆和谋的结合。

班超的谋，体现在以夷制夷。班超在西域，兵不多，可班超会借力。打焉耆，班超发龟兹、鄯善等八国兵七万人——这七万人不是汉朝的兵，是西域诸国的兵。班超用西域的兵打西域，是班超的高招。班超知道，汉朝离西域太远，运兵运粮都难，只有以夷制夷，才能长久。

班超的韧，是西域那盘棋的定力。班超在西域三十一年，从四十二岁到七十一岁。北匈奴攻，班超守；大月氏来，班超挡。班超不急，一步步来，先鄯善，再于阗，再疏勒，再龟兹，再焉耆，三十一年把西域五十余国一个个收过来。班超的韧，是"不入虎穴，不得虎子"的韧——进了虎穴，就不出来，直到抓到虎子。

班超的苦，是"生入玉门关"的苦。班超在西域三十一年，老了想家。班超上书说"臣不敢望到酒泉郡，但愿生入玉门关"，这话听着心酸。一个定远侯，在西域立了大功，临老只求活着进玉门关。班超的妹妹班昭代兄求归，说班超"发落齿摇"，"犬马之齿七十有一"。班超最后回到洛阳，几个月就死了。班超这一辈子，给了西域，给了汉朝，自己只换来几个月的故乡。

**班昭：续成《汉书》的才女**

班昭是班彪的女儿，班固、班超的妹妹。班昭嫁同郡曹世叔，人称曹大家。班昭是班家最有学问的一个——班固写《汉书》，八表和天文志没写完就死了，和帝诏班昭续成。

班昭续《汉书》，是史学史上的一件大事。《汉书》是第一部纪传体断代史，班固写了大部分，可八表和天文志最难，班固没写完。班昭接过来，把八表和天文志补全。班昭的学问，连大儒马融都服气——马融跟着班昭学《汉书》。

班昭还代兄班超上书求归。班昭的上书写得恳切，说班超"犬马之齿七十有一"，"发落齿摇"，"乞归死于本国"。和帝看了，才召班超回来。班昭这一笔，救了班超的晚年。

班昭还写了一篇《女诫》，讲女子之道。《女诫》后来争议很大，可班昭的学问，没人否认。班昭是中国第一个女历史学家，也是第一个有著作传世的女学者。

**窦固：给班超机会的人**

窦固是窦融的侄子，光武、明帝时的外戚名将。公元73年，明帝派窦固击北匈奴，窦固出酒泉，至天山，击呼衍王，追至蒲类海。窦固这次出击，打通了去西域的路。

窦固的功劳，不只是打仗，是识人。窦固看中班超，派班超出使西域。窦固这一识，识出一个定远侯。窦固给班超三十六人，班超带着这三十六人，搅了西域三十一年。窦固的识人之明，是班超这盘棋的开端。

## 讲背景

班超通西域这事，得放在东汉初年的匈奴形势和西域的地缘格局里看。

先说匈奴。西汉武帝击匈奴，匈奴北遁。可到了东汉初年，匈奴又分南北。南匈奴归汉，北匈奴据漠北，屡寇边郡。光武时，光武偃武修文，不主动击匈奴。可北匈奴寇边不止，河西诸郡受其害。到了明帝，国力恢复，明帝决定击北匈奴。公元73年，窦固出酒泉击北匈奴，打通了去西域的路。班超就是这次出击里冒出来的。

再说西域。西域就是今天新疆及中亚一带，汉武帝时张骞通西域，宣帝时设西域都护。王莽时，西域断绝，北匈奴趁机控制西域。东汉初年，西域诸国受北匈奴压迫，屡次求汉派都护，光武不许，说"天下初定，未遑外事"。光武不学汉武帝，不四处征伐。可北匈奴据西域，威胁河西。明帝时，窦固击北匈奴，班超通西域，东汉重新经营西域。

再说西域的地缘。西域有南北两道，南道经鄯善、于阗、莎车，北道经车师、焉耆、龟兹、疏勒。两道在葱岭（帕米尔）汇合，往西是大月氏、安息、大秦。西域诸国林立，大的几万人，小的几百人。北匈奴控制西域，靠的是挑拨诸国矛盾。班超通西域，靠的也是挑拨诸国矛盾——只不过班超挑拨得更高明，让诸国归汉。

再说班超为什么能成。班超能成，一是胆，二是谋，三是借力。班超的胆，是三十六人夜袭匈奴使者；班超的谋，是以夷制夷；班超的借力，是借西域诸国的兵打西域。班超在西域，兵不多，可班超会借势。班超借汉朝的威，借西域诸国的兵，借诸国之间的矛盾，把西域五十余国一个个收过来。班超这一套，是张骞没做到的——张骞通西域，是探路；班超通西域，是经营。

再说甘英为什么没渡海。班超派甘英出使大秦，甘英走到条支（波斯湾），临大海欲渡。安息的船夫吓唬甘英，说海上有大风暴，思乡落泪的多了。甘英没敢渡海。安息为什么吓唬甘英？因为安息垄断了东西方贸易，汉朝直接通大秦，安息就赚不到中间的差价。安息的船夫，是替安息的利益说话。甘英没渡海，是东汉失去直通罗马的机会。可甘英走到波斯湾，已经是中国人西行最远的一次。

再说班超的难。班超在西域三十一年，难在两头。一头是北匈奴，屡次攻西域；一头是朝廷，朝里有人觉得西域没用，想放弃。公元76年，朝议欲罢西域都护，召班超还。班超还没走，疏勒、于阗的人抱着班超的腿哭，不让走。班超只好留下。班超在西域，是两头难——外有北匈奴，内有朝议。班超能撑三十一年，靠的是定力。

## 讲道理

范晔在《后汉书·班超传》末有一段赞。范晔说班超"定远慷慨，专功西遐。坦步葱、雪，咫尺龙沙"。大意是班超封定远侯，慷慨立功于西疆，从容走过葱岭、雪山，逼近大漠。范晔这段，把班超的气概写出来了——一个人，三十六人起步，从容走遍西域，这是何等的气概。

王夫之在《读通鉴论》里论班超。大意据王夫之说，班超以三十六人定西域，是奇功，可班超靠的是西域诸国的矛盾，不是汉朝的实力。王夫之指出，班超一走，西域又乱，说明班超的经营没有根基。王夫之这话，是清醒的——班超是奇才，可奇才不常有；班超一走，西域失控，是东汉的隐忧。

司马光在《资治通鉴》中论班超。大意据《资治通鉴》，司马光认为班超"以三十六人定西域"，是"奇功"。司马光论史，重"礼"重"名"，班超通西域，让西域五十余国归汉，是扬国威于异域，正合司马光的胃口。可司马光也注意到，班超之后，西域屡得屡失，是东汉国力不足的体现。

## 问道悟道

班超通西域这出戏，藏着两条让人开眼的规律。

**不入虎穴，不得虎子。**

班超带着三十六人，面对匈奴使者和鄯善王的变心，没有退路。班超说"不入虎穴，不得虎子"，趁夜火攻，一举全歼匈奴使者。班超这一搏，搏出一个定远侯。

不入虎穴不得虎子这事，历史上有例子。战国时赵武灵王胡服骑射，群臣反对，赵武灵王力排众议，终成强国。西汉时张骞出使西域，被匈奴扣了十几年，逃出来继续西行，终通西域。三国时诸葛亮隆中对，刘备三顾茅庐，诸葛亮出山，三分天下。班超、赵武灵王、张骞、诸葛亮，都是进了虎穴抓虎子的人。机会在险处，不敢冒险，就抓不到机会。可"入虎穴"不是莽，是算准了再进——班超算准了匈奴使者的人数和反应，才敢火攻。

**奇才不常有，制度比人长久。**

班超在西域三十一年，西域五十余国归汉。可班超一走，西域又乱。班超的儿子班勇后来经营西域，也没班超的气象。班超的奇才，是个人能力，不是制度。班超靠胆、靠谋、靠借力，可这些本事，传不下去。

奇才不常有这事，历史上有例子。诸葛亮治蜀，蜀汉强；诸葛亮一死，姜维撑不住，蜀汉亡。王猛辅苻坚，前秦强；王猛一死，苻坚淝水大败，前秦亡。张居正改革，明朝中兴；张居正一死，改革废，明朝衰。班超、诸葛亮、王猛、张居正，都是奇才，可奇才一走，事业就散。靠人的，人走政息；靠制度的，才能长久。班超通西域是奇功，可东汉没有把西域经营制度化，班超一走，西域失控，是东汉的遗憾。

## 结语

三十六人夜袭匈奴营，三十一年纵横西域——班超扔了笔，捡起的是一条没人走过的路。
