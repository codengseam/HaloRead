# 资治通鉴·后梁纪五·梁晋争霸

## 讲事情

朱温一死，后梁便走到了下坡路。北边的沙陀李存勖，承李克用之业，骑步精锐，步步紧逼。后梁与晋争霸十二年，从柏乡之战到汴梁城破，梁军败多胜少，朱友贞任人不明、宠信外戚，把老将王彦章都害了。923年十月，李存勖入汴，后梁亡——十六年的短命王朝，就此收场。

朱友贞即位后，第一桩大事便是应付北边的晋。沙陀李克用于908年卒，其子李存勖继晋王位。李存勖是李克用亲子（详见《资治通鉴·后唐纪一·同光灭梁》论李存勖灭梁建唐），骁勇善战，承父业与后梁争霸。早在朱温在世时，梁晋便已交锋。开平四年，公元910年，朱温遣王景仁（原名王茂章）率梁军攻镇州王镕、定州王处直，二镇求救于晋。李存勖率周德威等救之。乾化元年，公元911年正月，两军会于柏乡（在赵州，今河北柏乡）。梁军七万，号"龙骧""神捷"精锐；晋军仅周德威所部骑兵，加上镇定之兵，远少于梁。大意据《资治通鉴·后梁纪二》《旧五代史·梁书·太祖纪》。可梁军粮尽，士卒饥寒，周德威以骑兵扰其营，梁军大溃。柏乡之战，梁军精锐尽丧，李存勖一战奠定河北霸权——后梁自此转攻为守，晋转守为攻。柏乡之败，是后梁由盛转衰的转折点。

朱温死后（912，详见《资治通鉴·后梁纪一·朱温代唐》论朱温被弑与朱友贞即位），后梁内乱虽平，可朱友贞任人不明。贞明元年，公元915年，魏博节度使杨师厚卒。杨师厚是后梁宿将，握魏博六州精兵，"银枪效节都"是其麾下亲军。朱友贞恐魏博势大难制，欲分魏博六州为天雄、昭德两镇。魏博军士不愿分迁，银枪效节都哗变，叛梁降晋。大意据《资治通鉴·后梁纪四》《旧五代史·梁书·末帝纪》。李存勖亲率军入魏州，收银枪效节都为己用——这支精骑，后来成了灭梁的先锋。魏博兵变降晋，是后梁失河北的关键——自此梁失黄河以北，退守黄河以南，战略上完全被动。魏博一失，后梁的北方屏障尽丧。

贞明四年，公元918年十二月，李存勖率晋军南下，欲一举灭梁。两军会于胡柳陂（在濮州，今河南濮阳东）。这一战，晋军初战大败——周德威，李克用、李存勖父子倚重的老将，沙陀骑兵的统帅，战死阵中。大意据《资治通鉴·后梁纪五》《旧五代史·唐书·周德威传》。李存勖痛失良将，可奋力反击，梁军亦大败。胡柳陂之战，晋胜梁亦败——晋军损失惨重，无力直取汴梁，只得退兵。这一战，是梁晋争霸中最惨烈的一战。周德威之死，是晋的重大损失；可梁军未能乘胜，反被李存勖反击击溃——可见后梁已无可用之将、可乘之机。胡柳陂之后，梁晋相持于黄河两岸，进入拉锯。

龙德三年（即同光元年），公元923年四月，李存勖在魏州称帝，国号唐（后唐），建元同光，是为后唐庄宗。大意据《资治通鉴·后唐纪一》《旧五代史·唐书·庄宗纪》（后唐建国主场详见《资治通鉴·后唐纪一·同光灭梁》）。李存勖称帝，是后梁灭亡的丧钟。这年闰四月，后唐军开始大举南下。九月，后唐军攻杨刘（在郓州，今山东东阿东北），渡过黄河。十月，后唐军逼近汴梁。

后梁此时，已是众叛亲离。名将王彦章（王铁枪）守中都（今山东汶上），兵少将寡，又无后援。王彦章是郓州寿张人，善用铁枪，军中号"王铁枪"，忠勇善战，是后梁最后的柱石。可朱友贞听信赵岩、张汉鼎等谗言，曾罢王彦章，用段凝为帅。段凝是朱友贞的姻亲，暗害王彦章，拥重兵在黄河北岸，观望不救。大意据《资治通鉴·后梁纪六》《旧五代史·梁书·王彦章传》。王彦章兵败被俘，李存勖爱其勇，欲降之。王彦章不屈，说"豹死留皮，人死留名"——豹子死了留下皮，人死了留下名。大意据《新五代史·王彦章传》。李存勖见其志不可夺，杀之。王彦章之死，是后梁最后的忠魂。

王彦章既没，段凝拥兵不救，汴梁城内已是惊弓之鸟。朱友贞众叛亲离，左右散尽。公元923年十月，李存勖军入汴梁。朱友贞自杀（一说为其所杀），后梁亡。大意据《资治通鉴·后梁纪六》《旧五代史·梁书·末帝纪》。后梁自907年朱温代唐，至923年朱友贞亡国，凡三世十六年。这十六年，是梁晋争霸的十六年——从柏乡到胡柳陂到汴梁城破，后梁一路败退，晋一路南下，霸权易手。沙陀李氏取代朱梁，成为中原新主。

## 讲人物

**朱友贞：任人不明而亡国的末代之君**

朱友贞是后梁末帝，在位十一年（912-923）。朱友贞之败，败在"任人不明、亲小远贤"。朱友贞宠信姐夫赵岩、表兄张汉鼎、姻亲段凝等外戚佞幸，疏远老将。王彦章（王铁枪）忠勇善战，是后梁最后的柱石，可朱友贞听信谗言，曾罢王彦章，用段凝为帅——段凝是朱友贞的姻亲，暗害王彦章，最后拥兵不救，致后梁速亡。大意据《旧五代史·梁书·王彦章传》。

朱友贞的悲剧，是"武夫治国的悖论"的典型。朱友贞是武夫之后，可他本身并非善战的武夫——他靠诛兄得位，靠的是禁军与外戚，不是沙场之功。朱友贞亲小远贤，用段凝不用王彦章，是因为段凝是他的姻亲，王彦章是耿介老将——姻亲听话，老将直言。可听话的庸才误国，直言的良将救国。朱友贞选了听话的庸才，弃了直言的良将——后梁之亡，亡在朱友贞的任人不明。可见武夫治国，最怕的不是不会打仗，是不会用人——用佞幸则国亡，用贤将则国存。

**李存勖：灭梁建唐的沙陀雄主**

李存勖是李克用亲子（详见《资治通鉴·后唐纪一·同光灭梁》论李存勖称帝与亡国），沙陀族，骁勇善战，承父业与后梁争霸。李克用与朱温是世仇（汴州之变朱温纵火欲杀李克用，详见《资治通鉴·唐纪十二·黄巢起义与朱温篡唐》），李克用临终以三矢赐李存勖，命其报三仇——一讨刘仁恭，二讨契丹，三灭朱温。大意据《新五代史·伶官传序》。李存勖藏矢于庙，每战前请矢以行。李存勖承父业，先平刘守光（刘仁恭之子），再破契丹，最后灭后梁——三矢之愿，一一兑现。

李存勖之强，强在沙陀骑兵。沙陀是西突厥别部，骑兵骁勇，是平黄巢、灭后梁的军事主力。柏乡之战、胡柳陂之战、入汴之役，沙陀骑兵所向披靡。李存勖本人亦善战，常亲冒矢石——这份骁勇，是李存勖灭梁的本钱。可李存勖灭梁后，却因宠伶人而亡国（详见《资治通鉴·后唐纪一·同光灭梁》论伶官乱政与庄宗亡国）。可见马上得天下者，未必能马上治天下——李存勖灭梁是雄主，治国却是昏君。

**王彦章：忠勇不屈的铁枪之将**

王彦章是郓州寿张人，善用铁枪，军中号"王铁枪"，是后梁最后的柱石。王彦章从朱温起，历事后梁三世，忠勇善战。可朱友贞任人不明，听信赵岩、张汉鼎等谗言，曾罢王彦章。后梁末年，晋军压境，朱友贞无奈，复用王彦章。王彦章受命于危难，率兵御晋，可兵少将寡，又无后援。大意据《旧五代史·梁书·王彦章传》。

王彦章兵败被俘，李存勖爱其勇，欲降之。王彦章不屈，说"豹死留皮，人死留名"——豹子死了留下皮，人死了留下名。又云"臣无面目见梁主"——臣没有面目见梁主。大意据《新五代史·王彦章传》。李存勖见其志不可夺，杀之。王彦章之死，是后梁最后的忠魂——后梁三世，朱温荒淫，友珪弑父，友贞任佞，唯有王彦章，是从头到尾的忠勇之将。王彦章"豹死留皮"的话，是武夫忠义的极致——身可死，名不可辱。可见忠臣良将，纵遇昏主，亦守其节；纵死沙场，亦全其名。王彦章与段凝的对比，正是后梁的缩影——忠将不用，佞将误国。

## 讲背景

梁晋争霸这事，得放在沙陀族崛起与草原骑兵南侵、藩镇格局演变、朱友贞任人不明与外戚乱政、以及契丹南侵的牵制四重背景里看。

先看沙陀族崛起与草原骑兵南侵。沙陀是西突厥别部，本居北庭（今新疆吉木萨尔），后内迁代北。沙陀骑兵骁勇，是唐末五代最精锐的军事力量。李克用率沙陀骑兵入援平黄巢（详见《资治通鉴·唐纪十二·黄巢起义与朱温篡唐》），自此沙陀成为中原争霸的一强。沙陀骑兵的特点，是骑射娴熟、机动迅捷、冲击力强——柏乡之战，周德威以少胜多，靠的就是沙陀骑兵的机动；胡柳陂之战，李存勖能反击取胜，亦靠沙陀骑兵的冲击。沙陀李氏灭后梁、建后唐，靠的就是这支骑兵。沙陀骑兵的崛起，是五代前半叶的主线——草原骑兵南下，与中原武夫争天下，是五代军事格局的核心。

再看藩镇格局演变。后梁是朱温以宣武节度使起家建国的，本身就是一个大藩镇。后梁的版图，远不及唐——北有沙陀李克用父子据河东，南有杨行密据淮南，西有李茂贞据凤翔，蜀有王建，岭南有刘隐。后梁实际控制的，只是中原汴洛一带。藩镇割据的格局，并未因朱温代唐而改变——朱温只是众多藩镇中最大的一个。柏乡之战后，后梁失河北；魏博兵变后，后梁失黄河以北；胡柳陂之后，后梁只能退守黄河以南。可见后梁之亡，是藩镇格局演变的必然——后梁本是大藩镇，被更强的藩镇（沙陀晋）取代。

再看朱友贞任人不明与外戚乱政。朱友贞是诛兄得位的，得位不正，故格外倚重外戚佞幸。赵岩是朱友贞姐夫，张汉鼎是朱友贞表兄，段凝是朱友贞姻亲——这些人围绕朱友贞，把持朝政，排挤老将。王彦章忠勇，却被谗罢免；段凝庸碌，却因姻亲而领兵。可见朱友贞之任人，不是看才能，是看亲疏——亲则用，疏则弃。这份"任人唯亲"，是后梁速亡的内因。外戚乱政，是五代的通病——后唐明宗时亦有此弊（详见《资治通鉴·后唐纪七·明宗之治》论明宗之治与外戚）。可见武夫之朝，本无礼法制约，外戚佞幸乘机用事，是王朝速亡的催化剂。

再看契丹南侵的牵制。梁晋争霸期间，北方契丹正在崛起。契丹首领耶律阿保机（辽太祖）于916年称帝建契丹国（后改辽）。契丹骑兵屡屡南侵，骚扰幽燕。李存勖灭梁前，曾分兵御契丹——刘守光据幽州，李存勖先灭刘守光，再御契丹，才能专心南下灭梁。大意据《辽史·太祖纪》。契丹的南侵，牵制了晋的兵力；可晋仍能灭梁，可见晋之强。契丹此时虽未深入中原，可已是北方大患——后唐末年石敬瑭借兵契丹灭后唐（详见《资治通鉴·后唐纪七·明宗之治》论石敬瑭借兵），契丹自此深入中原。可见契丹南侵，是五代后半叶的祸根。

## 讲道理

王夫之在《读通鉴论》卷二十八（五代上）里论梁晋争霸，话说得极痛。大意据王夫之说，朱温、李存勖、石敬瑭、刘知远、郭威之流，"窃据唐之京邑"，皆不足称天子——"初无君天下之志，天下亦无君之之心，燎原之火，旋起旋灰"。王夫之以为，五代不过是"战国"，不是"代"——藩镇武夫相搏相噬，毒民争地，全无立国之规模。大意据《读通鉴论》卷二十八。王夫之把梁晋争霸放在"藩镇相噬、燎原旋灰"的位置上——后梁与晋争了十二年，争的不是天下苍生，是彼此的志欲；百姓死于兵燹，田地化为战场。王夫之论朱友贞之亡，大意据王夫之说，朱友贞任人不明，用佞弃贤，是"含毒奋爪以相攫"的典型——后梁之亡，亡在自毁柱石。大意据《读通鉴论》卷二十八。

胡三省在《资治通鉴音注》后梁纪部分，对魏博兵变与王彦章之死的批注，多落在"藩镇倒戈、忠将见弃"上。大意据胡三省注，魏博银枪效节都本是后梁精锐，一旦叛降于晋，反成灭梁先锋——可见藩镇之兵，只认利害，不认主君；今日为你所用，明日为敌所用。胡三省注王彦章之死，是带着敬意的——王彦章"豹死留皮"，是武夫忠义的极致；可朱友贞不能用，反罢之，是自毁柱石。胡三省的潜台词是：后梁之亡，非晋之强，是梁之自毁——忠将不用，佞将误国，藩镇倒戈，三者交织，后梁焉得不亡。大意据《资治通鉴音注》后梁纪部分。

司马光在《资治通鉴·后梁纪五》里对梁晋争霸的记述，亦带其一贯史观。大意据司马光说，后梁之亡，亡于朱友贞任人不明、外戚乱政——朱温虽暴，尚能用人；朱友贞不暴，却不能用贤。可见王朝之亡，未必亡于暴君，多亡于庸主。司马光的史观，是"治乱在人主之德"——朱友贞德之不足（任人不明），必然亡国；后世之君当鉴朱友贞之失，亲贤远佞。大意据《资治通鉴·后梁纪五》。

## 问道悟道

梁晋争霸与后梁之亡这出戏，藏着两条让人开眼的规律。

**武夫治国的悖论。**

朱友贞是后梁末帝，武夫之后，本身却非善战的武夫——他靠诛兄得位，靠的是禁军与外戚，不是沙场之功。朱友贞任人不明，亲小远贤——用姻亲段凝，不用忠将王彦章；听赵岩、张汉鼎的谗言，罢王彦章。结果是段凝拥兵不救，王彦章兵败被杀，后梁速亡。这是"武夫治国的悖论"的典型——武夫以兵权得天下，可武夫不善于治国；治国需要用人，用贤则存，用佞则亡。朱友贞用佞弃贤，是武夫治国的典型败局。

武夫治国的悖论这事，历史上有几个对照。三国时蜀汉，刘备以武夫起家，可刘备善用人——用诸葛亮治国，用关羽张飞征战，故能鼎足三分。刘备死后，刘禅庸碌，亲黄皓远贤臣，蜀汉速亡。大意据《三国志·蜀书·后主传》。西晋，司马氏以权臣篡魏，可司马炎之后，惠帝昏愚，八王之乱，五胡乱华，西晋速亡——武夫权臣之后，若不能用人治国，必然速亡。大意据《晋书·惠帝纪》。后梁朱友贞、蜀汉刘禅、西晋惠帝，都是武夫或权臣之后，本身非善治之君，任人不明，必然亡国。可见武夫以马上得天下，不能以马上治天下——马上得天下靠的是兵权，治天下靠的是用人；用贤则存，用佞则亡。朱友贞之败，败在不会用人——武夫之朝，最怕的不是不会打仗，是不会用人。

**藩镇倒戈与北方霸权易手。**

魏博兵变降晋，是后梁失河北的关键。魏博节度使杨师厚卒，朱友贞欲分魏博六州，银枪效节都哗变，叛梁降晋——这支精骑，反成了灭梁的先锋。柏乡之战后，后梁失河北；魏博兵变后，后梁失黄河以北；胡柳陂之后，后梁退守黄河以南。霸权一步步易手，从后梁转到沙陀晋——这是藩镇倒戈与北方霸权易手的典型。

藩镇倒戈与北方霸权易手这事，历史上有几个对照。东汉末，董卓之乱后，关东诸侯倒戈——可关东诸侯各怀鬼胎，未能一举取代董卓。大意据《后汉书·董卓传》。曹魏末，司马氏篡魏——可司马氏是权臣篡位，非藩镇倒戈。大意据《晋书·宣帝纪》。唐朝安史之乱，河北三镇降将叛唐——可安史之乱是叛乱，非藩镇倒戈易主。大意据《旧唐书·安禄山传》。魏博兵变不同——魏博本是后梁的藩镇，一旦哗变降晋，反成灭梁先锋。这是藩镇倒戈直接导致霸权易手的典型。可见藩镇之兵，只认利害，不认主君；今日为你所用，明日为敌所用。王朝之靠藩镇，如靠山靠水——水能载舟，亦能覆舟。后梁靠藩镇起家（朱温本是大藩镇），亦因藩镇倒戈而亡——这是五代"以藩镇得天下、以藩镇失天下"的循环。要到宋兴，赵匡胤杯酒释兵权，削藩镇之权，才结束这个循环——可那已是五十余年后的事了。

## 结语

梁晋争霸十二年，后梁一路败退，晋一路南下——柏乡之败定河北，魏博兵变失河北，胡柳陂后守河南，汴梁城破亡后梁；武夫治国的悖论让朱友贞用佞弃贤，藩镇倒戈让霸权易手——十六年的短命王朝，就此收场。
