# 资治通鉴·唐纪六·二圣临朝

## 讲事情

贞观二十三年，公元 649 年五月，太宗驾崩，年五十二。太子李治即位，是为高宗，年二十二。太宗临终前，命长孙无忌、褚遂良辅政——长孙无忌是高宗的舅父（长孙皇后之兄，长孙皇后是长孙无忌之妹——《旧唐书·长孙无忌传》），褚遂良是太宗信任的诤臣。这一对托孤组合，是太宗留给高宗的最后一道保险。可这道保险，没保住几年。

高宗即位，改元永徽。永徽初年，朝政还算太平——长孙无忌、褚遂良主持，沿贞观旧制。可高宗的后宫，暗流涌动。王皇后是太原王氏，出身名门，端庄而无子。萧淑妃得宠，生子女。王皇后怕萧淑妃压过自己，想找个人分萧淑妃的宠。她找到了一个人——武氏。

武氏就是后来的武则天。她本名武照（后自造"曌"字），并州文水人，父亲武士彟是唐朝开国功臣，封应国公。武氏十四岁入宫，为太宗才人，赐号"武媚"——大意据《旧唐书·则天皇后本纪》。才人是太宗的嫔妃，武氏在太宗后宫十多年，地位不高。这是唐宫的一桩特例：武氏先是太宗的才人，后来又成了高宗的皇后——父子共妻，在唐宫里不算罕见（唐宫有收继婚的胡风遗存），但在正史里仍是需要说清的一笔。

太宗崩后，武氏依例出家，入感业寺为尼。永徽元年，公元 650 年五月，高宗行香感业寺，与武氏相遇。两人相对而泣——高宗在太宗生前已与武氏有情，此时旧情复燃。王皇后听闻，主动劝高宗把武氏接回宫，想用武氏分萧淑妃的宠。永徽二年，武氏还俗入宫，拜昭仪。这一手引狼入室，是王皇后后来被废的伏笔。——《资治通鉴·唐纪十五》。

武氏入宫后，迅速得宠，又生了儿子李弘。王皇后和萧淑妃都被冷落。王皇后这才醒悟——她引武氏入宫分萧淑妃的宠，结果武氏把王皇后和萧淑妃一起压了下去。王皇后与萧淑妃反联手对付武氏，可已经晚了。武氏心狠手辣——传说她亲手掐死自己的女儿嫁祸王皇后（此事《资治通鉴》记"或曰"存疑，未坐实）。大意据《资治通鉴·唐纪十五》。

永徽六年，公元 655 年，废后之争到了决战。高宗想废王皇后立武氏。长孙无忌、褚遂良坚决反对。褚遂良在朝堂上叩头流血，说："武氏经事先帝，众所共知，天下耳目，安可蔽也。陛下若立为皇后，万代之后，谓陛下何！"——武氏侍奉过先帝，天下人都知道，陛下若立她为后，后世怎么议论陛下！大意据《旧唐书·褚遂良传》《资治通鉴·唐纪十五》。高宗大怒，要把褚遂良拉出去。长孙无忌也反对，但没褚遂良那么激烈。

可高宗心意已决。这时，李义府、许敬宗这些寒门出身的大臣站出来支持立武氏。李义府被长孙无忌排挤，要贬出去，他孤注一掷，上表请废王皇后立武氏，赌赢了——武氏立后，李义府成了功臣。许敬宗也借机上位。这群寒门新贵，借废后之争，把长孙无忌、褚遂良这些关陇旧勋打下去。永徽六年十月，高宗下诏废王皇后、萧淑妃为庶人，立武氏为皇后。王皇后、萧淑妃后被武氏所害——传说武氏断其手足，投酒瓮中，称"骨醉"。大意据《资治通鉴·唐纪十五》。

立武氏为后只是第一步。显庆四年，公元 659 年，许敬宗构陷长孙无忌谋反——借太子洗马韦季方"朋党案"把刀子指向长孙无忌。高宗惊得落泪，说："我家不幸，亲戚间屡有异志……舅若果尔，朕决不忍杀之。"许敬宗搬出汉文帝哭杀舅父薄昭的旧事，逼高宗"当断不断，反受其乱"。高宗终究没亲自过问，下诏削长孙无忌太尉及封邑，流放黔州。同年七月，许敬宗遣人赴黔州，"再鞫无忌反状，至则逼无忌令自缢"。凌烟阁二十四功臣之首、太宗托孤的顾命大臣，就这么死在了西南的贬所里。《资治通鉴》冷冷记下一句："自是政归中宫。"大意据《资治通鉴·唐纪十五》《旧唐书·长孙无忌传》。

这四个字，是二圣临朝格局的起点。长孙无忌一死，朝中再无人能挡武后的路。可武后真正走到台前，靠的不是阴谋，是高宗的身体。显庆五年，公元 660 年十月，高宗"初苦风眩头重，目不能视，百司奏事，上或使皇后决之"。《资治通鉴》记了一句要紧话："后性明敏，涉猎文史，处事皆称旨。由是始委以政事，权与人主侔矣。"——皇后脑子灵光，又读过书，处理奏报件件合皇帝心意。一来二去，政事就交到了她手里，权柄与皇帝不相上下。——《资治通鉴·唐纪十六》。

可这时武后还只是"幕后"。真正让她走到帘前的，是麟德元年，公元 664 年那场未遂的废后风波。宦官王伏胜告发武后引道士郭行真入宫行厌胜之术，高宗大怒，密召西台侍郎上官仪起草废后诏书。上官仪刚把诏书写完，武后已得消息，赶到高宗面前自诉。高宗一见她，竟"羞缩不忍"，把责任全推到上官仪头上："此皆上官仪教我。"武后当即命许敬宗构陷上官仪与废太子李忠通谋，上官仪父子下狱处死，李忠赐死。大意据《资治通鉴·唐纪十六》。

自此，朝堂上出现了一道奇景。《资治通鉴》记："自是上每视事，则后垂帘于后，政无大小，皆与闻之。天下大权，悉归中宫，黜陟、杀生，决于其口，天子拱手而已，中外谓之'二圣'。"——皇帝坐在前头，皇后坐在帘后，大小政事都要她点头。天下人管这叫"二圣"。——大意据《资治通鉴·唐纪十六》。到了上元元年，公元 674 年八月，高宗下诏追尊祖宗帝后，"皇帝称天皇，皇后称天后"。天皇天后，并称天下，这在中国历史上是头一遭。从永徽六年的废王立武，到上元元年的天皇天后，武后用了近二十年，一步步把"皇后"这个后宫之位，建构成了与皇帝并坐的"天后"。二圣临朝的格局，就此成形。武后如何从二圣走向独圣、最终代唐称帝，详见《唐纪·武周代唐·二圣临朝》及《唐纪·武周代唐·女主称帝》。

## 讲人物

**李治：软弱的继位者**

李治是太宗第九子，长孙皇后第三子。他被立为太子，是因为最不会夺嫡（详见《资治通鉴·唐纪五·废立太子》）。可"不夺嫡"的温和，到了帝位上，就是软弱。高宗这一辈子，是被三个人架着走的一生——前半段被长孙无忌、褚遂良架着（永徽之治还算清明），后半段被武则天架着（二圣临朝，天子拱手）。

高宗最大的毛病，是当断不断。废王皇后立武氏，他心意已决，可面对褚遂良叩头流血的直谏，他又犹豫；长孙无忌反对，他又不敢硬来。最后靠李义府、许敬宗这些寒门把长孙无忌打下去，才立成武氏。这一段，高宗还算做了决断。可到了麟德元年废后风波，他密召上官仪起草废后诏书，武后一到，他竟"羞缩不忍"，把责任推给上官仪——这一推，把上官仪推上了死路。一个连废后都不敢认账的皇帝，怎能不被武后压在帘后？高宗的软弱，是二圣临朝的土壤——没有高宗的软弱，就没有武后的临朝。太宗当年立李治，赌的是"温和者不夺嫡"，可他没料到——温和者不夺嫡，却会被别人夺权。高宗这个太子，是太宗无奈之下的无奈之选，也是唐朝由盛转衰的转折点。

**武则天：从才人到天后的女主**

武则天这一辈子，是中国历史上最传奇的女人。她十四岁入宫，为太宗才人，赐号"武媚"——才人是低阶嫔妃，她在太宗后宫十多年，地位不高。太宗崩后，她出家感业寺，本该青灯古佛了此一生。可高宗行香感业寺，旧情复燃，她被接回宫，拜昭仪。这一年，她二十七岁。从昭仪到皇后，她用了四年（永徽二年入宫，永徽六年立后）；从皇后到天后，她又用了近二十年。这条路，她走得步步为营，步步算计。

武则天的能耐，在两样：一是心狠，二是能干。心狠，是她能掐死女儿嫁祸王皇后（虽存疑），能害王皇后萧淑妃"骨醉"，能逼死长孙无忌、杀上官仪。这份狠，是女主立足的根——在男尊女卑的朝堂上，一个不狠的女主，活不下来。能干，是她"性明敏，涉猎文史，处事皆称旨"——她读过书，脑子灵，处理奏报件件合高宗心意。高宗风眩不能视事，她接过来，朝政照样运转。这份能干，是女主立足的本——没有能干，狠也只是匹夫之狠。武则天是狠与能干的结合，这才是她能走到天后之位的真本事。她不是靠美色（美色只能让她入宫），不是靠阴谋（阴谋只能让她上位），是靠能干让她坐稳。这一点，胡三省看得最清楚——他注"权与人主侔矣"时，是真心佩服这个女人的本事。大意据《资治通鉴音注》唐纪部分。

**王皇后：引狼入室的失位者**

王皇后是太原王氏，出身名门。她端庄而无子，这是她最大的软肋——在后宫里，无子就没有根基。萧淑妃得宠生子，王皇后不安。她想找个人分萧淑妃的宠，找到了武氏。这一手引狼入室，是王皇后后来被废的伏笔。她以为武氏是自己的棋子，可武氏一旦得宠生子，就不再是任何人的棋子——武氏把她和萧淑妃一起压了下去。

王皇后的悲剧，在于自作聪明。她引武氏入宫，本是想用武氏分萧淑妃的宠，保住自己的后位。可她没料到，武氏不是她能用的人——武氏心比天高，志在后位。王皇后引狼入室，结果狼把主人吃了。废后之后，王皇后被废为庶人，又被武氏所害。临死前，王皇后说："愿皇上万岁，武昭仪承恩日久，愿来世生为猫，武氏为鼠，吾扼其喉以报。"大意据《资治通鉴·唐纪十五》。这话是诅咒，也是无力——一个被自己引进来的狼吃掉的人，到死只能诅咒。王皇后之败，败在引狼；可引狼这事，在历史上屡见不鲜——以为能驾驭的，往往驾驭不了。

**长孙无忌：托孤重臣的末路**

长孙无忌字辅机，长孙皇后之兄，太宗的舅子兼布衣之交。他是凌烟阁二十四功臣之首，太宗临终托孤的第一人。太宗把高宗交给他，是把唐朝的命运交给他。可这份托孤之重，也成了他的催命符。辅政的舅父，权倾朝野，新帝长大必被嫌——这是托孤重臣的通病。长孙无忌反对废王皇后立武氏，触怒武氏；他又排挤李义府这些寒门，得罪寒门新贵。武氏要立后，寒门要上位，两股势力合流，长孙无忌就成了靶子。

显庆四年，许敬宗借"朋党案"构陷长孙无忌谋反。高宗惊得落泪，说"舅若果尔，朕决不忍杀之"——可他终究没亲自过问，下诏流放，任许敬宗逼死。这一"不忍"里，有几分真不忍，几分乐得甩手，谁也说不清。可结果是清楚的——凌烟阁二十四功臣之首，太宗托孤的顾命大臣，被逼自缢于黔州贬所。太宗当年抢剑定策立李治，长孙无忌是首功；如今李治坐视长孙无忌被逼死，这一进一出，是托孤重臣最深的反讽。你拥立的人，最终杀了你——这便是托孤的代价。长孙无忌的悲剧，详见《资治通鉴·唐纪六·二圣临朝》本篇。

## 讲背景

二圣临朝这事，得放在关陇集团与寒门崛起、唐宫的收继婚俗、以及三省六部的权力结构三重背景里看。

先看关陇集团与寒门崛起。关陇集团是唐初立国的基本盘——长孙无忌、褚遂良、房玄龄、杜如晦，多是关陇旧勋或关东士族（详见《资治通鉴·唐纪三·贞观之治》）。永徽初年，长孙无忌、褚遂良辅政，关陇集团把持朝政。可关陇集团独大，寒门出身的大臣就难上位。李义府被长孙无忌排挤要贬出去，许敬宗也不得志。这群寒门新贵，需要一个机会上位——废王立武之争，就是这个机会。李义府、许敬宗支持立武氏，不是真心拥护武氏，是借废后之争把长孙无忌这些关陇旧勋打下去。武氏立后，关陇集团失势，寒门新贵上位——这是一场借后宫之争完成的前朝洗牌。陈寅恪先生论此极精，大意据陈寅恪《唐代政治史述论稿》。可见废王立武，不只是一桩后宫事，是关陇集团与寒门新贵的一次权力交接。武氏能立后，靠的不只是高宗的宠爱，是寒门新贵的政治支持。

再看唐宫的收继婚俗。武氏先是太宗才人，后是高宗皇后——父子共妻，在汉家礼法里是大忌，可在唐宫里却行得通。这不是偶然。唐朝皇室有鲜卑血统，李渊的祖父李虎是西魏八柱国之一，关陇集团胡汉合流，收继婚（父死子继其妾、兄死弟继其嫂）是胡俗遗存。所以褚遂良谏"武氏经事先帝"时，高宗并不以为耻——在唐宫的风气里，这不算什么大逆不道。可褚遂良的话戳中的不是礼法，是名声——"万代之后，谓陛下何"。武氏的出身问题，成了她合法性建构的先天短板，她后来要花大力气弥补。这一点，是理解武后一辈子建构合法性的钥匙——她的每一步，都在洗"先帝才人"这个底色。从皇后到天后到临朝称制到代唐称帝，她要建构的，是一个"先帝才人"如何名正言顺统治天下的合法性。大意据《旧唐书·则天皇后本纪》。

再看三省六部与谏官制度。废王立武之争，长孙无忌、褚遂良在门下省行使封驳之权——褚遂良叩头流血直谏，是门下省诤谏的极致。可这套制度，挡不住高宗的决心——高宗绕过门下，下诏废后。可见三省六部的封驳之权，在皇帝决心面前是空的——皇帝肯听，封驳有效；皇帝不听，封驳无效。这是制度诤臣的局限（详见《资治通鉴·唐纪三·贞观之治》论"诤臣之能，一半在诤臣一半在君主"）。褚遂良被贬死爱州，长孙无忌被逼自缢——门下省的两个长官，一个贬死一个自缢，三省六部的封驳之权，至此名存实亡。武后临朝，再无制度能挡她。可见制度是死的，人是活的；制度能挡肯听的皇帝，挡不住不听的皇帝，更挡不住临朝的女主。

再看高宗的身体与武后的才能。显庆五年，高宗风眩头重，目不能视——这是高宗一辈子的病根。一个不能视事的皇帝，朝政就得有人接。接的人是武后——她"性明敏，涉猎文史，处事皆称旨"。这一接，就接出了二圣临朝。可见女主临朝，不是女主强抢，是君主病弱。汉武帝晚年病，霍光辅政；唐高宗病，武后临朝——君主病弱，权力就会外移，不是移到外戚就是移到宦官或权臣。武后的特殊，在于她是皇后，能正大光明地接。这正大光明，让她比外戚宦官更难制衡——外戚可以除，宦官可以诛，可皇后是君主的正妻，除她等于废君主。这是女主临朝的制度优势，也是制度困境。

## 讲道理

王夫之在《读通鉴论》卷二十里论高宗与武后，话说得不客气。大意据王夫之说，高宗之弱，是二圣临朝的根——没有高宗的弱，就没有武后的强。可王夫之也指出，武后之能，不是一朝一夕——她从才人到皇后到天后，每一步都是算计。王夫之最叹的是高宗的"羞缩不忍"——一个连废后都不敢认账的皇帝，怎能不被武后压在帘后？王夫之论高宗，是把他放在"懦主"的位置上——太宗英武，生了个懦弱的儿子，这是帝王家最大的不幸。可王夫之也不全怪高宗——他说武后"性明敏，涉猎文史"，是真心能干；这份能干，连王夫之都不得不承认。王夫之论武后，是既恨她的狠，也服她的能；既骂她的篡，也承认她的治。这份分寸，是史家的难处。大意据《读通鉴论》卷二十。

胡三省在《资治通鉴音注》唐纪部分，对二圣临朝的批注，多落在"权柄下移"上。大意据胡三省注，权柄下移，是君主失柄之渐。高宗病弱，权柄移到武后；武后一旦接手，就再不放手。胡三省生在宋元之际，亲见南宋权臣、后宫、宦官交替专权，他对"权柄下移"格外敏感。他的潜台词是：权柄一旦下移，就难收回——高宗想让武后暂代，可这一代就代到底，连高宗自己都收不回。胡三省的注，是把"权柄下移"看作王朝衰乱的通病——君主只要让出权柄，无论让给谁，都是让出江山。他注"自是政归中宫"四字时，是带着亡国之痛的——这四个字，是唐朝由盛转衰的起点。大意据《资治通鉴音注》唐纪部分。

司马光在《资治通鉴·唐纪十五》"臣光曰"里对废王立武有一段议论。大意据司马光说，高宗之失，失在不听诤臣。长孙无忌、褚遂良力谏，高宗不听，反而贬死二人——这是"君暗臣直"的反面。司马光最看重的是"名分"——武氏经事先帝，名分不正；立她为后，是乱名分。名分一乱，国本就动摇。司马光写废王立武，是写给宋朝皇帝看的——他要宋帝听诤臣、正名分，不要学高宗。司马光的史观，一贯是"治乱在人主之德"——高宗之弱，是德之不足；武后之强，是高宗之弱养出来的。德之修与不修，是治乱的分水岭。大意据《资治通鉴·唐纪十五》臣光曰。

## 问道悟道

二圣临朝这出戏，藏着两条让人开眼的规律。

**女主干政的合法性建构。**

武后从才人到皇后到天后，每一步都在建构合法性。她的起点是先帝才人——这个身份是她的先天短板，褚遂良一句"武氏经事先帝，众所共知"戳的就是这个短板。要补这个短板，她用了近二十年。立后靠的是高宗的宠爱和寒门的支持——这是后宫与朝堂两路并进。临朝靠的是高宗的病弱和自己的能干——这是君主失柄与女主接柄的巧合。天后靠的是天皇天后并称的诏书——这是名分上的建构。从才人到天后，她建构的，是一个"先帝才人"如何名正言顺统治天下的合法性。这份合法性建构，是女主干政的核心——没有合法性，女主就是"牝鸡司晨"；有了合法性，女主就是"天后临朝"。

女主干政的合法性建构这事，历史上有几个对照。汉高祖刘邦死后，吕后临朝称制。吕后的合法性，来自她是汉惠帝刘盈的生母、高祖的皇后——母以子贵，后以帝显。吕后临朝，靠的是太后的名分，可她没有"能干"做支撑，只靠狠，所以一死诸吕就被诛，临朝不到八年。《汉书·高后纪》。北魏文明太后冯氏，临朝称制二十余年。冯氏的合法性，来自她是献文帝的养母、孝文帝的太皇太后——母以子贵。可冯氏比吕后强在能干——她推行均田制、三长制，是北魏汉化的推手。冯氏临朝能成，一半靠太后的名分，一半靠能干。《魏书·文成文明皇后传》。辽朝萧太后（萧燕燕），临朝称制二十余年。萧太后的合法性，来自她是辽圣宗的生母、景宗的皇后——母以子贵。萧太后也极能干，澶渊之盟跟宋真宗对峙，不落下风。《辽史·圣宗纪》。吕后、冯太后、萧太后、武则天，都是女主临朝，可结局不同——吕后只靠狠不靠能干，临朝八年身死族灭；冯太后、萧太后、武则天，都靠狠加能干，临朝都长久。可见女主干政的合法性，一半在名分（太后/皇后的身份），一半在能干。名分给的是起点，能干给的是延续。武则天的特殊，在于她的起点最弱——先帝才人，名分最不正，所以她要花最多力气建构合法性，从皇后到天后到临朝称制到代唐称帝，每一步都是合法性的一层加固。这是女主干政合法性建构的极致，也是中国历史上唯一一个走到"代唐称帝"的女主。

**托孤重臣的末路。**

长孙无忌是太宗托孤的第一人，凌烟阁二十四功臣之首。他辅政永徽，权倾朝野。可这份托孤之重，也成了他的催命符。他反对废王立武，触怒武氏；排挤寒门，得罪新贵。两股势力合流，他就成了靶子。显庆四年被构陷谋反，流放黔州，被逼自缢。太宗当年抢剑定策立李治，长孙无忌是首功；如今李治坐视长孙无忌被逼死——你拥立的人，最终杀了你。这便是托孤重臣的末路。

托孤重臣的末路这事，历史上有几个对照。汉武帝托孤霍光，霍光辅政昭帝、宣帝，权倾朝野。霍光死后，霍氏一族被汉宣帝诛灭——霍光拥立的汉宣帝，最终族灭霍氏。《汉书·霍光金日磾传》。魏明帝托孤司马懿，司马懿辅政曹芳，可司马懿发动高平陵之变，夺了曹魏的权——托孤重臣反过来篡了位。《晋书·宣帝纪》。明神宗托孤张居正，张居正辅政十年，推行改革，权倾朝野。张居正一死，明神宗清算张家，抄家夺爵。《明史·张居正传》。霍光、司马懿、张居正、长孙无忌，都是托孤重臣，可结局不同——霍光是死后被族灭，张居正也是死后被清算，长孙无忌是生前被逼死，司马懿是反过来篡位。可见托孤重臣的末路，有三种：一是生前被逼死（长孙无忌），二是死后被清算（霍光、张居正），三是反过来篡位（司马懿）。这三种结局，都源于同一个根——托孤重臣权太大，新帝长大必被嫌。辅政时结党，新帝亲政必被清算；辅政时揽权，新帝亲政必被夺。这是托孤重臣的结构性困境——你要辅政，就得有权；有权，就有被清算的风险；要躲过清算，要么像司马懿那样篡位，要么像诸葛亮那样早死（诸葛亮鞠躬尽瘁死而后已，是托孤重臣唯一善终的特例，靠的是不揽私权）。长孙无忌没看穿这一层，所以他死；张居正没看穿，所以他死后被清算；霍光没看穿，所以他族灭。可见托孤之难，难在"辅政"与"揽权"的分寸——辅政可以，揽权必祸。长孙无忌辅政永徽，本可善终，可他排挤寒门、反对废后，是揽权太甚，所以被清算。这是托孤重臣末路的最深教训。

## 结语

懦主病弱让出权柄，女主能干接成天后——二圣临朝是"女主合法性建构"的二十年布局，也是"托孤重臣末路"的又一面镜子。
