# 资治通鉴·后周纪五·世宗柴荣

## 讲事情

后周世宗柴荣，是五代最后的雄主，亦是《资治通鉴》全书的终点。他即位六年，南征北伐，几乎把燕云失地收了一半，却在三十九岁的盛年猝然崩殂——天不假年，统一大业半途而废。他崩后半年，陈桥兵变，赵匡胤黄袍加身，五代终，宋兴。司马光写《资治通鉴》，自三家分晋（前403）写起，至此世宗崩（959）收笔——一千三百六十二年的治乱，在这一年画上句点。

后周的建国，承后汉之亡而来。乾祐三年（950），后汉隐帝自毁柱石，诛三大权臣，又逼反大将郭威。郭威自邺城起兵，隐帝兵败被杀，后汉亡（后汉亡国主场详见《资治通鉴·后汉纪一·后汉短祚》）。广顺元年（951）正月，郭威在澶州被诸军拥立，裂黄旗披身，遂还汴即位，国号"周"，是为后周太祖。大意据《资治通鉴·后周纪一》《旧五代史·周书·太祖纪》。郭威本寒微出身，邢州尧山人，年少颈刺飞雀，人称"郭雀儿"，从军后历事后唐、后晋、后汉三朝，至后汉已手握重兵。郭威即位后，革五代弊政——罢苛法，减赋税，罢户部营田务还民，毁佛铸钱以充国用——后周得有几年休养。大意据《资治通鉴·后周纪一》。

郭威在位三年，于广顺三年（953）崩。郭威亲子皆于后汉时为隐帝所杀，遂以养子柴荣继位。柴荣本是郭威内侄兼养子——柴氏是郭威之妻（圣穆皇后），亦是柴荣的姑母（柴荣父柴守礼之妹），故郭威是柴荣姑父；柴荣本姓柴非郭，幼时被郭威收为养子（人物血缘核验见 _figure_lineage.md，柴荣本姓柴非郭、郭威内侄兼养子非亲子）。柴荣即位，是为后周世宗，改元显德。大意据《资治通鉴·后周纪三》《旧五代史·周书·世宗纪》。

柴荣即位之初，便遭强敌压境。显德元年（954）正月，北汉主刘崇（刘知远弟，沙陀刘氏，据太原）趁郭威新丧、柴荣新立，结契丹为援，率三万兵南下，欲复后汉之仇。柴荣力排众议，决意亲征。三月，周汉两军会战于高平（今山西高平）。战初，周军右翼樊爱能、何徽临阵奔逃，周军阵势动摇；柴荣亲冒矢石，督阵冲杀，禁军将赵匡胤、张永德等力战，大破北汉军。刘崇仅以身免，逃回太原。大意据《资治通鉴·后周纪三》《旧五代史·周书·世宗纪》。高平之战，是柴荣立威之战——一个新君以亲征大胜强敌，自此朝野归心。战后柴荣诛樊爱能等临阵脱逃之将以正军法，后周军威大振。

柴荣的雄心不止于自保。他采纳比部郎中王朴《平边策》之议——"先南后北、先易后难"，先取南唐江北之地以富国力，再北伐幽云以收失地。显德二年到三年（955—956），柴荣一征南唐，攻寿州，久围不下。显德四年（957），柴荣二征南唐，亲征淮上，大破南唐军于紫金山。显德五年（958），柴荣三征南唐，南唐主李璟力屈，献江北十四州（庐、寿、光、黄、舒、蕲、和、濠、泗、楚、扬、泰、通、滁），去帝号称臣。大意据《资治通鉴·后周纪四》《旧五代史·周书·世宗纪》。三征南唐，取江北十四州，是柴荣南向的最大战果——后周自此据有淮南，国力大盛。

显德六年（959）三月，柴荣北伐幽云。这是石敬瑭割燕云（936）以来，中原第一次主动北伐收复失地。周军北上，宁州、莫州、瀛州不战而降，周军连取瀛、莫、易三州，又拔瓦桥、益津、淤口三关。柴荣本欲乘胜直取幽州（契丹南京），可诸将以为契丹势强，劝止。柴荣不从，决意进兵。正在此时，柴荣忽染重病，遂不得不班师。大意据《资治通鉴·后周纪五》《辽史·穆宗纪》。北伐取瀛、莫、易三州，是中原收复燕云的第一步——柴荣几乎要把石敬瑭割出去的燕云收回来一半了。可天不假年，柴荣一病，大业半途而废。

显德六年（959）六月，柴荣崩于万岁殿，年三十九岁。柴荣在位仅六年，却南征北伐，几欲统一——史称"五代第一明君"。柴荣崩后，其子宗训即位，是为恭帝，年仅七岁。主少国疑，军权在外。显德七年（960）正月，契丹与北汉合兵南下，朝廷遣殿前都点检赵匡胤北征。大军行至陈桥驿，诸将黎明时分拥赵匡胤为帝，裂黄袍披其身——是为"陈桥兵变"。赵匡胤遂还师汴梁，恭帝禅位，赵匡胤即位，国号"宋"，是为宋太祖。大意据《资治通鉴·后周纪五》《宋史·太祖本纪》。五代终，宋兴。司马光《资治通鉴》记事起周威烈王二十三年三家分晋（前403），讫后周世宗显德六年（959），紧接着的960正月陈桥兵变、赵匡胤建宋，便是这一千三百六十二年通鉴史的最后回响——五代乱世，至此终结；两宋三百二十年，自此开端。

## 讲人物

**郭威：被逼反又革弊政的太祖**

郭威是后周太祖，邢州尧山人，寒微出身，年少颈刺飞雀，人称"郭雀儿"。从军后历事后唐、后晋、后汉三朝，骁勇善战，至后汉时已是手握重兵的枢密使。郭威本忠事后汉，直到隐帝屠其家属，悲愤交加，才以"清君侧"起兵（郭威被逼反始末详见《资治通鉴·后汉纪一·后汉短祚》论郭威被逼反又建后周的太祖）。大意据《旧五代史·周书·太祖纪》《新五代史·周本纪》。

郭威即位后，革五代弊政——这是他对后周最大的贡献。罢苛法，减赋税，罢户部营田务还民（把官营屯田还归百姓），毁佛铸钱以充国用。郭威为政清廉，生活俭朴——他曾对柴荣说："我若死，当衣纸衣，坟以瓦棺，勿修地宫，勿置石人石马。"这份薄葬之嘱，是五代君主少有的俭德。大意据《资治通鉴·后周纪一》。郭威在位三年，革弊政、休养民力，为柴荣后来的南征北伐打下了国力基础。可见郭威虽以兵变得国，可他能革弊政、休养民——这份"以治国续兵变"的本事，是五代君主少有的。郭威与柴荣，是后周两代雄主——郭威打地基，柴荣起高楼；郭威革弊政，柴荣征四方。可惜柴荣起楼到一半便崩了，后周的高楼遂成半拉子工程。

郭威以养子柴荣继位，是养子继统的少数成功案例。柴荣是郭威内侄兼养子——柴氏是郭威之妻，亦是柴荣姑母，故郭威是柴荣姑父；柴荣本姓柴，幼时被郭威收为养子。这份"内侄兼养子"的双重关系，让柴荣既有血亲之亲，又有养子之正，继统较少争议。可柴荣早卒（959），后周旋亡——可见纵有养子继统的成功，亦难长久（养子继统的合法性困境详见《资治通鉴·后唐纪七·明宗之治》）。郭威养柴荣，是五代养子继统最成功的一例，却也是最短命的一例——成功与短命集于一身，正是五代的写照。

**柴荣：英主短祚的五代第一明君**

柴荣是后周世宗，本姓柴非郭，郭威内侄兼养子。柴荣即位时年三十三岁，正值壮年，胸怀大志。他即位之初便立下大愿："朕当以十年开拓天下，十年养百姓，十年致太平。"——三十年太平之愿。大意据《新五代史·周本纪》。可惜天不假年，柴荣在位仅六年便崩——三十年的宏愿，只活了六年的命。

柴荣其人，是五代第一明君。他能征——高平之战亲征刘崇，大破北汉契丹联军；三征南唐取江北十四州；北伐幽云取瀛莫易三州。他能治——革郭威未尽之弊，整顿军纪，汰冗员，均田赋，抑佛铸钱以充国用，兴修水利以利农桑。他能用人——王朴献《平边策》定"先南后北"之略，柴荣用之；赵匡胤高平之战崭露头角，柴荣拔为殿前都指挥使。大意据《资治通鉴·后周纪三至五》《旧五代史·周书·世宗纪》。柴荣的明，是治乱兼备的明——既能打仗，又能治国；既能用将，又能纳谏。这在五代君主中绝无仅有——朱温能打不能治，李存勖能打不能治，李嗣源能治不能打，石敬瑭既不能打又不能治，刘知远捡漏不能治，郭威能治不能征；唯独柴荣，能打能治能用能纳，是五代最完整的明君。

可柴荣最大的悲剧，是英主短祚。959年他北伐幽云，连取三州三关，正欲直取幽州，忽染重病，不得不班师。六月崩于万岁殿，年三十九岁。柴荣若再多活十年，燕云可复，天下可一——这几乎是后世史家的一致惋惜。可天不假年，柴荣崩，统一大业半途而废，恭帝年幼，军权在外，半年后陈桥兵变，后周亡。柴荣的英主短祚，是五代最大的遗憾——一个本可统一天下的雄主，倒在离统一最近的地方。这份遗憾，让后世读史者无不扼腕——王夫之、胡三省论柴荣，皆以"天不假年"为痛。

**赵匡胤：陈桥兵变终结五代的宋太祖**

赵匡胤是涿郡人，后周宿将。高平之战，赵匡胤随柴荣亲征，临阵力战，崭露头角，柴荣拔为殿前都指挥使，后升殿前都点检——掌禁军精锐。大意据《宋史·太祖本纪》《旧五代史·周书·世宗纪》。柴荣北伐幽云时，途中得一木牌，上书"点检作天子"——点检要做天子。柴荣疑之，遂以赵匡胤代张永德为殿前都点检。这一换，本是为防点检篡位，却阴差阳错地把赵匡胤推上了篡位的位置——柴荣防点检，点的正是赵匡胤。

柴荣崩，恭帝年幼，赵匡胤以殿前都点检掌禁军。显德七年（960）正月，契丹与北汉合兵南下，朝廷遣赵匡胤北征。大军行至陈桥驿，赵匡胤之弟赵匡义（后改名光义）与赵普等密谋，黎明时分率诸将拥赵匡胤为帝，裂黄袍披其身。赵匡胤遂还师汴梁，恭帝禅位，赵匡胤即位，国号"宋"，是为宋太祖。大意据《资治通鉴·后周纪五》《宋史·太祖本纪》。陈桥兵变，是兵权反噬的最后一笔——柴荣以殿前都点检掌禁军，赵匡胤正以此位夺了后周的天下。可见五代兵权在将的痼疾，到后周末年仍未解——柴荣防了点检，却把赵匡胤点上了点检；赵匡胤正是以点检之位，行篡代之事。

赵匡胤建宋后，吸取五代教训——杯酒释兵权，以文制武，立宗法制度，终结了五代"以兵变得位、以兵变失位"的循环。这份"以制度固兵权"的本事，是赵匡胤对五代速亡律的根本回应。可见赵匡胤虽以兵变得国，却能以制度防兵变——这是他与五代其他兵变君主最大的不同。五代终，宋兴；兵权反噬的循环，终被制度化解——可这化解，是建立在柴荣打下的国力基础上的。柴荣奠其基，赵匡胤成其业——这是后周到宋的传承。

**王朴：献《平边策》定统一之略的谋臣**

王朴是后周谋臣，字文伯，东平人。柴荣即位后，欲知天下大计，群臣多献"守成"之策，独王朴献《平边策》，提出"先南后北、先易后难"——先取南唐江北以富国力，再伐北汉、契丹以收失地。大意据《新五代史·王朴传》。柴荣用其策，故有三征南唐取江北十四州、北伐幽云取瀛莫易三州之功。王朴之策，是后周统一方略的纲领——柴荣南征北伐，皆循此略。

王朴其人，深谋远虑。他曾为柴荣卜算，言"臣窥唐末以来治乱，知陛下三十年之内当致太平"——可见其对柴荣的统一之愿深信不疑。可惜王朴亦早卒——显德六年（959）四月，王朴卒，年四十五岁；柴荣闻之恸哭。两个月后，柴荣亦崩。王朴与柴荣，是后周统一大业的两个支柱——王朴献略，柴荣执行；王朴先柴荣两个月而卒，柴荣旋亦崩。这两人的相继早逝，让后周的统一大业彻底断绝。可见英主短祚，亦折其谋臣——柴荣失王朴，如失其脑；脑与躯俱亡，大业遂废。大意据《新五代史·王朴传》《资治通鉴·后周纪五》。

## 讲背景

世宗柴荣这事，得放在五代统一的趋势、燕云失守与北伐收复、兵权在将与禁军体系、英主短祚的宿命四重背景里看。

先看五代统一的趋势。五代是分裂走向统一的过渡期。后梁、后唐、后晋、后汉四朝，皆短命，皆忙于内争；到后周郭威、柴荣，中原王朝才开始有意识地走向统一。郭威革弊政打地基，柴荣南征北伐起高楼——后周是五代唯一有统一之志且有统一之力的王朝。柴荣三征南唐取江北十四州，北伐幽云取瀛莫易三州，几乎把南北两线都打开了。可惜柴荣崩于959，统一大业半途而废；赵匡胤建宋，承后周国力基础，才得以完成统一——宋初削平荆南、后蜀、南汉、南唐、吴越、北汉，皆建立在柴荣打下的地基上。可见五代之统一，非一朝之功，是后周到宋的接力——柴荣奠其基，赵匡胤成其业。后周虽短祚，却是五代走向统一的关键一环。

再看燕云失守与北伐收复。石敬瑭936年割燕云十六州，是中原千年北防的断送（燕云失守与千年北防详见《资治通鉴·后晋纪一·石敬瑭借兵建晋》）。到959年柴荣北伐，已是燕云失守二十三年后——这是中原第一次主动北伐收复失地。柴荣连取瀛、莫、易三州，又拔瓦桥、益津、淤口三关，几乎要把燕云收回来一半了。可柴荣染病班师，大业半途而废——燕云的收复，要到明初徐达、常遇春北伐（1368）才完成，距柴荣北伐又过了四百零九年。可见燕云之失，贻害之深——柴荣几乎收回，却功亏一篑；此后宋辽、宋金数百年拉锯，燕云始终是中原的心腹大患。柴荣北伐取三州三关，是燕云失守后中原第一次"反攻"——虽未竟全功，却证明了燕云可复、北族可败。可惜天不假年，柴荣一崩，北伐遂止，燕云遂再失四百年。

再看兵权在将与禁军体系。五代兵权在将不在君，是速亡律的根子（速亡律详见《资治通鉴·后晋纪六·出帝亡国》论短命王朝的速亡律）。郭威、柴荣两代，虽革弊政，却未真正解决兵权在将的问题。柴荣整顿禁军——汰老弱，选精锐，设殿前都点检掌禁军精锐。可这一整顿，反而把禁军精锐集中到了殿前都点检一人手中——柴荣以赵匡胤为殿前都点检，正是把禁军交到了赵匡胤手里。柴荣崩，恭帝年幼，赵匡胤正以此位行陈桥兵变。可见柴荣整顿禁军，本是为强中央军权；可禁军越精锐，点检的权力越大，反成了篡位的资本。这份"强禁军则强点检、强点检则危皇权"的悖论，是五代兵权格局的死结——不整顿则禁军弱，整顿则点检强；柴荣整顿禁军，反给赵匡胤铺了路。要到赵匡胤建宋，杯酒释兵权，才真正化解此结。

最后看英主短祚的宿命。五代君主，雄主多短祚。后唐庄宗李存勖，灭梁称帝三年即亡于兴教门之变（同光灭梁详见《资治通鉴·后唐纪一·同光灭梁》）；后周世宗柴荣，南征北伐六年即崩。雄主短祚，是五代的宿命——能打的君主多亲征，亲征则多积劳，积劳则多早卒；能治的君主多勤政，勤政则多耗神，耗神则多损寿。柴荣亲征高平、亲征淮南、亲征幽云，六年三度亲征，积劳成疾，终崩于盛年。这份英主短祚的宿命，是五代最深的遗憾——柴荣若再多活十年，燕云可复，天下可一；可天不假年，柴荣倒在离统一最近的地方。可见英主短祚，是历史的最大"如果"——可惜历史没有"如果"。

## 讲道理

王夫之在《读通鉴论》卷三十（五代下，论周）里论柴荣，话说得极痛。大意据王夫之说，柴荣是五代第一明君，南征北伐，几欲统一，可惜天不假年，崩于盛年——这是历史最大的"如果"。王夫之以为，柴荣若再多活十年，燕云可复，天下可一；可天不假年，柴荣一崩，统一大业半途而废，后周旋亡。王夫之把柴荣放在五代君主的最上格——朱温、李存勖、石敬瑭、刘知远皆不能比，独柴荣有统一之志且有统一之力。大意据《读通鉴论》卷三十。王夫之亦论陈桥兵变，以为赵匡胤以点检之位行篡代，是兵权在将的极致——柴荣防点检，点的正是赵匡胤；柴荣整顿禁军，反给赵匡胤铺了路。可见兵权在将的痼疾，到后周末年仍未解。

胡三省在《资治通鉴音注》后周纪部分，对柴荣北伐的批注，亦多带痛惜。大意据胡三省注，柴荣北伐幽云，连取瀛莫易三州三关，几乎要把燕云收回来一半了；可染病班师，大业半途而废——这是中原收复燕云最近的一次，也是最遗憾的一次。胡三省是宋元之际人，亲眼见过蒙古骑兵据燕云南下灭宋，故其注柴荣北伐，字字带血——他写的不是柴荣一朝的功亏一篑，而是他亲身经历的亡国之痛的远因。胡三省的潜台词是：柴荣若复燕云，则宋辽、宋金、宋元数百年的南北拉锯或不致发生；可柴荣崩，燕云遂再失四百年——这份遗憾，是历史最深的痛。大意据《资治通鉴音注》后周纪部分。胡三省亦注王朴《平边策》，以为王朴先南后北之略，是后周统一方略的纲领；可惜王朴与柴荣相继早卒，大业遂废。

司马光在《资治通鉴·后周纪五》里对柴荣崩殂的记述，作为全书的终卷，亦带其一贯史观与一份深沉的收束。大意据司马光说，柴荣是五代第一明君，能征能治能用能纳，可惜英主短祚，崩于盛年——这是五代最大的遗憾。司马光以《资治通鉴》记事起三家分晋（前403），讫后周世宗显德六年（959），紧接着的960正月陈桥兵变、赵匡胤建宋，便是这一千三百六十二年通鉴史的最后回响。司马光的史观，是"治乱在人主之德"——柴荣有德，故能征能治；可英主短祚，天不假年，统一大业半途而废。大意据《资治通鉴·后周纪五》。三家的史观，殊途同归：柴荣是五代最后的雄主，亦是历史最大的"如果"——天不假年，大业半废，五代终，宋兴，通鉴止。

## 问道悟道

世宗柴荣这出戏，藏着两条让人开眼的规律。

**英主短祚与天命难违。**

柴荣在位仅六年，南征北伐，几欲统一；959年北伐幽云，连取三州三关，正欲直取幽州，染病班师，旋崩于万岁殿，年三十九岁。柴荣若再多活十年，燕云可复，天下可一——这几乎是后世史家的一致惋惜。可天不假年，柴荣倒在离统一最近的地方。这份英主短祚，是历史最大的"如果"，可惜历史没有"如果"。柴荣崩后，恭帝年幼，半年后陈桥兵变，后周亡——一个本可统一的雄主，倒在盛年；一个本可长治的王朝，亡于主少。可见英主短祚，不但是雄主的悲剧，更是王朝的悲剧——雄主一崩，根基未稳的王朝便经不起任何风浪。

英主短祚与天命难违这事，历史上有几个对照。蜀汉诸葛亮，六出祁山，欲复中原，积劳成疾，病逝五丈原（234），年五十四岁——诸葛亮北伐未竟，与柴荣北伐未竟前后呼应（鞠躬尽瘁详见《资治通鉴·魏纪三·诸葛亮北伐》）。前秦苻坚，统一北方，欲灭东晋一统天下，淝水一战（383）大败，旋为部将所杀——苻坚统一未竟，亦是天不假年之痛。大意据《晋书·苻坚载记》。后唐庄宗李存勖，灭梁称帝，三年即亡于兴教门之变——雄主而短祚，与柴荣同构（同光灭梁详见《资治通鉴·后唐纪一·同光灭梁》）。诸葛亮、苻坚、李存勖、柴荣，都是英主短祚的典型——能征能治，却倒在盛年或盛业。可见英主短祚，是历史的宿命——天命难违，非人力可挽。要免此憾，须培养接班人、立稳固制度——可柴荣亲子年幼，制度未立，遂一旦崩殂，大业遂废。这份遗憾，是给所有"雄主一身系天下"者的警示——一人之身，撑不起一个王朝；雄主之外，须有制度与接班。

**陈桥兵变与兵权反噬。**

陈桥兵变，是兵权反噬的最后一笔。柴荣整顿禁军，设殿前都点检掌精锐；柴荣崩，恭帝年幼，赵匡胤正以此位行篡代之事。柴荣防点检，点的正是赵匡胤；柴荣整顿禁军，反给赵匡胤铺了路。这份"强禁军则强点检、强点检则危皇权"的悖论，是五代兵权格局的死结——不整顿则禁军弱，整顿则点检强；柴荣整顿禁军，反成赵匡胤篡位的资本。可见兵权在将的痼疾，到后周末年仍未解；纵是雄主如柴荣，亦未能真正把兵权收归中央。

陈桥兵变与兵权反噬这事，历史上有几个对照。后唐庄宗李存勖，灭梁后以伶人乱政，禁军将郭从谦反，兴教门之变李存勖死——禁军反噬，与陈桥兵变同构（同光灭梁详见《资治通鉴·后唐纪一·同光灭梁》）。后汉隐帝逼反郭威，郭威以兵权行篡代——亦是兵权反噬（后汉亡国详见《资治通鉴·后汉纪一·后汉短祚》）。西汉，吕后死后，周勃、陈平等以兵权废少帝迎立文帝——亦是兵权在外戚与功臣间的博弈（诸吕之乱详见《资治通鉴·汉纪五·诸吕之乱》）。李存勖、后汉隐帝、西汉少帝、后周恭帝，都是兵权在外、主少国疑而被反噬的典型。可见兵权反噬的根子，在"兵权在将不在君、主少国疑"——主强则兵权在君，主少则兵权反噬。要免此祸，须以制度收兵权于中央——赵匡胤建宋后正是以此为本，杯酒释兵权，以文制武，立宗法制度，才真正终结了五代兵权反噬的循环。可见兵权反噬是表，制度不立是里；陈桥兵变是五代兵权反噬的最后一笔，亦是制度重建的起点——五代终，宋兴，兵权反噬的循环至此终结。

## 结语

资治通鉴全书至此终结——自三家分晋（前403）至此世宗崩（959），一千三百六十二年的治乱，在这一位英主短祚的雄主身上画上句点：柴荣南征北伐几欲统一却天不假年，崩后半年陈桥兵变、赵匡胤建宋，五代终、宋兴，一个千年纪元的治乱史就此收笔。
