# 资治通鉴·唐纪四·渭水之盟

## 讲事情

武德九年，公元 626 年，是唐朝最凶险的一年。六月初四，李世民在玄武门杀兄屠弟，逼父让位；八月初九，他即皇帝位，改元贞观。可皇位还没坐热，北边的狼烟就烧到了家门口。

八月，突厥颉利可汗、突利可汗合兵十余万，从泾州入寇，一路南下，势如破竹。那时唐朝刚经历玄武门之变，人心未定，朝堂上下一片惊惶。颉利的算盘很清楚：唐朝自己杀了半个皇室，正虚弱，正是勒索的好时机。——《资治通鉴·唐纪七》。

突厥铁骑直扑长安。八月，颉利进至渭水便桥之北，离长安城仅四十里。京师震骇，有人劝太宗弃城避敌，有人劝迁都樊邓。太宗一概不听。他做了一个让所有人都捏一把汗的决定——亲自出城，到渭水边跟颉利对话。

太宗带了侍中高士廉、中书令萧瑀、将军周范等六骑，从玄武门出来，直奔渭水。隔水就是颉利的大营，连营十余里，旌旗蔽日。太宗勒马渭水之上，隔着河水责备颉利负约——突厥此前曾与唐朝通好，颉利也曾遣使求婚，如今背盟入寇，于理不通。大意据《资治通鉴·唐纪七》《旧唐书·突厥传》。

颉利见太宗只带六骑出城，反而心里没底。草原上的规矩，敢这么做的，要么是真有伏兵，要么是有恃无恐。颉利正犹豫，唐军后队陆续赶到——太宗早已布置诸军列阵于后，旌甲连绵。颉利见唐军容严整，又怕太宗有诈，不敢轻举妄动。双方隔水对话一番，最后斩白马为盟。颉利答应退兵，太宗答应给一笔金帛。这一场"渭水之盟"，太宗花钱买和，颉利满载而归。

可这笔账，太宗记在心里。《资治通鉴》记他后来跟侍臣说过，"将欲取之，必固与之"——要取他，先给他，等他骄惰不备，一举可灭。大意据《资治通鉴·唐纪七》。这话是渭水之盟的注脚：盟是假，蓄势是真。太宗心里清楚，渭水之盟是耻辱，可这份耻辱，他忍下了。

忍下的代价是三年。贞观元年、二年，太宗一门心思养精蓄锐。他每天带数百人在显德殿前习射，亲自教士兵弯弓射箭，赏罚分明。有人劝他："您是天子，在殿前教人射箭，万一有狂徒混进来，怎么办？"太宗不听，说："王者视四海如一家，何必猜忌自己的将士？"——大意据《资治通鉴·唐纪八》。这一手，是把府兵训练成精锐。同时，太宗对突厥的属部极尽拉拢——薛延陀、回纥、奚、契丹，一个一个分化瓦解。

机会来了。贞观三年，公元 629 年，突厥连遭天灾，雪深过膝，羊马死损大半，属部薛延陀、回纥相继叛离。代州都督张公瑾上奏，陈突厥可灭之状六条：一是颉利政乱，二是突利背叛，三是薛延陀自立，四是回纥不服，五是雪灾饥荒，六是华人北奔熟知虚实。太宗大喜，遂命李靖为定襄道行军总管，李勣为通漠道行军总管，柴绍为金河道行军总管，薛万彻为畅武道行军总管，合兵十余万，分道并进。——《旧唐书·李靖传》《资治通鉴·唐纪九》。

贞观四年，公元 630 年正月，李靖率先动手。他手里只有三千精骑，却要直捣颉利牙帐。李靖赌的是突厥料定唐军不敢孤军深入，防备必松。果然，唐军一夜突至定襄城南恶阳岭，颉利大惊："唐兵若不倾国而来，靖岂敢孤军至此？"一日数惊。李靖又离间其心腹，突厥骁将康苏密来降。李靖趁夜攻入定襄，颉利仓皇北逃铁山。——《新唐书·李靖传》。

太宗接到捷报，高兴坏了，对侍臣说："李陵以步卒五千绝漠，然卒降匈奴，其功尚得书竹帛。靖以骑三千，深入虏庭，克复定襄，古未有辈，足澡吾渭水之耻矣！"——李陵带五千步兵深入沙漠，最后降了匈奴，功劳还能写进史书；李靖带三千骑兵，深入敌庭，克复定襄，自古没有过这样的，足以洗刷渭水之耻！大意据《资治通鉴·唐纪九》。

颉利退守铁山，假意请降，想拖到春暖草长，再卷土重来。太宗将计就计，派鸿胪卿唐俭去抚慰，又密令李靖进兵。李靖对副将张公谨说："此韩信破齐之计也，唐俭何足惜！"——这是韩信破齐的计策，唐俭一条命算什么！选精骑一万，带二十日粮，趁大雾夜袭阴山。前锋苏定方带二百骑摸到离牙帐七里处，突厥才发觉。颉利单骑逃命，部众溃散。李靖斩首万余，俘男女十余万，杀隋义成公主。颉利逃到沙钵罗部，被大同道行军副总管张宝相生擒，押送长安。——《资治通鉴·唐纪九》。

东突厥，这个称霸草原近半个世纪的庞然大物，就此覆灭。颉利被押到长安，太宗没杀他，赦其死罪，授右卫大将军，安置在长安。太宗对群臣说："昔渭水之盟，朕引为深耻；今雪之矣。"大意据《旧唐书·突厥传》。

贞观四年三月，西北诸蕃震动。四夷君长齐聚长安，诣阙请上太宗为"天可汗"。太宗半开玩笑地问："我为大唐天子，又下行可汗事乎？"群臣与四夷皆呼万岁。此后朝廷以玺书赐西北君长，皆称"天可汗"。——《资治通鉴·唐纪九》。一个中原皇帝，被草原各族尊为共主，这是亘古未有的事。从渭水之盟的屈辱到天可汗的尊号，太宗用了不到四年。这四年，是"示弱"与"蓄势"的活教材。渭水便桥的详情，唐纪专栏侧重正史视角的民族政策另有详述（详见《唐纪·贞观之治·民族政策》），本篇资治通鉴侧重通鉴视角。

## 讲人物

**李世民：忍辱蓄势的帝王**

李世民是李渊的次子，玄武门之变后即位。渭水之盟这出戏，把他的本事显了两样：一是敢赌，二是能忍。敢赌，是他敢带六骑出玄武门，到渭水边跟十几万突厥铁骑对话。这份胆气，不是莽——他身后早布置了诸军列阵，旌甲连绵。他是赌颉利看不穿虚实。草原上的霸主，最怕的就是被看穿底牌；太宗偏偏亮出一个看不懂的牌面，让颉利自己疑神疑鬼。这一手"示之以不可测"，是渭水之盟能成的关键。

能忍，是太宗更大的本事。渭水之盟是耻辱，可太宗忍下了。贞观元年、二年，他每天带兵在显德殿前习射，亲自教士兵弯弓射箭。有人劝他怕有狂徒行刺，他不听，说"王者视四海如一家，何必猜忌自己的将士？"这话听着是大度，实是练兵——三年下来，唐军从一个刚经历内乱的疲弱之师，练成了一支能夜袭阴山、一战灭国的精锐。太宗这份"以耻为砺"的本事，是渭水之盟能雪耻的根本。他不是不怒，是把怒压成磨刀石。一怒就动手的是匹夫，一怒就忍住、忍到能动手再动手的，才是帝王。

太宗还有一样常被忽略的本事——离间。他对突厥属部的分化，从贞观元年就开始。薛延陀、回纥、奚、契丹，一个一个被他拉拢过去。等突厥内乱一起，这些属部反戈一击，颉利就再没有回旋余地。太宗灭东突厥，靠的不只是李靖的兵，更是三年离间的功。这一手"以夷制夷"，太宗用得极熟。后来贞观年间经营西域、安北，多靠这套离间分化的手腕。详见《资治通鉴·唐纪三·贞观之治》。

**颉利可汗：骄盛而败的草原霸主**

颉利可汗是启民可汗的第三子，始毕可汗、处罗可汗的弟弟。突厥汗位继承，向来是兄终弟及——始毕传处罗，处罗传颉利。颉利即位后，立侄子突利为小可汗，分管东部。这个安排，本是草原上的老规矩，可埋下了叔侄相争的祸根。颉利其人，史书说他"勇而无谋，骄而无恩"——大意据《旧唐书·突厥传》。他凭突厥控弦数十万的兵威，自恃天下无敌。

渭水之盟那一场，颉利是赢家——他勒索了一笔金帛，满载而归。可他不知道，这一赢，恰恰输掉了警惕。他以为唐朝怕他，以为太宗不过是个新登基、立足未稳的皇帝。他没料到，太宗把这一回的耻辱记成了磨刀石。颉利回草原后，依旧骄纵，政令苛刻，又连遭天灾，雪深过膝，羊马死损。属部薛延陀、回纥相继叛离，他压不住。等李靖三千骑夜袭定襄，颉利仓皇北逃，一日数惊——那个曾勒索长安的草原霸主，连自己的牙帐都保不住。

颉利的败，不只是天灾，更是人祸。他侄子突利管东部，被属部欺负，颉利不但不帮，还借故治罪，逼得突利降唐。叔侄反目，是突厥内乱的导火索。颉利靠兵威立国，却不懂得"威"要靠"恩"养——一味靠压，属部迟早反。等颉利被擒押送长安，太宗问他："你为什么这么快就败了？"颉利答不上来。一个草原霸主，到死都没想明白自己是怎么输的。大意据《旧唐书·突厥传》。颉利的悲剧，是所有骄盛者的悲剧——看得见自己的强，看不见自己的弱；看得见别人的怕，看不见别人在忍。

**李靖：三千骑灭国的名将**

李靖字药师，京兆三原人，是隋朝名将韩擒虎的外甥。韩擒虎曾夸这个外甥："可与论孙吴之术者，惟斯人矣。"大意据《旧唐书·李靖传》。李靖年轻时被人告发谋反，李渊要杀他，李世民救下他——这一救，救出了唐朝第一名将。李靖后来南平萧铣、东定辅公祏，每到一处都立大功，可他从不揽权，又处处谨慎。

灭东突厥这一仗，是李靖一生的顶点。太宗把先锋给了他，他手里只有三千精骑，却要直捣颉利牙帐。李靖赌的是突厥料定唐军不敢孤军深入。他这一赌，赌赢了。三千骑一夜突至定襄，颉利一日数惊，仓皇北逃。可李靖真正的杀招在后头——颉利假意请降，太宗派唐俭去安抚，又密令李靖进兵。李靖对副将张公谨说："此韩信破齐之计也，唐俭何足惜！"这一句，是把唐俭当弃子——唐俭一条命，换颉利一颗头。李靖选精骑一万，带二十日粮，趁大雾夜袭阴山。前锋苏定方二百骑摸到离牙帐七里处，突厥才发觉。颉利单骑逃命，被擒。

李靖的"狠"，是名将的"狠"——他看得清什么时候该舍一条命换大局。可这份狠，也给李靖惹了祸。灭东突厥后，有人告李靖"纵兵掠物"。太宗大怒，责让李靖。李靖不敢辩。后来太宗缓过来，说"隋将史万岁破突厥，有功不赏反被诛；朕不这样，朕要录公之功，赦公之罪"——大意据《旧唐书·李靖传》。李靖这才明白，功大震主的滋味。此后李靖阖门自守，宾客亲戚一律不见。一个灭国擒王的名将，晚年要靠"自污"保身——这跟当年李渊酗酒自污求安，何其相似。李靖活得比李渊明白，他不是不敢出头，是知道出头就是死。贞观二十三年，李靖病逝，年七十九，善终。这善终，是他"自污"换来的。

## 讲背景

渭水之盟这事，得放在突厥汗国的结构和唐朝初年的虚弱里看。

先看突厥汗国。突厥自六世纪中叶崛起，到隋末唐初，已是草原上最大的霸主。突厥汗国分东西两部——东突厥在漠北漠南，西突厥在西域中亚。东突厥是唐朝北边最大的威胁。突厥的汗位继承，是个老大难——阿史那氏为可汗族，可汗之位向来兄终弟及与父死子继并行，谁强谁上位。启民可汗传始毕，始毕传处罗，处罗传颉利，都是兄终弟及；颉利又立侄子突利为小可汗分管东部。这个安排，看着是分权，实是埋雷——叔侄各掌一部，迟早相争。突厥官制有"设"（或察）、"特勒"、"吐屯"等，分管各地兵马民政。突厥控弦之士数十万，隋末唐初极盛，连李渊起兵都要借突厥兵马（详见《资治通鉴·唐纪一·晋阳起兵》）。可突厥的强盛，全靠兵威和掠夺维持——一旦属部叛离、天灾饥荒，就立刻崩盘。颉利之败，根子在此。

再看唐朝初年的虚弱。武德九年，唐朝刚经历玄武门之变，李世民杀兄屠弟、逼父让位，人心未定。突厥颉利可汗看得清楚，趁乱入寇。那时长安城防空虚，禁军刚经历政变，士气不稳；府兵尚未恢复；北边诸州又被突厥蹂躏。颉利兵临渭水，长安随时可能陷落——这不是危言耸听，是实情。太宗能做的，只有亲出渭水，赌颉利不敢动手。这是玄武门之变留下的烂摊子。玄武门之变是太宗上位的代价，渭水之盟是这笔代价的延续——一个靠政变上来的皇帝，外敌就敢趁虚而入。详见《资治通鉴·唐纪二·玄武门之变》。

再看府兵制与唐初军事。唐初立国靠府兵——兵农合一，平时为农，战时为兵，自备弓刀衣粮。府兵归十二卫统领，折冲府分驻各地，关中集中了大部分折冲府。可玄武门之变后，府兵也受了冲击——禁军刚政变，士气不稳；诸州折冲府尚未整训。太宗贞观元年、二年做的，就是把府兵重新整训成精锐。他每天带兵在显德殿前习射，亲自教士兵弯弓——这一手，是把府兵从疲弱练成精锐。等贞观三年出兵灭东突厥，唐军已是一支能夜袭阴山、一战灭国的铁军。没有这两年的整训，就没有灭东突厥的胜利。府兵制是唐初军事的底子，太宗把这底子重新擦亮，才敢跟突厥决战。

再看唐朝的边防格局。贞观以前，唐朝对突厥基本是守势——李渊起兵时借突厥兵，武德年间也屡次和亲纳贡。突厥视唐朝为附庸，动辄入寇勒索。太宗即位后，决意翻盘。可翻盘不能蛮干，要等时机。突厥连遭天灾、属部叛离，就是时机。太宗对突厥属部的离间分化，从贞观元年就开始——拉拢薛延陀、回纥，孤立颉利。这套"以夷制夷"的策略，是太宗边防的核心。灭东突厥后，太宗不杀颉利，安置在长安，授右卫大将军——这一手"以德服人"，是给草原各族看的。西北诸蕃请上太宗为"天可汗"，正是这一手的效果。从此唐朝的版图从东海到葱岭，唐朝不再附庸于突厥，反成了草原各族共尊的共主。渭水之盟到天可汗，是唐朝边防从守势转攻势的转折点。

## 讲道理

王夫之在《读通鉴论》卷二十里论太宗渭水之盟，话说得透彻。大意据王夫之说，太宗能雪渭水之耻，靠的是两样：一是能忍，二是能等。渭水之盟是忍，贞观三年出兵是等。忍的时候不露锋芒，等的时候养精蓄锐。突厥骄惰不备之时，一举可灭。王夫之最赞的是太宗的"忍"——他说天下之患，多在不能忍；能忍的，往往能成大事。太宗带六骑出渭水，是忍的极致；贞观三年一击灭突厥，是忍的回报。王夫之又说，太宗的"忍"不是怯——怯的人忍了就忘，太宗忍了记成磨刀石。这一层分别，是王夫之论太宗最精微处。

胡三省在《资治通鉴音注》唐纪部分，对突厥盛衰的批注，多落在"夷狄之盛衰有数"上。大意据胡三省注，夷狄之强，靠的是天时人和——天时顺则强，人和聚则盛；天灾一起、内部分裂，立刻崩盘。突厥控弦数十万，盛极一时，可雪灾一起、属部叛离，三年就败。胡三省生在宋元之际，亲见金兵、蒙古兵的盛衰，他对"夷狄之盛衰有数"格外感慨。他的潜台词是：夷狄之强不足畏，畏的是自己不强；自己强了，夷狄之盛不过待衰。太宗能灭突厥，不是突厥弱，是唐朝强；不是突厥偶然败，是突厥必败。胡三省把渭水之盟到灭东突厥，看作"自强"与"待敌之衰"两条线交会的必然结果。

司马光在《资治通鉴·唐纪九》"臣光曰"里对灭东突厥有一段议论。大意据司马光说，太宗能雪渭水之耻，是因为他"屈己蓄势"——肯放下身段，养精蓄锐。司马光最看重的是"忍"——他认为太宗的功劳不在出兵，在出兵前的三年准备。出兵是一时，蓄势是长久。司马光写渭水之盟，是写给宋朝皇帝看的——他要宋帝学太宗忍辱蓄势，不要学宋廷一见夷狄就慌。司马光的史观，一贯是"治乱在人主之德"——太宗能雪耻，是因为他肯修德、肯忍辱；颉利败，是因为他骄惰不修德。德之修与不修，是胜负的分水岭。大意据《资治通鉴·唐纪九》臣光曰。

## 问道悟道

渭水之盟这出戏，藏着两条让人开眼的规律。

**示弱与蓄势的辩证。**

太宗带六骑出渭水，是示弱；贞观三年一击灭东突厥，是蓄势后的反击。这两件事看似相反，实是一体——示弱是蓄势的前奏，蓄势是示弱的收束。没有渭水之盟的示弱，就没有三年喘息的蓄势；没有三年蓄势的底气，也不敢贸然出兵。太宗的"将欲取之，必固与之"，说的是这条规律——要取他，先给他；先示弱，让对方骄惰不备，再一举可灭。

示弱与蓄势这事，历史上有几个对照。春秋时越王勾践败于会稽，向吴王夫差称臣，亲为夫差牵马尝粪，三年忍辱。夫差以为勾践服了，放松警惕。勾践回国后卧薪尝胆，十年生聚，十年教训，最后灭吴。勾践的"尝粪称臣"是示弱，"卧薪尝胆"是蓄势，灭吴是收束。这一套，跟太宗渭水之盟到灭东突厥，是一个模子。《史记·越王勾践世家》。再看汉初，汉高祖刘邦白登之围被匈奴冒顿单于围困七日，靠陈平贿赂阏氏才脱身。脱身后刘邦采纳刘敬的建议，行和亲之策，送宗室女为单于阏氏，岁奉絮缯酒米。这一和亲，忍了六十年。到汉武帝时，汉朝国力蓄足，卫青、霍去病大举击匈奴，封狼居胥，雪白登之围的耻。白登之围是示弱，和亲是蓄势，汉武帝击匈奴是收束。《汉书·匈奴传》。勾践灭吴、汉朝雪白登之耻、太宗灭东突厥，都是同一个"示弱—蓄势—收束"的回路。可见示弱不是怯，是策略；蓄势不是慢，是等待。能不能成，关键看蓄势的功夫做足没有——勾践蓄了二十年，汉武帝蓄了六十年，太宗蓄了三年。蓄得越足，收束越干脆。可也有反例——南宋对金称臣纳贡，本该是示弱蓄势，可宋廷示弱了却忘了蓄势，一味偏安，最后被蒙古所灭。可见示弱只是前奏，蓄势才是关键；只示弱不蓄势，示弱就成了真弱。

**离间瓦解与以夷制夷。**

太宗灭东突厥，靠的不只是李靖的兵，更是三年离间的功。贞观元年起，太宗就拉拢突厥属部薛延陀、回纥、奚、契丹，一个一个分化出去。等突厥内乱一起，雪灾加上属部叛离，颉利就再没回旋余地。这一手"离间瓦解"，是太宗灭东突厥的暗功——明面上是李靖三千骑夜袭定襄，暗地里是三年离间把突厥掏空。

离间瓦解这事，历史上有几个对照。汉武帝击匈奴，也用离间——派张骞通西域，联络大月氏夹击匈奴；又离间匈奴属部。匈奴之衰，一半是汉武帝打的，一半是属部叛离的。《汉书·匈奴传》。唐初唐高祖起兵时，也曾离间突厥——李渊借突厥兵，又防突厥，一边和亲一边分化。可见"以夷制夷"是中原王朝对草原的常策。可这一手也有反噬——唐朝后来扶持回纥制突厥，回纥坐大后，反过来勒索唐朝，安史之乱时借回纥兵平叛，回纥在洛阳大掠。可见离间是一把双刃剑——用得对，瓦解强敌；用过头，养出新敌。太宗灭东突厥用得恰到好处，可唐朝后来扶持薛延陀、回纥，一个一个又成了新患。夷狄之患，灭一个起一个，离间只能解一时，不能解一世。胡三省注里感慨的"夷狄之盛衰有数"，反过来也是"以夷制夷之患有数"——用夷制夷，终有用尽之时。

## 结语

六骑出渭水示弱买和，三年蓄势一击擒颉利——渭水之盟是"示弱蓄势"的活教材，也是"离间瓦解"暗功收束的明证。
