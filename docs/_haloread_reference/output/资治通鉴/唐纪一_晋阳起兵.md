# 资治通鉴·唐纪一·晋阳起兵

## 讲事情

大业十三年，公元 617 年，天下已经乱成一锅粥。杨广躲在江都不肯回来，瓦岗军围困洛阳，窦建德称雄河北，杜伏威横行江淮，各路英雄各自割据。这一年，太原留守李渊五十一岁，奉命镇守北边，防备突厥。

李渊这个人，论出身是顶天的。他是西魏八柱国之一李虎的孙子，袭封唐国公，跟隋室杨家是表亲——独孤皇后是李渊的姨母，杨广是他的表弟。可这份出身，在隋末反倒成了他的累赘。杨广猜忌宗室勋旧，李渊早就被人告过谋反，靠酗酒自污、收受贿赂才躲过一劫。他心里清楚，要么起来干，要么等着被人收拾。

起兵的念头，最先不在李渊，在他二儿子李世民身上。李世民那年十八岁，是个胆大包天的少年。他看到天下大乱，悄悄散财结客，养了一帮敢死之士。他劝父亲起兵，李渊不肯。李世民又去找父亲的亲信裴寂商量。裴寂是晋阳宫副监，管着离宫的人马财帛。李世民知道裴寂跟父亲关系铁，就让裴寂出面。

裴寂想了个法子。他安排晋阳宫里的宫人私下侍奉李渊——晋阳宫是杨广的离宫，宫人是皇帝的人，私侍大臣是死罪。李渊不知就里，喝醉了酒，受了宫人。第二天裴寂来见，把事情挑明，说：二郎已安排好起兵，你如今犯了宫禁，不反也得反了。李渊大惊，起初还要把李世民送官，可事已至此，终于下了决心。大意据《资治通鉴·唐纪一》《旧唐书·裴寂传》。

五月甲子，李渊在晋阳誓师。可起兵第一关就是突厥。突厥那时正强，李渊若带兵南下，后路就被突厥抄了。李渊派刘文静去见始毕可汗，送厚礼，约定打下长安后，土地归李渊，子女玉帛归突厥。始毕可汗大喜，送马千匹、兵两千助阵。借外兵平内乱，这一手不光彩，却是当时的实情。李渊稳住北边，才敢放心南下。

南下第一仗是西河郡。西河郡守不肯降，李渊派大儿子建成、二儿子世民各带一军去攻。两个年轻人怕约束不住部下，跟士兵同吃同住，秋毫无犯。百姓大喜，西河一战而下，往返九日。李渊高兴，说："以此用兵，天下横行也。"大意据《资治通鉴·唐纪一》。

接下来是霍邑。霍邑守将宋老生是隋朝的虎牙郎将，挡在通往关中的路上。李渊到了霍邑下，正赶上连阴雨，粮草不济，又传突厥要抄后路。李渊想退兵，建成、世民劝他：宋老生轻躁，一战可擒，现在退了，人心就散了。李渊不听，下令退军。李世民在帐外大哭，李渊叫他进来问，世民说：现在退兵，前面散了，后面突厥追来，我们就死无葬身之地。李渊这才改主意。天一晴，建成、世民诱宋老生出城，伏兵斩之，霍邑乃下。——《资治通鉴·唐纪一》。

河东守将屈突通挡在黄河边上。李渊不打屈突通，绕过他渡河入关。关中是李渊的根本——他是关陇集团的人，关中的人心向着他。李渊女儿平阳公主早已在鄠县起兵响应，聚众七万，号称"李氏当兴"。李渊渡河后，平阳公主与李世民各统一军，会师长安城下。

十一月，李渊攻入长安。进城之后，李渊做了一件极聪明的事——他没有自己称王称帝，而是立代王杨侑为皇帝，遥尊杨广为太上皇。代王杨侑那时才十二岁，是杨广的孙子，正好在长安。李渊让自己当了"大丞相"，假黄钺，掌天下兵马，实权在握，名分却留给杨家。这一手"尊隋"的戏，演了大半年。

大业十四年，公元 618 年三月，江都兵变，杨广被宇文化及缢死。消息传到长安，李渊"哭之恸"——当然，这哭几分真几分假，谁也说不清。可隋亡的事实摆在眼前，戏不必再演了。五月，李渊逼杨侑禅位，自己称帝，国号唐，建元武德。一个三百年的王朝，就这么开了张。隋亡的细节，详见《资治通鉴·隋纪八·隋亡唐兴》。

## 讲人物

**李渊：能忍能断的唐国公**

李渊字叔德，陇西成纪人，西魏八柱国李虎之孙，唐国公李昞之子。他七岁袭爵唐国公，少年时即受独孤姨母照拂，仕隋历任千牛备身、谯陇二州刺史。杨坚在位时，李渊是皇室姻亲，风光得很。可杨广即位后，宗室勋旧一个一个被打压，李渊的几个堂兄弟都因谗被诛。李渊心里那点不安，从那时就种下了。

李渊最大的本事，是能忍。杨广召他入朝，有人告他谋反，他干脆装病不出，又故意喝酒收贿，把自己弄得一身俗气，让杨广觉得他没出息。这一招"自污以求安"，是他在炀帝朝活下来的命根子。可忍归忍，他不是没主意的人。李世民、裴寂劝他起兵，他先是推托，等晋阳宫的局做实了，他立刻翻脸下决心——这份翻脸的本事，跟"忍"是一体两面。忍的时候能忍到让人看不起，断的时候能断到让人措手不及。

李渊建国后立大儿子建成为太子，二儿子世民为秦王，三儿子元吉为齐王。他把军权大量交给世民，让世民南征北战；又把政务交给建成，让建成留守监国。李渊原以为这样分权，能平衡两个儿子。可他没有想到，军权握在世民手里，世民功高震主，建成不安，元吉助建成——这盘棋从武德元年就埋下了玄武门的祸根。李渊是个能开国的皇帝，却不是个能理家的父亲。详见《资治通鉴·唐纪二·玄武门之变》。

**李世民：十八岁劝父起兵的少年**

李世民是李渊的次子，建成为长，世民为次——《旧唐书·高祖纪》《旧唐书·隐太子建成传》俱载。李世民生于开皇十八年，公元 598 年，晋阳起兵那年十八岁。他少年时即有胆有识，史书说他"玄鉴深远，临机果断"——《旧唐书·太宗本纪》。这些帝王相的套话之外，有一点是实情：李世民是个敢想敢干的人。

晋阳起兵这件事，李渊是名义上的主帅，可真正下决心、推动全过程的，是李世民。他散财结客，结交刘文静、刘弘基、长孙顺德这些豪杰；他劝父亲，被父亲拒绝，又找裴寂商量；他安排晋阳宫的局，逼父亲表态。十八岁的少年，办事之深之细，让裴寂这个老江湖都佩服。起兵之后，西河、霍邑、绛郡，每战他都亲自上阵。霍邑之战，李渊想退兵，李世民帐外大哭劝谏，李渊才改了主意。这一哭，是李渊父子关系里值得细品的一幕——儿子劝老子，老子不听，儿子哭着硬劝，老子才回头。

李世民能打仗，更会用人。他在起兵途中收了尉迟敬德、秦叔宝、程知节这些后来的名将。他对手下推心置腹，部下也肯为他卖命。晋阳起兵时他还年轻，可他那份笼络人心、驾驭豪杰的本事，已经初见端倪。这份本事，后来成就了贞观之治，也酿成了玄武门之变。李世民这一辈子的功过，从他十八岁劝父起兵那天就开始铺展。

**裴寂：推李渊下水的旧交**

裴寂字玄真，蒲州桑泉人。他是李渊的旧交，两人在隋朝就相识，李渊留守太原时，裴寂做晋阳宫副监，两人常常饮酒达旦。裴寂这个人，没有大才，却有一份机变。李世民找他商量起兵，他出的主意是"以宫人私侍李渊"——这一招够狠，把李渊逼到墙角，不反也得反。裴寂的算盘是：事成之后，他是首功。

裴寂后来果然成了唐朝第一任宰相——武德年间他是尚书右仆射，位极人臣。李渊待他极厚，跟他一起吃饭，称他"裴监"而不名。可裴寂到底不是宰相之才。武德年间，他领兵打刘武周，大败；他又在朝中坚持旧法，跟李世民的人马搞不到一块。玄武门之变后，李世民上台，裴寂很快被贬斥。贞观三年，公元 629 年，李世民借故把裴寂流放静州，裴寂死在路上。起兵那天，裴寂把李渊推下水；李渊死后，李世民把裴寂踢出去。一进一出之间，是功臣与帝王难两全的老规矩。大意据《旧唐书·裴寂传》。

**刘文静：出谋借突厥的智囊**

刘文静字肇仁，彭城人，仕隋为晋阳令。他是裴寂的友人，又是李世民最早拉拢的谋士。起兵前后，刘文静做了两件大事：一是替李渊起草檄文、布署起兵事务；二是奉命出使突厥，借兵借马。出使突厥这一趟，是晋阳起兵的生死手——若没有突厥的许诺，李渊南下时后路就被抄了。刘文静带回了始毕可汗的兵两千、马千匹，李渊这才放心南下。这一趟差事办得漂亮，刘文静自认为功不在裴寂之下。

可刘文静的命运比裴寂更惨。武德二年，公元 619 年，刘文静跟裴寂争权，口出怨言，被人告发"谋反"。李渊命裴寂审案，裴寂落井下石，李渊竟将刘文静处死。刘文静临刑前，抚胸长叹："高鸟尽，良弓藏，此言果不谬！"——《旧唐书·刘文静传》。这话本是韩信的话，刘文静借来用在自己身上，分外贴切。起兵首功，转眼成了刀下鬼——功臣之难，古今一例。

## 讲背景

晋阳起兵这事，得放在隋末的群雄格局和关陇集团的老底子里看。

先看群雄格局。大业十三年，公元 617 年，杨广困在江都，天下群雄并起。河南有瓦岗军，李密杀翟让后称魏公，围攻洛阳；河北有窦建德，自称长乐王，割据乐寿；江淮有杜伏威，纵横淮南；陇右有薛举、薛仁果父子，称西楚霸王；凉州有李轨，自称河西大凉王；马邑有刘武周，勾结突厥，自称皇帝。这堆人里，李渊看起来不起眼——他守着太原一隅，兵不算多，地不算大。可李渊有两个别人没有的优势：一是关中的基本盘，二是关陇集团的根基。这两样东西，是他能从群雄里杀出来的本钱。

关陇集团，是理解北周到隋唐三百年政治的钥匙。西魏时，宇文泰据关中，以八柱国、十二大将军为核心，结成一个胡汉合流的军事贵族集团。八柱国里，宇文泰、李虎（李渊的祖父）、独孤信（杨坚的岳父、李渊的外祖父）、杨坚之父杨忠，都在其列。这个集团内部通婚联姻，利益一体——独孤信的长女是北周明帝皇后，四女是李渊的母亲（元贞皇后），七女是隋文帝的独孤皇后。所以李渊跟杨广的关系，不只是表兄弟，更是关陇集团内部通婚的产物。杨坚代北周建隋，是集团内部的权力转移；李渊代隋建唐，也是同一集团内部的换牌。陈寅恪先生论此极精，大意据陈寅恪《唐代政治史述论稿》。这就是为什么李渊入关中，关陇旧勋纷纷归附——他们本是一家人。

府兵制是关陇集团的军事基础。府兵制起源于西魏、北周，特点是兵农合一——平时为农，战时为兵，自备弓刀衣粮。府兵归十二卫统领，折冲府分驻各地，关中集中了大部分折冲府。李渊起兵时，太原附近的府兵是他最初的本钱。入关中后，关中的府兵源源不断地为他补充兵员。府兵制的优势是兵源稳定、不废农时、不耗国帑；它的前提是土地均分（均田制）和兵农合一的户籍管理。唐初百年府兵制运转良好，到武则天以后均田制崩溃，府兵制才跟着瓦解。李渊起兵时，正是府兵制的全盛期。

三省六部，是唐初的政治骨架。三省是中书省（起草诏令）、门下省（审核封驳）、尚书省（执行）。六部归尚书省，分吏、户、礼、兵、刑、工。这套制度隋朝已经设立，李渊建唐沿用，到贞观年间才完备。三省分权的妙处，在于决策、审核、执行三道关互相制衡——诏令不能一人说了算，门下可以封驳。这套制度后来成为贞观之治的政治基础，魏征做谏议大夫时，就在门下省行使封驳之权。三省六部是制度层面，府兵制是军事层面，均田制是经济层面——这三套制度是唐初立国的三根支柱。

唐初分封与宗室也是个背景。李渊建唐后，要分封宗室。他封建成太子，世民秦王，元吉齐王；又封一堆堂兄弟子侄为王。李渊的本意是"以宗室屏藩皇室"，可他没想到，分封越多，宗室相残的可能就越大。封王的有兵有地，互相不服，又都姓李——这种格局，迟早要出事。果然，武德年间建成、世民争权，元吉助建成，演成玄武门之变。后来李世民上台，自己又分封一堆儿子，结果太子承乾、魏王李泰相争，几乎重演一遍。宗室分封这把双刃剑，唐朝从一开始就没解决好。

## 讲道理

王夫之在《读通鉴论》卷二十里论唐高祖李渊，话说得不客气。大意据王夫之说，李渊之能起兵，不是他自己有多大决断，是被儿子李世民和裴寂这些人推着走的。李世民散财结客、谋划起兵，李渊起初不肯，是被晋阳宫的局做实了才被迫表态。可李渊的功劳，在于他能"顺时"——既然被推下水了，他就顺势而为，按部就班：先稳突厥，再南下，入长安立代王，等杨广一死才受禅。这份"顺时"的本事，是李渊的长处。王夫之的高明之处，在于他分得清"决断"与"顺时"——李渊没有决断的勇气，却有用顺时之智，所以能成事。

胡三省在《资治通鉴音注》唐纪部分，对晋阳起兵一事的批注，多落在"借夷狄以平内乱"上。李渊借突厥兵平隋，胡三省看得极清楚——这是开了一个坏头。大意据胡三省注，借夷狄之兵，无论说得多么好听，终究是引狼入室。突厥借给李渊两千兵一千马，李渊答应打下长安后土地归己、子女玉帛归突厥。这买卖不光彩，可李渊做了。胡三省生在宋元之际，亲眼见过金兵、蒙古兵借给中原军阀的事，他对"借夷狄"格外敏感。后来唐朝一直受突厥、回纥、吐蕃之患，安史之乱借回纥兵平叛，回纥在洛阳大掠；唐末借沙陀兵平乱，沙陀最后坐大成了后唐——这条"借夷狄"的链子，从晋阳起兵就开始了。胡三省的注，是把后来几百年的祸患，追溯到这个开头。

司马光在《资治通鉴》里记晋阳起兵，叙事克制，"臣光曰"却点到要害。司马光关心的是"名分"——李渊既受隋朝恩命，又代隋建唐，于君臣之伦有亏。可司马光也承认，隋朝已亡在即，李渊算是"应时而起"。他在"臣光曰"里反复申明的，是"治乱之机，在于人主之德"。李渊能成，是因为隋政已坏，民心思变；李渊若不能修德，唐朝也未必长久。这话后来应验——李渊自己没修出什么治世，倒是儿子李世民修出了贞观之治。司马光写李渊，是把他放在"前车之鉴"与"后事之师"的中间位置上。大意据《资治通鉴·唐纪一》臣光曰。

## 问道悟道

晋阳起兵这出戏，藏着两条让人开眼的规律。

**儿子劝父与时机之择。**

李渊起兵这件事，名义上的主帅是李渊，可真正下决心的是十八岁的李世民。李世民劝父亲起兵，被父亲拒绝；他不放弃，找裴寂商量；他又安排晋阳宫的局，逼父亲表态。霍邑之战，李渊想退兵，李世民帐外大哭，硬把父亲劝回头。从头到尾，李渊是被儿子推着走的——儿子比老子有决断，这是李渊父子关系里的微妙之处。

儿子劝父这事，历史上有几个例子。三国时孙策起兵，也是少年志气。孙策之父孙坚战死襄阳，孙策十七岁，渡江依附袁术，后借兵渡江，平定江东，奠定孙吴基业。孙策比他父亲孙坚更能成事——孙坚是一员猛将，孙策是一个开国之君。儿子胜过父亲的，不止李世民一个。《三国志·孙破虏讨逆传》。北周时，宇文邕（周武帝）劝其堂兄宇文护不要专权，宇文护不听，宇文邕隐忍十二年，最后亲手杀宇文护，亲政后灭北齐，统一北方。宇文邕的隐忍与决断，也是儿子辈胜过长辈的例子。《周书·武帝纪》。可儿子辈的决断，是一把双刃剑——决断用对了，是开国之功；决断用错了，就是骨肉之祸。李世民劝父起兵用对了，可他后来逼父让位（详见《资治通鉴·唐纪二·玄武门之变》），就走到了另一面。时机择得对，是好事；时机择得急，就成祸事。十八岁的少年劝父起兵，是时机；二十七岁的秦王逼父让位，是急切。同一个李世民，前后两副面孔。

**名分之用与禅代之机。**

李渊入长安后，做了一件极聪明的事——他没有自己称王称帝，而是立代王杨侑为皇帝，遥尊杨广为太上皇，自己当"大丞相"。这一手"尊隋"的戏，演了大半年。直到杨广死于江都，李渊才逼杨侑禅位。这一套程序，看着是礼数，实是策略——名分留给杨家，实权攥在自己手里，等时机成熟再受禅代隋。

名分之用这事，历史上早有先例。曹操"挟天子以令诸侯"，迎汉献帝都许，自己当司空、丞相，名分留给汉家，实权握在自己手里。曹操一辈子没称帝，把代汉的事留给了儿子曹丕。《三国志·魏书·武帝纪》。司马氏代魏，也是先掌实权，后受禅让——司马懿、司马师、司马昭三代掌权，到司马炎才受禅建晋。《晋书·宣帝纪》。李渊的做法，跟曹操、司马氏一脉相承：先掌实权，立傀儡，等时机成熟再受禅。这种"名分之用"，是权臣夺位的标准程序。它的妙处在于：名分未失时，旧朝人心可借；名分一让，新朝合法性就有了。可这一套程序，也是有代价的——名分给得越久，旧朝余孽越难清；禅让演得越熟，合法性越虚。曹魏禅让汉家，演了一代；司马氏禅让曹魏，演了三代；南朝禅让演了四回，每一回都更短命。李渊这一回演得还算干净——半年就受禅，可唐朝后来也遭了武则天代唐、安史之乱的报应。名分之用是策略，可策略用多了，就成了惯例；惯例多了，就成了诅咒。曹丕那句"舜禹之事，吾知之矣"——《三国志·魏书·文帝纪》裴注，是说穿了这套把戏的底牌：所谓禅让，不过是装个样子罢了。

## 结语

十八岁少年劝父起兵，五十一岁唐国公顺势受禅——晋阳起兵是关陇集团内部的换牌，也是"儿子劝父、名分之用"两条规律的开场。
