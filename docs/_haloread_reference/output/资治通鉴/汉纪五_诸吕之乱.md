# 资治通鉴·汉纪五·诸吕之乱

## 讲事情

诸吕之乱，是吕后死后汉朝最惊险的一夜。这一夜，决定了刘家的天下是姓刘还是姓吕。

吕后八年，公元前180年七月，吕后病重。吕后掌权八年，把吕家的人安插遍朝野——侄子吕产为相国，居南军；吕禄为上将军，居北军。南军守宫城，北军守京城，长安的兵权，全在吕家手里。吕后又封吕氏为王——吕台为吕王，吕产为梁王，吕禄为赵王，吕通为燕王，违了刘邦当年"非刘氏而王者，天下共击之"的白马之盟。

吕后知道自己一死，功臣集团和刘家诸王必反吕氏。吕后临终，嘱咐吕产、吕禄：我死之后，你们千万不要为我送丧，一定要握住兵权，守住皇宫，别让人困住你们。

八月，吕后死。吕产、吕禄握着南北军，不发丧，暗中谋议。他们要干的，是趁乱诛功臣、灭刘家，把天下改成吕家。

可这事走漏了风声。朱虚侯刘章是刘家宗室，娶的是吕禄的女儿。刘章从妻子那里探到吕氏的密谋，派人密告哥哥齐王刘襄（齐悼惠王刘肥之子），让齐王起兵西进，自己跟弟弟东牟侯刘兴居在长安做内应，里应外合，诛诸吕，立齐王为帝。

齐王刘襄举兵，发兵西攻吕国（济南）。齐王还发檄文，列数诸吕罪状，号召天下共讨。吕产闻讯，派颍阴侯灌婴率兵东击齐。

灌婴走到荥阳，停了。灌婴是跟着刘邦打天下的老将，心里向着刘家。灌婴跟部下商量：诸吕想篡刘氏，我若击齐，等于帮诸吕。不如屯兵荥阳，跟齐王和诸侯联络，等吕氏一动，就联合讨之。灌婴屯兵荥阳，跟齐王讲和，等长安的动静。

长安这边，功臣集团也没闲着。丞相陈平、太尉周勃，早想动手，可手里没兵——南北军在吕氏手里。陈平、周勃商量，想了个法子：曲周侯郦商的儿子郦寄，跟吕禄交好。把郦商抓起来，逼郦寄去骗吕禄。

郦寄去找吕禄，说：高祖定天下，刘氏立九王，吕氏立三王，都是大臣议定的，布告诸侯。如今太后崩，你挂上将军印，不赶紧回赵国就藩，反而留长安握兵，必惹大臣诸侯疑心。你何不交出将印，把兵权还给太尉，请梁王（吕产）交相国印，跟大臣盟约，回封国？这样齐王必退兵，大臣也安心，你高枕做赵王。

吕禄信了郦寄。可吕氏族人意见不一。吕禄的姑姑吕嬃（吕后之妹，樊哙之妻）大怒，骂吕禄：你当将军却要把兵交出去，吕氏要完了！吕嬃把家里的珠玉宝器全扔到堂下，说：我不再为别人守这些东西了！

吕禄犹豫不决，派人去问吕产和吕家老人。有人劝吕产急速入宫控制局面。可吕产、吕禄还是拿不定主意。

八月，平阳侯曹窋（曹参之子）代行御史大夫事，见相国吕产议事。正巧郎中令贾寿从齐地回来，贾寿责备吕产：你不早去封国，现在齐王已反，灌婴屯荥阳，你还想等什么？贾寿劝吕产赶紧入宫。曹窋偷听到，赶紧跑去告诉陈平、周勃。

周勃要进北军，可北军被吕禄控制，进不去。周勃想了个法子——假借皇帝符节。这时襄平侯纪通尚符节，周勃让纪通假传旨意，许周勃入北军。周勃又命郦寄跟典客刘揭去劝吕禄：皇帝让太尉接管北军，你赶紧交印回赵国， 不然大祸临头。

吕禄这下彻底慌了。他相信郦寄，觉得郦寄不会骗自己。吕禄解下上将军印，交给刘揭，刘揭转交周勃。吕禄怅然离去。

周勃进了北军，站在军门前，对将士们喊：为吕氏右袒，为刘氏左袒！——帮吕氏的露出右臂，帮刘氏的露出左臂！

军中皆左袒。全军露出左臂，一片刘家。

周勃这下握住了北军。可南军还在吕产手里。周勃派朱虚侯刘章去协助，又命平阳侯曹窋带兵守住相国府。刘章带兵入宫，在未央宫门前遇上吕产。吕产逃到郎中府，被刘章追上，杀在郎中府厕所里。

吕产一死，诸吕大势已去。周勃派人分头捕杀吕氏，无论老少，一律斩首。吕禄被杀，吕嬃被笞杀，燕王吕通被杀，鲁王张偃（吕后外孙）被废。长安城里，吕氏被诛灭。

少帝刘弘是吕后立的，皇后是吕禄的女儿，功臣们不放心。功臣们商量：少帝和吕氏是一党，留着是后患。可少帝毕竟是名义上的皇帝，不能明杀。功臣们对外宣称：少帝和惠帝的几个儿子，都不是惠帝亲生（是吕后从外面抱来的），不该继承皇位。于是废少帝，另选新帝。

选谁？有人提议齐王刘襄，齐王是高祖长孙，又首举义兵。可大臣们反对——齐王的舅舅驷钧，是个恶人，立齐王，等于又来一个吕氏。又有人提淮南王刘长，刘长年少，母家也恶。最后选中代王刘恒——高祖现存最年长的儿子，母亲薄氏谨良，仁孝闻天下。

功臣们派人迎代王入京。这一迎，迎出了文景之治。

## 讲人物

**周勃：木讷厚重的开国功臣**

周勃是沛人，出身寒微，原本以织薄曲（蚕具）为生，还常给人吹箫办丧事。周勃跟刘邦起兵，打天下，是一员猛将。周勃为人木讷，不善言辞，每次跟儒生议事，老是叫人家"趣为我语"——赶紧直说，别绕弯子。可刘邦临终，吕后问谁能托孤，刘邦说：周勃重厚少文，然安刘氏者必勃也，可令为太尉。——《史记·高祖本纪》。刘邦看人，看到骨子里——周勃话不多，可关键时刻靠得住。

诸吕之乱这一夜，周勃的"重厚少文"成了定海神针。周勃进不了北军，假借符节；进了北军，一句"为吕氏右袒，为刘氏左袒"，把人心试了出来。周勃不是聪明人，可周勃是厚道人。厚道人关键时刻不慌，不动摇。刘邦说"安刘氏者必勃"，一语成谶。

周勃后来为太尉、为丞相，可他不善文治。文帝时周勃罢相归国，有人告周勃谋反，周勃下狱，吓得拿千金贿赂，又靠薄太后（文帝之母）说情才免。周勃出狱叹：吾尝将百万军，然安知狱吏之贵乎！——我曾率百万大军，哪知道狱吏这么威风！——《史记·绛侯周勃世家》。周勃这一叹，是功臣的悲凉——打天下的功臣，在太平年月，反被狱吏拿捏。

**陈平：六出奇计的智囊**

陈平是阳武人，跟周勃不同，陈平是智囊。陈平跟刘邦，"六出奇计"，可这六计，陈平自己说都是阴谋，见不得光。白登之围的奇计是一计，诸吕之乱的谋划也是一计。

吕后当政时，陈平当丞相，整天喝酒玩乐，不理政事。吕后死后，陈平跟陆贾商量。陆贾说：天下安，注意相；天下危，注意将。将相和调，则士务附；士务附，则天下虽有变，即权不分。你跟太尉周勃搞好关系，把兵权跟相权拧成一股，诸吕就翻不了天。陈平听了，用陆贾的计，跟周勃结交，两人合力，才有了诛诸吕这一夜。

陈平的好处，是能忍。吕后封吕氏为王，陈平不反对，不吭声，等时机。陆贾劝他结交周勃，他照办。陈平的"事成"之谋，是把功夫下在平时——平时把将相关系理顺，关键时刻才能一击成功。陈平后来对人说：我多阴谋，是道家所忌，我这一辈子，怕是要败在阴祸上。可陈平的阴谋，救了刘氏的天下。

陈平的明白，还在于他知道自己的计谋是双刃剑。阴谋能成事，也能损德。陈平的后人，果然没落——陈平曾孙陈何因抢人妻被弃市，封国被除。陈平的预言，应验了。

**吕产吕禄：守不住的权柄**

吕产、吕禄是吕家的核心。吕后把南军给吕产，北军给吕禄，是把长安的命脉交给这俩侄子。可吕产、吕禄，守不住。

吕禄的毛病，是轻信。郦寄来劝他交兵权，他信了。姑姑吕嬃骂他，他不听。吕禄信郦寄，是因为郦寄是他朋友。可朋友的话，在权力面前，不一定靠得住。吕禄交出兵权的那一刻，吕氏就完了。吕嬃摔珠玉，是看明白了——吕禄一交兵，吕氏死定了。

吕产的毛病，是迟疑。贾寿劝他赶紧入宫控制局面，他还在犹豫。等他反应过来要入宫，周勃已经握住北军，刘章已经带兵杀来。吕产逃到郎中府，死在厕所里。吕产要是早一步入宫，占住未央宫，跟吕禄南北呼应，鹿死谁手，还未可知。可吕产迟疑，错过了最后的机会。

吕产、吕禄的失败，不是没兵没权，是没胆没断。吕后给他们兵权，没给他们胆识。权柄在握，不敢用；时机到了，不敢断。这样的权臣，守不住权，也保不住命。

**刘章：刘家的虎子**

朱虚侯刘章，是齐悼惠王刘肥的次子，娶吕禄的女儿。刘章年轻，却是刘家宗室里最猛的一个。吕后当政时，一次宴饮，吕后让刘章当酒吏。刘章说：臣是将门之子，请以军法行酒。吕后允了。

酒过三巡，刘章唱《耕田歌》：深耕穊种，立苗欲疏；非其种者，鉏而去之。——深耕密种，立苗要疏；不是这品种的，锄掉！刘章这是借种苗讽诸吕——不是刘家的"种"，都该锄掉。吕后听了，默然。

席间有个吕氏族人喝醉逃席，刘章追上去，一剑斩了，回报吕后：有亡酒一人，臣谨行法斩之。——有个人逃酒，臣按军法斩了。吕后大惊，可事先允了军法，不好发作。从此诸吕都怕刘章。

诸吕之乱这一夜，刘章是杀吕产的人。刘章带兵入宫，追杀吕产于郎中府厕所。刘章的勇，是刘家宗室里少见的勇。功臣们本来想立齐王刘襄，可刘章这一夜的功劳太大，大臣们反倒忌惮——刘章太猛，立了齐王，刘章兄弟势大，又是一个吕氏。所以最后立了温和的代王刘恒。刘章的勇，救了刘氏，却也断送了哥哥齐王的帝位。

## 讲背景

诸吕之乱这事，得放在吕后称制八年的格局里看。

先说吕后为什么能称制。吕后名吕雉，是刘邦的结发妻。刘邦打天下，吕后吃尽苦头——被项羽俘虏两年，险些被烹；刘邦宠戚夫人，想废吕后的儿子刘盈（惠帝），立戚夫人的儿子刘如意。吕后靠张良、商山四皓保住刘盈的太子位。刘邦死后，吕后报复戚夫人，砍手砍脚，弄成"人彘"，还请惠帝去看。惠帝看了，大哭，说：这不是人干的事！我是你的儿子，我治不了天下了。从此惠帝不理朝政，日夜饮酒作乐，七年后病死。

惠帝死后，吕后立少帝，自己称制——以太后身份临朝，行皇帝权力。吕后称制八年，是汉朝皇权最集中的八年。吕后的狠，让功臣集团不敢动； 吕后的权术，让刘家诸王不敢反。可吕后一死，压制的力量没了，矛盾全爆发。

再说吕后为什么封吕氏为王。刘邦有白马之盟——非刘氏而王者，天下共击之。吕后封吕氏为王，是违盟。可吕后有她的算盘——刘家诸王跟吕后没血缘，吕氏跟吕后有血缘。吕后要保吕家，就得让吕氏掌权掌兵。吕后封吕氏为王，是为身后铺路。可吕后没想到，她一死，吕氏根本守不住。吕后给吕氏权，没给吕氏威望——吕氏的权，是吕后的权；吕后没了，吕氏的权就是空中楼阁。

再说功臣集团的力量。汉初的功臣集团，是跟着刘邦打天下的人——萧何、曹参、张良、陈平、周勃、灌婴、樊哙、夏侯婴。这些人，跟刘邦有生死之交，跟刘家有君臣之义。吕后当政，功臣集团忍着；吕后一死，功臣集团就动了。功臣集团的力量，在于他们手里有兵有将有威望。灌婴屯兵荥阳，周勃握北军，陈平掌相权——这三股力量合起来，吕氏根本不是对手。

再说南北军的格局。南军守宫城，北军守京城，是长安的两支禁军。吕后把南军给吕产，北军给吕禄，是把长安的命脉交给吕氏。可南北军的将士，多是功臣集团的旧部。周勃一句"为吕氏右袒，为刘氏左袒"，全军左袒——将士们心里向着刘家，不向着吕家。吕氏握着南北军的印，没握住南北军的心。这是吕氏的致命伤——权在手里，心不在。

再说为什么立代王刘恒。功臣们选新帝，有三条标准：一是高祖血脉，二是母家不恶，三是本人仁孝。齐王刘襄是高祖长孙，首举义兵，可母家驷钧恶；淮南王刘长年少，母家也恶；代王刘恒是高祖现存最年长的儿子，母亲薄氏谨良，本人仁孝闻天下。功臣们选刘恒，是选一个不会报复功臣、不会重用外戚的皇帝。功臣们的算盘，是保自己的身家——立个温和的皇帝，功臣集团才能安稳。这一选，选出了文景之治，是功臣们没想到的意外之喜。

## 讲道理

司马光在《资治通鉴》里对诸吕之乱有议论。大意据臣光曰，吕后之贵，由于高祖；吕氏之祸，由于吕后。吕后欲以吕氏代刘氏，是背高祖之盟，逆天下之心。然吕氏之败，非独天意，亦人谋也。周勃、陈平之谋，灌婴之兵，刘章之勇，合而诛之。司马光看诸吕，看的是"逆天意、失人心"——吕后违盟，是逆；吕氏无威，是失人心。逆天失人，败是必然。

王夫之在《读通鉴论》里论周勃之功。大意据王夫之说，周勃之安刘，非独一军之左袒也。周勃能入北军，在于陈平之谋；能杀吕产，在于刘章之勇；能成大事，在于灌婴之屯荥阳。诸吕之诛，是将相和、内外合之功， 非一人之力。王夫之看周勃，看的是"合力"——安刘氏不是周勃一个人的功劳，是功臣集团合力的功劳。王夫之也指出，周勃的功劳，在于他能容人、能用刘章，不争功，才成大事。

班固在《汉书·高后纪》赞里论吕后。大意据班固说，吕后称制，政不出房户，天下晏然，刑罚罕用，民务稼穑，衣食滋殖。可吕后封吕氏为王，违高祖之盟，终致吕氏之祸。班固看吕后，看的是两面——治国有方，弄权招祸。班固比司马迁更客观，既肯定吕后的治国，也批评吕后的弄权。

## 问道悟道

诸吕之乱这出戏，藏着两条让人脊背发凉的规律。

**权在手里，心不在，权是空的。**

吕产握南军，吕禄握北军，长安的兵权全在吕氏手里。可周勃一句"为吕氏右袒，为刘氏左袒"，全军左袒。将士们心里向着刘家，不向着吕家。吕氏的兵权，是吕后给的；吕后没了，兵权就是空的。

权在手里心不在这事，历史上不止一桩。东晋时，桓温专权，想篡位，可王谢世家大族不买账。桓温病重，谢安、王坦之拖延禅让诏书，桓温等不到诏书就死了。——《晋书·桓温传》。桓温握着兵权，可世族的心不在他那边，篡位篡不成。权，不只是印信兵符，更是人心。印信兵符能调兵，人心才能让兵真为你卖命。

**朋友的话，在权力面前，不一定靠得住。**

吕禄交兵权，是信了朋友郦寄。郦寄劝他交权，他信了；姑姑吕嬃骂他，他不信。吕禄信朋友，不信亲人。可郦寄的话，是陈平、周勃逼郦寄说的——郦寄的父亲郦商被功臣集团抓了，郦寄不得不来骗吕禄。吕禄信的朋友，是被人拿刀架在脖子上来骗他的。

朋友的话在权力面前靠不住这事，历史上也多。战国时，孙膑跟庞涓是同学，庞涓请孙膑到魏国，却陷害孙膑，挖了孙膑的膝盖骨。——《史记·孙子吴起列传》。孙膑信朋友庞涓，差点送命。权力面前，朋友可能变敌人，亲人的话反倒更实在。吕嬃骂吕禄，是真为吕家着急；郦寄劝吕禄，是被人胁迫。吕禄信错人，丢了命，也丢了吕家。

## 结语

一夜左袒，袒出了刘家的天下，也袒出了功臣集团的分量。
