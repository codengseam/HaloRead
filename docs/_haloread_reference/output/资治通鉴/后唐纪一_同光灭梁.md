# 资治通鉴·后唐纪一·同光灭梁

## 讲事情

李存勖灭后梁建后唐，是五代最辉煌的一刻，也是五代最速亡的一刻。庄宗李存勖马上得天下，三年便失天下——宠伶人、杀功臣，最后被伶人郭从谦弑于兴教门。养兄李嗣源入洛阳即位，是为后唐明宗。同光灭梁与庄宗亡国，是"马上得天下不能马上治天下"的典型。

龙德三年（即同光元年），公元923年四月，李存勖在魏州（今河北大名）称帝，国号唐（后唐），建元同光，是为后唐庄宗。大意据《资治通鉴·后唐纪一》《旧五代史·唐书·庄宗纪》。李存勖是李克用亲子，沙陀族，承父业与后梁争霸十二年（梁晋争霸详见《资治通鉴·后梁纪五·梁晋争霸》主场）。称帝后，李存勖旋即大举南下。这年十月，后唐军入汴梁，后梁亡——后梁末帝朱友贞自尽，后梁三世十六年而亡。同光元年十月入汴，是后唐建国的标志性事件。李存勖以沙陀骑兵灭中原王朝，是五代第一个以少数民族政权入主中原的先例。

灭梁之后，李存勖迁都洛阳。后唐以洛阳为都，沿用唐之宫殿。李存勖自以为复兴唐室，可他并非李唐宗室——沙陀李氏赐姓李，与唐室并无血缘。李存勖建国号"唐"，是借唐之名号令天下，是其政治合法性的人为建构。可这份合法性，本就根基不固——沙陀李氏非李唐后裔，复兴唐室只是名号，不是实质。这为后唐的速亡埋下了隐患。

李存勖灭梁后，志得意满，渐荒于政事。庄宗宠伶人——伶人就是优伶、戏子，李存勖好戏曲，常自傅粉墨与伶人共戏于庭，自号"李天下"。大意据《资治通鉴·后唐纪一》《新五代史·伶官传》。伶人得宠，纷纷干政——景进、史彦琼、郭从谦等伶官，借庄宗之宠把持朝政。伶人史彦琼为武德使，掌宫廷宿卫；伶人景进常为庄宗耳目，访察朝臣；伶人郭从谦（艺名郭门高）被任为从马直指挥使，掌禁军。大意据《新五代史·伶官传》。伶人本是优伶戏子，借庄宗之宠得掌军政，是后唐朝政的奇观——伶官乱政，自此开端。庄宗以马上得天下，复以伶人治天下，是"马上得天下不能马上治天下"的典型。

同光三年，公元925年，李存勖命皇子李继岌与枢密使郭崇韬率军灭前蜀。郭崇韬是后唐谋臣，灭梁功臣，多谋善断。郭崇韬率军入蜀，前蜀主王衍投降，前蜀亡。大意据《资治通鉴·后唐纪一》《旧五代史·唐书·郭崇韬传》。可郭崇韬灭蜀后，被庄宗与刘皇后冤杀。刘皇后是庄宗正妻，与伶人勾结，素忌郭崇韬。刘皇后矫诏，命皇子李继岌杀郭崇韬于蜀中。大意据《旧五代史·唐书·郭崇韬传》。郭崇韬之死，是后唐自毁柱石——灭梁功臣、灭蜀功臣，一朝被冤杀，朝臣寒心，将帅离心。郭崇韬之死，是庄宗亡国的伏笔。

同光四年（即天成元年），公元926年二月，贝州（今河北清河）军士皇甫晖等哗变，推赵在礼为首，据魏州（贝州兵变亦称魏州兵变）。大意据《资治通鉴·后唐纪一》《旧五代史·唐书·庄宗纪》。庄宗遣养兄李嗣源讨之。李嗣源是李克用养子（非亲子，详见下文讲人物），与李存勖为养兄弟，时任蕃汉内外马步军总管，是后唐宿将。可李嗣源率军至魏州，其部下亦叛，拥李嗣源为主。李嗣源本无反意，可部下拥立，箭在弦上，不得不发。大意据《资治通鉴·后唐纪一》。李嗣源遂率军南下，直指洛阳。

公元926年四月，李嗣源军逼近洛阳，洛阳城中大乱。伶人郭从谦率从马直禁军哗变，攻兴教门。庄宗闻变，率亲卫抵抗，中流矢。庄宗以优伶出身、借宠掌禁军的郭从谦，正是他一手提拔的——可郭从谦反弑其主。大意据《资治通鉴·后唐纪一》《新五代史·伶官传》。庄宗李存勖死于兴教门之变，年四十二岁。在位不过三年（923-926），灭梁的雄主，最后死于自己所宠的伶人之手。马上得天下，复以伶人失天下——庄宗亡国，是"以暴易暴、以佞自毁"的典型。

庄宗既死，李嗣源入洛阳，即皇帝位，是为后唐明宗，改元天成。大意据《资治通鉴·后唐纪一》《旧五代史·唐书·明宗纪》。李嗣源是李克用养子，以养兄继养弟之位（详见《资治通鉴·后唐纪七·明宗之治》论养子继统与明宗之治）。后唐自此进入明宗一朝——明宗李嗣源改革，五年小康，是五代少有的治世。可后唐亦不过十四年（923-936），明宗之后，养子李从珂夺位，石敬瑭借兵契丹，后唐亡（详见《资治通鉴·后唐纪七·明宗之治》论后唐之亡）。

## 讲人物

**李存勖：灭梁建唐却因伶人亡国的庄宗**

李存勖是李克用亲子，沙陀族，骁勇善战。李克用临终以三矢赐李存勖，命其报三仇——一讨刘仁恭，二讨契丹，三灭朱温（详见《资治通鉴·后梁纪五·梁晋争霸》论李存勖三矢之愿）。李存勖藏矢于庙，每战前请矢以行。承父业，先平刘守光（刘仁恭之子），再破契丹，最后灭后梁——三矢之愿，一一兑现。923年四月称帝建后唐，十月入汴灭梁，是五代最辉煌的一刻。

可李存勖灭梁后，志得意满，渐荒于政事。庄宗宠伶人，自傅粉墨与伶人共戏于庭，自号"李天下"。大意据《新五代史·伶官传》。伶人得宠干政，景进、史彦琼、郭从谦等借宠把持朝政。庄宗以马上得天下，复以伶人治天下——这份"以马上得天下、以伶人治天下"的悖论，是庄宗亡国的根子。926年四月，伶人郭从谦弑庄宗于兴教门。庄宗在位不过三年，灭梁的雄主，最后死于自己所宠的伶人之手。李存勖是五代"骤兴骤亡"的典型——能灭梁，不能守梁；能得天下，不能治天下。

**郭崇韬：灭梁灭蜀却被冤杀的谋臣**

郭崇韬是后唐谋臣，灭梁功臣，多谋善断。郭崇韬本是李克用旧部，历事李克用、李存勖父子。灭梁之役，郭崇韬多献奇策，是庄宗的首席谋臣。大意据《旧五代史·唐书·郭崇韬传》。同光三年（925），郭崇韬率军灭前蜀，前蜀主王衍投降，前蜀亡。郭崇韬灭蜀，是后唐武功的巅峰。

可郭崇韬灭蜀后，被庄宗与刘皇后冤杀。刘皇后是庄宗正妻，与伶人勾结，素忌郭崇韬。刘皇后矫诏，命皇子李继岌杀郭崇韬于蜀中。大意据《旧五代史·唐书·郭崇韬传》。郭崇韬之死，是后唐自毁柱石——灭梁功臣、灭蜀功臣，一朝被冤杀，朝臣寒心，将帅离心。郭崇韬的悲剧，是"功臣自毁"的典型——功高震主，必然见疑；功臣不自全，必然自毁。郭崇韬之死，是庄宗亡国的伏笔——杀功臣则将帅离心，将帅离心则兵变频发，兵变频发则亡国。

**李嗣源：以养兄继位的后唐明宗**

李嗣源是李克用养子（非亲子），沙陀人，本名邈佶烈，李克用养以为子，赐姓李。大意据《旧五代史·唐书·明宗纪》《新五代史·唐本纪》。李嗣源与李存勖为养兄弟，年长于李存勖。李嗣源是后唐宿将，历事李克用、李存勖，蕃汉内外马步军总管，掌后唐兵权。李嗣源本无反意，可926年魏州兵变，李嗣源奉命讨之，其部下亦叛，拥李嗣源为主。李嗣源箭在弦上，不得不发，率军南下洛阳。

庄宗兴教门之变死后，李嗣源入洛阳即位，是为后唐明宗，改元天成。李嗣源是李克用养子，以养兄继养弟之位——这是养子继统的又一典型（养子继统的合法性困境详见《资治通鉴·后唐纪七·明宗之治》）。李嗣源不识字，可他能听谏、能用人——明宗一朝，任用冯道、任圜等，五年小康，是五代少有的治世。李嗣源是五代少有的明君，可他的继位本身，已是养子继统的产物——后唐的合法性问题，始终未解。

## 讲背景

同光灭梁与庄宗亡国这事，得放在沙陀族入主中原与少数民族政权、伶官与宦官乱政、养子继承制深化、以及禁军与兵变四重背景里看。

先看沙陀族入主中原与少数民族政权。沙陀是西突厥别部，本居北庭，后内迁代北。沙陀骑兵骁勇，是唐末五代最精锐的军事力量。李克用率沙陀骑兵入援平黄巢（详见《资治通鉴·唐纪十二·黄巢起义与朱温篡唐》），自此沙陀成为中原争霸的一强。李存勖以沙陀骑兵灭后梁，建后唐——这是五代第一个以少数民族政权入主中原的先例。沙陀李氏建后唐、后晋（石敬瑭亦是沙陀）、后汉（刘知远亦是沙陀），三代皆沙陀——五代后半叶，是沙陀族主政中原的时期。可沙陀是少数民族，入主中原后，面临合法性建构的难题——李存勖建国号"唐"，是借唐之名号令天下，可他并非李唐宗室，这份合法性根基不固。少数民族政权入主中原，如何建构合法性，是五代的根本问题。

再看伶官与宦官乱政。庄宗宠伶人，伶人得宠干政——景进、史彦琼、郭从谦等伶官，借庄宗之宠把持朝政。伶人本是优伶戏子，借庄宗之宠得掌军政，是后唐朝政的奇观。伶官乱政，与唐末宦官专权，是同构的——都是君主亲信近幸，近幸借宠干政，最后反噬君主。唐末宦官专权延续百年（详见《资治通鉴·唐纪十一·元和中兴》论宦官弑君与家奴之祸），后唐伶官乱政不过数年，可结局相同——都是近幸反噬君主。可见君主亲信近幸，无论近幸是宦官、是伶人、是外戚，最后都会反噬君主。伶官乱政，是庄宗亡国的直接原因——郭从谦正是伶官出身，被庄宗提拔掌禁军，最后反弑其主。

再看养子继承制深化。李嗣源是李克用养子，以养兄继养弟之位——这是养子继统的深化。五代养子继承，是沙陀与中原武夫集团的共同现象（详见《资治通鉴·后梁纪一·朱温代唐》论养子继承制萌芽）。朱温欲立养子友文，激发亲子友珪弑父；李嗣源以养兄继位，又为后唐末年养子李从珂夺位埋下隐患（详见《资治通鉴·后唐纪七·明宗之治》论养子继统的合法性困境）。养子继承，破坏了嫡长子继承的稳定性，埋下了养子与亲子争位的内乱隐患。后唐两代以养子继位（李嗣源养于李克用，李从珂养于李嗣源），养子继统的合法性问题，始终未解。

再看禁军与兵变。庄宗之亡，亡于禁军哗变——伶人郭从谦率从马直禁军哗变，弑庄宗于兴教门。禁军是庄宗的亲军，可禁军反噬其主，是后唐的奇耻。禁军反噬，与魏博兵变降晋（详见《资治通鉴·后梁纪五·梁晋争霸》论藩镇倒戈），是同构的——都是兵权反噬主君。可见兵权是双刃剑——今日为你所用，明日反噬其主；禁军是皇帝的亲军，可禁军反噬，是亡国最直接的方式。五代之亡，多亡于禁军哗变——后梁亡于段凝拥兵不救，后唐庄宗亡于禁军哗变，后唐末帝亡于石敬瑭借兵（详见《资治通鉴·后唐纪七·明宗之治》论后唐之亡）。可见禁军与兵变，是五代王朝速亡的催化剂。

## 讲道理

王夫之在《读通鉴论》卷二十九（五代中）里论后唐，话说得极痛。大意据王夫之说，李存勖、石敬瑭、刘知远是沙陀三部的小夷，"窃据唐之京邑"，不足称天子。王夫之论李存勖灭梁，大意据王夫之说，李存勖以沙陀骑兵灭后梁，是"夷盗窃而不可纵"——沙陀以少数民族入主中原，本无君天下之志，亦无立国之规模。王夫之论庄宗宠伶人而亡国，大意据王夫之说，庄宗以马上得天下，复以伶人治天下，是"淫虐猥贱，不复有生人之理"——能灭梁不能守梁，能得天下不能治天下。王夫之把李存勖放在"骤兴骤亡、以佞自毁"的位置上——灭梁是武功的巅峰，宠伶人是政治的深渊；以佞自毁，必然速亡。大意据《读通鉴论》卷二十九。

胡三省在《资治通鉴音注》后唐纪部分，对庄宗宠伶人与郭崇韬之死的批注，多落在"近幸乱政、功臣自毁"上。大意据胡三省注，伶官乱政与唐末宦官专权，是同构的——都是君主亲信近幸，近幸借宠干政，最后反噬君主。胡三省注郭崇韬之死，是带着痛惜的——郭崇韬灭梁灭蜀，功高震主，必然见疑；庄宗听信刘皇后与伶人谗言，冤杀郭崇韬，是自毁柱石。胡三省的潜台词是：庄宗之亡，亡于自毁——杀功臣则将帅离心，宠伶人则朝政乱，二者交织，庄宗焉得不亡。大意据《资治通鉴音注》后唐纪部分。

司马光在《资治通鉴·后唐纪一》里对同光灭梁与庄宗亡国的记述，亦带其一贯史观。大意据司马光说，李存勖灭梁，是武功的巅峰；可灭梁后志得意满，宠伶人、杀功臣，是政治的深渊。司马光的史观，是"治乱在人主之德"——庄宗能灭梁，是武功之盛；庄宗亡国，是德之不足。能灭梁者，未必能守梁；能得天下者，未必能治天下——治天下在人主之德，不在武功之盛。大意据《资治通鉴·后唐纪一》。欧阳修在《新五代史·伶官传序》里亦有一段名言，大意据欧阳修说，"忧劳可以兴国，逸豫可以亡身"——忧劳能使国家兴盛，逸豫能使自身灭亡；"祸患常积于忽微，而智勇多困于所溺"——祸患常积累于微小之处，而智勇之人多困于自己所溺爱的事物。大意据《新五代史·伶官传序》。欧阳修的话，是对庄宗亡国最精辟的总结。

## 问道悟道

同光灭梁与庄宗亡国这出戏，藏着两条让人开眼的规律。

**伶官乱政与马上得天下的悖论。**

李存勖以沙陀骑兵灭后梁，建后唐——是五代最辉煌的一刻。可灭梁后，庄宗宠伶人，自傅粉墨与伶人共戏于庭，自号"李天下"。伶人得宠干政，景进、史彦琼、郭从谦等借宠把持朝政。郭从谦被任为从马直指挥使，掌禁军——伶人掌禁军，是后唐朝政的奇观。926年四月，伶人郭从谦率禁军哗变，弑庄宗于兴教门。庄宗以马上得天下，复以伶人失天下——这是"马上得天下不能马上治天下"的典型，也是"伶官乱政"的极致。

伶官乱政与马上得天下的悖论这事，历史上有几个对照。唐末宦官专权，宦官掌神策军，废立皇帝——可宦官之祸延续百年，比伶官乱政久远。大意据《旧唐书·宦者传》。西汉末王莽，以外戚权臣篡汉——可王莽是权臣，非近幸，且王莽建新后仍试图治国。大意据《汉书·王莽传》。东汉末十常侍乱政，宦官专权——可十常侍之乱，是宦官专权的延续，非伶官。大意据《后汉书·宦者传》。后唐庄宗宠伶人，与唐末宦官、东汉十常侍，是同构的——都是君主亲信近幸，近幸借宠干政，最后反噬君主。可见君主亲信近幸，无论近幸是宦官、是伶人、是外戚，最后都会反噬君主。庄宗之亡，亡于自宠——以佞自毁，是亡国最直接的方式。马上得天下靠的是兵权，治天下靠的是用人；用伶人则朝政乱，用贤臣则朝政清。庄宗选了伶人，弃了贤臣——后唐之速亡，亡在庄宗的以佞自毁。

**兵变弑君与禁军反噬。**

庄宗之亡，亡于禁军哗变——伶人郭从谦率从马直禁军哗变，弑庄宗于兴教门。禁军是庄宗的亲军，可禁军反噬其主。郭从谦本是伶人，被庄宗提拔掌禁军——这份提拔，是庄宗自掘坟墓。禁军反噬，与魏博兵变降晋（详见《资治通鉴·后梁纪五·梁晋争霸》论藩镇倒戈），是同构的——都是兵权反噬主君。可见兵权是双刃剑——今日为你所用，明日反噬其主；禁军是皇帝的亲军，可禁军反噬，是亡国最直接的方式。

兵变弑君与禁军反噬这事，历史上有几个对照。唐末，神策军由宦官世掌，废立皇帝——可神策军是宦官控制的工具，非禁军自发的反噬。大意据《旧唐书·宦者传》。隋末，宇文化及率禁军哗变，弑隋炀帝于江都——可隋炀帝之弑，是禁军哗变的典型，与庄宗之亡同构。大意据《隋书·炀帝纪》。后梁，朱友珪率控鹤指挥使禁军弑朱温——可朱友珪是皇子借禁军弑父，非禁军自发的反噬（详见《资治通鉴·后梁纪一·朱温代唐》论弑父篡位）。后唐庄宗、隋炀帝，都是禁军哗变弑君的典型——禁军本是皇帝的亲军，可禁军一旦哗变，皇帝便首当其冲。可见禁军反噬，是亡国最直接的方式——禁军近在咫尺，一旦反噬，皇帝无处可逃。要防禁军反噬，须以德服禁军、以恩结禁军——可庄宗宠伶人、杀功臣，禁军离心，焉得不反噬。兵权是双刃剑，今日弑君的刀，明日就反噬到自己头上——这是五代的铁律。

## 结语

同光灭梁是五代最辉煌的一刻，庄宗亡国是五代最速亡的一刻——马上得天下不能马上治天下，宠伶人杀功臣则朝政乱，伶官反弑其主则国亡；以佞自毁，以兵变亡，三年而失天下。
