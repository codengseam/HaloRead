# 资治通鉴·后唐纪七·明宗之治

## 讲事情

后唐明宗李嗣源，是五代少有的明君。他不识字，可他能听谏、能用人，五年小康，是五代乱世的喘息。可明宗一死，养子李从珂夺位，石敬瑭借兵契丹割燕云十六州，后唐亡——十四年的短命王朝，就此收场。明宗之治与后唐之亡，是"养子继统的合法性困境"与"割地借兵的代价"的双重悲剧。

李嗣源入洛阳即位后，是为后唐明宗，改元天成（926起，详见《资治通鉴·后唐纪一·同光灭梁》论李嗣源以养兄继位）。李嗣源是李克用养子（非亲子），沙陀人，本名邈佶烈，李克用养以为子，赐姓李。大意据《旧五代史·唐书·明宗纪》《新五代史·唐本纪》。李嗣源即位时已六十岁，年事已高，可他能听谏、能用人。李嗣源不识字，每阅奏章，须人诵读——可他常能于诵读中察觉是非，从谏如流。这是武夫难得的本事——多数武夫不识字，亦不听谏；李嗣源不识字，却能听谏，是五代少有的明君。

明宗一朝，任用冯道、任圜等为相。冯道是五代名臣，历事四朝十帝（后唐、后晋、后汉、后周），号称"长乐老"。任圜是后唐谋臣，明宗用其为相，整治朝政。大意据《旧五代史·冯道传》《旧五代史·任圜传》。明宗改革，罢庄宗之弊政——罢伶官之权，汰冗员，减赋税，恤民困。天成、长兴间（926-933），中原粗安，史称"明宗小康"——这是五代乱世少有的喘息。大意据《资治通鉴·后唐纪七》。明宗曾夜焚香祝天，说"某蕃部之人，因乱为众所推，愿天早生圣人，为生民主"——我是蕃部之人，因乱被众人推举，愿上天早生圣人，做百姓之主。大意据《旧五代史·唐书·明宗纪》。这话，是明宗的自知之明——他知道自己非天命所归，只愿天下早有真主。这份自知，是五代君主少有的清醒。

可明宗晚年，亦有隐忧。明宗长子李从璟（原名李从审）早卒，次子秦王李从荣骄横。长兴四年，公元933年，明宗病重。李从荣见父病危，误以为父已死，率兵入宫，欲夺位。大意据《资治通鉴·后唐纪七》《旧五代史·唐书·秦王从荣传》。明宗闻变，命禁军讨之，李从荣兵败被杀。明宗见亲子谋反，悲愤交加，病势加剧。十一月，明宗崩，年六十七岁。大意据《旧五代史·唐书·明宗纪》。明宗在位七年（926-933），是五代少有的治世，可他的晚年，被亲子谋反的悲剧笼罩——秦王李从荣之乱，是后唐由治转乱的转折点。

明宗崩后，第三子李从厚即位，是为后唐闵帝。大意据《资治通鉴·后唐纪七》《旧五代史·唐书·闵帝纪》。李从厚年幼（即位时年仅二十），柔弱不能制。可后唐的养子——李从珂，时任凤翔节度使，见闵帝柔弱，遂生异心。李从珂是李嗣源养子（非亲子），本姓王，平山人，李嗣源养以为子，赐姓李。大意据《旧五代史·唐书·末帝纪》《新五代史·唐本纪》。应顺元年（即清泰元年），公元934年，李从珂反于凤翔，率兵东进。闵帝遣兵讨之，可李从珂是宿将，讨军反叛降李从珂。李从珂入洛阳，废闵帝，即位为后唐末帝，改元清泰。大意据《资治通鉴·后唐纪七》。李从珂以养子夺亲子之位——这是养子继统的合法性困境的又一典型（养子继统的根源详见《资治通鉴·后梁纪一·朱温代唐》论养子继承制萌芽）。

李从珂即位后，亦不得安。石敬瑭是李嗣源女婿（娶李嗣源女永宁公主），沙陀人，时任河东节度使，镇太原。石敬瑭与李从珂素有嫌隙——李从珂是养子，石敬瑭是女婿，二人皆以非亲子继统，互不相下。清泰三年（即天福元年），公元936年，石敬瑭反于太原。可石敬瑭兵少，敌不过后唐大军。石敬瑭遂向契丹借兵——割燕云十六州，岁输帛三十万匹，并称契丹主耶律德光为"父皇帝"，自称"儿皇帝"。大意据《资治通鉴·后唐纪八》《辽史·太宗纪》《旧五代史·晋书·高祖纪》。契丹主耶律德光（辽太宗）率骑兵南下，助石敬瑭。后唐军大败。

公元936年闰十一月，契丹立石敬瑭为晋帝（后晋高祖，详见即将续写的《资治通鉴·后晋纪一·石敬瑭借兵》主场）。李从珂见大势已去，自焚于洛阳玄武楼，后唐亡。大意据《资治通鉴·后唐纪八》《旧五代史·唐书·末帝纪》。后唐自923年李存勖称帝，至936年李从珂自焚，凡三世四帝十四年。后唐之亡，亡于养子继统的合法性困境（李从珂夺位），亡于石敬瑭借兵契丹——割燕云十六州，使中原屏障尽丧，契丹自此深入中原。后唐虽亡，可燕云十六州之失，贻害四百余年——直到明初徐达、常遇春北伐（1368），中原才重新收复燕云。割地借兵的代价，是后唐留给后世最大的祸根。

## 讲人物

**李嗣源：不识字却能听谏的明君**

李嗣源是李克用养子（非亲子），沙陀人，本名邈佶烈，李克用养以为子，赐姓李。大意据《旧五代史·唐书·明宗纪》。李嗣源即位时已六十岁，年事已高，可他能听谏、能用人。李嗣源不识字，每阅奏章，须人诵读——可他常能于诵读中察觉是非，从谏如流。这是武夫难得的本事——多数武夫不识字，亦不听谏；李嗣源不识字，却能听谏，是五代少有的明君。

李嗣源改革，罢庄宗之弊政——罢伶官之权，汰冗员，减赋税，恤民困。天成、长兴间，中原粗安，史称"明宗小康"。明宗曾夜焚香祝天，说"某蕃部之人，因乱为众所推，愿天早生圣人，为生民主"——我是蕃部之人，因乱被众人推举，愿上天早生圣人，做百姓之主。大意据《旧五代史·唐书·明宗纪》。这话，是明宗的自知之明——他知道自己非天命所归，只愿天下早有真主。这份自知，是五代君主少有的清醒。可明宗晚年，被亲子李从荣谋反的悲剧笼罩——秦王李从荣之乱，是后唐由治转乱的转折点。明宗是五代少有的明君，可他的明，止于自身；他的治，止于一代——养子继统的合法性困境，让后唐无法长治。

**李从珂：以养子夺位的后唐末帝**

李从珂是李嗣源养子（非亲子），本姓王，平山人，李嗣源养以为子，赐姓李。大意据《旧五代史·唐书·末帝纪》。李从珂是后唐宿将，骁勇善战，历事李嗣源、李从厚。934年，李从珂反于凤翔，率兵东进，废闵帝李从厚，即位为后唐末帝。李从珂以养子夺亲子之位——这是养子继统的合法性困境的又一典型。

李从珂的悲剧，是"养子继统合法性不足"的典型。养子夺位，得位不正，朝野不服；养子继统，破坏嫡长子继承的稳定性，埋下内乱隐患。李从珂夺位后，亦不得安——石敬瑭是李嗣源女婿，与李从珂素不相下，遂借兵契丹。936年，石敬瑭借契丹兵灭后唐，李从珂自焚于洛阳玄武楼。李从珂以养子夺位，最后亦被养子继统的反噬所灭——养子继统的合法性困境，是后唐速亡的根子。可见养子继统，是五代的特产，也是五代的祸根；朱温欲立养子友文（详见《资治通鉴·后梁纪一·朱温代唐》），李嗣源以养兄继位（详见《资治通鉴·后唐纪一·同光灭梁》），李从珂以养子夺位——养子继统的循环，贯穿五代。

**石敬瑭：割地借兵的儿皇帝**

石敬瑭是沙陀人，李嗣源女婿（娶李嗣源女永宁公主），时任河东节度使，镇太原。石敬瑭与李从珂素有嫌隙——李从珂是养子，石敬瑭是女婿，二人皆以非亲子继统，互不相下。936年，石敬瑭反于太原，向契丹借兵——割燕云十六州，岁输帛三十万匹，并称契丹主耶律德光为"父皇帝"，自称"儿皇帝"。大意据《资治通鉴·后唐纪八》《辽史·太宗纪》《旧五代史·晋书·高祖纪》。

石敬瑭割燕云十六州，是中原屏障的断送。燕云十六州（幽、蓟、瀛、莫、涿、檀、顺、新、妫、儒、武、云、寰、应、朔、蔚），大致相当于今北京、天津、河北北部、山西北部——是中原抵御北方游牧的天然屏障。石敬瑭割此十六州予契丹，使中原门户洞开——契丹骑兵自此可长驱直入中原。这份割地的代价，贻害四百余年——直到明初徐达、常遇春北伐（1368），中原才重新收复燕云。石敬瑭称"儿皇帝"，是中原王朝对少数民族政权的奇耻——以臣称君，以子称父，是华夏正统的崩塌。可见割地借兵，是引狼入室的极致——借外兵以争内位，是自毁屏障；割地以换帝位，是断送根本。石敬瑭之祸，是五代最大的遗毒——燕云之失，贻害四百年。

## 讲背景

明宗之治与后唐之亡这事，得放在养子继承制深化与合法性危机、契丹南侵与燕云屏障、五代更替加速、以及藩镇格局演变四重背景里看。

先看养子继承制深化与合法性危机。后唐两代以养子继位——李嗣源养于李克用（详见《资治通鉴·后唐纪一·同光灭梁》论李嗣源以养兄继位），李从珂养于李嗣源。养子继统，是五代沙陀与中原武夫集团的共同现象（养子继承制萌芽详见《资治通鉴·后梁纪一·朱温代唐》）。可养子继统，破坏了嫡长子继承的稳定性——养子与亲子争位，养子与养子争位，是五代内乱的根源。朱温欲立养子友文，激发亲子友珪弑父；李嗣源以养兄继位，又为李从珂夺位埋下隐患；李从珂以养子夺位，又为石敬瑭借兵埋下隐患。可见养子继统，是五代的特产，也是五代的祸根——养子继统的合法性危机，让五代王朝无法长治。

再看契丹南侵与燕云屏障。梁晋争霸期间，契丹已在崛起（详见《资治通鉴·后梁纪五·梁晋争霸》论契丹南侵的牵制）。耶律阿保机于916年称帝建契丹国（后改辽）。到后唐末年，契丹主已是耶律德光（辽太宗）。石敬瑭割燕云十六州，使中原屏障尽丧——契丹骑兵自此可长驱直入中原。燕云十六州是中原抵御北方游牧的天然屏障，北依燕山、西依太行，是华北平原的北界。割此十六州，等于把华北门户拱手让人。这份割地的代价，贻害四百余年——契丹、女真、蒙古，相继据燕云南下中原。可见燕云屏障，是中原的根本；割燕云以换帝位，是断送根本。石敬瑭之祸，是五代最大的遗毒。

再看五代更替加速。五代（后梁、后唐、后晋、后汉、后周）是唐亡后中原的五个短命王朝，前后不过五十三年（907-960）。后梁十六年（907-923），后唐十四年（923-936），后晋十一年（936-947），后汉四年（947-950），后周十年（951-960）——越到后来，王朝越短命。后唐之亡，亡于养子继统与割地借兵——这是五代更替加速的典型。后唐之后，后晋亦不过十一年便亡（石敬瑭之侄石重贵亡国，详见即将续写的《资治通鉴·后晋纪一·石敬瑭借兵》）。可见五代更替，是"以兵变得位、以兵变失位"的循环——养子继统、借兵外族、禁军反噬，三者交织，让五代王朝速兴速亡。

再看藩镇格局演变。后唐是沙陀李氏建国的，本身亦是一个大藩镇。后唐的版图，承后梁而稍扩——灭前蜀（925），据蜀；可蜀地旋失（925后不久即叛）。后唐实际控制的，仍是中原汴洛一带。藩镇割据的格局，并未因后唐灭梁而改变——后唐只是众多藩镇中最大的一个。后唐末年，石敬瑭据河东，李从珂据凤翔——藩镇各自割据，朝廷号令不出洛阳。可见后唐之亡，亦是藩镇格局演变的必然——后唐本是大藩镇，被更强的藩镇（石敬瑭借契丹）取代。

## 讲道理

王夫之在《读通鉴论》卷二十九（五代中）里论后唐与石敬瑭借兵，话说得极痛。大意据王夫之说，李存勖、石敬瑭、刘知远是沙陀三部的小夷，"窃据唐之京邑"，不足称天子。王夫之论明宗李嗣源，大意据王夫之说，李嗣源不识字，可他能听谏、能用人，是五代少有的"差近正"之君——比朱温、李存勖之流略胜一筹，可仍非天命所归。王夫之论石敬瑭割燕云借兵，话说得极痛——大意据王夫之说，石敬瑭以"儿皇帝"事契丹，是华夏正统的崩塌；割燕云十六州，是断送中原屏障。王夫之把石敬瑭放在"夷盗窃而不可纵"的极致位置上——借外兵以争内位，割地以换帝位，是引狼入室的极致。大意据《读通鉴论》卷二十九。王夫之甚至以为，五代之乱，根子在"中国致之"——非夷狄之强，是中原自毁；石敬瑭借兵契丹，是中原自毁的极致。大意据《读通鉴论》。

胡三省在《资治通鉴音注》后唐纪部分，对明宗之治与石敬瑭借兵的批注，亦多落在"养子继统、割地借兵"上。大意据胡三省注，明宗李嗣源不识字却能听谏，是五代少有的明君——可他的明，止于自身；养子继统的合法性困境，让后唐无法长治。胡三省注石敬瑭割燕云，是带着痛惜的——燕云十六州是中原屏障，割此十六州，等于把华北门户拱手让人；契丹自此深入中原，贻害四百余年。胡三省的潜台词是：后唐之亡，亡于养子继统与割地借兵——养子继统破坏合法性，割地借兵断送屏障，二者交织，后唐焉得不亡。大意据《资治通鉴音注》后唐纪部分。

司马光在《资治通鉴·后唐纪七》里对明宗之治与后唐之亡的记述，亦带其一贯史观。大意据司马光说，明宗李嗣源是五代少有的明君，五年小康是五代乱世的喘息——可明宗之治，止于一代；养子继统的合法性困境，让后唐无法长治。司马光论石敬瑭割燕云，是带着痛惜的——割燕云十六州，是中原屏障的断送；称"儿皇帝"，是华夏正统的崩塌。司马光的史观，是"治乱在人主之德"——明宗有德，故能小康；石敬瑭无德，故割地借兵。可明宗之德，止于一代；养子继统的合法性危机，让后唐无法长治——可见治乱在人主之德，亦在制度之善；养子继统之制不立，后唐焉能长治。大意据《资治通鉴·后唐纪七》。

## 问道悟道

明宗之治与后唐之亡这出戏，藏着两条让人开眼的规律。

**养子继统的合法性困境。**

后唐两代以养子继位——李嗣源养于李克用，以养兄继养弟之位；李从珂养于李嗣源，以养子夺亲子之位。养子继统，是五代沙陀与中原武夫集团的共同现象——养子多是军中骁将，比亲子更能打仗，故被立为嗣。可养子继统，破坏了嫡长子继承的稳定性——养子与亲子争位，养子与养子争位，是五代内乱的根源。李从珂以养子夺位，又为石敬瑭借兵埋下隐患；石敬瑭是女婿，亦以非亲子继统，与李从珂互不相下。可见养子继统，是五代的特产，也是五代的祸根。

养子继统的合法性困境这事，历史上有几个对照。后梁朱温欲立养子友文，激发亲子友珪弑父——养子继统引发内乱（详见《资治通鉴·后梁纪一·朱温代唐》论弑父篡位与因果回旋）。后周郭威养柴荣（柴荣本姓柴，郭威内侄兼养子），柴荣继位为后周世宗——可柴荣是养子继统的少数成功案例，且柴荣早卒（959），后周旋亡。大意据《旧五代史·周书·世宗纪》。后唐李嗣源、李从珂，与后周柴荣，都是养子继统——可后唐养子继统引发内乱，后周养子继统虽成功却短命。可见养子继统，合法性不足，朝野不服，必然引发内乱；纵有成功，亦难长久。要破此困境，须重建嫡长子继承的稳定性——可五代武夫重军功轻血缘，养子继统之风难改。要到宋兴，确立嫡长子继承与宗法制度，养子继统的合法性困境才得以化解——可那已是五十三年后的事了。

**割地称帝与燕云屏障的丧失。**

石敬瑭割燕云十六州，岁输帛三十万匹，称契丹主耶律德光为"父皇帝"，自称"儿皇帝"——这是中原王朝对少数民族政权的奇耻。燕云十六州是中原抵御北方游牧的天然屏障，北依燕山、西依太行，是华北平原的北界。割此十六州，等于把华北门户拱手让人——契丹骑兵自此可长驱直入中原。这份割地的代价，贻害四百余年——契丹、女真、蒙古，相继据燕云南下中原。直到明初徐达、常遇春北伐（1368），中原才重新收复燕云。可见割地借兵，是引狼入室的极致——借外兵以争内位，是自毁屏障；割地以换帝位，是断送根本。

割地称帝与燕云屏障的丧失这事，历史上有几个对照。西周，幽王烽火戏诸侯，犬戎入寇，西周亡——可西周之亡，是外族入侵，非割地借兵。大意据《史记·周本纪》。西晋，八王之乱，借匈奴、鲜卑等胡族兵，五胡乱华，西晋亡——可西晋是借胡兵内乱，非割地借兵。大意据《晋书·惠帝纪》。后梁，朱温篡唐，未割地借兵。大意据《旧五代史·梁书·太祖纪》。石敬瑭割燕云，与西周犬戎、西晋五胡，是同构的——都是中原屏障尽丧，外族深入中原。可石敬瑭之祸，比西周犬戎、西晋五胡更甚——石敬瑭是主动割地借兵，称"儿皇帝"，是华夏正统的主动崩塌。可见割地借兵，是引狼入室的极致——借外兵以争内位，割地以换帝位，是断送根本；引狼入室，狼一旦入室，便不再走。石敬瑭之祸，是五代最大的遗毒——燕云之失，贻害四百年；儿皇帝之耻，贻羞千古。要防引狼入室，须以自强为本，不借外兵——可石敬瑭为争帝位，不惜割地借兵，是五代无德的极致。

## 结语

明宗之治是五代乱世的喘息，后唐之亡是养子继统与割地借兵的双重悲剧——养子继统破坏合法性让后唐无法长治，石敬瑭割燕云十六州借兵契丹断送中原屏障贻害四百年；十四年的短命王朝，就此收场。
