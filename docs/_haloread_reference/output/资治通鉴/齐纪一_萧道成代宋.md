# 资治通鉴·齐纪一·萧道成代宋

## 讲事情

升明三年，公元 479 年，建康城里的宋顺帝刘准，才十一岁，被一群大人架着走完了人生最后一场戏。

要讲清楚这场戏，得先回到两年前。刘宋到了末年，出了一位荒唐得出奇的小皇帝——后废帝刘昱，史称苍梧王。这刘昱十岁登基，十四五岁就开始自己拎刀出门杀人。他常带一帮亲信夜出，逢人便杀，路上行人有躲不及的，就被他亲手剖腹取婴、锤劈行路。朝中大臣早朝时，谁也不敢保证能活着回家。阮佃夫、申伯宗这些老臣，一个个死在他手里。萧道成那时是刘宋的右卫将军、中领军，掌禁兵，功劳大，威望高，刘昱盯上了他。

有一次刘昱闯进萧道成府中，正赶上萧道成白昼裸身睡觉。刘昱看他肥胖的肚子，竟拿肚子当箭靶，画个记号拉弓就要射。萧道成吓得冷汗直流，苦苦劝住，刘昱这才换了响箭，一箭射中肚脐——大意据《南史·宋本纪》。可萧道成心里明白：这样的皇帝，迟早要他的命。

于是萧道成下了决心。元徽五年，公元 477 年七月，萧道成结交刘昱的亲信杨玉夫、杨万年等人，趁刘昱七夕夜醉卧仁寿殿，派人将其弑杀，年仅十五。刘昱一死，萧道成立刘准为帝，是为宋顺帝，改元升明。刘准那年才九岁。萧道成以司空、录尚书事、骠骑大将军辅政，从此把刘宋的权柄攥在了手里。

接下来的两年，是禅让戏的预备。萧道成先平了荆州刺史沈攸之的反抗，又清除了袁粲、刘秉这些忠于刘宋的势力。朝中大事，全凭他一句话。升明二年，公元 478 年，萧道成进位太傅、扬州牧，加剑履上殿、入朝不趋、赞拜不名。升明三年，公元 479 年，进相国，封齐公，加九锡。没几天，进齐王，增封十郡。这一套加九锡、封公、封王、加十郡的程序，跟当年曹丕代汉、司马炎代魏、刘裕代晋一模一样——剧本是现成的，照着演就行。

四月甲午，禅让的正日子到了。宋顺帝该把皇位让给齐王。可这十一岁的小皇帝，临了却缩了。据《南史·宋本纪》记载，顺帝不肯出宫，逃到佛盖之下——躲到佛像的伞盖底下去。王敬则带兵来请，顺帝哭着问是不是要杀他，王敬则答——大意据《南史·齐本纪上·高帝》，不过是换个地方住罢了，跟当年你们刘家取代司马家没什么两样。顺帝哭得泣不成声。这一幕，让人想起前废帝刘子业杀弟刘子鸾时，刘子鸾临终那句让后人唏嘘千年的话——「愿后身勿复生帝王家。」《宋书·孝武十四王传》。这话本是刘子鸾说的，可放在顺帝身上，竟也字字贴切——像是为整个南朝末代帝王作的注。当年刘裕代晋，把晋恭帝司马德文杀了；如今萧道成代宋，刘准的下场，早已写好。

禅让礼毕，宋顺帝被降为汝阴王，迁居丹阳。可建康城里哪容得下旧主。五月，监守的人把这位十一岁的废帝害死在秣陵——一说死于丹阳。萧道成下诏为他举哀，追谥为顺帝，礼数都做足。仪式是体面的，命却是没了。

萧道成称帝，国号齐，建元建元。他就是齐高帝。他在位四年，建元元年到建元四年，公元 479 到 482 年。这四年，他做了一件最让后人议论的事——诛杀刘宋宗室。刘裕的子孙，从文帝、孝武帝到明帝诸子，凡是还有血脉留存的，几乎被扫荡一空。沈攸之起兵时打的旗号是「刘氏复兴」，萧道成要的恰恰相反：让刘氏再无复兴的可能。

建元四年，公元 482 年三月，萧道成病重。临终前他留下一道遗诏，话里带着不安——「治道未泰，朕所未安。」大意据《南齐书·高帝纪》。治道还没走上轨道，我心里不踏实。这话是一个开国皇帝临死前的清醒。他知道，自己代宋建国，靠的是权谋与兵威，不是民心；他知道，南朝的禅让戏唱了三回，每一回都埋下新祸；他更知道，自己的子孙能不能坐稳这江山，他还看不准。萧道成享年五十六岁，太子萧赜继位，是为齐武帝。南齐二十三年的短命王朝，就此开场。

## 讲人物

**萧道成：深知「未泰」的开国之君**

萧道成，字绍伯，小名斗将，祖籍东海兰陵，南渡后居于晋陵武进，是兰陵萧氏。他父亲萧承之，是刘宋的汉中太守、龙骧将军，靠军功起家。萧道成少年时随父征战，后入仕刘宋，从一名普通将领一步步爬上来。他外貌有特点——腰腹大，史书说他「龙颡钟声，鳞文遍体」，这些帝王相的描写，多半是后人的附会，但能看出他是个魁梧的人。

萧道成能成事，靠的是两点：一是能打仗，二是能忍耐。元徽二年，公元 474 年，桂阳王刘休范起兵叛乱，直扑建康。当时朝中一片慌乱，萧道成主张坚守台城，待敌疲再击。他亲自上阵，在新亭击溃叛军，平了这场几乎倾覆刘宋的乱子。这一仗，让他名震朝野，也让他握住了禁军大权。可他没因此翘尾巴，反而更沉得住气。后废帝刘昱要拿他的肚子当箭靶，他能忍；朝中有人挤兑他，他能忍；该出手时——比如废刘昱、立顺帝——他下手也快。能忍又能断，这是他的本事。

他代宋之后，自己也明白这江山来路不正。他常说一句话——「吾本布衣素族，因藉时来，遂隆大业。」大意据《南齐书·高帝纪》。我本是布衣素族出身，借着时运，才成就了大业。这话里有自谦，也有自省。所以他建国后，崇尚节俭，自己穿粗布，后宫不用珠玉；他亲理政务，留心刑狱，想做出点治世的样子。可他心里始终不踏实——「治道未泰，朕所未安」成了他临终的牵挂。一个开国皇帝，临死前说的是「我不安」，而不是「我功成」，这份清醒，在南朝皇帝里少见。

**宋顺帝刘准：哭着躲进佛盖下的末代之君**

刘准，字仲谋，是宋明帝刘彧的第三子。公元 477 年，萧道成弑杀后废帝刘昱后，把他推上皇位，时年九岁。他这个皇帝，从头到尾是个傀儡。即位诏书是萧道成的人写的，朝政是萧道成的人把持的，连他能不能活到明年，都是萧道成说了算。

刘准一生最被记住的，是禅让那天的哭。十一岁的孩子，被架着走完禅让礼，哭着躲进佛像底下。这不是他一个人的哭，是整个南朝末代帝王的哭。前废帝刘子业杀弟刘子鸾时，刘子鸾临终曾叹「愿后身勿复生帝王家」——《宋书·孝武十四王传》；晋恭帝司马德文被刘裕所杀前，处境同样凄凉；后来齐和帝萧宝融禅让给萧衍，也是一般下场。这一句「勿复生帝王家」，道尽末代帝王的不由自主。他禅让后不到一个月，就被害死于监中，年不过十一二岁。一个孩子，因生在帝王家，丢了性命。

**王敬则：唱禅让戏的武夫**

王敬则，临淮射阳人，屠狗出身，后投军，是萧道成的心腹猛将。他这个人，粗豪，敢干，没那么多礼义顾忌。禅让那天，宋顺帝躲进佛盖下不肯出来，是王敬则带兵去「请」的。他对着哭哭啼啼的小皇帝，说了一句分量极重的话——大意据《南史·齐本纪上·高帝》，萧道成代宋，就像当年你们刘家代司马家一样。这话一针见血：你们刘宋的天下，就是这么从司马家手里抢来的；如今萧家从你们手里抢，一报还一报，谁也别喊冤。王敬则不懂大道理，可他这话，把禅让戏的底牌掀了个干净——所谓的禅让，不过是兵强马壮者夺权的体面说法。

**褚渊：被讥为「可怜」的佐命之臣**

褚渊，字彦回，河南阳翟人。他出身名门，是刘宋的驸马——尚宋文帝之女南郡公主。按说他是刘宋的姻亲旧臣，可萧道成代宋时，他不但没殉节，反而带头劝进、捧玺绶，成了佐命第一功臣。褚渊貌美，当时有人见他，叹了一声「可怜」——古人「可怜」是可爱之意。可到了后世，「可怜」二字变了味，褚渊也就被史家讥为「一身事两朝」的软骨头。他和王俭一道，替萧道成草定禅让礼仪，把这场戏演得有模有样。褚渊一生，享尽荣华，可身后名声不好。一个人的操守，有时就压在最后这一桩「迎新送旧」上。

## 讲背景

萧道成代宋，不是孤立的一桩事，是南朝禅让传统的又一次上演。从东晋到南陈，二百年间，禅让戏唱了四回——刘裕代晋、萧道成代宋、萧衍代齐、陈霸先代梁，加上更早的曹丕代汉、司马炎代魏，这是一条绵延三百年的篡位剧本。剧本的核心是「禅让」二字。古时的禅让，传说是尧让位于舜、舜让位于禹，是圣王主动让贤的美谈。可到了汉魏以后，禅让变了味——成了权臣夺权的遮羞布。曹丕代汉后，自己都说了一句大实话：「舜禹之事，吾知之矣。」——舜禹禅让那回事，我现在算明白了。《三国志·魏书·文帝纪》裴注。意思是：所谓禅让，不过是装个样子罢了。从此每一代权臣夺位，都照着这套剧本演：先封公、加九锡、赐殊礼，再封王、加十郡，最后小皇帝「主动」让位。礼数齐全，体体面面，可底下全是刀兵。萧道成代宋，是这套剧本的第三次重演，演得熟极而流。

再看宋齐之际的南北对峙。刘宋末年，北魏正是孝文帝的祖父辈在位，国力正盛。刘宋丢了淮北四州及豫州淮西之地，疆域已从刘裕时的全盛退缩到淮河一线。萧道成代宋前后，北方有冯太后主政、孝文帝即将亲政，正在酝酿一场大改革。南边内部禅代忙乱，北边埋头蓄力，南北天平正在悄悄向北倾斜。萧道成心里也清楚，所以建国后他没有大举北伐，而是采取守势，与北魏通好——永明年间南齐与北魏通使，就是萧道成打下的底子。这种「内禅外和」的格局，是南齐初年的基本国策。

还有一层，是刘宋宗室屠戮的循环。刘宋自孝武帝以后，宗室相残就成传统：前废帝刘子业杀叔父刘义恭，把他的眼睛挖出来用蜜浸着叫「鬼目粽」；宋明帝刘彧杀侄子刘子业、又杀诸弟诸侄，把刘裕的子孙几乎杀绝。萧道成代宋后诛刘宋宗室，某种程度上是「继承」了刘宋宗室自相屠戮的恶风——既然刘家自己都杀绝了自己，外人下手时也少了心理负担。这种宗室相残的恶性循环，成了南朝政治的一大痼疾。萧道成大概没想到，他自己打下的南齐，二十三年后会重蹈覆辙——萧鸾篡位时，把萧道成、萧赜的子孙也几乎杀了个干净。历史开起玩笑来，从不手软。

再看兰陵萧氏的崛起。萧道成是兰陵萧氏，这个家族本是南渡的次等士族，在门阀森严的南朝原本排不上号。萧道成靠军功起家，借刘宋末年乱局一跃成为开国皇帝，是南朝「寒人掌机要」大势的一个突出样本。他建国后，提拔了一批寒门武将与实际人才，对门阀政治有所冲击。可萧道成毕竟是借门阀体系上来的，他不敢、也无力彻底推翻门阀——褚渊、王俭这些顶级士族，照样是佐命首功。所以南齐一朝，门阀与寒人并存的格局没有根本改变，只是寒人的分量重了些。这是理解南齐政治的一把钥匙。

## 讲道理

王夫之在《读通鉴论》里论南朝禅代，话说得极透。大意据王夫之说，所谓禅让，不过是「以欺天下、自欺其心」的一场戏。萧道成、萧衍这些人，夺了人家的江山，却偏要装出受之不情的样子，让小皇帝三让三辞，自己还要「辞不受命」装几回。这一整套仪式，骗不了天下人，也骗不了他们自己——他们心里清楚，江山是抢来的，不是让来的。可他们仍要演，因为禅让这层皮，能给篡位披上一层「天命」的合法性。王夫之看穿的是：禅让戏越演越熟，说明南朝的「合法性」越来越经不起推敲，只能靠仪式来补。仪式越繁，底气越虚。大意据《读通鉴论》卷十五。

胡三省在《资治通鉴音注》里，对南朝禅代的批注，往往落在「养虎自遗患」与「前事不忘后事之师」上。胡三省生在宋元之际，亲身经历了亡国之痛，他注《通鉴》时，对这种「以力夺国」的循环格外敏感。他批注萧道成诛刘宋宗室，意在前车：当年刘裕怎么对待晋恭帝，如今萧道成就怎么对待宋顺帝；将来萧衍怎么对待齐和帝，也是照方抓药。胡三省的潜台词是：你今天种什么因，明天就收什么果。萧道成临终那句「治道未泰，朕所未安」，在胡三省看来，恐怕正是不安于此——他知道自己种下的，是一颗什么样的种子。大意据《资治通鉴音注》齐纪部分。

司马光在《资治通鉴》里记萧道成代宋，叙事克制，不轻易发议论，可字里行间有褒贬。他记萧道成崇尚节俭、留心政务，是肯定；记他诛刘宋宗室、行篡逆之事，是贬。司马光的史观，一贯是「正统」与「名分」——他认为君臣之伦不可乱，禅让是乱臣贼子的把戏。所以《通鉴》记南朝禅代，多写其残酷与虚伪，少写其「顺天应人」的粉饰。萧道成遗诏那句「治道未泰，朕所未安」，被司马光原样保留下来，这是史家的高明之处——不评论，让皇帝自己的话说话。一个篡位者临终的不安，比史家千言万语的批评都更有力。

## 问道悟道

萧道成代宋留给后人的，是两条反复在历史上上演的规律。

**第一条：禅让的剧本与代价。**

从曹丕代汉，到司马炎代魏，到刘裕代晋，再到萧道成代宋，禅让戏唱了四回。剧本一模一样：先掌大权，再封公封王，加九锡加十郡，最后小皇帝「主动」让位，权臣还要三让三辞装几番，最后「不得已」受禅。仪式之繁、礼数之全，把篡位粉饰成「受命于天」。

可这剧本是有代价的。代价之一，是合法性的虚无。每一回禅让都把「君臣之伦」凿掉一层，凿到最后，南朝的皇帝谁也不信「忠君」二字——既然你能禅让我，我为什么不能禅让别人？于是南朝一百七十年，禅让戏唱了四回，皇位像走马灯一样换。代价之二，是末代帝王的人头。晋恭帝被刘裕杀，宋顺帝被萧道成杀，齐和帝后来被萧衍杀，每一回禅让都以前朝末帝的血收场。前废帝刘子业杀弟刘子鸾时，刘子鸾临终那句「愿后身勿复生帝王家」——《宋书·孝武十四王传》，是为这一整套剧本作的注。

这事历史上有清晰的循环。西汉末王莽代汉，演禅让戏演得最足，可不到十五年就被赤眉绿林推翻，新朝二世而亡。《汉书·王莽传》。曹魏代汉，没过四五十年就被司马氏以同样手法篡掉，曹奂禅让司马炎时的场面，和他爷爷曹丕受禅时如出一辙。《三国志·魏书·三少帝纪》。这正应了一句老话：你用这套手段夺的天下，别人也会用这套手段夺走。禅让的剧本，演一次是夺位，演四次就成了诅咒——诅咒皇位坐不稳。萧道成大概也隐约感到这诅咒，所以临死才说「治道未泰，朕所未安」。

**第二条：兔死狗烹与前朝宗室的处置。**

萧道成代宋后，做的一件最让人不齿的事，是诛杀刘宋宗室。刘裕的子孙，凡是有血脉留存的，几乎扫荡一空。这事的逻辑是冷酷的：留着前朝宗室，就是留下复辟的火种；不如斩草除根，绝后患。

可这事历史上有反复的教训，结论却常常被忽略。汉高祖刘邦得天下后，韩信、彭越这些大功臣一个个被杀，「狡兔死，走狗烹」成了千古定论。《史记·淮阴侯列传》。可刘邦杀的是功臣，萧道成杀的是前朝宗室，对象不同，逻辑一脉——都是「坐稳江山就要清人」。明太祖朱元璋得天下后，胡惟庸案、蓝玉案，牵连数万，功臣宿将几乎杀绝，连他自己的儿子们都未能保全。《明史·太祖本纪》。可这些「清人」的王朝，结局都不太好——汉有诸吕之乱，明有靖难之役。清得越狠，留下的裂缝越大。

更直接的前车，是刘裕自己。刘裕代晋，把晋恭帝司马德文杀了，开南朝杀废帝的先例。从那以后，南朝每一代禅让，废帝都活不成。萧道成杀宋顺帝，是照着刘裕的方子抓药；后来萧衍杀齐和帝，又照着萧道成的方子。这条「杀废帝」的链子，从刘裕一直拉到陈霸先，二百年不断。可杀了废帝，江山就稳了吗？南朝禅代最频，皇位却最不稳——杀一个废帝，堵不住下一个萧道成。兔死狗烹这事，看着是除患，实则是种祸。萧道成临终的不安，或许正源于此：他知道，自己开的这个头，将来会还到子孙身上。果然，南齐二十三年而亡，萧道成的子孙在萧鸾篡位时被杀得几乎断绝——一报还一报，分毫不差。

## 结语

一场禅让戏，十一岁末帝哭着躲进佛盖下，留下一声为末代帝王而发的叹息——萧道成代宋，是南朝篡位剧本的第三次重演，也是「治道未泰」一语成谶的开端。
