# 资治通鉴·陈纪一·陈霸先建国

## 讲事情

陈朝的开国，不像是一场创业，倒像是一场收尾——把梁朝剩的那点残局收拾起来，凑成一个最小、也最弱的南朝。

要讲陈霸先建国，得从梁朝怎么塌的说起。梁武帝萧衍在位四十八年，江南太平了小半个世纪。可太清二年，公元 548 年，从北方投降来的羯人侯景起兵反梁，围台城一百三十余日，次年城破，八十六岁的萧衍饿死台城。这场侯景之乱，把自永嘉南渡以来二百余年的江南元气，毁掉了一大半（侯景之乱本事详见《资治通鉴·梁纪·侯景之乱》）。乱平之后，梁室已经只剩一副空架子。

侯景是王僧辩、陈霸先联手平掉的。这两个人，本是平乱的双璧，也是这场建国戏里最要紧的一对冤家。王僧辩出身太原王氏，是门阀世族；陈霸先出身寒微，吴兴长城人，岭南小吏起家。两人一文一武，一北一南，联手把侯景赶下了台。乱平后，梁元帝萧绎在江陵即位，王僧辩、陈霸先分掌建康与京口的兵权，一为骠骑大将军，一为司空，共撑梁室的半壁江山。

可梁室的半壁，也没撑住多久。承圣三年，公元 554 年，西魏大举南下。西魏权臣宇文泰派于谨、杨忠、宇文护率军攻江陵。江陵是梁元帝的都城，城破之日，萧绎被杀，江陵朝廷一夜覆灭。这一仗，把梁室打回了原形——江陵没了，梁元帝没了，剩下建康一隅，群龙无首。梁朝的招牌，眼看就要倒。

江陵陷落之后，王僧辩、陈霸先在建康共立萧方智为太宰、承制。萧方智是梁元帝的第九子，年纪尚幼，是梁室仅剩的血脉之一。这本是两人商量好的安排，可接下来的一件事，把这对盟友推向了决裂。

原来北齐趁火打劫。北齐文宣帝高洋要把从前在北方扣留的萧渊明送回梁朝当皇帝。萧渊明是梁武帝之侄、贞阳侯，论辈分是萧方智的堂叔。北齐的意思很明白：你梁朝立萧方智，是我北齐不乐意；我给你送个萧渊明回来，你迎他即位，梁朝就成了我北齐的附庸。北齐甚至派兵护送萧渊明到边境，扬言不发兵迎立便要兴师问罪。

王僧辩扛不住了。他手里兵虽不少，可江陵新败，江南残破，实在打不起与北齐的另一场仗。他先是反对，几次派人去说理，北齐态度强硬。最后，王僧辩妥协了——绍泰元年，公元 555 年，他迎萧渊明入建康即位，降萧方智为太子。这一妥协，等于把梁朝的废立之权，交到了北齐手里。

陈霸先不肯。他驻兵京口，听说王僧辩迎立萧渊明，连发数书反对，王僧辩不回。陈霸先看得很清楚：王僧辩这一让，南朝就成了北齐手里的傀儡，他与王僧辩一同浴血打下来的局面，要拱手送给北方。可他也清楚，王僧辩兵力不弱，正面相争未必能赢。于是他选了一条最险也最快的路——袭杀。

绍泰元年九月，陈霸先自京口起兵，率部将侯安都、杜棱等，水陆并进，直扑建康。王僧辩毫无防备。陈霸先的兵马突入建康，直扑王僧辩的府第。王僧辩被擒，父子二人俱被杀。陈霸先废萧渊明，复立萧方智为帝，自己进位司徒、丞相、扬州牧，封陈国公，从此把梁室的实权攥在了手里。这一手袭杀盟友，快、准、狠，是陈霸先一生里最见决断的一刻——也是后人议论最多的一笔。

接下来，是禅让戏的预备。陈霸先先击退了徐嗣徽、任约勾结北齐的反扑，又击败了王僧辩旧部、湘州刺史王琳的反抗。内外既定，太平二年，公元 557 年十月，梁敬帝萧方智禅位于陈王。萧方智那年才十三岁。陈霸先三让三辞，照着曹丕、司马炎、刘裕、萧道成、萧衍演过的那套剧本，受禅称帝，国号陈，改元永定，都建康。他就是陈武帝。

陈武帝在位的时间短得可怜。从永定元年十月称帝，到永定三年六月病逝，前后不过二十一个月。这二十一个月里，他做的大事是平定王琳的叛乱、击退北齐的进犯、安抚江南残破的州郡。永定三年，公元 559 年六月丙午，陈霸先病逝于建康璇玑殿，享年五十七岁。他临终时，太子未立——陈霸先诸子多早夭（长子陈克曾立为太子，亦早卒），没有成年的嫡子可以继位。群臣议定，迎立其兄陈谈先之子陈蒨即位，是为陈文帝。拥立陈蒨的，正是袭杀王僧辩时立下大功的侯安都。陈霸先一支的血脉至此断了，皇位传到了他哥哥那一系——这是陈朝开国时埋下的第一颗隐忧：开国之君无子可传，只能兄终弟及、再及于侄。

陈霸先建国，就讲到这里。从江陵陷落、王僧辩迎立萧渊明、陈霸先袭杀盟友、到敬帝禅让、定都建康，前后不过三年。可这三年的代价，是梁朝最后的脸面，是王僧辩父子的人头，也是南朝自刘裕以来最小的一个版图——陈朝的地盘，仅剩长江以南、三峡以东，比刘宋、萧梁都小了一大圈。

## 讲人物

**陈霸先：寒门杀出来的开国之君**

陈霸先，字兴国，吴兴长城人。他出身寒微，跟刘裕、萧道成这些寒门武将出身的开国皇帝是一脉。他早年在岭南做小吏，因平定交州李贲之乱崭露头角，一步步爬上来。论门第，他不如王僧辩；论资历，他不如萧衍的旧臣。可他有两样本事，是别人比不了的——一是能打仗，二是敢决断。

陈霸先的军旅生涯，是从岭南开始的。大同年间，交州土豪李贲起兵反梁，割据数年。陈霸先率军南下，苦战数年，于太清元年前后平定李贲，由此名震岭南。这份苦战的底子，后来成了他平侯景、抗北齐的本钱。侯景之乱起，他从岭南千里北上勤王，与王僧辩会师，联手攻破建康，平定侯景（详见《资治通鉴·梁纪·侯景之乱》）。这一路，陈霸先打的是硬仗，靠的是真本事。

他最让人议论的，是袭杀王僧辩。王僧辩是他的盟友，是一同平乱的双璧。可陈霸先认定王僧辩迎立萧渊明是引狼入室，便不顾同盟之义，先下手为强。这一手，从政治上说是果断——保住了南朝不被北齐牵着走；从道义上说是寡恩——盟友说杀就杀，毫无顾忌。乱世里，盟友翻脸比翻书还快，这是那个时代的常态，可陈霸先这一手，仍显得凉薄。也正因如此，他后来终其一生都未能真正收服王僧辩的旧部——王琳、王頠这些王僧辩的部将，先后据湘州、奔北齐，成了陈朝前期的肘腋之患。

陈霸先建国后，知道自己接的是个烂摊子。他崇尚节俭，留心政务，史书说他「披图御历，揔己躬亲」——大意据《陈书·高祖本纪》，登基之后事必躬亲。可他没来得及做什么大事，二十一个月就病逝了。一个寒门出身的开国之君，打了一辈子仗，临到坐上龙椅，却没享几天福，也没留下一个成年的儿子继位。这是陈霸先个人的悲剧，也是陈朝开国的先天不足。

**王僧辩：妥协一时、丧命一夕的柱国**

王僧辩，字君才，太原祁人，出身门阀世族。他父亲王神念，从北魏南奔入梁，是梁朝名将。王僧辩以门荫入仕，随父征战，后来成为梁元帝萧绎手下的头号大将。论门第、论资历，他都在陈霸先之上。

王僧辩的能耐，是平侯景。侯景之乱时，他与陈霸先联手，率军东下攻建康，苦战经年，终于克复台城，诛杀侯景。这一仗，是南梁的存亡之战，也是王僧辩一生的顶点。可这一仗打完，他就遇到了平生最难的一道题——北齐要把萧渊明送回来当皇帝。

王僧辩的选择，是妥协。他不是不想抗，他抗了几次，可北齐态度强硬，又有大军压境。江南刚经侯景之乱，江陵又新败，他实在凑不出与北齐决战的本钱。他迎立萧渊明，是认了南梁的弱，认了北齐的强。可这一认，就把自己的命也搭进去了。陈霸先从京口杀来，王僧辩措手不及，被擒于府中。临难之际，王僧辩对陈霸先说了一句——大意据《南史·王僧辩传》，他抱怨陈霸先不讲信用。可这话已经晚了。

王僧辩的悲剧，是弱国大臣的悲剧。他不是奸臣，他只是个识时务的将相；可识时务到了引狼入室的地步，就成了国家的罪人。他与陈霸先的分歧，不是忠奸之分，是两条路线之争——一条是忍辱负重、向北方低头；一条是拼死一搏、保南朝独立。陈霸先赢了，因为他够狠；王僧辩输了，因为他太懂权衡。乱世里，太懂权衡的人，往往死得最快。

**徐陵：两朝文柄、玉台新咏的主持者**

陈朝的文坛，绕不开一个人——徐陵。徐陵，字孝穆，东海郯人。他出身东海徐氏，是南朝著名的文学世家。其父徐摛，梁时任太子左卫率，是梁简文帝萧纲「宫体」诗文的引路人；徐陵承父学，少年时即以文才名动江南，与庾信齐名，世称「徐庾」。梁简文帝为太子时，徐陵父子并侍东宫，宫体文风，半出其手。

徐陵在梁官至尚书左丞。侯景之乱中，他被侯景所执，几经辗转，后入北齐留滞多年。陈霸先建国后，徐陵南归，陈文帝、陈宣帝两朝，他都位居显要，官至尚书左仆射、左光禄大夫、太子少傅。陈朝的许多册命诏诰，多出其手——大意据《陈书·徐陵传》。他还奉命编选《玉台新咏》十卷，收录汉魏以来情爱题材的诗赋，是南朝文学最重要的总集之一，其中收的《古诗为焦仲卿妻作》（即《孔雀东南飞》），就是赖此书得以流传后世。

徐陵这个人，是陈朝「文」的一面镜子。陈朝以寒门武将开国，可立国之后仍要靠门阀文士撑场面、定礼仪、草诏册。徐陵历仕梁陈两朝，享尽荣华，又以文采立身——这跟同时期北朝的文人命运很不一样。他活到陈宣帝太建年间，八十余岁善终。乱世里的文士，能这样善始善终的不多。从徐陵身上，能看出陈朝虽小，仍是衣冠南渡的正统所在——南朝虽弱，文脉未断。

## 讲背景

陈霸先建国，不是孤立的一桩事。要读懂它，得把它放在南朝禅代的大循环、侯景之乱后的江南残局、南北强弱的逆转，以及佛教盛行的时代空气里去看。

先说南朝禅代的大循环。从刘裕代晋（420）算起，到陈霸先代梁（557），南朝一百三十余年间，禅让戏唱了四回——刘裕代晋、萧道成代宋、萧衍代齐、陈霸先代梁（前三回详可参看《资治通鉴·齐纪一·萧道成代宋》《齐纪五·萧衍代齐》）。每一回都是同一套剧本：权臣掌大权，封公、封王、加九锡、加十郡，最后小皇帝「主动」让位。陈霸先代梁，是这套剧本的第四次重演。可跟前几回不同的是，陈霸先接手的，是南朝有史以来最烂的一个烂摊子——梁朝经侯景之乱，江南十室九空；江陵又新陷于西魏，梁室仅剩建康一隅。陈霸先的陈国，从建国那天起，就是一个「先天不足」的弱政权。前几位禅让的开国皇帝，刘裕有北伐灭南燕、后秦的武功，萧道成有平桂阳之乱的威望，萧衍有襄阳起兵的厚基；唯独陈霸先，是裹着一身战尘、踩着盟友的尸首上的台。

再说侯景之乱后的江南残局。侯景之乱前，南朝与北朝还能分庭抗礼，梁武帝朝江南繁华了近五十年；乱后，南朝元气大伤，再无北伐之力（详见《资治通鉴·梁纪·侯景之乱》）。梁元帝萧绎虽在江陵勉强撑起一个朝廷，可他猜忌宗室、屠戮骨肉，又与西魏交恶，自取灭亡。江陵陷落后，江南连一个名义上的共主都没有。王僧辩、陈霸先在建康立萧方智，本是权宜之计；可北齐一插手，立储之争就变成了南北博弈。陈霸先袭杀王僧辩，看似是梁室内部的政变，实则是南朝对北齐干预的一次反抗——陈霸先保的不是萧方智，是南朝不被北方牵着走的最后一口气。可这口气的代价，是江南再无一个让各方心服的共主，分裂与内战的隐患，从此埋进了陈朝的根基里。

接着说南北强弱的逆转。陈霸先建国前后，北方正在剧变。西魏（后为北周）攻陷江陵，吞并梁的雍州、荆州；北齐则把势力推到长江北岸，先后占淮南之地。南朝自刘裕时全盛的疆域，到陈霸先手里，退缩到长江以南、三峡以东，连巴蜀、襄阳、淮南都丢了。这是自永嘉南渡以来，南朝版图最小的一回。南北实力的天平，彻底倒向北方。三十年后，隋文帝杨坚挥师南下，陈朝不堪一击——这颗种子，就是陈霸先建国时埋下的。陈霸先未必看不到这点，可他无力回天。一个开国皇帝，明明知道自己的江山是个弱政权，却仍要把这戏演下去，这是开国者的无奈，也是开国者的勇气。

还有一层，是寒门与门阀的消长。陈霸先寒微出身，岭南小吏起家，靠军功一步步爬上来，是南朝「寒人掌机要」大势的一个突出样本。他建国后，提拔了一批寒门武将——侯安都、杜棱、吴明彻的先辈们——对门阀政治有所冲击。可陈霸先毕竟是借门阀体系上来的，他不敢、也无力彻底推翻门阀。所以陈朝立国，仍要靠王通、徐陵这些门阀文士撑场面、定礼仪。陈朝一朝，门阀与寒人并存的格局没有根本改变，只是寒人的分量重了些，门阀的架子还在。这是理解陈朝政治的一把钥匙——它是个寒门武将当家、门阀文士装点门面的混合政权。

最后说佛教。梁武帝崇佛，南朝佛教盛极一时。陈朝承梁之后，佛教依旧盛行。陈武帝陈霸先本人，史载曾舍身庄严寺，舍群臣奉赎——大意据《陈书·高祖本纪》。这种「皇帝舍身、群臣赎回」的把戏，梁武帝演过好几回，陈霸先也照演了一次。江南士族与佛教的关系，在陈朝依旧紧密。佛寺遍布建康，僧尼数万。佛教在乱世里给江南百姓以精神寄托，也给南朝的皇室以「积福修德」的自我安慰。可这层佛光，掩盖不住陈朝政治的衰弱与江山的残破——江南残破到连赋税都收不齐，佛寺却金碧辉煌。这种反差，是南朝末年的一个真实写照。

## 讲道理

王夫之在《读通鉴论》里论陈霸先，话说得相当克制。大意据王夫之，陈霸先之代梁，跟前几朝的禅代一样，都是「以力夺之」的篡弑，没什么「天命」可言。可王夫之也承认，陈霸先有一桩别人比不了的长处——他是真有战功的人。从岭南平李贲，到平侯景，到抗北齐，陈霸先的江山是一刀一枪打出来的，不是白捡的。王夫之看重的，是陈霸先在江南残破之际，仍能撑住南朝不倒的那份「撑」的功夫。可他也点出陈霸先的短处——袭杀王僧辩，是「背盟」之举，开了乱头。一个靠背盟上台的皇帝，他的江山，从根子上就缺了一份「信」字。大意据《读通鉴论》卷十六。

胡三省在《资治通鉴音注》里，对陈朝纪事的批注，往往落在「弱」与「裂」二字上。胡三省生在宋元之际，亲历亡国之痛，他注《通鉴》时，对这种「地蹙势弱、内斗不休」的末世政局格外敏感。他批注陈霸先袭杀王僧辩，意在前车：王僧辩一死，梁室再无柱石；陈霸先虽得了位，可王僧辩的旧部、北齐的兵锋，从此成了陈朝甩不掉的包袱。胡三省的潜台词是：你今天杀的是盟友，明天反你的就是盟友的旧部。大意据《资治通鉴音注》陈纪部分。

司马光在《资治通鉴》里记陈霸先建国，叙事克制，少发议论。可他在《陈纪》开篇的那段「臣光曰」式的评述里——大意据《资治通鉴·陈纪》，强调陈霸先的功业是「因乱而起」，而其失在于「以诈力取国」。司马光的史观，一贯是「正统」与「名分」，他认为君臣之伦不可乱，禅让是乱臣贼子的把戏。可在陈霸先这件事上，司马光也留了几分余地——他记陈霸先的勤政、节俭、安抚江南，是肯定；记他袭杀王僧辩、行篡逆之事，是贬。一个篡位者，能在二十一个月里让江南稍安，已是乱世里的难得。司马光的高明，是不一棍子打死，也不替他粉饰——史家之笔，该褒则褒，该贬则贬。

## 问道悟道

陈霸先建国留给后人的，是两条反复在历史上演的规律。

**第一条：盟友反目的代价。**

陈霸先与王僧辩，本是平侯景的双璧，是患难里的盟友。可一旦到了立储之争、到了南北博弈，盟友转眼就成了死敌。陈霸先袭杀王僧辩，是先下手为强；可这一杀，开了一个乱头——王僧辩的旧部王琳据湘州反，王頠奔北齐引兵南下，陈朝前期的内战，几乎都跟这次背盟有关。一个靠背盟上台的政权，它的合法性从根子上就存疑，它的内部就永远缺一份「信」字。

盟友反目这事，历史上有几处清晰的例子。东汉末，袁绍与曹操本是讨董的盟友，可一旦到了争中原，反目成仇，官渡一战定生死——《三国志·武帝纪》。唐末，朱温与李克用本是镇压黄巢的盟友，可转头就互攻，梁晋相争数十年，最后李克用之子李存勖灭梁——《旧五代史·梁书》。最切近的，是太平天国——洪秀全与杨秀清起事时是结义兄弟，定都天京后杨秀清权势过盛，洪秀全密令韦昌辉诛杨，韦昌辉屠东王府，翼王石达开负气出走，天国由此转衰。陈霸先、袁绍与曹操、朱温与李克用、洪杨，都是盟友反目，都付出了大代价。盟友之所以能成盟友，靠的是「利同」；盟友之所以反目，多半也是因为「利不再同」。可一旦反目，前头共患难攒下的那点信任，往往就成了被对方捅进来的刀——因为对方最知道你的软肋在哪里。

**第二条：末世开国的先天不足。**

陈霸先建国，接的是南朝最烂的一个摊子——江南残破、江陵新陷、版图最小、强敌环伺。他自己又无子可传，二十一个月就病逝。这样一个「先天不足」的政权，从建国那天起，就埋下了速亡的种子。果然，陈朝三十三年而亡，是南朝四代里最短的一代（宋六十年、齐二十三年、梁五十五年、陈三十三年）。

末世开国的先天不足，历史上有不少例子。蜀汉昭烈帝刘备，在夷陵之战后白帝城托孤，国力大损，蜀汉从一开始就是个偏安弱国，诸葛亮六出祁山也补不回来——《三国志·先主传》。东晋元帝司马睿，衣冠南渡后靠王导、王敦撑场面，皇权旁落，门阀专权，东晋一百余年始终是个「王与马共天下」的弱势政权——《晋书·元帝纪》。南宋高宗赵构，靖康之变后南渡称帝，半壁江山、强敌在北，终其一朝始终是个「偏安」之局——《宋史·高宗纪》。陈霸先、刘备、司马睿、赵构，都是在残局里开国的皇帝。他们的共同特点是：起家时手里本钱就薄，强敌压境，内部还未必齐心。这类「先天不足」的开国，往往能撑一时，难撑长久。开国皇帝的本事，不在于抢到多少地盘，在于能不能在残局里把「本钱」养厚——把残破的江南治理好，把内斗的火苗压下去，把成年的继承人留下。陈霸先三样都没做到，所以陈朝的速亡，从他建国那天起就已成定局。

## 结语

一个寒门武将，踩着盟友的尸首、接下江南最烂的残局建立陈朝——陈霸先建国，是南朝禅代剧本的第四次重演，也是「先天不足」一语成谶的开端。
