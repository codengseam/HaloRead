# 资治通鉴·后晋纪六·出帝亡国

## 讲事情

后晋的亡国，是从一句硬话开始的。出帝石重贵不愿再给契丹当孙子，景延广替他把这句硬话撂了出去——可撂话的人没准备好兵，于是这句话成了催命符。

天福七年（942），石敬瑭崩，年五十一岁。石敬瑭的亲子多早夭，遂养兄子石重贵为子。石重贵本是石敬瑭兄长石敬儒之子，过继给石敬瑭为养子（人物血缘核验见 _figure_lineage.md，重贵为敬瑭兄子非亲子）。石敬瑭临终，托孤于宰相冯道与景延广。石重贵即位，是为后晋出帝。大意据《资治通鉴·后晋纪六》《旧五代史·晋书·少帝纪》。

出帝即位，第一桩便是与契丹的礼数之争。石敬瑭在世时，向耶律德光称"儿皇帝"，后晋对契丹是臣是子，年年进贡。石重贵即位后，宰相景延广主张"称孙不称臣"——孙子辈分可以认，臣子的名分不能要。景延广对契丹使节说："先帝是北朝所立，故称臣称子；今上乃中国自立，可称孙，不可称臣。"大意据《资治通鉴·后晋纪六》《新五代史·晋本纪》。这话传到契丹，耶律德光大怒——称孙不称臣，等于后晋要摆脱儿皇帝的附庸名分，这是耶律德光绝不能容忍的。桑维翰闻此，力谏不可，以为"激怒契丹必招大祸"，请出帝照旧称臣以安契丹（桑维翰其人详见《资治通鉴·后晋纪一·石敬瑭借兵建晋》论桑维翰献策割地又以身殉）。出帝不听，反外放桑维翰。从此后晋与契丹交恶。

耶律德光遂以此为借口，自开运元年（944）起，三度南伐。第一次，开运元年正月，契丹军入寇，前锋至澶州。出帝石重贵亲征，契丹军起初得利，可后晋军拼死抵抗，契丹不利而退。大意据《资治通鉴·后晋纪六》。第二次，开运二年（945），契丹再南下，阳城一战，后晋军李守贞、符彦卿等力战，契丹大败，耶律德光险些被擒，乘骆驼狼狈北逃。这是后晋抗契丹最辉煌的一战，可这一胜也麻痹了出帝——他以为契丹不过如此，遂纵情声色，怠于政事。第三次，开运三年（946），契丹再南下。这一次，出帝派姑父杜重威（娶石敬瑭妹宋国公主）为元帅，率大军北上迎敌。杜重威怯懦畏战，又心怀异志，在滹沱河畔率全军投降契丹。大意据《资治通鉴·后晋纪六》《旧五代史·晋书·少帝纪》。后晋主力尽丧，门户洞开。

开运四年（即天福十二年，947）正月，契丹军长驱直入，攻破汴梁。出帝石重贵欲自焚，为左右所救，遂出降。后晋自936年建国，至947年亡，凡二世二帝十一年。大意据《资治通鉴·后晋纪六》《新五代史·晋本纪》。耶律德光入汴，废出帝为负义侯，迁之于黄龙府（今吉林农安一带）。石重贵一行被押解北上，沿途饥寒，公主、宫人皆为契丹军所掠，境况极惨。石重贵在北地苟活十八年，于964年病死于建州——一位中原皇帝，竟以亡国俘虏的身份，客死北荒。大意据《旧五代史·晋书·少帝纪》。

话说回耶律德光入汴后，在汴梁服中原冠冕，改服中国衣冠，改国号"大辽"，欲做中原皇帝。可契丹兵入城后"打草谷"——四出劫掠，纵兵抢夺，汴梁城中百业凋敝，百姓群起反抗。中原各地藩镇、义军纷纷起兵驱契丹。耶律德光见中原不可久据，于947年三月仓皇北撤，归途中病死于杀胡林（耶律德光扶立儿皇帝始末详见《资治通鉴·后晋纪一·石敬瑭借兵建晋》论耶律德光）。契丹一退，中原权力真空，这就给了河东节度使刘知远以可乘之机——947年二月，刘知远在太原称帝，建后汉（刘知远建汉主场在《资治通鉴·后汉纪一·后汉短祚》）。后晋亡，后汉兴，五代又翻过一页。

## 讲人物

**石重贵：硬话出口却无备的出帝**

石重贵本是石敬瑭兄长石敬儒之子，过继给石敬瑭为养子（养子继统，详见 _figure_lineage.md）。石重贵少时随石敬瑭，亦能打仗，可论机变心术，远不及养父。石敬瑭崩，石重贵以养子继位——又是养子继统（养子继统的合法性困境详见《资治通鉴·后唐纪七·明宗之治》，养子继统萌芽详见《资治通鉴·后梁纪一·朱温代唐》）。可石重贵继位时，面对的是个比石敬瑭更难的局面——石敬瑭好歹有"儿皇帝"的屈辱名分换几年太平，石重贵却连这份屈辱名分都不愿守。

石重贵其人，志大才疏。他听景延广的话，撂"称孙不称臣"的硬话，骨子里是不甘为儿皇帝——这份不甘，本无可厚非。可撂硬话得有底气：要么兵精粮足可一战，要么忍辱负重图自强。石重贵两头不靠——既不愿称臣，又不修武备；既抗了契丹两次，又在阳城大捷后纵情声色。开运二年阳城一战，后晋军险些擒了耶律德光，这本是后晋扭转局面的良机，可石重贵不乘胜经营，反以为契丹不足虑。等到开运三年契丹再南下，他派姑父杜重威为帅——杜重威是皇亲国戚，可也是庸才，一战降敌，葬送了后晋最后的主力。大意据《资治通鉴·后晋纪六》《旧五代史·晋书·少帝纪》。石重贵亡国，亡在硬话出口而无备——撂话是慷慨的，可光有慷慨撑不起一个王朝。

石重贵亡国后的境况，是中原皇帝最不堪的落幕。他被废为负义侯，迁于黄龙府，沿途宫眷被掠，自己沦为阶下囚。一位中原皇帝，竟以亡国俘虏身份客死北荒——这份屈辱，比石敬瑭称儿皇帝更甚。石敬瑭是主动屈膝换帝冠，石重贵是亡国被俘客死异乡。父子两代，一个以屈辱得国，一个以屈辱亡国——后晋的兴亡，成于一屈，毁于一硬，可谓莫大的讽刺。

**景延广：撂硬话却撑不住的权臣**

景延广是后晋权臣，字航川。石敬瑭在世时，景延广已受重用；出帝即位，景延广以托孤大臣身份执掌朝政，势倾朝野。出帝即位后的"称孙不称臣"，正是景延广一手主张。大意据《资治通鉴·后晋纪六》《新五代史·景延广传》。

景延广撂硬话，骨子里是想替后晋洗掉儿皇帝的屈辱名分。他对契丹使节那番话——"先帝北朝所立，故称臣；今上中国自立，可称孙不可称臣"——逻辑上说得通，可逻辑通不等于实力够。景延广以为后晋有阳城大捷在前，可撑住"称孙不称臣"的体面；可他没算到，体面是要兵撑的，后晋的兵根本撑不起这份体面。桑维翰力谏称臣以安契丹，景延广不听，反排挤桑维翰出朝。结果契丹三度南伐，后晋亡国——景延广的硬话，成了催命符。

景延广其人，亦非全无才能。阳城一战，后晋军能死战，与景延广督军不无关系。可他志大才疏，撂硬话而无备，排异己而失众心。后晋亡后，景延广被契丹所擒，耶律德光责让他"主张称孙不称臣"之事。景延广无辞以对，自扼而死——一说是被杀，但《新五代史》从其自扼说。大意据《新五代史·景延广传》。景延广一生，撂硬话是其狂，自扼而死是其悔——硬话出口容易，撑住却难。他与桑维翰恰成对照：桑维翰献媚策割地是其罪，以身殉国是其节；景延广撂硬话抗敌是其勇，撂话无备误国是其失。一个以媚误国，一个以硬误国，殊途同归于亡国之祸——这正是五代权臣进退两难的两副面孔。

**耶律德光：灭晋称帝却死于归途的辽太宗**

耶律德光其人，在《资治通鉴·后晋纪一·石敬瑭借兵建晋》已详写其扶立儿皇帝始末（前文已述）。本篇侧重其灭晋称帝又北撤身死的一段。947年正月，耶律德光入汴，灭后晋，废出帝为负义侯。耶律德光本欲长据中原——他改服中国衣冠，在汴梁即皇帝位，改国号"大辽"，以"大辽皇帝"自居。大意据《辽史·太宗纪》《新五代史·四夷附录》。

可耶律德光低估了中原的反抗。契丹兵入城后"打草谷"四掠，激起中原军民奋起，各地义军、藩镇纷纷起兵驱契丹。耶律德光见中原不可久据，于947年三月仓皇北撤。归途中，耶律德光病死于杀胡林（今河北栾城一带），年四十六岁。大意据《辽史·太宗纪》。耶律德光扶立儿皇帝、灭后晋、入汴称帝，本是为长久据有中原；可他死于归途，他的"大辽"在中原的统治只维持了几个月。耶律德光的悲剧，是"扶得起儿皇帝，坐不稳中原帝位"——他可以靠兵变扶一个傀儡，却无法靠兵变统治一个文明体。契丹真正想要据有中原，还要等到后来金、元两朝——而那两朝的据有，亦都建立在燕云已失的根基上。耶律德光灭晋而身死，是契丹南下中原的第一次受挫，亦是中原反抗之力的一次证明。

## 讲背景

出帝亡国这事，得放在五代速亡的节奏、契丹南下的深入、主战主和的拉锯、儿皇帝名分的进退困境四重背景里看。

先看五代速亡的节奏。五代是"以兵变得位、以兵变失位"的循环。后梁十六年（907—923），后唐十四年（923—936），后晋十一年（936—947），后汉四年（947—950），后周十年（951—960）——越到后来，王朝越短命。后晋十一年，已算中等。可这十一年的前七年是石敬瑭称儿皇帝换来的"太平"，后四年是石重贵抗契丹的苦战——真正的"治世"几乎没有。后晋之亡，是五代速亡律的典型——立国靠借外兵，亡国亦因借外兵反噬；兴亡都系于契丹一念。后晋亡后，后汉不过四年即亡（详见《资治通鉴·后汉纪一·后汉短祚》），后周十年亦亡于陈桥兵变（详见即将续写的《资治通鉴·后周纪五·世宗柴荣》）。可见五代之速亡，根子在"以兵变得位"——兵变可立君，亦可废君；兵权在将不在君，王朝便无法长治。

再看契丹南下的深入。石敬瑭割燕云后，契丹骑兵不再受长城阻隔，自幽蓟南下一马平川（燕云失守的千年北防详见《资治通鉴·后晋纪一·石敬瑭借兵建晋》论燕云失守与千年北防）。后晋之亡，是契丹借燕云通道南下的第一次成功——947年契丹入汴，是契丹第一次灭亡一个中原王朝。此后契丹、女真、蒙古相继南下，都以燕云为跳板。后晋亡国，是燕云之失的第一个直接后果——石敬瑭割燕云才十一年，契丹便借此灭了他的王朝。可见割燕云之祸，不是远祸，是近祸；不是后代的祸，是当世的祸。石敬瑭亲手埋下的雷，十一年就在他养子脚下炸了。

再看主战主和的拉锯。后晋朝廷上，景延广是主战派，桑维翰是主和派。景延广主张"称孙不称臣"，以为后晋有阳城大捷可一战；桑维翰力主称臣安契丹，以为激怒契丹必招大祸。两派之争，本质上是"屈辱求存"与"宁折不弯"的路线之争。后晋出帝听景延广的，撂硬话抗契丹——这本是中原王朝摆脱附庸名分的正当诉求。可正当的诉求，需有相应的实力支撑。后晋兵虽能战，却不足以长期抗契丹；后晋财政虽勉强维持，却撑不起连年大战。景延广的主战，是"无备之战"；桑维翰的主和，是"屈辱之和"——两条路都通向亡国，只是亡法不同。可见主战主和之争，关键不在战与和本身，而在战有战备、和有和本——无备之战与无本之和，同样误国。

最后看儿皇帝名分的进退困境。石敬瑭称儿皇帝，是后晋立国的根基，亦是后晋亡国的祸根。称儿则国体沦为附庸，不称儿则契丹翻脸——这是个进退两难的困局。石敬瑭在世时，靠称儿保命；石重贵即位后，欲摆脱儿皇帝名分，遂招契丹三度南伐。可见儿皇帝名分，是个套上了就摘不下的枷锁——戴上则失主权，摘下则失性命。石敬瑭给后晋套上这枷锁时，只算自己一代的帝位，没算后世的进退。等到石重贵想摘摘枷锁，枷锁已经勒进了肉里。这份进退困境，是儿皇帝体制的根本代价——以主权换存续，存续是借来的，摘不摘枷锁都不由自己。

## 讲道理

王夫之在《读通鉴论》卷二十九（五代中）里论后晋之亡，话亦极痛。大意据王夫之说，景延广主张"称孙不称臣"，本是为后晋洗儿皇帝之耻，志不可谓不正；可志大才疏，撂硬话而无备，反速其祸。王夫之以为，后晋之亡，亡于石敬瑭割燕云种下的根——儿皇帝体制是个枷锁，戴上摘不下；石重贵想摘，摘不动，反被勒死。王夫之把石敬瑭与石重贵并论，以为父子两代，成于屈膝、毁于硬撑——一个以屈辱得国，一个以屈辱亡国，是儿皇帝体制必然的结局。大意据《读通鉴论》卷二十九。王夫之亦论杜重威降敌，以为后晋之亡，杜重威一身之罪虽大，根子仍在石敬瑭割燕云——燕云一失，契丹随时可南下，后晋始终处于被动挨打的境地，纵有良将，亦难回天。

胡三省在《资治通鉴音注》后晋纪部分，对景延广"称孙不称臣"的批注，亦多带惋惜。大意据胡三省注，景延广之志在洗儿皇帝之耻，可志大才疏，无备而撂硬话，反速后晋之亡。胡三省以为，主战主和之争，关键不在战与和，而在战有战备、和有和本——后晋两头不靠，故亡。胡三省亦注阳城大捷，以为后晋本有扭转之机，可出帝不乘胜经营，反纵情声色，错失良机。大意据《资治通鉴音注》后晋纪部分。胡三省的潜台词是：后晋之亡，非一战之失，乃全程的进退失据——立国靠屈膝，抗敌无备，胜而不进，败则降敌，每一步都错。

司马光在《资治通鉴·后晋纪六》里对出帝亡国的记述，亦带其一贯史观。大意据司马光说，出帝石重贵以养子继位，又欲摆脱儿皇帝名分，志不可谓不正；可志大才疏，硬话出口而无备，遂速其祸。司马光以为，后晋之亡，亡于石敬瑭割燕云称儿皇帝种下的根——这个根，是石敬瑭一代种下、后晋一代承受的祸。司马光的史观，是"治乱在人主之德"——石重贵无德无能，故亡国；可司马光亦深责石敬瑭之先罪，以为养子继统与儿皇帝体制的祸根，是石敬瑭一代埋下的。大意据《资治通鉴·后晋纪六》。三家的史观，殊途同归：后晋之亡，是儿皇帝体制的必然结局——屈膝得国者，终以屈膝亡国；硬撑抗敌者，无备则速亡。

## 问道悟道

出帝亡国这出戏，藏着两条让人开眼的规律。

**短命王朝的速亡律。**

五代王朝，一个比一个短命——后梁十六年、后唐十四年、后晋十一年、后汉四年、后周十年。越到后来，王朝越短命。这份速亡律，根子在"以兵变得位"——兵变可立君，亦可废君；兵权在将不在君，王朝便无法长治。后晋之亡，是速亡律的典型：立国靠契丹兵变上位，亡国亦因契丹兵变反噬——兴亡都系于外兵一念。石敬瑭借契丹兵得国，十一年后契丹兵灭了他的王朝；杜重威率军降敌，亦是兵权在将的写照。可见借外兵立国者，外兵反噬是迟早的事；兵权在将者，将降则国亡。

短命王朝的速亡律这事，历史上有几个对照。秦，二世而亡（前221—前207），十五年——秦以武力得天下，亦以苛政失天下，是速亡律的上古典型。西晋，统一不过三十七年（265—316），八王之乱即亡——西晋以篡代得国，内乱即亡（西晋之亡详见《资治通鉴·晋纪五·八王之乱》）。南朝陈，三十三年（557—589），亡于隋——陈以篡代得国，外敌一击即亡（陈亡于隋详见《资治通鉴·陈纪九·陈后主亡国》）。秦、西晋、陈、后晋，都是速亡律的典型——立国不正者，根基不固，内乱外患一来即亡。可见速亡律的根子，在"得国不正"——以篡代、以兵变、以借外兵得国者，合法性不足，朝野不服，王朝便难长久。要破速亡律，须以正道得国、以制度固本——可五代武夫重兵变轻制度，速亡律难破。要到宋兴，以杯酒释兵权、立文制武，速亡律才得以化解——可那已是后晋亡后十三年的事了。

**称孙不称臣的进退失据。**

景延广主张"称孙不称臣"，撂硬话抗契丹。这份硬话，骨子里是中原王朝摆脱附庸名分的正当诉求。可撂硬话得有底气——要么兵精粮足可一战，要么忍辱负重图自强。后晋两头不靠——既不愿称臣，又不修武备；既抗了契丹两次，又在阳城大捷后纵情声色。这份进退失据，是后晋亡国的直接原因。称孙不称臣，本是个折中——既想保体面，又想避战祸；可折中往往两头不讨好——体面保不住，战祸也避不开。景延广的硬话，是典型的"无备之战"——撂话是慷慨的，可光有慷慨撑不起一个王朝。

称孙不称臣的进退失据这事，历史上有几个对照。南宋，韩侂胄主张北伐金朝，撂硬话抗金——可南宋兵不足、将不精，开禧北伐大败，韩侂胄被杀，南宋反割地赔款，是"无备之战"的典型。大意据《宋史·韩侂胄传》。明末，崇祯帝面对后金（清），既不愿和议，又无力决战，进退失据，最终亡国——崇祯帝的"既不和又不战"，与景延广的"称孙不称臣"前后呼应。大意据《明史·庄烈帝纪》。南宋韩侂胄、明末崇祯、后晋景延广，都是进退失据的典型——撂硬话容易，撑住却难；不愿屈膝又无力抗敌，便只能两头落空。可见进退失据的根子，在"志大才疏"——有志无才，有愿无力。要免此祸，须量力而行——可量力者往往被骂为软弱，硬撑者往往被赞为气节，于是进退失据者代代不绝。后晋之亡，亡于硬话出口而无备——这份教训，是给所有"撂硬话者"的一面镜子：话出口前，先摸摸自己的兵够不够。

## 结语

一句"称孙不称臣"的硬话，撑不起一个王朝的脊梁——后晋亡于进退失据，成于屈膝、毁于硬撑，是儿皇帝体制必然的结局。
