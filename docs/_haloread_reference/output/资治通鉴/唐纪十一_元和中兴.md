# 资治通鉴·唐纪十一·元和中兴

## 讲事情

安史之乱后，藩镇割据折腾了半个世纪，到宪宗一朝，终于硬气了一回。削西川、平镇海、克淮西、定淄青，四方藩镇一时俯首，史称"元和中兴"。可这中兴像一盏回光——宪宗一死，藩镇复叛，唐廷又跌回割据的泥潭。宪宗本人，也死在宦官的刀下。

这事要从永贞内禅说起。贞元二十一年，公元805年正月，德宗崩，太子李诵即位，是为顺宗。顺宗中风失语，不能视事，倚翰林待诏王叔文、王伾用事。王叔文是越州山阴人，以棋待诏进身，顺宗为太子时便与他亲厚。顺宗即位，王叔文与王伾主政，推行新政——罢宫市（宫中宦官强买民物的弊政）、罢五坊小儿（宦官以搜捕珍禽为名勒索）、惩贪官、抑制宦官与藩镇。又引刘禹锡、柳宗元等入朝，时号"二王刘柳"。大意据《资治通鉴·唐纪二百三十六》《旧唐书·王叔文传》。这是中唐少有的一次革新，史称"永贞革新"。

可革新触了宦官的逆鳞。宦官俱文珍、刘光琦掌神策军，王叔文欲夺其军权，以范希朝为神策军使。宦官大惧，密谋废立。王叔文又轻躁，与韦执谊等结党，朝士多不附。元和元年——顺宗在位仅八月，宦官俱文珍逼顺宗内禅，传位太子李纯，是为宪宗。顺宗被尊为太上皇，次年崩。王叔文、王伾贬死，刘禹锡、柳宗元等八人远贬为司马，史称"二王八司马"。大意据《资治通鉴·唐纪二百三十六》。永贞革新，百日而败——败在革新者无兵权，宦官有神策军。

宪宗李纯即位，年二十八，正是锐意有为之时。他先用宰相杜黄裳。杜黄裳劝宪宗以法度裁制藩镇，"非振作威令，无以慑服四方"。大意据《旧唐书·杜黄裳传》。元和元年，公元806年，西川节度使韦皋死，其支度副使刘辟自为留后，求朝廷节钺。宪宗初即位，姑息许之。刘辟益骄，又求都统三川。宪宗怒，用杜黄裳策，以高崇文率兵讨之。高崇文是幽州人，骁勇善战，长驱入蜀，擒刘辟，传首长安。这是宪宗削藩的第一仗。大意据《资治通鉴·唐纪二百三十七》《旧唐书·高崇文传》。元和二年，公元807年，镇海节度使李锜据润州反。李锜是宗室，久镇浙西，骄横不法。宪宗发兵讨之，李锜为部下所擒，械送长安处死。两战皆捷，宪宗声威大振。

最难的一仗，是淮西。淮西在蔡州（今河南汝南），自安史之乱后由李希烈、吴少诚、吴少阳相继割据，"有地虽少，然地势险要"，是割据河北与河南的钉子。元和九年，公元814年，淮西节度使吴少阳死，其子吴元济匿丧不报，自领军务，发兵四出焚掠。宪宗发诸道兵讨之。可淮西军悍，官军久攻不下，相持数年。大意据《资治通鉴·唐纪二百三十九》《旧唐书·吴元济传》。

元和十年，公元815年六月，宰相武元衡上朝，至靖安坊东门，刺客突起杀之，割其首而去。裴度亦遇刺，伤首堕沟，赖毡帽厚得不死。这是藩镇王承宗、李师道派刺客干的——武元衡主平淮西，藩镇恨之入骨。大意据《资治通鉴·唐纪二百三十九》《旧唐书·武元衡传》。宰相被刺于京师，朝野震动。有人劝宪宗罢兵，宪宗怒曰："度全，必能平贼。"——裴度若保全，必能平贼。即以裴度为相。裴度主战益坚，淮西之役，不可半途而废。

元和十二年，公元817年，裴度自请督师淮西。宪宗以裴度为宰相兼彰义节度使、淮西宣慰招讨使，赴郾城督师。裴度到前线，奏罢诸道监军的宦官中使，使诸将得专兵权。十月十五日，唐邓节度使李愬趁大雪夜，率军九千自文城出发，长驱奔袭蔡州。"风雪阴晦，旌旗裂，人马冻死者相望"，李愬冒死督军。半夜至城下，蔡军不备。李愬令将士登城，黎明入城，吴元济仓皇不及御，被擒于牙城。传首长安。淮西平。大意据《资治通鉴·唐纪二百四十》《旧唐书·李愬传》。雪夜入蔡州，是军事史上奇袭的经典——风雪之夜长途奔袭，趁敌不备一鼓成擒。

淮西既平，淄青李师道震恐。李师道是平卢淄青节度使，割据齐鲁，曾遣刺客杀武元衡。元和十三年，公元818年，宪宗发兵讨之。李师道惧，欲献地归顺，又犹豫不决。其部将刘悟反戈，杀李师道，传首长安。淄青平。大意据《资治通鉴·唐纪二百四十》。至此，"河朔三镇"亦遣使入朝谢罪（虽未真服，然一时不敢抗命）。四方藩镇俯首，"元和中兴"达于鼎盛。

可这鼎盛，转眼即逝。宪宗削藩成功，靠的是他的刚明果断与裴度等人的死力。可宪宗晚年，信方士，服丹药，性情暴躁，动辄捶打左右。元和十五年，公元820年正月，宦官陈弘志（一作王守澄）弑宪宗于中和殿，对外宣称"药发暴崩"。大意据《资治通鉴·唐纪二百四十一》《旧唐书·宪宗本纪》。一代中兴之主，死于宦官之手。宪宗一死，穆宗柔弱，河朔三镇复叛，"元和中兴"的成果一朝尽失。这中兴，是唐朝最后的一盏回光——亮得猛，灭得也快。

## 讲人物

**李纯：削平藩镇却死于宦官的中兴之主**

宪宗李纯是顺宗长子，永贞内禅即位，年二十八。他是中唐最有作为的皇帝——刚明果断，知人善任，削平四方藩镇，缔造了"元和中兴"。宪宗的刚明，在于他敢用兵。中唐的皇帝，从代宗到德宗，对藩镇多是姑息——怕一打就乱，索性放任。宪宗不同，他认准了"以法度裁制藩镇"，西川、镇海、淮西、淄青，一个个打下来。宰相武元衡被刺，他不动摇；平淮西相持数年，他不罢兵。这份定力，是中唐皇帝里少见的。

可宪宗的悲剧，在于他刚明却不察身边的祸根。他削藩成功，靠的是裴度、李愬这些文武；可他身边，是宦官掌神策军。宦官拥立他即位（俱文珍逼顺宗内禅），也最终杀了他。宪宗晚年信方士柳泌，服丹药，性情大变，"暴怒无常"，左右动辄被打被杀。宦官陈弘志惧，遂弑宪宗。大意据《旧唐书·宪宗本纪》。宪宗一生对外刚明，对内却栽在宦官手里——他削了藩镇的权，却没削宦官的权；宦官的权，比藩镇更近，更要命。宪宗之死，是"中兴之主死于家奴"的悲剧——他能制外，却不能制内。

**李愬：雪夜入蔡州的奇袭名将**

李愬是中唐名将李晟之子。李晟曾平朱泚之乱，收复长安，封西平郡王。李愬以父荫起家，初不见重。淮西之役相持数年，宰相李吉甫荐李愬为唐邓节度使。李愬到任，故意示弱，"抚其疮痍"，不急着打仗，麻痹吴元济。又优待降卒，得淮西虚实。元和十二年十月，李愬趁大雪夜，率军九千奔袭蔡州。一路风雪，人马冻死，李愬冒死督进。半夜至城下，蔡军全无防备——淮西割据三十余年，官军从未打到蔡州城下，吴元济做梦也想不到有人敢雪夜奔袭。李愬令将士登城，黎明入城，擒吴元济。大意据《旧唐书·李愬传》。雪夜入蔡州，是奇袭战的极致——以"示弱—麻痹—猝击"的连环计，趁敌不备一鼓成擒。李愬之功，不在兵多，在算准了吴元济的"想不到"。后人论奇袭，必提李愬雪夜入蔡州。

**裴度：督师淮西的宰相**

裴度是河东闻喜人，进士出身。元和十年武元衡被刺，裴度亦遇刺伤首，几乎丧命。有人劝宪宗罢兵，裴度伤愈入朝，力主不可罢兵——"淮西，腹心之疾，不可不除"。宪宗以裴度为相。元和十二年，裴度自请督师淮西。他到前线，第一件事就是奏罢监军的宦官中使——诸道军中皆有宦官监军，将领进退皆受掣肘。裴度让诸将得专兵权，李愬方能放手奇袭。大意据《旧唐书·裴度传》。裴度之相，在于他既主战，又知兵；既敢担，又肯放权。淮西之平，李愬是奇袭之功，裴度是督师之相，缺一不可。裴度历仕宪、穆、敬、文四朝，四朝元老，是中晚唐难得的贤相。

**王叔文：百日而败的革新者**

王叔文是永贞革新的核心。他以棋待诏进身，顺宗为太子时便亲厚。顺宗即位，王叔文主政，推行新政——罢宫市、惩贪、抑宦官藩镇，引刘禹锡柳宗元等。可王叔文的革新，败在两条：一是无兵权。他欲夺宦官神策军权，以范希朝为神策军使，可宦官俱文珍先发制人，禁军不从。革新者手中无兵，遇政变便如纸糊。二是结党轻躁。王叔文与韦执谊等结党，朝士多不附，孤立无援。宦官一逼，顺宗内禅，百日革新尽废。大意据《旧唐书·王叔文传》。王叔文的悲剧，是"书生变法无兵权"的悲剧——革新若不先握住兵权，必败于有兵权的宦官。王叔文贬死，刘禹锡柳宗元远贬，二王八司马成了永贞革新的殉葬品。

## 讲背景

元和中兴这事，得放在藩镇割据格局延续、宦官专权与神策军、牛李党争萌芽、以及盐铁专卖与财政四重背景里看。

先看藩镇割据格局延续。安史之乱平定后，河北三镇割据（详见《资治通鉴·唐纪十·安史之乱平定》论藩镇割据）。到宪宗朝，藩镇格局已成——河朔三镇（魏博、成德、卢龙）半独立，淮西、淄青、西川、镇海各有割据。藩镇"赋入自私，不朝献于廷"，父死子继、自署官吏，朝廷的命令出不了长安。宪宗削藩，就是要把这些藩镇重新纳入朝廷的法度。可削藩不易——河朔三镇根深蒂固，宪宗也只能让它们"遣使谢罪"，并未真服；淮西、淄青是硬打下来的。可见藩镇割据，是安史之乱后唐朝的沉疴，宪宗的中兴只是暂时压住了，没除根。

再看宦官专权与神策军。神策军是中晚唐的禁军，由宦官掌之（详见《资治通鉴·唐纪十·安史之乱平定》论宦官专权根在肃宗）。到宪宗朝，神策军已由宦官世掌——俱文珍、吐突承璀、陈弘志、王守澄，皆掌神策军。宦官不仅掌禁军，还监军于诸道。淮西之役，诸道军中皆有宦官监军，将领进退受掣——裴度督师，第一件事就是罢监军。可裴度罢的是前线的监军，罢不了朝中的宦官。宪宗削了藩镇的权，却没削宦官的权；宦官拥立他，也杀了他。永贞内禅是宦官逼的，宪宗之死是宦官干的——宦官废立皇帝，已成中晚唐的常态。可见宦官专权，是比藩镇更近、更致命的祸根。

再看牛李党争萌芽。元和后期，朝中已现朋党之端——李吉甫、李德裕父子为一派（后称"李党"），牛僧孺、李宗闵为一派（后称"牛党"）。元和三年，牛僧孺等对策，直言时政，李吉甫恶之，出牛僧孺等为地方官。这是牛李党争的开端。大意据《旧唐书·李吉甫传》《旧唐书·牛僧孺传》。牛李党争，是中晚唐朝士的内耗——两党互相倾轧，长达四十年，到文宗朝愈演愈烈。宪宗朝朋党尚浅，可已埋下祸根。朋党一开，朝士忙于内斗，无暇治国——这是元和中兴无法持久的内因之一。

再看盐铁专卖与财政。削藩打仗，要钱。中唐财政，靠两样：一是盐铁专卖，二是转运。安史之乱后，第五琦、刘晏改革盐法，实行盐铁专卖，朝廷岁入大增。到宪宗朝，盐利岁入占朝廷岁入之半。淮西之役相持数年，军费浩繁，全靠盐利支撑。可见元和中兴，背后是财政的支撑——有盐利，才打得起仗。可盐铁专卖也是双刃剑——盐利重，则民食贵；民食贵，则百姓困。中晚唐的财政，靠专卖支撑，靠专卖剥削，是饮鸩止渴。

## 讲道理

王夫之在《读通鉴论》卷二十六里论元和中兴，话说得既许又惜。大意据王夫之说，宪宗削藩，是中唐少有的"振作"，可这振作是"力取"而非"德服"——靠兵威压服藩镇，非靠德政感化。力取则难久，宪宗一死，藩镇复叛，"德不足以持之"。王夫之论宪宗之死，大意据王夫之说，宪宗刚明而不知察身边之祸——他能制藩镇，不能制宦官；能平外，不能安内。王夫之把元和中兴放在"回光返照"的位置上——中兴是有的，可这中兴没有根基，是力取的强光，一闪即灭。大意据《读通鉴论》卷二十六。

胡三省在《资治通鉴音注》唐纪部分，对裴度罢监军、李愬雪夜入蔡州的批注，多落在"将得专兵则胜，宦官监军则败"上。大意据胡三省注，淮西相持数年不克，根在宦官监军掣肘；裴度一罢监军，李愬便能放手奇袭，蔡州立下。胡三省对"宦官监军败事"深恶痛绝——他注九节度之溃（详见《资治通鉴·唐纪十》论胡注），又注淮西之胜，前后对照，意在说明：将帅专兵则胜，宦官掣肘则败。胡三省注宪宗之弑，是带着痛的——中兴之主死于宦官，是"家奴反噬其主"。大意据《资治通鉴音注》唐纪部分。

司马光在《资治通鉴·唐纪二百四十一》"臣光曰"里对宪宗有一段议论。大意据司马光说，宪宗之明，在于知人善任、以法度裁制藩镇；宪宗之失，在于信方士、近宦官——明于外而暗于内。司马光写宪宗，是写给宋朝皇帝看的——他要宋帝学宪宗之明，戒宪宗之失。司马光的史观，是"治乱在人主之德"——宪宗之明是德之用，宪宗之失是德之不足；中兴是德之功，速衰是德之亏。大意据《资治通鉴·唐纪二百四十一》臣光曰。

## 问道悟道

元和中兴这出戏，藏着两条让人开眼的规律。

**中兴的代价与回光返照。**

宪宗削平四方藩镇，缔造元和中兴，可这中兴是"力取"的强光——宪宗一死，藩镇复叛，中兴成果一朝尽失。可见中兴有两种：一种是有根基的真中兴，国力真恢复；一种是没根基的假中兴，靠强人一时压制，强人一去便崩。元和中兴属后者——宪宗的刚明是个人之能，不是制度之固；人存政举，人亡政息。中兴的代价，是要把"力取"变成"德服"、把个人之能变成制度之固，否则只是回光返照。

中兴的代价与回光返照这事，历史上有几个对照。西汉末王莽篡汉，光武帝刘秀起兵，扫平群雄，再造汉室，是为"光武中兴"。光武中兴是有根基的真中兴——刘秀减税、省官、释放奴婢，国力真恢复，东汉延续近二百年。《后汉书·光武帝纪》。唐朝元和中兴，是力取的回光——宪宗死，穆宗弱，河朔复叛，中兴速衰。《旧唐书·穆宗本纪》。清朝同治中兴，曾国藩、李鸿章镇压太平天国，洋务运动兴办工业，一时有"中兴"气象。可同治中兴也是回光——根基在汉族督抚的私人武力与列强的扶持，非朝廷真振作；甲午一战，"中兴"原形毕露。《清史稿·穆宗本纪》。光武中兴、元和中兴、同治中兴，都是中兴，可结局不同——光武有根基，所以持久；元和、同治无根基，所以是回光。可见中兴的关键，不在一时削平外患，在能否建立持久的制度根基；力取的中兴是回光，德服与制度的中兴才是真光。

**宦官弑君与家奴之祸。**

宪宗是中兴之主，却死于宦官陈弘志之手。宦官本是皇帝的家奴——掌宫禁、司洒扫，是奴才。可家奴一旦掌了兵权（神策军），就由奴变主，能废立皇帝、弑君篡命。宪宗刚明，能制藩镇，却不能制宦官——因为他削不了宦官的兵权。永贞内禅是宦官逼的，宪宗之死是宦官干的；中晚唐的皇帝，从宪宗起，多死于宦官之手。可见宦官之祸，祸在"家奴掌兵"——家奴无兵，则安分；家奴掌兵，则反噬其主。

宦官弑君与家奴之祸这事，历史上有几个对照。秦朝赵高是宦官，秦始皇死，赵高与李斯矫诏立胡亥，后又弑秦二世，秦速亡。赵高是家奴掌权反噬的最早典型——家奴掌了诏令兵权，便由奴变主，弑君篡命。《史记·秦始皇本纪》。东汉末十常侍专权，宦官张让等杀大将军何进，引董卓入京，东汉名存实亡。这是宦官与外戚之争，宦官胜，却引狼入室，汉亡。《后汉书·宦者传》。唐朝宦官掌神策军，弑宪宗、敬宗、文宗，废立皇帝如弄儿——家奴掌禁军，皇帝成傀儡。这是宦官之祸的极致。《旧唐书·宦官传》。明朝魏忠贤，号称"九千岁"，自称厂臣，排斥东林，祸乱朝政。魏忠贤虽未弑君，然家奴专权之祸，与唐代宦官无异。《明史·宦官传》。赵高、十常侍、唐代宦官、魏忠贤，都是家奴掌权反噬，可结局不同——赵高被秦王子婴诛，十常侍被袁绍诛，唐代宦官终被朱温诛（尽杀宦官于白马驿），魏忠贤被崇祯诛。可见家奴掌权，终必反噬其主；而家奴之祸的根，在君主以家奴掌兵——以奴制主，必受其噬。可见宦官之祸，祸根在"家奴掌兵"四字——家奴无兵则安，家奴掌兵则反；欲除宦官之祸，必先夺其兵权。王叔文革新失败，正在于想夺宦官兵权而未能；裴度督师能胜，正在于先夺前线宦官监军之权。

## 结语

中兴的代价是回光返照、家奴掌兵必反噬其主——元和中兴是"力取的中兴一闪即灭"，也是"中兴之主死于家奴"的关口，唐亡之祸由此埋下。
