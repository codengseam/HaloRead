# 资治通鉴·唐纪十·安史之乱平定

## 讲事情

渔阳鼙鼓动了八年，终于收了场。可这场收场，收得并不干净——叛将摇身一变成了节度使，河北三镇的割据从此扎下根，宦官借平叛之功把持了中枢。安史之乱是平了，可唐朝的命，也平去了一半。

至德二载，公元757年，肃宗在灵武即位已一年（详见《资治通鉴·唐纪九·马嵬驿之变》论太子北上灵武）。这年正月，叛军内部先出了一件大事——安庆绪杀了父亲安禄山。安禄山在洛阳称帝后，双目渐盲，性情暴躁，左右动辄捶打。其子安庆绪惧，与宦官李猪儿合谋，趁夜入帐，以大刀斫杀安禄山，"帐下埋其尸"。安庆绪即位，然暗弱不能服众，叛军人心渐散。大意据《资治通鉴·唐纪三十二》《旧唐书·安禄山传》。父子相杀，是叛军内耗的开端——这桩弑父的戏，后来在史思明父子身上又演了一遍。

肃宗在灵武调兵平叛。他以广平王李俶（即后来的代宗）为天下兵马元帅，以朔方节度使郭子仪为副元帅，又借回纥兵。回纥太子叶护率精骑四千来助。借兵是有代价的——肃宗与回纥约定："克城之日，土地、士庶归唐，金帛、女子皆归回纥。"——城破之后，土地和百姓归唐朝，金银布帛和妇女全归回纥。大意据《资治通鉴·唐纪三十二》。这一约，是后来洛阳两遭回纥劫掠的祸根。

至德二载九月，唐回联军与叛军决战于长安城西香积寺。叛军安庆绪部十万列阵，官军亦十万，回纥精骑绕至叛军之后夹击。一战叛军大败，死者六万余。官军收复长安。十月，广平王、郭子仪率军东进，再收复洛阳。安庆绪北逃，退守邺城。两京既复，肃宗自凤翔回长安，迎太上皇玄宗还京。大意据《资治通鉴·唐纪三十二》《旧唐书·肃宗本纪》。这收复两京的首功，郭子仪居之。可两京虽复，叛军主力未灭——安庆绪犹据邺城，史思明犹拥兵范阳。

史思明这人，是安禄山的同伙，也是安庆绪的催命符。至德二载史思明见官军势大，曾一度降唐。肃宗授其范阳长史，然不放心，欲以兵权换之。史思明惧，于乾元二年，公元759年复反。这年，朝廷命九节度使围邺——郭子仪、李光弼等九路节度使，合兵六十万围安庆绪于邺城。可肃宗不设统帅，怕功臣难制，只以宦官鱼朝恩为"观军容使"监军。六十万兵无人统一调度，又遇城久不下、粮草不继。大意据《资治通鉴·唐纪三十四》《旧唐书·鱼朝恩传》。乾元二年三月，史思明率范阳兵来援，官军无统帅应敌，又逢大风沙，诸军皆溃，"战马万匹，惟存三千；甲仗十万，委弃殆尽"。大意据《资治通鉴·唐纪三十四》。九节度围邺之溃，是平叛中最大的败仗——败不在兵少，败在无统帅、宦官监军。史思明随后进邺城，杀安庆绪，收其众，回范阳称大燕皇帝。

自此叛军由史氏父子接手。史思明凶悍善战，一度再陷洛阳，与官军相持。可史思明也步了安禄山的后尘——上元二年，公元761年二月，史思明欲立少子史朝清，长子史朝义惧而杀父，史思明死于其子之手。大意据《资治通鉴·唐纪三十六》《旧唐书·史思明传》。安禄山被安庆绪杀，史思明被史朝义杀——两代叛首，皆死于亲子之手。父子相残，是叛军速亡的内因。

宝应元年，公元762年四月，肃宗崩，宦官李辅国、程元振拥太子李俶即位，是为代宗。代宗以仆固怀恩为元帅，再借回纥兵东讨。仆固怀恩是铁勒族蕃将，其女嫁回纥登里可汗，借回纥兵最为便利。十月，唐回联军再收洛阳，史朝义大败。广德元年，公元763年正月，史朝义穷途，自林中缢死（一说为部下所杀），传首长安。大意据《资治通鉴·唐纪三十八》《旧唐书·仆固怀恩传》。历时八年的安史之乱，到此平定。

可平定的代价，极重。一是回纥两次劫掠洛阳——收复洛阳后回纥纵兵大掠，死者万计，"府库积畜，荡然无遗"；二是仆固怀恩恐贼平宠衰，奏请以安史降将分帅河北——田承嗣为魏博节度使、李宝臣为成德节度使、李怀仙为卢龙节度使。大意据《资治通鉴·唐纪三十八》。这河北三镇，"赋入自私，不朝献于廷"，父死子继、自署官吏，开了藩镇割据的祸根。安史之乱是平了，可埋下的藩镇之祸，又折腾了唐朝一百五十年。

## 讲人物

**李亨：倚宦官疑功臣的平叛之主**

肃宗李亨是玄宗第三子，马嵬驿后北上灵武即位（详见《资治通鉴·唐纪九·马嵬驿之变》论危局中枢转移）。他在危局中接过唐朝的烂摊子，重用郭子仪、李光弼，借回纥兵，收复两京，续了唐朝一百五十年的命——这份续命之功，不可没。可肃宗这个皇帝，当得并不舒心。他疑功臣、倚宦官，把平叛的权柄交给了宦官。九节度围邺，他不设统帅，以宦官鱼朝恩为观军容使监军——六十万大军无人统一，败于无统帅。他对郭子仪也猜忌，邺城之败后竟召还郭子仪，夺其兵权。

肃宗的悲剧，是危局中的君主之疑。他自己是在危局中靠兵权上位的，所以最怕别人也靠兵权上位。郭子仪功高，他怕；李光弼功高，他也怕。可他不信功臣信谁？信宦官。李辅国、程元振、鱼朝恩，都是肃宗一朝抬起来的宦官。肃宗崩后，宦官李辅国竟对代宗说"大家但内里坐，外事听老奴处分"——陛下只管在宫里坐着，外面的事听老奴安排。大意据《旧唐书·宦官传》。宦官敢对皇帝这样说话，根子就在肃宗开的头——他倚宦官以制功臣，结果功臣被制住了，宦官却制不住了。肃宗续了唐朝的命，也种下了宦官专权的祸根。

**郭子仪：功高而能自全的名将**

郭子仪是华州郑县人，武举出身。安史之乱起，他以朔方节度使率军平叛，是收复两京的首功之将。香积寺之战、收复长安、再复洛阳，郭子仪居功第一。可郭子仪最难得的，不是他的功劳，是他的"自全"。功高震主，是历朝功臣的死局——可郭子仪活到了八十五岁，八子七婿皆显贵，寿终正寝。

郭子仪的自全，靠的是一份"放得开"。他大门不闭，任人出入——有人来见他，他让姬妾出来捧砚，毫无避讳。鱼朝恩派人挖他父亲的坟，他不报私仇，说"臣久主兵，不能禁暴，军士摧毁人坟墓，固亦有之。此乃天谴，非人事也"。大意据《旧唐书·郭子仪传》。代宗召他还朝，他"朝闻命，夕就道"——早上接到命令，晚上就上路，不带一兵一卒。郭子仪把兵权交得干脆，把姿态放得极低。他的儿子郭暧尚升平公主（代宗之女），小两口吵架，郭暧脱口而出："汝倚乃父为天子邪？我父薄天子而不为！"——你以为你爹是天子就了不起了？我爹是不稀罕当天子罢了！公主大怒入宫告状。郭子仪闻讯，立刻绑了郭暧入宫请罪。代宗笑说："鄙谚有之：'不痴不聋，不作家翁。'儿女闺帏之言，何足听也。"大意据《资治通鉴·唐纪四十》。这桩"醉打金枝"的本事，正是郭子仪功高自全的写照——儿子敢说大话，老子赶紧请罪；功越大，姿态越低。郭子仪的自全，是功臣在猜忌之君手下的活法。

**李光弼：军令严整却死于疑惧**

李光弼是契丹族，其父李楷洛是契丹酋长。李光弼治军严，与郭子仪并称"李郭"。太原之战，他以数千兵挡史思明十万众，是平叛中的守城名战。河阳之战，他大破史思明。可李光弼的结局，不如郭子仪。鱼朝恩、程元振屡次催他出兵，他不敢抗命又不愿进，加之朝廷猜忌，他"不敢入朝"，最后"愧愤成疾"而卒。大意据《旧唐书·李光弼传》。李光弼的死，是功臣在宦官掣肘、君主猜忌下的憋屈之死——他不是死于敌手，是死于自己心里的怕。郭子仪放得开，所以自全；李光弼放不开，所以忧愤。同样是功臣，一个善终，一个早逝，分野就在这份"放得开"上。

**仆固怀恩：平叛功臣的功高自毁**

仆固怀恩是铁勒族蕃将，朔方军宿将。安史之乱末，他是平定河北的最后功臣——借回纥兵、灭史朝义，传首长安。可仆固怀恩的结局，是功臣的反面。他平贼后，怕朝廷"狡兔死走狗烹"，留兵自固。宦官骆奉先诬他谋反，朝廷又遣使慰劳兼察之。仆固怀恩惧，竟举兵反，引回纥、吐蕃入寇，最后死于鸣沙。大意据《旧唐书·仆固怀恩传》。仆固怀恩的反，是疑惧逼出来的——他不是真心要反，是怕被杀才反。可这一反，功臣变叛臣，一世英名尽毁。仆固怀恩是郭子仪的对照：同样是平叛功臣，郭子仪功高自全，仆固怀恩功高自毁。仆固怀恩的自毁，是功臣在猜忌环境下的另一种结局——放不开，又信不过朝廷，索性拥兵自保，结果保出了一个反贼的名声。

## 讲背景

安史之乱平定这事，得放在藩镇割据格局、宦官专权与神策军、借兵回纥与外援反噬、以及平叛统帅缺失四重背景里看。

先看藩镇割据格局。安史之乱起于节度使坐大——安禄山一人兼三镇（详见《资治通鉴·唐纪八·渔阳鼙鼓》论胡将与边防依赖）。平叛八年，朝廷为速平贼，又封了一堆节度使。到安史平定，仆固怀恩奏请以安史降将分帅河北——田承嗣魏博、李宝臣成德、李怀仙卢龙。这三镇"赋入自私，不朝献于廷"，父死子继、自署官吏，朝廷不能制。大意据《资治通鉴·唐纪三十八》。藩镇割据的格局，就此形成。可见安史之乱平定，不是平了藩镇，是换了藩镇的主人——由安史叛将，换成了朝廷册封的节度使。可这些节度使，和安禄山并无二致——拥兵自重，割据一方。唐朝中晚期的藩镇之祸，根子就在这次平叛的"姑息"。

再看宦官专权与神策军。肃宗在灵武即位，身边无人可依，唯信宦官。李辅国掌禁军，鱼朝恩为观军容使监九节度之军，程元振亦掌兵权。宦官监军，是肃宗一朝的发明——他不信功臣，信宦官，把军权交给宦官监督。九节度围邺，六十万兵无统帅，以宦官鱼朝恩监之，结果大败。这败，败在宦官不懂军事却握统帅之权。肃宗之后，宦官专权愈演愈烈——代宗朝李辅国、程元振，德宗朝神策军正式由宦官掌之。神策军是中晚唐的禁军，由宦官掌，废立皇帝——宪宗、敬宗、文宗皆死于宦官之手。可见宦官专权，根子在肃宗开的"以宦官制功臣"的头（详见《资治通鉴·唐纪九·马嵬驿之变》论禁军逼宫）。

再看借兵回纥与外援反噬。肃宗借回纥兵平叛，是"驱虎吞狼"之计。可回纥恃功，岁岁索财，每岁赐回纥绢二万匹。更甚者，收复洛阳后，回纥纵兵大掠，死者万计，"府库积畜，荡然无遗"。大意据《资治通鉴·唐纪三十二》。借兵是有代价的——回纥出兵不是为唐朝尽忠，是为财为利。城破之日，回纥先抢金帛女子，再轮到唐朝收拾残局。借兵外援，是饮鸩止渴——不借，平不了叛；借了，引狼入室。回纥之祸，是平叛的附加成本，也是唐朝国力衰弱后不得不借外援的无奈。

再看平叛统帅缺失。九节度围邺，六十万兵，竟不设统帅——这是肃宗的猜忌之过。他怕郭子仪、李光弼功大难制，所以不设统帅，以宦官监军。可六十万兵无统一调度，又遇史思明来援，必败。这是猜忌功臣的代价——君主怕功臣拥兵自重，结果把仗打输了。可见统帅缺失，不是无人可用，是君主不敢用。肃宗的猜忌，让六十万兵白白溃散，又把平叛的时间拖长了数年。

## 讲道理

王夫之在《读通鉴论》卷二十六里论安史之乱平定后的藩镇割据，话说得极痛。大意据王夫之说，朝廷平叛，急于收功，以安史降将帅河北，是"以贼治贼"——贼未真平，只是换了个名目。王夫之论仆固怀恩之反，是"功臣之不能自处，亦君之不能处功臣"——功臣不会自处，君主也不会处功臣，两相疑惧，必至于反。王夫之把仆固怀恩放在"功臣自毁"的位置上——他不是天生反贼，是猜忌之君逼出来的反贼。大意据《读通鉴论》卷二十六。王夫之又论宦官监军，大意据王夫之说，以宦官监军，是"以奴制主"——把统帅之权交给奴才，把功臣放在奴才之下监督，功臣必不服，军必败。九节度之溃，根子在宦官监军。

胡三省在《资治通鉴音注》唐纪部分，对九节度围邺之败、宦官监军的批注，多落在"以不习武之宦官统六军"上。大意据胡三省注，鱼朝恩不识兵事，却握观军容使之权，六十万兵无所适从，安得不败。胡三省生在宋元之际，他对"宦官监军败事"格外敏感——他亲见宋朝权臣误国，对宦官、外行指挥内行深恶痛绝。胡三省注仆固怀恩之反，是带着痛的——功臣平贼，反被猜忌逼反，是朝廷自毁长城。胡三省的潜台词是：君主疑功臣，功臣必自保，自保则成反贼，这是猜忌之君自食其果。大意据《资治通鉴音注》唐纪部分。

司马光在《资治通鉴·唐纪三十八》"臣光曰"里对安史平定后藩镇割据有一段议论。大意据司马光说，朝廷以安史降将帅河北，是"养痈遗患"——图一时之安，留百年之患。司马光写藩镇割据，是写给宋朝皇帝看的——他要宋帝鉴唐之失，勿姑息武人。司马光的史观，是"治乱在人主之德"——肃宗猜忌功臣、倚信宦官、姑息降将，是德之不足；藩镇割据、宦官专权，是德之不足的代价。大意据《资治通鉴·唐纪三十八》臣光曰。

## 问道悟道

安史之乱平定这出戏，藏着两条让人开眼的规律。

**功臣的自全与自毁。**

平叛功臣里，郭子仪功高自全，活到八十五岁寿终正寝；仆固怀恩功高自毁，举兵反叛死于鸣沙。同样是平叛首功，一个善终一个反贼，分野就在"放得开"与"放不开"。郭子仪大门不闭、姬妾捧砚、朝命夕行、绑子请罪——把姿态放得极低，把兵权交得极干脆。仆固怀恩留兵自固、惧而生疑、疑而举兵——把兵权攥得极紧，把退路堵得极死。可见功臣之祸，不在功大，在放不开；不在君疑，在自疑。功越大，越要放得开——放得开是退路，放不开是死路。

功臣自全与自毁这事，历史上有几个对照。战国时王翦率六十万大军灭楚，出征前向秦王政"请美田宅园池甚众"——接连五次求田问舍。有人怪他贪，王翦说：秦王多疑，今空国中之兵专委于我，我不多求田宅为子孙业，秦王能不疑我？王翦以贪自污，是功臣自全的高招——把"贪"作护身符，让君主放心。《史记·白起王翦列传》。汉初韩信功高盖主，却不肯交兵权，又羞与绛灌同列，终被吕后斩于长乐钟室。韩信是功臣自毁的典型——放不开兵权，又放不下身段，结果身死族灭。《史记·淮阴侯列传》。明朝徐达功高，朱元璋赐蒸鹅，徐达含泪食之而卒（传说）。徐达之死，是功臣在猜忌之君下的无奈——纵然自全如郭子仪，也未必都善终。《明史·徐达传》。南宋岳飞功高，却不肯拥兵自重，又触高宗之忌，终被赐死风波亭。岳飞之死，是功臣的另一种悲剧——他不是自毁，是被毁；可根子仍在君主猜忌。《宋史·岳飞传》。王翦、郭子仪、徐达，是功臣自全的活法；韩信、仆固怀恩、岳飞，是功臣自毁或被毁的结局。可见功臣之祸，根子在君主猜忌与功臣自处两端——君主不疑、功臣放得开，则两全；君主疑、功臣又放不开，则两败。

**借兵外援与引狼入室的代价。**

肃宗借回纥兵平叛，约"克城之日，金帛女子皆归回纥"。城破之后，回纥纵兵大掠洛阳，死者万计。借兵是饮鸩止渴——不借，平不了叛；借了，引狼入室。回纥出兵不是为唐朝尽忠，是为财为利。可见借兵外援，是衰弱之国的无奈——自己打不了，就借别人打；可借来的兵，是要付代价的，且这代价往往比预想的更大。

借兵外援与引狼入室这事，历史上有几个对照。西周末年，申侯借犬戎之兵攻周幽王，犬戎入镐京，杀幽王，西周亡。申侯借犬戎，是借兵反噬的最早典型——借来容易送走难，犬戎赖在关中不走，平王不得不东迁洛邑。《史记·周本纪》。唐肃宗借回纥，回纥岁岁索财，两次劫掠洛阳，是借兵反噬的中唐典型。《旧唐书·回纥传》。唐末朝廷借沙陀李克用兵平黄巢，沙陀遂坐大，后来李克用之子李存勖灭梁称帝（后唐）——借兵平贼，借出一个新王朝。这是借兵反噬的晚唐典型。《旧五代史·唐书》。西晋时司马越借鲜卑兵，五胡乱华之祸由此而起——借兵外援，引出了整个南北朝的乱局。《晋书》。申侯借犬戎、肃宗借回纥、唐朝借沙陀、司马越借鲜卑，都是借兵外援，可结局相同——借来的兵，赖着不走；引来的狼，回头咬主。可见借兵外援，是饮鸩止渴——解了一时之渴，种了百年之毒。衰弱之国借外援，必付代价；这代价，往往是引狼入室、反受其噬。

## 结语

功臣自全自毁分两途、借兵回纥引狼入室——安史之乱平定是"功臣在猜忌之君手下的两样结局"，也是"衰弱之国借外援必付代价"的关口，藩镇割据与宦官专权由此扎下根。
