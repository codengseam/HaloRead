# 资治通鉴·汉纪十七·霍光辅政

## 讲事情

后元二年，公元前87年，汉武帝病重。

临终前两天，他下了一道诏：立幼子刘弗陵为皇太子，时年八岁。拜奉车都尉霍光为大司马大将军，受遗诏辅少主；驸马都尉金日磾为车骑将军，太仆上官桀为左将军，搜粟都尉桑弘羊为御史大夫，四人同受遗诏，共辅幼主。

第二天，武帝崩于五柞宫。

八岁的昭帝即位，政事全托给霍光。霍光是霍去病的同父异母弟弟，霍去病是卫子夫的外甥——霍光算是卫霍外戚集团的余脉。武帝选他，一是因为他侍奉自己二十多年，出入禁闼，未尝有过，沉稳谨慎；二是因为他跟卫太子一系有渊源，又跟新立的幼帝没血缘牵扯，用他辅政，各方都能接受。

辅政的头几年，四个人还算相安无事。始元元年（前86），金日磾病逝，剩下三人。上官桀跟霍光本是儿女亲家——上官桀的儿子上官安娶了霍光的女儿，生了个女儿。上官安想把这女儿送进宫当昭帝的皇后，霍光嫌孩子太小（才六岁），没答应。上官安另走门路，通过昭帝姐姐鄂邑长公主的关系，把女儿送进了宫，立为皇后。上官家因此得意，可跟霍光的梁子也结下了。

上官桀、上官安父子转而联合燕王刘旦（武帝第三子，本觊觎皇位）、桑弘羊（因封赏不满）和鄂邑长公主，结成一党，要扳倒霍光。他们让燕王上书，告霍光「专权自恣，疑有非常」，请求入京宿卫。昭帝那年十四岁，看了奏章，扣下不发。

第二天霍光上朝，不敢进殿。昭帝问：「大将军在哪？」上官桀说：「因燕王告发，不敢进。」昭帝下诏召霍光进殿。霍光进殿，摘下帽子叩头请罪。昭帝说：「将军戴好帽子。这奏章是假的，将军无罪。」霍光问：「陛下怎么知道是假的？」昭帝说：「将军去广明校阅郎官，是最近的事；调校尉的事，也才十天。燕王离京师数千里，怎么会这么快就知道？况且将军要真想谋反，用不着靠一个校尉。」

一个十四岁的孩子，把这事看得这么透。在场的尚书和左右侍中听了，都惊了。上官桀一党的人还想再告，昭帝发怒：「大将军是先帝托付的忠臣，谁再敢诬陷，治罪！」

上官桀一计不成，再生一计，密谋由鄂邑长公主设宴请霍光，埋伏甲士杀他，再废昭帝、立燕王刘旦。阴谋败露，元凤元年（前80），霍光先下手，诛上官桀、上官安父子，桑弘羊同谋被族灭，燕王刘旦、鄂邑长公主自杀。从此霍光独揽大权，「威震海内」。

元平元年（前74），昭帝崩，年仅二十一，无嗣。

霍光召丞相以下商议，迎立昌邑王刘贺。刘贺是武帝的孙子，昌邑哀王刘髆之子，按辈分是昭帝的侄子。刘贺入京即位，可这位新皇帝荒淫无度——据《汉书》所记，二十七天内干了一千一百二十七件荒唐事。霍光忧惧，召亲信田延年、张安世商议，决定废帝。

霍光召集群臣于未央宫，开门见山：「昌邑王行昏乱，恐危社稷，怎么办？」满朝文武吓白了脸，没人敢吭。田延年离席按剑，说：「先帝托孤将军，以天下寄将军，是因为将军忠贤能安刘氏。今群下鼎沸，社稷将倾，将军若不决断，汉家宗庙将绝。今日之议，不得有迟疑！谁迟疑，我请剑斩之。」

霍光谢道：「九卿责备我是对的。」于是群臣叩头：「万姓之命在于将军，唯大将军令。」

霍光请出太后（昭帝皇后，霍光的外孙女，年仅十五）坐镇，召刘贺伏前听诏。尚书令宣读奏章，历数刘贺罪状，由太后批准废黜。刘贺被废，削去王号，归故国，赐汤沐邑二千户。前后在位二十七日。

废了刘贺，霍光又选了一个继承人——武帝曾孙刘病已，戾太子刘据的孙子，巫蛊之祸时系于狱中，由丙吉护着活下来，长在民间。霍光先封他为阳武侯，随即即皇帝位，是为宣帝。

宣帝即位时十八岁，霍光仍秉政。宣帝对霍光又敬又畏——每次霍光陪乘，宣帝「如有芒刺在背」。地节二年（前68），霍光病逝，宣帝以帝王之礼厚葬，谥宣成侯。

可霍光一死，霍家的好日子就到头了。霍光的妻子霍显（名显，本霍光侍婢，后为继室）当年为了让女儿当皇后，买通女医淳于衍，毒死了宣帝的发妻许皇后。这事瞒了多年，霍光死后渐渐败露。霍氏一门骄横跋扈，霍光的儿子霍禹、侄孙霍山霍云并居要职，奴仆成群，横行长安。宣帝一步步削夺霍氏兵权，霍氏恐惧，密谋造反。地节四年（前66），事败，霍禹被腰斩，霍显及诸女昆弟皆弃市，霍氏族灭，连坐诛灭者数千家。

一代权臣，死后三年，宗族尽灭。

## 讲人物

**霍光：忠而不退的权臣**

霍光这个人，是汉朝权臣里最矛盾的一个。

他忠。武帝托孤，他受遗诏辅政，二十年兢兢业业，没起过篡逆之心。上官桀谋反，他平了；昭帝无嗣，他立了昌邑王；昌邑王荒淫，他废了；又从民间找来宣帝，把皇位还给了武帝的嫡系血脉。班固说他「受襁褓之托，任汉室之寄，匡国家，安社稷，拥昭立宣，虽周公、阿衡何以加此」——受托辅佐襁褓中的幼主，担起汉室的寄托，匡正国家、安定社稷，拥立昭帝、废立宣帝，就是周公、伊尹也超不过他。这份评价，把霍光抬到了周公的高度。

可他也不退。武帝托孤时他是大司马大将军，到宣帝朝他还是大司马大将军。昭帝十四岁就能识破上官桀的阴谋，说明这孩子聪明，可霍光没还政。宣帝十八岁即位，已是成年人，霍光还是「秉政」，宣帝跟他同车都「如芒刺在背」。他不贪财，不贪色，可他贪的是权——或者说，他放不下权。

霍光的来历，得从霍去病说起。霍去病是卫子夫的外甥，卫青的外甥，武帝朝的冠军侯、骠骑将军，封狼居胥的那位。霍去病出征匈奴时，路过河东，认下了生父霍仲孺，还把同父异母的弟弟霍光带到长安。那时霍光十来岁，霍去病安排他做郎官，从此入仕。霍去病早逝，霍光却凭着谨慎稳重，在武帝身边侍奉二十多年，「出入禁闼二十余年，出则奉车，入侍左右，小心谨慎，未尝有过」——出则奉车，入则侍奉左右，小心谨慎，从没出过差错。武帝晚年选托孤大臣，第一个就选了他。

霍光的谨慎，有个出名的故事。他在宫里行走，每次停步、转身，都踩在同一处，分毫不差。这种近乎刻板的规矩，是他在武帝身边活下来的本事——武帝刻薄寡恩，丞相换了十几个，杀了好几个，霍光能在武帝身边二十年不出错，靠的就是这份谨慎。

可谨慎救得了他生前，救不了他身后。他大权独揽二十年，亲党遍布朝堂，儿子霍禹、侄孙霍云霍山并居显职，妻子霍显毒杀许皇后，家族骄横已极。他活着，没人敢动；他一死，新账旧账一起算。班固说他「不学亡术，暗于大理」——没学问、没手段，看不清大局。这话狠，但也准。霍光看得清怎么辅政，看不清怎么收场。

**上官桀：从亲家到死敌**

上官桀本是武帝朝的老臣，骁勇善战。他年轻时做羽林期门郎，随武帝出行，遇大风，车盖吹不开，他扛着车盖遮风，武帝赏识。后来随李广利伐大宛，因功封安阳侯。武帝托孤，他是四人之一，拜左将军。

上官桀跟霍光本是亲家，可权力面前亲家也靠不住。他儿子上官安想送女儿入宫当皇后，霍光不答应，上官家另走门路办成了，从此两家生隙。上官桀联合燕王、桑弘羊、鄂邑长公主，要扳倒霍光。昭帝识破阴谋，上官桀不死心，密谋兵变，事败被族灭。

上官桀的悲剧，是辅政大臣之间权力分配失衡的典型。武帝托孤四人，本是想互相制衡。可金日磾早死，剩下三人，上官桀和桑弘羊联合对抗霍光，制衡变成了党争。党争一败，就是灭族。

**汉昭帝：少年老成的短命天子**

昭帝刘弗陵八岁即位，二十一岁崩，在位十三年。

他最出名的一件事，就是十四岁识破上官桀的阴谋。一个十四岁的孩子，能从奏章的细节里看出破绽——燕王离京数千里，不可能这么快知道霍光调校尉的事；霍光真要谋反，用不着靠一个校尉。这份判断力，连成年大臣都未必有。

昭帝在位期间，霍光秉政，继续武帝晚年的休养生息政策。始元六年（前81），召开「盐铁会议」，让贤良文学跟桑弘羊辩论盐铁专营的利弊。这场辩论是中国经济思想史上的一件大事，桓宽后来整理成《盐铁论》。昭帝还下诏减田租、免徭役，百姓稍稍喘了口气。

可惜昭帝短命，二十一岁就崩了，没留下子嗣。他的早逝，把皇位继承的难题又抛给了霍光。

**汉宣帝：从牢狱到龙椅**

宣帝刘询（原名刘病已）的身世，是汉朝皇室最坎坷的一支。

他是武帝的曾孙，戾太子刘据的孙子。巫蛊之祸时，他出生才几个月，被系于郡邸狱。廷尉监丙吉可怜这个婴儿，找女囚哺养，护着他活了下来。后来武帝病，望气者说长安狱中有天子气，武帝下令把狱中囚犯全杀了。使者到丙吉管的狱前，丙吉闭门不纳，说「皇曾孙在此，他人无辜死者犹不可，况亲曾孙乎」——皇曾孙在这里，别人无辜被杀都不行，何况是亲曾孙？使者回去告状，武帝却醒悟了，说「天使之然」，大赦天下。刘病已这才出狱，由祖母史家抚养，长在民间。

宣帝长在民间，知道百姓的苦。他即位后，常说「庶民所以安其田里而亡叹息愁恨之心者，政平讼理也」——百姓能安心种地、没有叹息愁恨，靠的是政事公平、诉讼合理。这话是一个长在民间、亲历过底层生活的皇帝才说得出的。

宣帝对霍光，又敬又畏。霍光活着，他「如芒刺在背」；霍光死后，他才亲政。他收拾霍氏，不是公报私仇——霍显毒杀许皇后是真，霍氏骄横谋反也是真。可他下手之狠，也透着多年隐忍的反弹。

## 讲背景

霍光辅政二十年，不是他一个人的事，是武帝留下的政治格局。

武帝晚年托孤，选了四个人，是有讲究的。霍光代表卫霍外戚集团，金日磾是匈奴休屠王太子，降汉后忠心耿耿，代表归附的少数民族将领；上官桀是陇西上邽人，骁勇老将，代表军功集团；桑弘羊是洛阳商人之后，代表理财官僚。四个人出身不同、背景不同，互相制衡，谁也吃不掉谁。这是武帝的政治设计——他怕一个人辅政会篡位，四个人互相盯着，皇权就稳了。

可这个设计有漏洞。金日磾始元元年就病死了，剩下三人。上官桀和桑弘羊联合，对抗霍光，制衡变成了二对一。偏偏昭帝聪明，十四岁就站在霍光这边，上官桀一党反而被铲除。霍光从此独大，制衡彻底失效。

武帝托孤的另一个安排，是把昭帝的姐姐鄂邑长公主留在宫里照顾幼帝。可长公主跟上官桀勾结，成了上官安送女儿入宫的门路，后来又卷入谋反。武帝的设计，人算不如天算。

霍光能独揽大权，还因为他手里有兵。大司马大将军这个职位，是武帝晚年设的，统领天下兵马。霍光既掌政权又掌兵权，朝中无人能制。上官桀想动他，得靠阴谋；阴谋败露，就是灭族。

再说废昌邑王、立宣帝这件事。霍光废帝，在中国政治史上是惊天大事。伊尹放太甲、霍光废昌邑，并称「伊霍之事」，后来成了权臣废立的代名词。霍光敢废帝，因为他手里有兵、有太后（自己的外孙女）背书、有田延年按剑威慑群臣。可这一废，也开了个坏头——权臣能废立皇帝，皇权的神圣性就打了折扣。后来曹操、司马懿废立皇帝，都拿「伊霍之事」做借口。

霍光立宣帝，也有他的算盘。宣帝是武帝嫡系血脉（戾太子刘据一支），立他名正言顺；宣帝长在民间，朝中无党，好控制；宣帝年纪轻（十八岁），霍光还能继续辅政。可霍光没算到，宣帝不是昌邑王那种荒淫之辈，而是一个有主见、有手段的皇帝。宣帝隐忍多年，等霍光死了才动手，这份耐性，比霍光还强。

霍氏之族，根子在霍光。霍光大权独揽二十年，亲党遍布朝野，家族骄横已极。霍光妻子霍显毒杀许皇后，霍光生前知道后没追究，反而替她遮掩——这是霍光最大的失算。毒杀皇后是大罪，遮掩一时，遮掩不了一世。霍光一死，这事就成了霍氏的催命符。加上霍禹、霍云、霍山骄纵不法，奴仆横行，民怨沸腾，宣帝要收拾霍氏，名正言顺。

## 讲道理

司马光在《资治通鉴》卷二十五里有一段「臣光曰」，专论霍光。

司马光说：「霍光之辅汉室，可谓忠矣。然卒不能庇其宗，何也？夫威福者，人君之器也。人臣执之，久而不归，鲜不及矣。」——霍光辅佐汉室，算得上忠了。可最终保不住自己的宗族，为什么？因为威福是君主的器物，臣子执掌太久不归还，很少有不招祸的。

司马光接着说：昭帝十四岁就能识破上官桀的阴谋，「固可以亲政矣」，何况宣帝十九岁即位，「聪明刚毅，识知民情」，「岂可使之终其在位乎」——昭帝早该亲政了，宣帝更是聪明刚毅、深知民情，怎么能让他一直不能亲政？「光久专大柄，不知避去，多置亲党，充塞朝廷，使人主蓄愤于上，吏民积怨于下，切齿侧目，待时而发」——霍光久专大权，不知退避，亲党塞满朝廷，让君主在上面蓄愤，吏民在下面积怨，咬牙切齿等着时机发作。「幸得终其天年，而其子孙骄侈，又足以激怒之，此其所以不免也」——他侥幸得终天年，可子孙骄奢，又足以激怒君主，这就是他免不了灭族的原因。

司马光这段话，把霍光的悲剧说透了：忠是忠，可权不还，祸就埋下了。「威福者，人君之器也」——这句话是整段的核心。权是君主的，臣子借来用可以，用完得还。不还，君主早晚要夺回去，夺的时候就是清算的时候。

班固在《汉书·霍光传》的赞里，给了霍光一个更全面的评价。他说霍光「受襁褓之托，任汉室之寄，匡国家，安社稷，拥昭立宣，虽周公、阿衡何以加此」——这是肯定他的功绩，把他比作周公、伊尹。可班固话锋一转：「然光不学亡术，暗于大理，阴妻邪谋，立女成后，湛溺盈溢之欲，以增颠覆之祸，死财三年，宗族诛夷，哀哉！」——可霍光没学问、没手段，看不清大局，纵容妻子的邪谋，把女儿立为皇后，沉溺于过分的欲望，增加了覆灭的祸患，死后才三年，宗族被诛灭，可悲啊。

班固的「不学亡术，暗于大理」八个字，是千古定论。霍光的「不学」，不是不识字，是不懂进退存亡的大道理。他看得清怎么辅政，看不清怎么收场；看得清怎么掌权，看不清怎么放权。这份「暗」，是权臣的通病——权力在手时太舒服，舍不得放，放不下就招祸。

王夫之在《读通鉴论》里论霍光，有一层更深的批评。他认为霍光废昌邑王、立宣帝，虽是不得已，可「人臣废置其君，终非令名」——臣子废立君主，终究不是美名。霍光开了这个头，后世权臣效仿，都拿「伊霍之事」做幌子。王夫之看的是长远影响：一个霍光废帝是忠，十个权臣废帝就是乱。霍光一人之事小，开的风气大。

王夫之还指出，霍光之祸不在他自己，在他「不知教子」。霍光谨慎一辈子，可儿子霍禹、侄孙霍云霍山骄横不法，他不管；妻子霍显毒杀许皇后，他遮掩。家不齐，族怎么保？王夫之这话点出权臣的另一面——自己谨慎不够，还得管住家人。权力越大，家人越容易骄纵，骄纵就是祸根。

## 问道悟道

霍光这出戏，藏着几条经得起琢磨的道理。

**权力是借来的，借主一走，债就到期。**

霍光的全部权力，来自武帝的托孤。武帝把幼帝托给他，他才有大司马大将军的位子。可这个权力的来源是君主，君主换了、君主长大了，权力的合法性就动摇了。

昭帝十四岁就能亲政，霍光没还；宣帝十八岁即位，霍光还是「秉政」。他大概觉得，自己忠心耿耿，皇帝不会动他。可他忘了，权力不是他的，是借来的。借主（皇帝）长大了，要收回借出去的东西，天经地义。霍光不还，皇帝只能等他死了再收——收的时候，连本带利一起算。

历史上这类例子太多。曹操挟天子以令诸侯，权力够大，可曹丕一篡位，曹操的「忠汉」就成了笑话。诸葛亮辅政，权力也大，可他「鞠躬尽瘁，死而后已」，死后蜀人痛哭，没被清算。差别在哪？诸葛亮放权——他虽秉政，可没把亲党塞满朝堂，没让家人骄横不法，临终还安排了蒋琬、费祎接班，没让权力传给子孙。霍光恰恰相反，亲党遍布、家人骄横，权力成了他家的私产。私产越积越多，皇帝的恨就越积越深。

**废立之权，是双刃剑。**

霍光废昌邑王，当时看是英明——昌邑王确实荒淫。可这事开了个头：权臣能废皇帝。伊尹放太甲，是商朝的事；霍光废昌邑，是汉朝的事。从此「伊霍之事」成了权臣废立的借口。曹操废伏皇后、杀董贵人，司马懿废曹芳，桓温废司马奕，都拿「伊霍之事」做幌子。

霍光废帝是忠，可他立的这个规矩，被不忠的人用了。这就是双刃剑——你用的时候是利器，别人用的时候就是凶器。权臣废立，自己用是忠臣，别人学就是乱臣。霍光大概没想到，他废昌邑王的英明，会成了后世篡权者的遮羞布。

**家不齐，权越大祸越大。**

霍光谨慎一辈子，自己没出大错。可他的家，是个烂摊子。妻子霍显毒杀许皇后，他遮掩；儿子霍禹骄横，他不管；侄孙霍云霍山横行，他不约束。他大概觉得，自己大权在握，家人出点格没事。可他忘了，权力越大，家人的骄纵越招恨；他一死，这些恨全要算账。

《大学》里说「修身、齐家、治国、平天下」，齐家在治国之前。家不齐，国治不了，自己也保不住。霍光治国有功，齐家失败，最终宗族尽灭。这份教训，比他的功业更值得记住。

诸葛亮也辅政，也大权在握，可诸葛亮的家齐——他死后家无余财，儿子诸葛瞻、孙子诸葛尚后来守绵竹战死，一门忠烈。霍光的家不齐，死后族灭；诸葛亮的家齐，死后流芳。同样是权臣，齐不齐家，结局天差地别。

**功高震主，进退失据。**

霍光的功，够大了：受托孤之重，平上官桀之乱，废昌邑王，立宣帝，安汉室二十年。这份功，汉朝开国功臣里没几个比得上。可功越高，主越忌。宣帝跟他同车「如芒刺在背」——这五个字，是功高震主最生动的写照。

功高震主的臣子，出路只有两条：要么急流勇退，像张良那样功成身退；要么把权力传下去，像曹操那样让儿子接班。霍光两条都没走——他不退，也不篡，卡在中间。卡在中间最危险：皇帝忌他，可他没篡位，皇帝不能明着动他；他不退，皇帝只能等他死。等他死了，新账旧账一起算。

功臣要善终，最难的不是立功，是知退。范蠡帮越王勾践灭吴，功成身退，泛舟五湖；文种不肯退，被勾践赐死。同样是灭吴功臣，退的活，不退的死。霍光不是不想退，是退不了——他一退，霍家的荣华富贵全没了，他舍不得。舍不得，就埋下了灭族的祸根。

## 结语

权臣之忠，贵在知退；不知退者，身虽全而族必灭——霍光之祸，不在不忠，在不退。
