# 资治通鉴·唐纪七·开元盛世

## 讲事情

开元盛世，是唐朝最辉煌的二十九年。可这盛世的起点，是两场政变。

第一场是唐隆政变。景龙四年，公元710年六月，韦后毒杀中宗李显，立温王李重茂为少帝，临朝称制，想学武则天再走一遍。相王李旦的第三子李隆基，时年二十六，与太平公主联手，发动唐隆政变——率万骑攻入太极宫，诛韦后、安乐公主、上官婉儿。相王李旦复位，是为睿宗。李隆基以功立为太子。——《资治通鉴·唐纪二十五》。

可睿宗复位，朝政在太平公主手里。太平公主是武则天的女儿，权倾朝野，"宰相七人，五出其门"——七个宰相里五个是她的人。大意据《资治通鉴·唐纪二十五》。太平公主忌太子李隆基英武，谋废之。睿宗夹在中间，先让了一步。先天元年，公元712年八月，睿宗传位太子，李隆基即位，是为玄宗，改元先天。睿宗自己做太上皇，"三品以上除授及大刑政仍决于上皇"——三品以上官员任免和大刑政仍决于太上皇。

这传位不是真退。太平公主还在，太上皇还在，玄宗坐的龙椅是悬的。先天二年，公元713年七月，玄宗先下手——与高力士、王毛仲等率兵诛太平公主一党，赐死太平公主。太上皇睿宗正式归政。这便是先天政变。玄宗改元开元。这一年，玄宗二十九岁。从唐隆政变到先天政变，三年两场政变，玄宗才坐稳了帝位。这两场政变，是开元盛世的前传——没有这两场政变，就没有玄宗的亲政，也就没有开元之治。

玄宗亲政，第一件事是找宰相。开元元年，他想起一个人——姚崇。姚崇本名元崇，字元之，陕州硖石人，历仕武则天、中宗、睿宗三朝，是个老臣。玄宗在渭水之滨召见姚崇，姚崇不拜相，先提了"十事要说"——十条政见，玄宗条条答应，姚崇才拜相。大意据《新唐书·姚崇传》。这十条，大意是行仁政、不任外戚、不任宦官、纳谏、省刑罚、节用、礼接大臣等——是开元之治的施政纲领。姚崇拜相，第一桩大功是灭蝗。开元三、四年，山东大蝗，朝廷里有人主张修德禳灾，不敢捕蝗。姚崇力排众议，主张扑埋焚杀——烧埋扑杀蝗虫。百姓有怨，"杀虫太多，有伤和气"。姚崇说："杀虫救人，祸归于崇，不关公等。"大意据《旧唐书·姚崇传》。姚崇一力主持，蝗灾渐息。这一手灭蝗，是开元之治的第一功——不靠天恩，靠人为。

姚崇罢相，玄宗问谁可代。姚崇荐宋璟。宋璟，邢州南和人，耿直刚正，"守法持正"——按规矩办事，不徇私情。宋璟为相，量才授任，百官不敢干以私。大意据《旧唐书·宋璟传》。宋璟在朝，犯颜直谏，玄宗也敬他三分。开元八年，宋璟罢相。

宋璟之后，张说拜相。张说，字道济，洛阳人，兼文武。开元十一年，张说奏改政事堂为中书门下，下设五房——吏、枢机、兵、户、刑礼五房，分理政务。这是唐代宰相制度的一大改。大意据《资治通鉴·唐纪二十七》。张说又奏改府兵为募兵，长从宿卫——选十二万长从兵，轮番宿卫京师。这是府兵制崩坏、募兵制兴起的开端。开元十三年，公元725年，玄宗封禅泰山。张说主其事。封禅是帝王盛事，自秦皇汉武以来只有几位帝王行过。玄宗封禅，是开元全盛的标志。可张说封禅用人多私——封禅随从多是他的人，封官太滥。中书令张说终被弹劾罢相。大意据《旧唐书·张说传》。

张说之后，宇文融括户。开元九年，宇文融上奏括逃户——清查逃亡隐漏的户口。隋末唐初战乱，户口隐漏甚多，不纳赋税。宇文融为劝农使，分行天下，括得逃户八十余万——八十万户，田亦无数。大意据《旧唐书·宇文融传》。这一手括户，让唐朝的税源大增，是开元盛世的家底。可宇文融为人倾险，结党营私，终被贬死。

开元二十二年，公元734年，张九龄拜相。张九龄，韶州曲江人，岭南第一位宰相，有文名，直言敢谏。开元二十四年，公元736年，安禄山兵败被押送长安。张九龄主张按军法斩之，说："禄山狼子野心，面有逆相，请因其罪诛之，以绝后患。"大意据《旧唐书·张九龄传》。玄宗爱才，赦安禄山，说"卿勿以王夷甫看禄山，枉害忠良"。玄宗没听张九龄的话。这一赦，后来要了唐朝的命。同年，张九龄因直言忤玄宗，罢相，贬荆州长史。张九龄罢相，是开元由盛转衰的拐点。张九龄一罢，李林甫代之独相。

李林甫，唐宗室旁支，为人"柔佞多狡"，"面柔而有狡计"——面上一团和气，肚里全是算计。时人叫他"口蜜腹剑"——嘴里甜，肚里藏着刀。大意据《资治通鉴·唐纪三十》。李林甫为相，第一桩事是杜绝言路。他对谏官说："明主在上，群臣顺之不暇，何用谏？君不见立仗马乎？食三品料，一鸣即斥去，悔之何及！"——你看仪仗马，吃三品料，叫一声就踢出去，后悔来不及。谏官自此噤声。大意据《新唐书·李林甫传》。李林甫又奏请多用蕃将为节度使——理由是蕃将善战，且无党援，可专用。这奏请，是李林甫固宠之计——他怕汉将为相，夺他的位。可这一奏，让安禄山这些蕃将坐大了三镇兵权。大意据《资治通鉴·唐纪三十》。李林甫独相十九年，开元末到天宝末，朝政日坏。

开元二十八年，公元740年，玄宗年过半百，一桩事先静后动——寿王妃杨氏，被度为女道士，号太真。寿王李瑁是玄宗第十八子，杨氏是寿王妃，是玄宗的儿媳。这一桩翁媳之间的变故，是唐宫收继婚的胡风遗存（详见《资治通鉴·唐纪六·二圣临朝》论唐宫收继婚俗）。玄宗宠武惠妃，武惠妃死（737），玄宗郁郁。有人进言寿王妃杨氏美，玄宗见之，命度为女道士，号太真，入太真宫。大意据《旧唐书·杨贵妃传》。这一桩事，在开元末不显，可到了天宝，杨太真还俗入宫，册为贵妃，便是天宝乱事的引子。

开元二十九年，公元741年，开元末。这一年，玄宗改元天宝。从先天政变到开元末，二十九年，玄宗从一个二十九岁的青年，走到了五十六岁的老人。盛极而衰，自此始。

## 讲人物

**李隆基：两场政变起家的盛世天子**

李隆基是睿宗李旦第三子，废太子李成器的弟弟。他被立为太子，不是嫡长——嫡长是李成器。李成器让太子位给李隆基，说"国家安则先嫡长，国家危则先功"——太平时立嫡长，危难时立有功的。李隆基有唐隆政变之功，李成器让位。这是李成器的识相，也是李隆基的运气。李隆基这个太子，是抢来的——不是抢嫡长，是抢来的帝位。

李隆基的能耐，在前半生是英武果断。唐隆政变，他二十六岁，联合太平公主诛韦后；先天政变，他二十九岁，诛太平公主。三年两场政变，他下手快、下手狠。这一份英武果断，是他前半生的本事——没有这两场政变，他坐不稳帝位。李隆基后半生，却是另一副面孔。开元初，他用姚崇、宋璟，从谏如流；开元末，他用李林甫，杜绝言路。同一个人，前半生用直臣，后半生用佞臣——不是李隆基变了，是他坐稳了。坐稳之后，听谏是累赘，享乐是奖赏。李隆基的悲剧，是"享国日久"——皇帝坐太久，就会从明君变成昏君。太宗在位二十三年，临终还能听得进长孙无忌的话；玄宗在位四十三年，开元末已经听不进张九龄的话。享国日久，是明君变昏君的催化剂。

李隆基最深的矛盾，是盛世天子与享乐皇帝的撕扯。开元初，他节制、勤政、纳谏；开元末，他宠李林甫、纳杨太真、信高力士。同一个人，前后判若两人。这撕扯的根，在权力无人制衡——皇帝做久了，没人能挡他享乐。开元盛世是李隆基的功劳，开元转衰也是李隆基的罪过。这便是"盛衰同源"——盛也玄宗，衰也玄宗。

**姚崇：灭蝗救民的实际宰相**

姚崇历仕三朝，是个老练的能臣。他最大的本事，不是纸上谈兵，是办实事。开元三、四年大蝗，朝廷主张修德禳灾的人多——蝗是天灾，捕杀有伤和气。姚崇力排众议，说："彼委骨肉于沟壑，乃以修德禳灾，是欺天也。"大意据《旧唐书·姚崇传》。他一力主持扑杀，蝗灾渐息。这一手灭蝗，是开元之治的第一功——不靠天，靠人为。

姚崇的能耐，在权变。姚崇不是腐儒，他不拘常格。他临终前曾对人说，自己一生不学无术，唯济世用事。大意据。姚崇的权变，是能臣的本事——办成事，比守规矩要紧。可姚崇的权变，也有代价——他纵子受贿，开元四年被人告发，玄宗借机让他罢相。姚崇的罢相，是他纵子的代价，也是他权变的反噬。可姚崇罢相荐宋璟，是识人——他知道自己短在刚直，宋璟长在刚直，让宋璟补他的短。这一荐，是姚崇的格局。

**宋璟：守法持正的诤臣**

宋璟与姚崇并称"姚宋"。宋璟的本事，在刚直。"守法持正"——按规矩办事，不徇私。他犯颜直谏，玄宗也敬他。宋璟为相，量才授任，百官不敢干以私。他选官，不论门第，只论才干——这是开元年间的用人风骨。

宋璟最深的本事，在知进退。开元八年，宋璟因"恶钱"之争（禁私铸恶钱）引起市场骚动，自请罢相。他不恋权——办不成的事，就退。这一点，是宋璟的清白。姚崇纵子受贿被罢，宋璟因恶钱之争自请罢——两个宰相，一个被动罢，一个主动退。宋璟的退，是他比姚崇更清白的地方。宋璟罢相后，闭门养志，寿终正寝。宋璟这一辈子，是诤臣的典范——刚直、清白、知退。可宋璟这一类诤臣，在开元末就断了——李林甫杜绝言路，诤臣再也上不来。

**李林甫：口蜜腹剑的固宠权相**

李林甫是唐宗室旁支，为人"柔佞多狡"。他的本事，不在治国，在固宠。他用了三招：一是杜绝言路——"立仗马"之喻，让谏官噤声；二是结党营私——拉拢宦官、妃嫔，结成党援；三是用蕃将——奏请多用蕃将为节度使，固自己的宠。

李林甫最毒的一招，是用蕃将。他怕汉将为相夺他的位，奏请用蕃将——蕃将善战，且无党援，可专用。玄宗准了。这一准，让安禄山这些蕃将坐大了三镇兵权。李林甫这一奏，是固自己宠的，却是唐朝的催命符。李林甫活着时，安禄山还怕他——"禄山见林甫，虽隆冬常汗洽衣"——安禄山见李林甫，大冬天衣服都汗湿。大意据《旧唐书·安禄山传》。李林甫一死（752），安禄山就不怕谁了。三年后，范阳起兵。李林甫口蜜腹剑，固了自己十九年的宠，却养出了安禄山这头狼。这便是固宠之臣的代价——你固自己的宠，就把王朝的命运搭进去了。

## 讲背景

开元盛世这事，得放在三省六部与政事堂改制、府兵制崩坏与募兵制、节度使与边防重兵、以及蕃将崛起四重背景里看。宦官与外戚的伏笔，也在此埋下。

先看三省六部与政事堂改制。唐初三省六部，政事堂是宰相议事之所。开元十一年，张说奏改政事堂为中书门下，下设五房分理政务。这是宰相制度的重大改制——政事堂从议事之所，变成行政实体。大意据《资治通鉴·唐纪二十七》。这一改，让宰相的行政权加重，可也埋下专相的伏笔——宰相权一重，遇到李林甫这种独相，专权就更难制衡。开元初姚崇、宋璟并相，相互制衡；开元末李林甫独相十九年，无制衡。制度是同一个制度，可人变了，制度的效力就变。可见制度是死的，人是活的——制度能挡肯听的宰相，挡不住专权的独相。

再看府兵制崩坏与募兵制。唐初府兵制——兵农合一，府兵轮番宿卫京师，自备弓矢衣粮。府兵制的基础，是均田制——农民有田，才能自备弓矢衣粮当府兵。可到了玄宗朝，均田制崩坏——土地兼并严重，农民失田，府兵逃亡。开元十年，张说奏改府兵为募兵，长从宿卫——选十二万长从兵，轮番宿卫京师。这是府兵制崩坏、募兵制兴起的开端。大意据《旧唐书·张说传》。府兵制崩坏，京师宿卫靠募兵——这是京师武力衰弱的开端。府兵制下，京师有兵可调；募兵制下，京师宿卫靠募，边防靠边镇。这一改，让唐朝的武力重心从京师移到了边镇——京师空虚，边镇重兵。这是后来安史之乱的制度根源（详见《资治通鉴·唐纪八·渔阳鼙鼓》论府兵崩坏与京师空虚）。

再看节度使与边防重兵。唐初边防用行军总管、大都督，临时调遣。开元天宝间，边防设十大节度使——安西、北庭、河西、朔方、河东、范阳、平卢、陇右、剑南、岭南。节度使总揽一镇军政财权，久任不代。天宝元年，十大节度使兵四十九万，而京师宿卫不满八万——边兵四十九万，京兵八万，外重内轻。大意据《新唐书·兵志》。节度使久任，兵将相附——兵认将，将认兵，渐渐成了私家军。安禄山兼三镇节度使（平卢、范阳、河东），兵十五万——一镇节度使的兵，比京师宿卫还多。这是节度使制度的危险——边将坐大，京师制不住。可见节度使制度，是唐朝边防的需要，也是唐朝的催命符——边防靠节度使，节度使反过来乱唐。

再看蕃将崛起。唐初用汉将，李靖、李勣、苏定方。开元天宝间，蕃将崛起——安禄山（粟特）、哥舒翰（突厥）、高仙芝（高句丽）、封常清（汉人，但起于边事）。蕃将崛起，一是唐廷边防的需要——蕃将善骑射，熟边情；二是李林甫的固宠之计——他怕汉将为相，奏请用蕃将。这两股合力，让蕃将坐大边镇。安禄山兼三镇节度使，便是蕃将崛起的极致。可见蕃将崛起，一半是边防的需要，一半是权相的固宠——可这两股合力，养出了安禄山这头狼。

再看宦官与外戚的伏笔。玄宗朝，宦官高力士得宠。高力士是宦官，不是外戚（详见 prompts/narrative/_figure_lineage.md 提醒：高力士宦官非外戚）。高力士权盛，"太子呼之为兄，诸王呼之为翁，驸马辈直呼爷"——太子叫他兄，诸王叫他翁，驸马叫他爷。大意据《旧唐书·高力士传》。宦官干政，自高力士始，后来唐朝宦官专权废立皇帝，根子就在玄宗朝。外戚方面，开元间杨氏未起，杨贵妃入宫在天宝（745册贵妃），杨国忠用事在天宝末。开元间外戚不显，可宦官已起。宦官与外戚，是唐朝中晚期的两大祸根，根子都在玄宗朝。

## 讲道理

王夫之在《读通鉴论》卷二十三里论玄宗开元盛衰，话说得极重。大意据王夫之说，玄宗之治，前后判若两人——开元用姚崇、宋璟，从谏如流；天宝用李林甫、杨国忠，杜绝言路。同一个人，前后为何如此不同？王夫之的答案是"享国日久"——皇帝坐太久，就会从明君变昏君。玄宗即位二十九年（开元末），已经听不进张九龄的话；即位四十余年（天宝末），已经把朝政交给李林甫、杨国忠。享国日久，是明君变昏君的催化剂。王夫之最叹的是张九龄罢相——张九龄预言安禄山反，玄宗不杀；张九龄直言忤旨，玄宗罢之。这一赦一罢，是开元转衰的拐点。王夫之论玄宗，是把他放在"盛衰同源"的位置上——盛也玄宗，衰也玄宗，盛衰的根，都在皇帝一人的心。王夫之又论，开元之治非制度之功，乃人治之效——姚崇、宋璟在则治，李林甫在则乱；同一制度，治乱悬殊，可见制度因人而异，盛衰由心而定。大意据《读通鉴论》卷二十三。

胡三省在《资治通鉴音注》唐纪部分，对开元转衰的批注，多落在"权柄下移"与"边兵外重"上。大意据胡三省注，玄宗朝边兵四十九万，京兵八万，外重内轻——这是节度使制度埋下的祸根。胡三省生在宋元之际，亲见南宋权臣、外患，他对"外重内轻"格外敏感。胡三省的潜台词是：京师空虚，边镇重兵，一旦边将反，京师制不住——这是王朝覆亡的通病。他注李林甫奏用蕃将时，是带着后见之明的痛——这一奏，养出了安禄山。胡三省的注，是把"外重内轻"看作唐朝催命符的——节度使制度一旦失衡，王朝就难逃倾覆。大意据《资治通鉴音注》唐纪部分。

司马光在《资治通鉴·唐纪三十》记李林甫奏用蕃将一事，"臣光曰"有一段议论。大意据司马光说，李林甫固宠之私，养出安禄山之祸——这是私心误国的典型。司马光写李林甫，是写给宋朝皇帝看的——他要宋帝防权相固宠，不要让私心误国。司马光的史观，一贯是"治乱在人主之德"——玄宗之盛，是德之修；玄宗之衰，是德之不修。盛衰的分水岭，在皇帝一人的心。大意据《资治通鉴·唐纪三十》臣光曰。

## 问道悟道

开元盛世这出戏，藏着两条让人开眼的规律。

**盛世转衰的拐点。**

开元盛世二十九年，盛极而衰。拐点在哪？在张九龄罢相。开元二十四年，张九龄因直言忤玄宗罢相，李林甫代之。这一罢一替，是开元由盛转衰的拐点。张九龄在，言路通；李林甫代，言路塞。张九龄在，诤臣上得来；李林甫代，诤臣噤声。开元盛世的"君明臣直"，是张九龄这些诤臣撑着的；开元转衰，是李林甫这些固宠之臣断的。

盛世转衰的拐点这事，历史上有几个对照。西汉文景之治，盛极而衰，拐点在武帝穷兵。文景蓄养民力，武帝开边耗尽——盛在蓄民力，衰在穷兵。大意据《汉书·武帝纪》。东汉光武中兴，盛极而衰，拐点在章和之后外戚宦官交替专权。光武退功臣进文吏，章和之后外戚宦官专权——盛在退外戚进文吏，衰在外戚宦官专权。大意据《后汉书·光武帝纪》。西晋太康之治，盛极而衰，拐点在武帝死后的八王之乱。太康蓄养，八王耗尽——盛在蓄，衰在耗。大意据《晋书·惠帝纪》。文景、光武、太康、开元，都是盛极而衰，可拐点不同——文景拐在穷兵，光武拐在外戚宦官，太康拐在内乱，开元拐在权相固宠。可见盛世转衰的拐点，各有各的根，可规律一致——盛世的"盛"，是有人撑着的；撑的人一去，盛就转衰。文景之盛是萧何曹参这些与民休息之臣撑的，光武之盛是退功臣进文吏撑的，开元之盛是姚崇宋璟张九龄这些诤臣撑的。撑的人一去，盛就转衰。这是盛世转衰拐点的最深规律——盛世不在皇帝一人，在撑着盛世的人。皇帝用撑盛世的诤臣，则盛世；用固宠的佞臣，则转衰。开元用姚崇宋璟，盛；用李林甫，衰。盛衰的根，在宰相的代谢。

**相位代谢与能臣的枯竭。**

开元二十九年，宰相代谢清晰可见——姚崇、宋璟、张说、宇文融、张九龄、李林甫。这一串代谢，是有规律的：能臣先用、后枯、再被佞臣代。姚崇权变能办事，可纵子受贿被罢；宋璟刚直清白，可因恶钱之争自请罢；张说兼文武改制，可封禅用人私被弹劾罢；宇文融括户增税，可结党被贬死；张九龄直言敢谏，可忤旨被罢。五个能臣，各有各的短，先后被罢。能臣一罢，佞臣代之——李林甫独相十九年。

相位代谢与能臣枯竭这事，历史上有几个对照。汉武帝朝，丞相代谢——窦婴、田蚡、公孙弘、李蔡，先后被罢被杀。能臣先枯，后任用酷吏张汤、佞臣李广利。大意据《汉书·武帝纪》。明万历朝，内阁代谢——张居正死后，张四维、申时行相继，能臣枯，党争起。大意据《明史·张居正传》。汉武帝朝、明万历朝、唐玄宗朝，都是能臣先枯佞臣代——能臣先枯，佞臣趁虚而入。可见相位代谢的规律：能臣有短，先后被罢；佞臣无短（因为他们不办事，不犯错），长期固宠。能臣的短，是他们办事的代价；佞臣的长，是他们不办事的红利。这是相位代谢的最深悖论——办事的能臣先枯，不办事的佞臣久留。开元用姚崇宋璟张九龄，盛；用李林甫，衰——盛衰的根，在相位代谢。能臣代谢顺畅，则盛世；佞臣久留，则转衰。开元盛世是能臣代谢的成果，开元转衰是佞臣久留的代价。

## 结语

开元盛世二十九年，盛在能臣代谢、诤臣撑场；转衰在张九龄罢相、李林甫独相——盛衰同源，根在相位代谢与皇帝一人的心。
