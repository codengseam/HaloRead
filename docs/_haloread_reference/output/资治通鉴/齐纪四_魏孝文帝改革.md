# 资治通鉴·齐纪四·魏孝文帝改革

## 讲事情

要讲魏孝文帝改革，得先说清一桩体例：这件事的主角是北魏的皇帝，可《资治通鉴》是编年体，按年记事，孝文帝改革的大半发生在南齐永明年间，所以这些事被系于「齐纪」对应年份。章节虽叫齐纪，故事却是北方的——这是读《通鉴》南北朝部分要记牢的。

拓跋宏，公元 467 年生，471 年五岁即位，是为魏孝文帝。他登基时太小，朝政由祖母冯太后主持。冯太后是文成帝拓跋濬的皇后，是孝文帝的嫡祖母——孝文帝生母李夫人按北魏「子贵母死」的旧制被赐死，孝文帝自幼由冯太后抚养。这冯太后不简单，她是长乐信都人，汉人出身，临朝称制近十五年，是北魏实际上的掌舵人。公元 476 年，她毒杀献文帝拓跋弘——也就是孝文帝的亲生父亲，把权力攥得更紧。一个祖母，杀了孙子的父亲，又把孙子抚养长大，这关系剪不断理还乱。可正是这位冯太后，给孝文帝改革打下了底子。

冯太后主政时，北魏已经出了一连串动摇国本的毛病。鲜卑骑兵打天下行，治天下不行；官吏没俸禄，靠搜刮度日，贪腐横行；宗主督护制下，户口隐匿严重，朝廷收不上税也征不到兵。冯太后是汉人，懂汉法，她着手用汉制改造北魏。太和八年，公元 484 年，颁行班禄制——给官吏发俸禄，同时严惩贪赃，贪一匹布者处死。太和九年，公元 485 年，采纳李安世的建议，颁行均田制——男子十五岁以上受露田四十亩，妇人二十亩，身死还田，桑田为永业。太和十年，公元 486 年，采纳李冲的建议，颁行三长制——五家立邻长，五邻立里长，五里立党长，校定户口，均平赋役。这三件——班禄、均田、三长——是冯太后的手笔，也是孝文帝改革的前奏。没有冯太后打的地基，孝文帝后来的大动作立不住。

太和十四年，公元 490 年，冯太后去世，孝文帝开始亲政。他守丧三年，哀毁骨立，礼数极尽。这位由冯太后养大的皇帝，对祖母的感情是真切的——尽管祖母杀了他的父亲。亲政后，孝文帝把改革推向更深处。他要做的，不止是几条制度，而是把整个鲜卑族迁离故土、改换面目。

太和十七年，公元 493 年八月，孝文帝以「南伐」为名，亲率步骑三十余万自平城南下。一路阴雨连绵，群臣叫苦不迭。九月到了洛阳，雨更大。群臣跪在泥水里泣谏，请求停止南伐。孝文帝却骑在马上说——「今者兴发不小，动而无成，何以示后？若不南伐，即当迁都于此。」《魏书·高祖纪》。现在我们兴师动众，劳而无功，怎么向后人交代？要是不继续南伐，那就迁都到这里！群臣本就不愿南伐送死，一听只要迁都就能不南伐，纷纷同意。任城王元澄——拓跋澄，孝文帝的宗室长辈——最先支持。这场「南伐」，从头到尾是个幌子，真正的目的是迁都。孝文帝不敢直说要迁都——平城是鲜卑老巢，贵族在平城有根有业，谁都不愿走——只能用「南伐」逼群臣就范。这一手，是政治上的高明，也是无奈。

太和十八年，公元 494 年，正式迁都洛阳。从此北魏的政治中心从塞北平城，迁到了中原腹地洛阳。这一迁，意义深远——它意味着鲜卑要从草原民族，变成中原王朝。

迁都之后，孝文帝连下几道诏令。太和十八年，禁胡服，鲜卑人不得再穿窄袖左衽的胡服，一律改穿汉人衣冠。太和十九年，禁胡语——「三十以下，见在朝廷之人，语音不听仍旧，若有故为，当加降黜。」《魏书·高祖纪》。三十岁以下在朝为官的，不准再说鲜卑话，谁敢违犯，降职罢黜。太和二十年，公元 496 年，改汉姓——皇族拓跋氏改姓元氏，孝文帝自己改名元宏。丘穆陵改穆，步六孤改陆，贺赖改贺，独孤改刘，贺楼改楼，勿忸于改于——鲜卑八大贵族，统统改汉姓。他还鼓励鲜卑贵族与汉人四大士族（崔、卢、李、郑）通婚，自己带头娶汉族大姓之女。

这一套改革，叫「汉化」。它把一个鲜卑的草原政权，改造成了一个中原的汉化王朝。可改革触动了鲜卑旧贵族的根本利益，反弹极大。带头反对的，是太子元恂。元恂是孝文帝长子，本名拓跋恂，不爱读书，体胖怕热，怀念平城的旧俗。太和二十年，公元 496 年，元恂趁孝文帝出巡，谋逃回平城，被发觉。孝文帝大怒，废元恂为庶人，拘禁河阳。太和二十一年，公元 497 年，又赐死元恂，年仅十五岁。一个皇帝，为了推行改革，亲手赐死了自己的太子——这是汉化改革付出的最痛的代价。

太和二十三年，公元 499 年三月，孝文帝南征途中，崩于谷塘原，年仅三十三岁。他在位二十九年，亲政近十年。临终遗诏，以任城王元澄、咸阳王元禧、彭城王元勰辅政。元勰是孝文帝的弟弟，贤能有德，是北魏宗室里的清流。孝文帝一死，他的改革基本被保留下来——均田、三长、汉化，都延续到了北齐北周，甚至影响隋唐。可鲜卑旧贵族的怨气没消，二十多年后六镇之乱爆发，北魏很快分裂为东魏西魏。孝文帝改革，是北魏的高光，也是北魏由盛转衰的转折——他改得太急、太深，把鲜卑这个马上民族，改得自己都不认识自己了。

## 讲人物

**元宏：把鲜卑改成汉人的皇帝**

拓跋宏，后改名元宏，魏孝文帝。他五岁即位，由冯太后抚养长大，亲政时已二十多岁。这位皇帝有几个特点：一是汉文化修养极高。他自幼读经史，通《诗》《书》，能写一手好文章，与汉人士大夫谈经论史，不相上下。一个鲜卑皇帝，比汉人还汉人，这是冯太后教育的结果，也是他自身的资质。二是雷厉风行。一旦认准了改革，就一往无前，不留退路。迁都时用「南伐」逼群臣，禁胡服胡语时不许讨价还价，连太子反对也敢赐死。这份决断，是改革者必备的胆气。

三是情深义重。他对冯太后的感情是真切的——冯太后杀了他的父亲，他却守丧哀毁，亲政后仍追念不已。他对弟弟元勰信任有加，临终托付。他对汉文化的向往也是真的——他不是装出来的「汉化」，是发自内心地认同中原文明。可正是这份「真」，让他的改革做得太绝——他爱汉化，爱到要抹去鲜卑的痕迹；他决断，决断到连太子都不留。孝文帝三十三岁就死了，他没看到改革的长远后果。他要是多活二十年，也许能补上改革的裂缝；可惜天不假年。他留下的，是一个半汉化的北魏，和新旧两派鲜卑贵族深埋的怨。

**冯太后：杀子养孙的铁腕女主**

冯太后，长乐信都人，汉人出身，文成帝拓跋濬的皇后。她是北魏实际的改革奠基人，也是孝文帝的嫡祖母兼养祖母。她临朝称制近十五年，手段强硬——公元 476 年，她毒杀献文帝拓跋弘，即孝文帝的亲生父亲，把权力牢牢握住。可她又把孝文帝抚养成人，亲自教导他汉文化，把自己改革的未竟之业交到孙子手里。一个祖母，杀了孙子的父亲，又把孙子培养成自己的接班人——这关系复杂得让人唏嘘。

冯太后的功劳，在于她打下了孝文帝改革的地基。班禄、均田、三长，都是她主政时颁行的。她用汉法改造北魏的思路，被孝文帝全盘继承并推向深入。没有冯太后，就没有孝文帝改革。可冯太后也留下了隐患——她毒杀献文帝，开了北魏宫廷政变的恶例；她临朝称制，也让外戚与宗室的矛盾埋下种子。她是一个有远见也有铁腕的女主，可她的铁腕，也埋下了北魏后来动荡的伏线。

**元恂：不愿汉化的太子**

元恂，本名拓跋恂，孝文帝长子。他被立为太子，本该是汉化改革的接班人。可元恂不爱读书，怀念平城旧俗，体胖怕热，受不了洛阳的闷热。他不爱穿汉服，不喜说汉话，与父亲孝文帝的理念完全相反。公元 496 年，他趁孝文帝出巡，谋逃回平城，被发觉。孝文帝大怒，废他为庶人，次年赐死，年仅十五岁。

元恂的悲剧，是改革中代际冲突的极端样本。他不是坏孩子，只是个怀念故土、不适应新环境的少年。可在父亲眼里，他成了改革的阻力；在鲜卑旧贵族眼里，他又成了反汉化的旗帜。一个十五岁的少年，被夹在新旧两种力量之间，最终成了改革的祭品。孝文帝赐死亲子，是改革者最痛的决断——为了改革能推行到底，连骨肉至亲都不能留。元恂一死，鲜卑旧贵族的反汉化暗流反而更深了。

**元澄：支持改革的宗室重臣**

元澄，本名拓跋澄，任城王，是孝文帝的宗室长辈。他是孝文帝改革最得力的宗室支持者。迁都时，群臣反对，元澄率先表态支持，替孝文帝分担了压力。改革推行中，他也是孝文帝的左膀右臂。一个改革要成功，光有皇帝的决断不够，还得有宗室重臣背书——元澄就是那个背书人。他懂改革的方向，也懂如何在宗室内部斡旋。孝文帝临终，把他列为辅政首臣，是托付改革成果的延续。元澄一生，是鲜卑贵族里「识大体」的代表——他明白，鲜卑要么汉化图存，要么被时代淘汰。

## 讲背景

魏孝文帝改革，要放在鲜卑汉化的长进程里看。

鲜卑拓跋氏本是草原游牧民族，从大兴安岭嘎仙洞一路南迁，到建立北魏统一北方，用了百余年。可游牧民族打天下行，治天下难。北魏初年，鲜卑贵族还过着部落式的生活，官吏没俸禄，靠战利品和搜刮度日；宗主督护制下，汉族豪强隐匿户口，朝廷收不到税也征不到兵；鲜卑与汉族之间，语言、服饰、风俗、婚姻都隔着一道墙。这样的政权，靠武力维持一时可以，长久下去非乱不可。北魏前期已多次爆发起义——盖吴起义、各族反抗——都是这种「马上治天下」撑不住的信号。改革，是北魏求存的必经之路。

鲜卑汉化，其实从道武帝、太武帝时就开始了。道武帝拓跋珪重用汉人士族，太武帝拓跋焘用崔浩改革，可崔浩最终因「国史之狱」被杀——鲜卑旧贵族反扑，汉化受挫。冯太后主政后，重启汉化，用李冲、李安世这些汉臣，颁行班禄、均田、三长。孝文帝亲政后，把汉化推向极致——不止改制度，还要改语言、改服饰、改姓氏、改婚俗，要把鲜卑整个变成汉人。这是汉化进程的第三波，也是最深的一波。

平城与洛阳的地理之争，是迁都的根。平城在今山西大同，地处塞北，是鲜卑的龙兴之地。可平城偏北，控制中原不便，且冬季苦寒，不利于农业。更关键的是，平城是鲜卑旧贵族的老巢，他们在此根深蒂固，反对改革的力量集中在这里。洛阳在中原腹地，是东汉、曹魏、西晋的故都，是汉文化的中心。迁都洛阳，意味着北魏的政治重心从草原移向中原，从鲜卑移向汉文化。这一迁，地理上是几百里，文化上是几百年——鲜卑要从马上民族，变成衣冠王朝。可这一迁，也丢掉了平城六镇的根基，二十多年后六镇之乱，根子就在这里。

均田制与三长制，是改革的经济与户口基础。均田制把荒地分给农民，限制豪强兼并，让流民重新附籍；三长制重建基层户籍，邻长、里长、党长逐级管理，把隐匿在宗主督护下的户口挖出来。这两条制度，让北魏的税收和兵源有了保障，也让北方残破的经济得以恢复。均田制延续到隋唐，是中国古代土地制度的一大支柱。孝文帝改革能在经济上立得住，靠的就是均田与三长这两条。这是改革的硬基础——没有它，汉化只是改改衣冠姓氏，撑不住一个王朝。

再看北方门阀的背景。北方的汉族士族——清河崔氏、范阳卢氏、赵郡李氏、荥阳郑氏——是中原的文化传承者。北魏要汉化，必须借助这些士族。孝文帝定姓族，把鲜卑八姓与汉人四姓并列，鼓励通婚，就是要把鲜卑贵族与汉人士族捆在一起，共享权力。这既是对汉文化的尊重，也是政治上的联盟——用通婚换支持，用姓族换认同。可这也埋下了新问题：鲜卑旧贵族被冷落，新旧贵族的矛盾激化，这是六镇之乱的远因。

## 讲道理

王夫之在《读通鉴论》里论魏孝文帝改革，话分两层。大意据王夫之说，孝文帝改革的功劳，在于把一个草原政权改造成了中原王朝，让北方免于彻底的胡化，保住了汉文化的根脉——这是大功。可王夫之也直指其弊：孝文帝改得太急、太深，把鲜卑旧俗一笔抹杀，激起新旧两派的深仇。赐死太子元恂，是改革最痛的代价；可这一赐死，没消弭反对，反而把怨气压得更深。王夫之的判断是：改革不在快慢，在「顺」与「逆」。顺乎大势的改革，慢些也成；逆乎人心的改革，再快也是埋祸。孝文帝顺了汉化的大势，却逆了鲜卑旧俗的人心，所以改革留下成果的同时，也留下了北魏分裂的种子。大意据《读通鉴论》卷十六。

胡三省在《资治通鉴音注》里，对孝文帝迁都的批注，眼光独到。大意据胡三省说，孝文帝迁都洛阳，是高明的一手——它把北魏的政治中心移到中原，让鲜卑从草原民族变成中原王朝。可这一迁，也丢掉了平城六镇的屏藩。六镇是北魏北边的军事重镇，驻着鲜卑精锐，本是保卫平城的。迁都后，六镇地位一落千丈，镇户从国之干城变成被遗忘的边兵。胡三省预见到：六镇之怨，必起于此。果然，孝文帝死后二十多年，六镇起义爆发，北魏很快分裂。胡三省的潜台词是：迁都移的是政治中心，可不能把军事根基一起移走。丢了根基的迁都，是把大树连根拔起再栽，能不能活，看天意。大意据《资治通鉴音注》齐纪魏事部分。

司马光在《资治通鉴》里记魏孝文帝改革，对孝文帝本人是高度肯定的。他记孝文帝勤政好学、崇尚文治、留心刑律，是把他写成一位贤君。可司马光也如实记下了赐死太子元恂、激化鲜卑旧怨的事，不加粉饰。司马光的史观，重「用人」与「教化」——他认为孝文帝能成事，靠的是识人用人、力行教化；可他未能化解新旧矛盾，是改革未竟之处。司马光写这一段，大概是想提醒宋朝皇帝：改革要敢为，也要会化人；敢为而不化人，改革留一半。大意据《资治通鉴》齐纪魏事部分。

## 问道悟道

魏孝文帝改革留给后人的，是两条关于「改」的规律。

**第一条：汉化（胡化）的强制与代价。**

孝文帝的汉化，是「胡人学汉人」的强制改革。他禁胡服、禁胡语、改汉姓、通汉婚，用行政命令把鲜卑整个改造成汉人。这改革方向对不对？对——它让北魏融入中原，保住了汉文化的北传，也让北方免于彻底胡化。可改革的方式，是「强制」。强制，就必然有代价。代价之一，是鲜卑旧贵族的反扑——元恂谋逃、六镇之乱，都是被强压下去又爆发的怨气。代价之二，是文化身份的撕裂——鲜卑人改了姓、换了服、说了汉话，可心里未必真成了汉人。这种身份的撕裂，是强制改革最深层的代价。

这事历史上有几处对照，最有名的是反向参照——赵武灵王的「胡服骑射」。赵武灵王是战国赵国君主，他做的是「汉人学胡人」的改革：让汉人脱掉宽袍大袖，改穿胡人短装，学骑射。方向与孝文帝相反，方式一样——强制。胡服骑射让赵国军力大增，可赵武灵王晚年因继承人问题被困沙丘，活活饿死。《史记·赵世家》。胡服骑射是「汉学胡」，孝文帝汉化是「胡学汉」，一正一反，都是强制改风俗。强制改革能见效，可改革者的命运都不太好——赵武灵王饿死沙丘，孝文帝三十三岁早逝，改革者常以身殉改革。

再看一处反例——清初的「剃发易服」。清军入关后，强制汉人剃发留辫、改穿满服，是「汉人学满人」的强制改革。这一强制，激起江南激烈抵抗，扬州十日、嘉定三屠，血流成河。剃发易服与孝文帝汉化，方向相反，逻辑一样——用强制改风俗。区别在于：孝文帝汉化是大势所趋（汉文化先进），所以虽有代价终能消化；剃发易服是逆势而行（满文化相对落后），所以激起更惨烈的反弹。这说明：强制改革的成败，不全在「强不强」，在「顺不顺势」。顺势的强制，痛一阵能消化；逆势的强制，痛几代都消不掉。

**第二条：改革者与既得利益的代际冲突。**

孝文帝改革最痛的一幕，是赐死太子元恂。元恂不愿汉化，谋逃平城，被父亲赐死。这一幕的实质，是改革者与既得利益（此处是鲜卑旧俗）的代际冲突——儿子守旧，父亲求新，父子反目，骨肉相残。这种代际冲突，是改革中最难化解的矛盾。

这事历史上有几处深刻的例子。商鞅变法，触动秦国旧贵族利益，秦孝公一死，商鞅被车裂，全家被杀。可商鞅之法存下来了，秦国由此强盛，终灭六国。《史记·商君列传》。商鞅是「身死而法存」的典型——改革者死了，改革活了。孝文帝赐死元恂，是另一面——改革者活，反对者死。可元恂的死，没消弭鲜卑旧贵族的怨，反而让六镇之乱埋得更深。王安石变法，触动既得利益，新旧党争几十年，变法反复，北宋元气大伤。《宋史·王安石传》。王安石是「法与身俱困」的典型——改革者和改革一起被困在党争里。

这三处——商鞅、孝文帝、王安石——说明改革的一条铁律：触动既得利益的改革，必有反弹。反弹的方式不同：商鞅遇反扑身死，孝文帝杀子压反弹，王安石陷入党争泥潭。可反弹的必然是一样的。改革的智慧，不在「敢不敢触动」，在「触动之后怎么消化反弹」。孝文帝敢动，可没消化好反弹——他改得太急太深，把怨气压到炸。一个高明的改革者，不仅要敢改，还要会化——化解反弹，消化怨气，让新旧两派最终融合。孝文帝会改不会化，所以改革留下一半成果，也留下一半祸根。

## 结语

一个鲜卑皇帝，把族人的姓氏、衣冠、语言全改了，连太子反对都赐死——魏孝文帝改革，是「顺势而强改、改得深却化不尽」的典型，它让北方汉化图存，也让北魏埋下分裂的种子。
