# 资治通鉴·汉纪一·韩信拜将

## 讲事情

汉元年，公元前206年，韩信拜将，是楚汉相争里最戏剧性的一幕。一个钻过别人裤裆的小官，一夜之间成了汉军大将，一番话把项羽的底牌翻了个底朝天。

韩信是淮阴人，年轻时家里穷，没正经行当，常跟人混饭吃。韩信在南昌亭长家吃了几个月闲饭，亭长老婆烦了，故意提前吃饭，韩信来了没饭，明白意思，走了。

韩信在城下钓鱼，一个漂母看韩信饿，给他饭吃，一连几十天。韩信说：我将来一定重报。漂母生气说：大丈夫不能自食，我是可怜你，谁图你报答？

淮阴屠中少年看韩信不顺眼，拦住他说：你长得高大，带剑，其实是胆小鬼。你要不怕死，刺我；怕死，从我胯下爬过去。韩信看了看他，低头从他胯下爬过去。满街人都笑韩信胆小。

项梁渡淮，韩信仗剑从军，跟着项梁。项梁败死，韩信又跟项羽，做郎中。韩信几次献策，项羽不用。韩信跑了，投了刘邦，做连敖，管粮草的小官。

韩信犯了法当斩，同伙十三人都斩了。轮到韩信，韩信仰头看，看见滕公夏侯婴，说：上不欲就天下乎？何为斩壮士？——汉王不想得天下吗？为什么斩壮士？——《史记·淮阴侯列传》。夏侯婴奇其言，又看韩信相貌不凡，放了他，跟刘邦说。刘邦拜韩信为治粟都尉，可还是没重用。

韩信跟萧何谈了几次，萧何奇之。可刘邦不重用，韩信觉得没指望，跑了。

有人报告萧何：丞相也跑了。刘邦大怒，如失左右手。过了一两天，萧何回来见刘邦。刘邦又怒又喜，骂萧何：你为什么跑？萧何说：臣不敢跑，臣去追跑的人。刘邦问：追谁？萧何说：韩信。刘邦骂：诸将跑了几十个你不追，追韩信？你骗我。萧何说：诸将易得，至如韩信，国士无双——诸将容易得，像韩信这样的，国士无双。——《史记·淮阴侯列传》。汉王要长王汉中，韩信没什么用；要争天下，非韩信不可。刘邦说：那我就用他。萧何说：你向来傲慢，拜大将像呼小儿，这是韩信走的原因。你要拜他，选良日，斋戒，设坛，具礼。刘邦说：好。

刘邦要拜大将，诸将都高兴，人人以为是自己。到了拜将那天，竟是韩信。一军皆惊。

拜完将，刘邦问韩信：丞相数言将军，将军有什么计策教寡人？韩信辞谢，问刘邦：现在东向争权天下的，是不是项王？刘邦说：是。韩信问：大王自己估量，勇悍仁强，比项王怎么样？刘邦沉默，说：不如。韩信说：臣也以为大王不如。可臣曾经事奉项王，请允许我说说项王的为人。

韩信说：项王喑恶叱咤，千人皆废——项王一怒吼，千人都吓废。——《史记·淮阴侯列传》。可项王不能任属贤将，这是匹夫之勇。项王待人恭敬慈爱，说话和气，人有病，涕泣分食饮。可人有大功该封爵时，印都磨好了，舍不得给，这是妇人之仁。

韩信又说：项王虽霸天下，不居关中，都彭城，背义帝之约，凭亲爱封王，诸侯不平。项王所过，无不残灭，天下不亲。项王名叫霸王，其实失天下之心。所以项王强，其实容易弱。

韩信献策：大王入武关，秋毫无犯，除秦苛法，秦民无不欲大王王秦者。现在大王举兵东向，三秦可传檄而定。刘邦大喜，自以为得韩信晚。于是部署诸将，准备东进。

## 讲人物

**韩信：忍辱负重的奇才**

韩信这个人，是淮阴平民，家里穷，没正经行当。可韩信好带剑，少年时忍了胯下之辱。这份忍，不是胆小，是有大志向的忍。

韩信的贫寒，是韩信的底色。韩信母亲死，韩信穷得办不起丧事，可韩信找了一块高地葬母，墓地旁边能置万户。韩信的志向，从葬母就看得出——他要的是万户侯的排场。韩信在南昌亭长家吃闲饭，亭长老婆烦了，提前吃饭，韩信来了没饭。韩信看出意思，走了，再没回来。韩信在城下钓鱼，饿得不行，一个漂母给他饭吃，一连几十天。韩信说将来重报，漂母生气说：大丈夫不能自食，我是可怜你，谁图你报答？漂母这话，是激韩信的话。韩信后来封楚王，找到漂母，赐千金。韩信的报恩，是真报恩。韩信找到南昌亭长，赐百钱，说：你是个小人，做好事不能坚持。韩信的报怨，也是真报怨。

韩信的胯下之辱，是韩信最出名的故事。淮阴屠中少年拦韩信：你长得高大，带剑，其实是胆小鬼。你要不怕死，刺我；怕死，从我胯下爬过去。韩信看了看他，低头从他胯下爬过去。满街人都笑韩信胆小。韩信封楚王后，找到那个少年，没杀他，封他做中尉，跟诸将说：这是个壮士。他辱我时，我不能杀他吗？杀他没名目，所以忍了，才有今天。韩信的忍，是有大志向的忍——忍一时之辱，成一世之功。

韩信先跟项梁，项梁败死；又跟项羽，做郎中，几次献策不用；再投刘邦，做连敖，犯法当斩。韩信这一路，是怀才不遇的一路。韩信跟项羽时，数次献策，项羽不用。韩信看项羽不能用人，跑了。韩信投刘邦，做连敖，又犯法当斩。韩信这一路，是怀才不遇的一路。韩信的志向，在跟萧何谈兵时露出来。萧何一听，就知道韩信不是寻常人。

韩信拜将后，一番话把项羽的"匹夫之勇，妇人之仁"说透，又献策东向争天下。刘邦听了大喜，自以为得韩信晚。后来韩信明修栈道、暗渡陈仓，还定三秦；背水一战，破赵；潍水壅水，破齐；垓下十面埋伏，灭项羽。韩信一个人，打下了汉朝一半的天下。可韩信的下场不好。项羽派人劝韩信三分天下，韩信不肯，说刘邦解衣推食，自己不能背义。蒯通也劝韩信反，三分天下，韩信不听。后来有人告韩信反，刘邦用计擒韩信，降为淮阴侯。最后吕后和萧何骗韩信入宫，斩于长乐钟室。韩信临死叹："吾悔不用蒯通之计"——我后悔没用蒯通的计。——《史记·淮阴侯列传》。

韩信论项羽，是韩信拜将后的重头戏。韩信说项羽"匹夫之勇"——项王喑恶叱咤，千人皆废，可不能任属贤将。又说项羽"妇人之仁"——待人恭敬慈爱，人有病涕泣分食饮，可人有功该封爵，印磨好了舍不得给。韩信这两句话，把项羽的病根说透了。后来楚汉相争，项羽的败，果然败在不能任将、舍不得封赏上。韩信跟过项羽，对项羽看得清楚。韩信的清楚，是从切身经历来的——韩信在项羽手下不被用，所以知道项羽不能用人。

**萧何：识人的丞相**

萧何是沛人，跟刘邦同乡，做沛主吏掾。刘邦还是亭长时，萧何就护着他。刘邦起兵，萧何跟着，做丞督，管后勤。

萧何的本事，是识人。萧何在沛县时，就识刘邦。刘邦还是亭长，常被差役押送徒众去咸阳，别人送三百钱，萧何送五百。萧何对刘邦，是早有看好。萧何的识人，从刘邦就开始。

萧何识韩信，是萧何识人的高峰。韩信做治粟都尉，萧何跟他谈了几次，就知道韩信是国士无双。韩信跑了，萧何来不及报告，连夜去追。这就是"萧何月下追韩信"。诸将跑了几十个，萧何不追；韩信一跑，萧何连夜追。萧何知道，诸将易得，韩信难求。萧何追韩信时，有人报告刘邦"丞相也跑了"，刘邦大怒，如失左右手。萧何回来，刘邦又怒又喜。萧何说：诸将易得，至如韩信，国士无双。萧何这话，是把韩信的分量说清楚——诸将容易得，韩信这样的，国士无双。

萧何追回韩信，跟刘邦说：你要长王汉中，韩信没什么用；要争天下，非韩信不可。萧何这话，是把刘邦的志向问出来。刘邦要争天下，所以拜韩信为大将。萧何还劝刘邦：你向来傲慢，拜大将像呼小儿，这是留不住人的原因。要拜韩信，得选良日，斋戒，设坛，具礼。刘邦听了，照办。刘邦能听萧何的话，是刘邦的本事。

萧何镇守关中，是萧何对汉朝的大功。刘邦在前线打项羽，萧何在关中守。萧何"镇国家，抚百姓，给馈饷，不绝粮道"——镇国家，抚百姓，供粮饷，粮道不绝。——《史记·高祖本纪》。刘邦几次败给项羽，萧何从关中送兵送粮，刘邦输了再补。没有萧何，刘邦撑不到垓下。后来刘邦论功行赏，萧何第一，"功臣皆莫敢言"——功臣都没话说。——《史记·萧相国世家》。萧何的功，是后勤的功，是镇守的功，不是冲锋陷阵的功，可萧何的功，比冲锋陷阵的功还大。

**夏侯婴：慧眼的滕公**

夏侯婴是沛人，跟刘邦同乡，做滕令，所以叫滕公。夏侯婴跟刘邦交好，刘邦起兵，夏侯婴一直跟着。

夏侯婴跟刘邦的交情，从沛县就开始。夏侯婴做沛县厩司御，就是管马车的。刘邦那时做亭长，跟夏侯婴开玩笑，误伤夏侯婴。有人告刘邦伤人，刘邦否认。夏侯婴为刘邦作证，说自己没被伤，被关了一年多，打了几百杖。夏侯婴为刘邦受这罪，是夏侯婴对刘邦的义。后来夏侯婴一直跟着刘邦，做太仆，管车马。夏侯婴是刘邦的"车神"——一辈子给刘邦驾车，从沛县到关中到荥阳到垓下，夏侯婴的车，载着刘邦的命。

韩信犯法当斩，夏侯婴路过，听见韩信喊"上不欲就天下乎"，奇其言，又看韩信相貌不凡，放了他。夏侯婴这一放，放出了一个大将。夏侯婴还跟刘邦说韩信，刘邦拜韩信为治粟都尉。虽然没重用，可夏侯婴是第一个识韩信的人。夏侯婴识韩信，跟萧何识韩信不一样——夏侯婴是路过听见，萧何是谈兵识出。可夏侯婴是第一个，没有夏侯婴，韩信早斩了，萧何也没机会识韩信。

夏侯婴后来一直跟着刘邦，做太仆，管车马。彭城之战，刘邦大败，夏侯婴驾车带刘邦逃命。路上遇见刘邦的一儿一女（后来的鲁元公主和汉惠帝），夏侯婴把他们抱上车。刘邦嫌车慢，几次把儿女踹下车，夏侯婴几次停车抱回。夏侯婴对刘邦说：虽然急，可以慢慢走，怎么能扔孩子？夏侯婴这一抱，抱出了汉朝的下一代——汉惠帝刘盈。刘邦踹儿女，是刘邦的狠；夏侯婴抱儿女，是夏侯婴的仁。夏侯婴的仁，救了汉朝的太子。

## 讲背景

韩信拜将这事，得放在刘邦入汉中的局势和楚汉的力量对比里看。

公元前206年，项羽分封诸侯，自立为西楚霸王，都彭城。刘邦被立为汉王，王巴、蜀、汉中，都南郑。项羽把关中分给三个秦降将——章邯为雍王、司马欣为塞王、董翳为翟王，挡着刘邦东进的路。这就是"三秦"。项羽这一手，是把刘邦关在巴蜀汉中，让三秦挡着。项羽觉得巴蜀是穷山恶水，刘邦翻不了天。可项羽错了——巴蜀虽偏，可也安全；刘邦在巴蜀休养，没人打他。后来刘邦还定三秦，出关跟项羽争天下，巴蜀汉中成了刘邦的根据地。

刘邦去汉中，一路上将士逃亡。刘邦自己是被逼去的，手下将士多是关东人，想回家。韩信也是这时候跑的——韩信觉得刘邦不重用他，待在南郑没指望。刘邦入汉中，走的栈道。栈道是在悬崖上搭的木道，险峻难行。刘邦听张良的计，烧栈道，示无东意——让项羽以为刘邦不回来了。这一烧，项羽放心了，可刘邦也把自己关在了汉中。后来韩信拜将，明修栈道、暗渡陈仓，就是利用栈道做文章——明面上修栈道，吸引章邯注意；暗地里从陈仓故道出兵，打章邯一个措手不及。

再说当时的拜将规矩。拜大将，要设坛，斋戒，具礼，不是随便叫来就拜。刘邦向来傲慢，对士人没好脸色，洗脚时见儒生是常事。郦食其来见刘邦，刘邦正让两个女人洗脚。郦食其说：你要是想灭秦，不该这样见长者。刘邦赶紧停止洗脚，穿好衣服，请郦食其上坐。刘邦的傲慢，是刘邦的毛病；可刘邦能改，是刘邦的长处。萧何劝刘邦：你拜大将像呼小儿，这是留不住人的原因。要拜韩信，得选良日，斋戒，设坛，具礼。刘邦听了，照办。拜将坛制度，是当时的规矩——筑坛场，斋戒，设礼，然后拜将。刘邦能听萧何的话，按规矩拜将，是刘邦对韩信的尊重，也是刘邦对人才的尊重。

再说刘邦的用人格局。刘邦用人，不看出身，只看本事。刘邦手下，张良是韩国贵族，萧何是沛县小吏，韩信是淮阴平民，陈平是盗嫂受金之人，樊哙是屠狗的，灌婴是布贩的，周勃是吹箫的，娄敬是戍卒。这些人，出身五花八门，可刘邦都能用。刘邦的用人，是"因能授职"——按本事给官。项羽用人，只看亲贵，不看本事。韩信、陈平都跟过项羽，都不被用，都跑了。刘邦的用人格局，是刘邦得天下的根。

再说楚汉的人才争夺。楚汉相争，不只是兵争，也是人才争。项羽那边，范增是谋臣，龙且、钟离昧、季布、英布是武将。可项羽不能用——范增被气走，英布被逼反，韩信、陈平跑了。刘邦那边，张良、萧何、陈平、韩信、郦食其、随何，都是从各处来的。陈平从项羽那边跑来，刘邦用；韩信从项羽那边跑来，刘邦用。刘邦的人才，一半是从项羽那边挖来的。项羽的人才，越打越少；刘邦的人才，越打越多。人才的得失，是楚汉相争的关键。

再说韩信的军事思想。韩信打仗，有几个特点：一是用间，二是奇正相生，三是因敌制胜。韩信明修栈道暗渡陈仓，是奇正相生——明修是正，暗渡是奇。韩信背水一战，是置之死地而后生——把自己人逼到绝地，逼出士气。韩信潍水壅水破齐，是用水——壅水上流，等楚军半渡，放水淹。韩信垓下十面埋伏，是用围——把项羽围在垓下，四面楚歌，攻心为上。韩信的军事思想，是从实战来的，不是从书本来的。韩信学兵法，没学完；可韩信打仗，打的是活兵法。

再说韩信为什么先跟项羽再跟刘邦。韩信是淮阴人，楚地人。项梁起兵会稽，渡河北上，韩信仗剑从军。韩信跟项羽，做郎中，几次献策，项羽不用。韩信看项羽不能用人，跑了。韩信投刘邦，做连敖，又犯法当斩。韩信这一路，是怀才不遇的一路。韩信的才，要遇到识才的人才能用。萧何是识才的人，所以韩信拜将。

再说韩信论项羽。韩信跟过项羽，对项羽的为人看得清楚。韩信说项羽"匹夫之勇"——项王喑恶叱咤，千人皆废，可不能任属贤将。又说项羽"妇人之仁"——待人恭敬慈爱，人有病涕泣分食饮，可人有功该封爵，印磨好了舍不得给。韩信这两句话，把项羽的病根说透了。后来楚汉相争，项羽的败，果然败在不能任将、舍不得封赏上。

## 讲道理

司马迁在《史记·淮阴侯列传》末尾有一段"太史公曰"。司马迁说韩信"天下已集，乃谋畔逆"——天下已定，却谋反叛。——《史记·淮阴侯列传》。司马迁又叹：假令学道谦让，不伐己功——假使学道谦让，不夸功。——《史记·淮阴侯列传》。司马迁看韩信，看的是"才"与"德"的失衡——韩信才高，德不够，所以下场不好。

蒯通是韩信的谋士。韩信破齐后，蒯通劝韩信三分天下。大意据蒯通说，韩信功高震主，威震天下，归楚楚人不信，归汉汉人害怕，不如自立。韩信不听，说刘邦解衣推食，自己不能背义。后来韩信被杀，临死叹"吾悔不用蒯通之计"。蒯通看韩信，看的是"功高震主"的危险——韩信功太高，无论归楚归汉，都让人怕。蒯通的话，韩信当时不听，后来应验了。

王夫之在《读通鉴论》里对韩信有议论。大意据王夫之说，韩信之败，败在不知止。韩信破齐后，请刘邦封自己为假齐王。刘邦大怒，张良、陈平踩刘邦脚，刘邦改口封韩信为齐王。韩信这一请，是韩信自取祸端的开始。王夫之看韩信，看的是"知进不知退"——韩信只知进，不知退，所以招祸。


司马光在《资治通鉴》臣光曰中评韩信：「汉之所以得天下者，大抵皆信之功也」——汉朝得天下，大半是韩信的功劳。但韩信灭齐自王、固陵不至，已埋下祸根。

扬雄《法言·重黎》将韩信与黥布并列：「忠不终而躬逆，焉攸令」——忠心不终、亲身造反，哪有令名可言。
## 问道悟道

韩信拜将这事，藏着两条让人开眼的规律。

**人才要遇到识才的人，才能用。**

韩信先跟项梁，项梁败死；再跟项羽，献策不用；再投刘邦，做连敖，犯法当斩。韩信这一路，是怀才不遇。直到夏侯婴听见韩信喊"上不欲就天下乎"，放了他；萧何跟韩信谈兵，识他是国士无双。夏侯婴是第一个识韩信的人，萧何是真正用韩信的人。没有夏侯婴，韩信早斩了；没有萧何，韩信跑了没人追。

人才要遇到识才的人才能用这事，历史上有几处例子。战国时商鞅在魏国，魏相公叔痤临死推荐商鞅，魏王不用。商鞅入秦，秦孝公重用，商鞅变法，秦国强盛。——《史记·商君列传》。三国时诸葛亮在隆中，没人识。刘备三顾茅庐，诸葛亮出山，辅刘备成三分之业。——《三国志·诸葛亮传》。韩信、商鞅、诸葛亮，都是人才，都要遇到识才的人才能用。人才常有，识才的人不常有。

**功高震主，是功臣的大险。**

韩信破魏、破赵、破齐，连百万之军，名震天下。韩信请刘邦封自己为假齐王，刘邦大怒。后来有人告韩信反，刘邦用计擒韩信，降为淮阴侯。最后吕后斩韩信于长乐钟室。韩信的功太高，高到让刘邦不安。

功高震主是功臣的大险这事，历史上有例子。战国时白起为秦将，破赵于长平，坑赵卒四十万。白起功高，秦昭王忌，赐剑令白起自尽。——《史记·白起王翦列传》。春秋时文种帮越王勾践灭吴，功高，勾践赐剑令文种自尽。——《史记·越王勾践世家》。韩信、白起、文种，都是功高震主。功太高，主不安；主不安，功臣就险。功高的人，要么知止，要么有祸。

## 结语

萧何追回一个人，追出了汉朝四百年。
