# 资治通鉴·后梁纪一·朱温代唐

## 讲事情

唐朝灭亡的丧钟，是在洛阳椒殿敲响的；可后梁建国的戏，要从汴州的白马驿说起。朱温篡唐建梁，建的不是个安生朝廷，而是个屠戮朝臣、父子相杀的短命王朝。十六年后梁，三世相传，朱温被亲子所弑，弑父者又被亲弟所诛——这出戏的开场，正是本篇要讲的。

天祐元年，公元904年，朱温弑昭宗于洛阳椒殿，立昭宗第九子李柷为帝，是为哀帝，年十三（详见《资治通鉴·唐纪十二·黄巢起义与朱温篡唐》论朱温弑昭宗为唐亡丧钟）。朱温弑君之后，朝臣犹有不服者。朱温欲禅代，可唐朝的衣冠世族、清流朝士还在，这些人是朱温称帝的障碍。天祐二年，公元905年六月，朱温在心腹李振、柳璨的怂恿下，于白马驿（在滑州，今河南滑县）杀裴枢、独孤损、崔远等朝臣三十余人。大意据《资治通鉴·后梁纪一》《旧五代史·梁书·太祖纪》。这一案，史称"白马驿之祸"。李振其人，是朱温的谋主之一，屡试不第，素恨朝士。白马驿之祸时，李振对朱温说："此辈自谓清流，宜投于浊流。"——这些人自命清流，该投到浊流里去。朱温听之，命将死者尸体投于黄河。大意据《新五代史·李振传》。清流投浊流，是唐末衣冠世族被武夫集体屠戮的标志——百年来依靠科举门阀撑起的中朝文官体系，至此一朝断绝。白马驿之祸杀的不止三十余人，杀的是唐朝的士族根基。

白马驿之后，朝臣噤声。天祐四年，公元907年四月，哀帝禅让，唐亡。朱温即皇帝位，国号梁（后梁），建元开平，建都汴州（今开封），升汴州为开封府。大意据《资治通鉴·后梁纪一》《旧五代史·梁书·太祖纪》（禅让本事详见《资治通鉴·唐纪十二·黄巢起义与朱温篡唐》主场）。朱温以宣武节度使起家，汴州是其根本，故建都于此，不都洛阳长安。这一建都选择，开启了五代以汴梁为中原中心的政治格局——开封地处平原，四战之地，无险可守，可漕运便利；五代梁晋汉周皆都汴，正因漕运之便。朱温建梁后，封赏功臣，以张全义为河南尹，镇洛阳；以朱友裕（早卒）、朱友珪、朱友贞、朱友文（养子）诸子分镇要地。后梁的版图，远不及唐，亦不及晋——北有沙陀李克用父子虎踞河东，南有杨行密据淮南（后之杨吴），西有李茂贞据凤翔，蜀有王建称帝（前蜀），岭南有刘隐（后之南汉）。五代十国的"十国"格局，自此成形。

朱温建梁后，年事已高（称帝时已五十五岁），渐荒于政事。朱温好色，且乱伦——其诸子多在外镇守，朱温常召诸子之妻入宫侍寝。养子朱友文之妻王氏美，朱温尤宠之。大意据《资治通鉴·后梁纪一》《新五代史·梁本纪》。朱温晚年，欲传位于养子朱友文，以其妻王氏得宠故。可朱温亲子朱友珪、朱友贞皆在侧，这份"舍亲子立养子"的念头，埋下了弑父之祸。朱友珪时任控鹤指挥使，掌禁军，是朱温身边的实权之人；朱友贞时为东京留守，在大梁（开封）。养子得宠、亲子失位，是后梁内乱的引子。

乾化二年，公元912年六月，朱温病重，命内侍召朱友文来，欲付以大位。朱友珪之妻张氏亦在侧，密告朱友珪。朱友珪大惧——若朱友文继位，自己不仅无位，恐怕性命难保。大意据《资治通鉴·后梁纪一》《旧五代史·梁书·郢王友珪传》。六月二十二日，朱友珪率亲信左仆射冯廷谔等，半夜闯入万春门。朱温闻变，惊起，问："反者谁？"朱友珪答："非他人也。"朱温怒曰："我固疑此贼，恨不早杀之。"朱友珪命冯廷谔刺朱温，刃出于背。大意据《旧五代史·梁书·郢王友珪传》。朱温年五十六岁，死于亲生子之手。朱温一生弑唐昭宗、屠前朝宗室、杀白马驿朝臣，最后死于亲子朱友珪的刀下——弑君者人亦弑之，屠人者人亦屠之，因果回旋，丝毫不爽。

朱友珪弑父后，秘不发丧，矫诏立自己为帝。可弑父之事，纸包不住火。朱友珪遣使往汴州，召朱友贞。朱友贞知其事，与姐夫赵岩、表兄张汉鼎、旧将袁象先等密谋诛友珪。大意据《资治通鉴·后梁纪一》《旧五代史·梁书·末帝纪》。乾化二年（朱友贞诛友珪在六月后不久，旋即即位），朱友贞联合袁象先率兵入宫，朱友珪走投无路，命冯廷谔刺死自己及妻。朱友贞即位，是为后梁末帝（也称均王）。后梁三世，朱温→朱友珪→朱友贞，不过二世相传而内乱两次。弑父者被弑，屠兄者继立——后梁的开国，就是一场父子兄弟相杀的血腥戏。这出戏，开了五代"以兵变得位、以兵变失位"的头（详见《资治通鉴·后梁纪五·梁晋争霸》论后梁之亡）。

## 讲人物

**朱温：弑君篡唐又被亲子所弑的开国之君**

朱温是宋州砀山人，本黄巢部将，中和二年（882）降唐赐名全忠，借平黄巢之功坐大，借诛宦官之名挟天子，最后弑昭宗、篡唐建梁（详见《资治通鉴·唐纪十二·黄巢起义与朱温篡唐》论朱温为降唐反噬的掘墓人）。朱温以武夫起家，靠的是兵权——宣武节度使镇汴州，是朱温的根基。可朱温建梁后，并未改变武夫治国的本色。他屠戮朝臣（白马驿之祸），乱伦诸媳，猜忌诸子，把后梁的朝堂搞成了武夫的私堂。

朱温晚年的荒淫，是他败亡的引子。诸子多在外镇守，朱温召诸媳入宫侍寝，养子朱友文之妻王氏尤得宠。朱温欲传位于友文，这份"舍亲子立养子"的念头，直接激发了朱友珪的弑父之心。朱温一生算计，最后算计到自己头上——他弑唐昭宗于椒殿，其子弑他于万春门；他屠前朝宗室于九曲池，其子屠他于寝宫。可见弑君篡位者，往往也死于自己人之手——因果循环，报应不爽。朱温是后梁的开国之君，也是后梁三世相杀的祸根。他建梁时已五十五岁，在位不过五年，便死于亲子之手——以暴力得天下者，必以暴力失天下。

**朱友珪：弑父篡位的二世短命之君**

朱友珪是朱温第二子（长子朱友裕早卒），母不详。朱友珪时任控鹤指挥使，掌禁军——这份禁军兵权，是朱友珪能弑父的本钱。可朱友珪弑父后，并未能坐稳皇位。朱友珪弑父之事，秘而不发，矫诏自立，可纸包不住火。朱友珪在位不过数月，便被其弟朱友贞联合袁象先诛杀。

朱友珪的悲剧，是"弑父者不得善终"的典型。弑父篡位，得位不正，朝野不服；秘不发丧，矫诏自立，更显心虚。朱友珪靠禁军弑父，朱友贞靠禁军诛兄——同一支禁军，今日助你弑父，明日助人诛你。可见靠兵变得位的，必然也败于兵变；禁军是双刃剑，今日弑君的刀，明日就反噬到自己头上。朱友珪是后梁的二世，也是后梁第一个被弑的皇帝——弑父者被弑，循环不爽。

**朱友贞：诛兄即位却终亡国的末代之君**

朱友贞是朱温第三子，母曰元贞皇后张氏（朱温正妻）。朱友贞时为东京留守，在大梁（开封）。朱友珪弑父后，朱友贞联合姐夫赵岩、表兄张汉鼎、旧将袁象先等诛友珪，即位为后梁末帝（也称均王）。朱友贞在位十一年（912-923），是后梁三帝中在位最久的，可也是末代之君。

朱友贞之败，败在"任人不明、亲小远贤"。朱友贞宠信赵岩、张汉鼎等外戚佞幸，疏远老将。后梁与晋争霸，名将王彦章（王铁枪）忠勇善战，可朱友贞听信谗言，罢王彦章，用段凝（朱友贞姻亲）为帅，致后梁军屡败（详见《资治通鉴·后梁纪五·梁晋争霸》论朱友贞任人不明致梁亡）。朱友贞是诛兄得位的，得位亦不正——弑兄者与弑父者，并无本质不同。朱友贞最后在923年李存勖入汴时自尽，后梁亡。可见诛兄篡位者，亦不得善终——后梁三世，朱温弑唐昭宗，友珪弑朱温，友贞诛友珪，三世相杀，无一善终。

**李振：怂恿屠戮朝臣的落第谋主**

李振是后梁谋主之一，原唐朝官员，屡试不第，素恨朝士。白马驿之祸，李振是主要的怂恿者——他对朱温说"此辈自谓清流，宜投于浊流"，将裴枢等三十余朝臣投尸黄河。大意据《新五代史·李振传》。李振的恨，是落第文人对科举正途的恨——他自己考不中，便恨考中的人。这份恨，借朱温的刀，杀了唐朝的清流。李振后来在后梁官至崇政院使，可后梁亡后，李振被后唐庄宗李存勖诛杀。大意据《旧五代史·李振传》。李振一生怂恿武夫屠戮文官，最后也死于武夫之手——为虎作伥者，终为虎所噬。李振的悲剧，是落第文人投靠武夫的反噬——以为借武夫之刀雪自己之恨，可武夫之刀不分敌我，最后也砍到自己头上。

## 讲背景

朱温代唐建梁这事，得放在五代更替规律、武夫当国与文官体系崩溃、养子继承制萌芽、以及沙陀与朱梁争霸四重背景里看。

先看五代更替规律。五代（后梁、后唐、后晋、后汉、后周）是唐亡后中原的五个短命王朝，前后不过五十三年（907-960），却换了五姓八姓十一帝。五代更替的规律，是"以兵变得位、以兵变失位"——每一个王朝的开国之君，都是前朝的藩镇或权臣，靠兵变篡位；每一个王朝的末代之君，都死于下一个藩镇的兵变。后梁朱温篡唐，是五代的开始；后周柴荣（郭威养子）死后赵匡胤陈桥兵变建宋（960），是五代的结束。朱温代唐，开了这个"以兵变得位"的头——可见五代不是正常的朝代更替，而是藩镇武夫的连环兵变。朱温建梁时，已五十五岁，不过十六年便被沙陀李存勖所灭；后梁的短命，预示了五代皆短命的宿命。

再看武夫当国与文官体系崩溃。白马驿之祸，是文官体系崩溃的标志。唐朝的中朝文官，靠科举与门阀撑起，衣冠世族世代为官。可朱温是武夫出身，看不起文官，更要除掉这些"自命清流"的障碍。李振一句"清流投浊流"，把裴枢等三十余朝臣投尸黄河。这一案，杀的不止是三十余人，杀的是唐朝文官体系的根基——自此以后，武夫当国，文官噤声，朝堂成了武夫的私堂。五代的政治，是"以马上得天下，复以马上治之"的武夫政治——文官不被信任，不被重用，朝政全凭武夫的好恶。后梁如此，后唐亦如此（详见《资治通鉴·后唐纪一·同光灭梁》论伶官乱政与武夫政治）。可见白马驿之祸，是五代武夫政治的开端，也是文官体系在五代一蹶不振的根子。

再看养子继承制萌芽。朱温晚年欲传位于养子朱友文（朱温之养子，本姓康），这是五代养子继承制的萌芽。五代养子继承，是沙陀与中原武夫集团的共同现象——李克用养李嗣源（后唐明宗），李嗣源养李从珂（后唐末帝），郭威养柴荣（后周世宗），皆是养子继统。养子继承的兴起，是因为武夫集团重视军功与能力，超过血缘嫡庶——养子多是军中骁将，比亲子更能打仗。可养子继承，破坏了嫡长子继承的稳定性，埋下了养子与亲子争位的内乱隐患。朱温欲立养子友文，激发了亲子友珪的弑父——这是养子继承制第一次引发内乱。后唐明宗养李从珂，李从珂又夺亲子李从厚之位（详见《资治通鉴·后唐纪七·明宗之治》论养子继统的合法性困境）。可见养子继承，是五代的特产，也是五代的祸根。

再看沙陀与朱梁争霸。朱温篡唐建梁时，北方最大的对手是沙陀李克用。李克用是沙陀族（西突厥别部），其父李国昌本姓朱邪，赐姓李。沙陀骑兵骁勇，是平黄巢的主力。朱温与李克用争天下，二人结仇甚深——汴州之变朱温纵火欲杀李克用，自此结为世仇（详见《资治通鉴·唐纪十二·黄巢起义与朱温篡唐》论朱李争霸）。李克用于908年卒，其子李存勖继晋王位。李存勖是李克用亲子，骁勇善战，承父业与后梁争霸。朱温在世时，梁晋互有胜负；朱温死后（912），后梁渐衰，李存勖渐强——923年李存勖称帝建后唐，十月入汴梁亡（详见《资治通鉴·后梁纪五·梁晋争霸》论梁亡与后唐建国）。沙陀李氏与朱梁的争霸，是五代前半叶的主线；沙陀骑兵的骁勇，是后唐灭梁的军事本钱。

## 讲道理

王夫之在《读通鉴论》卷二十八（五代上）里论后梁，话说得极痛。大意据王夫之说，朱温是盗贼出身，与安禄山同类——"朱温，盗也，与安禄山等"；李克用、李存勖、石敬瑭、刘知远是沙陀三部的小夷，郭威是无名窃夺，都不足称天子。王夫之甚至以为"五代不足称代"——五代算不上正常的朝代更替，是藩镇武夫的窃据，与战国相类，应称"战国"而非"五代"。大意据《读通鉴论》卷二十八。王夫之把朱温放在"盗贼窃据"的位置上——朱温以贼平贼起家，借平黄巢之功坐大，借诛宦官之名挟天子，最后弑昭宗篡唐建梁，全无君天下之志，亦无立国之规模。王夫之论朱温之死，大意据王夫之说，朱温弑唐昭宗，其子弑他；屠前朝宗室，其子屠他——弑君者人亦弑之，屠人者人亦屠之，因果循环，丝毫不爽。王夫之的潜台词是：朱温之流，窃据唐之京邑，本非天命所归，必然速亡；以暴易暴，循环不爽。大意据《读通鉴论》卷二十八。

胡三省在《资治通鉴音注》后梁纪部分，对白马驿之祸的批注，多落在"武夫屠戮文官、文官体系一朝断绝"上。大意据胡三省注，白马驿之祸杀的不止是裴枢等三十余人，杀的是唐朝的衣冠世族与文官根基——清流投浊流，是衣冠之族被武夫集体屠戮的标志。胡三省注朱温乱伦诸媳，是带着痛惜的——朱温以武夫篡唐，本无礼法可言，可乱伦诸媳、欲传养子，是连武夫的基本伦常都丢了。胡三省的潜台词是：后梁之速亡，根子在朱温失德——失德之君，难有善终；失国之君，必有失德之由。大意据《资治通鉴音注》后梁纪部分。

司马光在《资治通鉴·后梁纪一》里对朱温代唐与白马驿之祸的记述，亦带其一贯史观。大意据司马光说，朱温篡唐，是唐末藩镇割据、宦官专权、朋党相争三害累积的恶果——唐亡非朱温一人之罪，是唐朝自毁的必然（详见《资治通鉴·唐纪十二·黄巢起义与朱温篡唐》臣光曰论唐亡于纪纲不振）。可朱温建梁后，并未吸取唐亡的教训——屠戮朝臣，是重蹈唐末宦官专权之覆辙；乱伦诸媳，是重蹈唐末君主失德之覆辙；猜忌诸子，是重蹈唐末储位之争之覆辙。司马光的史观，是"治乱在人主之德"——朱温德之不足，必然速亡；后世之君当鉴朱温之失，勿使三害复起。大意据《资治通鉴·后梁纪一》。

## 问道悟道

朱温代唐与后梁三世相杀这出戏，藏着两条让人开眼的规律。

**弑父篡位与因果回旋。**

朱温弑唐昭宗于椒殿，立哀帝；天祐四年篡唐建梁。可朱温建梁不过五年，便被亲子朱友珪所弑——朱友珪半夜闯入万春门，命冯廷谔刺朱温，刃出于背。朱温一生弑君篡位、屠戮宗室，最后死于亲子之手。这是"弑君者人亦弑之"的典型——以暴力得位者，必然以暴力失位；弑父者必被弑，屠兄者必被屠。后梁三世，朱温→朱友珪→朱友贞，朱温弑唐昭宗，友珪弑朱温，友贞诛友珪——三世相杀，无一善终。

弑父篡位与因果回旋这事，历史上有几个对照。南朝刘宋，刘劭弑宋文帝刘义隆（元凶劭），其弟刘骏（宋孝武帝）又起兵诛刘劭——弑父者被诛，循环不爽。大意据《宋书·元凶劭传》。南朝萧梁，侯景乱梁，梁武帝困死台城——侯景以叛将反噬，旋即自立为汉帝，不久亦败死——以暴力得位者，必以暴力失位。大意据《梁书·侯景传》。后梁朱温、刘宋刘劭、萧梁侯景，都是弑君篡位或叛将窃据者，结局都是被诛。可见弑父篡位，得位不正，朝野不服，必然速亡。朱温之死，是因果回旋的典型——他弑唐昭宗，其子弑他；他屠前朝宗室，其子屠他；他乱伦诸媳，其子乱他的宫闱。以暴易暴，循环不爽；弑父者必被弑，这是历史的铁律。要避此祸，须以德服人、以正得位——可五代武夫，有几人肯听？

**屠戮朝臣与士族体系的终结。**

白马驿之祸，朱温在李振怂恿下，于白马驿杀裴枢、独孤损、崔远等朝臣三十余人，投尸黄河。李振说"此辈自谓清流，宜投于浊流"。这一案，杀的不止是三十余人，杀的是唐朝的衣冠世族与文官根基——百年来依靠科举门阀撑起的中朝文官体系，至此一朝断绝。白马驿之祸，是五代武夫政治的开端，也是文官体系在五代一蹶不振的根子。

屠戮朝臣与士族体系终结这事，历史上有几个对照。东汉末，董卓专权，屠戮朝臣——可董卓之屠，规模与系统性不及白马驿。大意据《后汉书·董卓传》。曹魏末，司马氏专权，高平陵之变屠曹爽党羽——可司马氏屠的是政敌，未及整个士族阶层，且司马氏建晋后仍重用士族。大意据《晋书·宣帝纪》。唐朝武则天时期，酷吏来俊臣、周兴大兴告密，屠戮李唐宗室与朝臣——可武则天屠戮是为了改朝换代，未及文官体系的根本，且武则天晚年仍用狄仁杰等文臣。大意据《旧唐书·酷吏传》。白马驿之祸不同——朱温是武夫，李振是落第文人，二人合谋，把唐朝的清流朝士集体投尸黄河，是"以暴力彻底终结士族体系"。此后五代，武夫当国，文官噤声，朝政全凭武夫好恶。可见士族与文官体系，是王朝的政治根基；屠戮朝臣，是自毁根基——根基既断，王朝必然速亡。后梁之短命，根子之一就是白马驿之祸断了文官体系的根。要到宋兴，重用文臣，"与士大夫共治天下"，士族文官体系才得以重建——可那已是五十余年后的事了。

## 结语

朱温代唐建梁，开了五代武夫篡位的头；白马驿之祸断了士族文官的根；弑父篡位的因果回旋，让后梁三世相杀、十六年而亡——以暴力得天下者，必以暴力失天下。
