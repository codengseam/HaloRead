# 资治通鉴·后汉纪一·后汉短祚

## 讲事情

后汉是五代最短命的王朝——立国不过四年（947—950），二世而亡。一个沙陀武夫趁契丹北撤的空档捡了个帝位，他的儿子又因猜忌权臣把自家的柱石一一砍倒，最终被自家大将反噬。后汉的短祚，是"趁乱捡漏"与"自毁柱石"的双重悲剧。

开运四年（即天福十二年，947）正月，契丹入汴，后晋亡（后晋亡国主场详见《资治通鉴·后晋纪六·出帝亡国》）。契丹主耶律德光在汴梁改国号"大辽"，欲做中原皇帝。可契丹兵"打草谷"四掠，激起中原军民奋起，各地义军、藩镇纷纷驱契丹。耶律德光见中原不可久据，于947年三月仓皇北撤，归途病死于杀胡林。大意据《资治通鉴·后汉纪一》《辽史·太宗纪》。这一段"权力真空"，便是后汉建国的契机。

在这场大乱里，河东节度使刘知远一直在太原观望。刘知远是沙陀人（非汉人），本是后晋名将，石敬瑭时镇河东，握有重兵。契丹灭后晋时，刘知远按兵不动，既不援后晋，也不抗契丹——他在等。等到947年二月契丹入汴、中原大乱，刘知远遂在太原称帝，国号"汉"，是为后汉高祖。大意据《资治通鉴·后汉纪一》《旧五代史·汉书·高祖纪》。刘知远是沙陀人，却以"汉"为国号——这是借汉室旧名以张华夏正统，与石敬瑭借"晋"名同理。沙陀三朝（后唐、后晋、后汉）皆以汉晋旧名为国号，是为了在华夏正统的谱系里争一个名分——可名分是借的，根基是空的。契丹北撤后，刘知远自太原南下，入汴梁，定都于此。后汉的版图，大体承后晋而稍缩。

刘知远在位仅一年。乾祐元年（948），刘知远崩，年五十四岁。其子刘承祐即位，是为后汉隐帝。大意据《资治通鉴·后汉纪一》《旧五代史·汉书·隐帝纪》。隐帝即位时年少（约十八岁），朝政皆决于三大权臣——杨邠、史弘肇、王章。杨邠为宰相，总机要；史弘肇为侍卫亲军都指挥使，掌禁军；王章为三司使，掌财赋。三人皆先帝旧臣，势倾朝野，目无少主。大意据《资治通鉴·后汉纪一》《新五代史·汉本纪》。

隐帝年少，可他有少年人的自尊。三大权臣当朝专横，议事不让他插嘴；隐帝欲有所作为，三人辄以"陛下但禁声"——陛下只管闭嘴——压之。隐帝积怨日深，遂生杀心。乾祐三年（950）十一月，隐帝与亲信李业、聂文进等密谋，在朝堂上伏兵杀杨邠、史弘肇、王章三人，夷其族。大意据《资治通鉴·后汉纪一》《旧五代史·汉书·隐帝纪》。这三人皆先帝柱石，一夕尽诛，朝野震动。

可隐帝杀三大臣还不罢休。他忌惮在外领兵的大将郭威——郭威是后汉宿将，时任邺都（今河北临漳）留守，节制河北。隐帝密遣使赴邺，欲杀郭威；又遣人赴汴梁杀郭威家属。郭威妻柴氏（柴荣姑母，详见人物段）与郭威诸子、婴儿，尽被诛于汴梁。大意据《资治通鉴·后汉纪一》《旧五代史·周书·太祖纪》。郭威闻家属被屠，悲愤交加，遂以"清君侧、诛奸臣"为名，自邺城起兵南下。隐帝遣兵迎击，可后汉军多附郭威，一战即溃。隐帝亲自出城督战，兵败，逃至赵村，为近臣郭允明所杀（一说为乱兵所杀）。大意据《资治通鉴·后汉纪一》。后汉自947年建国，至950年隐帝被杀，凡二世二帝四年——是五代最短命的王朝。

隐帝被杀后，郭威入汴，初立刘氏宗族刘赟为帝（刘赟是刘知远弟刘崇之子），自己率军北上御契丹。可行至澶州，诸军将士拥郭威为帝，裂黄旗披其身——是为"澶州兵变"。郭威遂还汴，废刘赟，于广顺元年（951）正月即位，国号"周"，是为后周太祖（郭威建后周主场在《资治通鉴·后周纪五·世宗柴荣》）。大意据《资治通鉴·后周纪一》《旧五代史·周书·太祖纪》。后汉亡，后周兴，五代又翻过一页。后汉的四年，是五代最仓促的一笔——立国靠捡漏，亡国靠自毁，匆匆而来，匆匆而去。

## 讲人物

**刘知远：趁乱捡漏的沙陀高祖**

刘知远是沙陀人（非汉人），本名刘知远，后晋时为名将，石敬瑭镇河东时即随其征战。石敬瑭建后晋后，刘知远镇河东，握有重兵。大意据《旧五代史·汉书·高祖纪》《新五代史·汉本纪》。刘知远其人，骁勇善战，亦能忍——这是他能成事的关键。

刘知远最大的本事，是"等"。947年契丹灭后晋，中原大乱，刘知远在太原按兵不动，既不援后晋，也不抗契丹。他在等什么？等契丹与中原两败俱伤，等权力真空出现。等到947年二月契丹入汴、三月耶律德光北撤身死，刘知远遂在太原称帝，趁势南下，入汴梁，捡了一个现成的帝位。这份"等"的本事，是五代武夫中少有的——多数武夫急躁冒进，刘知远却沉得住气。可这份"等"，亦是其短——他捡的是别人打烂的摊子，根基本就空。

刘知远以沙陀人之身，国号称"汉"，是借汉室旧名以张华夏正统。这与石敬瑭借"晋"名（攀晋朝）、李克用李存勖借"唐"名（攀唐朝）同理——沙陀三朝皆以汉晋唐旧名为国号，是为了在华夏正统的谱系里争一个名分。可名分是借的，根基是空的——刘知远虽入汴，可后晋留下的烂摊子未整，藩镇未服，财政未稳，他仅在位一年即崩，连根基都来不及打。刘知远留给隐帝刘承祐的，是一个名分虽正、根基却空的王朝。这份"趁乱捡漏"得来的帝位，注定了后汉的短命——根基不固的王朝，经不起任何风浪。

刘知远临终，托孤于杨邠、史弘肇、王章、郭威等人。可他没料到，他托的这群柱石，成了隐帝心头最大的刺——隐帝一夕尽诛三大臣，又逼反郭威，亲手把后汉推向灭亡。刘知远捡漏得国，他的儿子自毁柱石亡国——成也捡漏，亡也自毁，是后汉短祚的双面写照。

**郭威：被逼反又建后周的太祖**

郭威是后汉宿将，邢州尧山人（今河北邢台一带），本是寒微出身，年少时颈上刺一飞雀，人称"郭雀儿"。郭威从军后，历事后唐、后晋、后汉三朝，骁勇善战，屡立战功，至后汉时已是手握重兵的大将。大意据《旧五代史·周书·太祖纪》《新五代史·周本纪》。

郭威与刘知远关系极深。刘知远镇河东时，郭威是其心腹；刘知远称帝，郭威出力甚多。后汉建国后，郭威为枢密使，掌天下兵权。隐帝即位，郭威以顾命大臣身份辅政，本与杨邠、史弘肇、王章并列为先帝托孤之臣。可隐帝忌惮权臣，一夕尽诛杨邠等三人，又密谋杀郭威，更诛郭威家属——郭威妻柴氏、诸子、婴儿，尽死于汴梁。大意据《资治通鉴·后汉纪一》《旧五代史·周书·太祖纪》。

郭威被逼反，是后汉亡国的直接原因。他本无反心——他若反，早在契丹入汴时便可趁乱自立。他一直忠事后汉，直到隐帝屠其家属，悲愤交加，遂以"清君侧"为名起兵。郭威的反，是"被逼的反"——这份"被逼"，亦是五代武夫的共同处境：兵权在将，君猜则将反，将反则君亡。郭威入汴后，初立刘赟为帝，自己率军北上——这本是给后汉留一条后路。可澶州兵变，诸军拥立，郭威遂废刘赟，建后周。郭威建后周，是后汉自毁柱石的反噬——隐帝杀三大臣、逼反郭威，亲手把自家的柱石变成了新王朝的太祖。郭威其人，后文世宗柴荣篇再详（郭威建后周、养柴荣继位详见《资治通鉴·后周纪五·世宗柴荣》）。

**杨邠、史弘肇、王章：被一夕尽诛的三大柱石**

杨邠、史弘肇、王章，是后汉隐帝朝的三大权臣，亦是被隐帝一夕尽诛的三大柱石。三人皆先帝刘知远旧臣，受顾命辅政。杨邠为宰相，总机要，为人刚直，议事不让；史弘肇为侍卫亲军都指挥使，掌禁军，治军严酷；王章为三司使，掌财赋，聚敛刻薄。三人皆以"先帝旧臣"自居，目无少主。大意据《资治通鉴·后汉纪一》《新五代史·汉本纪》。

三人当朝专横，是后汉亡国的导火索。议事时，隐帝欲有所主张，三人辄以"陛下但禁声"压之——陛下只管闭嘴，朝廷的事我们做主。这话，是少年皇帝最不能忍的屈辱。史弘肇更曾当众说："有臣等在，陛下何忧？"——有我们在，陛下愁什么。这份傲慢，把隐帝的自尊踩进了泥里。隐帝年少，可他有少年人的血性——他要的不是被供养的傀儡位，而是真正的皇权。于是950年十一月，隐帝伏兵朝堂，一夕尽诛三人，夷其族。

可隐帝杀三人，杀的是后汉的柱石。杨邠总机要，史弘肇掌禁军，王章握财赋——三人一去，后汉的军政财三权顿时失了主心骨。更要命的是，隐帝杀三人还不够，又去逼反在外领兵的郭威——三大柱石已诛，第四根柱石郭威又被逼反，后汉焉得不亡？三人之死，是后汉自毁柱石的极致——隐帝为泄一己之愤，把先帝托付的柱石尽数砍倒，最终自己也死于乱兵之手。三人的悲剧，是"权臣当朝"与"少主自尊"的必然冲突——权臣不知退让，少主不知隐忍，冲突一起，玉石俱焚。

## 讲背景

后汉短祚这事，得放在沙陀三朝的族源正统、五代速亡的节奏、权臣辅政与少主自尊的冲突、被逼反的兵权格局四重背景里看。

先看沙陀三朝的族源正统。后唐、后晋、后汉三朝，建立者都是沙陀人——李克用、李存勖、李嗣源是沙陀（详见《资治通鉴·后唐纪一·同光灭梁》）；石敬瑭是沙陀（详见《资治通鉴·后晋纪一·石敬瑭借兵建晋》）；刘知远也是沙陀。沙陀本是西突厥别部，入主中原后，并未真正建立稳固的统治根基——他们靠军功立国，靠养子、女婿等拟血缘关系维系内部，靠藩镇体系瓜分地盘。沙陀三朝皆以汉晋唐旧名为国号——后唐攀唐朝、后晋攀晋朝、后汉攀汉朝——是为了在华夏正统的谱系里争一个名分。可名分是借的，根基是空的。刘知远以沙陀人之身国号称"汉"，是借汉室旧名以张华夏正统；可这份正统，是名分上的正统，不是根基上的正统。后汉四年即亡，连根基都来不及打——可见借来的正统，撑不起一个王朝。

再看五代速亡的节奏。五代王朝，一个比一个短命——后梁十六年、后唐十四年、后晋十一年、后汉四年、后周十年。越到后来，王朝越短命。后汉四年，是五代最短命的。这份速亡律，根子在"以兵变得位、以兵变失位"——兵变可立君，亦可废君；兵权在将不在君，王朝便无法长治（速亡律详见《资治通鉴·后晋纪六·出帝亡国》论短命王朝的速亡律）。后汉之亡，是速亡律的极致——刘知远捡漏得国，根基本空；隐帝自毁柱石，逼反郭威；郭威一反，后汉即亡。立国靠捡漏，亡国靠自毁，后汉的四年，是五代速亡律最浓缩的写照。

再看权臣辅政与少主自尊的冲突。后汉隐帝朝的三大权臣杨邠、史弘肇、王章，皆先帝托孤之臣，势倾朝野，目无少主。三人当朝专横，议事不让隐帝插嘴，"陛下但禁声"的话出口，把隐帝的自尊踩进泥里。这份冲突，是"权臣辅政"体制的根本矛盾——权臣以"先帝旧臣"自居，少主以"正统天子"自期，权臣不知退让，少主不知隐忍，冲突一起便玉石俱焚。历史上权臣辅政与少主自尊的冲突代代不绝——霍光辅汉昭帝、诸葛亮辅刘禅、鳌拜辅康熙，都是同构。可霍光能善终，是霍光知退让、汉昭帝早卒；鳌拜被擒，是康熙能隐忍待时。后汉隐帝不能隐忍，三大臣不知退让，遂两败俱亡。

最后看被逼反的兵权格局。五代兵权在将不在君，是速亡律的根子。郭威本无反心——他若反，早在契丹入汴时便可趁乱自立。他一直忠事后汉，直到隐帝屠其家属，悲愤交加，才以"清君侧"为名起兵。郭威的反，是"被逼的反"。这份"被逼"，是五代武夫的共同处境——兵权在将，君猜则将反，将反则君亡。隐帝猜忌郭威，诛其家属，本是为固皇权；可这一手逼反郭威，反把后汉推向灭亡。可见猜忌兵权在将的大将，是把双刃剑——不猜则将可能坐大，猜则将可能反；隐帝一猜，郭威一反，后汉即亡。这份被逼反的兵权格局，是五代武夫政治的根本困境——君将之间，信任难立，猜忌则反，遂成死局。

## 讲道理

王夫之在《读通鉴论》卷三十（五代下，论汉周）里论后汉之亡，话亦极痛。大意据王夫之说，后汉之亡，亡于隐帝自毁柱石——杨邠、史弘肇、王章三人虽专横，皆是先帝柱石；郭威本忠臣，被逼反乃起兵。王夫之以为，隐帝一夕尽诛三大臣，又逼反郭威，是"自毁其藩"的极致——不为外敌所灭，先为自家所毁。王夫之把后汉之亡归咎于"少主不知隐忍，权臣不知退让"——隐帝年少气盛，三大臣傲慢无礼，冲突一起便玉石俱焚。大意据《读通鉴论》卷三十。王夫之亦论刘知远捡漏得国，以为根基本空——趁契丹北撤捡个帝位，名分虽正，根基不固，故经不起任何风浪。

胡三省在《资治通鉴音注》后汉纪部分，对隐帝诛权臣的批注，亦多带惋惜。大意据胡三省注，隐帝诛杨邠等三人，是"自毁柱石"——三人虽专横，皆先帝托孤之臣，朝政军权财赋系于一身；一夕尽诛，后汉顿时失了主心骨。胡三省以为，隐帝杀三人还不够，又逼反郭威，是"自毁其藩"的极致——四大柱石去其三，再逼反最后一位，后汉焉得不亡。胡三省亦注郭威被逼反，以为郭威本忠臣，屠其家属乃起兵，是"被逼的反"——这份"被逼"，是五代武夫的共同处境。大意据《资治通鉴音注》后汉纪部分。胡三省的潜台词是：后汉之亡，非外敌之祸，乃自毁之祸——隐帝为泄一己之愤，把先帝柱石尽数砍倒，最终自己也死于乱兵之手。

司马光在《资治通鉴·后汉纪一》里对后汉短祚的记述，亦带其一贯史观。大意据司马光说，隐帝年少，不能隐忍，三大臣专横，不知退让，两败俱亡。司马光以为，权臣辅政当以谦退为本，少主自尊当以隐忍为先——权臣不退则少主疑，少主不忍则权臣死，此乃辅政之大忌。司马光的史观，是"治乱在人主之德"——隐帝无德无能，故自毁柱石；可司马光亦深责三大臣之骄横，以为权臣当以谦退保身，不当以傲慢取祸。大意据《资治通鉴·后汉纪一》。三家的史观，殊途同归：后汉之亡，是自毁柱石的必然结局——少主不知隐忍，权臣不知退让，玉石俱焚，王朝短祚。

## 问道悟道

后汉短祚这出戏，藏着两条让人开眼的规律。

**养子承祧与家天下终结。**

后汉是五代承前启后的一环——它承后晋之亡而立，又启后周之兴而亡。可后汉的承启，有一个隐而未发的关节：刘知远传位于亲子刘承祐（隐帝），未走养子继统的路子。这在五代沙陀三朝中是少有的——后唐两代以养子继位（李嗣源养于李克用、李从珂养于李嗣源，养子继统的合法性困境详见《资治通鉴·后唐纪七·明宗之治》），后晋以养子石重贵继位（石重贵养于石敬瑭，详见《资治通鉴·后晋纪一·石敬瑭借兵建晋》），唯独后汉走的是亲子继统。可这份亲子继统，并未让后汉长治——隐帝年少无能，自毁柱石，四年即亡。

可见养子继统是五代的特产，亲子继统在五代亦非长治之道——根子不在养子与亲子之分，而在"得国不正、根基不固"。后唐养子继统引发内乱，后晋养子继统亦亡国，后汉亲子继统亦短祚——可见五代之速亡，不在继统形式，而在"以兵变得位"的根本痼疾。养子承祧与家天下终结这事，折射的是五代家天下秩序的彻底崩解——无论养子亲子，都撑不起一个长治的王朝。要终结这乱局，须重建家天下之外的制度根基——可五代武夫重兵变轻制度，养子继统与亲子继统的争论不过是皮相之争。要到宋兴，以文制武、立宗法制度，家天下才得以重建——可那已是后汉亡后十一年的事了。

养子承祧与家天下终结这事，历史上有几个对照。后周郭威养柴荣继位（柴荣本姓柴，郭威内侄兼养子，详见《资治通鉴·后周纪五·世宗柴荣》），是养子继统的少数成功案例——可柴荣早卒（959），后周旋亡，可见纵有成功，亦难长久。南宋高宗无子，养太祖后裔赵昚为嗣（宋孝宗），是养子继统的成功案例——可南宋偏安，养子继统未引发内乱，是特例。大意据《宋史·孝宗本纪》。后唐李嗣源、李从珂，后晋石重贵，后汉隐帝（亲子继统），后周柴荣——五代五朝的继统，养子亲子皆有，可无一长治。可见家天下的长治，不在继统形式，而在制度根基——养子继统引发内乱是表，制度不立才是里。

**诛权臣与自毁柱石。**

隐帝一夕尽诛杨邠、史弘肇、王章三大臣，又逼反郭威——是后汉亡国的直接原因。三大臣虽专横，皆先帝柱石，朝政军权财赋系于一身；一夕尽诛，后汉顿时失了主心骨。更要命的是，诛三大臣还不够，又去逼反在外领兵的郭威——四大柱石去其三，再逼反最后一位，后汉焉得不亡。隐帝诛权臣，本是为固皇权；可这一手自毁柱石，反把后汉推向灭亡。可见诛权臣是双刃剑——权臣当朝固可虑，可一夕尽诛则失柱石，王朝顿成空壳。

诛权臣与自毁柱石这事，历史上有几个对照。汉景帝诛晁错——七国之乱时景帝听袁盎言诛晁错，本欲息藩王之怒，可晁错一死，七国仍反，景帝反失一良臣。大意据《汉书·袁盎晁错传》。明崇祯帝诛袁崇焕——崇祯中后金反间，凌迟袁崇焕，自毁辽东柱石，明亡辽东尽失。大意据《明史·袁崇焕传》。唐肃宗听宦官谗杀高仙芝、封常清——安史之乱初，肃宗听边令诚谗杀二将，自毁平叛柱石，唐军大败。大意据《旧唐书·封常清传》。汉景帝诛晁错、明崇祯诛袁崇焕、唐肃宗杀高仙芝封常清、后汉隐帝诛三大臣——都是自毁柱石的典型。权臣当朝固可虑，可诛权臣须有后手——无后手则失柱石，王朝顿成空壳。可见诛权臣易，补柱石难——一夕尽诛是痛快的，可痛快之后是空壳，空壳则亡。要免此祸，须以制度制衡权臣，不一夕尽诛——可隐帝年少气盛，无此城府，遂速其祸。

## 结语

四年短祚，立国靠捡漏、亡国靠自毁——后汉隐帝一夕尽诛三大权臣又逼反郭威，把先帝柱石尽数砍倒，最终自己也死于乱兵之手，是五代自毁柱石最浓缩的一笔。
