# 子路——勇而好义的悲剧

## 一、讲事情

子路初见孔子那一段，是孔门弟子里最有戏剧性的开场。

《史记·仲尼弟子列传》记：子路性鄙，好勇力，志伉直，冠雄鸡，佩猳豚，陵暴孔子。孔子设礼稍诱子路，子路后儒服委质，因门人请为弟子。

什么意思？子路年轻时是个粗野汉子，头上插着雄鸡毛，腰里挂着公猪皮——这是当时武士的打扮，雄鸡好斗，公猪凶猛，戴这些就是显摆自己勇猛。他碰见孔子，不但不敬，还欺负孔子。孔子没跟他硬来，慢慢用礼乐开导他。子路后来换了儒服，通过门人请求做了孔子的弟子。

这一段太有画面感。一个插着鸡毛、挂着猪皮的莽汉，撞上一个文质彬彬的读书人，按常理该是读书人挨揍。可孔子不慌不忙，用礼把子路给"诱"了过来。"诱"这个字用得妙——不是压服，是吸引。子路这种人，你越压他越犟；用礼乐一化，他反而服了。

子路一进孔门，就成了孔子身边最特别的人。他比孔子小九岁，可跟孔子说话从没大没小。孔子周游列国，子路多半是赶车的。孔子夸他能治军——"由也，千乘之国，可使治其赋也"（《论语·公冶长》），可也最常敲打他。

子路好勇，孔子没少说他。《公冶长》记："道不行，乘桴浮于海。从我者其由与？"子路闻之喜。子曰："由也好勇过我，无所取材。"——孔子叹气说："道要是行不通，我就乘着木筏浮海而去。跟着我的，怕是仲由吧？"子路一听，高兴坏了——老师最看重我！孔子紧接着泼一盆冷水："仲由好勇超过了我，可就是没地方取木材。"

这话有两层。表面是说子路光有勇，没材料造筏子。深层是说：子路啊，你勇是勇，可勇得没个裁度。孔子不是否定子路的勇，是嫌他的勇没有"裁"——没有节制，没有方向。

可孔子心里头，是认子路的。《公冶长》还有一段："子路有闻，未之能行，唯恐有闻。"——子路听到一个道理，还没来得及去做，就怕又听到新的。这话听着像批评，其实是夸。夸什么？夸子路是真干的人——听到一个道理，立刻要去做，做不完就不敢再听新的。这种"知行合一"的劲头，孔门弟子没几个比得上。

最有名的是子路治蒲。子路做卫国蒲邑的大夫，治了三年。孔子路过蒲地，《孔子家语·辩政》记了三称其善的故事：

孔子入其境，曰："善哉！由恭敬以信矣。"——刚进蒲地边境，孔子就说："好啊！仲由恭敬而取信于民了。"入其邑，曰："善哉！由忠信而宽矣。"——进了城邑，孔子又说："好啊！仲由忠信而宽厚了。"至其庭，曰："善哉！由明察以断矣。"——到了庭堂，孔子再说："好啊！仲由明察善断了。"

子路治蒲三年，孔子三称其善——入境称其"恭敬以信"，入邑称其"忠信而宽"，至庭称其"明察以断"。从边境到城邑到庭堂，一路夸过来。这是孔子少有的对子路的全面肯定。可见子路不是只会舞刀弄枪的莽夫，他能治民，能宽厚，能明断。

可子路的结局，孔子早就预感到了。《先进》记："闵子侍侧，闇闇如也；子路，行行如也；冉有、子贡，侃侃如也。子乐。'若由也，不得其死然。'"——闵子骞侍立身旁，恭敬正直；子路，刚强外露；冉有、子贡，和乐从容。孔子看了高兴，可又说："像仲由这样，怕是不得好死吧。"

"行行如也"——刚强外露的样子。孔子一看子路那副刚强相，就替他担心。这话不是诅咒，是忧虑——性格刚强、好勇斗狠的人，在那个乱世，最容易死在祸难里。

后来这话应验了。卫国发生蒯聩之乱，子路在孔悝之难中遇难。《左传·哀公十五年》记得详细：子路当时不在城里，听到消息赶回去。城门关了，有人劝他别进去，子路说："食焉，不避其难。"——吃了人家的俸禄，就不能躲避人家的祸难。他冲进城，要烧蒯聩挟持孔悝的台子。蒯聩派人打子路，子路的帽缨被砍断。子路说了一句千古流传的话："君子死，冠不免。"——君子就算死，帽子也要戴正。他放下剑去系帽缨，被敌人砍成了肉酱。

"结缨而死"——系着帽缨死去。这一幕，是子路一生的句号。

孔子在鲁国听说卫国发生内乱，叹了一口气："嗟乎，由死矣！"——《史记·仲尼弟子列传》。没多久，卫国报丧的人来了。孔子正在吃饭，听到子路被剁成肉酱，把嘴里的肉酱吐了，叫人把家里的肉酱全倒掉——"覆醢"。

## 二、讲人物

子路这个人，最难写，因为他太"满"。

他粗野。初见孔子时"冠雄鸡，佩猳豚，陵暴孔子"，活脱脱一个市井泼皮。进了孔门，性子也没全改。《论语·先进》记："子路使子羔为费宰。子曰：'贼夫人之子。'子路曰：'有民人焉，有社稷焉。何必读书，然后为学？'子曰：'是故恶夫佞者。'"——子路推荐子羔去做费邑长官。孔子说："你这是害人家。"子路嘴硬："那里有百姓有社稷，何必非读书才算学？"孔子骂："所以我讨厌嘴利的人。"子路跟老师顶嘴，理不直气也壮，可爱也可气。

他忠诚。孔子在陈蔡绝粮，子路愠见曰："君子亦有穷乎？"——《论语·卫灵公》。他不是抱怨自己穷，是替老师不平——君子怎么也落到这步田地？这份不平，是忠。孔子周游列国，子路一路赶车、护卫、张罗。

他直率。孔子去见南子——卫灵公的夫人，名声不好。子路不高兴，"子见南子，子路不说"——《论语·雍也》。孔子急得发誓："予所否者，天厌之！天厌之！"——我要是做了不对的事，老天厌弃我！老师被学生逼得发誓，孔门里也就子路能干出来。可这份直率，正是子路的可贵——他不阿谀，不奉承，老师做得不对他就皱眉。

他重义。子路治蒲，治了三年，孔子三称其善。可他做蒲大夫，不是为了官，是为了义。后来卫国蒯聩作乱，挟持孔悝。子路当时是孔悝的蒲大夫，完全可以躲——城门关了，有人劝他别进。可他说"食焉，不避其难"——吃了人家的饭，就不能躲人家的祸。这是义。

可子路也有他的毛病。他好勇，勇得过头。孔子说他"由也好勇过我，无所取材"——《论语·公冶长》。又说他"若由也，不得其死然"——《论语·先进》。孔子不是不爱子路的勇，是怕这勇没个收束，迟早要出事。

孔子对子路，是又爱又忧。爱他的真，忧他的刚。子路是孔门弟子里跟孔子最亲的几个之一——颜回是精神传人，子贡是外交能手，子路是贴身护卫。孔子出门，子路赶车；孔子有难，子路挡着。可越爱子路，越替他担心——这么刚的人，乱世里怎么活？

孔子说"由也好勇过我"，这话里有惋惜。惋惜什么？惋惜子路的勇没经过"裁"——没有礼来节制，没有义来定向。勇是好的，可勇得没裁度，就成了危险。孔子这句话，是预言，也是遗恨。

## 三、讲背景

子路的出身，得先说清楚。

《史记·仲尼弟子列传》记："子路，卞人也。"卞是鲁国的一个小地方。子路出身底层——春秋时候，"国人"是城里的自由民，"野人"是城外的乡下人，地位低一等。子路就是从底层出来的。

这出身决定了子路的底色。他没受过贵族教育，不懂礼乐，年轻时候就是个好勇斗狠的莽汉。他"冠雄鸡，佩猳豚"，是武士的打扮，也是底层人往上爬的方式——靠一身力气，靠一股狠劲，博个出身。这种出身的人，性子直，脾气暴，可也最讲义气，最重承诺。

再说卫国的乱局。子路后来死在卫国，得说清楚卫国为什么乱。

卫国的事，是一笔糊涂账。卫灵公的夫人南子，跟太子蒯聩关系不好。蒯聩想杀南子，没成功，逃到了晋国。卫灵公死了，蒯聩的儿子辄继位，就是卫出公。可蒯聩还活着——他是太子的爹，是国君的爹，可他人在晋国，回不来。这下就尴尬了：儿子做了国君，爹在外面流亡。

蒯聩不甘心，他想回国夺位。他借助晋国的力量，又联合卫国的大夫孔悝，搞政变。蒯聩混进孔悝家，挟持孔悝，逼孔悝拥立自己。卫出公跑了，蒯聩上位，就是卫庄公。

子路为什么卷进去？因为子路是孔悝的蒲大夫。蒲是孔悝的封地，子路是孔悝家臣的头儿。蒯聩挟持孔悝，等于动了子路的主君。按当时的规矩，家臣有义务为主君赴难——"食焉，不避其难"。子路听到消息，赶回卫国都城，要救孔悝，结果死在乱兵里。

这里头有个背景：春秋时期的"士为知己者死"风气。那时候的家臣跟主君，不是简单的雇佣关系，是带有半人身依附性质的恩义关系。主君给你俸禄、封地、地位，你就得为主君效死。这种风气，是周代宗法制度的延伸——臣对君，像子对父。

子路死的时候，其实已经不在孔悝身边了。他当时在蒲邑，听到都城出事，完全可以不回去——蒲邑离都城远，他又是地方官，守好蒲邑就行。可子路不这么想。他觉得吃了孔悝的饭，孔悝有难他就得去。

可这劲儿，孔子是不赞成的。孔子说过："邦有道，不废；邦无道，免于刑戮"——《论语·公冶长》。国家有道就出来做事，国家无道就保住自己别受刑戮。孔子还说过："危邦不入，乱邦不居"——《论语·泰伯》。危险的国家不进去，动乱的国家不待着。孔子自己是这么做的——他周游列国，看到哪个国家要乱，就赶紧走。

子路偏偏反着来。卫国都乱了，他还往里冲。这就是孔子为什么早就预感"若由也，不得其死然"——子路这性子，迟早要栽在乱局里。

## 四、讲道理

子路的"勇"，得分层次看。

第一层是血气之勇。子路初见孔子时"冠雄鸡，佩猳豚，陵暴孔子"，这是血气之勇——凭一身力气，凭一股狠劲，谁都不怕。这种勇，动物也有，公鸡斗、公猪咬，都是血气。朱熹《论语集注》在"由也好勇过我，无所取材"下注："夫子美其勇于为善，而惜其无所取材以裁之也。"——孔子欣赏子路勇于为善，可惋惜他没有材料来裁度自己。朱熹点出一个"裁"字——勇要有裁度，没裁度的勇是血气，有裁度的勇才是德。

第二层是义理之勇。子路跟孔子学了之后，勇没丢，可勇里头多了"义"。他治蒲三年，"恭敬以信""忠信而宽""明察以断"——这不是血气之勇能干出来的，是义理之勇。他听到道理就去做，"子路有闻，未之能行，唯恐有闻"——《论语·公冶长》。朱熹注："前所闻者既未及行，恐复有所闻而行之不给也。"——前面听到的还没做，怕又听到新的来不及做。朱熹这一注，点出子路的"行"——他不是听听说说，是真干。这种"知行合一"的劲头，是义理之勇的根。

第三层是死节之勇。子路结缨而死，"君子死，冠不免"——这是死节之勇。可这层勇，孔子是不认可的。孔子讲"危邦不入，乱邦不居"，讲"邦无道，免于刑戮"。子路偏偏往乱邦里冲，孔子早就预感他要出事。钱穆《论语新解》论子路，认为子路的勇是血气与义理的混合——他学了义理，可血气没全化掉，遇到大事，血气就压过义理。子路的悲剧在于勇有余而智不足——勇是够了，可智谋不够，看不清大局，分不清该不该死。

钱穆这一分析，点破了子路悲剧的必然性。子路不是不义，是不智。他吃了孔悝的饭，觉得该为孔悝死——这是义。可孔悝是被蒯聩挟持的，子路去救孔悝，等于跟蒯聩作对——蒯聩是来夺位的，背后有晋国支持，子路一个人能挡得住？这是不智。义是义了，可义得没看清楚局势，就成了无谓的牺牲。

朱熹在"若由也，不得其死然"下注："先事而出，言其必死于祸难也。"——孔子在事情发生之前就说出这话，预言子路必死于祸难。朱熹点出"先事而出"四个字——孔子是提前预见的。为什么能预见？因为孔子看透了子路的性格——刚强好勇，不知变通，乱世里必死。这不是孔子神机妙算，是孔子对弟子性格的深刻了解。

子路的悲剧，是性格的悲剧。他的勇是美德，可这美德没经过足够的"裁"，遇到大事就失控。孔子一辈子想"裁"子路的勇，可没裁成。子路最后死在自己最突出的品质上——勇。

## 五、问道悟道

从子路身上，能悟出几条让人开眼的规律。

**勇是美德，也是危险。**

勇是美德，这没人否认。可勇也是危险——勇得没节制，勇得没方向，就成了祸。子路勇了一辈子，最后死在勇上。孔子早就看出这点，所以说"由也好勇过我，无所取材"。

这事古今中外都有印证。古希腊亚里士多德讲"中道"——勇敢在怯懦与鲁莽之间。子路的勇偏向鲁莽那一端。他还说，勇敢要靠理性指导，没有理性的勇敢是蛮勇。子路结缨而死，理性告诉他该走——"危邦不入"，可血气让他往里冲。理性没压住血气，勇就成了蛮勇。

中国历史上这样的例子不少。项羽力能扛鼎，勇冠三军，可最后乌江自刎——勇得没谋，败给了刘邦。勇是利器，可利器没鞘，伤人也伤己。

**忠诚需要智慧。**

子路忠于孔悝，吃了人家的饭就为人家的祸赴难。这份忠诚是真的，可这份忠诚没智慧。孔悝是被挟持的，蒯聩背后有晋国，子路一个人冲进去，救不了孔悝，还搭上自己。这是愚忠。

孔子讲"危邦不入，乱邦不居"——这不是教人怕死，是教人有智慧。该死的时候死，不该死的时候别白死。子路之死，是"不该死而死"。孔子自己周游列国，遇到危险就跑——在宋国被桓魋追杀，他换装走人；在陈蔡被围，他设法脱身。孔子不是不忠，是不愚忠。

这事有反证。管仲辅佐公子纠，公子纠被齐桓公杀了，管仲没殉死，反而辅佐齐桓公成就霸业。子路和子贡都质疑管仲"非仁者与"——《论语·宪问》。孔子却说："管仲相桓公，霸诸侯，一匡天下，民到于今受其赐。"——管仲辅佐齐桓公，一匡天下，百姓到今天还受他的恩赐。孔子还说："岂若匹夫匹妇之为谅也，自经于沟渎而莫之知也？"——难道要像匹夫匹妇那样守小信，在沟渠里上吊没人知道？管仲不死，是智；子路死，是不智。

**性格决定命运。**

子路的命运，从他"冠雄鸡，佩猳豚"那天就定了。他性子刚，好勇，认死理——这种性格，太平年月是良将，乱世里就是炮灰。孔子看透了这点，所以说"若由也，不得其死然"。

性格决定命运，这事古今皆然。韩信能忍胯下之辱，所以成大事；可他后来居功自傲，不肯反刘邦，最后死在吕后手里——能忍小辱，不能忍大疑，性格的矛盾决定结局。性格里的"直"和"刚"，是美德，也是催命符。

可性格决定命运，不等于命运不可改。孔子一辈子想"裁"子路的勇，就是想改他的命。子路学了义理，勇里头多了"义"，治蒲三年政绩斐然——这说明性格是可以修炼的。只是子路修炼得不够，遇到大事，血气又压过义理。性格的修炼是一辈子的事，不到死不能说成。

子路的悲剧，不是他不够好，是他不够"裁"。他的勇、他的忠、他的义，都是真的，可这些真的东西没经过足够的裁度，最后一起把他推向了结缨而死。这才是子路悲剧最让人心痛的地方——他不是死于恶，是死于善没经过裁度。

---

## 结语

勇是美德，可美德没经过裁度，就成了催命的刀。
