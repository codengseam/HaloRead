# 学而时习——学习的本质是什么

## 一、讲事情

《论语》二十篇，开篇第一句，不是讲仁，不是讲礼，不是讲政，是讲学。

"学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？"——《论语·学而》

这三句话，是孔子一生学问的总纲。后人读《论语》，往往急着往后翻，去找那些讲仁、讲孝、讲政的大道理，却把开篇这三句轻轻放过了。其实孔子把"学"放在第一位，不是随便排的——他这一辈子，最看重的就是这件事。

有一回，孔子跟人说起自己，把"学"在他心里的分量交代得清清楚楚。

"十室之邑，必有忠信如丘者焉，不如丘之好学也。"——《论语·公冶长》

十户人家的小村子，一定有忠信像我这样的人，只是不如我这样好学罢了。

这话听着谦虚，其实分量极重。孔子不跟人比忠信——十户小村就有跟他一样忠信的人，忠信不算稀罕。孔子只跟人比一样东西：好学。这一样，没人比得过他。孔子不自称"博学"，自称"好学"。这两个字差一字，意思差了十万八千里。博学是已经学富五车，好学是还在拼命地学。一个说的是结果，一个说的是态度。孔子选的是态度。

再说颜回。颜回是孔子门下最得意的弟子，可颜回让孔子最看重的，不是聪明，不是德行，是"好学"。

季康子问孔子："弟子孰为好学？"孔子答："有颜回者好学，不幸短命死矣，今也则亡。"——《论语·雍也》

季康子问弟子里有谁好学。孔子说，有个颜回好学，可惜短命死了，现在没有了。

"今也则亡"——现在没有了。这四个字，是一个老师对弟子最高的评价，也是最深的痛惜。孔子门下弟子三千，贤者七十二，可孔子说"好学"的，只颜回一个。颜回好学，好在哪里？《论语·雍也》记孔子的话："不迁怒，不贰过。"——不把怒气撒到别人身上，同样的错不犯第二回。

这两条，听着简单，做起来难如登天。人不迁怒，得先管住自己的情绪；不贰过，得先认错，再改错。这两件事，靠的不是聪明，是功夫。颜回的"好学"，不是书读得多，是做人的功夫下得深。

还有子夏。子夏比孔子小四十四岁，是孔门后期的弟子。子夏有一句话，把"学"讲得极透：

"日知其所亡，月无忘其所能，可谓好学也已矣。"——《论语·子张》

每天知道自己所没有的——今天不懂的，弄懂；今天不会的，学会。每月不忘自己所能的——已经学会的，常温习，别丢掉。这叫好学。

子夏这话，把"学"拆成了两头：一头是"知新"，一头是"温故"。两头都得顾，只顾一头不算好学。

孔子还说过一句话，把"学"的境界分了三层：

"知之者不如好之者，好之者不如乐之者。"——《论语·雍也》

知道的人不如喜好的人，喜好的人不如以之为乐的人。

第一层，知道——懂了，会了，可心里未必上心。第二层，好——喜欢，主动去追。第三层，乐——以之为乐，乐在其中，停不下来。孔子把"学"的最高境界定为"乐"。学到最后，不是苦差事，是乐趣。这跟开篇"不亦说乎"的"说"（悦）正好呼应——学的本质，是悦，是乐。

## 二、讲人物

**孔子：以"好学"自许的人**

孔子这个人，最容易被后人贴上"圣人"的标签，可他自己从来不这么看自己。他怎么看自己？看《论语·述而》里那段有名的话：

"默而识之，学而不厌，诲人不倦，何有于我哉？"

默默记住所学的，学习不觉得满足，教导别人不觉得疲倦，这三件事我能做到哪一件呢？

这话是谦虚，可也是真心话。孔子一辈子都在学，到老不辍。他晚年喜《易》，"读《易》，韦编三绝"——翻《易经》翻到穿竹简的牛皮绳子断了三回。——《史记·孔子世家》。一个六十多岁的老人，把编竹简的牛皮绳翻断三回，这是什么样的劲头？

孔子还说过："吾尝终日不食，终夜不寝，以思，无益，不如学也。"——《论语·卫灵公》。我曾经整天不吃饭、整夜不睡觉地去想，没用，不如学。这话透出孔子的一个底色：他不是天生的智者，他是靠学堆出来的。他也会苦思冥想，也会钻牛角尖，可他发现光想没用，得学。这份自我反省，正是"好学"的根。

孔子自称"好学"，不是自夸，是自况。他十五岁"志于学"——《论语·为政》，到七十岁"从心所欲不逾矩"——《论语·为政》，五十五年里，一直在学，一直在进。这份"好学"，是他跟别人最大的不同。忠信可以天生，好学全凭功夫。孔子把"好学"当成自己唯一的长项，等于告诉所有人：我没什么了不起，我只是比别人更肯学。

**颜回：唯一的"好学"之人**

颜回比孔子小三十岁，鲁国人，家里穷得叮当响，"一箪食，一瓢饮，在陋巷"——《论语·雍也》。颜回的穷，不是没本事，是不求富贵。可颜回让孔子最看重的，不是安贫乐道，是"好学"。

《史记·仲尼弟子列传》记颜回："回年二十九，发尽白，蚤死。孔子哭之恸，曰：'自吾有回，门人益亲。'"——自从有了颜回，弟子们更亲近了。这话透出一层意思：颜回不光自己好学，还能带动别人。一个好学的弟子，能把整个学风带起来。这是颜回的"好学"超出个人的地方。

颜回死后，鲁哀公问孔子："弟子孰为好学？"孔子答："有颜回者好学，不迁怒，不贰过。不幸短命死矣，今也则亡，未闻好学者也。"——《论语·雍也》。注意最后一句："未闻好学者也"——再没听说有谁好学了。孔子门下弟子三千，孔子说"好学"的，只颜回一个。这份评价，比任何赞美都重。

颜回的"好学"，不是读得多，是改得快。"不迁怒，不贰过"——这两条，是修身功夫，不是读书功夫。颜回把"学"落到了"行"上，这正是孔子"学"的真义。颜回死后，孔子哭得稀里哗啦："噫！天丧予！天丧予！"——《论语·先进》。一个老师为弟子哭成这样，不光是痛失爱徒，是痛失一个真正懂"学"、真正在"学"的人。

## 三、讲背景

要明白孔子为什么把"学"看得这么重，得回到春秋那时候，看看"学"到底指什么。

今天我们说"学"，多半指读书——上学、学知识、学技能。可春秋那会儿，"学"的含义宽得多。

那时候的"学"，包括三大块：学做人、学做事、学礼乐。学做人，是修身；学做事，是治国；学礼乐，是文化。这三块合在一起，才叫"学"。光读书，不算学。

周朝贵族教育，有"六艺"：礼、乐、射、御、书、数。——《周礼·地官·保氏》。礼是礼仪，乐是音乐，射是射箭，御是驾车，书是识字写字，数是算术。六艺里，没有一项是光靠书本就能学会的。射箭得拉弓，驾车得握缰，礼乐得演练，书数得操练。每一项都是"学"和"习"分不开——学了得练，练了才算会。

这就是"学而时习之"的"习"。"习"字的本义，是鸟儿学飞。《说文解字》："习，数飞也。"——鸟儿反复飞。小鸟学飞，不是看一遍就会，得一遍一遍扑腾翅膀，摔下来再飞，飞起来再摔，直到翅膀硬了，能自己上天。这个反复练的过程，就是"习"。

"时习"的"时"，也有讲究。朱熹说"时"是"时常"——经常。可还有一层意思：时，是适时、按时。不同的时节、不同的场合，习不同的东西。古人按节气安排学习，春诵夏弦，秋学礼，冬读书——各有各的时令。这跟农业社会的节律有关——春耕秋收，学习也得跟着天时走。

还有一层：时，是时机。学到的东西，要在合适的时机去用、去试。光会背，不会用，不算习。孔子说"温故而知新，可以为师矣"——《论语·为政》。温习旧的，能悟出新的，这才算把"习"做到家了。温故是习，知新是习出来的果实。

春秋那会儿，"学"还有一层特殊的意义。那时礼崩乐坏，旧秩序在塌，新秩序没立起来。贵族们忙着争霸，没人顾得上正经学问。典籍流散，礼乐荒废。孔子开私学，把贵族手里的学问拿出来教给平民，这本身就是一种"学"的复兴。孔子说"学而时习之"，是在一个学问断层的时代，重新把"学"立起来。

孔子还说过："生而知之者，上也；学而知之者，次也；困而学之，又其次也；困而不学，民斯为下矣。"——《论语·季氏》。生来就懂的，是上等；学了才懂的，是次一等；遇到困难才学的，又次一等；遇到困难还不学，那就是下等了。

孔子把自己放在哪一等？他说"我非生而知之者，好古，敏以求之者也"——《论语·述而》。我不是生来就懂的，是喜好古人之道，勤勉去求来的。孔子自认是"学而知之者"，不是"生而知之者"。这话跟"十室之邑"那句对看，就明白了——孔子最看重的，不是天赋，是后天的好学。

## 四、讲道理

"学而时习之"这五个字，两千多年来，注家不断。

朱熹《论语集注》开篇注"学"字，下了一个定义：

"学之为言效也。人性皆善，而觉有先后，后觉者必效先觉之所为，乃可以明善而复其初也。"

学，就是效法。人性都是善的，只是觉悟有先有后。后觉悟的人，必须效法先觉悟的人所做的，才能明白善、回到本初的善。

朱熹这个"效"字，下得极重。学不是凭空想，是效法——效法前人，效法先觉。前人怎么做的，我照着做；先觉怎么走的，我跟着走。这跟"习"的本义——鸟儿学飞——正好呼应。小鸟学飞，是效法大鸟；后觉学道，是效法先觉。学，从模仿开始；习，从反复练就。

朱熹还点出"明善而复其初"——学的目的是回到人本来的善。这话把"学"跟"性善"连起来了：人本性善，可被气质和习气蒙蔽了；学，就是擦掉蒙蔽，让善重新发亮。所以学不是从外面往里装东西，是从里面把本有的东西擦出来。这是理学家的根本看法。

钱穆在《论语新解》里分析"学"字，别有见地。钱穆指出，孔子时代的"学"，含义比今天宽得多——不只是读书，更包括做人和行事。学是"觉"，也是"效"；是知，也是行。钱穆强调，"学而时习之"的"习"，不是复习功课，是在生活中实践、演练。学了礼，得在言行里用出来；学了仁，得在待人接物上做出来。学和行分不开。

钱穆还点出一层：孔子说"学而时习之，不亦说乎"，这个"说"（悦）字是关键。学为什么悦？因为学有所得，得有所用，用有所成——这个从不懂到懂、从不会到会、从不能到能的过程，本身就是快乐的。这快乐不是外面给的，是从里面生出来的。

再说学与思的关系。孔子有一句最著名的话：

"学而不思则罔，思而不学则殆。"——《论语·为政》

只学不思，就迷惘；只思不学，就危险。

罔，是诬罔、迷惘——学了一堆，不消化，越学越糊涂。殆，是危险——光靠自己想，不吸收前人的学问，想出来的东西靠不住，容易走偏。

孔子这话，是把"学"和"思"放在天平的两端，谁也少不了谁。学是吸收，思是消化；学是材料，思是加工。光有材料没加工，材料堆成山也没用；光加工没材料，加工出来的东西是空的。这两条，缺一条都成不了事。

后来王阳明讲"知行合一"，跟孔子这套学问遥相呼应。王阳明在《传习录》里说："知是行的主意，行是知的功夫；知是行之始，行是知之成。"——知是行的方向，行是知的落实；知是行的开始，行是知的完成。知和行，是一件事的两面，不是两件事。

王阳明这话，可以拿来注"学而时习之"。学是知，习是行；学是始，习是成。学和习，本是一件事，分不开。孔子说"学而时习"，不是先学后习两步走，是学和习一体推进。学了就习，习中又学，循环不断。这正是"知行合一"的最早形态——只不过孔子没造这个词，王阳明造了。

朱熹的"效"、钱穆的"觉与行"、王阳明的"知行合一"，三家说法角度不同，根子一样：学不是光读书，学是知行合一的功夫。

## 五、问道悟道

"学而时习之"这五个字，往深了想，藏着三条经得起琢磨的规律。

**学是终身之事。**

孔子十五岁"志于学"，到七十岁"从心所欲不逾矩"，五十五年没停过。他晚年喜《易》，韦编三绝，到老还在学。孔子把"学"当成一辈子的事，不是阶段性任务。

这道理，古今中外皆然。庄子有句话："吾生也有涯，而知也无涯。"——《庄子·养生主》。生命有限，知识无限，以有限追无限，是终身的事业。苏格拉底七十岁了还说"我知道我一无所知"——越学越觉得自己不懂，越不懂越要学。这是真正好学之人的共同体会：学问越大，越知道自己差得远；越知道差得远，越不肯停。

宋代司马光编《资治通鉴》，编了十九年，到死才编完。明代李时珍写《本草纲目》，写了二十七年，三易其稿。这些人在别人眼里已经是学问大家，可他们自己觉得还差得远，非得学到死才算完。学问没有尽头，好学的人也没有止步的时候。

**学是知行合一。**

孔子说"学而时习之"，学和习是一体的。学了不习，等于没学；习了不学，习也走不远。颜回的"不迁怒，不贰过"，就是把学落到行上——学了不迁怒，就真的不迁怒；学了不贰过，就真的不贰过。学在行中见，行在学中长。

王阳明讲"知行合一"，是这个道理的延伸。知是行的主意，行是知的功夫。光知不行，不是真知；光行不知，是盲行。真知必能行，真行必有知。

这道理今天依然管用。学游泳，光看教材不下水，永远学不会；学管理，光读案例不上手，永远学不通。知识得在实践里检验，实践得在知识里升华。学和习，知和行，是同一件事的两面，分不开。孔子两千五百年前说的话，到今天一字不差地应验。

**学的目的是成人，不是知识本身。**

孔子说"学而时习之，不亦说乎"，这个"说"（悦）从哪里来？不是从知识多来，是从人变了来。学了，习了，人不一样了——眼界开了，心胸宽了，做事稳了，做人正了。这个"变"，才是学的真乐。

孔子还说过："古之学者为己，今之学者为人。"——《论语·宪问》。古时候的学者是为了充实自己，现在的学者是为了给别人看。为己，是修自己的人；为人，是装点门面。孔子推崇的是"为己之学"——学是为了成人，不是为了炫耀。

这跟今天的风气正好相反。今天的人学东西，多半为了考试、为了文凭、为了升职、为了赚钱。学是手段，不是目的。可在孔子看来，学本身就是目的——学是为了让自己成为一个更完整的人。知识是工具，成人才是目标。把工具当目标，是本末倒置。

苏格拉底也讲过类似的话。他说未经省察的人生不值得过——学问的目的不是积累知识，是省察人生，让自己活明白。东西方的智者，在这点上想到一块去了：学，归根到底，是为了成人。

---

## 结语

学不在多，在习；习不在久，在悦——悦的是人变了，不是书读了。
