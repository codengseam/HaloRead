# 孔子的终极关怀

## 一、讲事情

孔子这辈子，操心的不是一时一事，是一件事能不能传下去。这件事，他叫"道"。

《论语·述而》里，孔子说过十二个字，把自己一辈子的功夫全说尽了——"志于道，据于德，依于仁，游于艺。"

志，是心之所向；道，是人伦日用间当行的路。心先有个方向，才不会走岔。据，是执守；德，是行道有得于心。心里有了得，才能守得住。依，是不违；仁，是心德的全体。功夫到了，无时无刻不是仁。游，是玩物适情；艺，是礼乐射御书数。这些不是末节，是道所寄寓的地方，日用不可缺。这四句像四层台阶——立志、守德、依仁、游艺，从心之所向到日用涵泳，一层比一层深。孔子一辈子就在这四层上走。这不是空话，是他每天的日子。

再说孔子言志那一段。那天孔子坐着，颜渊、子路在旁边侍候。孔子来了兴致："盍各言尔志？"——何不各人说说自己的志向？子路性子急，先开口："愿车马衣轻裘与朋友共敝之而无憾。"——车马衣裘跟朋友共享，用坏了也不遗憾。颜渊接着说："愿无伐善，无施劳。"——不夸耀自己的好处，不把劳苦加给别人。子路反问："愿闻子之志。"——想听听老师的志向。孔子说了三句话——"老者安之，朋友信之，少者怀之。"（《论语·公冶长》）

子路的志向主角是"我"，颜渊的志向主角还是"我"，孔子的志向里没有"我"，只有"他"——老者、朋友、少者。老者要安，朋友要信，少者要怀。三代人都照顾到了，一个社会最要紧的事全在这十二个字里。这不是治国之术，是治人之心。

再说"吾道一以贯之"。孔子有一天忽然喊曾子："参乎！吾道一以贯之。"——曾参啊，我的道是一以贯之的。曾子答："唯。"——是。孔子出去后，门人问曾子："何谓也？"——什么意思？曾子说："夫子之道，忠恕而已矣。"——老师的道，就是忠恕两个字罢了。（《论语·里仁》）

这段对话短得惊人。孔子只说了一句"吾道一以贯之"，没解释；曾子只答了一个"唯"，没追问。师徒二人心里头那点东西，一个眼神就通了。可门人听不懂，曾子才替他们翻译——忠恕。忠是尽己，恕是推己及人。孔子一辈子讲的仁、讲的礼、讲的为政、讲的修身，归根到底就是这两件事——对自己尽到心，对别人推己及人。这是孔子的"一"，是他终极关怀的根。

再说孔子临终。鲁哀公十六年，公元前479年，孔子病重。一天早晨，他拄着杖在门口徘徊，唱了一首歌——"太山其颓乎？梁木其坏乎？哲人其萎乎？"——泰山要倒了吧？梁木要坏了吧？哲人要凋零了吧？（《礼记·檀弓上》）

子贡听见，赶紧往里走。孔子说："赐，尔来何迟也？"——赐啊，你怎么来得这么晚？接着说了一段话：夏人殡于东阶，周人殡于西阶，殷人殡于两柱之间。我昨晚梦见自己坐在两柱之间受人祭奠——我是殷人之后啊。孔子七天后去世。（《礼记·檀弓上》）

泰山倒了，梁木坏了，哲人凋零了——孔子自己知道，他这一走，是一个时代的结束。可他临终惦记的不是死，是"道"传没传下去。

再说曾子临终。曾子病重，把弟子叫到跟前，说："启予足！启予手！《诗》云：'战战兢兢，如临深渊，如履薄冰。'而今而后，吾知免夫！小子！"——看看我的脚！看看我的手！《诗经》说"战战兢兢，如临深渊，如履薄冰"，从今以后，我知道可以免了！小子们！（《论语·泰伯》）

曾子一辈子修身，到死才敢说一个"免"字。身体发肤受之父母，不敢毁伤——他到死保全了身子，也保全了德。这一句"吾知免夫"，是曾子对自己一生的交代。孔子传的道，到曾子这儿接住了。

## 二、讲人物

孔子的终极关怀，不是某一天突然有的，是他一辈子活出来的方向。

他年轻时立定志向——"吾十有五而志于学"（《为政》）。这个"学"，不是读书识字，是求道。所以他后来说"朝闻道，夕死可矣"（《里仁》）——早上听到道，晚上死也值了。这话听着极端，可细想，孔子是在说：道比命重。命有尽，道无穷。能闻道，这一辈子就没白活。

可孔子不是只顾自己闻道。他说过"士志于道，而耻恶衣恶食者，未足与议也"（《里仁》）——士人立志于道，却以穿破衣吃粗饭为耻，这种人就不值得跟他谈论了。这话有两层。一层是：志于道的人，不该在乎衣食。另一层更深：志于道，不是一个人的事，是要跟同道中人议的。道要传，要有人一起讲、一起行。孔子一辈子收弟子三千，贤者七十二，就是在找能跟他"议道"的人。

孔子心里头明白，道不能靠他一个人扛。他得有传人。颜回是第一个。颜回"不迁怒，不贰过"，"其心三月不违仁"（《雍也》），孔子把"克己复礼"的功夫传给他。可颜回早死，孔子痛呼"天丧予"（《先进》）——丧的不只是颜回，是道统。颜回一走，孔子把目光转向曾子。

曾子比孔子小四十六岁，在孔门里不算最聪明的。孔子说"参也鲁"（《先进》）——曾参迟钝。可曾子有一样别人比不了——踏实。他"吾日三省吾身"（《学而》）：为人谋而不忠乎？与朋友交而不信乎？传不习乎？每天反省三件事：替人谋事尽心了吗？跟朋友交往守信了吗？老师传的道温习了吗？这种日日不断的功夫，正是传道需要的根。

孔子看人看得准。聪明的弟子不少，子贡善辩、子路勇武、宰予利口，可传道不需要聪明，需要的是"守"。曾子能守。所以孔子临终前把"吾道一以贯之"传给曾子——不是传给子贡，不是传给子路，是传给那个"鲁"的曾参。这一传，传出了后世两千年的儒学道统。曾子传子思，子思传孟子，一线相承。

曾子也没辜负孔子。他一辈子"战战兢兢，如临深渊，如履薄冰"（《泰伯》），到死才敢说"吾知免夫"。这不是怕，是敬——对道的敬，对身的敬，对父母所授、孔子所传的敬。孔子临终叹"哲人其萎"，曾子临终说"吾知免夫"——一个忧道之传，一个全道之守。师徒二人，一个传，一个接，把"道"这根接力棒交下去了。

## 三、讲背景

孔子的终极关怀，得放在春秋末期的大背景里看。

那时候，周天子名存实亡，诸侯争霸，大夫专权，家臣作乱。鲁国被三桓瓜分，季氏的家臣阳虎一度执国命——"陪臣执国命"（《季氏》）。天下大乱，礼崩乐坏。更要命的是，"学在官府"的传统瓦解了。西周以来，典籍、礼乐、学问都掌握在王室和贵族手里。平王东迁后，王室衰微，典籍散佚，官学解体。老子是周守藏室之史，可连他都管不住那些典籍的流失。

这就是孔子面临的危机——不只是政治上的乱，是文化上的断。礼乐没人懂了，典籍没人读了，三代以来的文明火种眼看要灭。孔子急。他说"文王既没，文不在兹乎"（《子罕》）——文王之后，文化不就在我这儿吗？这话不是自大，是焦虑。天要丧斯文，后死者就接不上；天不丧斯文，匡人能拿我怎么样？孔子把自己当成了文化的承载者——这份自觉，就是他终极关怀的根。

所以孔子干了几件事。第一，开私学。"有教无类"（《卫灵公》），谁都能来学。这打破了贵族对知识的垄断，让学问从官府走向民间。第二，整理典籍。删《诗》《书》，定礼乐，赞《周易》，作《春秋》。《诗经》三千多篇删成三百零五篇；晚年喜《易》"韦编三绝"（《史记·孔子世家》）。第三，周游列国。不是为做官，是为行道——"天下有道，丘不与易也"（《微子》），天下有道我就不出来改变了，正因为无道，我才要出来。

这三件事，指向同一个目标——保存文化火种。政治上的乱，是一时的；文化上的断，是万世的。孔子看得很清楚：国可以亡，朝可以换，可文化断了，这个文明就真没了。所以他的终极关怀，不是救一国一朝，是救一个文明。这超越了个人生死——他知道自己活不了多久，可道要传下去，典籍要留下来，弟子要散到各国去。他临终叹"哲人其萎"，可他心里头惦记的是"文不在兹乎"——文化还在不在？

这就是"终极"二字的分量。终极，不是最后，是最根本。孔子的终极关怀，是超越了个人利害、超越了生死、甚至超越了朝代更替的那件事——道能不能传，人能不能成全，文化能不能延续。

## 四、讲道理

孔子的终极关怀，可以从三个维度讲。

**道的传承。** 孔子一辈子"志于道"，可这道不是他发明的，是他从三代继承下来的。他说自己"述而不作，信而好古"（《述而》）——只传述不创作，信古好古。可他不是简单照搬，他把三代之道提炼成"吾道一以贯之"——忠恕。朱熹《论语集注》在"吾道一以贯之"下注："圣人之心，浑然一理，而泛应曲当，用各不同。曾子于其用处，盖已随事精察而力行之，但未知其体之一尔。"——圣人的心，浑然是一个理，可应对不同的事有不同的表现。曾子在用处已经随事精察力行，可还不知道体的"一"。朱熹点出"体一"二字——孔子之道，体是一，用是多。忠恕是体，仁礼是用。孔子传给曾子的，不是某一条具体的规矩，是那个"一"。

**人的成全。** 孔子言志"老者安之，朋友信之，少者怀之"，心里装的是天下人。治国的目的，不是国家的强大，是人的安顿。钱穆《论语新解》论此章，说孔子之志"非为一己"，是"以天下为心"。钱穆又论"志于道据于德依于仁游于艺"，认为这是孔门为学之全功——道是方向，德是得于心，仁是依凭，艺是涵泳，四者层层递进，归结到人的成全。孔子的终极关怀，不是抽象的道，是活生生的人——老者能安，朋友能信，少者能怀，这才是道在人间的样子。

**文化的延续。** 孔子整理典籍、开私学、传弟子，干的都是文化延续的事。司马迁在《史记·孔子世家》末尾写了一段有名的赞语："《诗》有之：'高山仰止，景行行止。'虽不能至，然心乡往之。余读孔氏书，想见其为人。适鲁，观仲尼庙堂车服礼器，诸生以时习礼其家，余祗回留之不能去云。天下君王至于贤人众矣，当时则荣，没则已焉。孔子布衣，传十余世，学者宗之。自天子王侯，中国言《六艺》者折中于夫子，可谓至圣矣！"

司马迁这段话，点破了孔子终极关怀的价值。天下的君王贤人多了去了，活着的时候荣耀，死了就完了。可孔子一个布衣，传了十几世，学者都宗奉他。帝王将相换了无数茬，孔子的道没换——"自天子王侯，中国言《六艺》者折中于夫子"。这就是文化延续的力量。政治靠权，权会丢；文化靠人，人会传。孔子看准了这一点，所以他把终极关怀放在文化上，不是政治上。

朱熹《论语集注》在"朝闻道夕死可矣"下注："道者，事物当然之理。苟得闻之，则生顺死安，无复遗恨矣。"——道是事物当然之理。能闻道，活着顺，死了安，没有遗恨了。朱熹这个注，把孔子的终极关怀说到了底——闻道，则生死皆安。这不是说闻道就可以死，是说闻了道，死也没什么遗憾了。因为道已经传下去了，人已经成全了，文化已经延续了。

## 五、问道悟道

从孔子的终极关怀，能琢磨出几条让人开眼的规律。

**人的伟大在于超越自我。**

孔子言志，志里没有"我"，只有"他"——老者、朋友、少者。曾子临终，惦记的是"传不习乎"——老师传的道温习了吗。这师徒二人，心里装的不是自己。人活着，能超越自我，才谈得上伟大。这话听着玄，其实有根。一个人只为自己活，他的世界就那么大；一个人为天下人活，他的世界就无限大。孔子活着的时候，是个失败的政客、一个四处碰壁的老头；可他死了两千多年，还被人记得——因为他活过了自己。

古今中外，伟大的人都这样。苏格拉底被雅典判了死刑，朋友安排好了逃路，他不走，说逃了就违背了自己一辈子讲的道理。他喝下毒堇汁死了，可他的思想通过柏拉图传了两千多年。苏格拉底的终极关怀不是自己的命，是真理的传。释迦牟尼本可以做王子，享尽荣华，他抛下王宫去求解脱之道——他的终极关怀不是王位，是众生的苦。耶稣被钉在十字架上，他本可以呼召天使来救，可他没有——他的终极关怀不是自己的死，是人的救赎。这四个人——孔子、苏格拉底、释迦牟尼、耶稣——活法不同，可有一点相同：他们都活过了自己。人的伟大，不在拥有多少，在超越多少。

**文化的传承是最大的不朽。**

孔子临终叹"哲人其萎"，可他心里头惦记的是"文不在兹乎"。他知道自己会死，可文化不能断。所以他删《诗》《书》、定礼乐、传弟子——把文化的火种散出去。司马迁说"孔子布衣，传十余世"——一个布衣，传了十几世。帝王将相的功业，多则几百年，少则几十年；可孔子的道，传了两千多年。为什么？因为政治靠权，权会丢；文化靠人，人会传。

这事古今皆然。秦始皇统一六国，可秦朝只传了二世；可秦始皇统一文字、统一度量衡，这个文化的影响传了两千年。汉武帝打匈奴、通西域，功业赫赫，可汉朝四百年就亡了；可汉武帝独尊儒术，这个文化的格局影响了中国两千年。司马迁受宫刑，写《史记》，他自己说"藏之名山，传之其人"——《报任安书》。司马迁的终极关怀不是自己的屈辱，是历史的传。《史记》传了两千多年，司马迁也活了两千多年。文化传承的力量，比权力长久得多。这是最大的不朽——不是肉体的长生，是精神的延续。

**终极关怀决定人生格局。**

孔子一辈子颠沛流离，困于陈蔡，被人笑成"丧家之狗"。可他没垮，因为他心里有个"道"。这个"道"，就是他的终极关怀。一个人有没有终极关怀，决定了他能扛多大的事。没有终极关怀的人，遇到点挫折就垮——因为没有比挫折更大的东西撑着他。有终极关怀的人，再大的挫折也扛得住——因为他心里有比挫折更大的东西。

颜回穷得"一箪食一瓢饮"，可他"不改其乐"（《雍也》），因为他有"道"。子路结缨而死，临死还系帽缨，因为他有"义"。曾子"战战兢兢"一辈子，到死才说"吾知免夫"，因为他有"传"。这师徒几人，活法不同，可都有个比命更重的东西。这个东西，就是终极关怀。它让人活得有方向，死得有交代。孔子说"朝闻道，夕死可矣"——有了道，死也值了。这话不是悲壮，是坦然——一个人找到了比命更重的东西，命就轻了，死就安了。

---

## 结语

孔子的终极关怀，不在功名利禄，在道传不传、人全不全、文续不续——他活过了自己，所以活成了不朽。
