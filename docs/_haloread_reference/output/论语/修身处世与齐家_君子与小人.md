# 君子与小人——儒家的人格理想

## 一、讲事情

孔子讲了一辈子"君子"，也讲了一辈子"小人"。这两个词，在他嘴里不是骂人的话，是两面镜子——一面照人该往哪儿走，一面照人别往哪儿去。

最有画面感的一次，是孔子跟子路说"坦荡荡"。

那天师徒几个闲坐，子路刚从外头进来，风风火火，脸上还带着没褪干净的急躁。子路这个人，性子直，心里装不住事，高兴不高兴全写在脸上。孔子看了他一眼，慢悠悠说了一句——

"君子坦荡荡，小人长戚戚。"——《论语·述而》

君子心胸平坦宽广，小人老是愁眉苦脸。

这事不是凭空说的。子路本事大，胆子更大，可就是容易患得患失。一遇到事，心里七上八下，脸上挂不住。孔子这话是点他——你这样下去，离君子还远。君子不是没烦恼，是心里头有根定得住的东西，烦恼来了过得去；小人不一样，心里没根，一点风吹草动就翻江倒海。

再说"喻于义"那一段。

《论语·里仁》记孔子的话："君子喻于义，小人喻于利。"——君子明白的是义，小人明白的是利。

这话听着简单，分量却重。"喻"是明白、通晓的意思。君子一开口一动手，心里盘算的是"这事该不该做"——义；小人一开口一动手，心里盘算的是"这事对我有什么好处"——利。不是说君子不要利，是小人眼里只有利，看不见义。孔子那个年代，诸侯争霸，大夫擅权，人人都在算利。整个春秋，就是一部"喻于利"的乱世史。孔子看在眼里，才把"义"和"利"摆在人面前——你选哪个，你就是哪种人。

还有一段，是论"和"与"同"。

《论语·子路》记："君子和而不同，小人同而不和。"——君子与人和谐，却不盲从；小人盲从，却不和谐。

这话精妙。"和"是各有各的主张，却能相处融洽；"同"是表面上附和，心里头各怀鬼胎。君子跟你处得好，不代表他什么都听你的——该争的争，该反对的反对，争完了还是朋友。小人正好反过来，当面点头哈腰，背后捅刀子，看着一团和气，其实一盘散沙。晏子曾用做羹汤打比方——要有水有酱有盐有梅，各种味道调和才好喝；全是水或全是盐，谁喝得下去？《左传·昭公二十年》记的这段话，跟孔子"和而不同"是一个意思。可见那个年代，有识之士都在想人和人怎么相处，才能既不丢自己，又不伤别人。

最见孔子骨气的，是陈成子弑简公那一次。

齐国大夫陈恒（即陈成子）把齐简公杀了。这事传到鲁国，孔子做了件让所有人都没想到的事——《论语·宪问》记："孔子沐浴而朝，告于哀公曰：'陈恒弑其君，请讨之。'"

沐浴，是斋戒沐浴，表示郑重。孔子不是随随便便上朝，是洗了澡换了衣服，以最庄重的礼节去见鲁哀公，说：陈恒杀了他的国君，请出兵讨伐。

这事听着就离谱。鲁国那时候弱得不行，齐国是强国，鲁国去讨伐齐国，不是找死吗？

鲁哀公也这么想，推托说："告夫三子。"——你去跟三家说吧。三家是鲁国的实际掌权者，哀公说了不算。

孔子出来，叹了一句："以吾从大夫之后，不敢不告也。"——我曾在朝廷做过大夫，这事不能不报。明知道说了没用，还是得说。因为我是大夫，大夫见国君有难不说话，还算什么大夫？

他真的去了三家。三家果然不同意。

孔子又说了一句一模一样的话："以吾从大夫之后，不敢不告也。"——我是做过大夫的，不能不报。

两次碰壁，两次说同一句话。这话里头有无奈，更有硬气。孔子知道讨伐齐国不可能成功，可他还是要做。因为"君子喻于义"——这事该做，做了再说成不成；不是"喻于利"——先算划不划算再决定做不做。整个鲁国没人敢说话，就他一个老头子沐浴而朝，明知不可为而为之。这就是君子。

## 二、讲人物

讲君子小人，绕不开两个人——定标准的孔子，和往标准上靠的子路。

**孔子：君子是修养来的，不是天生的**

孔子自己就是君子的活样本。可他这个样本，不是天生就成的。

《论语·为政》记孔子自述："吾十有五而志于学，三十而立，四十而不惑，五十而知天命，六十而耳顺，七十而从心所欲不逾矩。"

十五岁立志求学，三十岁站得住，四十岁不再迷惑，五十岁懂得天命，六十岁什么话都听得进去，七十岁随心所欲却不越规矩。这一条路，走了五十五年。君子不是一天长成的，是一辈子磨出来的。

孔子最难得的，是他不把自己当圣人。他说"若圣与仁，则吾岂敢"——《论语·述而》。他只承认自己"学而不厌，诲人不倦"。君子在他嘴里，不是个高不可攀的帽子，是不停地做、不停地教的功夫。

这就跟"小人"划开了界限。小人不是坏人，是停下来的人。小人也曾经有过好心，可他守不住——遇到利，义就丢了；遇到难，节就垮了。君子和小人的差别，不在起点，在能不能一直走下去。

孔子论君子，处处讲"修养"二字。"君子求诸己，小人求诸人"——《论语·卫灵公》。出了事，君子先问自己哪里没做好；小人先怪别人、怪环境、怪运气。这一条，是君子修养的根子——把责任放回自己身上。"君子不可小知而可大受也，小人不可大受而可小知也"——《论语·卫灵公》。君子不一定在小事上聪明，可大事交给他，他担得住；小人不能担大事，可在小事上精明得很。看人不能看小处，得看大处。

**子路：从粗人到君子的成长**

子路是孔子门下最有意思的一个。他不是天生的君子，他是被孔子硬生生磨出来的。

《史记·仲尼弟子列传》记子路："性鄙，好勇力，志伉直，冠雄鸡，佩猳豚，陵暴孔子。"——性子粗鄙，好勇斗狠，志向刚直，头上插着雄鸡毛，腰里挂着野猪牙，还欺负过孔子。

这是子路的本来面目——一个街头混混，一个莽汉。可孔子看上他了。看上他什么？看上他的"直"。子路粗是粗，可不装；莽是莽，可不阴。这种人，调教好了，是块好料。

孔子调教子路，费了不少功夫。《论语·公冶长》记："子曰：'道不行，乘桴浮于海。从我者其由与？'子路闻之喜。"——孔子说：我的道行不通了，我坐木筏出海去，跟着我的大概是子路吧？子路一听，高兴坏了。孔子接着一句："由也好勇过我，无所取材。"——子路比我还好勇，没别的可取。一盆冷水泼下来。

这就是孔子教子路的法子——给一句甜的，再给一句辣的。子路好勇，这是他的长处，也是他的短处。勇过了头，就是莽；莽过了头，就要出事。

子路慢慢在改。《论语·子路》记子路问政，孔子答"先之，劳之"——自己带头，然后让百姓干活。子路请多说一点，孔子答"无倦"——不要倦怠。子路这个人，一旦认了理，就死磕到底。

可子路的底色，还是那个"勇"字。他最后死在卫国作乱里。《史记·仲尼弟子列传》记，卫国发生内乱，子路当时在卫国做官，赶去救主。帽子被打歪了，他说"君子死，冠不免"——君子就算死，帽子也要戴正。系帽带的时候，被人砍成了肉酱。

这话听着迂腐，可这是子路一辈子修行的结果。那个当年头上插鸡毛、腰里挂猪牙的莽汉，临死前惦记的是"君子"二字——帽子歪了要正过来。子路没修到孔子说的"不逾矩"，可他修到了"死，冠不免"。从粗人到君子，他走了一辈子，走到底了。

孔子听说卫国作乱，叹了一句："嗟乎，由死矣。"——唉，子路死了。他太了解子路——遇到事不会躲，一定会去，去了就一定会死。君子不是不会死，是死得其所。

## 三、讲背景

"君子"这个词，不是孔子发明的。可在孔子之前，"君子"的意思跟孔子讲的，完全不是一回事。

"君子"两个字，拆开看——"君"是统治者，"子"是子弟。"君子"本来的意思，是"国君之子"，也就是贵族子弟。《诗经》里"君子"出现了几十次，大多指有位的人。那时候的君子，是血统概念，是身份概念，不是道德概念。《尚书·无逸》里周公讲"君子"，指的也是在位的贵族。要当君子，得先有个好爹——生在公卿之家，就是君子；生在田野之间，就是小人。君子和小人，是阶级的划分，不是道德的评判。

这个划分，跟周代的封建制度是一体的。周天子分封诸侯，诸侯分封大夫，大夫下面是士，士下面是庶人。每一层都有对应的礼，全按血统定。君子就是这金字塔上头的人，小人就是底下的人。

可到了春秋，这个金字塔塌了。礼崩乐坏，不只是礼的形式崩了，是整个等级秩序崩了。诸侯不把天子放在眼里，大夫不把诸侯放在眼里，陪臣不把大夫放在眼里。血统定不了高低了，权力才定得了。

这时候就出了个新阶层——士。

士本来是大夫下面的一层，有贵族身份，可没多少封地。春秋后期，诸侯大夫争着养士，士成了香饽饽。孔子就是士出身——"吾少也贱，故多能鄙事"——《论语·子罕》。士这个阶层有个特点——身份不高不低，可流动性大。往上可以做卿大夫，往下可以混成庶人。靠什么？靠本事，靠学问，靠德行。血统靠不住了，只能靠自己。

孔子就是在这个节骨眼上，把"君子"这个词翻了个底朝天。

他说："君子"不是血统给的，是修养来的。你生在贵族之家，要是无德无能，那是"小人儒"；你生在陋巷穷户，要是好学修德，那是"君子儒"。"女为君子儒，无为小人儒"——《论语·雍也》——你要做君子式的儒，别做小人式的儒。

这一翻，翻天覆地。

从前君子是身份，现在君子是境界；从前君子靠爹，现在君子靠自己；从前君子是少数人的专利，现在君子是人人可以追求的理想。"君子之德风，小人之德草"——《论语·颜渊》。孔子这话里，"君子"还是指在位者，"小人"还是指百姓——可他紧接着说"草上之风必偃"，风吹过草，草一定倒。可见孔子不是不要"位"，他是要"位"和"德"统一起来——有位的人，更要有德。

可更革命性的是另一面——没有位的人，也可以有德。颜回一箪食一瓢饮，在陋巷，是穷光蛋；可孔子说他"三月不违仁"，是弟子里的君子第一人。子路粗鄙好勇，是莽汉出身；可他临死"冠不免"，修到了君子的边。

这是中国思想史上一次了不起的解放。从前人的命运由血统决定，现在人的命运由自己决定。你做君子还是做小人，不看你爹是谁，看你自己的选择。孔子把"君子"从血统里解放出来，给了所有人一条往上走的路。这条路不好走，可它开着。这就是春秋那个乱世里，孔子给普通人留下的一道光。

## 四、讲道理

君子和小人的根本区别在哪里？两千年来，注家们各有说法。挑几家最有分量的来看。

**朱熹：君子循理，小人役于物**

朱熹《论语集注》在"君子坦荡荡小人长戚戚"下注："坦，平也。荡荡，宽广貌。戚戚，多愁貌。君子循理，故常舒泰；小人役于物，故多忧戚。"

这话分两层。君子"循理"——顺着天理走，心里有根，所以平坦宽广；小人"役于物"——被外物牵着鼻子走，患得患失，所以愁眉苦脸。君子心里有个"理"做主，外头的事再变，心里不乱；小人心里没有"理"，全靠外头的东西撑着——有钱就高兴，没钱就发愁；有权就神气，没权就丧气。外头的东西一变，小人就跟着变，所以"长戚戚"。

朱熹又在"君子喻于义小人喻于利"下注："喻，犹晓也。义者，天理之所宜。利者，人情之所欲。"——喻是通晓。义是天理该做的，利是人情想要的。义是"天理之所宜"——天理觉得该做的，不管有没有好处；利是"人情之所欲"——人情想要的，不管该不该做。君子心里通晓的是天理，小人心里通晓的是欲望。路子根本不同。

朱熹又在"君子和而不同"下注："和者，无乖戾之心。同者，有阿比之意。"——和是没有违逆之心，同是有阿谀附和之意。"和"是心里不跟人别扭，能容人，能处人；"同"是嘴上附和，心里头算计。君子跟人相处，求的是"和"——各有各的立场，能合作就合作，不能合作也不翻脸；小人跟人相处，求的是"同"——表面上跟你一样，背地里为自己打算。

**钱穆：君子是一种人格理想**

钱穆《论语新解》论君子，最着力的一点，是讲清楚"君子"在孔子这里已经从"位"转成了"德"。钱穆指出，孔子所谓君子，不是指在位的人，是指有德的人。这一判断，是理解《论语》里所有"君子"的钥匙。

论"君子坦荡荡小人长戚戚"，钱穆讲君子心地光明，所以坦荡荡；小人患得患失，所以长戚戚。这比朱熹更直白——君子心里亮堂，没什么好藏的；小人心里头算计，得失一字压在心头，所以放不开。

论"君子固穷小人穷斯滥矣"，钱穆讲得尤其动情。这话出自《论语·卫灵公》——孔子在陈国断粮，子路饿得发脾气，问"君子亦有穷乎？"孔子答："君子固穷，小人穷斯滥矣。"——君子穷也守得住，小人一穷就乱了。钱穆说，君子虽穷，能守其道；小人穷，则无所不为。顺境里看不出来，一穷，原形毕露。

钱穆还特别点出"君子求诸己小人求诸人"这一条——君子凡事反求诸己，小人凡事责求诸人。出问题先找自己的原因，不先怪别人。钱穆说这是君子小人分途之大端——君子和小人分岔的根本路口。

两家对照，朱熹从"理"上讲，钱穆从"人"上讲。可两家都认一个核心——君子和小人的差别，不在外头，在心里头。

## 五、问道悟道

君子和小人这两个词，往深里想，藏着几条经得起琢磨的规律。

**人格是可以选择的。**

孔子把"君子"从血统里解放出来，最大的意义是——做人这件事，成了选择。

从前你是什么人，由你爹决定；现在你是什么人，由你自己决定。生在贵族之家，可以做成小人；生在陋巷穷户，可以修成君子。颜回是穷光蛋，可他是君子第一人；季氏是鲁国权臣，可孔子骂他"八佾舞于庭，是可忍也，孰不可忍也"——《论语·八佾》。血统给不了你君子，权力也给不了你君子。君子是自己修来的。

这事放到今天依然成立。一个人是什么样的人，不看他什么出身，看他怎么选择。苏格拉底是石匠的儿子，成了哲学的化身；林肯是穷木匠的孩子，成了伟大的总统。出身定起点，选择定终点。

**道德修养是终身之事。**

孔子自述"十五志于学，七十从心所欲不逾矩"，走了五十五年。颜回"三月不违仁"，已经是天花板。子路修了一辈子，临死才修到"冠不免"。君子不是一次成就，是一辈子的事。

"君子固穷"那一段最见功夫。顺境里谁都能当君子，一穷就露馅。孔子在陈国断粮，"从者病，莫能兴"——《论语·卫灵公》。跟的人都饿爬下了，子路都发脾气了。孔子还在那里讲诵弦歌不衰——照样讲学、照样弹琴。修养不到，饿三天就垮了；修养到了，饿死也守得住。

这跟任何一种修行都相通。佛家讲"念念相续"，一念觉悟不算数；基督教讲"忍耐到底，必然得救"。道德修养没有终点，只有过程。曾子说"死而后已"——《论语·泰伯》，到死才放下，就是这个意思。

**君子是一种境界，不是一种身份。**

这是孔子最革命性的洞察。从前君子是身份——有位就是君子，无位就是小人。孔子把这条界线抹了——有位无德，不算君子；无位有德，也是君子。君子从"你是什么"变成了"你修到什么程度"。

这一变，意义深远。它意味着道德不是阶级的附属品，是人人可以追求的境界；意味着人的价值不由外在地位决定，由内在修养决定；意味着哪怕你一无所有，只要心里有"义"，你也是君子。

这事古今中外都相通。斯多葛学派的哲人爱比克泰德是奴隶出身，可他是罗马帝国最受尊敬的道德教师——这跟孔子"君子求诸己"是一个路子。境界不分阶级，不分时代。君子这个理想，孔子提出来两千五百年了，到今天还是做人的标杆。因为它讲的不是某个时代的规矩，是人之为人的根本——你能不能守住义，能不能坦荡荡，能不能在穷的时候不滥。

---

## 结语

君子不是一种身份，是一种选择——选义还是选利，选坦荡还是选戚戚，选求己还是选求人，全在你自己。
