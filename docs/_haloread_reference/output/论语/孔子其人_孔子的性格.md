# 孔子的性格——既是圣人，也是性情中人

## 一、讲事情

孔子这个人，后人把他供上神坛，可你真去读《论语》，会发现他根本不是庙里那尊泥塑。他会哭，会笑，会发脾气，会自嘲，会想家——是个有血有肉的老头。

先说颜回死那一段。颜回是孔子最得意的弟子，比孔子小三十岁，安贫乐道，一箪食一瓢饮，在陋巷，人不堪其忧，回也不改其乐。孔子把他当传人，当儿子，当知音。鲁哀公十四年，颜回英年早逝。孔子哭之恸——哭得撕心裂肺。跟从的人劝："子恸矣！"——老师哭得太过了。孔子擦着眼泪说："有恸乎？非夫人之为恸而谁为？"——我哭得过分了吗？我不为这个人哭得过分，还为谁哭得过分？——《论语·先进》。说着又喊："噫！天丧予！天丧予！"——老天这是要我的命啊！老天这是要我的命啊！——《论语·先进》。

这一声"天丧予"，不是一个道德家在作秀，是一个老人在送自己的精神骨肉。颜回一死，孔子觉得自己的道没人传了。

再说子路。子路比孔子小九岁，性子直，脾气暴，跟孔子一辈子没大没小。孔子周游列国到了陈，听说卫国发生政变。孔子叹气说："由也死矣！"——仲由怕是死了吧？——《史记·仲尼弟子列传》。没过多久，卫国果然来报：子路在卫国蒯聩之乱里遇难，被人剁成了肉酱。孔子正在吃饭，一听这消息，把嘴里的肉酱全吐了，叫人把家里的肉酱都倒掉，盖着眼泪叹："噫！天祝予！"——老天断了我的路啊！"祝"就是"断"的意思。子路这一死，孔子又少了一条胳膊。

这两段放一起看，就知道孔子不是冷面道学。他爱弟子，爱得深，也爱得痛。

可孔子也有另一面。他到了齐国，听了韶乐——传说是舜时的音乐。孔子一听，整个人就痴了。三个月吃饭，不知肉味——嘴里嚼着肉，尝不出肉味来。孔子感叹："不图为乐之至于斯也！"——没想到音乐能达到这样的境界！——《论语·述而》。一个能为一首曲子痴三个月的人，是个懂得美的人。孔子讲仁讲礼，可他没说仁就是苦，礼就是板着脸。他心里有美，有乐，有那种被艺术击中的颤动。

再说一件趣事。孔子的弟子言偃，字子游，做了武城宰——武城的长官。子游治武城，用礼乐教化百姓，到处弦歌不辍。孔子到了武城，听见满城弹琴唱歌，扑哧一笑，说："割鸡焉用牛刀？"——杀鸡哪用得着宰牛的刀？——《论语·阳货》。这话是打趣子游——治个小地方，用得着这么大的阵仗？子游不卑不亢，回了一句："昔者偃也闻诸夫子曰：'君子学道则爱人，小人学道则易使也。'"——老师从前教过我，君子学了道就爱人，小人学了道就好使唤。孔子一听，回头对跟来的弟子说："二三子！偃之言是也。前言戏之耳。"——同学们，言偃说得对，我刚才跟他开玩笑呢。——《论语·阳货》。

这一段太妙了。孔子打趣弟子，弟子拿孔子自己的话顶回来，孔子当场认账，还跟其他弟子解释"我是开玩笑"。这样的老师，哪里是后世那种一言九鼎、说一不二的"圣人"？他肯认错，肯服理，肯在学生面前放下架子。

还有一段思乡。孔子在陈国待了三年，一直没机会行道。有一天他叹气："归与！归与！吾党之小子狂简，斐然成章，不知所以裁之。"——回去吧！回去吧！我家乡那帮小子志大才疏，文采倒是斐然，就是不知道怎么裁剪。——《论语·公冶长》。孔子想家了。他想的不只是鲁国的乡土，更是家里那帮年轻弟子——他们有才气，可没人调教，长歪了怎么办？这一声"归与"，是一个老师对学生的牵挂，也是一个老人对故土的思念。

## 二、讲人物

孔子这个人，难就难在不是一个面。

他是严肃的道德家。宰予白天睡觉，孔子骂："朽木不可雕也，粪土之墙不可杇也。"——《论语·公冶长》。原壤老而无礼，叉开腿坐在地上等孔子，孔子拿起拐杖敲他的小腿，骂："老而不死是为贼。"——《论语·宪问》。该骂的时候，孔子骂得比谁都狠。

可他也是爱开玩笑的老人。"割鸡焉用牛刀"是打趣，"前言戏之耳"是认账。孔子还自嘲："吾少也贱，故多能鄙事。"——我小时候穷，所以会干不少下贱活。——《论语·子罕》。别人夸他多能，他说那是因为自己没出身、什么都得自己干。有人问他是圣人吗？他说："若圣与仁，则吾岂敢？"——要说圣和仁，我可不敢当。——《论语·述而》。他给自己画的像，是"发愤忘食，乐以忘忧，不知老之将至云尔"——发愤起来忘了吃饭，快乐起来忘了忧愁，连自己快老了都不知道。——《论语·述而》。这是个活得热气腾腾的人。

他既严厉又温和。对子路，他动不动就敲打——"由也好勇过我，无所取材"——《论语·公冶长》；可子路遇难，他哭得把肉酱都吐了。对颜回，他夸"贤哉回也"——《论语·雍也》；颜回一死，他喊"天丧予"。严是严在道理上，温是温在情分上，这两样在孔子身上不打架。

他既坚持原则又懂得变通。阳货想让孔子见他，孔子不见；阳货送了蒸小猪，孔子按礼该回拜，就趁阳货不在家的时候去——既守了礼，又没真见。——《论语·阳货》。公山弗扰在费邑叛乱，召孔子，孔子动了心——"夫召我者，而岂徒哉？如有用我者，吾其为东周乎？"——《论语·阳货》。子路不高兴，孔子也不恼，只是说自己不是白叫去的。孔子一辈子想行道，可他没死等一个机会，机会来了他敢接，原则破了他又收得住。他自己总结过四个字——"毋意，毋必，毋固，毋我"——《论语·子罕》。不凭空猜测，不绝对肯定，不固执己见，不自我中心。这四个"毋"，是孔子性格里"变通"的根。

司马迁在《史记·孔子世家》里写孔子，没把他写成神。司马迁写孔子在陈蔡绝粮，"从者病，莫能兴"——跟从的人都病得起不来，子路愠见曰："君子亦有穷乎？"——君子也有走投无路的时候？孔子答："君子固穷，小人穷斯滥矣。"——君子走投无路还能守住，小人一穷就乱了。——《论语·卫灵公》。司马迁写这一段，写的是孔子在最狼狈的时候，还是孔子。穷不穷是命，守不守是人。

## 三、讲背景

要懂孔子的性格，得先懂春秋时人对"圣人"的理解。

今天说"圣人"，想到的是道德完美、不食人间烟火的神人。可春秋时说"圣人"，说的是"通人"——通达事理、博学多能的人。《尚书》里"圣人"多指聪明睿智之人，不一定是道德楷模。孔子自己说"圣人，吾不得而见之矣，得见君子者斯可矣"——《论语·述而》。可见在孔子眼里，圣人是稀罕物，君子才是修养的目标。

所以孔子活着的时候，没人把他当"圣人"供着。弟子们敬他，可也敢跟他顶嘴、跟他开玩笑。子路动不动就怼他，子游拿他的话堵他，他都不恼。这种师生关系，是春秋时的风气——弟子不是奴仆，老师不是神像。

再说君子修养的标准。春秋时讲"六艺"——礼、乐、射、御、书、数。君子不光要懂道德，还要会礼节、懂音乐、能射箭、能驾车、会写字、会算数。孔子就是六艺皆通的。他年轻时做过季氏的委吏——管仓库，做过乘田——管牛羊，"会计当而已矣""牛羊茁壮长而已矣"——《孟子·万章下》。孔子不是养尊处优的贵族，是从底层干起来的。这份经历，让孔子懂得人间疾苦，也让他不端架子。

孔子的出身也关键。他祖上是宋国贵族，可到了他父亲叔梁纥，已经流落到鲁国。叔梁纥是武士，力能扛城门。孔子三岁丧父，母亲颜征在带他长大，家境贫寒。所以孔子说"吾少也贱，故多能鄙事"。这份"贱"，让孔子没染上贵族的矫情。他后来周游列国，吃过苦，挨过饿，被人围过，被人笑过"累累若丧家之狗"——《史记·孔子世家》。孔子听了反而笑："然哉！然哉！"——是啊是啊，就是这样。能笑着认下"丧家之狗"的人，是个心里有底的人。

再说鲁国的文化土壤。鲁国是周公的封地，保存周礼最全。孔子生在鲁国，从小耳濡目染，"十五而志于学"——《论语·为政》。鲁国的礼乐传统，是孔子性格里"温而厉、恭而安"的根。礼让他知道分寸，乐让他懂得和顺。没有鲁国的文化，长不出孔子这个人。

## 四、讲道理

《论语·述而》里有一句，是弟子给孔子画的像："子温而厉，威而不猛，恭而安。"——老师温和而严厉，威严而不凶猛，恭敬而安详。这十二个字，是讲孔子性格最准的一句话。

朱熹在《论语集注》里注这一句说："惟圣人全体浑然，阴阳合德，故其中和之气见于容貌之间者如此。"——只有圣人全体浑然，阴阳合德，所以中和之气显现在容貌之间是这样。朱熹点出"阴阳合德"四个字，是看透了这十二个字的玄机——温和属阳，严厉属阴；威严属阳，不猛属阴；恭敬属阳，安详属阴。常人温则不厉，厉则不温；威则必猛，不猛则无威；恭则不安，安则不恭。能把这些相反的特质融在一身，不偏不倚，这是修养到了极致才有的气象。朱熹说"惟圣人"，不是说孔子是神，是说这种平衡常人做不到。

钱穆在《论语新解》里讲孔子性情，特别强调一个"真"字。钱穆说，孔子哭颜回"哭之恸"，不是失态，是真情的流露；孔子闻韶"三月不知肉味"，不是矫情，是真美的沉醉。钱穆认为，儒家的修养不是灭情，而是养情——把真情养到正处，喜怒哀乐都发而中节。钱穆这一解，把孔子从"道学先生"的形象里救了出来——孔子不是无情的人，是情到深处自然正的人。

《论语·学而》里子禽问子贡：老师每到一个国家，必能听到那个国家的政事，是求来的还是人家给的？子贡答："夫子温、良、恭、俭、让以得之。"——老师靠温和、善良、恭敬、节制、谦逊得来的。钱穆《论语新解》解这五个字，说这不是五种手段，是五种气象——温是待人接物的温度，良是心地的本善，恭是对人对事的态度，俭是欲望的节制，让是不争的胸襟。五种气象聚在一起，人家见了就愿意把事告诉他。钱穆这一解，点破了孔子性格的"磁场"——不是靠权势压人，是靠气象化人。

## 五、问道悟道

从孔子的性格里，能悟出几条让人开眼的规律。

**真性情与道德修养不矛盾。**

后世有种误解，以为道德修养就是灭情——灭喜怒哀乐，灭七情六欲，修成一尊木像。可你看孔子，颜回死他哭之恸，子路死他吐肉酱，闻韶他三月不知肉味，思归他叹"归与归与"。他的情比谁都真，比谁都深。可这些情，没让他失了分寸——哭归哭，他没去殉葬；痛归痛，他没去寻死。情发到正处，就是修养。

这事古今中外都有印证。《中庸》讲"喜怒哀乐之未发谓之中，发而皆中节谓之和"——情没发时是中，发得恰到好处是和。情不是敌人，发而不中节才是敌人。古希腊亚里士多德讲"中道"——勇敢在怯懦与鲁莽之间，节制在纵欲与麻木之间。情感的美德不是无情，是情到正处。孔子哭颜回，是情到正处；子路死他吐肉酱，是情到正处。真性情与道德修养，在孔子身上是一回事，不是两回事。

**圣人是"人"不是"神"。**

后世把孔子捧成"大成至圣先师"，供进孔庙，烧香磕头。可你看《论语》里的孔子，会开玩笑，会认错，会想家，会被骂"丧家之狗"还笑着认。他不是神，是人——一个把人的好品质活到极致的人。

这事有个反例。明清两代把孔子越捧越高，孔子就成了牌位，成了科举的工具，成了统治的符号。结果五四一来，"打倒孔家店"，孔子从神坛跌到泥里。捧得越高，摔得越惨。为什么？因为神像是立不住的——神像没有血肉，经不起推敲。可《论语》里的孔子，那个会哭会笑会自嘲的孔子，从来没倒。因为他是人，人有血肉就经得起看。

西方也有类似的道理。苏格拉底被神谕说成"雅典最聪明的人"，他自己不信，到处去问那些被认为聪明的人，发现他们其实不聪明，最后得出"我比他们聪明一点，就是我知道自己不知道"。苏格拉底没把自己当神，他承认自己的无知，反而成了最智慧的人。孔子说"若圣与仁，则吾岂敢"，跟苏格拉底那句"我知道自己不知道"是一个路子——承认自己的有限，反而成就了无限。

**情感的真挚是修养的根基。**

孔子能成为孔子，根子在一个"真"字。他对颜回的痛是真的，对韶乐的痴是真的，对故土的念是真的，对弟子的牵挂是真的。这些真，是他一切修养的底子。没有这个底子，礼是空的，乐是虚的，仁是假的。

这事有反证。王莽谦恭未篡时，礼数周全，可那是装的，所以一篡位就露馅。曹操礼贤下士，可他对人的好里有算计，所以"宁教我负天下人"。情不真，修养就是面具，面具戴不久。孔子之所以是孔子，不是他礼数最周全，是他的情最真——真的痛，真的乐，真的牵挂，真的自嘲。情真了，礼才有根；情假了，礼就是壳。

## 结语

圣人不是无情的人，是把真情活到正处的人。
