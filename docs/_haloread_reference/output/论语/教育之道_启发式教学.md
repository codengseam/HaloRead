# 启发式教学——不愤不启，不悱不发

## 一、讲事情

孔子讲学，有一句话成了两千年来中国教育的纲领——"不愤不启，不悱不发"（《论语·述而》）。

这话听着玄，其实就一个意思：学生心里没憋出那股劲儿，老师不开口；学生嘴上没卡到那个说不出来的地步，老师不点拨。非得你自己先琢磨、先憋、先卡，老师才肯伸手。

这话不是空喊。孔子是真这么教的。

先说子贡那回。

子贡是孔子门下最会做生意的弟子，家里有钱，脑子又快。有一天他来见孔子，抛出一句话："贫而无谄，富而无骄，何如？"——《论语·学而》。意思是：一个人穷了不巴结人，富了不翘尾巴，怎么样？

这话搁今天，听着已经挺高了。穷人多半容易低三下四，富人多半容易目中无人，能守住"不谄不骄"，算是有骨气。

孔子却不慌不忙，回了句："可也。未若贫而乐，富而好礼者也。"——还行。但不如另一种：穷却乐道，富却好礼。

这一下，把子贡的境界往上推了一层。"不谄不骄"是消极的——我不做什么；"乐道好礼"是积极的——我去做什么。前者是守住底线，后者是主动攀登。

子贡是聪明人，立刻接住。他没有再问"什么叫乐道好礼"，而是脱口而出一句诗："《诗》云：'如切如磋，如琢如磨'。其斯之谓与？"

"如切如磋，如琢如磨"出自《诗经·卫风·淇奥》，原本是讲加工骨角玉石的工艺——切了再磋，琢了再磨，一遍比一遍精细。子贡拿这话来形容修身——做人的功夫，也是这样一层层打磨上去，没有止境。

孔子一听，笑了："赐也，始可与言《诗》已矣！告诸往而知来者。"——赐啊（子贡名叫端木赐），现在可以跟你谈《诗》了！告诉你过去的，你能推出未来的。

这就是启发。孔子没给子贡讲"乐道好礼"的具体含义，只点了一句"未若"。可这一"未若"，子贡自己悟出了"切磋琢磨"——修身是一辈子打磨的事，不是一锤定音。

再说子夏那回，更精彩。

子夏比孔子小四十四岁，是文学科的高材生（《论语·先进》："文学：子游、子夏"）。有一天他读《诗经》，读到一句"巧笑倩兮，美目盼兮，素以为绚兮"——这是讲美人的：笑起来好看，眼波流转好看，素底子上添了彩就更绚烂。

子夏没看懂"素以为绚兮"那句，来问孔子："老师，这诗是什么意思？"

孔子答了四个字："绘事后素。"——画画的事，后于素底。意思是：先有白底子，才能画花。

这话听着像讲画画的常识。可子夏一琢磨，突然悟出另一层："礼后乎？"——那么礼，是不是也是"后"的？是不是也得先有个底子，礼才立得住？

这一问，把孔子也惊住了。孔子立刻说："起予者商也！始可与言《诗》已矣。"——启发我的人，是你商啊（子夏名叫卜商）！现在可以跟你谈《诗》了。

"起予者商也"——这五个字分量极重。孔子一生教人，极少说弟子"启发了我"。颜回都没得过这话。子夏这一悟，让老师都觉得眼前一亮。

子夏悟的是什么？素是底子，绘是花纹。底子是人的真情实感、质朴本性，花纹是礼。礼不能凭空立，得立在真情之上。没有真情做底子，礼就是空的——一套虚礼、假礼。这跟孔子说的"人而不仁，如礼何"（《论语·八佾》）是一个路子。

孔子只说了"绘事后素"，子夏自己推出了"礼后乎"。老师给一个引子，学生自己悟出一套道理。这就是启发。

还有颜回。

颜回是孔子最得意的弟子，悟性最高。孔子自己说："回也闻一以知十，赐也闻一以知二。"——《论语·公冶长》。颜回听到一，能推出十；子贡听到一，能推出二。

孔子还说过一段耐人寻味的话："吾与回言终日，不违，如愚。退而省其私，亦足以发，回也不愚。"——《论语·为政》。我跟颜回讲一整天，他一句反驳都没有，像个傻子。可他退下去，自己琢磨，能把我说的发挥出来——颜回不傻。

颜回的"发"，不是当场接话，是回去以后自己想通了，还能往前推。这种悟，比子贡的"闻一知二"更深一层。子贡是当场反应快，颜回是事后沉淀深。

孔子还有一句更狠的话："举一隅不以三隅反，则不复也。"——《论语·述而》。给你举出一个角，你不能推出另外三个角，那我就不再教了。

这话听着冷，其实是热。孔子不是嫌弃学生笨，是知道——你不肯自己想，我再讲也是灌水。灌进去的水，留不住；自己悟出来的理，才扎得下根。

## 二、讲人物

启发式教学，背后是孔子作为老师的智慧——他不抢学生的悟。

**孔子：不抢功的老师**

孔子有一句名言："温故而知新，可以为师矣。"——《论语·为政》。能从旧知识里悟出新东西，才能当老师。可他自己教弟子时，常常反过来——让弟子从旧里悟新，他只在关键处点一下。

这种教法，需要克制。老师最怕的，就是忍不住把答案说出来。子贡问"贫而无谄富而无骄"，孔子大可直接讲"乐道好礼比不谄不骄高一层"——可他没长篇大论，只一句"未若"，让子贡自己往下接。子夏问诗，孔子大可讲一通"礼立于情"的哲学——可他只说"绘事后素"四个字，让子夏自己推出"礼后乎"。

这份克制，是高手才有的功夫。新手老师恨不得把所有都讲明白，高手老师知道——讲明白了，学生就不再想了。

孔子还有一句更见格局的话："起予者商也。"——子夏启发了我。一个老师能当着弟子说"你启发了老师"，这是何等的胸襟。孔子不把自己当真理的垄断者，他把教学看成师生共同长进的过程。这种姿态，比任何方法都珍贵。

**子贡：闻一知二的领悟力**

子贡这人，《史记·仲尼弟子列传》说他"利口巧辞"——嘴皮子利索，脑子转得快。孔子常拿他跟颜回比，说他"闻一以知二"。子贡自己也认这个账，他对孔子说："回也闻一以知十，赐也闻一以知二。"——《论语·公冶长》。这话是子贡主动说的——他知道自己不如颜回。

可子贡的长处在别处。他能在老师一句话里听出弦外之音。"未若贫而乐富而好礼"——一般人听到这就完了，子贡却接出"如切如磋如琢如磨"。他不是死记老师的话，是把老师的话当成引子，往自己已有的知识里接。诗经里那句"切磋琢磨"，他早就读过，可直到老师点出"乐道好礼"，他才把那句诗和修身的道理连起来。

这就是启发式教学的妙处——它激活学生已有的积累。子贡读过的诗、想过的事，被孔子一句话点活，自己长出新东西来。

子贡后来成了大商人、外交家。《史记·货殖列传》记他"结驷连骑，束帛之币以聘享诸侯"，所到之处国君与之分庭抗礼。他靠的不是死知识，是那种"告诸往而知来者"的本事——从已知推出未知，从眼前看出远处。这种本事，正是孔子启发出来的。

**子夏：文学之才与深思之悟**

子夏在孔门属文学科，跟子游并列。可他的"文学"不是今天写文章的意思，是熟悉古代文献、能从经典里读出道理。

子夏问"巧笑倩兮"，孔子答"绘事后素"，他立刻推出"礼后乎"。这一悟，不是凭空来的。子夏对《诗》熟，对礼也熟，两个东西在他脑子里本来就有，只是没连上。孔子一句"绘事后素"，像一把钥匙，把两扇门一下打开了。

子夏后来到西河讲学，《史记·仲尼弟子列传》说"子夏居西河教授，为魏文侯师"。他的弟子里有李悝、吴起这些变法名臣。一个能从"绘事后素"悟出"礼后乎"的人，教出来的弟子自然不是死读书的料。子夏这一脉，把孔子的启发式传了下去——魏文侯尊师重道，李悝变法图新，背后都有那种"举一反三"的精神。

**颜回：默识心通的至高境界**

颜回的悟，跟前两位不一样。子贡是当场接话，子夏是当场发问，颜回是当场不响。

"吾与回言终日，不违，如愚"——孔子讲一天，颜回一句不吭，像没听懂。可"退而省其私，亦足以发"——他回去自己琢磨，能把我说的发挥出来。

颜回的悟是内向的。他不抢着表现，不急着反驳，他把老师的话咽下去，慢慢消化，消化完了长成自己的东西。这种悟，比嘴上的快更深。

孔子说颜回"非助我者也，于吾言无所不说"——《论语·先进》。颜回对我说的没有不高兴的，可他从不质疑、从不反驳——太顺了，顺到老师都没机会被弟子逼着再想深一层。这是颜回的长处，也是他的短处——他悟得太深，深到老师都觉得少了点碰撞的火花。

可正因为颜回能"闻一知十"，孔子给他的也是最高的东西。"克己复礼为仁"（《论语·颜渊》）——这话孔子只对颜回说，没对子路、子贡说。为什么？因为只有颜回接得住。给子路讲"克己复礼"，子路会问"怎么做"；给颜回讲，他立刻能推出"非礼勿视，非礼勿听，非礼勿言，非礼勿动"——四个下手处，自己就长出来了。

## 三、讲背景

孔子这套启发式，不是凭空想出来的，是春秋那个时代逼出来的。

先说当时的教育格局。春秋以前，知识垄断在贵族手里，学在官府。贵族子弟进官学，老师是史官、乐官，教的是固定的礼仪、射御、书数。教法是灌输式——老师讲，学生记，记住了就算会。为什么灌输？因为官学教的是"术"，是具体技能，不需要学生悟，只需要学生会做。

孔子开私学，情形变了。他收的弟子，有贵族也有平民，有老有少，有性子急的也有性子慢的。更关键的是——孔子教的不是"术"，是"道"。仁是什么？礼是什么？君子是什么？这些问题没有标准答案，背一万遍也没用，得自己想通。

想通这事，没法灌输。你跟子贡说"仁是爱人"，他记住了，可他不一定懂。非得他自己从"贫而无谄"想到"如切如磋"，从切磋琢磨里悟出修身是一辈子的事——这时候"仁"才在他心里活起来。

再说弟子的水平差异。孔门弟子三千，贤者七十二，可这七十二人水平差得很远。颜回能闻一知十，子贡闻一知二，樊迟问仁连"爱人"都要再问一遍。一个老师面对这种参差，没法讲统一的课。

孔子怎么办？他因材施教（详见前一章），更因势启发——看准弟子卡在哪里，点一下，让他自己往前走。子贡卡在"不谄不骄"自满，孔子点一句"未若"；子夏卡在"素以为绚"不解，孔子点一句"绘事后素"。点完就停，剩下的路学生自己走。

还有一层：春秋是礼崩乐坏的时代。旧秩序塌了，新秩序没立起来。弟子们问的，不是"怎么做"的技术问题，是"该怎么活"的根本问题。这种问题，老师给不了答案——因为乱世没有现成答案。孔子只能引导弟子自己去想、去悟、去立自己的根。

启发式教学，根子在这里——它不是一种技巧，是一种不得已，也是一种必然。教"道"只能启发，教"术"才能灌输。孔子教的是道，所以他必须启发。

## 四、讲道理

"不愤不启，不悱不发"这八个字，后世注家讲得最透的，是朱熹。

朱熹《论语集注》在这一章下注："愤者，心求通而未得之意。悱者，口欲言而未能之貌。启，谓开其意。发，谓达其辞。"

这话拆开来看——"愤"不是愤怒，是心里想通却通不了的那股憋劲；"悱"不是别的，是嘴上想说却说不出的那道坎。"启"是开意，让学生心里的意思通起来；"发"是达辞，让学生嘴上的话顺出来。

朱熹又注："物之有四隅者，举一可知其三。反者，还以相证之义。复，再告也。"——一个东西有四个角，举出一个，另外三个该能推出。"反"是反过来印证，"复"是再讲一遍。学生不能举一反三，孔子就不再讲了。

朱熹的注，点出启发式的两个关键：一是时机——非得学生"愤悱"了才开口，早了没用；二是尺度——只举一隅，剩下三隅让学生自己推，全说了就没了。

《礼记·学记》里有一段话，跟孔子这套教法互为表里："君子之教喻也：道而弗牵，强而弗抑，开而弗达。道而弗牵则和，强而弗抑则易，开而弗达则思。和易以思，可谓善喻矣。"

"道而弗牵"——引导而不硬拉。"强而弗抑"——督促而不压制。"开而弗达"——点开而不说透。引导不硬拉，师生就和睦；督促不压制，学生就觉得容易；点开不说透，学生就会自己思考。和睦、容易、思考，这三样齐了，才叫"善喻"——善于启发。

"开而弗达"四个字，正是"举一隅不以三隅反则不复也"的另一种说法。开一个口子，不说到头；举一个角，不点全四个角。剩下的，留给学生。

《学记》还有一句更狠："善待问者如撞钟，叩之以小者则小鸣，叩之以大者则大鸣。"——善于答问的人像敲钟，轻轻敲响得轻，重重敲响得重。学生问得浅，老师答得浅；学生问得深，老师答得深。子贡问"贫而无谄"，孔子答"未若贫而乐"——这是中等力度的敲；子夏问"巧笑倩兮"，孔子答"绘事后素"——这是轻敲，可子夏接得住，自己撞出大响。

朱熹和《学记》合起来看，启发式教学的道理就清楚了：教育不是灌水，是点火；不是替学生想，是让学生自己想；不是把答案端上来，是把学生推到答案门口，让他自己推门进去。

## 五、问道悟道

启发式教学这八个字，往深了想，藏着几条经得起琢磨的规律。

**真正的学习来自内在驱动。**

孔子说"不愤不启"——学生不憋到那股劲儿，不开口。这话反过来就是：学习得先有"愤"。心里想通却通不了，嘴上想说却说不出的那股劲，是学习的真正燃料。没有这股劲，老师讲得再好也是对牛弹琴。

这道理古今中外都一样。古希腊的苏格拉底搞"产婆术"——不直接给答案，靠提问让对方自己把真理"生"出来。可苏格拉底提问之前，先得让对方陷入困惑——你以为懂的，被他一问发现不懂；你以为对的，被他一问发现自相矛盾。这"困惑"就是孔子说的"愤"。非得你先意识到自己不懂、先急着想懂，真理才能从你嘴里"生"出来。

苏格拉底审问美诺的奴隶小童，问一个几何问题。小童一开始以为自己懂，被苏格拉底一步步问下来，发现自己其实不懂——这时候小童急了，开始自己琢磨。苏格拉底没给答案，只继续问，小童最后自己推出了正确结论。这跟孔子"举一隅不以三隅反则不复也"是一个路子——老师只引路，学生自己走。

**思考比答案更重要。**

孔子给子贡的，不是"乐道好礼"四个字的定义，是一句"未若"——一个引子。子贡接出"如切如磋如琢如磨"，这诗他早读过，可直到那一刻才跟修身的道理连上。连上的瞬间，比记住一百个定义都值钱。

为什么？因为答案是可以背的，思考是长出来的。背下来的答案会忘，长出来的思考不会。子贡一辈子记得"切磋琢磨"，不是因为他背过，是因为他自己悟过。

这道理今天依然管用。一道数学题，老师讲了你会做，换个数就不会——因为你背了过程没懂原理。可如果你自己憋半天，卡在某个地方，老师点一下，你突然通了——这道题你这辈子都不会忘。通的那一瞬间，知识就从老师的变成了你的。孔子说"举一隅不以三隅反则不复也"，要的就是这个"反"——学生自己反过来印证，知识才扎得下根。

**举一反三是智慧的特征。**

"举一隅以三隅反"——给一个角，推出三个角。这不只是学习方法，是智慧的特征。

智慧跟知识的差别在哪？知识是一个一个的，智慧是一通百通的。子贡闻一知二，颜回闻一知十——差距不在记多少，在能从一里推出多少。能推出十的，是抓住了理；只能推出二的，是抓住了事。理是通的，事是死的。

这规律放之四海皆然。中医讲"同病异治、异病同治"——同一个病在不同人身上治法不同，不同的病根子相同治法相通。为什么？因为医者抓住了"理"，不是死守"症"。兵法讲"兵无常势，水无常形"（《孙子兵法·虚实篇》）——战场千变万化，可兵法的理是通的。能举一反三的将领，才能以不变应万变。

孔子教弟子，要的就是这种"通"。子夏从"绘事后素"悟出"礼后乎"——画画的理通到礼上去了。这种跨领域的贯通，是智慧最显眼的样子。死知识只能解决见过的问题，活智慧能解决没见过的问题。启发式教学教的不是知识，是那种"通"的能力。

---

## 结语

教人不在灌答案，在点一把火——学生心里先烧起来，老师只在火苗将熄时拨一拨。
