# 90%产能：1877年全国炼油版图的归一

## 一、入戏

1879年11月某个深夜，克利夫兰，标准石油总部二楼办公室。

屋里没点灯，只有窗外的月光和远处伊利湖上油轮的灯火。一个瘦削的男人站在窗前，背对着屋里那张巨大的美国地图，一动不动。

地图上钉满了红色图钉——每一颗代表一家被标准石油收购或控股的炼油厂。从克利夫兰到匹兹堡，从匹兹堡到费城，从费城到纽约，从纽约到巴尔的摩，红色图钉沿着东海岸连成一串，再从克利夫兰往西，零零星星撒进俄亥俄、印第安纳、伊利诺伊几州。

这人就是约翰·洛克菲勒，四十岁。他站在窗前，看着湖上的灯火，手里攥着那本翻得发软的账本A。

门吱呀一声推开，弗拉格勒端着两杯咖啡走进来。

## 二、破题

要解决的问题只有一个——标准石油从克利夫兰一个地方炼油厂主，怎么一步一步变成了全美炼油的垄断者，又该怎么面对这个位置带来的代价。

金手指在这一章里翻到了最难的一页——账本A第一次帮洛克菲勒算的不是"怎么赢"，而是"赢了之后怎么办"。洛克菲勒翻开账本A，翻到那一页——上面写着两行字：

"全美炼油产能占比：约90%。垄断阈值——超过即引来监管。监管红线？"

他对弗拉格勒说："亨利，90%这个数，我算了好几年。可算到这一天，我反倒睡不着了。"

## 三、推进

### 一、归一之路

标准石油走到90%这一步，不是一夜之间的事，是从1872年南方改良公司那场惊天的密约失败之后，一点一点啃下来的。

1872年，南方改良公司密约曝光，全国独立炼油厂主群情激愤，洛克菲勒差点被赶出油区。他吸取的教训不是"别垄断"，而是"别让人看出来在垄断"。从那以后，他换了个打法——不靠密约，靠收购；不靠铁路回扣，靠管道；不靠一纸协议，靠一家一家的股权。

克利夫兰是第一站。1872年那场"克利夫兰大屠杀"，二十六家独立炼油厂，标准石油吃下二十二家。到1877年，克利夫兰全城的炼油产能，几乎全在标准石油名下。

接着是匹兹堡。匹兹堡是当时全美第二大炼油中心，独立炼油厂林立。洛克菲勒没硬来，他派代理人一家一家谈——条件不高，给现金加标准石油的股权，让原厂主留任当经理。多数厂主算不过账：自己单干一年挣五千，并入标准石油一年分八千还不用操心销路。一家签，家家签。到1878年，匹兹堡的炼油产能，标准石油拿了过半。

费城和纽约是更难的骨头。这两地的炼油厂主资本厚、关系硬，不肯轻易就范。洛克菲勒换了个法子——他不买厂，他掐管道。标准石油的管道铺到费城和纽约城外，原油要进城，得走标准石油的管子；走管子就得按标准石油的价。一年之内，费城和纽约的独立炼油厂，一半因为原油成本太高关了门，一半主动来谈收购。到1879年，这两地的炼油产能，标准石油也拿了过半。

加上油区一带的小炼油厂、巴尔的摩的几家、西海岸零星的几家，到1879年底，标准石油控制的全美炼油产能——约九成。

九成。这个数，洛克菲勒在账本A上算了好几年，算到这一天，终于成了真。

### 二、垄断的代价

可九成这个数，不是只有洛克菲勒一个人在算。

1879年这一年，反对标准石油的声音，从油区一路传到了华盛顿。

油田主先闹。原油价格被标准石油压得太低——油区一口井，打出来一桶油，卖给标准石油才挣一毛多，连打井的本钱都收不回。油田主们组织了"石油生产者联盟"，几次开会，决议都是一条：要求标准石油提高原油收购价。可决议归决议，标准石油是唯一的买家——你不卖给它，你的油就只能烂在井里。

独立炼油厂主跟着闹。那些还没被吞掉的小厂主，凑在一起办了份报纸，专门登标准石油的"黑料"——回扣、压价、逼收购，一桩一桩往外抖。报纸印数不大，可每一份都送到国会议员和州议员的办公桌上。

政客们也嗅到了味儿。1879年，俄亥俄州议会有人提出动议，要求调查标准石油是否垄断；纽约州议会也有人跟上。动议没通过——那年头反托拉斯这个词还没写进法律，要再等十一年才有《谢尔曼法》。可动议提出来这件事本身，就是个信号。

洛克菲勒收到这些消息时，没开会让大家讨论对策。他一个人坐在办公室里，把账本A翻到那一页——"全美炼油产能占比：约90%"——盯着看了很久。

### 三、90%的反思

那天夜里，弗拉格勒推门进来，看见洛克菲勒对着账本A发呆。

"约翰，怎么了？"

洛克菲勒没抬头，只把账本A推过去，指着那行"90%"："亨利，你看这个数。"

弗拉格勒看了一眼："九成。好事啊。咱们算了好几年。"

"是好事。"洛克菲勒说，"可也是坏事。"

他翻到下一页，上面是他自己写的几行字：

"九成是顶。再往上，没人敢跟我们做生意——怕被吞。
九成是线。线这边是生意，线那边是官司。
九成是靶。靶子越大，越多人想打。"

弗拉格勒看完，没吱声。他跟洛克菲勒合伙这么多年，头一回见洛克菲勒把"赢"写成"靶"。

洛克菲勒从抽屉里拿出一支红铅笔，在那行"90%"旁边，慢慢画了一个圈，又在圈边写下四个字——

"监管红线？"

"亨利，"洛克菲勒把笔放下，"我们从克利夫兰那二十二家厂起步，一路吃到九成。可这九成，是别人让我们吃的，还是我们硬抢的？"

弗拉格勒想了想："一半一半。"

"那一半'硬抢'的，迟早要还。"洛克菲勒说，"今天他们没法还，明天会有办法。十年后、二十年后，总有办法。"

弗拉格勒没接话。他知道洛克菲勒说的不是没有道理——可他也知道，箭已经射出去了，收不回来。

## 四、高潮

夜更深了。办公室里只剩煤油灯一豆光。

弗拉格勒把咖啡递给洛克菲勒，自己端着另一杯，靠在窗边。两人都没说话，屋里只有湖风拍打窗框的声音。

良久，弗拉格勒开口："约翰，90%——是终点吗？"

洛克菲勒没立刻答。他端着咖啡，走到那张巨大的美国地图前，伸手摸了摸克利夫兰那颗最早钉上去的红色图钉——那是1872年"大屠杀"那夜钉的，钉子头都磨亮了。

他又摸了摸纽约那颗、费城那颗、匹兹堡那颗。每一颗，都代表一场收购、一次谈判、一笔账本A上算过的账。

最后，他的手停在地图最右边——大西洋海岸线那一条蓝线边上。那里没有红色图钉。那里是海，海的另一边是欧洲，是亚洲，是整个还没被任何一家石油公司碰过的世界。

洛克菲勒转过身，看着弗拉格勒，慢慢说：

"亨利，90%是终点。"

他顿了一下，又补了一句：

"也是起点。下一步——是跨国。"

弗拉格勒愣住。他端着咖啡的手停在半空："跨国？"

"美国这九成，我们已经吃到了顶。再往上吃，就是监管、是官司、是拆分。"洛克菲勒走回桌边，重新翻开账本A，在新的一页上写下两个字——

"欧洲。"

"美国的煤油，欧洲还烧不起灯。一桶煤油运到伦敦，能卖克利夫兰的三倍价。运到上海，能卖五倍。"洛克菲勒一边写一边说，"我们手里有九成的产能，有管道，有油轮码头。下一步，是把这九成的煤油，卖到全世界去。"

弗拉格勒放下咖啡杯，走到地图前，看着大西洋那一片蓝："约翰，你是说——咱们不在美国这九成里头死磕了？"

"不是不死磕。"洛克菲勒把账本A合上，"是在死磕之前，先把盘子做大。盘子一大，九成就不显眼；盘子一小，九成就是靶子。"

他走回窗边，重新看向伊利湖上的灯火。湖那头，一艘油轮正缓缓驶向湖口，明天一早它会经过布法罗，沿着运河到纽约港，再从纽约港装上远洋船——

"亨利，"洛克菲勒低声说，"美国的九成，是咱们的根。可根扎完了，得往上长。"

弗拉格勒站到他身边，也看向湖上那艘油轮的灯火。两人都没再说话。

地图换挡完成的瞬间，就停在这一刻——

从克利夫兰一个地方炼油厂主，到全美九成产能的垄断者，再到即将跨过大洋的跨国帝国。这一夜，是分水岭。

## 五、余味

**OPC 翻译桥**：

标准石油控制90%炼油产能——这是垄断阈值，超过即引来监管。洛克菲勒自己都把"监管红线"写进了账本A。OPC启示：单一渠道占比超70%就是红色警报。今天起，审视你的收入结构——某个客户、某个平台、某个渠道占你收入70%以上？这是垄断阈值的反面：你不是垄断者，你是被垄断者。一个大客户占你70%的订单，他一砍价你就死；一个平台占你70%的流量，它一改算法你就没；一个渠道占你70%的销售，它一封号你就停。30天内必须把单一渠道占比降到50%以下，否则这个渠道一改规则你就死。洛克菲勒的解法是"跨国"——把盘子做大，让九成不显眼。你的解法是什么？是开拓新客户、新平台、新产品，还是把鸡蛋从一个篮子挪到三个篮子？别等到监管红线那天才动，那时候已经晚了。

---

1879年11月那个深夜，克利夫兰标准石油总部。

弗拉格勒走后，洛克菲勒一个人又站到窗前。伊利湖上的油轮灯火已经看不见了，只剩湖面上一片漆黑。

他重新翻开账本A，翻到那一页——"全美炼油产能占比：约90%"旁边那个红圈，还在。圈边那四个字"监管红线？"，也还在。

他没擦掉那个圈，也没擦掉那四个字。他知道，这四个字不是写给今天的，是写给十年后、二十年后的——写给那个迟早会来的、他算得出却躲不开的一天。

他在那一页的下方，又添了一行新字：

"1879年11月。全国归一。下一步——跨国。欧洲。亚洲。"

写完，他合上账本A，吹熄煤油灯。

屋里一片黑。只有窗外月光照着那张美国地图——地图上红色图钉钉得密密麻麻，可地图最右边，大西洋海岸线那一条蓝边之外，还是一片空白。

那片空白，等着被填上。

三年后，1882年，标准石油托拉斯成立——那张密密麻麻的红色图钉网，将被一份法律文件正式捆成一个整体。再过几年，标准石油的煤油会以"船用油"的名义，第一次装上远洋轮，运往伦敦、汉堡、上海。

而这一切的起点，就是1879年这个深夜，洛克菲勒站在窗前，对弗拉格勒说的那句话——

"90%是终点，也是起点。下一步——是跨国。"

湖面上起风了。窗框被吹得咯吱响。

洛克菲勒没动。他看着窗外那片漆黑的湖，轻声说了一句，像是说给自己听——

"亨利，咱们这一代，把美国的灯点亮了。"

"下一代，得把全世界的灯点亮。"
