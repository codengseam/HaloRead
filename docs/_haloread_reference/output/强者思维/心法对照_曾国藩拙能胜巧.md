# 曾国藩拙能胜巧

## 一、入戏

有个名校毕业的年轻人，入职头三年顺风顺水。他脑子快，开会发言总能切中要害；他眼力好，老板要什么他三秒就能嗅出来；他手也快，活儿一来找捷径、套模板、借工具，半天交差。三年里他晋升两次，成了同批里最快冒头的人。

第四年他负责一个真要啃硬骨头的项目——需要把一套没人愿意碰的旧系统重构。他的快招全失灵了：模板套不上、工具用不了、捷径走不通。他熬了两个月交出一版，上线就崩。老板问他怎么回事，他答得也快："这套系统本身就有问题，前任留的坑太深。"老板没说话，把项目转给了同部门一个比他笨、比他慢的同事。

那位同事接手的方式让他嗤之以鼻——一行行读旧代码、一个个问业务方、一页页写文档、一段段做测试。三个月没出任何"亮眼成果"，团队私下都说他磨叽。可半年后那套系统稳稳上线了，bug 比重构前少了一半。第二年那位同事升职，他被调到一个边缘岗位。

他不服气很久。直到有次偶然读到曾国藩的家书，看到"结硬寨，打呆仗"五个字，他愣住了——原来真正能啃下硬骨头的，从来都是肯下笨功夫的人，聪明人反倒啃不下。聪明人擅长短跑，笨人才能跑长跑；聪明人擅长的活儿都是"看起来漂亮"，笨人擅长的活儿都是"做出来扎实"。这世道奖励短跑，但厚东西都是长跑磨出来的。

## 二、破题

曾国藩这个人是研究"拙"最好的样本。他考秀才考了七次，前六次都落榜，主考官还把他的卷子当作反面教材张榜公示，批四个字"文理欠通"。换作一般人，要么放弃科举，要么换个赛道。曾国藩把那张公示的卷子带回家，贴在书桌前，逐字逐句琢磨自己到底差在哪。第七次中了。

这种"被羞辱也不绕开，硬把短处磨平"的劲头，贯穿了他一辈子。后来他带湘军打太平天国，用的还是这一招。湘军每到一个地方，第一件事不急着打仗，是挖壕沟、筑高墙、扎硬寨，哪怕对面敌军只有几百人也要这么干。挖完壕沟不主动出击，等敌人来攻，靠壕沟墙垒消耗对方。打不下来就围，围几个月甚至一年，把敌人围到粮绝自溃。打下来就推进几十里，再挖再扎。一年推进两三百里，慢到让人发指。

他自己上奏朝廷时大白话写过：「臣不善骑马，未能身临前敌……但知结硬寨，打呆仗，从未用一奇谋、施一方略制敌于意计之外」——大意据曾国藩奏折。一个统帅，公开承认自己不善骑马、不能临阵、不会奇谋、没有方略，只会结硬寨打呆仗。这话在崇尚奇谋妙算的文化里几乎是自污。可就是这套笨办法，把横扫半个中国的太平天国熬垮了。

「城贼猛扑，凭濠对击，坚忍不出，最为合法」——大意据《曾国藩家书·致九弟》。敌人猛扑过来，靠壕沟对射，坚忍不出战，这才是最合规矩的打法。一个"坚忍不出"，道尽了拙的真意——不接对方的节奏，不打对方希望你打的那种仗，按自己的笨办法一寸一寸推进。

民间常有一句话挂在曾国藩名下："天下之至拙能胜天下之至巧"。这句话的出处其实存疑——《挺经》是后人辑录，此句是否为曾氏原话未见可靠原文。但即便这句话不是他亲口说的，"拙能胜巧"四个字依然是他一辈子行事的总纲，也是这一章要讲的事。

## 三、方法论

曾国藩的"拙"落到操作上，是三层系统。

**第一层：不取巧的系统思维。**

"结硬寨打呆仗"看似是战术，本质是系统思维。它的核心在于不依赖运气。奇谋妙算依赖的是对手犯错、时机凑巧、天时地利——这些都是不可控的。曾国藩把所有不可控的因素从胜利公式里剔除，只保留可控的：挖壕沟、筑墙、扎营、围困、推进。每一步都自己做主，每一步都不指望对方配合。胜利不押在对手犯错上，押在自己的笨功夫累积到对手熬不住。

这是拙的第一重底子：把命运攥在自己手里。聪明人喜欢赌时机、赌对手、赌运气，曾国藩不赌——他把赌局改造成工程，把"等运气"改成"做工程"。系统思维恰恰是笨人最该走的路，因为笨人赌不起，只能靠系统。

落到强者思维上，这意味着你的胜利公式里要尽可能多放"自己能控制的变量"，少放"指望外界的变量"。控制二分法划的是边界，拙诚是边界内的执行方式——能控制的那一半，用笨功夫做到极致；不能控制的那一半，干脆不指望。

**第二层：打脱牙和血吞的逆境姿态。**

曾国藩一生吃过无数次败仗，三次想投水自尽，被同僚排挤、被皇帝猜忌、被朋友背刺。他怎么熬过来的？他在家书里写过一句很重的话——「好汉打脱牙和血吞」。原文语境是他说自己怄气从不说出，一味忍耐，徐图自强，把这句谚语当作生平咬牙立志之诀。

打脱牙和血吞的要点，是把屈辱吞下去当燃料，而非逆来顺受。聪明人吃亏的第一反应是找人说、找地方评理、找机会扳回——这些都是能量外耗。曾国藩的拙是反过来：先把屈辱咽下，不让它外溢成抱怨和辩解；再在内部把它转化成"我下次怎么做"的复盘。这是拙的第二重底子：逆境里不浪费能量在外求，把所有能量收回来磨自己。

「吾生平长进，全在受挫辱之时」——大意据《曾国藩家书》。我一辈子长进，全在受挫受辱的时候。这句话把挫折和长进绑成了因果关系——长进发生在挫折之中，而非挫折之后。聪明人在挫折里想的是"怎么脱身"，曾国藩在挫折里想的是"这次能磨出什么"。前者是逃，后者是磨。

**第三层：笨功夫磨真本事。**

曾国藩的日课是出了名的严苛——早起、静坐、读史、写字、写日记、谨言、养气、保身，每日不辍，连打仗最紧的时候都没断。他在家书里反复告诫子弟不可走机巧一路，「万不可走入机巧一路，日趋日下也」。他自己也坦承"吾自信亦笃实人，只为阅历世途，饱更事变，略参些机权作用，便把自家学坏了"，把"机巧"视为对自家本心的污染。

这一层最深。拙不是不会巧，是会巧而不用。曾国藩带兵打仗后期当然会用机谋，但他始终把机谋当工具，不当根基。根基是那套日复一日的笨功夫——读书、写字、静坐、自省。这些事短期看不出效果，聪明人都不屑做；但十年二十年下来，它们在人身上下出来的根，比任何巧办法都深。

笨功夫的真意，是把真本事磨进身体里。聪明人靠脑子记住的东西，遇压就忘；笨人靠身体磨出来的反应，遇压才显。曾国藩的湘军为什么比太平军能熬？因为湘军每天扎营挖壕的笨功夫，把"守"磨成了肌肉记忆，太平军的奇袭对这种肌肉记忆无效。

三层系统串起来：不取巧（边界）+ 打脱牙和血吞（逆境姿态）+ 笨功夫磨真本事（根基）。这就是曾国藩版的强者思维——靠系统、靠忍耐、靠笨功夫，把聪明人熬死。

## 四、避坑

讲曾国藩的"拙"，有三个坑最常见。

**坑一：把"拙"理解为"笨"。**

拙是选择上的拙，而非智力上的笨。曾国藩本人绝对不笨——他二十七岁中进士、入翰林，平定太平天国、办洋务、建江南制造局，一辈子干的事没一件是笨人能干的。他的"拙"是明知有巧路也不走，明知有捷径也不抄，主动选择那条看起来笨的路。这是聪明人选了笨，比笨人选了笨难得多。把拙等同于笨，就丢了曾国藩最值钱的那一层——对巧的克制。

**坑二：把"结硬寨打呆仗"理解为保守。**

很多人读这五个字，以为曾国藩是个保守的统帅，不敢冒险、不会进攻。这是误读。结硬寨打呆仗的目的是为了稳赢——曾国藩打的不是"避免失败"的仗，是"必胜"的仗。他宁可慢，也不要把胜利押在不可控的奇谋上。这是另一种意义上的激进：他对胜利的执着比任何奇谋派都深，所以才能接受最慢的路径。保守是不敢动，拙诚是不动则已、一动必成。

**坑三：用拙诚否定聪明。**

有人读完曾国藩，跑去反对一切聪明、反对一切效率、反对一切工具，认为只有笨办法才是正道。这是另一个极端。曾国藩自己办洋务、造枪炮、译西学，从来不排斥新工具。他反对的是"以机巧为根基"，并非反对工具本身。聪明是好的，工具是好的，效率是好的——只要它们建在拙诚的根子上。把拙和巧对立起来，就误读了曾国藩。拙是根，巧是枝；无根之巧一压就断，有根之巧越用越利。

## 五、践行

曾国藩的拙诚落到今天，可以做三件可量化的事。

**第一件：找出你生活里的"奇谋依赖"。**

花一周时间记录你做事的方式，重点标记那些"靠捷径完成"的事——套模板的方案、借工具的速成、抄近路的决策、靠人脉拿到的资源。记录下来是为了看清楚：你的胜利公式里，有多少押在"运气/捷径/关系"上，有多少押在"自己能控制的笨功夫"上。

如果前者超过一半，你的系统是脆弱的——只要捷径断了、关系冷了、运气转了，你就崩。曾国藩把押在前者的部分降到最低，所以他能熬过任何不利。这一周记录完，挑一项最依赖捷径的事，下周改用笨办法做一遍，感受一下差别。

**第二件：建立"打脱牙"日记。**

每天晚上花十分钟，写下今天让你不舒服、想抱怨、想辩解的三件事。写下来是为了"吞下去"——写完就合上，不外溢。

这一步模仿的是曾国藩"怄气从不说出，一味忍耐"的功夫。关键在"不外溢"三个字。聪明人吃亏的最大损耗，在于吃亏之后花大量能量找人诉说、寻求认同、构筑受害者叙事。把这些能量收回来，全部转成"我下次怎么做"的复盘，你的能量账户会立刻多出一大笔。

一个月后翻这本日记，你会看到自己的"受挫模式"——哪些事反复让你不爽、哪些人反复触发你的反应。看清模式，模式就开始松动。

**第三件：定一项"笨功夫"日课，雷打不动做一年。**

挑一件你认可但一直懒得做的笨事——可以是每天读半小时书、每天写五百字、每天静坐十五分钟、每天做一组俯卧撑——定下来，雷打不动做一年。重点是"不间断"，强度可以放低。

曾国藩日课的力量在"不间断"三个字。间断一次，根就断一截；不间断，根就一直在长。一年下来，这件事会从"需要意志力去做"变成"身体自动会做"，这时候真本事就磨进身体了。

聪明人最容易在这一步上栽跟头——他们觉得这种笨功夫"效率太低"，不如找更聪明的方法。可一年后回头看，那些找聪明方法的人还在原地找，笨功夫的人已经走出很远。这就是拙能胜巧的现场版。

三件并做，一年后再回头看那个让你"嗤之以鼻"的笨同事，你会懂——他赢的是那套别人看不上的笨办法，而不是那一次项目。曾国藩一百多年前就讲透了的事，到今天依然有效。
