## 概念

### 这章考什么

- **科目**：中级网络工程师综合知识、案例分析
- **考频**：高频，VLAN/STP 计算几乎是每年案例题
- **分值**：中级约 8-12 分
- **难度**：中等偏上，CRC 计算和 STP 选举需动手练

数据链路层把物理层的原始比特流变成可信的帧流，局域网技术全部建立在这层之上。交换机、VLAN、生成树是网络工程师每天打交道的家伙。

## 原理

### 【中级重点】数据链路层功能

链路层五大职责：成帧、物理寻址、差错控制、流量控制，再加介质访问控制。成帧给比特流加分隔，物理寻址用 MAC 地址定位网卡，差错控制用 CRC 检错，流量控制防止发快收慢，介质访问控制解决多主机共享信道冲突。

#### 成帧方法

- **字符计数法**：帧头记录帧总长，缺点是计数位出错全乱
- **字符填充法**：用特殊字符标记起止，数据中出现特殊字符就转义
- **位填充法**：用标志位 01111110 标记帧边界，数据中连续 5 个 1 后插 0，HDLC、PPP 采用

### 【中级重点】差错控制

#### CRC 循环冗余校验

CRC 用多项式除法检错。发送方把数据后面补 k 个 0（k 等于生成多项式位数减 1，也就是最高次幂），用模 2 除法除以生成多项式，余数就是 CRC 校验码，替换掉补的 0。

模 2 除法是不借位的异或除法。示例：数据 1010，生成多项式 10011（x⁴+x+1），补 4 个 0 得 10100000，模 2 除：

```
        1010
     ---------
10011 ) 10100000
        10011
        -----
          1110
          10011
          -----
           1111
           10011
           ----
            1100
```

余数 1100 即 CRC 校验码，发送数据为 10101100。接收方用收到的完整数据除以生成多项式，余数为 0 则无误。

#### 海明码

海明码 Hamming Code 能纠一位错。校验位数 r 满足：

> 2^r ≥ m + r + 1

m 为数据位数。例如 4 位数据，2^r ≥ 4+r+1，r=3 时 8≥8 成立，需 3 位校验位。校验位放在 2 的幂次位置（1、2、4、8…），其余位置放数据位。

### 【中级重点】流量控制

- **停等协议 Stop-and-Wait**：发一帧等一个 ACK，效率低
- **后退 N 帧 GBN（Go-Back-N）**：发送窗口>1、接收窗口=1，出错后从错帧重传所有后续
- **选择重传 SR（Selective Repeat）**：发送窗口和接收窗口都>1，只重传出错帧

GBN 窗口大小 ≤ 2^n - 1（n 为序号位数），SR 窗口大小 ≤ 2^(n-1)。

### 【中级重点】介质访问控制

#### CSMA/CD

载波监听多路访问/冲突检测 Carrier Sense Multiple Access with Collision Detection，有线以太网用。四句口诀：

1. 先听后发：发前先听信道空闲
2. 边发边听：发送时持续监测
3. 冲突停发：检测到冲突立即停
4. 延迟重发：随机退避后重试

最小帧长 = 2 × 传播时延 × 带宽（即争用期内能发完的最小帧）。10Mbps 以太网争用期 51.2μs，最小帧长 64 字节，这就是以太网帧不能小于 64 字节的根源。

#### CSMA/CA

无线局域网用，因为无线难检测冲突，改用避免。增加 RTS-CTS 握手预留信道，隐藏终端问题靠 RTS/CTS 缓解。

### 【中级重点】以太网

IEEE 802.3 标准。帧结构：

| 字段 | 前导码 | SFD | 目的MAC | 源MAC | 类型 | 数据 | FCS |
|---|---|---|---|---|---|---|---|
| 字节 | 7 | 1 | 6 | 6 | 2 | 46-1500 | 4 |

前导码 7 字节同步时钟，SFD 1 字节标记帧开始，目的/源 MAC 各 6 字节，类型 2 字节标识上层协议（0x0800 是 IP），数据 46-1500 字节，FCS 4 字节 CRC。加上前导码和 SFD，最小帧 64 字节，最大帧 1518 字节。

MAC 地址 48 位，前 24 位 OUI 厂商代码，后 24 位厂商分配。第一字节最低位 0 表示单播、1 表示组播，次低位 0 表示全局唯一、1 表示本地管理。

### 【中级重点】VLAN

IEEE 802.1Q 在以太网帧源 MAC 和类型间插入 4 字节 Tag，其中 12 位 VLAN ID，理论 4096 个 VLAN，减去保留实际可用 4094 个。

端口模式：

| 模式 | 收发 | PVID |
|---|---|---|
| Access | 只属一个 VLAN，发出去剥 Tag | 有 |
| Trunk | 多 VLAN 通过，发出去带 Tag（除 Native） | 有 |
| Hybrid | 灵活，可指定哪些 VLAN 带 Tag 哪些剥 | 有 |

VLAN 间路由三种方式：单臂路由（路由器子接口）、三层交换机（最常用）、外部路由器多端口。

### 【中级重点】STP 生成树

IEEE 802.1D，解决交换环路和广播风暴。核心机制：

1. **选举根桥**：桥 ID = 优先级（默认 32768）+ MAC 地址，最小者当根
2. **选举根端口**：非根交换机到根开销最小的端口
3. **选举指定端口**：每条链路选一个到根开销最小的端口
4. **阻塞其余端口**：剩下端口阻塞，消除环路

端口状态：阻塞 Disabled → 监听 Listening（15s）→ 学习 Learning（15s）→ 转发 Forwarding。从阻塞到转发约 50 秒，这是 STP 收敛慢的根因。

**RSTP**（802.1w）快速生成树，引入替代端口和备份端口，收敛降到 1-2 秒。**MSTP**（802.1s）多生成树，把多个 VLAN 映射到多个实例，每个实例独立计算生成树，平衡负载。

### 【中级重点】PPP 与 HDLC

PPP 点对点协议 Point-to-Point Protocol，三阶段：

1. **LCP 链路建立**：协商参数（MRU、认证方式）
2. **认证**：PAP 两次握手明文、CHAP 三次握手加密（挑战-应答）
3. **NCP 网络层协商**：IPCP 协商 IP 地址

HDLC 高级数据链路控制，思科串行链路默认封装，同步传输，标志位 01111110 加位填充。

### 【中级重点】WLAN 无线局域网

IEEE 802.11 系列标准：

| 标准 | 频段 | 最高速率 | 调制 |
|---|---|---|---|
| 802.11a | 5GHz | 54Mbps | OFDM |
| 802.11b | 2.4GHz | 11Mbps | DSSS |
| 802.11g | 2.4GHz | 54Mbps | OFDM |
| 802.11n | 2.4/5GHz | 600Mbps | MIMO-OFDM |
| 802.11ac | 5GHz | 6.9Gbps | MU-MIMO |
| 802.11ax（WiFi6） | 2.4/5GHz | 9.6Gbps | OFDMA |

AP+AC 架构：瘦 AP 只负责射频，AC 集中管理转发。WPA2 用 AES-CCMP，WPA3 引入 SAE 替代 PSK 抗暴力破解。

## 实践

### 【高级衔接】数据中心网络架构

传统三层架构（接入-汇聚-核心）适合南北向流量为主的 Web 业务。现代数据中心虚拟化迁移、分布式存储带来大量东西向流量，三层架构的汇聚层带宽瓶颈暴露。

Spine-Leaf 脊叶架构每台 Leaf 交换机连接所有 Spine，任意两台 Leaf 间两跳可达，无阻塞带宽容易做。但 Leaf 数量受 Spine 端口数限制，扩展时需同步扩 Spine。

VXLAN 用 MAC-in-UDP 封装，把二层帧封装进 UDP 报文跑在三层网络上，16M 个 VNI 解决 4094 个 VLAN 不够用的问题，是大二层网络和云数据中心多租户隔离的基础。

**真题示例（2024年·案例分析·VLAN与STP相关·考生回忆）**：某网络三台交换机三角连接，给出优先级和 MAC，问根桥、根端口、指定端口、阻塞端口。解题步骤：先比桥 ID 选根桥（优先级小的优先，相同比 MAC 小的），再各交换机算到根开销选根端口，每条链路比两端到根开销选指定端口，余下阻塞。

## 速查/自测

### 以太网帧结构速记

前导7 + SFD1 + 目MAC6 + 源MAC6 + 类型2 + 数据46-1500 + FCS4 = 最小64/最大1518。

### VLAN 端口模式对比

| 模式 | Tag 处理 | 用途 |
|---|---|---|
| Access | 收加发剥 | 接终端 |
| Trunk | 除 Native 外都带 | 交换机互联 |
| Hybrid | 灵活指定 | 华为常用 |

### STP 端口状态

阻塞 → 监听（15s）→ 学习（15s）→ 转发，收敛约 50s。

### 自测题

1. 数据 1101，生成多项式 1001，求 CRC 校验码。答：补 3 个 0 得 1101000，模 2 除 1001 得余数 001，发送 1101001。
2. 三台交换机优先级 32768/32768/4096，MAC 各异，谁是根桥？答：优先级 4096 的。
3. 802.11ac 工作在哪个频段？答：5GHz。

一句话小结：链路层管帧，VLAN 切广播域，STP 防环路，CRC 要会算，STP 要会选。
