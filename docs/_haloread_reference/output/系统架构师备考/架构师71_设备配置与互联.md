## 概念

### 这章考什么

- **科目**：中级网络工程师案例分析（重点）、综合知识
- **考频**：必考，配置命令是案例题主体
- **分值**：中级约 10-15 分
- **难度**：中等，命令要熟练，ACL/NAT 是常考点

案例题一半分值在配置命令。思科 IOS 是默认参照，华为 VRP 命令对照即可。ACL、NAT、VLAN、OSPF 配置是必练项。

## 原理

### 【中级重点】路由器基本配置

思科 IOS 模式：用户模式 > 特权模式（enable）> 全局配置模式（configure terminal）> 接口模式（interface）。

```
Router> enable
Router# configure terminal
Router(config)# hostname R1
R1(config)# enable secret cisco       ! 加密密码
R1(config)# line vty 0 4              ! 远程登录
R1(config-line)# transport input ssh
R1(config-line)# login local
R1(config)# username admin privilege 15 secret cisco
R1(config)# ip domain-name exam.com
R1(config)# crypto key generate rsa   ! 生成 SSH 密钥
```

接口配置：

```
R1(config)# interface GigabitEthernet0/0
R1(config-if)# ip address 192.168.1.1 255.255.255.0
R1(config-if)# description To-LAN
R1(config-if)# no shutdown
```

### 【中级重点】静态路由

```
R1(config)# ip route 192.168.2.0 255.255.255.0 10.0.0.2
! 目的网络 掩码 下一跳
R1(config)# ip route 0.0.0.0 0.0.0.0 10.0.0.2          ! 默认路由
R1(config)# ip route 192.168.3.0 255.255.255.0 10.0.0.2 200  ! 浮动静态，管理距离 200
```

### 【中级重点】OSPF 配置

```
R1(config)# router ospf 1
R1(config-router)# router-id 1.1.1.1
R1(config-router)# network 192.168.1.0 0.0.0.255 area 0
R1(config-router)# network 10.0.0.0 0.0.0.3 area 1
R1(config-router)# passive-interface default
R1(config-router)# no passive-interface GigabitEthernet0/0
R1# show ip ospf neighbor
R1# show ip route ospf
```

反掩码 wildmask 是掩码按位取反：/24 掩码 255.255.255.0，反掩码 0.0.0.255。

### 【中级重点】ACL 访问控制列表

| 类型 | 范围 | 匹配条件 |
|---|---|---|
| 标准 ACL | 1-99、1300-1999 | 只看源 IP |
| 扩展 ACL | 100-199、2000-2699 | 源目 IP、协议、端口 |
| 命名 ACL | 名字 | 灵活 |

标准 ACL 放靠近目的，扩展 ACL 放靠近源，减少无用流量。

```
R1(config)# access-list 10 deny 192.168.1.0 0.0.0.255
R1(config)# access-list 10 permit any
R1(config)# interface GigabitEthernet0/0
R1(config-if)# ip access-group 10 in

R1(config)# ip access-list extended WEB
R1(config-ext-nacl)# permit tcp 192.168.1.0 0.0.0.255 any eq 80
R1(config-ext-nacl)# permit tcp 192.168.1.0 0.0.0.255 any eq 443
R1(config-ext-nacl)# deny ip any any
```

ACL 末尾隐含 deny any，记得显式 permit。

### 【中级重点】NAT

| 类型 | 说明 |
|---|---|
| 静态 NAT | 一对一，公网 IP 直接映射 |
| 动态 NAT | 一对一，从地址池取 |
| PAT/NAPT | 多对一，端口多路复用，最常用 |

```
R1(config)# access-list 1 permit 192.168.1.0 0.0.0.255
R1(config)# ip nat inside source list 1 interface GigabitEthernet0/1 overload
R1(config)# interface GigabitEthernet0/0
R1(config-if)# ip nat inside
R1(config)# interface GigabitEthernet0/1
R1(config-if)# ip nat outside
```

overload 关键字启用 PAT。

### 【中级重点】交换机配置

VLAN：

```
SW1(config)# vlan 10
SW1(config-vlan)# name SALES
SW1(config)# interface FastEthernet0/1
SW1(config-if)# switchport mode access
SW1(config-if)# switchport access vlan 10
SW1(config)# interface GigabitEthernet0/1
SW1(config-if)# switchport trunk encapsulation dot1q
SW1(config-if)# switchport mode trunk
SW1(config-if)# switchport trunk allowed vlan 10,20,30
```

STP：

```
SW1(config)# spanning-tree vlan 10 priority 4096   ! 优先级必须是 4096 倍数
SW1(config)# spanning-tree mode rapid-pvst
```

端口安全：

```
SW1(config-if)# switchport port-security
SW1(config-if)# switchport port-security maximum 2
SW1(config-if)# switchport port-security mac-address sticky
SW1(config-if)# switchport port-security violation restrict
! protect 丢包不告警、restrict 丢包告警、shutdown 关端口
```

链路聚合 LACP：

```
SW1(config)# interface range GigabitEthernet0/1-2
SW1(config-if-range)# channel-group 1 mode active
SW1(config-if-range)# channel-protocol lacp
SW1(config)# interface Port-channel1
SW1(config-if)# switchport mode trunk
```

### 【中级重点】防火墙配置

安全域：trust 内网、untrust 外网、dmz 隔离区。安全策略五元组：源域、目的域、源地址、目的地址、服务，动作 permit/deny。NAT 在策略外单独配。

### 【中级重点】VPN

#### IPsec

两阶段：

- **IKE 阶段 1**：主模式（6 条消息）或野蛮模式（3 条消息），建立管理连接 SA，协商加密算法、认证算法、DH 组、认证方式
- **IKE 阶段 2**：快速模式（3 条消息），建立数据连接 SA，协商 ESP/AH、加密、认证、传输模式/隧道模式

ESP 协议号 50 加密+认证，AH 协议号 51 只认证不加密。传输模式只保护 payload，隧道模式保护整个原 IP 包并加新头，站点到站点用隧道模式。

```
R1(config)# crypto isakmp policy 10
R1(config-isakmp)# encryption aes 256
R1(config-isakmp)# hash sha256
R1(config-isakmp)# authentication pre-share
R1(config-isakmp)# group 14
R1(config)# crypto isakmp key cisco address 10.0.0.2
R1(config)# crypto ipsec transform-set TS esp-aes esp-sha256-hmac
R1(config)# crypto map MAP 10 ipsec-isakmp
R1(config-crypto-map)# set peer 10.0.0.2
R1(config-crypto-map)# set transform-set TS
R1(config-crypto-map)# match address VPN-ACL
```

#### SSL VPN

基于浏览器，无需客户端，适合远程办公接入。分无客户端（Web 资源）、瘦客户端（端口转发）、厚客户端（AnyConnect 全隧道）三种模式。

#### GRE

通用路由封装 Generic Routing Encapsulation，隧道协议，把任意网络层协议封装进 IP。GRE 本身不加密，常与 IPsec 结合：GRE over IPsec 既支持组播（路由协议）又加密。

## 实践

### 【高级衔接】架构师网络设备选型清单

防火墙吞吐量计算：业务峰值流量 × 安全策略开销系数（约 1.5）× IPS/AV 开销系数（再 1.5-2）。某企业峰值 2Gbps，启用 IPS 选 6Gbps 以上防火墙。还要看并发连接数（百万级起）、新建连接速率（万 cps 起）。

交换机背板带宽 = 端口数 × 端口速率 × 2（全双工）。48 口万兆交换机背板至少 48×10×2=960Gbps。包转发率看小包 64 字节，64B at 10Gbps = 14.88Mpps。

BGP 多线接入：机房接多家运营商，用 AS_PATH 和 Local_PREF 调优回程，地理就近选路。

SD-WAN 软件定义广域网：控制平面集中、数据平面分布、链路动态选路、应用感知。多分支企业替代传统专线，互联网+MPLS 混合链路智能调度。

**真题示例（2024年·案例分析·NAT配置相关·考生回忆）**：内网 192.168.1.0/24 通过路由器外网接口 200.1.1.1 上网，写出 PAT 配置。答：

```
access-list 1 permit 192.168.1.0 0.0.0.255
ip nat inside source list 1 interface GigabitEthernet0/1 overload
interface GigabitEthernet0/0
 ip nat inside
interface GigabitEthernet0/1
 ip nat outside
```

## 速查/自测

### 常用思科命令清单

| 操作 | 命令 |
|---|---|
| 进特权 | enable |
| 进全局 | configure terminal |
| 改名 | hostname R1 |
| 接口配 IP | ip address x.x.x.x mask |
| 静态路由 | ip route net mask next-hop |
| OSPF | router ospf 1 |
| 查看 OSPF 邻居 | show ip ospf neighbor |
| 查看路由表 | show ip route |
| 查看接口 | show ip interface brief |
| 保存 | write memory |

### ACL 两类对比

| 类型 | 编号 | 匹配 | 部署位置 |
|---|---|---|---|
| 标准 | 1-99 | 源 IP | 靠近目的 |
| 扩展 | 100-199 | 五元组 | 靠近源 |

### NAT 三种类型

静态一对一、动态池、PAT 端口多路复用。

### IPsec 两阶段

阶段 1 主模式建管理 SA，阶段 2 快速模式建数据 SA。

### 自测题

1. OSPF network 命令 192.168.1.0 0.0.0.255 area 0 中反掩码是什么？答：0.0.0.255（掩码 255.255.255.0 按位取反）。
2. ACL 末尾隐含什么？答：deny any。
3. IPsec 隧道模式保护什么？答：整个原 IP 包，加新 IP 头。

一句话小结：命令是案例题命根，ACL 靠源放目的、扩展靠源放源头，NAT 记 PAT 最常用，IPsec 两阶段别混。
