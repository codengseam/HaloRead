# 称谓与见面

## 一、入戏

成都玉林路的一家老茶馆里，外地来的小张进门就喊：“老板，来碗茶。”堂倌头也不抬，旁边一位大爷笑着替他纠正：“小伙子，在成都要喊老师，喊老板人家当你是买铺子的。”小张愣了一下，怯生生改口：“老师，来碗茶。”堂倌立马笑开了：“要得，三花还是竹叶青？”

一个称谓换过来，关系的温度就变了。

同样一幕放到重庆朝天门码头边的苍蝇馆子，老板娘听见你喊“老板娘”，多半会笑着回一句：“莫喊老板娘，喊姐！老得很咯。”重庆人不爱绕弯子，称谓来得快、来得直，几分江湖气都压在一句“哥”“姐”“兄弟”里。

称谓这一关，是中国人交往的第一道关口。还没开口说话，先看你“会不会喊人”。喊对了，后面的事顺理成章；喊错了，再怎么赔笑脸都拧着。礼这套东西，说到底就是教人怎么“开口”——而称谓，是开口的第一句。

## 二、破题

称谓这一章要讲的“道”，是五个字：**就高不就低，就尊不就卑**。

金正昆在《金正昆谈礼仪》里反复讲一个例子：一位教授来访，你对外介绍他，要称“教授”，不能称“老师”。表面看两者都是尊称，可在中国人的语境里，“教授”是职衔、是体制内的位置，“老师”是泛称、是谁都能当的。把教授降格成老师，看似谦虚，实则是把对方的位置压低了一截。

这就是“就高不就低”——同样的身份，挑那个更体面、更准确、更高一档的称谓来用。

为什么要“就高”？因为称谓不是简单的代号，它是关系网的定位系统。中文里“您”和“你”分开，“伯伯”“叔叔”“舅舅”分得清清楚楚，“张总”“李局”“王主任”一个职务一个叫法——每一声称呼，都在确认对方在你心里的位置。彭林在《彭林说礼》第二集“学会称呼”里讲得更直接：称谓是礼的开端，称呼错，后面全错。

这一点和西方很不一样。英文里“you”一个词通吃上下，“uncle”“aunt”不分父系母系，朋友之间直呼其名是亲切，职场里 John、Mary 叫得欢。这是契约社会、个体本位的底色——人和人之间默认平等，称谓不必承担定尊卑的任务。

中国不一样。中国是关系本位、差序格局（费孝通语），称谓承担的是“序”的功能——长幼、尊卑、亲疏，全靠称谓来标定。所以中国人学礼，第一课不是别的，就是“喊人”。

见面礼也是同样的逻辑。握手、点头、拱手、鞠躬，动作本身不重要，重要的是谁先伸手、谁先点头、动作做到几分。金正昆讲握手有一条铁律：地位高者先伸手。这一条背后，还是那个“序”字。

## 三、方法论

### 第一步：称谓三系统——辈分、职务、泛称

中国人喊人，逃不开三套系统。

**辈分系统**——叔伯姑舅、爷爷奶奶、哥哥姐姐。这套是血缘和准血缘的称谓，讲究精准。彭林在《彭林说礼》里特别提醒：父系母系不能混，舅舅不是叔伯，姑父不是姨父。农村里这套还严，城市里松了一些，但基本规则没变——辈分高一级，称谓就得跟着升一级。

**职务系统**——张总、李局、王主任、刘校。这是职场和官场的称谓，讲究“就高不就低”。金正昆举例：一位副总主持工作，对外介绍时称“总”不称“副”，这是给面子，也是给位置。反过来，把正的喊成副的，是得罪人。

**泛称系统**——先生、女士、老师、师傅、同志。这是陌生人之间的礼节性称谓。这套最有地域特色，也最容易出错。

三套系统怎么选？基本原则是：**有职务称职务，没职务称辈分，两者都无称泛称，泛称就高不就低**。

举个实战例子。你陪一位长辈去对方单位拜访，对方是处长，但你不知道。进门第一句，宁可称“领导”也别称“师傅”——“领导”是就高，“师傅”在城里有时带蓝领意味，就低了。

### 第二步：握手规矩——谁先伸手，谁后伸手

握手看似简单，金正昆讲得极细，有几条铁律要记牢。

**第一，用右手**。左手在许多文化里是不洁的，中国虽没那么严格，但右手是默认。

**第二，地位高者先伸手**。下级见上级，等上级伸手；晚辈见长辈，等长辈伸手。你先伸手而对方没接，那一秒钟的尴尬，比说错话还难受。

**第三，主客有别**。客人来访，主人先伸手，表示欢迎；客人告辞，客人先伸手，表示“不用再送”。这一条最常被搞反——主人陪客人到门口，主人伸手要握，客人却先伸手，结果两个人都伸手又都缩回去，场面一度很僵。

**第四，力度与时长**。不是越紧越好，也不是越久越亲。握住、用力一分、停两秒、自然松开，刚好。掌心出汗、死死不放、敷衍一碰——都失礼。

### 第三步：介绍顺序——先把谁介绍给谁

介绍也有规矩，金正昆讲得很清楚：**先把地位低的介绍给地位高的，先把年轻的介绍给年长的，先把客人介绍给主人**。

为什么这么排？因为地位高、年长、主人这一方，是被尊重的对象，需要先“知道”对方是谁，再有姿态地接受介绍。地位低的一方被先介绍，等于先报到、先亮相，符合“序”的逻辑。

介绍时还要注意称呼的“就高”——“王总，这位是我们公司的小李，李明”——这里“王总”是尊称，“小李”是谦称，一尊一谦，关系就摆正了。

### 第四步：名片三不准与递接规矩

金正昆有“名片三不准”：不能随意涂改、不提供私宅电话、不印两个以上头衔。

递名片要双手，正面朝对方、文字正向对方——对方接到即可读，不必再翻。接名片也要双手，接过来先看一眼，记住姓名职务，再郑重收进名片夹或口袋。看都不看就塞兜里，等于告诉对方“你这个人不重要”。

名片还有一个细节：现场不要在对方名片上写写画画。有的人习惯接到名片就当场记笔记，这在西方商务礼仪里尚可，在中国容易被解读为“瞧不起人”——仿佛对方的身份信息需要你重新整理才记得住。

### 第五步：川渝“老师”泛化的江湖底色

回到开篇那个茶馆场景。川渝人喊“老师”，是有来历的。

这个“老师”和教书无关，是市井之间的泛化尊称。一个不认识的人，你不知道他姓什么、做什么的，喊“先生”太正式，喊“师傅”嫌生分，喊“老师”刚好——既抬高对方一分，又不显得套近乎。

成都“老师”多用于真正的师生关系或文艺界，市井破冰用“哥老倌”“嬢嬢”更亲切——人和人之间不远不近，留三分余地。

重庆的“老师”才是真正的“万能破冰词”——菜市场、出租车、路边摊都喊“老师”，据中国新闻周刊调研，多数（普遍估计约七成）重庆人常用“老师”称呼陌生人。朝天门码头扛活的、解放碑做生意的，几句话就称兄道弟，这是码头文化的底子——快、直、讲义气，没那么多弯弯绕。

但要提醒一句：川渝“老师”的泛化也有副作用。在正式场合，把真正的教授、医生、领导一概喊“老师”，会显得失敬——你不知道对方的实际位置，就默认压低一级。所以“老师”是好用的市井泛称，但不是万能称谓。正式场合，还是回到职务或辈分系统更稳妥。

## 四、避坑

**坑一：直呼长辈姓名**

年轻人图省事，微信里直接“@张建华”，或者当面喊长辈名字。在老一辈听来，这是大忌。即便长辈说“别客气，叫我名字就行”，你也别真这么叫——客气话不能当真。该喊叔喊叔，该喊伯喊伯，这是底线。

**坑二：把“老公”“老婆”挂在嘴边**

彭林在公开讲座里多次批评“老公”“老婆”这对称呼。他认为“老公”在古代是太监的俗称，把丈夫叫“老公”不是与时俱进，而是退。这个观点学界有争议，但彭林的提醒有道理：称谓背后有文化，盲目跟风互联网用语，有时是把自己的文化根给刨了。在正式场合或长辈面前，“先生”“太太”“爱人”更得体。

**坑三：单手递名片、接名片不看**

职场新人常犯的错。单手递名片显得随意，接名片即收不看，等于把对方的人脸和名字一并忽略了。最严重的是把对方职务和姓记混了——下回见面把“张总”喊成“李总”，那一瞬间对方脸色就会变。

**坑四：握手时机错乱**

最常见的错是客人来访时客人先伸手——你以为这是主动示好，其实是抢了主人的礼。还有一种是隔着老远就伸手，让对方不得不快步上前接，这种“远距离握手”既尴尬又失礼。等距离近了、目光对上了、对方有伸手迹象了，再自然伸手。

## 五、践行

本周可以做的五件小事：

1. **盘点你最近一周喊错过的人**。回想有没有把职务喊低、把辈分喊错、把姓记混。列出来，下次见面先在心里默念一遍正确称谓。

2. **练习握手的主客之分**。下次去别人家或别人单位做客，进门等主人先伸手；告辞时你先伸手。试一次就记住了。

3. **名片整理**。把你收到的名片翻一遍，挑三张最常用的联系人，把姓名、职务、上次见面场合默背一遍。下次见面能直接喊出“张总，上次在您公司见面”，对方一定刮目相看。

4. **观察川渝“老师”的使用边界**。在成都和重庆分别待一天，记录“老师”出现在哪些场合、对哪些人。你会发现市井泛称和正式场合的称谓完全不是一套系统。

5. **写一张称谓小抄**。把你常打交道的十个人——领导、长辈、同事、客户——的“正确称谓”写下来。看似多此一举，关键时刻能救命。
