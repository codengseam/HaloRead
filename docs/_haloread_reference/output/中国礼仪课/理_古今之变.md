# 古今之变

## 一、入戏

假设朱熹穿越到今天重庆的一家老火锅店。

九宫格铁锅咕嘟咕嘟翻着红油，毛肚鸭肠在篓子里起伏，十来个人围炉而坐，啤酒瓶碰得叮当响。这位写过《家礼》、定下后世几百年婚丧嫁娶范式的老先生，会皱眉，还是会举杯？

我猜他先不举杯，先看三件事：座次是不是有序——长辈和主宾坐在哪里；敬酒是不是由尊及卑——晚辈先敬还是长辈先发；动筷是不是长辈先——晚辈会不会抢着夹第一筷子。如果这三条在，朱熹大概会捋捋胡子，端起面前的酒杯，对身边人说：「礼在，火锅亦是礼。」若三条都不在，那锅里烫的再是金贵的毛肚，对他而言也只是吃食，不是礼。

若把场景换到成都人民公园的鹤鸣茶社，朱熹大概会有另一种反应。竹椅一排排斜靠着，盖碗茶掺到八分满，老人们一下午不挪窝，桌上几碟瓜子花生，谁来了点点头、谁走了挥挥手，没有敬酒、没有座次、没有抢着夹第一筷子——但每一杯茶续到谁那里、谁先开口说什么话、谁的孩子该让谁带开，照样有一套不显眼的次序在跑。朱熹若到此处，大概也会说：礼在，茶馆亦是礼。

这个想象不是抬杠。它要说的只有一句：礼的形式可以全变，但礼的内核，三千年没怎么变。

## 二、破题

要讲清古今之变，得先把两个字拆开——「礼」和「仪」。

《礼记》里有一句被反复引用的话：「礼，时为大」——《礼记·礼器》。意思是礼最大的原则，是合乎时宜。时代变了，礼的形态就得跟着损益，可增可减，可改可废。但损的是「仪」，不是「礼」。

「仪」是形式——冠礼怎么戴帽、婚礼怎么六礼、丧礼怎么服三年、跪拜怎么磕、长袍马褂怎么穿、书信怎么落款。这些东西今天基本都改了：冠礼简化成十八岁成人礼，婚礼从「纳采、问名、纳吉、纳征、请期、亲迎」六礼简化成领证加婚宴，丧礼从三年之丧简化成三日告别，跪拜改鞠躬握手，长袍改西装中山装，竹简书信改微信表情包。

「礼」是内核——序、和、敬（“让”是“序”在具体场合的实操）。金正昆在《社交礼仪教程》里反复强调一句「礼者敬人也」，礼的本质是尊重；彭林在《彭林说礼》里讲「人之所以为人者，礼义也」，礼是人与禽兽的分界。这一层东西，从周公到孔子到朱熹到今天，没有变过。

变的是形，不变的是道。这就是古今之变的核心命题。

## 三、方法论

讲透古今之变，可以分四步走。

**第一步：识别形式之变。**

哪些是「仪」的更迭？这一层要诚实地承认变化，不要装作没变。称谓上，「老爷」「太太」「大人」退场，「同志」「先生」「女士」登场，近年又有「老公」「老婆」流行——彭林在公开讲座中批评这是「退」而不是「进」，因为「老公」在古时本指太监。场合上，从宗庙朝堂、宗祠家庙，转向职场、商务、互联网。工具上，从飞鸽传书到电话到微信视频。这些变化承认下来，心里才有底，不会把过时的「仪」当成永恒的「礼」去强求。

**第二步：抓住内核之常。**

形式变了，但「序、和、敬」这三个字还在（“让”作为“序”的实操，下文一并展开）。

敬——金正昆讲「礼者敬人也」，彭林讲「恭敬之心」，西方 Emily Post 也讲 Consideration 与 Respect，跨文化相通。

让——孔融让梨的故事今天还在讲，饭桌上长辈先动筷、晚辈后动筷，是「让」的活态。

序——长幼有序、尊卑有别，虽形式不再像古时那么严，但在家庭饭桌、职场会议、政务场合里仍存。圆桌也分主位，电梯也分先后。

和——「礼之用，和为贵」——《论语·学而》。礼的目标是让人和人之间处得舒服，不是制造隔阂。

这四个字落到川渝，长出的形还分两路。重庆的码头文化把这四字催得“硬”——袍哥茶碗摆阵、兄弟伙结义、码头工分账，敬是当面敬、让是抢着让、序是当场序、和是喝到一起和，节奏快、动作大、外显明显。成都的闲适文化把这四字养得“软”——茶馆里你谦我让、走人户不催不急、邻里间一个橘子一袋瓜子的礼数，敬是心里敬、让是嘴上让、序是默认序、和是相处舒服，节奏缓、动作小、外隐深。两路形不同，序、和、敬的道却一字不差——这是古今之变在川渝两地各自活出来的样本。

抓住这四个字，就抓住了「礼」之所以为礼的根。形式可以全换，这四个字不能丢。

**第三步：主动适应新场景。**

古今之变最大的几个转折点要心里有数。春秋「礼坏乐崩」，孔子重塑礼的精神；五四新文化运动「打倒孔家店」，礼治被混同于旧礼教一并抛弃，差不多一个世纪；建国后五六十到七十年代，阶级斗争为纲，传统价值遭激烈批判；改革开放后物质欲望释放，礼仪被抛一边；互联网时代去中心化，人人成为自主体，情绪化思维上升。学者姜义华在《论「礼治」的当代意义》（具体篇名待核实）中指出，礼治在百年里经历了战争革命、阶级斗争、GDP 挂帅、网络时代四大挑战，在浴火中重生。

每个新场景都要求礼主动调整形态。互联网时代不能要求人人写八行书，但「微信问事先问在不在」「群聊不深夜打扰」「朋友圈点赞不算真问候」这些新规矩，本质就是「仪」的更新。刘丰在《礼的损益与中华文明的连续性》（待核实）里讲，「变礼」是礼制发展的内在机制——能变的礼才有生命力，才不至被时代抛弃。

**第四步：损益有度——不复古，不虚无。**

这是最难的一步。汤勤福、葛金芳在《中华传统礼制内在价值及其现代转换》（待核实）中提出，传统礼制须经「现代性转换」——守望本土传统，又以现代性为标准审视、转换并汲取古典精义。礼仪学者范智用一句话概括为「传统礼仪现代化，西方礼仪中国化，中西礼仪共荣化，现代礼仪全球化」。

落到操作上：不强行恢复跪拜，但保留鞠躬时的庄重；不要求婚礼走六礼，但保留「纳征」背后那份郑重——彩礼不是炫富，是定约；不让丧礼回到三年之丧，但保留三日告别里那份「临丧毋苟免」的真诚。礼可以损，可以益，但不可废。

## 四、避坑

讲古今之变，最容易踩两个坑。

**第一个坑：复古派。** 觉得古礼都好，恨不得全盘恢复——重新穿汉服行冠礼，重新行跪拜，重新走六礼。这种思路错在把「仪」当成了「礼」。汉服当然可以穿，但穿上汉服不等于懂礼；行跪拜而不懂为何跪，比不行更糟。古人讲「礼，时为大」，连古人都知道礼要随时损益，今天的人何必刻舟求剑。

**第二个坑：虚无派。** 觉得古礼全过时，把「礼教吃人」泛化——一提传统礼仪就贴标签「封建糟粕」「父权压迫」。这种思路错在把「仪」的过时当成了「礼」的过时。跪拜过时了，不代表「敬」过时；三年之丧过时了，不代表「慎终追远」过时。鲁迅批评的是吃人的礼教，不是敬与和本身。

两个坑的共性都是分不清「礼」与「仪」。复古派把仪当礼，虚无派把仪的过时当成礼的过时。分清这两个字，古今之变就有了准星。

## 五、践行

本周可以做的几件事：

一、列出你家里至今还在用的三条老规矩。比如长辈先动筷、年节不说不吉利话、红白事要随份子。问问自己：这些规矩守的是什么？是敬，是序，是和？

二、列出三条你或你的家庭已经放弃的老规矩，并写下放弃的理由。比如不再磕头拜年、不再守三年之丧、不再男尊女卑。判断：放弃的是「仪」还是「礼」？若只是仪，没问题；若连敬也丢了，要警惕。

三、下一次在饭桌上，主动观察一顿饭里的「序」——谁先坐、谁先动筷、谁先敬酒、谁最后吃。看看这顿饭的「礼」还在不在，跟形式新旧无关。

四、翻一翻《礼记·曲礼》前几段，挑一句你能背下来、且本周能用上的——比如「临财毋苟得，临难毋苟免」。在某个具体场景里试一次，感受古礼的内核如何在你身上还活着。

五、想一想你正在用的新工具（微信、视频、表情包），哪些是承载了「敬」的新形态，哪些只是敷衍流程。把前者留下，后者诚实面对。

六、挑一件你在重庆朋友家和成都朋友家分别经历过、却做法不同的事（敬酒方式、走人户礼物、邻里相处皆可），写下来对比：形差在哪？道同在哪？这一对比，“形可损益、道需承续”就不再是书本上的话，是你身上的活经验。
