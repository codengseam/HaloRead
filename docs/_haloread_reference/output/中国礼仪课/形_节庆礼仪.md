# 节庆礼仪

## 一、入戏

除夕夜，成都的小李家族群里此起彼伏地弹出一长串“新年快乐”表情包——卡通金元宝、跳动的大红字、转场的烟花动画，一个比一个热闹。小李的妈妈在群里默默发了一个 200 块的红包，附一句“一家人平平安安”。三秒钟被抢光，几个侄子侄女回了句“谢谢嬢嬢”，再无下文。

同一时间，重庆磁器口的庙会正人山人海。老陈陪八十岁的母亲去逛了一圈，给母亲买了一串糖油果子、一挂红灯笼，回家路上母亲笑着说：“今年灯会好看，明年还来。”老陈心里一暖——这一句话，比群里几百条表情包都重。

过年的“年味儿”是怎么淡的？多半是从群发表情包、群抢红包开始的。礼的形式简化到一定程度，礼意就稀释了。这一章就讲节庆里的礼——怎么过、怎么走、怎么敬，才能让“年味儿”回来。

## 二、破题

节庆这一章要讲的“道”，是五个字：**礼的时间化**。

什么意思？节庆是礼的年度演练。春节团圆、清明祭祖、端午驱邪、中秋赏月、重阳敬老——每一个节日，都是把“敬天法祖”“长幼有序”“团圆和睦”的礼义，年度性地重演一遍。

《礼记·月令》把一岁四时的礼都安排得明明白白：孟春之月「迎春于东郊」，仲夏之月「命乐师修鼗鞞鼓」，季秋之月「享帝于明堂」。古人的节庆不是放假，是行礼——通过仪式把“敬”刻进时间的循环里。

彭林在《人文清华讲坛·人何以为人》里讲「礼乐皆得，谓之有德」，节庆正是礼乐的年度重演。春节拜年、清明扫墓、中秋赏月、重阳登高，每一个仪式都在强化“我们是谁、从哪来、向哪去”。

节庆还有一层社会功能：**人情维护的年度续约**。中国是关系本位社会，关系网需要持续维护，否则就疏了。节庆就是关系网的“年度续约日”——春节走亲戚、中秋送月饼、清明一起扫墓，都是在续人情的约。一年不走动，关系就淡一截；三年不走动，关系就断了。

所以节庆不是放假，是行礼；不是休息，是续约。把节庆当假期过，就丢了节庆的礼意；把节庆当礼过，节庆才有味。

## 三、方法论

### 第一步：春节——拜年、压岁钱、走动

春节是节庆之王，规矩也最多。

**拜年顺序**——由亲及疏、由长及幼。初一拜本家长辈，初二回娘家（女婿拜岳家），初三起拜亲友。顺序乱了等于乱辈分，是大忌。

**拜年方式**——当面拜年为上，电话拜年次之，微信拜年又次之。当面拜年要磕头（部分地区）或鞠躬，要说吉利话（“新年好”“身体健康”“万事如意”）。微信拜年要带称呼、不群发——群发无称呼的表情包，是最廉价之礼，不如不发。

**压岁钱**——长辈给晚辈，红包金额按年龄和亲疏定。给自家孩子百元起步，给亲友孩子几十到几百不等。压岁钱的礼义是“压祟”——压住邪祟，保孩子平安，不在金额，在心意。

**走动**——春节走动是关系网的年度续约。该走的亲友一定要走，该带的礼物一定要带（年货、糖果、茶叶）。走动不只在拜年当天，整个正月都可以“补走”。该走的不走，等于单方面断约。

### 第二步：清明——祭祖扫墓，敬天法祖

清明是祭祖之节，礼义最重。

**扫墓**——亲自到墓前扫除、献花、焚香、祭酒。不能到场的，可视频参与或在家设位遥祭。扫墓时要肃穆，不嬉笑、不大声、不晒游玩照（与祭祖氛围冲突）。

**敬天法祖**——清明不只是怀念逝者，更是确认“我从哪里来”。一家人同去扫墓，是给晚辈上“家族根”的一课。父母带着孩子讲祖辈的故事，比任何家训都管用。

**清明禁忌**——清明前后不说不吉利话，不穿艳色，不办喜事（部分地区）。扫墓后可踏青，但踏青照不宜晒朋友圈与祭祖照并列。

### 第三步：端午——驱邪、裹粽、赛龙舟

端午的礼义是“驱邪避瘟”。

**裹粽送粽**——家家裹粽子，邻里互送。送粽是轻礼，重在“我记着你”。端午送粽不必贵重，自家裹的几个，比超市的礼盒更显人情。

**赛龙舟**——集体性仪式，体现“和”的礼义。川渝部分地区有龙舟赛，沿江村镇更盛。

**挂艾草菖蒲**——门上挂艾草、菖蒲，驱邪避瘟。这是古礼的遗存，今天仍在农村延续。

### 第四步：中秋——团圆、月饼、赏月

中秋是团圆之节。

**团圆**——一家人围坐赏月、吃月饼。在外不能归的，要打电话、视频，“天涯共此时”。

**月饼相赠**——中秋送月饼是惯例。金正昆讲“年节的礼物重在传递爱心和情感”，月饼就是这份情感的载体。送月饼不必追求天价，合适就好。

**赏月**——一家人赏月、聊天、吃月饼。这是中秋的“仪式时间”，比月饼本身更重要。

### 第五步：重阳——敬老、登高、赏菊

重阳是敬老之节。

**敬老**——重阳看望长辈，送寿礼（寿桃、寿面、菊花酒），陪长辈登高、赏菊。这是“老吾老以及人之老”的年度演练。

**登高**——重阳登高是古俗，“避灾”之意。今人可陪长辈爬山、逛公园，意在陪伴。

**茶馆敬老**——川渝茶馆在重阳节常有“免费请老人喝茶”的活动，是本土敬老的特色。一碗几块钱的盖碗茶，请老人坐一下午，比送贵礼更暖。

### 第六步：川渝庙会、灯会与茶馆敬老

回到开头那两个场景。川渝节庆有独特烟火气，是中式节庆礼仪里最具地方特色的一种。

**春节逛庙会**——成都有武侯祠大庙会，重庆有磁器口庙会。庙会不只是逛街，是“看戏、吃小吃、买年货、会亲友”的复合仪式。一家人逛庙会，比群里抢红包更像过年。

**元宵看灯会**——自贡灯会天下闻名，是川渝节庆的高光。灯会不只是看灯，是“夜晚集体出动、共赏繁华”的礼——古之上元观灯遗意。

**清明祭祖后吃火锅**——川渝人扫墓归来，一家人吃顿火锅“团聚”。看似不搭，实则是“祭祖之后回归生者”的本土诠释——先敬逝者，再聚生者，一冷一热，一肃一闹，礼意全在。

**中秋椒盐月饼**——川渝确有椒盐月饼，虽“麻辣月饼”更多是网络段子。川渝月饼偏咸偏酥，与广式甜月饼不同，是本地口味。

**重阳茶馆敬老**——川渝茶馆重阳免费请老人喝茶，是本土敬老的特色。盖碗茶三才（茶盖为天、茶碗为人、茶船为地）暗合“天地人”哲学，请老人喝茶，是把老人置于“天地人”之中最尊的位置。

**成都和重庆的差别**——成都节庆更“绵”，庙会灯会节奏慢，一逛一天；重庆节庆更“烈”，磁器口人挤人，热热闹闹挤一天收场。一个“绵”，一个“烈”，是两种节庆气。

去浪漫化地说一句：川渝节庆的烟火气也有它的局限。庙会灯会人山人海，有时失于喧闹，少了节庆本该有的“敬”。清明吃火锅虽是本土智慧，但若只顾吃不顾祭，就丢了清明的礼意。节庆的礼，不在繁文缛节，也不在喧闹吃喝，而在“一家人坐下来吃顿饭、敬一碗茶、说一句吉利话”的本真——这个本真，需要仪式感来托住。

## 四、避坑

**坑一：群发祝福无称呼**

群发“新年快乐”给几百人，没称呼、没定制，是最廉价之礼。收到的人知道你是群发，回也不是不回也不是。不如不发，或不如挑十位最重要的亲友，逐一手写拜年信息。

**坑二：春节说不吉利话**

春节忌讳“死”“破”“碎”“没”等字。打碎碗要说“碎碎（岁岁）平安”，不说“完了”。这不是迷信，是节庆的礼意——年度之始，给一年定个吉利的调。

**坑三：清明晒游玩照**

清明扫墓与踏青本是一事，但晒游玩照到朋友圈，与祭祖氛围冲突。亲友看到你在玩，会想“你都没去扫墓吗”。清明朋友圈宜静，不宜晒。

**坑四：重阳只发朋友圈不真敬老**

重阳节发个朋友圈“祝天下老人健康长寿”，却不给自己父母打电话、不陪长辈登高，是“表演敬老”。敬老不在朋友圈，在一通电话、一次登门、一碗茶。

## 五、践行

本周可以做的五件小事：

1. **今年春节做两件事**——①给三位长辈逐一手写拜年信息（非群发）；②给一位久未走动的亲友打个电话拜年。试一次就知道，手写的分量比群发重多少。

2. **清明亲自扫一次墓**。如果祖坟在本地，亲自去扫；如果在外地，视频参与或在家设位遥祭。感受“敬天法祖”的礼义。

3. **中秋不送天价月饼**。挑一盒合适价位的月饼，附一张手写卡片，送给一位长辈。礼轻情意重，是真的重。

4. **重阳陪长辈登高或喝茶**。不发了朋友圈就算，而是真陪长辈爬一次山、逛一次公园、喝一碗茶。陪伴是最好的敬老礼。

5. **盘点你的“年度续约”清单**。列出一年里该走动的亲友名单——春节走几家、中秋走几家、清明一起扫墓的有几家。该走的走起来，关系网才不疏。
