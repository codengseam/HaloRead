# 饭局与座次

## 一、入戏

成都宽窄巷子旁一家老川菜馆，外地来的老李做东请几位本地朋友。菜上齐了，老李端起酒杯：“来来来，我先敬大家一杯，自罚三杯开场。”对面老周笑着按住他的手：“李哥，在成都莫搞这一套，慢慢来，先吃菜。”老李一愣——他在山东待了二十年，“三杯开场”是规矩，怎么到成都就不灵了？

同一幕换到重庆解放碑的江湖菜馆，画风又变了。主人老陈端杯就喊：“兄弟伙，走一个！”满桌跟着举杯，但下一位想敬酒的，没人硬逼别人喝完。你想喝就喝，不想喝抿一口也行，“喝舒服比喝醉酒重要”是默认共识。

南北饭局的差别，就在这一杯酒里。北方的“感情深一口闷”到江浙变成“侬随意呀”，到广东变成“你随意啦”，到了川渝又变成“喝舒服就行”。同一个“敬”字，载体完全不同。

饭局这一章，就讲这套载体背后的“道”。

## 二、破题

饭局这一章要讲的“道”，是五个字：**饭局即礼场**。

中国人谈生意、结朋友、办事情，多半不在办公室，而在饭桌上。一桌饭吃下来，关系的深浅、人情的厚薄、格局的大小，全在里面。所以饭局不是吃饭，是行礼。

饭局的礼义闭环有四个环节：**座次定序、敬酒见敬、劝酒见情、买单见让**。

- 座次——序的空间化（上一章已详谈，这里只点到为止）。
- 敬酒——敬的外化。谁先敬、敬谁、怎么敬，每一步都在确认尊卑亲疏。
- 劝酒——情的载体。劝到几分、怎么劝、劝不动怎么办，体现的是主人对客人的体谅还是裹挟。
- 买单——让的演练。抢买单不是吝啬反向，是“让”的礼在中国人情社会里的极致表达。

金正昆在《金正昆谈礼仪》里有一句很要紧的话：“关心有度，关心过度是一种伤害。”这句话放在饭局上最贴切——劝酒劝到对方难受，关心就变成了伤害；点菜点得让对方没得吃（不顾忌口），热情就变成了负担。

彭林讲《礼记·乡饮酒义》时也提过，古之乡饮酒礼，主人迎宾、揖让、升堂、献酬，每一步都有讲究，目的是“明尊长、睦乡里”。今天的饭局看似随意，骨子里还是这套——序、敬、情、让，一个不少。

## 三、方法论

### 第一步：点菜——数量、荤素、忌口

点菜是饭局的第一关，点得好不好，直接定了这一局的格调。

**数量**——基本规则是“人数+2”或“人数+1道凉菜+1道汤”。八个人就点十道菜，太多浪费，太少寒酸。要留余，不要正好。

**荤素搭配**——一桌菜里荤素比例大约七三或六四，看客人身份。商务场合荤多素少显隆重，朋友聚会荤素平衡更舒服。一定要有一道“硬菜”压桌——一条整鱼、一只整鸡、一份烤鸭，这是面子；再配几道下饭菜，这是里子。

**忌口**——这是最容易被忽视、却最关键的一条。点菜前一定要先问客人有没有忌口：宗教忌（清真、素食）、健康忌（痛风不喝汤、糖尿病少甜）、个人忌（不吃辣、不吃香菜、不吃内脏）。问忌口不是多事，是基本的尊重。

川渝饭局点菜还有一个本地规矩：必点一道“烫菜”（火锅或冒菜类）和一道“凉菜”（凉拌鸡、夫妻肺片之类）。烫菜是热络，凉菜是开胃，一热一凉，节奏就出来了。

### 第二步：敬酒——顺序、分寸、姿势

敬酒是饭局的高潮，规矩也最细。

**顺序**——由尊及卑、由长及幼。主人先敬主宾，再敬副宾，依次往下；晚辈敬长辈，要等长辈先敬完主人再开始。一桌人乱敬一气，等于把序搞乱了。

**分寸**——敬酒不劝酒。金正昆讲“交往以对方为中心”，敬酒时也是这个原则：你敬你的，对方喝多少由对方定。强迫对方喝完，是把“敬”变成了“压”。

**姿势**——晚辈敬长辈，酒杯要低于对方杯口，双手举杯；下属敬领导同理。碰杯时让自己的杯沿低于对方杯沿半寸，是“就低”的礼。如果是大圆桌够不着，用杯底轻磕桌面代替碰杯，叫“桌面碰杯”。

**敬酒词**——敬酒要说话。不说一句“我敬您”就干杯，显得敷衍。敬酒词要有：称谓+一句由头+一句祝愿。比如“王总，感谢您这次的支持，我敬您一杯”——三要素齐全，得体。

### 第三步：劝酒——南北有别，入乡随俗

劝酒是饭局最见性情、也最容易出事的环节。

**北方劝酒**——山东“三杯不过岗”开场，主陪副陪分工明确，敬酒顺序一丝不苟；东北“感情深一口闷”，称兄道弟显江湖；内蒙古银碗斟满烈酒，不喝即不给情面。北方劝酒的背后是“以酒载情”，喝得多等于情谊深，这是北方的礼。

**南方劝酒**——广东“你随意啦”高频，少有强硬劝酒；江浙“侬随意呀”温和，浅斟低吟，量力而饮；南方讲究“小酌怡情，大饮伤身”。南方劝酒的背后是“以茶载道”，喝得舒服比喝得多重要。

**川渝劝酒**——川渝是南北中介。火锅沸腾配白酒满杯，比广东热烈；但不流行山东式硬劝，讲究“喝舒服比喝醉酒重要”。川渝人会说“兄弟，整一个”，但你说“今天真喝不动了”，他不会硬逼——这是码头文化的底子，认“自愿”。

金正昆那句“关心有度，关心过度是一种伤害”，放在劝酒上最贴切。北方人去南方做客，硬劝南方客人喝酒，是失礼；南方人去北方做客，一句“我不喝”扫了主人的兴，也是失礼。入乡随俗，是劝酒的总原则。

### 第四步：买单——抢买单的学问

买单这一关，最能看出一个人的格局。

**中式抢买单**——这是中国人情社会的特色。一桌饭吃完，几个人抢着买单，谁抢到谁有面子。抢买单不是吝啬的反向，而是“让”的礼——把“我付”的姿态做出来，让对方感到被尊重。

**抢买单的边界**——做东的应该抢得最凶，因为这是你的局；客人可以象征性抢一下，但不能真抢。如果客人抢到了，主人反而没面子。

**AA 制的兴起**——年轻人中间越来越流行 AA。这是从西方引入的，与传统“抢买单”形成张力。商务场合一般不 AA（请客方买单），朋友聚会 AA 渐多。AA 的好处是不欠人情债，坏处是少了“让”的礼意。

**川渝买单**——川渝人也会抢买单，但抢得没那么“凶”。“安逸”心态让买单少了几分面子战、多了几分人情味。常常是做东的提前把账结了，等大家吃完说“我已经结了”，大家笑骂一句“龟儿子又抢”——这事儿就过去了。比北方动辄在收银台推搡半天，川渝的买单更“轻”。

### 第五步：餐桌礼仪——细节见教养

饭局上还有一堆零碎规矩，看着不起眼，但都关乎教养。

**长辈先动筷**——菜上桌，长辈不动筷，晚辈不能先夹。这是“序”在餐桌上的落地。

**不翻鱼**——吃整鱼时，吃完一面不要翻过来，要把鱼骨剔掉再吃另一面。“翻鱼”谐音“翻船”，沿海地区尤其忌讳。

**不敲碗**——筷子敲碗是乞丐的动作，饭桌上大忌。

**夹菜不越盘**——伸手越过别人的盘子夹菜，是失礼。够不着的菜请人代夹，或转桌。

**布菜用公筷**——给客人布菜要用公筷，不能用自己的筷子。这一点过去不讲究，现在越来越被重视。

**等齐再吃**——Debrett's 也讲这一条：主人或女主人许可后方可开吃。中餐一般等主人说“大家动筷子吧”再开始。

## 四、避坑

**坑一：把北方拼酒强加于南方客人**

北方人做东，硬劝南方客人“感情深一口闷”，结果对方喝到难受、甚至喝出事。这是把“我的礼”强加给“你的身体”，本意是敬，结果是伤。金正昆说“交往以对方为中心”，劝酒也得以对方为准。

**坑二：把“你随意”当真随意**

南方主人说“你随意啦”，是给你留余地，但你不能真一口不喝。该举杯时要举杯，该抿一口时要抿一口，“随意”是不强求，不是不参与。把“随意”当“放弃”，是对主人好意的辜负。

**坑三：抢买单弄巧成拙**

两个人抢买单，抢到在收银台推搡、甚至弄坏东西，那是失态。抢买单的“抢”是姿态，不是真打架。做东的提前结账，是最体面的做法。

**坑四：翻鱼、敲碗、越盘夹菜**

这三条是饭桌大忌，犯了就显得没教养。尤其是在长辈或商务客人面前，一条都犯不得。

## 五、践行

本周可以做的五件小事：

1. **下次做东前画一张座次图，再列一张点菜清单**。清单上写明：人数、菜品数量、荤素比例、忌口问询结果。这是基本功。

2. **练习敬酒词**。下次饭局敬酒前，先在心里默念一遍“称谓+由头+祝愿”三要素，开口前组织好语言。不说“我敬您”就干杯，是最低标准。

3. **观察一次南北饭局差异**。如果你有机会跨南北做客，记录主人劝酒的方式、敬酒的顺序、买单的姿态。南北之别在这一桌饭上最集中。

4. **建立“忌口档案”**。把你常一起吃饭的亲友的忌口记下来——谁不吃辣、谁痛风、谁清真。下次点菜直接调出，对方一定感动。

5. **练一次“提前结账”**。下次做东，吃到一半借口上洗手间，悄悄把账结了。回来说“已经结了”，比在桌上抢半天体面得多。川渝式的“轻买单”，值得学。
