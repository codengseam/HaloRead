## 入戏

老陈在一家制造业工厂干了二十年，从流水线做到车间主任。他常想，自己这辈子大概就这样了——没本金创业，没技术做产品，没资源搞人脉，财富这趟车，他连站票都没买到。

三十五岁那年，他开始每月拿两千块买一只宽基指数基金。同事笑他："两千块能干啥？还不如买两箱好酒喝。"

老陈没理。十年后，那笔钱加上复利，已经是他一年的工资。更关键的是，这十年里他没多加一天班、没多投一分精力，账户却悄悄长出了一截。他突然意识到一件事：自己虽然没当上老板，但这十年里，他一直是全社会几千家上市公司里那些最赚钱公司的隐名股东——别人在替他赚钱。

老陈没看清的是，他用的那只小小的指数基金，本质上是一根杠杆。

## 破题

纳瓦尔把财富的杠杆分成四种：劳动力、资本、代码、媒体。前两种是老钱杠杆，后两种是新钱杠杆。问题在于，对大多数普通人来说，雇人要本钱要管理能力，写代码要技术，做媒体要内容能力——四根杠杆，哪根都不好拿。

资本杠杆听起来最遥远：要有钱才能用钱生钱。可指数基金干了一件事，它把资本杠杆的门槛压到了几乎为零。你不必认识任何投资人，不必凑一百万起投，不必看懂财报，每个月几百块就能买下整个市场的一个切片。你买的不是一只股票，是一整条河流——河流里漂着这个国家最能赚钱的一批公司。

这就是指数基金作为杠杆的真正含义：它放大的是"你和社会经济增长之间的连接"。你不擅长选股没关系，你不预测涨跌没关系，你只要押注"这个社会长期会往前走"这一件事。指数基金把这押注变成了任何人都能下的注。

约翰·博格半个世纪前推动的指数革命，本质不是发明了一种新基金，而是让普通人也能用上资本杠杆——让一个车间主任，也能搭上整个经济的整体增长。

## 方法论

把指数基金当杠杆来用，有三层要诀。

**第一层：选最宽的河，别选最热的溪。** 杠杆放大的是趋势，趋势越宽越稳，放大效果越持久。宽基指数（覆盖全市场或主流市场的那一类）就是最宽的河——它不挑行业、不赌风口，把整个经济的增长都装进来。热门行业指数像溪水，涨得猛也干得快，普通人很容易在溪流断水时被困住。用杠杆，先求稳，再求大。

**第二层：把成本压到最低，让杠杆不漏气。** 杠杆的效率取决于摩擦力。一只年费百分之一点五的基金和一只年费零点一的基金，差的不只是数字，是几十年后差出几倍的最终收益。博格在《共同基金常识》里反复讲：成本是长期投资唯一确定的敌人。你用杠杆是为了放大收益，不是为了养肥基金公司。同样跟踪一个指数，谁费率低选谁，这是用杠杆的第一条纪律。

**第三层：用定投把杠杆装上自动挡。** 资本杠杆最大的敌人不是市场下跌，而是你自己的手。一次性押注，你会被波动吓得中途跳车；分批定投，等于把"持续注入资本"这件难事，变成像交水电费一样的机械动作。孟岩在《投资第一课》里强调，普通人最大的优势不是信息、不是眼光，而是有持续的工资现金流——把工资变成定投，就是把现金流接到了资本杠杆上，让杠杆自己转。

这三层合起来，就是普通人用指数基金的完整公式：宽基 + 低成本 + 定投 + 长期拿住。听起来朴素，但这正是它能成为杠杆的原因——它不挑人，谁都能照着做。

## 避坑

第一坑：把行业指数当宽基来定投。行业指数波动极大，定投它等于在赌一个行业的未来。宽基是"赌整个人类往前走"，行业是"赌某个赛道赢"，难度天差地别。普通人用杠杆，先押大趋势，别押小赛道。

第二坑：看着涨了就加仓，看着跌了就停投。这恰恰把杠杆用反了——杠杆应该在便宜的时候多吸筹码，你却在贵的时候追加、便宜的时候撤退。定投的纪律一旦被情绪打断，杠杆就断了。

第三坑：频繁换基金。每换一次都有成本，每换一次都在赌"另一只更好"。格雷厄姆在《聪明的投资者》里早就说过，投资者最大的敌人往往是他自己。你用杠杆是为了省心，不是为了给自己加活。

第四坑：以为指数基金稳赚。它放大的是趋势，趋势长期向上，但中途会有数年的回撤。如果你三年内要用这笔钱，别碰它——杠杆需要时间发挥作用，时间不够，杠杆会反过来咬你。

## 践行

1. 本月发工资后，选一只覆盖主流市场的低成本宽基指数基金，设成自动定投，金额设成你三五年内不心疼的数。先让杠杆转起来，比纠结买哪只重要。

2. 把"宽基 + 低成本 + 长期"六个字写在你能看见的地方。下次想追热门、想止损、想换基金时，先看一眼这六个字。

3. 给这笔钱定一个最低期限：五年内不动。杠杆需要时间，你给它的每一分钟，它都在替你工作。
