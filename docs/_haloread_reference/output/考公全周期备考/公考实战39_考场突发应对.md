# 第39章：考场突发应对，遇到看不懂的题怎么办

考场突发=抢分底线。突发不可怕，可怕的是卡在突发里出不来，把后面能拿的分也赔进去。这章给每个突发场景配一套"3秒自救+30秒过渡"SOP，让你被绊一下能立刻爬起来继续跑。

## 概念

### 学情把脉：突发面前的两种崩

考场上遇到突发，常见两种崩法。一种是"卡死型"——遇到一道看不懂的题，脑子一懵，反复读题反复纠结，5分钟过去人还钉在这道题上，后面五道简单题没时间做。一种是"连锁崩型"——一道题没处理好，情绪垮了，后面越做越慌，会的也错，整场考试被一道题带崩。

两种崩法的共同病灶：没有预演过突发应对，临场靠本能反应，而本能反应往往是错的（死磕、慌乱、放弃整场）。突发应对不能靠临场发挥，得靠考前预演的肌肉记忆——遇到X，做Y，3秒切换。

### 本质透视：抢分底线思维

考场突发的本质是"局部损失不可避免，全局得分必须保住"。一道题不会、一次涂卡出错、一阵身体不适，都是局部损失。应对的唯一目标是把局部损失控制在局部，不让它扩散成全局崩盘。

打比方，飞机遇到气流颠簸，飞行员不会去研究"为什么有气流"，而是按SOP调整高度和姿态保持平稳飞行。考生遇到突发也一样——不追问"为什么这题这么难"，按SOP处理完立刻回到正常做题节奏。

3秒自救是"瞬时反射"（阻止损失扩大），30秒过渡是"恢复常态"（回到做题节奏）。两段式应对，缺一不可。下面五个高频突发场景，每个都配这两段。

## 原理

### 突发一：完全看不懂的题

**行测场景**：一道题读了两遍没看懂在问什么。

- **3秒自救**：标记题号，跳过，不纠缠。心里默念"跳"，手已经在翻下一题。
- **30秒过渡**：做两道后面的简单题找手感，确认自己"还能做题"（重建自信），然后继续正常节奏。这道题留到该模块结束若有余时再回头，没余时最后统一蒙。

**申论场景**：小题问的角度完全没思路。

- **3秒自救**：不空着，回材料找相关段落。申论小题答案90%在材料里，"看不懂题"往往是因为没回材料。
- **30秒过渡**：把材料里跟题目关键词相关的句子圈出来，提炼成要点写上去。哪怕不是满分答案，踩到材料关键词就有分。空着是零分，写材料相关内容至少拿踩分。

### 突发二：写不完怎么办

**申论大作文写不完**：剩20分钟大作文还差一半字数。

- **3秒自救**：停笔5秒，做4-7-8呼吸一次，压住慌乱。
- **30秒过渡**：立刻切到"保框架"模式——开头已有则保留，没有则写3句开头；三个分论点各写1句核心句（哪怕不展开）；结尾写2句收尾。框架完整、字数尽量往要求线靠，比纠结某一段展开重要。禁交白卷，半篇框架分远高于空白。

**行测写不完**：剩5分钟还有10题没做。

- **3秒自救**：停做新题，立刻涂卡确认已做的都涂了。
- **30秒过渡**：剩余未做题目全部统一蒙同一选项（习惯蒙C或B），蒙完核一次题号无空缺。抢分底线是"不空题"，蒙了就有25%期望。

### 突发三：涂卡出错

**纸卷涂卡串行**：发现涂错位（如第20题涂到了第21格）。

- **3秒自救**：停笔，不慌，举手示意监考员。
- **30秒过渡**：向监考员说明情况申请更换机读卡（多数考场有备用卡）。换卡后优先把已做答案重新涂上，再继续做题。若监考员不允许换卡，用橡皮擦干净重涂，宁可慢不可错位到底。

**机考填答出错**：误选了答案想改。

- **3秒自救**：直接重新点选正确选项（机考可改）。
- **30秒过渡**：确认选项已更新，继续下一题。机考的好处是可改，但注意分段提交后该模块锁死不能改——所以提交前一定核一遍。

### 突发四：身体突发

**腹泻**：

- **3秒自救**：举手示意监考员申请如厕（考场规则允许，需陪同）。
- **30秒过渡**：快去快回，回来先做2道简单题找回节奏，别想"刚才浪费时间"。预防：考前一天禁生冷油腻、禁不常吃的食物，考前早餐吃常吃的。

**头痛/头晕**：

- **3秒自救**：闭眼10秒，深呼吸3次，按揉太阳穴。
- **30秒过渡**：喝口水，先做不需要大量计算的模块（如言语）缓一缓，状态恢复再做资料分析。预防：考前保证睡眠，咖啡因别超量（过量反而心慌）。

**女性考生月经**：

- **3秒自救**：确认备用卫生用品在手（考前已备）。
- **30秒过渡**：必要时举手如厕处理，回来继续。预防：算准周期，提前用药物调理（遵医嘱），考场备好用品和止痛药（需确认允许携带）。

### 突发五：考场干扰

**邻座干扰**（抖腿、翻卷子很响）：

- **3秒自救**：戴耳塞（若允许）或用纸巾轻塞耳朵，不与邻座交流。
- **30秒过渡**：把注意力锚定在题干第一个词上，强迫自己读完这道题。干扰最大的是"注意力被吸引"，锚定在题目上就能屏蔽。

**噪音干扰**（考场外施工、车辆）：

- **3秒自救**：不抬头看，继续做题。
- **30秒过渡**：噪音持续就接受它存在，告诉自己"所有人都在同一噪音下考，不是只针对我"。专注度是可以训练的，模考时故意开点背景音练抗干扰。

## 实践

### 易错陷阱与避坑指南

1. **死磕看不懂的题**：单题超90秒还没思路必须跳。死磕是考场第一大敌。
2. **申论小题空着**：空着是零分，写材料相关词就有踩分。绝不空题。
3. **大作文交白卷**：半篇框架分远高于空白。再赶也要写满框架。
4. **涂卡不举手自己改**：自己涂改可能被判标记，出问题举手走官方流程。
5. **身体不适硬扛**：该如厕就举手，硬扛影响整场。预防永远优于应对。

### 最新考情与命题趋势

机考普及后，部分突发场景减少（如涂卡串行），但新增了机考特有的突发：电脑死机、断网、输入法卡顿。

- **电脑死机**：3秒自救是举手示意监考员，机考系统通常有自动保存，重启后数据不丢。30秒过渡是确认恢复后的题号进度，继续做。
- **输入法卡顿**：申论打字时输入法突然切不出。3秒自救是切换输入法或用系统自带输入法。30秒过渡是确认能正常打字后继续，不反复调试。

> ⚠️ 考情差异提示：机考地区突发应对以"举手找监考员"为第一原则，监考员有标准处理流程；纸卷考试涂卡问题需自行小心。考前务必参加机考模拟（若有），熟悉设备故障的应对。

命题趋势上，机考分段提交让"时间到点强制提交"成为新突发源——某模块没做完就被强制跳转。应对是每模块内即做即决，不留尾巴。

## 速查/自测

### 在职抓手（模板F场景化应急型）

在职考前一周做突发预演，选3个场景模拟：

- **场景1（10分钟）**：拿一套做过的行测，随机抽5题假设"看不懂"，练"3秒标记跳过"的反射。
- **场景2（15分钟）**：申论大作文限时20分钟写框架，练"保框架"模式。
- **场景3（5分钟）**：练4-7-8呼吸法4次，确保考场上能秒用。
- 降级目标：实在没时间，至少把5个突发的3秒自救动作背熟，考前一晚默念一遍。

### 在校抓手（模板F场景化应急型）

在校时间充裕，做完整突发预演：

- **每周1次"突发注入"模考**（30分钟）：故意制造突发（如某题限时30秒必须跳、大作文倒计时20分钟必须保框架），练临场SOP。
- **5个突发场景全过一遍**：每个场景写出自己的3秒自救和30秒过渡，贴在书桌前。
- **抗干扰训练**：模考时故意开背景音，练专注力。

伪努力预警：在校生容易陷"只看突发清单不演练"——觉得知道怎么办就行了。但临场靠本能不靠记忆，必须练成反射。校验标准：随机报一个突发场景，3秒内能说出自救动作。

### 心态管理

突发发生时杏仁核必被激活，慌乱是生理本能不是你的错。关键是别让慌乱持续——3秒自救的"3秒"就是给杏仁核的反应时间，30秒过渡就是用呼吸和简单动作把前额叶拉回来。记住一个反常识事实：突发处理的好的考生，往往比一帆风顺的考生考得更好——因为突发逼你专注，反而进入心流。情绪安抚到此（占比已控20%内），剩下80%按SOP执行。

### 速查卡：5大突发3秒+30秒

| 突发 | 3秒自救 | 30秒过渡 |
|---|---|---|
| 看不懂的题（行测） | 标记跳过 | 做2道简单的找手感 |
| 看不懂的题（申论） | 不空着回材料 | 圈材料关键词写上去 |
| 写不完（大作文） | 呼吸停笔5秒 | 保框架：开头+3核心句+结尾 |
| 写不完（行测） | 停做涂卡 | 未做统一蒙同一选项 |
| 涂卡出错 | 举手示意监考员 | 换卡或擦净重涂 |
| 身体突发 | 举手如厕/闭眼呼吸 | 回来做简单题找回节奏 |
| 考场干扰 | 耳塞/锚定题干 | 强迫读完当前题 |

### 自测题

1. 行测遇到看不懂的题，你的3秒动作是什么？答"标记跳过"才合格。
2. 申论大作文写不完，框架保底结构是哪几部分？
3. 涂卡串行了第一反应是什么？答"举手"才对，别自己改。
4. 5个突发场景的3秒自救能脱口而出吗？不能就再背一遍。
5. 考前一周你预演过几个突发场景？

### 一句话小结

考场突发不是意外是常态——3秒自救止损、30秒过渡回节奏，把局部损失钉在局部，全局得分就保住了。
