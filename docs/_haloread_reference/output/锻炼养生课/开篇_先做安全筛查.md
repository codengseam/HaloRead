# 锻炼课·03｜先做安全筛查

## 一、那个倒在终点线前的人

二〇一九年秋天，某城市半程马拉松二十一公里处。

四十五岁的老陈已经跑了两小时十分钟。他平时不跑步，三个月前被同事拉来报名，想着"挑战一下自己"。他没做体检，没做心肺测试，买了一双跑鞋就开始练，每周跑两三次，最多跑过十公里。

比赛那天他状态不错，前十五公里配速稳定。到十八公里，他开始觉得胸口发闷，以为是累的，咬咬牙继续。二十公里，胸口像压了块石头，左肩发麻，冷汗直冒。

他没停下。他想冲过终点。

终点前两百米，老陈倒了。送到医院，急性心肌梗死。

老陈不是个案。每年马拉松赛道上都有人倒下，绝大多数是平时不运动、突然上量的人。他们不是输给了距离，是输给了"我以为我能行"。

## 二、身体不会提前通知你

很多人觉得，运动嘛，谁不会，跑就完了。

可心脏是个沉默的器官。冠状动脉堵了百分之七十，你可能毫无感觉，日常走路上下楼都没事。可一旦你猛地跑起来，心率飙升，心肌需氧量翻几倍，那条堵了七成的血管就供不上血了。

心肌缺血不会提前发请柬。它在你最自信的那一刻，给你一记闷棍。

更阴险的是，很多人倒下之前，身体其实给过暗示——只是被当成了"累的""练少了""坚持一下就好"。胸闷被当成岔气，头晕被当成低血糖，左肩发麻被当成姿势不对。运动猝死案例里，七成以上的人事前有过被忽略的预警信号。

这就是为什么所有正规运动指南，开篇第一件事是教你先筛自己，而不是教你练什么。

## 三、PAR-Q+：七道送命题

国际上有个最简单的自测工具，叫PAR-Q+，身体活动准备问卷。它就七道题，答完就知道自己能不能直接开练：

1. 医生是否曾告诉过你，心脏有问题，只能在医生指导下运动？
2. 你运动时是否曾感到胸痛？
3. 过去一个月内，你不运动时是否曾感到胸痛？
4. 你是否曾因头晕失去平衡，或曾晕倒？
5. 你是否有可能因运动而加重的骨骼或关节问题？
6. 医生是否正在给你开血压或心脏方面的药？
7. 你是否知道自己有不能运动的其他原因？

七题里只要有一题答"是"，就别急着开练，先去看医生。

这问卷不复杂，但它挡住了无数个老陈。

## 四、红旗症状：身体在喊救命

除了问卷，还有几个"红旗症状"，出现任何一个，立刻停，立刻就医：

胸痛、胸闷——尤其放射到左肩、下巴、后背的。这种放射痛是心肌缺血的招牌，别当成肌肉酸。
不明原因的头晕、眼前发黑。运动中突然发晕，可能是血压骤变，也可能是心律出了问题。
运动中异常气短——和平时同样的强度，突然喘得厉害。这往往是心肺功能跟不上的红灯。
心悸、心跳不规则。偶尔早搏正常，可要是心跳乱得让你发慌，停下来。
晕厥史——不管什么时候晕过，都得查清楚。

还有几类人，开练前必须先过医生这一关：已知或疑似心血管病、严重肥胖、进食障碍、控制不佳的高血压、糖尿病并发症、孕期且有特殊情况。

严重肥胖者别急着上跑步和跳绳。体重大，膝盖和腰椎承受的冲击成倍放大，先从游泳、骑车、快走这些低冲击项目入手，等体重下来再考虑承重运动。进食障碍者更得小心，运动可能被用来"消耗"，反而加重病情，必须在专业指导下进行。

这不是吓你，是让你能一直跑下去，而不是跑一次就结束。

## 五、筛查是买保险

有人觉得筛查麻烦，"我年轻，我没事"。

可猝死不挑年龄。三十五岁以下运动猝死，多是先天性心脏结构问题，比如肥厚型心肌病、冠状动脉畸形——这些人平时完全没症状，只有运动负荷下才暴露。三十五岁以上，多是冠心病，斑块破裂堵血管。

ACSM的建议很明确：中高强度运动前，先做PAR-Q+；有红旗或高危因素，加做医学评估和运动负荷测试。它算不上门槛，倒像一根安全带。

系安全带不代表一定会出事，可一旦出事，它就是救命的。

## 六、低风险也别浪

筛查过关，不代表可以胡来。

新手最该守的两条线：一是能说话但不能唱歌（中等强度），二是还能说短句但说不了长句（较高强度）。喘到说不出话，就过了。

第二条：循序渐进。第一周走，第二周快走，第三周慢跑。每周加量不超过百分之十。别和朋友圈比，别和年轻人比。你的对手只有昨天的自己。

还有一条容易被忽略：学会听身体的"软信号"。运动后正常的累是肌肉酸、睡一觉就好；不正常的是持续疲惫、关节刺痛、休息两三天还缓不过来。前者是训练痕迹，后者是预警。分不清的时候，宁可多歇一天，也别硬撑。

老陈后来救回来了。医生在他心脏里放了支架。出院那天医生跟他说："你想跑步可以，但先来我这做评估。"老陈点头。他现在每周快走五次，每次四十分钟，心率控制在一百二以内。他没再跑马拉松，但他活下来了。

## 七、结语

运动的第一个动作，是筛查，不是热身。先确认身体能承受，再谈挑战。
