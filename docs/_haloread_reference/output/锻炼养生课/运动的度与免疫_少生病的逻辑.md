# 锻炼课·24｜少生病的逻辑

## 一、马拉松赛后的怪事

每年波士顿马拉松赛后的一周，总有一件怪事。

跑者们陆续收到一封问卷——你赛后这几天，有没有嗓子疼、流鼻涕、发烧？回收上来的数据，年年都让研究者皱眉：跑完全程的人，赛后那几天报告上呼吸道感染症状的比例，明显高于平时，也明显高于没参赛的人。

这事儿搁谁都觉得反常。这些跑者平时身体好得很，训练规律，吃得也讲究，怎么跑完一场比赛，反而像换了个人似的容易生病？

更怪的是另一头。那些平时只是散散步、跳跳操、偶尔慢跑的中年人，一年到头感冒次数明显比久坐的同事少。他们没练得多猛，可就是不怎么生病。

一边是"练得猛的容易病"，一边是"练得适度的少生病"。这俩现象摆在一起，像是个悖论。

直到有个人把它画成了一条曲线。

尼曼自己就是个跑者，跑过超马。他不是坐在实验室里拍脑袋，是把自己和身边跑者的真实经历，跟免疫学数据对上了号。他发现一个规律：训练季里中等强度跑的时候，他几乎不感冒；可一旦备赛拉到高强度、长距离，赛后那一两周，嗓子疼、流鼻涕就找上门。他自己就是那条曲线的活样本。

## 二、那条J形的曲线

美国阿帕拉契州立大学的戴维·尼曼（David Nieman），研究运动和免疫几十年。他把不同运动量的人，感染上呼吸道疾病的风险，画在一张图上。

画出来是个"J"形。

曲线的左边，是久坐不动的人，感染风险偏高——不活动，免疫也懒洋洋的。曲线往右走一点，到中等强度运动的人，风险掉到最低——比久坐的人明显低一截。这是J的底部。

再往右，到高强度、大运动量的人，曲线又翘起来了——风险回升，甚至接近、超过久坐的人。这是J的右边那一笔。

这就是运动免疫学里著名的"J型曲线"。

意思很直白：**适度运动，免疫力往上走，少生病；过度运动，免疫力往下掉，反而容易病。** 运动量和感染风险，不是"越多越好"的直线，是个先降后升的J。

那个马拉松赛后的怪事，这下有了答案——跑全马是大强度，赛后免疫开了窗，病毒趁虚而入。而那些散步跳操的中年人，正好待在J的底部，免疫最舒服的位置。

这也解释了一个常见现象：有人开始运动后觉得身体变好了，很少感冒；可一旦迷上跑步、越跑越多，备赛期反而开始三天两头生病。他以为自己"体质变差了"，其实是把自己从J的底部，推到了右边那一笔。方向错了，越努力越糟。

## 三、肌肉也是个"内分泌器官"

为什么适度运动能养免疫？这里头有个挺颠覆的发现。

过去以为肌肉就是干活的——收缩、发力、扛重物。后来发现，肌肉远不止于此。运动的时候，肌肉会往外分泌一堆信号分子，叫"肌因子"。这玩意儿能跟免疫系统"对话"。

丹麦的贝格·佩德森（Bente Pedersen）做了一块开创性研究：运动时肌肉分泌的肌因子里，有好几种是抗炎的。也就是说，你一动起来，肌肉就在给全身发"灭火"信号，压住慢性炎症。

这个发现颠覆了人们对肌肉的认知。过去觉得肌肉就是个"发动机"，听大脑指挥收缩发力。佩德森的研究让人看到：肌肉还是个"内分泌器官"，会主动跟全身对话，调节免疫、调节代谢。你每动一下，肌肉就在替你"消炎"。难怪说运动是最好的药——这药是你身体自己造的，免费，还没副作用（前提是别过量）。

这事儿重要，因为慢性炎症是衰老和很多病的底子。

意大利的克劳迪奥·弗朗切斯基（Claudio Franceschi）提出过一个概念，叫"inflammaging"——炎症性衰老。人上了年纪，体内低度慢性炎症会慢慢升高，像文火炖着，一点一点啃骨头、啃血管、啃脑子，和很多老年病都脱不了干系。

而运动，恰恰是天然的"抗炎剂"。适度动起来，肌因子往外送抗炎信号，把那把文火压一压。这就是为什么规律运动的人，不光少感冒，连很多慢性病都来得晚、来得轻。

反过来，久坐不动的人，肌肉长期不出声，抗炎信号断了，慢性炎症没人压，慢慢烧。这就是为什么久坐被叫做"新型吸烟"——它不是让你立刻出事，是日积月累地、悄悄地，把身体往炎症和衰老那边推。

## 四、怎么待在J的底部

道理清楚了，方法就一句话：**中等强度，规律坚持，别过度。**

什么是中等强度？就是上一章说的那个"能说话但唱不动歌"的程度。快走、慢跑、骑车、游泳、跳操，到这个强度，持续二十到四十分钟，一周三到五次。这就是J的底部，免疫最舒服的位置。

几个实在的提醒：

**别突击加量。** 平时不动，突然去跑个十公里，那是直接把自己从J的左边甩到右边——从"久坐风险高"跳到"过度风险高"，中间那个舒服的底部你都没经过。免疫也受不了这种过山车，要么不动要么猛动，两头都不讨好。

**大强度后注意防护。** 跑了长距离、做了大强度训练，赛后那两三天免疫开窗，别往人堆里扎，别着凉，别熬夜，吃好睡好。让开窗期平平安安过去。

**生病了怎么办。** 普通感冒、症状在脖子以上（流鼻涕、打喷嚏），可以做一些轻量活动，散散步、拉伸拉伸，别硬练。症状在脖子以下（发烧、胸闷、拉肚子、浑身酸痛），停练休息，别运动——这时候运动不是养免疫，是添乱。

**慢病人群更要稳。** 有慢性病、年纪大、免疫力本就弱的，更要把强度压在中等，宁可少一点，别贪。养免疫是慢功夫，急不得。这类人最怕的不是练少了，是练猛了出事，稳比快重要。

**免疫力低下、正在治疗者要遵医嘱。** 免疫力低下、正在接受免疫抑制治疗或肿瘤治疗者，运动方案需在医生指导下制定，别自行加量。

还有一条容易被忽略：睡眠和营养，是免疫的另外两条腿。运动、睡眠、营养，三足鼎立，缺一条都站不稳。有人光练不管睡、不管吃，练得再对，免疫也养不起来。J的底部，是中等运动加够睡加吃对，三样凑齐才待得住。

## 五、一句话

适度运动是免疫的朋友，过度运动是免疫的敌人——少生病的逻辑，就藏在J的底部。你不用练成运动员，只要让自己稳稳待在那条曲线最低的地方，身体自然会替你把守好防线。
