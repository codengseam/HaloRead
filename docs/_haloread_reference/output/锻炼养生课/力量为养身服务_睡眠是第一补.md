# 锻炼课·12｜睡眠是第一补

## 一、六晚四小时的代价

斯坦福大学做过一个实验，让一群年轻健康的男生连续六晚每晚只睡四小时。

结果吓人。

六天之后，这些二十出头的小伙子，葡萄糖代谢能力下降了40%——这个数字已经踩到了糖尿病前期的门槛。他们的皮质醇飙升，睾酮水平掉到了比他们老十岁的人都不如。研究员说，照这个速度下去，再熬几周，他们的身体年龄会"老"出一个辈分。

这是马修·沃克在《我们为什么要睡觉》里记录的研究。沃克是加州大学伯克利分校的神经科学教授，研究睡眠二十多年。他说过一句话：睡眠是健康最被低估的杠杆，没有之一。

后来斯坦福又做了另一个实验，这次对象是篮球队员。让他们每天睡够十小时，持续几周。结果呢？罚球命中率提升9%，三分命中率提升9.2%，冲刺速度也快了。没有加练，没有换教练，只是多睡了。

你可能觉得奇怪：睡觉怎么能让人变强？训练不是在健身房里完成的吗？

## 二、肌肉是在休息时长出来的

这就说到一个很多人搞反的事。

训练的时候，你在做什么？你在撕裂肌肉。每一次举重、每一次冲刺，肌肉纤维都在产生微小的损伤。这个时候肌肉不会变强，它只会变弱、变酸、变累。

真正的生长，发生在训练之后——尤其是睡眠之中。

沃克在书里讲得很清楚：人入睡后，前三分之一的时间是深度睡眠（慢波睡眠）。就在这个阶段，脑垂体会分泌一天中最大量的生长激素。生长激素是干嘛的？修复组织、合成蛋白质、加固肌肉。换句话说，你白天在健身房撕裂的那些纤维，是在深夜的深度睡眠里被一针一线缝补、加固、变粗的。

如果你睡不够，这一针一线就少了。撕裂照旧，修复打折，练得越狠反而越虚。

睡眠还干另一件大事——清理大脑垃圾。大脑里有一套叫"类淋巴系统"的管道，白天几乎不工作，到了睡眠时才打开，把白天积累的代谢废物冲走，其中包括一种叫β-淀粉样蛋白的东西——这玩意儿堆积多了，和阿尔茨海默病脱不了干系。

所以熬夜不只是第二天没精神，是在给大脑攒垃圾。

## 三、运动和睡眠，是双向养身

那运动和睡眠是什么关系？

是双向的。

运动能改善睡眠，这是被反复证实的。规律运动的人，深度睡眠时间更长，入睡更快，半夜醒来的次数更少。沃克甚至建议，运动是改善睡眠最有效的非药物手段之一。

但有个细节：运动时段对睡眠有影响。对大多数人来说，早晨或下午运动对睡眠最有利。而高强度运动如果放在睡前一两小时，有些人会睡不着——因为体温升高、交感神经兴奋，身体还处在"战斗状态"。

不过这事因人而异。有人睡前练完倒头就睡，有人练完睁眼到天亮。判断标准很简单：练完看你睡得怎么样。睡得好就继续，睡不好就往前提。

还有一点：中等强度的有氧运动，比如快走、慢跑、骑车，对改善睡眠尤其友好。它不会过度刺激神经系统，又能消耗掉白天积攒的紧张感。很多人失眠，其实是身体"没累够"——白天坐一天，晚上脑子转个不停，身体却没释放过能量。动一动，把那股劲卸掉，睡眠自然就来了。

沃克还提过一个容易被忽略的细节：睡眠不是铁板一块，它分周期。一晚上大约四到五个周期，每个周期九十分钟，从浅睡到深睡再到做梦（快速眼动睡眠）。深度睡眠管身体修复，做梦的睡眠管记忆和情绪。运动对深度睡眠的加成最明显——练得规律的人，深度睡眠那一段占比更高，也就是身体修复得更充分。反过来，长期不运动的人，深度睡眠往往偏短，醒来还是觉得没睡透。

## 四、过度训练致失眠，是身体在报警

可这里有个反过来的警告，必须说清楚。

如果你最近训练很猛，突然开始失眠、入睡困难、半夜醒来再也睡不着——别以为是压力大，这可能是**过度训练**的信号。

沃克和运动科学界都提醒过：当训练量超过恢复能力，身体会进入一种"亢奋却疲惫"的状态。最典型的表现就是静息心率升高——早上醒来测一下心率，如果比平时高出5到10跳，说明身体还在"打仗"，没缓过来。这时候还硬练，只会越练越垮。

睡眠和训练是一对搭档。训练制造需求，睡眠完成修复。你给睡眠多少时间，睡眠就给你多少修复。你克扣睡眠去加练，等于拆东墙补西墙，墙迟早要塌。

运动员圈子里有句话：训练是贷款，睡眠是还款。光贷款不还款，利息能把你压垮。

## 五、安全警示

如果你有胸痛、胸闷、不明头晕、严重气短、晕厥史，或疑似心血管问题，先就医。长期失眠伴随静息心率持续升高、情绪低落、食欲下降，是过度训练的红旗信号，应立即减量休息，必要时寻求专业帮助。

别把"练不动还硬练"当毅力。真正的养身，是懂得什么时候该停下来。

## 六、结语

回到那个斯坦福实验。六晚四小时睡眠，把二十多岁的小伙子熬成了"准糖尿病人"。这是身体在告诉你：恢复这件事，比训练本身更重要。

你可以在健身房里挥汗如雨，可以举得很重、跑得很快，但如果晚上只睡五六个小时，那些训练的收益会大打折扣。那是在透支，不是养身。

睡眠是第一补，不是第二补，不是可选补。它是所有训练、所有恢复的地基。地基不牢，上面盖什么都晃。

好好睡觉，比多练一小时管用得多。
