# 锻炼课·01｜躺了十年站起来

## 一、黎明前的哈扎

坦桑尼亚东非大裂谷的旱季，天还没亮，一个叫马塔约的哈扎男人已经钻出草屋。

他赤脚踩在红土上，腰间别着一把木弓、几支箭、一个塑料桶。今天他要走十二公里去找蜂蜜。这十二公里爬猴面包树、钻灌木丛、躲黑曼巴蛇，步步真切。

马塔约今年四十二岁。在哈扎人里这算壮年。他静息心率五十出头，体脂不到百分之十五，血压和二十岁的小伙子差不多。他从不"锻炼"，因为他一天的活动量，已经是现代人一周的总和。

他也不懂什么叫有氧、无氧、心率区间。他只知道蜂蜜在树洞里，孩子等着吃饭，箭得磨利，水得去背。动，是活下去的前提。

人类学家丹尼尔·利伯曼跟踪哈扎人多年，得出一个让人坐不住的结论：哈扎人每天中高强度活动约一小时，轻体力活动三到六小时。而现代美国人，半数人一天中高强度活动几乎为零。

更有意思的是，哈扎人并不比现代人瘦因为"代谢快"。利伯曼测过他们的基础代谢，和城里人几乎一样。差别全在"动"上——他们把同样的能量，花在了活动里，而不是存进了腰围。

马塔约不知道什么叫"久坐危害"。他只是活着，活着就得动。

## 二、焊在椅子上的人

镜头切到上海，陆家嘴一栋写字楼三十七层。

下午三点，程序员老周盯着第三块屏幕，屁股已经焊在人体工学椅上六个小时。他中午叫了外卖，吃完没动；上午开了两个会，全程坐着；昨晚加班到凌晨一点，今天腰酸背痛，膝盖发僵。他今年三十八，体检报告上写着：脂肪肝、血压偏高、颈椎曲度变直、空腹血糖临界。

老周不是不努力。他办过三次健身卡，都去了不到十次。他买过跑步机，现在成了晾衣架。他下载过七个运动App，最长坚持了十九天。

他不是懒，是现代生活把他焊住了。

利伯曼把这种现象叫"进化失配"——人的身体是几百万年动出来的，可过去一百年，我们把它塞进了椅子、汽车、电梯、外卖里。身体还在等那个每天走十几公里的祖先，主人却变成了每天走不到两千步的"坐家"。

## 三、身体在等你动

人这套身体，从头到脚都是为运动设计的。

心脏每天跳十万次，泵七千升血，它不是给你躺着用的。骨骼是活的组织，受力才长得密，不受力就疏松——宇航员在太空待六个月，骨密度掉百分之一到二，相当于地面十年老化。肌肉更现实，你不用它，它就回收，三十岁后每年自然掉百分之一到二，不用进棺材，先坐轮椅。

这套机制叫"用进废退"。细胞不讲情怀，只讲信号。你给它压力，它就升级；你给它安逸，它就退役。

上世纪四十年代起，科学家做过一系列卧床实验：让健康的年轻人躺三周不动，结果触目惊心——心率变快、血压调节失灵、血容量下降、肌肉萎缩、骨密度流失，站起来时甚至会晕倒。三周卧床造成的身体退化，相当于衰老十几年。后来宇航员上了太空，在失重环境里待上半年，骨密度掉百分之一到二，肌肉掉百分之十以上——和卧床实验一个道理。身体这套系统，是为重力、为活动设计的，你把它撤了，它就按"快退休"处理。

那为什么躺着不动会病？因为身体把"不动"翻译成了"这人可能要冬眠，或者快死了"。于是代谢降速、肌肉分解、骨量流失、胰岛素敏感性下降——一套节能模式自动启动。它本意是帮你省能量熬过饥荒，可你天天坐办公室，饥荒永远不来，节能模式就成了慢性自杀。

WHO的数据很直白：久坐是第四大死亡风险因素，每年直接关联三百多万人死亡。每天坐超过八小时且不运动，死亡风险和肥胖加吸烟差不多。

## 四、NEAT：被忽略的火苗

有个概念叫NEAT，非运动性活动产热。说白了就是你不去健身房，光靠走路、站立、做家务、抖腿、爬楼梯这些"非运动"消耗的热量。

瘦人和胖人的NEAT差距能到每天三百到五百大卡。什么概念？相当于慢跑四十分钟。

哈扎人之所以瘦，秘诀在他们一天到晚都在动。利伯曼测过，哈扎人每天总能量消耗其实和现代人差不多——可他们把能量花在了活动上，而不是储存在脂肪里。

所以第一课不教你深蹲，不教你跑五公里，就一句话：先站起来。

## 五、从不动到微动

别一上来就想着翻盘。躺了十年的人，站起来都得慢慢来。

第一步，把"坐"换成"站"。能站着打电话就别坐着，能走着开会就别定会议室。每隔四十五分钟起来走两分钟，倒杯水，去趟厕所，伸个懒腰。这是给身体发信号：我还活着，别启动节能模式。

第二步，把"车"换成"腿"。最后一公里走回家，地铁少坐一站，电梯改楼梯。这些零碎的步数加起来，比你想的多。

第三步，找一个"动"的理由。遛狗、逛菜市场、陪孩子追跑、站着看剧。让动变成生活的一部分，而不是又一项待办事项。

老周后来没去健身房。他做了一件事：把办公桌换成了升降桌，上午站两小时，下午站一小时，每四十分钟绕办公室走一圈。三个月后，腰不疼了，体重掉了四斤，睡眠也好了。他没"锻炼"，他只是不再"不动"了。

## 六、结语

身体是动出来的，不是养出来的。你不动，它就替你决定怎么退场。
