# 锻炼课·35｜听身体的话

## 一、那场硬撑的深蹲，把他送进了医院

大刘是个自律的人。

自律到什么程度？健身计划排得像列车时刻表，周一深蹲、周三卧推、周五硬拉，雷打不动。朋友约饭不去，加班再晚也去练。

直到那个周五。

那天他连轴转了十四个小时，回到家已经晚上十一点。按计划该练硬拉。他犹豫了三秒，还是换衣服去了健身房。

第一组还行，第二组开始手抖，第三组弯腰拉起的瞬间，腰"咔"地一声，他直接跪在地上。腰椎间盘突出，卧床三周。

病床上他反复想：明明累得不行，为什么还要去？

答案是他信奉一句话——"越累越要练，练完就精神了"。这句话坑了无数人。

大刘的同事老赵，是另一种活法。老赵也加班，也累，但他有个规矩：累到什么程度，就练到什么程度。加班晚了，不去撸铁，改成散步半小时加十分钟拉伸；连轴几天实在没劲，就只做十分钟调息。从不硬撑。

五十岁的人，老赵比大刘还精神。大刘卧床那阵子，老赵来看他，笑着说：你这叫"用自律毁了自己"。

同样是累，一个硬撑进医院，一个降级反而养好了。差别在哪？在听不听身体的话。

## 二、累的时候，身体在说什么

先搞清楚，疲劳时身体到底发生了什么。

训练后肌肉会酸痛、会发炎，这是正常的。身体需要时间清理代谢废物、修补微小损伤、补充糖原。这个过程叫恢复。恢复完成了，你比之前更强；恢复没完成就再练，等于在没愈合的伤口上再划一刀。

疲劳分两种。一种是局部疲劳，比如昨天练了腿，今天腿酸，这是局部的，换个部位练就行。另一种是系统性疲劳，比如连续加班、睡眠不足、压力大，整个身体的恢复资源都被掏空了。这时候不管练哪个部位，都是在透支。

系统性疲劳时，身体的交感神经一直紧绷，皮质醇水平高，副交感神经被压着。这时候再去高强度训练，等于给已经过热的发动机再踩油门。

更糟的是，疲劳状态下动作会变形。神经控制不住肌肉，该稳定的关节稳不住，该发力的肌肉发不上力，代偿的肌肉乱使劲。大刘的腰，就是这么伤的——问题出在身体已经控不住那个重量，重量本身倒还在其次。

所以"越累越练越精神"是个假象。练完那一下确实会兴奋，那是应激激素硬顶上来的，透支的是后面几天的恢复能力。一次两次没事，长期这么干，要么受伤，要么越练越垮，进入过度训练。

## 三、主动恢复，比硬练和不动都强

那累的时候该怎么办？完全不动吗？

也不是。研究里有个概念叫"主动恢复"，意思是：在疲劳状态下，做低强度的活动，比完全躺着不动，恢复得更快。

听着反直觉，但有道理。

低强度活动——散步、慢骑、拉伸、调息——能让血液循环加快，把肌肉里的代谢废物更快带走，同时不至于造成新的损伤。它还激活副交感神经，把身体从"战斗状态"切回"修复状态"。

完全不动，血液循环慢，废物清理也慢，人反而更僵更沉。硬练，是制造新损伤。主动恢复，是帮身体加速清理，又不添乱。

所以低能量日的智慧，关键在于"降级"——把强度降下来，把目的从"练强"换成"养好"，避开"练或不练"的二选一。

这跟传统养生的思路是一致的。古人讲"动静相宜"，讲的是该动则动、该养则养——一直动或一直静都不对。累的时候，柔缓的导引、调息，正是主动恢复的古代表达。

## 四、降级，不是放弃

低能量日怎么练？记住一个原则：降级，不放弃。

降级有三档，按疲劳程度选。

轻档：只是有点累，睡眠还行。把今天的高强度降成中等。比如原计划跑步，改成快走；原计划大重量，减重三成、组数减半。动一动，出点微汗，就够。

中档：加班晚了，或者连着几天没睡好。放弃高强度，只做低强度。散步二十到三十分钟，加十分钟拉伸，加五分钟调息。不追求心率，不追求出汗，只追求"身体动起来、神经松下来"。

重档：连续透支、感冒初愈、或者身体明显发虚。这一档别练了，只做调息和最轻的拉伸，甚至就是好好睡一觉。这不是偷懒，这是训练的一部分——恢复也是训练。

怎么判断今天该用哪一档？听三个信号。

一是睡眠。昨晚没睡够六小时，或者睡得很差，至少降一档。

二是晨起心率。早上醒来测一分钟脉搏，比平时高五次以上，说明身体还在恢复中，降级。

三是主观状态。起床就觉得沉、不想动、对训练有抵触，这不是懒，是身体在喊累。降级，或者休息。

这三条任意一条亮红灯，就别上强度。三条都亮，直接进重档。

## 五、硬撑的代价，比你想的大

低能量日硬练，最直接的风险是受伤。大刘那种是急性的，更多人是慢性的——长期透支，关节和肌腱反复在疲劳状态下受力，一点点磨损，某天突然爆发。

更深的风险是过度训练综合征。这是身体被逼到极限后的"罢工"，表现为持续疲劳、表现下降、睡眠紊乱、情绪低落、容易生病，甚至静息心率长期偏高。一旦进入这个状态，可能要几周甚至几个月才能恢复，期间完全练不动。

很多人把过度训练当"瓶颈期"硬扛，越扛越深。其实身体早就报警了，只是被"自律"两个字盖住了。

自律是好品质，但盲目的自律是自我伤害。真正的自律，包括"该休息时能休息"这一条。能练是本事，能歇是更大的本事。

还有一类信号要单独说：如果疲劳伴随胸闷、心慌、不明头晕、气短，这不是普通累，可能是心血管问题的信号，立刻停练就医，别当成"低能量日"硬扛过去。有晕厥史、疑似心血管病的人，疲劳时尤其不能上强度。

## 六、身体是雇主，不是对手

大刘伤好之后，改了规矩。

他在计划表上加了三个字：恢复日。每周至少一天，只散步和调息。加班晚了，自动降级，不再纠结。半年后，他的训练成绩反而比之前好了——因为身体终于能恢复过来了。

他跟老赵说：以前我把身体当对手，要征服它；现在我把身体当雇主，它说歇我就歇。

老赵说：这就对了。身体给你发工资（健康和精力），你倒好，天天跟雇主对着干，能不被开除吗？

累的时候，身体在说话。它说的不是"我不行"，是"我需要修一修"。听懂这句话，比多练一组深蹲值钱得多。

养身养身，养的是身，不是养计划、养打卡记录。计划是死的，身体是活的。让死的计划迁就活的身体，这才叫会练。

该冲的时候冲，该养的时候养。能进能退，才是真功夫。
