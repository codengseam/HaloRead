# 锻炼课·31｜调息的力量

## 一、那一口气，定住了一条命

讲个真事。

2018年，法国自由潜水冠军纪尧姆·内里下潜到海底129米。没有氧气瓶，一口气。下潜时心率从七十掉到三十几，身体进入"哺乳动物潜水反射"。最惊险的在回程上浮，下潜反倒其次。回到水面那一刻，他必须用一种特殊的呼吸把肺重新撑开，节奏错了，轻则头晕目眩，重则晕厥黑视。

他上岸后做的第一件事，没去欢呼，闭眼，慢慢吐气，再慢慢吸气。三秒一吐，四秒一吸。教练在旁边数拍子。三十秒后，他睁眼，笑了。

记者问他：海底那么深，你怕不怕？

他说，怕。但呼吸一乱，人就废了。呼吸是潜水员唯一的遥控器。

把镜头拉回岸上。

北京，写字楼，下午三点。一个三十出头的项目经理刚被老板当众骂完，胸口发闷，手心冒汗，心跳像擂鼓。他躲进洗手间，蹲了五分钟没用。后来有人教他一个法子：鼻子吸气四秒，憋住两秒，嘴巴吐气六秒。做八遍。

五分钟后他出来，脸还是红的，但手不抖了，心跳从一百一回到八十。

同一口气，能在海底救命，能在格子间救命。

问题来了——呼吸这东西，人从生下来就会，凭什么慢下来几秒，就能管住心跳、压住焦虑？古人说的"调息"，到底调的是什么？

## 二、调息，调的是神经

先把一个误会拆掉。

很多人以为呼吸就是"换气"，把氧气吸进来、二氧化碳排出去。这话对一半。呼吸还有另一条线，直通大脑的"自动挡"——自主神经系统。

自主神经分两路：交感神经是油门，让你心跳加快、血压升高、准备战斗或逃跑；副交感神经是刹车，让你心跳减慢、消化启动、身体休整。两路平时此消彼长，像跷跷板。

呼吸，是唯一一个你能主动去拨动这架跷跷板的开关。

吸气时，心跳微微加快；呼气时，心跳微微减慢。这个一快一慢的差值，有个学名，叫心率变异性，缩写HRV。HRV越高，说明你的副交感神经越有劲，身体越能从紧张里弹回来。HRV太低，人就像一根绷紧的弦，一点小事就崩。

而呼气越长、越慢，副交感那一路就越占上风。这就是为什么潜水员上岸要慢吐气，为什么焦虑时"长呼短吸"能压住慌乱——你是在用呼吸给迷走神经发信号：安全了，松下来。

还有一层，在膈肌。

膈肌是胸腔和腹腔之间那块圆顶状的肌肉，呼吸的主引擎。大多数人紧张时只用肩膀和胸口浅浅地喘，膈肌几乎不动。深呼吸时膈肌下沉，挤压下面的肝、胃、肠，像给内脏做按摩；呼气时膈肌回升，又把静脉血推回心脏。这一上一下，本身就是循环的助力。

所以古人讲"调息"，调的不是空气，是神经和脏腑的节奏。这一层，现代研究已经摸得很清楚，属于可实证的范畴。

## 三、慢、深、匀，三个字够了

调息的法子，古今中外花样繁多，数呼吸、观呼吸、四七八呼吸法、箱式呼吸……名目一堆。剥掉外衣，核心就三个字：慢、深、匀。

慢，是把呼吸频率降下来。普通人每分钟呼吸十二到二十次，调息时压到六到十次，甚至更低。慢下来，副交感神经才肯接管。

深，是让气沉到腹部，而不是停在胸口。手放肚脐上，吸气时肚子鼓起，呼气时肚子回落。这就是膈肌呼吸，俗称腹式呼吸。

匀，是节奏稳。不忽快忽慢，不憋一下喘一下。匀了，神经才认这是"安全信号"。

具体怎么练？给一个最简的版本。

> 传统养生参考：本节内容属传统养生参考，不替代医疗。

找个安静地方坐下或躺下，一只手放胸口，一只手放肚子。鼻子吸气，数四拍，让放肚子的那只手被顶起来，胸口那只手尽量不动。稍停一拍。嘴巴微张吐气，数六拍，把气慢慢送走。一轮十拍，做八到十轮，大概三分钟。

就这三分钟，是调息的"最小有效剂量"。睡前做，助眠；紧张时做，压惊；早起做，开机。

不必追求花样，不必数到几百息。三个字守住，比一百种花架子都管用。

## 四、古人早把话说透了

调息这事，不是现代人的发明。

孙思邈讲调气，大意据《千金要方·养性》：和神调气之道，先须闭目静坐，左右手抱肚，意守丹田，鼻引口吐，绵绵若存。注意"绵绵若存"四个字——呼吸细、长、不断，像一根丝线，若有若无。这跟今天说的"慢深匀"是一个意思，只是古人用诗的语言说，今人用数据说。

《黄帝内经》里也有"呼吸精气，独立守神"的提法，把呼吸和精神的安定绑在一起。古人没有心率变异性这个词，但他们凭身体感知道：呼吸稳，神就稳。

现代这边，研究也跟上了。多项对照实验显示，把呼吸频率从每分钟十五次降到六次左右，受试者的HRV显著提升，焦虑量表评分下降，血压也有改善。机制指向迷走神经张力的变化。这谈不上玄学，属于可测量的生理反应。

需要说清边界：调息是日常养身和情绪调节的手段，不是治病的方法。真有焦虑症、高血压、心律失常，该看医生看医生，调息是辅助，不是替代。

## 五、有些气，不能这么憋

调息听着温和，但有禁忌，必须讲明白。

第一类，过度换气。有人为了"打通气血"，故意快速大口喘气，几分钟下来手脚发麻、头晕甚至抽搐。这是二氧化碳被吹掉太多，血液偏碱，神经兴奋性异常。自由潜水圈管这叫"过度换气黑视"，是出过人命的。日常调息绝不追求快和大，慢才是根本。

第二类，强力屏息。屏息能让心率进一步下降，有人喜欢憋很久来"练肺"。但屏息会升高胸腔压力、影响静脉回流，血压会先升后降。高血压、青光眼、心脑血管有问题的人，强力屏息是禁忌，可能诱发眼压飙升、血压骤变。孕妇也避免长时间屏息，胎儿供氧优先。

第三类，饭后立即深呼吸。胃里满满的时候大幅膈肌下沉，会顶到胃，引起反流不适。饭后半小时内别做调息，散散步就好。

还有一条总则：调息时若出现胸闷、头晕、心慌、手脚发麻，立刻停下来，恢复正常呼吸。身体在报警，别硬撑。

胸痛胸闷、不明头晕、严重气短、有晕厥史或疑似心血管病的人，调息前先问医生。这一条，所有呼吸练习通用。

## 六、把遥控器拿回手里

人这一辈子，呼吸大概六亿次，绝大多数是自动的，你根本没注意。

但只要你想，随时可以伸手去拨一下那个开关。慢一拍，深一点，匀住。三分钟，神经就听话。

古人说"息长命长"，未必是寿命的数字游戏，更像在说：能把呼吸稳住的人，也能把日子稳住。

调息调的不是气，是你和身体之间那条断了很久的电话线。重新接上，你会发现，身体一直在线，只是你太久没拨号。
