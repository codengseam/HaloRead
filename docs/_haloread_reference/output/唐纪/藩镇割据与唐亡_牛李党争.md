# 唐纪·牛李党争

## 一、讲事情

元和三年（808年）四月，长安宣政殿。唐宪宗李纯御殿试制科举人，开的是"贤良方正直言极谏科"。这科名头响亮，意思也直白——要敢说真话的人。

考生里有三个年轻人：伊阙尉牛僧孺、前进士李宗闵、陆浑尉皇甫湜。三人都是贞元、元和间的进士，血气方刚，笔下不留情。对策里他们指切时政，从宦官专权到藩镇割据，从赋税苛重到选官请托，一股脑全写了出来，"无所回避"。

主考官吏部侍郎杨於陵、吏部员外郎韦贯之看了，拍案叫好，定为上等。翰林学士裴垍、王涯复核，也无异议。宪宗览卷，颇为嘉许。

可卷子到了宰相李吉甫案头，事情变了味。李吉甫是赵郡李氏出身，宪宗朝两度拜相，正主推削藩的大业。在他眼里，这几个小官的对策字字句句都在揭他执政的短。更恼人的是，皇甫湜是翰林学士王涯的外甥，王涯参与复核却没申报回避——这里头有文章。

李吉甫跑到宪宗面前，一把鼻涕一把泪地告状。《资治通鉴》记了四个字："泣诉于上。"——宰相哭鼻子，这在中国历史上不多见。

宪宗扛不住，只好处置。考官杨於陵出为岭南节度使，韦贯之贬果州刺史再贬巴州，王涯贬虢州司马，裴垍罢学士。牛僧孺、李宗闵、皇甫湜三人"久之不调，随牒诸侯府"——长期挂在那里，没人理。直到元和七年李吉甫死了，李宗闵才入朝当了个监察御史。——《旧唐书·李宗闵传》《资治通鉴·唐纪五十七》

这就是后来四十年党争的第一粒火星。牛僧孺、李宗闵心里那口气，咽不下去。

十三年后，长庆元年（821年），火星点着了炸药桶。

这一年礼部侍郎钱徽知贡举。放榜前，宰相段文昌和翰林学士李绅各托了人——段文昌受了故刑部侍郎杨凭之子杨浑之的家藏书画，保荐他中第；李绅托了举子周汉宾。结果榜一放，杨浑之、周汉宾双双落第。中榜的十四人里，有李宗闵的女婿苏巢，有副主考杨汝士的弟弟杨殷士，还有给事中郑覃的弟弟郑朗——清一色权门子弟。

段文昌脸上挂不住，辞行那天在内殿面奏穆宗："钱徽所放进士郑朗等十四人，皆子弟艺薄，不当在选中。"穆宗问翰林学士李德裕、元稹、李绅，三人异口同声："诚如文昌言。"

穆宗命中书舍人王起、主客郎中白居易复试，题目皇帝亲拟——一诗一赋。复试结果惨不忍睹，十四人里十一人"辞律鄙浅，芜累甚多"，苏巢落第。

处分下来：钱徽贬江州刺史，李宗闵贬剑州刺史，杨汝士贬开江令。——《旧唐书·钱徽传》《资治通鉴·唐纪六十一》

《旧唐书·李宗闵传》写了一句要命的话："自是德裕、宗闵各分朋党，更相倾轧，垂四十年。"——从这一刻起，德裕和宗闵各立山头，互相倾轧，整整四十年。

李德裕是谁？李吉甫的儿子。当年李吉甫把牛僧孺、李宗闵压下去，如今儿子李德裕又在科场案里把李宗闵的女婿撸了下去。新仇旧恨，捆在一起。

大和年间，两党开始走马灯似的交替执政。

大和三年（829年），裴度荐李德裕入相，眼看要成。李宗闵使了暗劲，"为中人助宗闵者所沮"——宦官出手帮忙，李德裕没入成，复出镇浙西。李宗闵反手引牛僧孺同知政事，"二人唱和，凡德裕之党皆逐之"。——《旧唐书·李宗闵传》

大和七年（833年），李德裕终于入相。六月，罢李宗闵知政事，出为山南西道节度使。可李宗闵靠驸马沈立义结托女学士宋若宪、枢密使杨承和，没多久又翻身回朝。李德裕反被代之。——《旧唐书·李宗闵传》

两党斗到这个份上，已经不顾体面。最典型的一桩，是维州事。

大和五年（831年），西川节度使李德裕收复了被吐蕃占据的维州（今四川理县），吐蕃将领悉怛谋率众投降。这是西川几十年未有的大捷。李德裕上奏，请乘势经营，甚至可以直捣吐蕃腹心。

牛僧孺当时为相，一句话给否了。他说：吐蕃疆域万里，丢一个维州不算什么；可朝廷近来与吐蕃修好，若受其叛将，吐蕃必兴师问罪，"中国弃信，何以怀远"？不如把维州还给吐蕃，把悉怛谋交回去。

文宗听了牛僧孺的。维州还了，悉怛谋被交还吐蕃，吐蕃人当着唐使的面把他酷刑处死。——《资治通鉴·唐纪六十一》

这一手，寒了西川将士的心。后来司马光在《通鉴》里论这事，说"德裕所言者利也，僧孺所言者义也"——李德裕讲的是利，牛僧孺讲的是义。可王夫之不这么看，他认为司马光是被"朋党"和"夷夏"的复杂情态绕进去了，义利不能只看言辞表面。

大和九年（835年），甘露之变爆发。李训、郑注得势，两党通通遭贬。可这俩人也没撑多久，事败被杀。文宗被宦官仇士良软禁至死，曾对侍臣叹了一句千古名言："去河北贼易，去朝廷朋党难。"——除掉河北的藩镇容易，除掉朝中的朋党，难。——《新唐书·李宗闵传》

开成五年（840年），文宗驾崩。宦官仇士良拥立颍王李瀍即位，是为武宗。武宗一上台，就把李德裕从淮南节度使任上召回，拜为宰相。

这一拜，拜出了晚唐最后一次中兴。

会昌三年（843年），昭义节度使刘从谏死，侄子刘稹自立求袭位。朝中有人主张姑息，李德裕力排众议："不可许，许则诸镇效尤。"他亲自调度五路兵马围攻，严令诸镇不得私受降、不得纵兵剽掠。第二年，刘稹部将郭谊杀刘稹降，叛乱平定。这是安史之乱后，中央第一次用武力解决强藩自立继承问题。——《旧唐书·武宗纪》

会昌五年（845年），武宗下令灭佛。拆寺四千六百所，招提兰若四万所，还俗僧尼二十六万，收良田数千顷、奴婢十五万充两税户。钟磬铜像送度支铸"开元通宝"。佛教史上称"会昌法难"，可从财政上看，这一刀切下去，国库立时充实。——《旧唐书·武宗纪》

此外，李德裕还北破回鹘，内抑宦官，裁汰冗官。后人把这一段称为"会昌中兴"。李商隐在《会昌一品集》序里，称李德裕为"万古之良相"。

可中兴只维持了六年。会昌六年（846年）三月，武宗服丹药暴崩，年仅三十三。

即位的是宣宗李忱。宣宗为立储时受过牛党庇护，又忌惮李德裕"专权"。即位当月，罢李德裕相，贬太子少保分司东都；次年贬潮州司马；再贬崖州司户参军。——《旧唐书·宣宗纪》

大中三年（849年）正月，李德裕抵达崖州（今海南琼山）。年末卒于贬所，终年六十三。临终有诗："自是功高临尽处，祸来名灭不由人。"

牛党全面得势。白敏中、令狐绹相继为相。李德裕一贬再贬，连归葬都费了周折。一场四十年的党争，以李党的彻底覆灭告终。

## 二、讲人物

牛僧孺这个人，得从两头看。

他起家是漂亮的。元和三年制举，敢直言时政，触怒宰相，被压了多年不调。这种"直臣"底色，是牛党的招牌。后来他做到宰相，清廉自守，《旧唐书》本传说他"清慎端洁"。他还能写小说，《玄怪录》一卷，是唐代传奇的名篇。这样一个有才学、有骨气的人，怎么就成了党争的祸首？

问题出在他为相的做派上。维州事最能说明问题——李德裕在前线拼死拼活拿下的城、收降的将，他一句话就还回去了，还把降将交还敌国任其屠戮。这哪里是"义"？这是不顾国家边功、不顾将士性命，只为压李德裕一头。司马光说他"所言者义也"，是只看言辞不看用心。

牛僧孺对藩镇，一以贯之地姑息。他当政时，河朔三镇的事基本不闻不问，"以镇抚自安"。这种做派，和他元和三年对策里那个慷慨陈词的青年，判若两人。人在野时都敢直言，在朝时反倒畏缩——这是牛僧孺的矛盾。

李德裕呢，是另一副面孔。

他出身赵郡李氏，父亲李吉甫是宪宗朝名相。按规矩，他可以走门荫入仕。可他不乐意考进士，时人传他有过一句话："好骡马不入行。"——好骡马不进牲口队列。这是门阀子弟的骄傲，也是对进士科浮薄风气的鄙夷。——《新唐书·李德裕传》

这态度，注定了他和牛党的水火不容。牛党核心牛僧孺、李宗闵、杨嗣复，都是进士出身；李党核心李德裕、郑覃，都重经学、重门第。陈寅恪先生在《唐代政治史述论稿》里说，牛李两党的对立，根本在两晋南北朝以来的山东士族，与武后之后由进士词科进用的新兴阶级，两者互不相容。这话抓住了底子。

李德裕的才干是实打实的。他在浙西观察使任上革除淫祠、移风易俗；在西川节度使任上修筑筹边楼、整顿边防；会昌年间为相，平昭义、破回鹘、灭佛、裁冗官，桩桩件件都是实干。他执政时"政事皆出中书"，把宦官挤到一边，连仇士良都被迫致仕。清代王士禛说"李卫公一代伟人，功业与裴晋公伯仲"——和裴度并论，评价不可谓不高。

可李德裕也有他的短处。他打击牛党下手极狠，会昌年间把李宗闵一贬再贬到潮州，把牛僧孺贬到循州。他本人虽清廉，但"专权"之名不是空穴来风——武宗对他言听计从，朝政几乎一人独断。这种做派，在皇帝信任时是效率，在皇帝换人时就是罪状。

更深层的问题是，李德裕也脱不出党争的逻辑。他重门第、轻科举，主张"先试判案、观吏干"，反对以诗赋取士。这主张有其道理，可一旦和党争搅在一起，就成了打击牛党的工具。政策与人事，在他手里也分不太开。

牛僧孺和李德裕，都不是坏人，也都不全是好人。一个有直臣之名却姑息藩镇，一个有干臣之实却专权党同。人性在权力面前，从来不是非黑即白。

## 三、讲背景

牛李党争能闹四十年，不是几个人恩怨那么简单。背后有三条深层线索。

第一条，科举与门荫之争。

唐承隋制，以科举取士。到中晚唐，进士科成了读书人入仕的主渠道。据吴宗国先生《唐代科举制度研究》统计，宪宗至懿宗七朝一百三十三个宰相里，进士出身的一百零四个，占七成八。进士科考的是诗赋，重文采、重辞章，也重座主门生、同年关系。一个主考官放榜，榜上的人就是他一辈子的"门生"，同年及第的就是一辈子的"同年"。这张关系网，是牛党的根基。

可唐朝还留着另一条入仕的路——门荫。高官子弟可以凭父祖官品直接入仕，不必考科举。李德裕就是以门荫补校书郎起家。门荫出身的人，多出自山东士族旧家，重经学、重礼法、重门风，看不上进士科的浮华。

这两条路，本是国家选官的两种渠道，可一旦和人事纠葛缠在一起，就成了党争的底色。牛党是进士集团，李党是门荫集团——这是陈寅恪先生的经典论断。

第二条，士族与寒门之争。

山东士族，指的是魏晋以来绵延数百年的旧门阀——赵郡李氏、荥阳郑氏、清河崔氏、范阳卢氏、太原王氏。这些家族到中晚唐虽已没有政治特权，但社会声望仍在，家学礼法仍在。他们看不起武后之后靠科举爬上来的"新兴阶级"，觉得那帮人浮薄无行。

李党核心多出自这些旧家。李德裕是赵郡李氏，郑覃是荥阳郑氏。牛党核心则多是科举新贵。两党的对立，背后是旧门第与新进者的文化冲突。这不是简单的阶级斗争，而是两种文化传统、两种价值取向的碰撞。

第三条，对藩镇的态度之争。

安史之乱后，藩镇割据是唐朝的心腹大患。怎么对付？两党主张截然不同。

李党主强硬。李吉甫赞助宪宗削藩，李德裕会昌年间平昭义，一脉相承。他们主张中央用兵，遏制强藩，恢复权威。

牛党主姑息。牛僧孺为相时，对河朔三镇基本放任。维州事上，他宁可放弃已到手的边功，也要维持"和好"的表面。他们觉得，藩镇打不动，不如息事宁人。

这两种态度，各有各的道理。李党强硬，能振国威，可劳民伤财；牛党姑息，能省事，可养虎遗患。问题在于，这两条路线本可以因时制宜地调整，可在党争的框架下，成了你死我活的立场之争——你主战我就主和，你主和我就主战，政策成了党争的筹码。

还有一层背景不能忽略——宦官专权。自德宗把神策军交给宦官，宦官就掌握了禁军，进而把持皇帝废立。牛李两党要上位，都绕不开宦官。李宗闵靠枢密使杨承和翻身，李德裕靠武宗信任制住仇士良。两党嘴上骂宦官，背地里都借宦官的力。这才是中晚唐政治最深的溃疡。

## 四、讲道理

牛李党争，古代名家点评不少，挑几句最切中肯綮的。

头一句是唐文宗自己说的："去河北贼易，去朝廷朋党难。"——《新唐书·李宗闵传》。这是当事人最痛的体悟。河北藩镇再难，那是外患，看得见摸得着；朝中朋党是内耗，盘根错节，连皇帝都束手无策。文宗这话，道出了党争对皇权的消解。

第二句是司马光在《资治通鉴》里论维州事。他说"德裕所言者利也，僧孺所言者义也"——《资治通鉴·唐纪六十一》。司马光站在儒家义利之辨的立场，认为牛僧孺守信用、不贪得，是义；李德裕贪边功、弃信义，是利。可这个判断后世争议很大。南宋胡寅、朱熹从"理势"出发，不认同司马光，认为不能从言辞表面分别义利。司马光之所以偏向牛僧孺，和他自己在宋代新旧党争中的处境有关——他太警惕"贪功生事"了，以至于把具体的边防得失看得太轻。

第三句是王夫之《读通鉴论》里的长论，最见功力。他说："朋党兴，而人心国是如乱丝之不可理。"又说："邪正无定从，离合无恒势。"——朋党一起，人心和国是就像乱丝理不清；邪正没有固定的阵营，合离没有恒定的态势。

王夫之接着具体分析："李宗闵、牛僧孺攻李吉甫，正也；李德裕修其父之怨而与相排摈，私也。乃宗闵与元稹落拓江湖，而投附宦官以进，则邪移于宗闵、稹；而德裕晚节，功施赫然，视二子者有薰莸之异矣。"——《读通鉴论》卷二十六。

这段话极精彩。王夫之的意思是：一开始牛僧孺、李宗闵直言攻李吉甫，是正；李德裕报父怨排挤他们，是私。可后来牛党投靠宦官求进，邪就移到了牛党身上；而李德裕晚节建功立业，和牛党二人相比，就像香草和臭草的差别。薰莸，香草和臭草。

王夫之的高明处，在于他不贴标签。他不因为李德裕是"李党"就全盘否定，也不因为牛僧孺起家直言就一味肯定。他看的是具体事功、具体用心。这种辩证的眼光，比司马光"义利"二分更深刻。

李商隐在《太尉卫公〈会昌一品集〉序》里称李德裕"万古之良相"，是文人极致的推崇。李商隐写这序时，正受府主郑亚之命，而郑亚是李党要人，其时李德裕已贬死崖州——这是盖棺后的公论，不全是阿私。清代王士禛也说"李卫公一代伟人，功业与裴晋公伯仲"，把李德裕和裴度并列，可见后世对其功业的认可。

## 五、问道悟道

牛李党争这四十年，能提炼出几条本质规律。

第一条，党争一起，政策就成了人事的附庸。

维州事是最典型的例子。收复维州、收降悉怛谋，从军事和外交上看都是机会。可牛僧孺为压李德裕，一句话否了。这里头的判断标准，不是国家利益，而是党派利益——李德裕立的功，必须拆台。

这条规律古今皆然。北宋新旧党争，王安石变法时司马光反对，司马光执政后把新法不分青红皂白全废了，连保甲法、免役法这些本可保留的也一并扔掉。政策的好坏不再重要，重要的是谁立的法。明代东林党争，阉党和东林互相倾轧，辽东战事、剿寇方略都成了党争筹码，最后把明朝拖进深渊。

当派系利益凌驾于国家利益之上，再好的政策也会被扭曲，再坏的政策也会被推行。这是党争最致命的毒性。

第二条，人事与政策，本该分开，却最难分开。

李德裕重门第、轻科举，这主张有其道理——进士科重诗赋，确实选拔出一批浮薄之徒。可这主张从李德裕嘴里说出来，就是打击牛党的工具，因为牛党都是进士出身。牛僧孺姑息藩镇，也有其道理——安史之乱后朝廷确实打不动。可这主张从牛僧孺嘴里说出来，就是和李德裕对着干，因为李德裕主强硬。

政策和人事一纠缠，对的事也成了错的事。王夫之看透了这一点，所以他说"邪正无定从"——今天正的，明天可能就邪了，因为人事在变。

这条规律，在现代政治里也一样。西方两党制里，医保方案、移民政策、贸易关税，本是可以理性讨论的技术问题，可一旦贴上党派标签，就成了你死我活的立场。理性讨论的空间被压缩，妥协成了软弱，极端成了忠诚。

第三条，党争的根源，是"因私害公"的惯性。

司马光论维州事，纠结在义利之辨；可跳出维州看全局，牛李党争最深的病根，是私情压过了公议。这"私"不全是私心私利，更多是私见私情——我看你不顺眼，你做的事我就反对；我这一派的人，做错了我也护着。这种私情，比贪赃枉法更难治，因为它披着"原则"的外衣。

《左传》里讲"党锢之祸"，东汉党锢、唐代牛李、宋代新旧、明代东林，中国历史上党争反复重演。每一次都从几个人开始，每一次都越滚越大，每一次都以国力消耗告终。这背后的规律是：人性里"党同伐异"的本能，一旦没有制度约束，就会自我强化，直到把整个系统拖垮。

牛李党争给唐朝的伤害是实打实的。四十年里，朝政在内耗中空转，藩镇问题没解决，宦官问题没解决，财政问题没解决。会昌中兴那点起色，随着武宗一死、李德裕一贬，烟消云散。唐朝从宪宗时的"元和中兴"到宣宗时的"大中之治"，表面还有几段回光，可底子已经被党争蛀空。黄巢之乱不到四十年就爆发，唐朝离灭亡不远了。

## 结语

党争之害，不在争，在以私害公——政策成了人事的筹码，国事就成了党事的陪葬。
