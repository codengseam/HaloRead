# 唐纪·定鼎长安

## 一、讲事情

大唐开国这出戏，从大业十三年（617年）七月初五唱起，到武德元年（618年）五月甲子落幕，前后不过十个月。可这十个月里头，雨困霍邑、哭谏退兵、娘子军起、长安易主、江都惊变、禅让登基——一折接一折，折折都见功夫。

先说誓师。六月初五，李渊传檄各郡，宣布起义。可西河郡（今山西汾阳）不听令。李渊派李建成、李世民兄弟出征，临行撂下一句："除往返路程，只给三天军粮。"这是要试儿子们的成色。兄弟俩没让父亲失望——军纪严明，秋毫无犯，攻城时李世民冒险先登，李建成紧随其后。六月初十克城，只斩了指野鸟为鸾、阿谀隋炀帝的郡丞高德儒一人，余无所问。往返九日。李渊大喜，说了那句后来被温大雅记进《大唐创业起居注》的话："以此行兵，虽横行天下可也。"当天就定了"南下关中、西取长安"的大计。

可大计刚定，就遇上硬茬。代王杨侑镇守长安，得知李渊起兵，急命虎牙郎将宋老生率精兵两万屯霍邑（今山西霍州），左武侯大将军屈突通率数万骁果屯河东（今山西永济），一南一西，互为掎角。李渊三万人马沿汾水南下，过雀鼠谷，七月十四扎营贾胡堡，距霍邑五十余里。偏偏天不作美，连绵大雨，弓胶开解，粮道受阻。一困就是一个月。

更坏的消息传来——刘武周勾结突厥，要偷袭晋阳。李渊慌了。老巢若丢，前有坚城，后无归路，全军覆没。他召集将佐议事，意见一边倒：退兵。李渊碍于众议，下令撤军。可这时，十八岁的李世民不干了。

李世民连夜求见，侍卫通报"唐国公已寝"。李世民不硬闯，他跪在父亲帐外，放声大哭，哭声震帐。李渊被吵醒，召他进来，厉声问：又没死人，你嚎什么？李世民跪着说出了那段被司马光记进《资治通鉴》的话——"兵以义动，进战则克，退还则散。众散于前，敌乘于后，死亡无日，何得不悲？"起兵造反，是没有回旋的。前进则生，后退即死。父亲选了死路，儿子能不哭？李渊听完，沉吟。李世民又说：左军虽发，未去太远，右军未动，可追回。李渊笑了："吾之成败，皆在尔。知复何言，唯尔所为。"——成也是你们，败也是你们，由你们折腾去吧。

雨终于停了。太原粮草运到，突厥也安抚住。李渊问儿子：霍邑不能久留，怎么让宋老生出城？建成、世民笑道：宋老生勇而无谋，派轻骑挑战加谩骂，他必出。若还不出，就散布谣言说他跟我们一伙，隋军法严酷，他怕被告发，不敢不出。李渊抚掌大笑。

果然，轻骑一挑战，宋老生率三万人从东门、南门分道杀出。李渊、李建成在城东正面接战，李世民与段志玄从南面侧后冲杀。李世民手舞双刀，格杀数十人，两把刀全砍缺了，袖上尽是血。战至酣时，李渊让人大喊："老生被俘啦！"隋军斗志顿失，回头就跑。城门一关，把宋老生关在城外。老生慌不择路，连人带马摔进沟里，被斩。霍邑遂克。这一仗，三万对三万，是李渊起兵后第一场硬仗，打出了唐军的威风。

霍邑一破，屈突通就成了孤悬河东的一颗钉子。李渊不跟他死磕，绕道渡河，直扑关中。屈突通守河东，是隋朝的死忠，可他也知道大势已去。李渊派人劝降，屈突通哭了——"臣蒙国厚恩，历事两朝，岂可畏死贪生，背主求荣？"可他手下的人不干了。屈突通被部将桑显和劫持，不得已归降。李渊大喜，授他兵部尚书，封蒋国公。屈突通后来成了凌烟阁二十四功臣之一。

可李渊入关最大的助力，不是屈突通，是一个女人——他的三女儿平阳公主。

李渊起兵前，平阳公主与丈夫柴绍正在长安。李渊密召柴绍。柴绍犯难：一起去不行，独自走又怕妻子遭殃。平阳公主一句话定盘："君宜速去。我一妇人，临时易可藏隐，当别自为计矣。"——你快走，我一个妇人容易藏身，自有办法。柴绍走后，平阳公主回到鄠县（今陕西户县）李氏庄园，散尽家财，招了数百人，女扮男装，自称"李公子"，举旗响应父亲。

这"自有办法"四个字，分量重得很。短短三个多月，她派家僮马三宝游说胡商何潘仁归降，又收编李仲文、向善志、丘师利等义军，连克盩厔、武功、始平，聚兵七万。她申明法令，禁止侵掠，远近奔赴。隋朝名将屈突通在她手下连吃败仗。李渊渡河后，派柴绍迎她。平阳公主挑了一万精兵，与李世民会师渭北，与柴绍各置幕府，俱围长安。营中号曰"娘子军"——这就是"娘子军"三个字的来历。

十一月，长安城破。李渊没急着称帝。他迎立十三岁的代王杨侑为帝，改元义宁，遥尊隋炀帝为太上皇。自己任大丞相、尚书令，进封唐王。这一手，是政治上的高招——名义上还是隋臣，"尊隋"的旗号不倒，道义上就站得住脚。杀隋大臣阴世师、骨仪，余无所问，与民约法十二条，悉除隋苛禁。长安人心渐稳。

可天不假隋。义宁二年（618年）三月，江都兵变。隋炀帝杨广被宇文化及缢杀于江都宫。消息传到长安，隋朝最后一个象征没了。五月甲子，杨侑下诏禅位。李渊推让再三，群臣再请，李渊才在太极殿登基，国号唐，建元武德，定都长安。隋朝的大兴城改名长安城。一个叫"唐"的王朝，从此登上历史舞台。

## 二、讲人物

**李渊：被低估的开国之主**

后世一提大唐开国，眼睛都盯着李世民，李渊倒成了陪衬。可仔细看史料，李渊这个人，是真正的老谋深算。

李渊的功夫，下在起兵之前。他是西魏八柱国李虎之孙，与隋炀帝杨广是表兄弟，关陇贵族集团的核心人物。他到太原任留守后，对李世民说的那句话被温大雅记下来："唐固吾国，太原即其地焉。今我来斯，是为天与，与而不取，祸将斯及。"——唐国公的封号本就是我的，太原就是唐地。如今我来到这里，是天赐给我，不取反遭祸。这话说明，李渊到太原时，心里已经有了盘算。他让长子建成在河东"潜结英俊"，次子世民在晋阳"密招豪友"，自己则稳坐钓鱼台。

李渊最见功力的，是政治上的"尊隋"表演。他起兵不打反隋旗号，打的是"勤王定乱、迎回隋天子"的旗号，名义上拥护代王杨侑。攻下长安，立杨侑为帝，遥尊炀帝为太上皇。这一手，让他既得了实权，又保住了道义。直到炀帝被杀、隋朝名实俱亡，他才接受禅让。每一步都踩在"政治正确"上。后世说他是被李世民和裴寂"灌醉睡宫女"逼反的，这多半是后来的政治宣传——把开国的不臣之名推给儿子和部下，给父亲留个体面。

李渊的能听能断，从霍邑退兵一事看得最清楚。众议退兵，他碍于面子同意。可李世民一哭谏，他立刻改主意。王夫之在《读通鉴论》里说李渊"慎重之心，持之固，养之深"——这十个字，是李渊的真底色。

**李世民：十八岁的果决**

霍邑之战那年，李世民虚岁二十。可这个二十岁的少年，已经显出了日后贞观天子的影子。

雨困贾胡堡，众人都说退兵。李建成、李世民兄弟二人条分缕析地反驳：如今秋收在即，豆粟遍野，何愁乏粮？宋老生有勇无谋，一战可擒；刘武周与突厥貌合神离，不敢偷袭太原；李密贪图洛口仓藏，成不了大事；起兵为救苍生，遇敌即退，军心必散。这五条分析，条条切中要害。可李渊还是下令撤军。李世民不放弃，跪在帐外号哭，硬是把父亲哭醒了。

这一哭，哭出了大唐的命脉。若退回太原，李渊不过是割据一方的军阀，哪有后来的大唐？李世民的果决，在这一刻定了乾坤。霍邑战场上，他手舞双刀砍缺了两把刀，袖上尽血——这不只是谋略，更是敢拼命。后来贞观年间他回忆起兵往事，常说"朕少在太原，喜群聚博戏，暑不解衣"。这种少年气，到老都没消。

可这少年气里，也埋着后来的祸根。一个敢在父亲帐外号哭、敢身先士卒砍人的儿子，也是一个敢在玄武门杀兄屠弟的皇帝。霍邑城下的双刀，和玄武门前的弓箭，是同一个人的两面。

**李建成：被史书抹去的嫡长子**

李建成在开国这出戏里，戏份被后世削得厉害。可看温大雅的《大唐创业起居注》，李建成是起兵的核心人物之一。西河首战，他与李世民并肩先登；霍邑之战，他与父亲正面接战，吸引宋老生主力，给李世民侧后突袭创造机会。温大雅说他与世民"俱禀圣略，倾财赈施，卑身下士"——兄弟俩一个路数。

李建成身为嫡长子，起兵前一直在河东老家代行大家长职责，暗中联络豪强。李渊建大将军府，以建成为陇西公、左领军大都督，统左三军；以世民为敦煌公、右领军大都督，统右三军。左军在右军之上，这是嫡长子的位置。后来玄武门之变后，史书对李建成多有贬抑，说他"荒色嗜酒，畋猎无度"。可一个能在开国战争中独当一面的嫡长子，岂是庸才？《大唐创业起居注》是温大雅在武德年间写的，还没经过玄武门之变的政治清洗，里头的李建成，是个有谋略有担当的兄长。

**平阳公主：以军礼下葬的巾帼**

平阳公主是李渊与太穆皇后窦氏所生第三女，李世民的同母姐姐。她的故事，前面讲事情里已经说了。这里要说的是，她凭什么能以军礼下葬。

平阳公主的功，不在打仗本身，在"治军"。一个女子，三个月聚兵七万，靠的不是武力，是号令。史书说她"申明法令，禁兵士，无得侵掠，故远近奔赴者甚众"。军纪严明，不抢百姓，百姓才肯投奔。这与李渊西河之战"秋毫无犯"是一个路数——李家的人，都懂这个道理。

武德六年（623年），平阳公主去世。李渊下诏以军礼下葬——"加前后部羽葆鼓吹、大辂、麾幢、班剑四十人、虎贲甲卒"。太常官反对，说"以礼，妇人无鼓吹"。李渊反驳："公主亲执战鼓，率起义兵，辅成大业，她的功勋不亚于本朝功臣名将，堪比周之文母，列于十乱。应该以军礼下葬。"谥号"昭"——"明德有功曰昭"。她是中国历史上唯一以军礼下葬的公主。这一份荣光，不是李渊偏心，是平阳公主用七万兵马和一腔血性挣来的。

**屈突通：忠臣的最后一跪**

屈突通是这出戏里最让人唏嘘的角色。他是隋朝的死忠，镇守河东，阻击李渊。李渊绕道渡河后，屈突通派桑显和偷袭，被刘文静击退。李渊派人劝降，屈突通哭了，说出那句"臣蒙国厚恩，历事两朝，岂可畏死贪生，背主求荣"。可他的部下不干了，桑显和直接把他劫持，献给李渊。

屈突通见李渊时，李渊说："何相见晚耶？"屈突通哭着说："通不能尽人臣之节，力屈而至，为本朝之辱。"——我没尽到人臣的节操，力竭而来，是隋朝的耻辱。李渊敬重他的忠义，授兵部尚书，封蒋国公。后来屈突通随李世民平定王世充、窦建德，战功赫赫，名列凌烟阁二十四功臣。

屈突通这一跪，跪出了忠臣的无奈。隋朝气数已尽，他一个人挡不住。可他哭过、抵抗过，尽了人臣的本分。李渊敬他，敬的就是这份忠义——一个肯为旧主尽忠的人，也会为新主尽忠。这是用人的道理。

## 三、讲背景

定鼎长安这事，得放在关陇集团、隋末乱局、关中地利、政治旗号、突厥边患这五层背景上看。

先说关陇集团。陈寅恪先生在《唐代政治史述论稿》里提出"关陇集团"这个概念——从西魏到北周到隋唐，关中地区的军事贵族集团一直是权力的核心。西魏有"八柱国"：宇文泰、元欣、李虎（李渊祖父）、李弼、赵贵、于谨、独孤信、侯莫陈崇。这八家互相联姻，把持朝政。李渊的祖父李虎是八柱国之一，母亲是独孤信的女儿（与隋文帝皇后独孤伽罗是亲姐妹），妻子窦氏是北周武帝姐姐襄阳长公主的女儿。所以李渊与隋炀帝杨广是表兄弟，与北周皇室也是亲戚。这个集团从西魏到隋，已经统治了一百多年。隋炀帝的暴政，触动了这个集团的利益——他修运河、征高句丽、营江都，把资源和人口都往江南和辽东调，关陇集团的老本被掏空了。李渊起兵，本质上是关陇集团换一个代言人。所以李渊一呼，关中响应，不是偶然。

再说隋末乱局。大业七年（611年）王薄在长白山（今山东章丘）起义，唱"无向辽东浪死歌"，拉开反隋序幕。此后瓦岗军李密、河北窦建德、江淮杜伏威，群雄并起。隋炀帝大业十二年（616年）南下江都后再没回长安，中原大乱。到李渊起兵的大业十三年，隋朝对北方的控制已经名存实亡。这是李渊的天时——天下大乱，可关中还相对稳定，李渊据关中而望天下，进可攻退可守。

关中的地利，是这出戏的地理背景。关中东有崤函之险，西有陇蜀之阻，南有巴蜀之饶，北有胡苑之利。三面有险，独以一面东制诸侯。这是当年秦国灭六国的底子，也是后来汉朝立国的根本。李渊从太原南下，要进关中，必须先过霍邑、河东这一线。所以宋老生、屈突通死守这里——守住这里，就守住了关中的东大门。李渊绕过河东，从龙门渡河入关，是避实击虚。一入关，关中的粮、兵、险都归了他。平阳公主在关中起兵响应，等于给李渊铺好了红毯。

政治旗号是另一层背景。古代讲究"名正言顺"。李渊是隋朝的唐国公、太原留守，与隋炀帝是表兄弟。他若公然反隋，就是乱臣贼子，失了道义。所以他打"尊隋"旗号——名义上拥护代王杨侑，讨伐的是"乱隋"的奸佞，不是隋朝本身。攻下长安，立杨侑为帝，遥尊炀帝为太上皇，自己任大丞相、唐王。这一套，是学曹操"挟天子以令诸侯"和司马氏"以相代魏"的老办法。直到炀帝被杀，隋朝名实俱亡，他才接受禅让。每一步都踩在"政治正确"上，让天下人无话可说。温大雅在《大唐创业起居注》里把李渊写得处处"尊隋"，固然有美化成分，可这套政治表演的套路是真的。

最后说突厥边患。隋末突厥强盛，始毕可汗拥兵数十万。刘武周、梁师都、薛举等都勾结突厥。李渊起兵前，最怕的就是突厥与刘武周偷袭晋阳。所以他卑辞厚礼结好突厥，写给始毕可汗的信里自称"唐公"，措辞极谦。甚至答应"若入长安，民众土地入唐公，金玉缯帛归突厥"——土地归我，财宝归你。这是割肉饲虎的无奈。可李渊私下对刘文静说："胡骑入中国，生民之大蠹也。吾所以欲得之者，恐刘武周引之共为边患；又，胡马行牧，不费刍粟，聊欲藉之以为声势耳。数百人之外，无所用之。"——突厥骑兵进中国是百姓的大害。我要他们，只是怕刘武周勾结他们，顺便借点声势。几百人之外，没用。李渊心里清楚得很，突厥是虎狼，不能真靠。这一手"以夷制夷"，稳住了后方，才敢南下。

## 四、讲道理

王夫之在《读通鉴论》卷二十里专门评过唐高祖。王夫之说："人谓唐之有天下也，秦王之勇略志大而功成，不知高祖慎重之心，持之固，养之深，能为天下之主，非秦王之所可及也。"——世人都说唐朝得天下，是秦王李世民的勇略和志向大、功劳高。却不知道高祖慎重的心思，持得稳、养得深，能为天下之主，这不是秦王能比的。王夫之这话，是针对后世"重世民、轻高祖"的偏见说的。王夫之看李渊，看的是"慎重"二字——李渊起兵前观察形势、结纳人才、稳住突厥，每一步都不冒进。霍邑退兵一事，看似李渊犹豫，实则是他"慎重"的本性。李世民哭谏，他才下定决心。这种慎重，是开国之主该有的素质。王夫之这话的价值，在于把李渊从"被儿子推着走"的被动形象里解救出来，还原他作为开国决策者的本来面目。

《旧唐书·高祖本纪》末尾的史臣曰（刘昫等撰）说："高皇创图，谋深虑远，虽天命所归，亦资人谋之力。及至晋阳起兵，举事而有天下，盖圣人之大宝，岂徒然哉。"——高皇创基立业，谋深虑远，虽是天命所归，也靠人谋之力。到晋阳起兵，举事而有天下，圣人之大宝，岂是偶然。史臣看李渊，看的是"谋深虑远"——李渊的功，不在战场上的勇，在庙堂上的谋。这话与王夫之的"慎重"互为表里。史臣又说"天命所归，亦资人谋"——天命和人谋并重，这是公允的评价。不否认天命（时势），也不抹杀人谋（李渊的谋划）。

《新唐书·高祖本纪》的赞（欧阳修撰）说："自古受命之君，未有不由艰难者也。高祖之起，盖亦有不得已者。然其所以能成大业者，以其能用人也。"——自古受命之君，没有不经艰难的。高祖起兵，也有不得已之处。可他能成大业，在于能用人。欧阳修看李渊，看的是"能用人"——李渊自己不必是最能打的，可他能用最能打的人。建成、世民、平阳公主、屈突通、刘文静、裴寂，各尽其才。欧阳修这一赞，把李渊的用人哲学点出来了。

## 五、问道悟道

定鼎长安这事，藏着三条让人开眼的规律。

**合法性比实力更难造。**

李渊起兵时只有三万人，可他打的是"尊隋"旗号。这三万人的实力，远不如瓦岗军李密的百万之众，可李渊最终得了天下，李密却兵败降唐。差别在哪？在合法性。

李渊是隋朝的唐国公、太原留守，与隋炀帝是表兄弟。他起兵，名义上是"勤王"，是"尊隋"，是"废昏立明"。攻下长安，立代王杨侑为帝，遥尊炀帝为太上皇。这一套，让他在道义上站得住脚——我不是反贼，我是隋臣，我拥护隋朝的正统。直到炀帝被杀，隋朝名实俱亡，他才"不得已"接受禅让。每一步都踩在"政治正确"上。

反观李密。李密是瓦岗军首领，兵强马壮，可他打的是"反隋"旗号。他杀了隋朝的官员，占了隋朝的粮仓，名义上就是反贼。李密自称魏公，传檄天下，可天下诸侯没人真服他。李渊写信给他，他回信说"兄虽不肖，幸为盟主"——自封盟主。可这个盟主，没人认。李密最后兵败降唐，被李渊收编。

合法性这东西，看不见摸不着，可比实力更难造。实力可以靠打仗积累，合法性要靠名分、血统、旗号、仪式一层层建构。李渊有唐国公的爵位、关陇贵族的血统、隋臣的身份，这些都是合法性的资本。他用"尊隋"旗号把这些资本盘活，让天下人无话可说。这就是政治。

历史上这类事不少。曹操"挟天子以令诸侯"，用的是汉献帝的合法性；司马炎接受魏帝禅让，用的是曹魏的合法性；赵匡胤陈桥兵变黄袍加身，用的是后周的合法性。合法性这东西，本质上是"名正言顺"——名正了，言才顺；言顺了，事才成。李渊深谙此道。

**家族团队是开国的最大资本。**

李渊起兵，靠的不只是三万兵马，更是一个能干的家族团队。长子李建成统左三军，次子李世民统右三军，四子李元吉留守太原，女婿柴绍领右领军府长史，三女儿平阳公主在关中起兵响应。一家人，各当一面。

这个家族团队的合力，是李渊开国的最大资本。李建成在河东"潜结英俊"，李世民在晋阳"密招豪友"，平阳公主在关中聚兵七万。李渊还没起兵，三个孩子已经把人脉铺好了。起兵之后，建成、世民并肩作战，平阳公主独当一面。霍邑之战、长安之围，都是父子兄妹齐上阵。

这种家族团队的合力，历史上不多见。刘邦有萧何、张良、韩信，可这都是外人，不是家族。项羽有项梁、项伯，可项梁早死，项伯不成器。曹操有曹仁、曹洪、夏侯惇、夏侯渊，是家族团队，可曹操的儿子们没赶上开国。李渊的特别之处，在于他的儿女们个个能干，而且都在开国战争中独当一面。

可家族团队也是双刃剑。能共患难，未必能同富贵。李渊称帝后，立李建成为太子，李世民为秦王。兄弟俩一个有嫡长子的名分，一个有战功和兵权。这个矛盾，从开国那天就埋下了。武德九年（626年），玄武门之变，李世民杀兄屠弟，逼父退位。家族团队的开国资本，最终成了内讧的导火索。这道理，古今中外都一样——共患难易，同富贵难。

**关中是定鼎的根本。**

李渊从太原起兵，可他非要打进关中。为什么？因为关中是定鼎的根本。

关中的地利，前面讲背景时说过——东有崤函，西有陇蜀，南有巴蜀，北有胡苑，三面有险，独以一面东制诸侯。这是当年秦国灭六国的底子，也是汉朝立国的根本。李渊入关中，就有了粮、有兵、有险。关中稳，唐朝稳；关中乱，唐朝乱。后来安史之乱，唐朝靠关中和巴蜀撑住；黄巢起义攻破长安，唐朝就名存实亡了。

关中的价值，还在于它是关陇集团的老巢。李渊的祖坟、宗族、部曲，都在关中。他入关中，就是回家。关陇贵族见他回来，纷纷响应。平阳公主能在关中三个月聚兵七万，靠的就是关陇集团的人脉。这是地利之外的人和。

定都关中这规律，历史上一再重演。周朝都镐京（关中），灭商立国；秦国都咸阳（关中），灭六国统一；西汉都长安（关中），开四百年基业；隋朝都大兴（关中），统一南北；唐朝都长安（关中），开贞观开元盛世。反例也有——周平王东迁洛阳，周朝衰；东汉都洛阳，外戚宦官专权；唐末朱温逼昭宗迁都洛阳，唐朝亡。关中是进可攻退可守之地，洛阳是四面受敌之地。定都关中，是定鼎的根本。

## 结语

李渊从太原起兵到长安称帝，十个月走完了一个王朝的奠基礼——可真正定鼎的，不是那顶皇冠，而是"尊隋"的旗号、关中的地利、和一家人能共患难的合力。
