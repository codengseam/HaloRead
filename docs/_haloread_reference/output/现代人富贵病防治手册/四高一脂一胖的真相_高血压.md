# 血管里的高压锅

## 一、入戏：体检单上那个被忽略的箭头

老李拿到体检报告，扫了一眼血压栏：138/88。旁边没画箭头，他松了口气，"还行，没到140。"回到家照常喝酒吃肉熬夜。

半年后的一天清晨，他正洗脸，突然半边身子发麻，话也说不利索，紧急送医，诊断是脑卒中。妻子百思不得其解："他血压一直没事啊。"医生翻出体检单叹气：138/88 已经是"正常高值"，离高血压只差一步，而老李从没把它当回事。

## 二、破题：沉默的杀手

高血压的本质，是动脉里的血液对血管壁持续施加过高的压力。你可以把血管想象成一根水管，压力长期偏高，管壁就会变硬、变脆，连在管子上的心、脑、肾、眼这些"用水大户"也会被高压一点点冲坏。时间一长，心脏要更费力地把血泵出去，心肌肥厚、心衰；脑血管被冲得脆弱，某天破裂或堵塞就是中风；肾小球被高压破坏，慢慢走向尿毒症；眼底血管病变，视力模糊甚至失明。这就是医生常说的"靶器官损害"。

中国成人高血压患病率约27.5%，患者超过2.45亿——每四个成年人里就有一个。可它最阴险的地方在于：大多数人毫无感觉。头晕、头痛、耳鸣、后颈发紧这些症状，只在少数人身上出现，很多人直到中风、心梗、肾衰那天，才知道自己早就"中招"。所以医学界叫它"沉默的杀手"。它和癌症不同，癌症多少还有些信号，高血压却是悄无声息地腐蚀全身。

按《中国高血压防治指南（2024年修订版）》，在没吃降压药的情况下，非同日三次测诊室血压≥140/90 mmHg，或连续5—7天测家庭血压≥135/85 mmHg，就可以诊断为高血压。120—139/80—89 mmHg这个区间叫"正常高值"，就是老李待的那个危险地带。

## 三、方法论：把血压稳在安全线

**第一步：测对血压，别只信体检那一次。** 体检时紧张、赶路、刚喝过咖啡，都可能让血压虚高。建议买一台经过验证的上臂式电子血压计，安静坐五分钟再测，每次量两遍取平均值。家庭血压连续5—7天均值≥135/85 mmHg，就要去看医生。35岁以上的人，每年至少量一次。

**第二步：把盐罐子收一收。** 中国人吃盐太重，平均每天近10克，而《中国居民膳食指南（2022）》建议不超过5克。盐里的钠会把水分锁在血管里，血容量一上去，压力就上去。少盐不是不吃盐，是少用酱油、蚝油、咸菜、加工肉，炒菜出锅前再放盐，让咸味留在表面。同时多吃富含钾的蔬菜水果，钾能帮身体排钠。

**第三步：管住体重和酒杯。** 体重每减1公斤，血压大约能降1个毫米汞柱。腰围男超过90厘米、女超过85厘米，就要警惕。酒精则会直接让血压升高，长期大量饮酒的人，戒酒或限酒（男每日酒精<25克）后血压会明显回落。

**第四步：该吃药就吃药，长期规律。** 如果生活方式干预三到六个月血压仍不达标，或者一上来血压就很高，医生会开降压药。降压目标一般是<140/90 mmHg，能耐受的再降到<130/80 mmHg；合并糖尿病、肾病的更严格。药要每天按时吃，不能今天高了吃、明天低了停。王陇德院士反复讲过，慢性病防控核心是"吃动两平衡"，但已确诊的人，药物是底线，生活方式是加分项，两者缺一不可。

## 四、避坑：那些要命的"我以为"

**坑一：没头晕就没病，不吃药。** 高血压恰恰没症状才有杀伤力。等头晕了再吃，往往靶器官已经受损。临床上太多人中风前最后一次血压160，自己还觉得"精神好着呢"。

**坑二：血压降下来了就停药。** 降压药不是抗生素，吃完一个疗程就好。血压正常是药物压着的结果，一停就反弹，甚至比原来更高，这种"过山车"对血管伤害更大。该不该减量、换药，由医生说了算。

**坑三：靠芹菜、醋泡花生"降压"。** 这些食物当菜吃没毛病，但谁也不能靠它们把160的血压降到120。把偏方当主力，耽误的是正规治疗。

## 五、践行：本周三件事

1. 买一台上臂式电子血压计，每天早晚各量一次，记录一周，看看自己的真实血压。
2. 这一周把每天的盐控制在5克以内，炒菜少放酱油，用葱姜蒜柠檬提味。
3. 每天快走或慢跑30分钟，微微出汗即可，让血管也"松口气"。
