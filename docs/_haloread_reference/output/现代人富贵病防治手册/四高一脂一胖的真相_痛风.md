# 尿酸的账单

## 一、入戏：午夜脚趾的剧痛

凌晨三点，老陈被一阵剧痛惊醒。大脚趾根部又红又肿又烫，碰一下床单都像被针扎。他以为是扭伤，可白天明明没干重活。妻子扶他去医院，医生看了一眼就问："昨晚喝酒了？吃了海鲜？"老陈点头，昨晚同学聚会，啤酒配海鲜造了一顿。

"急性痛风发作。"血尿酸一查，560 μmol/L。老陈这才想起，去年体检尿酸就偏高，他嫌"没感觉"没管。这一夜，是他一年前欠下的尿酸账，连本带利一起还了。

## 二、破题：结晶在关节里"扎针"

尿酸是人体嘌呤代谢的产物，正常情况下由肾脏排出。可一旦生成太多或排不出去，血尿酸就会升高。血液里的尿酸浓度一旦超过它的溶解极限（体温37℃时约404.5 μmol/L），多余的尿酸就会以钠盐结晶的形式析出来，像一根根细小的针，沉积在关节、软组织和肾脏里。

这些"针"最爱待在温度低、血流慢的地方，所以大脚趾根部的第一跖趾关节是重灾区——半数以上痛风患者的第一次发作都在这里。免疫系统把这些结晶当成入侵者，派白细胞去围攻，结果引发剧烈的红肿热痛，这就是痛风急性发作。

它最大的误区是："痛风发作才治，不痛就不用管。"其实高尿酸血症（非同日两次血尿酸>420 μmol/L，2019年指南已不分性别统一为此标准）和痛风是同一个连续过程。中国高尿酸血症患病率13.3%，痛风1.1%，而且越来越年轻化。不痛不代表没事，结晶还在默默沉积，慢慢还会长出痛风石、堵出肾结石、拖垮肾功能。

## 三、方法论：把尿酸降下来，把账还清

**第一步：急性期先止痛，别急着降尿酸。** 痛风急性发作时，首要任务是抗炎止痛，而不是降尿酸——这时候突然降尿酸反而会诱发发作加重。指南推荐在发作24小时内尽早用药，常用小剂量秋水仙碱、非甾体抗炎药，必要时短期用糖皮质激素。等肿痛完全缓解两到四周后，再开始规律降尿酸治疗。

**第二步：缓解期把尿酸稳稳降下来。** 这是关键，也是很多人忽略的。降尿酸药常用三类：别嘌醇、非布司他（都属抑制尿酸生成的黄嘌呤氧化酶抑制剂）、苯溴马隆（促进尿酸排泄）。需要提醒的是，汉族人群使用别嘌醇发生严重超敏反应的风险较高，建议用药前先做HLA-B*5801基因检测。痛风患者血尿酸应长期控制在360 μmol/L以下，有痛风石或频繁发作的要更严格，控制在300 μmol/L以下，但不建议长期低于180。

**第三步：管住嘴，少嘌呤少果糖。** 动物内脏（肝、肾、脑）、浓肉汤、部分海鲜（沙丁鱼、贝类）嘌呤极高，急性期坚决不吃，缓解期也要少碰。啤酒和烈酒是痛风催化剂，能戒最好。含糖饮料里的果糖会加速尿酸生成，比海鲜还狠。取而代之，每天喝水2000毫升以上，帮助尿酸从尿里排走；多吃低脂奶制品、新鲜蔬菜、樱桃，研究显示这些对降尿酸有正向作用。

**第四步：控制体重，给代谢减负。** 肥胖会让尿酸生成增多、排出减少，BMI降下来，尿酸往往跟着降。但减重不能太快，饥饿或剧烈节食反而会诱发痛风发作，每月减2—4公斤比较稳妥。

## 四、避坑：痛风人的三个坑

**坑一：不痛了就停药，尿酸反复横跳。** 降尿酸是长期工程，结晶溶解需要时间，血尿酸达标后还得维持，自行停药几乎必定复发。曾正陪教授等专家反复强调，高尿酸血症是慢性病，要像管血压、管血糖一样管尿酸。

**坑二：急性期拼命降尿酸。** 发作时大把吃别嘌醇、非布司他，结果越降越痛。急性期只止痛，降尿酸留到缓解期。

**坑三：只忌海鲜不戒酒和甜饮料。** 海鲜嘌呤高没错，但啤酒本身含嘌呤前体还抑制尿酸排泄，果糖饮料更是隐形杀手。只盯海鲜、照喝啤酒奶茶，痛风照样来。

## 五、践行：本周四件事

1. 翻出体检单，看血尿酸值，超过420就预约复查一次（非同日两次确认）。
2. 这周每天喝水2000毫升以上，啤酒、白酒、含糖饮料全部暂停。
3. 暂停动物内脏、浓肉汤、贝壳类海鲜，用鸡蛋、低脂牛奶、豆腐替代蛋白质。
4. 每天快走30分钟，控制体重，但避免剧烈运动和突然受凉——这两样都易诱发发作。
