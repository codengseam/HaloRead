# 体重背后的真相

## 一、入戏：爬三楼喘成狗的中年人

老王四十二岁，身高一米七五，体重九十二公斤。他总说"我这是壮，不是胖"。直到有天陪儿子爬三楼回家，喘得说不出话，膝盖还隐隐作痛。体检报告：体质指数（BMI）30、血压偏高、空腹血糖6.5、血脂异常、脂肪肝——"富贵病全家桶"一项不落。

医生一句话戳破："你的体重，已经不是好不好看的问题，是还能不能健康活到老的问题。"

## 二、破题：肥胖是一种病，不是"福气"

很多人觉得胖点没事，甚至认为是"富态""有福气"。但现代医学早就把肥胖症定义为一种慢性病：身体里脂肪过度堆积，能量摄入长期大于消耗，代谢系统被压垮。

诊断先看体质指数（BMI）=体重（公斤）÷身高（米）的平方。中国标准：18.5—23.9正常，24—27.9超重，≥28就是肥胖。再看腰围，男≥90厘米、女≥85厘米叫"中心性肥胖"，也就是肚子大、内脏脂肪多的"苹果型"身材——这比均匀胖的"梨型"更危险，因为内脏脂肪会直接释放炎症因子，催生胰岛素抵抗。

据《中国居民营养与慢性病状况报告（2020）》，中国成人超重率34.3%、肥胖率16.4%，超重肥胖人数居全球第一，肥胖已成为我国第六大致死致残主要危险因素。它最大的误区是："胖就是馋和懒。"其实肥胖的成因很复杂，遗传、肠道菌群、睡眠不足、压力、某些药物都会推一把，单纯怪意志力是不公平的，也是不科学的。把它当病看，才能找到对的治法。

## 三、方法论：科学减重，别和身体硬碰硬

**第一步：定一个不贪心的目标。** 指南建议，超重和轻度肥胖者3—6个月减去原体重的5%—15%。一个90公斤的人，先减到77—85公斤就及格了，别一上来就要减30斤。目标定得越高，越容易半途而废和反弹。石汉平教授倡导"还营养为一线治疗"，强调减重不是挨饿，而是吃对——在保证蛋白质、维生素、矿物质的前提下，把总热量降下来。

**第二步：吃得对，比吃得少更重要。** 每天减少500—750千卡摄入比较稳妥，相当于少喝一杯奶茶加少吃一碗饭。结构上，碳水化合物占五到六成且多选全谷物，蛋白质一到两成选鱼禽蛋豆，脂肪两到三成多用植物油。蔬菜每天一斤，吃饭先菜后饭。指南还特别推荐适合中国人的"江南饮食"——多水产、多豆类、适量全谷物、少油少盐，比照搬西方节食法更可持续。

**第三步：动起来，而且要力量和有氧一起上。** 每周150分钟中等强度有氧（快走、慢跑、游泳）加2次力量训练。肌肉是身体的"耗能大户"，肌肉量上去了，基础代谢才高，瘦下来才不容易反弹。只做有氧、不练力量，减掉的往往是肌肉和水分。

**第四步：管住睡眠和压力。** 睡眠不足6小时，饥饿素升高、瘦素下降，第二天会控制不住想吃高热量食物；压力大时皮质醇升高，脂肪专门往肚子上堆。把作息和情绪理顺，减重事半功倍。

**第五步：必要时上药物或手术，别一个人扛。** BMI≥28、或≥24且有并发症，生活方式干预效果不好的，可在医生指导下使用药物，如GLP-1受体激动剂类药物（如司美格鲁肽）等，通过抑制食欲、延缓胃排空辅助减重。BMI≥35、或≥30合并糖尿病等且药物控制不佳的，可考虑减重代谢手术（如胃袖状切除术、胃旁路术），这是目前重度肥胖最有效的手段。药物和手术都不是作弊，是医学给重度肥胖者的正当工具，但都必须在医生评估和随访下进行。

## 四、避坑：减重路上的三个坑

**坑一：节食暴瘦，掉的全是水和肌肉。** 极低热量饮食（每天800千卡以下）必须在医生监督下短期进行，自己饿着瘦下来的，减的多是肌肉和水分，基础代谢崩了，恢复饮食后报复性反弹更胖。

**坑二：只看体重秤，不看腰围和体脂。** 有人体重没变，但腰围小了、肌肉多了，这是真瘦了；有人体重掉了，掉的却是肌肉，反而更虚。量腰围、看体脂率，比盯体重秤靠谱。

**坑三：迷信各种"速效"产品和代餐。** 网上打着"月瘦二十斤""排毒燃脂"旗号的产品，多含违禁成分或只是泻药利尿，伤的是肝肾。减重没有捷径，能长期坚持的，才是好方法。

## 五、践行：本周四件事

1. 算一下自己的BMI和量一下腰围，搞清自己在正常、超重还是肥胖哪个区。
2. 定一个3—6个月减5%—15%体重的目标，写在手机备忘录里，别贪多。
3. 这周每天少喝一杯含糖饮料、少吃一碗主食，多吃一份蔬菜和一份优质蛋白。
4. 每周快走或慢跑5次、每次30分钟，加2次深蹲、平板支撑等力量训练，连续坚持一个月。
