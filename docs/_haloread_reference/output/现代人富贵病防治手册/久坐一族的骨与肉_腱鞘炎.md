# 手腕的抗议

## 一、入戏：敲到一半，手腕卡住了

你是个程序员，一天码字十小时起步。那天写到兴头上，右手腕忽然一阵刺痛，拇指根部肿起一小块，活动时"咯噔"一下，像被什么东西卡住。你甩甩手继续敲，第二天更重了——大拇指一弯曲就弹响，得用另一只手掰直。再过一周，连拧瓶盖都疼。

这是腱鞘炎最常见的登场。鼠标和键盘用得越狠，手腕抗议得越早。它还有个"姐妹病"叫腕管综合征——正中神经在腕管里被压住，半夜被手指麻醒，甩甩手才缓。两种病常被混为一谈，其实机制不同，但都源于同一件事：手腕被过度使用了。

## 二、破题：摩擦出炎症，压迫出麻木

腱鞘是包在肌腱外面的双层管套，里层分泌滑液，让肌腱滑动顺畅。当手指、手腕反复做同一个动作——敲键盘、握鼠标、刷手机、抱孩子——肌腱和腱鞘之间摩擦加剧，滑液分泌跟不上，鞘壁充血增厚，肌腱在狭窄的鞘管里卡住，这就是狭窄性腱鞘炎。典型表现是活动时弹响、局部压痛，桡骨茎突腱鞘炎做握拳尺偏试验（Finkelstein试验）会剧痛。

腕管综合征是另一回事：腕管是腕骨和腕横韧带围成的"隧道"，正中神经和九条屈肌腱从中通过。手腕长期屈曲或肿胀，隧道里压力升高，正中神经被压，出现拇指、食指、中指麻木刺痛，夜间加重，甩手能缓解。人群患病率约3%-5%，女性是男性的3-6倍，与孕期激素、腕管解剖都有关。

最大的误区，是把手腕酸痛都当成"鼠标手"硬扛。腱鞘炎拖久会发展为慢性狭窄，肌腱粘连；腕管综合征拖久，大鱼际肌萎缩、握力下降，神经损伤难逆转。

## 三、方法论：让手腕歇下来，再换种用法

**第一步：制动，是最便宜也最有效的药。** 急性期用护腕或夹板固定手腕于中立位，2-3周内尽量减少触发动作。腱鞘炎可冷敷消肿（每次15分钟，每日3-4次）；腕管综合征夜间戴夹板保持腕中立，能显著降低夜间麻醒。筋伤最忌继续强拉硬拽，先让它歇住。

**第二步：消炎，分轻重度用药。** 轻中度用非甾体抗炎药（如布洛芬、双氯芬酸钠凝胶外涂）缓解炎症；顽固性腱鞘炎和腕管综合征可局部注射糖皮质激素（如曲安奈德），起效快，但同一部位每年不超过3次，多了会让肌腱变脆。注意：腱鞘炎封闭注射要打在鞘内不是肌腱里，必须由医生操作。

**第三步：换工具、换姿势，从根上减负。** 换人体工学键盘和垂直鼠标，让手腕保持中立而非背伸；键盘高度调到打字时手腕平直不翘。每打字45分钟，做一组手腕伸展：掌心向上，另一手轻压手指向后牵拉，停留15秒；再握拳松开重复10次，促进滑液循环。

**第四步：分清手术指征。** 腱鞘炎反复发作、出现扳机指无法伸直，可做腱鞘切开松解术；腕管综合征保守3-6个月无效、出现肌肉萎缩，需腕管切开减压。术后多数恢复良好，但前提是别拖到神经变性。

## 四、避坑：手腕养护的两个误区

**第一个坑：疼就使劲揉，越揉越肿。** 急性炎症期手法按摩、热敷会加重充血水肿，正确做法是制动加冷敷。等急性期过了（一般一周），再热敷促进循环。

**第二个坑：迷信小针刀"一针就好"。** 小针刀对部分慢性腱鞘炎有效，但属于有创操作，操作不当会伤及神经血管。一定要在正规医疗机构由有资质的医生做，别在街边理疗店随便让人扎。

## 五、践行：本周给手腕的五件事

1. 检查鼠标键盘高度，打字时手腕平直、不悬空、不背伸，必要时换垂直鼠标。
2. 设每45分钟提醒，做手腕伸展和握拳放松各一组。
3. 夜间手指发麻者，买一个腕部夹板，睡觉时保持手腕中立位。
4. 抱孩子、拧毛巾时双手交替，避免单手拇指长期外展发力。
5. 出现拇指弹响、手指持续麻木、握力下降，及时到骨科或手外科就诊。
