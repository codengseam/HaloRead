# 甜血的代价

## 一、入戏：体检空腹血糖7.2

小张三十二岁，程序员，体检空腹血糖7.2 mmol/L。他没当回事，"我又不渴不饿，瘦得跟竹竿似的，怎么会得糖尿病？"医生让他再查糖化血红蛋白（HbA1c），结果6.8%。

"已经是糖尿病了。"医生说。小张愣住。他不知道的是，中国成人糖尿病患病率已达11.9%，患者约1.4亿，居全球第一；更可怕的是，其中超过一半的人，和小张一样，根本不知道自己已经"甜血"了。

## 二、破题：胰岛素罢工的连锁反应

吃下去的米饭、面条、甜食，最后都会变成葡萄糖进入血液，给全身细胞供能。但葡萄糖进不了细胞，得靠一把"钥匙"——胰岛素来开门。胰岛素是胰腺分泌的激素，专门负责把血糖送进细胞里消耗掉。

2型糖尿病的问题出在两处：一是细胞对胰岛素越来越不敏感，叫"胰岛素抵抗"，门锁生锈了，钥匙打不开；二是胰腺长期超负荷工作，慢慢累趴下，胰岛素分泌越来越少。钥匙不够、锁又打不开，糖就只能堆在血里，血糖一路飙升。

它最大的误区是："瘦人不会得。"其实2型糖尿病的根源是代谢紊乱，胖人风险更高不假，但瘦人如果有家族史、久坐不动、爱吃精米白面甜饮料、长期熬夜，照样中招。早期很多人完全没有"三多一少"（多饮、多尿、多食、体重下降），等到口渴、消瘦、看东西模糊时，往往已经拖了好几年。

按《中国2型糖尿病防治指南》，空腹血糖≥7.0 mmol/L、或HbA1c≥6.5%、或口服葡萄糖耐量试验两小时血糖≥11.1 mmol/L，就能诊断为糖尿病。HbA1c反映的是近两三个月的平均血糖，比单次空腹更靠谱，所以体检别只查空腹血糖。

## 三、方法论：管住嘴迈开腿，把钥匙磨亮

**第一步：先看懂自己的血糖，定期复查。** 空腹血糖正常是3.9—6.1 mmol/L，6.1—7.0之间叫"糖尿病前期"，是悬崖边，及时干预能逆转。建议40岁以上、有家族史、超重、生过大孩子（出生体重≥4公斤）的女性，每年查一次空腹血糖加HbA1c。

**第二步：精制碳水换粗杂粮。** 白米饭、白馒头、甜饮料升糖最快，吃完血糖像坐火箭。把它们换成燕麦、糙米、杂豆、全麦，每顿饭先吃菜、再吃肉、最后吃主食，血糖上升会平缓很多。母义明教授强调过，糖尿病管理不是"不能吃"，而是"会吃"——量对了、顺序对了、粗细搭配了，比饿肚子管用。

**第三步：饭后动一动，给肌肉派活。** 肌肉是消耗血糖的大户，餐后快走20—30分钟，能显著压低餐后血糖高峰。每周再加两次力量训练，肌肉量上去了，胰岛素敏感性也会跟着改善。别小看这点，很多人靠运动把药量减了一半。

**第四步：该用药别硬扛。** 生活方式干预三个月HbA1c仍不达标（一般目标<7%），医生会开二甲双胍等口服降糖药，必要时加用胰岛素或新型降糖药。糖尿病可怕的不是吃药，是拖着不治。长期高血糖会像糖水泡零件一样泡坏全身血管：眼睛失明、肾脏衰竭要透析、脚趾溃烂要截肢、心脑血管堵塞要命。纪立农教授反复讲过糖尿病"三级预防"——一级防发病、二级防并发症、三级防致残，越早动手代价越小。

## 四、避坑：糖友常踩的三个坑

**坑一："是药三分毒，我先靠饮食控制。"** 早期轻症确实能靠饮食运动控制，但很多人用"控制"给自己拖延找借口，一拖拖到并发症出来。胰腺功能是会逐年衰退的，越晚用药，越被动。

**坑二：血糖一正常就停药。** 降糖药和降压药一样，是"压着"血糖的工具，停了就反弹。血糖波动比持续偏高更伤血管。

**坑三：迷信"降糖食物"和偏方。** 苦瓜、南瓜、秋葵当菜吃没问题，但没有哪样食物能代替药物。市面上打着"根治糖尿病"旗号的产品，基本是骗局。

## 五、践行：本周四件事

1. 看看体检单，如果空腹血糖在6.1以上，预约一次HbA1c复查。
2. 这周把白米饭的一半换成糙米或燕麦，吃饭顺序改成菜—肉—饭。
3. 每天三餐后各快走15分钟，不用跑，散步即可。
4. 含糖饮料这周全部换成白开水或淡茶，给胰岛放个假。
