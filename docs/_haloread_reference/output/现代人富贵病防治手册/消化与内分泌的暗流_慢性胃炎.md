# 胃的求救信

## 一、入戏：凌晨三点被痛醒的那个应酬季

凌晨三点，老陈被一阵闷痛从睡梦里拽醒。胃里像塞了块烧红的炭，反酸顶到嗓子眼，他撑着坐起来灌了半杯温水，压不下去。昨晚那场年终应酬，白酒黄酒轮着来，辣子鸡、毛血旺一筷子没少夹，他以为只是吃撑了，可这种凌晨被痛醒的事，这个月已经是第三次。

第二天他请了半天假去医院，医生让他做个呼气试验，结果阳性——幽门螺杆菌感染，慢性胃炎。老陈愣了一下："我胃口一直挺好的啊，怎么就胃炎了？"医生说，慢性胃炎最狡猾的地方，就是它常常不喊疼，等你感觉到，胃黏膜已经被磨了很久。

## 二、破题：胃黏膜上那场无声的消耗战

胃其实是个挺能忍的器官。它内壁覆着一层胃黏膜，分泌黏液和碳酸氢盐，把自己和强酸隔开，这套"黏液屏障"原本固若金汤。慢性胃炎的本质，就是这层屏障被一点一点破坏，炎症反复，腺体慢慢萎缩。

破坏屏障的头号嫌疑犯是幽门螺杆菌（Helicobacter pylori，HP）。它是目前已知唯一能在人胃酸里活下来的细菌，世界卫生组织早在1994年就把它列为Ⅰ类致癌物。它钻进黏液层底下，分泌尿素酶中和胃酸、放出毒素，让胃黏膜发炎、变薄，长期感染还会走向萎缩性胃炎、肠上皮化生，乃至胃癌。中国是HP高感染地区，现症感染率平均约44%，家庭聚集性极强，一人感染、全家阳性很常见。

帮凶还有一堆：长期饮食不规律、暴饮暴食、过辣过烫、烈酒空腹灌、熬夜；还有一类容易被忽略的——非甾体抗炎药（non-steroidal anti-inflammatory drugs，NSAIDs），像布洛芬、阿司匹林、双氯芬酸，它们在退烧止痛的同时，会直接削弱胃黏膜的保护因子。

最大的误区是一句话：胃痛忍忍就过去了。胃痛不是"上火"，是黏膜在喊救命。忍着、扛着、随便吞片止痛药压下去，等到吐血、黑便、胃穿孔才进急诊，往往已经从浅表性胃炎拖成了溃疡甚至更糟。

## 三、方法论：从查菌到养胃的四步走

慢性胃炎可防、可治，关键是有顺序，别乱来。

**第一步，先查菌，别凭感觉。** 反复腹胀、嗳气、反酸、餐后隐痛，或者有胃癌家族史，都建议做一次幽门螺杆菌检测。最常用的是尿素呼气试验（urea breath test，UBT），吹口气就行，灵敏度和特异度都在95%以上，无创、便宜。注意检查前要停质子泵抑制剂（PPI）两周、停抗生素四周，否则容易出现假阴性。家里人最好一起查，因为HP主要靠口-口、粪-口传播，共餐、夹菜、嚼碎喂孩子都是高危动作。

**第二步，阳性就规范根除，别半途而废。** 我国指南一线推荐铋剂四联疗法：一种质子泵抑制剂+铋剂+两种抗生素，疗程14天，规范服药根除率可达90%以上。难点不在方案，在依从性——药一天四次、吃满两周，很多人吃到第十天觉得好了就停了，反而培养出耐药菌。医生会根据当地耐药情况选抗生素组合，疗程结束四周后必须复查呼气试验，确认是否根除成功。根除成功后，胃炎的活动性炎症大多能消退，胃癌风险也明显下降。

**第三步，养胃不是喝粥，是改习惯。** 三餐定时、七八分饱、细嚼慢咽，比任何养胃茶都管用。少吃过烫、过辣、过咸和腌制熏烤食物，戒烟限酒，饭后别立刻躺下，睡前两小时不进食。《黄帝内经·素问》（战国至西汉成书）讲"食饮有节，起居有常"（传统养生参考），这八个字放到今天的胃病预防上，依然精准。

**第四步，定期胃镜，把风险掐在苗头。** 40岁以上、有HP感染史、有胃癌家族史、长期吸烟饮酒、爱吃腌制食品的人，建议每1到3年做一次胃镜。胃镜是唯一能直接看清黏膜、取活检、发现早期病变的手段。萎缩性胃炎、肠上皮化生、异型增生这些"癌前状态"，定期胃镜随访能让你在它还没变成癌之前就处理掉。

## 四、避坑：那些年我们坑过胃的方式

**坑一：胃痛了吃片布洛芬压一压。** 这是最危险的操作。布洛芬、阿司匹林这类NSAIDs本身就在伤胃，用它压胃痛等于火上浇油，临床上有相当比例的胃出血、胃溃疡就是吃止痛药吃出来的。胃痛该用的是铝碳酸镁、硫糖铝这类胃黏膜保护剂或抑酸药，分不清就找医生。

**坑二：症状一好就停药，复查也懒得做。** 四联疗法吃到第十天胃不疼了，自己把药停了；根除后没去复查呼气试验，第二年又复发。HP根除讲究"足量、足疗程、复查确认"，少一环都可能前功尽弃，还顺带制造耐药。

**坑三：一桌人夹菜不用公筷，全家反复感染。** 这是中国家庭最常见的循环：父母HP阳性，给孩子嚼食喂饭、用自己的筷子夹菜，孩子也感染；治好了又被家人传染回来。所以指南强调家庭同查同治，根除期间分餐、用公筷，是对家人最实在的保护。

## 五、践行：本周给胃的五条交代

1. **三餐定钟点**：早餐7到8点、午餐12点左右、晚餐18到19点，每餐七八分饱，把"饥一顿饱一顿"从日程里删掉。
2. **分餐公筷落地**：家里吃饭摆上公筷公勺，夹菜用它；有老人嚼碎喂孩子的习惯，温和地改掉。
3. **查一次HP**：全家成年人各做一次呼气试验，阳性一起治，阳性者治完四周后复查。
4. **清理药箱**：把布洛芬、阿司匹林这类NSAIDs从"胃痛急救药"的位置上拿下来，胃不舒服先找医生，别自己当大夫。
5. **约一次胃镜**：40岁以上或有胃癌家族史的人，把胃镜排进今年的体检日历，别等疼了才去。

胃不是被一顿饭吃坏的，是被上千顿乱吃的饭、上千个熬过的夜慢慢磨坏的。它发的每一封求救信，都值得你认真回。
