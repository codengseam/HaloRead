# 年纪轻轻，一身富贵病

## 一、入戏：体检单上密密麻麻的箭头

小林今年三十二，互联网公司产品经理，刚拿到年度体检报告，打开一看直接愣住：体质指数（BMI）28.4，超重；血压142/92，偏高；空腹血糖6.8，临界；尿酸480，已经超标；轻度脂肪肝；颈椎曲度变直。再往下翻，还有甘油三酯偏高、颈动脉斑块形成。他苦笑着把报告塞进包里："我才三十出头，怎么一身老年病？"

同事老周比他更惨，三十五岁，腰椎间盘突出压得坐不直，腱鞘炎让鼠标点不动，干眼症戴着两副眼镜上班，失眠两年靠褪黑素续命。两人一合计，部门里十个人，八个体检报告上都有箭头，剩那两个还是刚毕业的应届生。

这场景，你多半不陌生。我们这代人，明明吃得比父辈好、住得比父辈强、医疗条件比父辈优越，怎么反倒一身病？

## 二、破题：富贵病不是"富贵"病

"富贵病"这个词容易让人误会，以为是有钱人才得的病。其实它最早是说，营养过剩、活动太少那一类"生活方式病"——高血压、糖尿病、痛风、脂肪肝、肥胖、心脑血管病，再加上久坐带来的颈椎腰椎毛病、熬夜带来的失眠焦虑、屏幕带来的干眼脱发。它们不是"富贵"导致的，恰恰相反，是现代人生活方式出了岔子。

一句话概括富贵病的本质：吃得多、动得少、坐得久、睡得晚、想得太多。

注意，这里的关键不是"营养缺乏"，而是"营养过剩+生活方式紊乱"。父辈那代人缺油缺肉，得的是营养不良、感染性疾病；我们这代人油盐糖过剩、却久坐不动，得的是代谢病、慢性病。病种换了一茬，根源完全不同。所以靠"补"——补保健品、补营养品、补各种丸——是补不回来的，富贵病要"减"：减食量、减久坐、减熬夜、减焦虑。

这就解释了为什么年轻人也开始中招。富贵病不再是有钱人的专利，而是现代生活方式的"出厂配置"——只要你天天坐办公室、吃外卖、刷手机、熬夜加班，就有很大概率被它盯上。

## 三、方法论：二十年养出一代富贵病

富贵病不是一天爆发的。回看中国近二十年，几股力量叠加在一起，把整个社会的代谢指标推上了悬崖。

**第一股力量：城镇化与久坐办公。** 二十年前，多数人上下班骑车走路，工作时多少还要动一动。今天，地铁公交私家车解决了通勤，电梯解决了爬楼，外卖解决了取餐，从早到晚，人几乎可以不动弹地完成一天所有事。身体活动不足成为普遍现象，每周至少锻炼一次的成人比例不到四分之一。代谢是要"用"的，不用就堆。

**第二股力量：外卖文化与超加工食品。** 中国家庭人均每日烹调用油达43.2克，超过推荐值近一半；膳食脂肪供能比已升到34.6%，农村首次突破30%上限。餐馆和外卖为了好吃，油盐糖一样不少；超加工食品——方便面、含糖饮料、零食——便宜、方便、上瘾。18.9%的中小学生经常喝含糖饮料，成年人更不必说。能量进得多、出得少，剩下的全变成内脏脂肪。

**第三股力量：智能手机与屏幕生活。** 手机把我们的眼睛、颈椎、睡眠、心理同时绑架。低头看屏幕颈椎受压倍增，刷短视频让干眼症和失眠高发，社交比较和负面信息催生焦虑抑郁。中国干眼症患病率约21%-30%，脱发人数超2.5亿，焦虑障碍终生患病率7.6%——这些数字背后，都站着一部手机。

**第四股力量：内卷压力与睡眠透支。** 996、KPI、房贷、教育，把人压在长期应激状态里。皮质醇长期偏高，血糖血压跟着乱；夜里舍不得睡，熬夜刷手机把生物钟彻底打乱。睡眠不足直接推高肥胖、糖尿病、高血压风险，形成恶性循环。

把这些加在一起，再看国家卫健委《中国居民营养与慢性病状况报告（2020）》的数据，就不会觉得意外：中国成人超重肥胖率超过50%（超重34.3%+肥胖16.4%），高血压患病率27.5%，糖尿病患病率11.9%，高胆固醇血症8.2%；因慢性病导致的死亡占总死亡的88.5%。换算成身边的话：每两个成年人就有一个超重或肥胖，每四个就有一个高血压，每八九个就有一个糖尿病。

王陇德院士——中国工程院院士、原卫生部副部长——长期呼吁，慢性病是国家的重大公共卫生问题，防控核心是"吃动两平衡"。这句话听起来朴素，却点中了富贵病的命门：它们看似分散在各个科室，骨子里却有同一个根源——能量过剩、久坐不动、压力过大、睡眠不足。抓住根源，许多病就能一并挡住；只盯单点，往往是按下葫芦浮起瓢。

## 四、避坑：富贵病最常见的三个误解

**误解一：年轻就不会得富贵病。** 这是最大的错觉。今天痛风、脂肪肝、2型糖尿病、颈椎病在三四十岁甚至二十多岁人群中已经不稀奇。富贵病年轻化是近十年最明显的趋势，"我还年轻"恰恰是最危险的心理。

**误解二：体检正常就没事。** 体检只是快照，而且多数富贵病早期没症状。高血压要持续测才准，糖尿病前期空腹血糖可能还在6.1-6.9之间"看起来正常"，脂肪肝轻度时B超都可能漏诊。体检正常不等于健康，还要看腰围、看睡眠、看体力、看情绪——这些体检单不会写。

**误解三：靠保健品就能逆转。** 鱼油、辅酶Q10、葡萄籽、各种"护肝片"卖得火爆，但没有任何一种保健品能替代生活方式。石汉平教授讲过一句很到位的话："还营养为一线治疗"——意思是营养本身要被当成治疗手段，而不是靠几粒胶囊糊弄过去。富贵病的根在生活方式，根不动，保健品只是花钱买心理安慰。

## 五、践行：从这本手册开始

把这本手册当成一份自救说明书，按三个步骤用起来。

**第一，按章阅读。** 手册按"骨与肉—四高一脂一胖—头面五官—消化内分泌—精神神经—四大基石"的顺序展开，每一章对应你身上可能出问题的部位。先通读开篇和四大基石，建立全局观，再按自己最担心的章节细读。

**第二，对照自查。** 每读一章，对照自己的体检单、家族史和日常症状，看看有没有踩中。腰围量一量，血压量一量，睡眠时长记一记，把"可能有"的那几项标出来。

**第三，建立习惯。** 富贵病不是靠一次治疗解决的，是靠日复一日的生活方式逆转的。从这本手册里挑出三件最该改的事——比如少坐一小时、少喝一杯含糖饮料、早睡半小时——先做一个月，再迭代。别贪多，能坚持下来的小改变，胜过三天打鱼的大计划。

下一章，我们会讲怎么把这本手册用得更顺手，再帮你建一份属于自己的健康档案。
