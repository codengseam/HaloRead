# 肝脏的脂肪危机

## 一、入戏：体检B超上的"脂肪肝"三个字

小刘二十八岁，微胖，体检B超报告写着"轻度脂肪肝"。同事安慰他："现在十个人八个脂肪肝，没事。"他也就信了。三年后复查，"中度脂肪肝"，转氨酶也偏高，医生让他查血糖，果然已经是糖尿病前期。

他这才慌了。原来脂肪肝不是"小毛病"，它是代谢紊乱在肝脏亮起的红灯，红灯灭不灭，决定了下一步是肝炎、肝硬化还是肝癌。

## 二、破题：脂肪怎么跑进了肝细胞里

正常肝脏也含少量脂肪，但当肝细胞里脂肪（主要是甘油三酯）堆积超过肝脏重量的5%，就是脂肪肝。我们这里说的是"非酒精性脂肪性肝病"——不是因为酗酒，而是因为吃得多、动得少、胖起来的那种。近年学界已把它更名为"代谢相关脂肪性肝病"，名字变了，本质更清楚：它是代谢综合征在肝脏的表现，核心推手是胰岛素抵抗。

脂肪肝的进程像爬楼梯：先是单纯性脂肪肝，肝细胞被油塞满但还没发炎；接着是脂肪性肝炎，肝细胞被撑破、发炎、坏死；再往下是肝纤维化、肝硬化，少数人会发展成肝癌。中国成人脂肪肝患病率超过25%，部分地区高达30%，已取代慢性乙肝成为我国第一大慢性肝病。更扎心的是，2型糖尿病患者中脂肪肝检出率高达六到八成——糖和脂，本来就是一根藤上的瓜。

它最大的误区是："脂肪肝不是病，不用治。"轻度时确实能靠生活方式逆转，但放任不管，它绝不会自己消失，只会一路往坏的方向走。

## 三、方法论：减重，是脂肪肝最好的"药"

**第一步：搞清自己到了哪一步。** 单靠B超只能看出"有没有"，看不出"发不发炎、纤不纤维化"。如果转氨酶升高或合并糖尿病、肥胖，建议进一步做肝纤维化无创评估（如FibroScan弹性成像、血液指标评分），必要时肝穿刺。知道自己站在楼梯第几级，才知道要往回走多少。

**第二步：减重5%—10%，脂肪肝就能往回退。** 这是目前证据最充分、也最有效的方法。范建高教授主导的指南反复强调，减少体重和腰围是脂肪肝最重要的治疗措施。每个月减2—4公斤比较安全，减太快反而可能加重肝损伤。腰围男降到90厘米、女85厘米以内，肝脏的负担会明显减轻。

**第三步：吃得"瘦"一点，给肝脏减油。** 限制总热量，尤其是含糖饮料、精制碳水和高油食物——果糖在肝脏里直接转成脂肪，比肥肉还容易养出脂肪肝。多吃蔬菜、全谷物、优质蛋白（鱼、豆、瘦肉），用橄榄油、坚果里的好脂肪替代动物油和反式脂肪。有研究发现，地中海式饮食对脂肪肝改善明显。

**第四步：动起来，把多余的油烧掉。** 单纯节食会掉肌肉，配合运动才减得对。建议每周5次中等强度有氧运动（快走、慢跑、骑车），每次30分钟以上，再加2次力量训练。运动能改善胰岛素敏感性，从根子上断掉脂肪肝的燃料。

**第五步：必要时加保肝药，但别指望它单挑。** 如果已经是脂肪性肝炎或肝纤维化，医生会在生活方式干预基础上加用保肝抗炎药物，阻止肝病进展。但药物是辅助，生活方式才是主力——不减肥、照常吃喝，吃再多保肝药也压不住。

## 四、避坑：脂肪肝的三个坑

**坑一："我又不胖，不会有脂肪肝。"** 体重正常的人也可能得，叫"瘦人脂肪肝"，多和遗传、快速减重、营养不良、爱吃甜有关。瘦不是护身符，腰围粗、血脂高的人更要查。

**坑二：靠"保肝片"熬夜喝酒。** 体检发现脂肪肝，不戒酒不减肥，天天吞护肝片，等于一边堵漏一边放水。保肝药救不了继续作的人。

**坑三：快速节食暴瘦。** 饿肚子暴瘦会让脂肪快速分解涌入肝脏，反而加重脂肪肝甚至引发肝损伤。减肥要稳，不能猛。

## 五、践行：本周四件事

1. 翻出体检单，看B超有没有"脂肪肝"、转氨酶高不高，有的话预约一次肝纤维化评估。
2. 这周戒掉所有含糖饮料和糕点零食，把主食的三分之一换成杂粮。
3. 每天快走或骑车30分钟以上，每周加两次深蹲、俯卧撑等力量训练。
4. 量一下腰围，男超过90厘米、女超过85厘米，就给自己定个三个月减5%体重的目标。
