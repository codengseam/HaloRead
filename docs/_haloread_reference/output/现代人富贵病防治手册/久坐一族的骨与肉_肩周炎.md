# 冻住的肩膀

## 一、入戏：连扣内衣都成了难事

五十岁出头的母亲最近总是闷闷不乐。问她怎么了，她半天才说：夜里肩膀疼醒，翻个身都疼；早上穿衣服，手伸不到背后，连扣内衣都费劲。她以为是受凉，贴了一堆膏药没用。后来连梳头、晾衣服都做不到，胳膊像被冻住了一样，才肯去医院。

这是肩周炎最典型的模样。它又叫"冻结肩""五十肩"，好发于40-60岁，女性多于男性，普通人群发病率约2%-5%。它还有个特点：糖尿病患者发病率是普通人的2-4倍，血糖控制差的更易粘连、恢复更慢。久坐一族里，长期伏案、肩部受凉、缺乏活动的人，也是高发人群。

## 二、破题：冻住的不是骨头，是关节囊

很多人以为肩周炎是"肩关节发炎"，其实更准确的说法是粘连性肩关节囊炎——发炎纤维化的是包在肩关节外面的关节囊。

正常关节囊薄而有弹性，让肩关节自由转动。不知什么原因（可能与年龄退变、激素变化、微炎症有关），关节囊开始充血、增厚，接着纤维组织增生、粘连挛缩，关节囊容积缩小，肩关节活动范围一点点被"锁死"。整个过程分三期：疼痛期（2-9个月）以夜间痛为主；冻结期（4-12个月）疼痛减轻但僵硬达顶峰，梳头穿衣都难；解冻期（5-24个月）慢慢恢复。总病程1-3年，有自愈倾向，但约10%-20%的人会残留功能障碍。

最大的误区有两个：一是把所有肩痛都叫"肩周炎"。肩袖损伤、肩峰下撞击都会肩痛，但肩袖损伤是"自己抬不起来、别人帮能抬起来"，肩周炎是"自己抬不起来、别人帮也抬不起来"——主被动活动都受限。二是疼就不动，越不动粘得越紧。

《黄帝内经·素问·痹论》（战国至西汉成书）讲"风寒湿三气杂至，合而为痹也"——风、寒、湿交织侵入人体会形成痹证。肩周炎民间称"漏肩风"，正合此理，受凉是常见诱因之一。

## 三、方法论：止痛、松解、练活动度

**第一步：疼痛期，先消炎止痛。** 夜间痛剧烈时，口服非甾体抗炎药（如塞来昔布、依托考昔）缓解炎症；痛点明确的可在关节腔或滑囊注射糖皮质激素（如复方倍他米松），起效快，但每年不超过3次。这个阶段以静养为主，配合温和的钟摆运动——弯腰让患臂自然下垂，利用重力顺时针、逆时针画圈，幅度由小到大，每天3-5次每次5分钟。切忌暴力拉伸，会加重炎症和粘连。

**第二步：冻结期，重点在松解粘连。** 疼痛一减轻，就要主动练活动度，否则关节囊越粘越死。经典动作有三个：爬墙运动——面对墙站立，手指沿墙缓慢向上爬至微痛处停留5-10秒，每天10次；毛巾操——双手背后各握毛巾一端，健手带动患手向上拉，牵拉感保持10秒；外旋拉伸——患臂屈肘90°贴身，用门框或健手辅助向外旋转。训练以牵拉感为主，出现锐痛立即停。

**第三步：顽固病例，考虑手法松解或关节镜。** 保守3个月以上无效、活动严重受限的，可在麻醉下手法松解，或关节镜下松解粘连。术后必须立刻开始康复训练，否则会重新粘连。糖尿病患者更要严格控制血糖，否则恢复慢、复发率高。

## 四、避坑：肩痛养护的两个坑

**第一个坑：肩痛就当肩周炎，拼命爬墙。** 如果其实是肩袖撕裂，暴力锻炼会让撕裂扩大。肩痛第一步该做的是明确诊断：肩周炎主动被动都受限，肩袖损伤主动受限被动正常，磁共振成像（MRI）或超声能鉴别。诊断不清就猛练，是肩痛康复里最大的坑。

**第二个坑：疼就不动，等它自己好。** 肩周炎虽有自愈倾向，但"等"的代价是1-3年的功能丧失，还可能残留僵硬。疼痛期一过就该主动活动，"忍着痛练"比"养着等"恢复得快得多。

## 五、践行：本周给肩膀的五件事

1. 检查肩部是否受凉，空调和风扇别直吹肩颈，夜间肩部盖一层薄被。
2. 每天做钟摆运动3组，每组顺逆时针各20圈，弯腰让患臂自然下垂。
3. 面墙做爬墙运动，每天10次，记录手指到达高度，看着进度一点点往上。
4. 糖尿病患者监测血糖，糖化血红蛋白（HbA1c）控制在7%以下，降低粘连风险。
5. 肩痛超过两周不缓解，先做超声或磁共振明确是肩周炎还是肩袖损伤，再决定怎么练。
