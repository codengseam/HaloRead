## 概念

### 格局是什么

紫微斗数里的「格局」，指的是星曜在命盘上形成的特定组合模式。当某些主星、辅星、煞星按特定方式聚到特定宫位或特定三方四正里，就会构成一个有名字的「格」。古籍与近代讲义里录有百余格，本篇只挑最常用、最常被提起的几组来讲，重点是明白格局成立的条件，而非背名字。

需要先把心态摆正：格局是「趋势的命名」，不是「命运的封印」。一个格局成立，只意味着这张盘上某种星曜组合的能量比较突出，读盘时要多留意这一组的互动。格局本身不分好坏——有些古格名字听起来富贵（如紫府同宫），有些听起来凶险（如杀破狼），但实际吉凶要看星曜亮度、煞星同宫、四化引动等多重变量。这一点在 6-2 富贵贫贱格的实与虚里会专门展开。

### 格局的命名传统

格局的命名大致分三类：

- **以星曜组合命名**：如紫府同宫、日月并明、杀破狼，直接点出是哪几颗星聚在一起。
- **以星曜与宫位关系命名**：如日月反背（太阳太阴落于特定宫位）、马头带箭（擎羊坐于午宫命宫）。
- **以象意命名**：如火贪格（火星贪狼同宫主爆发）、刑囚夹命（擎羊与廉贞夹命宫）。

古格名多出自《紫微斗数全书》《紫微斗数全集》及清钞本，近代讲义又补入不少实战派总结的格局。本篇所选几组，是中州派、三合派、飞星派都提到的「公共语汇」。

### 三方四正在格局中的作用

绝大多数格局的成立，要看**三方四正**——本宫加三合两宫加对宫。三方四正的概念在 3-3 已经讲过：本宫 + 顺时针第四宫 + 逆时针第四宫 + 对宫。例如命宫的三方四正是命宫 + 财帛宫 + 事业宫 + 迁移宫。判断一个格局成立，常常不只看本宫坐星，还要看三方四正里有没有凑齐所需的星。

需要提醒的是，三方四正的权重在中州派、三合派里非常高；飞星派则相对弱化三方四正，更重四化飞星的来去。两种路数对同一格局的成立判断，可能宽严不一。本篇在描述成立条件时，先依传统三合派的严标准写，飞星派的变通用法在 6-2 再展开。

## 原理

### 紫府同宫

**成立条件**：紫微与天府同坐于命宫（或命宫三方四正里 strong 见到），主要出现在寅宫或申宫——因为紫微与天府在这两个宫位恰好同度。

**传统象意**：紫微是北斗主星、帝星；天府是南斗主星、财库。两颗主星同宫，叫「紫府同宫」，传统上主尊贵、有领导力、能掌大权大财。古格名很响亮。

**实际读法**：紫府同宫的盘，主星过强，往往需要左辅右弼、天魁天钺等吉星会照才稳，否则容易孤君无援。又因紫府同宫必在寅申，对宫必是七杀，七杀的冲劲会反过来牵制紫府的稳健。所以紫府同宫的人，看似尊贵，实则一生在「稳」与「冲」之间拉扯。

### 日月并明

**成立条件**：太阳、太阴分别坐于庙旺之宫，且能会照命宫。最典型的是太阳在巳宫、太阴在丑宫（或太阳在辰、太阴在戌），二曜皆旺，会照命宫。

**传统象意**：日月并明主名声显赫、事业有成、能得父母庇荫。古格录为吉格。

**实际读法**：日月并明的关键不在「并明」二字，而在二曜是否真的庙旺。若太阳落陷（如太阳在戌亥子丑）而太阴庙旺，叫「日月反背」，象意完全不同，主辛劳背井。所以判这一格，先查二曜亮度，再判吉凶。

### 杀破狼（七杀、破军、贪狼）

**成立条件**：七杀、破军、贪狼三颗星始终在三方四正里同会，只要命宫坐其中之一，其余两颗必在三合宫位。这是一个稳定成立的组合，凡是命坐杀破狼之一，都属此格。

**传统象意**：杀破狼主变动、开创、奔波。三颗星性格各异——七杀主肃杀与决断、破军主破立与开创、贪狼主欲望与交际。古格多视杀破狼为动荡之格，主一生多变。

**实际读法**：杀破狼的「变」并非坏——现代社会里，杀破狼格的人往往适合创业、销售、技术变革等需要闯劲的领域。古格的「动荡」贬义，部分是阶层语境产物（农业社会重稳定轻变动）。这一格的吉凶，要看是哪颗星坐命、有没有煞星同宫、有没有化禄权加持。杀破狼逢禄权，往往是创业有成之格；逢忌煞，则易成无序动荡。

### 火贪格（火星贪狼同宫）

**成立条件**：火星与贪狼同坐于命宫（或三方四正会入命宫），贪狼庙旺。又有铃星贪狼同宫的「铃贪格」，象意相近。

**传统象意**：火贪格主「爆发」——一夜暴富、突然成名、机遇陡至。古格录为吉格，谓「火贪相逢振武威」。

**实际读法**：火贪格的「爆发」是真的，但「爆发」之后能不能稳住，要看后续大限流年的引动。火贪格的人常人生起伏大——上去快，下来也快。现代派看这一格，多提醒「爆发后须守成」，不能只看「发」不看「守」。

### 七杀、破军单星坐命

杀破狼三颗星里，七杀与破军单星坐命时各有特定读法：

- **七杀坐命**：主肃杀、决断、独立。七杀在寅申巳亥四马地坐命，主一生奔波有成；在子午卯酉坐命，则刚而易折。
- **破军坐命**：主破立、开创、不守成。破军坐命的人，一生中常有「破旧立新」的事——换行业、换地方、换关系。古格多视破军为耗星，但现代社会里破军坐命者常是改革者、创业者。

### 机月同梁

**成立条件**：天机、太阴、天同、天梁四颗星在三方四正里会齐，命宫坐其中之一。这四颗星必在三方四正里相互会照，是一个固定的组合。

**传统象意**机月同梁主稳定、佐助、按部就班。古格谓「机月同梁作吏人」，主适合公职、文职、稳定的体制内工作。

**实际读法**：机月同梁格的人性格偏稳健、不爱冒险，适合稳定职业。这一格的吉凶较平，不会有火贪那样的爆发，也不会有杀破狼那样的剧烈动荡。现代派看这一格，多提醒「稳定是优势也是局限」——机遇来时要敢于跳出舒适区。

### 府相朝垣

**成立条件**：天府与天相在三方四正里会照命宫。天府是财库，天相是印星，二曜朝垣，主有财有印。

**传统象意**：府相朝垣主富贵双全、有财有权。古格录为吉格。

**实际读法**：府相朝垣的盘，结构稳健，财权两全的概率确实较高。但同样要看煞星与四化——若府相被煞星冲破，或化忌所伤，吉格也成空。古格的成立不等于吉格的实现。

## 实践

### 怎么判一个格局成立

实战中判一个格局是否成立，建议按下面顺序：

1. **看本宫坐星**：命宫坐什么主星？这是格局的根基。
2. **看三方四正会入什么**：三合宫与对宫里有没有凑齐格局所需的星？
3. **看星曜庙旺**：所需的星是否在庙旺之地？落陷则格局打折。
4. **看煞星同宫与会照**：有没有煞星冲破？煞星多则格局减分。
5. **看四化引动**：格局所需的星有没有被禄权科加持或被化忌所伤？
6. **看大限流年引动**：进入某大限后，格局所需的星有没有被引动？这是 6-3 的内容。

这六步是判格局的严标准。中州派、三合派大体按此判，飞星派则常省略三方四正，直接看四化飞星的引动。两种路数并陈，是 6-2 的主题。

### 几个格局的实战读法

举几个常见组合的实战读法：

- **命坐紫府同宫，左辅右弼来会**：君臣庆会，紫府有辅，主能掌大权、得人助。
- **命坐杀破狼，逢火铃同宫**：动荡中有爆发，常主创业有成，但起伏大。
- **命坐机月同梁，化科会入**：稳定中得名声，主适合学术、公职、专业领域。
- **命坐府相朝垣，煞星冲破**：财权双全之格被煞破，主有财权但辛苦。

### 流派差异提醒

本篇所选格局属中州派、三合派、飞星派都提到的公共语汇，但三派对格局的权重并不一致：

- **三合派**：重格局成立条件，看三方四正与星曜组合。一个格局成立，常常直接据此断吉凶。
- **中州派**：也看格局，但更重星曜庙旺与四化。格局只是辅助。
- **飞星派**：弱化格局，更重四化飞星的来去。一个格局成立与否，飞星派常常不深究，直接看四化。

这种差异在 6-2 会专门讨论。本篇先按三合派的严标准讲成立条件，让读者有一个公共底座。

1. 看本宫坐星。
2. 看三方四正会入。
3. 看星曜庙旺。
4. 看煞星同宫与会照。
5. 看四化引动。
6. 看大限流年引动。

### 自省问题清单

- 我能不能说出本篇六个格局各自的成立条件？说不全就回去重读原理段。
- 杀破狼为什么是「稳定成立」的格局？因为三颗星始终在三方四正同会。
- 火贪格的「爆发」之后，我有没有想过「守成」？
- 紫府同宫看似尊贵，为什么实战里常主「稳与冲的拉扯」？因为对宫必是七杀。
- 我命盘上有没有传统意义上的格局？成立条件查过没有？
- 三合派与飞星派对格局的权重差异，我能不能用自己的话说出来？
