## 概念

### 本篇定位：技法「怎么看」，不讲人生课题

5-3 讲了宫干飞化的三类区分（自化、化入、化出），5-4 讲了忌星追踪的四个动作（忌入、忌出、自化忌、生年忌）。本篇是把这两套工具拿到一张完整的盘上，做一次综合演练。

需要先把边界划清：本篇只讲**技法层面怎么看**——怎么把生年四化、宫干飞化、忌星追踪串成一张可读的图。至于「生年忌所落宫位=此生最大人生课题」这种自省应用，是第8章 8-2 的主题，本篇不重复。换言之，本篇是「方法论」，8-2 是「心法应用」，两者分工明确，看本篇时关注点要放在「怎么把盘读顺」上。

### 三层四化的优先级

综合实战第一个要解决的问题，是面对一张盘上密密麻麻的四化箭头，先看哪个、后看哪个。飞星派常用优先级是：

1. **生年四化**：先看生年禄权科忌落在哪四颗星、哪四个宫。这是先天格局的底座。
2. **命宫飞化**：命宫是一个人的「我」，命宫干令什么星化什么、飞向哪里，是个人主动性的核心。
3. **忌星追踪**：在生年忌与命宫飞忌的基础上，追踪忌的来去——谁化忌给我、我化忌给谁、哪个宫自化忌。
4. **大限四化叠加**：进入某个大限后，大限宫位的天干又起一组四化，与生年、宫干叠加。这一步本篇点到为止，详细在 6-3 大限章节展开。
5. **禄权科的辅助**：禄权科在忌的脉络清楚后，再看它们怎么解忌、怎么助忌。

这个顺序不是教条，但符合「先看大底色、再看主动力、再看执念、再看时段、再看化解」的递进逻辑。初学者按这个顺序练，能把一张盘从乱到顺。

### 串联的两个原则

把多层四化串成一张可读的图，有两个原则要守：

- **同星叠加原则**：同一颗主星上叠了多层四化（如生年禄 + 命干权 + 夫干忌），要先看这颗星所坐的宫位，再看每一化的来路。叠加不是简单加减，而是分清层次。
- **同宫叠加原则**：同一个宫位上落了多个四化（如生年忌 + 事业飞禄 + 财帛飞忌），要先看这个宫位是哪一个，再看每一化的来路。宫位是「场」，四化是「场上的力」。

这两个原则看似简单，实战中却被反复违背——很多人看到「生年忌 + 飞禄」就立刻下结论「忌被解了」，但解与不解要看飞禄的来路、看忌的来路、看自化情况，不能孤立判。

## 原理

### 生年 + 宫干的叠加判读

生年四化是先天格局，宫干飞化是后天互动。两者叠加，才能看出「先天给的格局，被后天怎么用」。常见的叠加模式：

- **生年禄 + 命宫飞禄同宫**：先天有禄，自己又主动加禄，主这一宫是「福上加福」，但也容易因福生惰。
- **生年禄 + 命宫飞忌同宫**：先天有禄，自己却主动加忌，主这一宫是「好牌被打烂」，自己把福气消耗。
- **生年忌 + 命宫飞禄同宫**：先天有忌，自己却主动加禄解忌，主这一宫是「自力化解」，是最值得称许的修心路径。
- **生年忌 + 命宫飞忌同宫**：先天有忌，自己又主动加忌，主这一宫是「执上加执」，最是难解。
- **生年忌 + 他宫飞禄同宫**：先天有忌，被外缘送禄解，主「外缘化解」——这个外缘是哪个宫，就是化解的来源。

这五种模式是叠加判读的基础。看盘时遇到生年忌的宫位，第一反应应该是「有没有飞禄来解？谁送来的？」

### 串联的逻辑：从忌出发

实战中，建议从忌出发串联，因为忌最重、最值得追踪。具体步骤：

1. **找生年忌在哪一宫**。这是先天大执念。
2. **看这一宫有没有自化**。自化禄解、自化忌加，分别处理。
3. **看哪些宫飞忌入这一宫**。这些宫是「加重执念的人事物」。
4. **看哪些宫飞禄入这一宫**。这些宫是「化解执念的人事物」。
5. **看生年忌所在宫的宫干飞向哪里**。生年忌的宫本身也起四化，它的飞禄权科忌去向，是执念的「出口」。
6. **看命宫飞忌去哪一宫**。这是个人主动执着的目标。

完成这六步，一张盘的核心四化脉络基本就清楚了。这六步不是机械流程，熟练后可以跳步、合并，但初学时按这六步走，最不容易漏。

### 串联示例：生年忌在命

举例说明。某人生年忌在命宫，先天自我认同纠结。再看：

- 命宫自化禄——命宫自己又化禄解了一部分忌，主「自我能化解」。
- 事业宫飞忌入命——工作的执着压在我身上，加重了我的自我消耗。
- 父母宫飞忌入命——父母也把期望压在我身上，又一份加重。
- 财帛宫飞禄入命——钱财上的好处给我带来缓解，主「财能解我忧」。
- 命宫干飞禄入事业——我把心思投向工作，工作的好处反哺我。
- 命宫干飞忌入夫妻——我对配偶执着，把执念投向婚姻。

这一组串联，读出来的故事是：这个人先天自我认同纠结（生年忌在命），但有自我化解的能力（命自化禄）；外缘里父母和工作都给他压力（父忌入命、事忌入命），唯独钱财上的进益能给他缓解（财禄入命）；他自己的执念投向了婚姻（命忌入夫），并把心力投给工作（命禄入事）。

化解的方向：把心力多投给工作与赚钱，少投给婚姻里的执着；同时学会拒绝父母与工作中的过度压力。这是技法层面给出的「修心路径」，至于具体怎么修，是 8-2 的主题。

## 实践

### 一个完整盘例：四化串联走一遍

下面用一个简化盘例，演示四化串联的完整流程。假设命主生于丙年，命宫在子。

**生年四化（丙干）**：

- 丙令天同化禄——天同坐于福德宫。
- 丙令天机化权——天机坐于事业宫。
- 丙令文昌化科——文昌坐于财帛宫。
- 丙令廉贞化忌——廉贞坐于夫妻宫。

第一层读象：先天格局上，精神享受有禄（福同禄）、事业有权（事机权）、财帛有科（财昌科）、婚姻有忌（夫廉忌）。这是一张「婚姻是先天课题、精神与事业先天顺」的盘。

**命宫飞化（命宫在子，天干丙，丙同禄机权昌科廉忌）**：

命宫天干与生年同，命宫飞化与生年四化重合。这种「命宫干同生年干」的盘，叫「命坐生年干」，主命主与先天格局贴合度高，自身的主动性放大了先天配置。具体到这张盘，命宫干化忌入夫妻，等于生年忌在夫妻又加重一次——婚姻是双倍执念。

**夫妻宫飞化（假设夫妻宫天干庚，庚令太阳化禄、武曲化权、太阴化科、天同化忌）**：

- 夫妻化禄入命（太阳坐命）——配偶给我送禄，感情给我好处。
- 夫妻化忌入福德（天同坐福德）——配偶的执着压到我的精神享受上，主感情影响我的心情。

读象：婚姻虽然先天有忌，但配偶本身给我送禄入命，是有滋养的；只是配偶也会把执念压到我的福德宫，影响我的精神状态。

**事业宫飞化（事业宫天干甲，甲令廉贞化禄、破军化权、武曲化科、太阳化忌）**：

- 事业化禄入夫妻（廉贞坐夫妻）——工作的好处落到婚姻上，给婚姻送禄解忌。
- 事业化忌入命（太阳坐命）——工作的执着压在我身上。

读象：事业是化解婚姻执念的来源（事禄入夫解夫生年忌），但工作本身也给我压力（事忌入命）。

**串联结论**：

- 先天大执念在婚姻（生年忌在夫）。
- 命宫自身又把执念投向婚姻（命忌入夫），是「执上加执」。
- 化解路径有两条：配偶给我送禄入命（夫禄入命）、事业给婚姻送禄（事禄入夫）。前者是「婚姻本身带来滋养」，后者是「工作成就感能解婚姻的执」。
- 工作本身给我压力（事忌入命），但同时也给我权（生年权在事）——工作虽然累但有权势，是值得投入的方向。

这样一张盘的化解方向就很清楚：不要在婚姻里钻牛角尖（命忌入夫要松手），把心力投给事业（事有权、事禄解夫忌），让婚姻的滋养自然来（夫禄入命）。

### 串联时常见的几个坑

实战中初学者容易踩的坑：

- **只看生年四化就下结论**。生年忌在哪一宫只是先天底色，还要看宫干飞化怎么引动。同样的生年忌在夫妻，命忌入夫与命禄入夫，故事完全不同。
- **把自化当作化解**。自化禄确实能解一部分忌，但自化的本质是「不持久」，自化禄解忌往往解得一时、解不了一世。要把它当作「自我缓解」，而非「根本化解」。
- **忽略忌的来路**。看到夫妻宫有忌，立刻归咎于「配偶不好」，但若是命忌入夫，是自己把执念压给配偶，反过来才对。忌的来路决定责任方。
- **把禄权科看作绝对的好**。禄太多主惰性，权太多主霸道，科太多主虚名。四化是力，力无善恶，看怎么用。
- **跨大限断事**。生年四化与宫干飞化是先天静态结构，断具体年份的事必须加大限四化与流年四化（6-3、6-4 内容）。本篇讲的是结构层，不是事件层。

### 与中州派路数的对照

中州派看四化，重星曜庙旺与三方四正组合。同样的盘，中州派可能先看夫妻宫廉贞的庙旺——廉贞在夫妻宫庙旺与落陷，读法截然不同。飞星派则相对弱化星曜亮度，更看重四化的来去与叠加。两种路数对同一张盘可能给出不同侧重的结论，并不冲突——一个看「性格与情境」，一个看「因缘与流向」。看盘时如能两造并陈，结论会更立体。本篇依飞星派写，但读者不妨用中州派的角度再回看一次盘例，比较两者的差异。

## 速查自测

### 综合串联六步法

1. 找生年忌在哪一宫（先天大执念）。
2. 看这一宫有无自化（自化禄解、自化忌加）。
3. 看哪些宫飞忌入这一宫（执念的加重方）。
4. 看哪些宫飞禄入这一宫（执念的化解方）。
5. 看生年忌所在宫的宫干飞向哪里（执念的出口）。
6. 看命宫飞忌去哪一宫（个人主动执着的目标）。

### 叠加判读五种模式

| 模式 | 象意 |
|---|---|
| 生年禄 + 命飞禄 | 福上加福，易生惰 |
| 生年禄 + 命飞忌 | 好牌打烂，自耗福气 |
| 生年忌 + 命飞禄 | 自力化解，最称许 |
| 生年忌 + 命飞忌 | 执上加执，最难解 |
| 生年忌 + 他宫飞禄 | 外缘化解，看是谁送来 |

### 自省问题清单

- 我能不能在一张盘上完成六步串联？做不到就回去重练。
- 我看四化时，是只看生年就下结论，还是把宫干飞化也串进去？
- 我有没有把自化禄当作「根本化解」？自化禄只是自我缓解。
- 我看到忌所在宫，第一反应是问「谁送来的」「谁解的」，还是直接归咎？
- 我能不能用飞星派与中州派两种路数看同一张盘？两造并陈会让结论更立体。
- 本篇的盘例，我自己用六步法走一遍，能不能得出相同的化解方向？
