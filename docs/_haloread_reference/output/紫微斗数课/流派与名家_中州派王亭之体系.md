## 概念

### 中州派在现代斗数史上的位置

紫微斗数在二十世纪后半叶的复兴与传播，中州派是一个绕不开的名字。这一派别由陆斌兆在近代奠基，由王亭之（谈锡永）公开教法、系统整理，使原本师徒口传的星曜性质与推断法得以成体系地走向公众。

中州派之所以重要，不在于它「最正统」（斗数本无唯一正统），而在于它把星曜性质、三方四正、格局三者整合成了一套可教学、可复现的解盘方法。这一方法经王亭之的讲义、评注、专著广泛传播，成为现代中文世界初学紫微斗数最常见的入门路径之一。

本篇客观陈列中州派的体系贡献，不涉优劣评判。其他流派（紫云派、飞星派、四化北派、倪海厦天纪）在后续 7-2、7-3、7-4 各篇平行介绍。

### 王亭之其人

王亭之，本名谈锡永，广东南海人，1935 年生。早年随陆斌兆学习中州派斗数，后又涉猎密宗、佛学、易经等多门学问，是近代华人文化圈少见的跨领域学者。他在斗数上的主要工作是把中州派讲义公开化、系统化，并对陆斌兆原讲义作评注。

王亭之对中州派的整理与公开，使这一派别从「师徒私授」转为「公开教法」。这一点在斗数传播史上有重要意义：它让大量自学者得以接触原本秘传的星曜性质与推断法，也间接推动了其他流派的公开化竞争。

### 中州派的三件套：星性、三方四正、格局

中州派的解盘方法论，可以概括为三件套：

1. **星曜性质**：每颗主星、辅星、煞星都有详细的性质描述，包括五行、阴阳、庙旺平陷、星性变化。
2. **三方四正**：以命宫为核心，看本宫、对宫与三合两宫的星曜组合，判断基调有无支持。
3. **格局**：识别古格成立条件，作为组合倾向的归纳名。

这三件套并非中州派独创，但中州派把它们整合成了一套完整的教学体系，是它的核心贡献。后续流派或强化其中一环（如紫云派重星曜组合与太岁入卦），或弱化某一环（如飞星派弱化格局），都是在这一基础上作调整。

## 原理

### 星曜性质：中州派的根基

中州派最被称道的是星曜性质的细致刻画。陆斌兆原著、王亭之补注的《紫微斗数讲义：星曜性质》是这一根基的集中呈现。

中州派对每颗星曜的描述包含几个维度：

- **基本性质**：星曜的五行、阴阳、核心象征（如紫微=帝星、天机=智星）。
- **庙旺平陷**：星曜在不同宫位的亮度差异，决定星性发挥程度。中州派用「庙、旺、得地、平、闲、陷」六阶来刻画。
- **同宫修正**：与其他星曜同宫时的性质变化（如紫微遇左辅右弼为帝有辅臣，遇擎羊陀罗为帝受困）。
- **化象变化**：被化禄、化权、化科、化忌引动时的性质转化。

这一套星性体系的优势是细致可教，劣势是初学者容易陷入「记星性」而忽略整体。中州派的教学对此有自觉，强调星性是基础而非目的。

### 三方四正：中州派的整合视角

中州派看盘不是孤立看一颗星，而是把命宫与三方四正作为一个整体看。这一视角的逻辑是：单星不足以定基调，必须看「命宫 + 官禄 + 财帛 + 迁移」这一组四宫的星曜组合，才能判断基调有无支持。

中州派对三方四正的权重在所有流派中属于最高之一。这与飞星派形成对照——飞星派这一步更多看宫干飞化，三方四正退居次要。这种权重差异是中州派与飞星派方法论分歧的集中体现，详见 7-3。

### 格局：中州派的归纳工具

中州派对格局的态度介于三合派与飞星派之间：

- 比三合派略宽：不要求古格成立的极致严格条件，允许一定程度的破格修正。
- 比飞星派严：仍然重格局名作为组合倾向的归纳工具，不弃用格局语言。

中州派对格局的功能定位与本专栏 6-5 所述一致：格局是趋势辅助而非命运判决。王亭之在讲义中多次提醒，古格成立的语境是古代社会，现代人要重新诠释「贵」「富」对应的领域。

### 化忌解读：中州派的视角

中州派对化忌的解读，重在「庙旺星性 + 执着点」：

- 化忌在某宫，看该宫坐星的庙旺平陷。
- 庙旺之星化忌，主「虽有执着而能化解」；落陷之星化忌，主「执着深而难解」。
- 化忌代表「欠缺与执着」，是中州派的核心定性。

这与飞星派/北派把化忌看作「果报之位、忌星追踪看因缘流向」的视角不同。两派对化忌的理解不在同一概念层，详见 7-3 的对照。

## 实践

### 中州派的教学体系

中州派在斗数传播上的另一贡献，是建立了一套层次分明的教学体系：

- **初级讲义**：星曜性质、起盘法、三方四正基础。适合入门。
- **深造讲义**：格局成立条件、四化解读、大限流年断法。适合进阶。
- **评注与专著**：对陆斌兆原讲义的评注，对古籍的整理与诠释。适合研究。

这一层次分明的教学体系，使中州派成为初学者最容易上手的流派之一。本专栏前七章的知识层结构（命盘骨架→十四主星→辅星煞星→四化→格局大限流年）即参照中州派的教学层次。

### 中州派解盘示例

中州派解盘的典型流程：

1. 看命宫坐星的庙旺与同宫修正。
2. 看三方四正的星曜组合，判断基调支持度。
3. 看生年四化所落宫位，定此生主题。
4. 扫格局，作为组合倾向的归纳。
5. 看大限流年，把潜在格局激活到具体时间段。

这一流程与本专栏 6-6 解盘六步法高度一致，因为六步法的基础架构即来自中州派。

### 中州派的局限

客观陈列中州派的贡献，也要陈列其局限：

- **重星性、轻飞化**：中州派对宫干飞化的处理不如飞星派/北派细致，宫位间因缘流向的追踪较弱。
- **古格依赖**：虽然比三合派宽松，但仍依赖格局名，对完全不用格局语言的现代解盘者不够友好。
- **传承争议**：中州派的「正统性」在斗数界有争议，部分学者认为王亭之公开的并非中州派全貌。本篇不介入此争议，仅指出存在不同声音。

这些局限不是否定中州派，而是说明：任何流派都有其方法论边界，中州派适合作为入门与基础，但深入研究时往往需要借鉴其他流派。

### 中州派对现代斗数教育的影响

中州派对现代斗数教育的影响是深远的。本专栏前七章的知识层，许多基础概念（星曜五行、庙旺平陷六阶、三方四正、起盘法）都参照了中州派的讲义体系。这一影响是公开可查的，不是本专栏的独创。

承认这种影响，是诚实；同时承认中州派的局限，是平衡。读者学完本专栏知识层后，若想深入研究星性，可读陆斌兆原著、王亭之补注的《紫微斗数讲义：星曜性质》。
