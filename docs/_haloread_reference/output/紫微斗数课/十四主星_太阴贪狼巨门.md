## 概念

### 太阴：母星的柔与财

太阴阴水，天府星系中的母星，又为主财之星。水主智而阴水主柔，故太阴主柔取、财富、母性。太阴的形象，是夜空中的明月——柔美、内敛、滋养万物，但也主阴柔之财与母性光辉。

太阴是十四主星中最大的"柔财"之星。与武曲的"刚财"相对——武曲之财靠刚毅决断得来，太阴之财靠柔取积蓄得来。太阴坐命的人，庙旺者主柔美有财、田宅丰足、感情细腻；落陷者主财来财去、感情多波折、母缘薄。

### 贪狼：欲望之星的多才

贪狼阳木，天府星系中的欲望之星，又为正桃花。木主仁而阳木主生发，故贪狼主欲望、才情、生发。贪狼的形象，是一位多才多艺的才子——欲望强、才情高、应酬多，但也容易纵欲、浮躁、难专一。

贪狼是十四主星中最"多才"的一颗。贪狼坐命的人，庙旺者主才情横溢、应酬有道、艺术天赋高；落陷者主纵欲浮躁、难专一、应酬过多而损身。贪狼的"欲望"，既是才情之源，也是纵欲之根——欲望得其所则为才情，欲望失其所则为纵欲。

### 巨门：暗星的口舌与疑

巨门阴水，天府星系中的暗星。水主智而阴水主暗，故巨门主口舌、暗昧、疑虑。巨门的形象，是一扇紧闭的暗门——主口舌是非、暗中行事、疑心重，但也主辩才与钻研之力。

巨门是十四主星中最"暗"的一颗。巨门坐命的人，庙旺者主辩才无碍、钻研有成、口才佳；落陷者主口舌是非、疑心重、人际关系紧张。巨门的"暗"，并非全凶——暗得其所则成钻研与辩才，暗失其所则成口舌与猜忌。

## 原理

### 太阴庙旺与落陷

太阴的庙旺，以亥宫为最——"月朗天门"，主柔美有财、田宅丰足、母性光辉。此外酉、戌、子等宫亦为庙旺。落陷的太阴，主要在卯、辰、巳等宫——其中卯为"月落沦陷"，主财来财去、母缘薄。

太阴的庙陷落差，与太阳并列为十四主星中最大的。同一颗太阴，在亥与在卯，吉凶几乎完全相反。这是中州派解读太阴的第一条铁则：先看庙陷，再论星性。

太阴还有男女命差异：男命太阴即妻子与母亲，太阴庙旺主妻运佳、母运佳；女命太阴即自身（命宫坐太阴时），太阴庙旺主自身柔美、旺夫益子。男命太阴落陷主妻运薄、母缘薄；女命太阴落陷主自身劳碌、感情多波折。

### 贪狼庙旺与落陷

贪狼的庙旺，主要在辰、戌、丑、未等宫支（土之地，土培木生）。庙旺的贪狼，主才情横溢、应酬有道、艺术天赋高、人际佳；落陷的贪狼（如在子、午等宫），主纵欲浮躁、难专一、应酬过多而损身。

贪狼的特殊性在于"欲望的双向"。贪狼本主欲望，欲望得其所则为才情与艺术，欲望失其所则为纵欲与浮躁。故贪狼的解读，关键看欲望的流向：

- 贪狼庙旺 + 文昌文曲：欲望化为才情，主艺术有成。
- 贪狼庙旺 + 火星铃星：火贪格、铃贪格，主突发横财。
- 贪狼落陷 + 化忌：欲望失其所，主纵欲、破财、损身。
- 贪狼 + 廉贞（三合或对照）：桃花组合，主感情丰富。

### 巨门庙旺与落陷

巨门的庙旺，主要在子、申、酉等宫支。庙旺的巨门，主辩才无碍、钻研有成、口才佳、学术有成；落陷的巨门（如在午、未等宫），主口舌是非、疑心重、人际关系紧张。

巨门的"暗"性，使其在解读时需要特别注意化星反应：

- 巨门化禄（辛干）：暗中有财，主口舌生财、钻研有成。
- 巨门化权（癸干）：口才有威，主辩才无碍、口才显达。
- 巨门化科（丁干）：口才有名，主学术有声望。
- 巨门化忌（丁干）：口舌大显，主是非、官非、疑心重。

巨门化忌是十四主星中较重的化忌之一，主口舌是非与疑心。但化忌也是动力源——巨门化忌的人若将这股钻研之力用于学术、法律、研究，反可大成。

## 实践

### 太阴的男女命差异

太阴的解读，男女命差异显著：

**男命**：太阴即妻子与母亲。太阴庙旺，主妻运佳、母运佳、田宅丰足；太阴落陷，主妻运薄、母缘薄、田宅有损。

**女命**：太阴即自身（命宫坐太阴时）。太阴庙旺，主自身柔美、旺夫益子、田宅丰足；太阴落陷，主自身劳碌、感情多波折、田宅有损。

这是太阴解读中最重要的性别差异。读者解盘时切忌男女命一概而论——同一颗太阴在亥，男命主妻运佳，女命主自身柔美，断语不同。

### 贪狼的桃花与艺术的转化

贪狼的解读，要在"桃花"与"艺术"之间找平衡：

- 贪狼庙旺 + 文昌文曲：桃花化为艺术，主才情横溢、艺术有成。
- 贪狼庙旺 + 火星：火贪格，主突发横财。
- 贪狼庙旺 + 铃星：铃贪格，主突发横财（与火贪格并列）。
- 贪狼落陷 + 化忌：桃花失其所，主纵欲、破财、损身。
- 贪狼 + 廉贞（三合）：桃花组合，主感情丰富。

贪狼的"火贪格"与"铃贪格"，是中州派解读中的著名格局——主突发横财。但此格的成立有严格条件：贪狼须庙旺、火星铃星须与贪狼同宫或三合、不可再有煞星冲破。这是贪狼解读中最辩证的一条。

### 巨门的口舌与暗化的用法

巨门的解读，要在"口舌"与"辩才"之间找平衡：

- 巨门庙旺 + 化禄：暗中有财，主口舌生财、钻研有成。
- 巨门庙旺 + 化权：口才有威，主辩才无碍。
- 巨门庙旺 + 文昌文曲：口才化为学术，主学业有成。
- 巨门落陷 + 化忌：口舌大显，主是非、官非、疑心重。
- 巨门 + 太阳（对照）：驱暗为明，主口才有显达之机。

巨门与太阳的对照，是中州派解读中的一条重要规则——太阳主光明，巨门主暗昧，太阳照巨门可"驱暗为明"。但此格的成立，须太阳庙旺——太阳落陷则无力驱暗。
