## 概念

### 十二宫是什么

紫微斗数的命盘是一个圆，被均分为十二个等份，每一份称为一个「宫位」。这十二个宫位各自对应人生的不同领域，合起来就构成一个人一生的「版图」——从自身的性格底色，到与他人的关系，再到物质基础、事业成就、精神世界，无一不在其中。十二宫不是抽象的概念，而是把人生切成十二个可观察的切面，让解盘者能逐项审视。

十二宫以「命宫」为起点，按逆时针方向依次排列。这个顺序是固定的：命宫、兄弟宫、夫妻宫、子女宫、财帛宫、疾厄宫、迁移宫、交友宫、官禄宫、田宅宫、福德宫、父母宫，共十二个宫位。无论命宫落在哪个地支位，其余十一宫的相对次序都不变。

### 十二宫的命名与含义

下表列出十二宫的命名、别称与对应的人生领域：

| 宫位 | 别称 | 人生领域 |
|---|---|---|
| 命宫 | 本命宫 | 先天格局、性格底色、整体人生主调 |
| 兄弟宫 | | 兄弟姐妹、平辈关系、合作同侪 |
| 夫妻宫 | 配偶宫 | 婚恋配偶、亲密关系、感情模式 |
| 子女宫 | 子息宫 | 子女、晚辈、学生、生育状况 |
| 财帛宫 | | 财富获取方式、理财能力、收支格局 |
| 疾厄宫 | | 身体健康、疾病倾向、体质特点 |
| 迁移宫 | | 出行、外出发展、外界环境、迁徙变化 |
| 交友宫 | 古称奴仆宫 | 朋友、部属、合作伙伴、人际网络 |
| 官禄宫 | 事业宫 | 事业、地位、工作取向、社会成就 |
| 田宅宫 | | 不动产、家庭环境、祖业、晚年安居 |
| 福德宫 | | 心性、福分、精神生活、内心世界 |
| 父母宫 | 相貌宫 | 父母长辈、荫庇、出身家世、相貌 |

十二宫构成人生版图——既覆盖外部社会关系（兄弟、夫妻、子女、交友、官禄、父母），也覆盖内在身心与精神（命宫、疾厄、福德），还覆盖物质基础（财帛、田宅），以及与外界的动态交换（迁移）。任何一个领域，都能在十二宫中找到对应的位置。

### 交友宫的命名演变

交友宫古称「奴仆宫」，反映古代社会主仆分明的人际结构；现代改称「交友宫」，泛指朋友、同事、部属、合作伙伴等广义人际网络。这是十二宫在历史流变中唯一的命名调整，含义也相应从「奴仆」扩展为「交往对象」。古籍《紫微斗数全书》中仍作「奴仆宫」，王亭之补注陆斌兆讲义时已统称「交友宫」，本专栏后文统一用「交友宫」。

## 原理

### 十二宫的排布逻辑

十二宫以命宫为锚点，按逆时针方向依次排列在十二地支位上。地支位是固定的（子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥），命宫的位置由生月和生时推算（具体安命宫法详见后篇起盘法），命宫一旦确定，其余十一宫的次序也就确定。

这种排布不是随意的，而是把人生领域按「由内向外、由近及远」的层次展开：

- 命宫是中心（自己）
- 兄弟、夫妻、子女是血亲与至近关系
- 财帛、疾厄是个体的物质与身体基础
- 迁移、交友是与外界的交往
- 官禄、田宅是社会成就与物质根基
- 福德、父母是精神世界与来路根源

十二宫的次序本身就在告诉解盘者：看一个人，先看他自己（命），再看他的至亲（兄夫子），再看他的物质身体（财疾），再看他与外界的关系（迁交），最后看他的社会成就与精神根基（官田福父）。

### 十二宫的几何对称

十二宫在几何上构成六组对宫，每组对宫含义互为表里：

- 命宫 ↔ 迁移宫：内在性格 ↔ 外在表现
- 兄弟宫 ↔ 交友宫：血亲平辈 ↔ 社交朋友
- 夫妻宫 ↔ 官禄宫：感情伴侣 ↔ 事业成就
- 子女宫 ↔ 田宅宫：下一代 ↔ 不动产家底
- 财帛宫 ↔ 福德宫：物质财富 ↔ 精神福分
- 疾厄宫 ↔ 父母宫：身体来路 ↔ 出身来路

这六组对称不是巧合，而是命盘设计者对人生结构的深刻洞察：每个领域都有它的「对镜」，内在与外在、物质与精神、来路与去处，互为映照。看命宫不能不看迁移，看财帛不能不看福德——对宫是本宫的「另一面」。

### 三层结构：本宫、对宫、三方

每个宫位都不是孤立的。看任何一个宫位，至少要看三层：

1. **本宫**：该宫本身坐什么星曜，是该宫领域的基调
2. **对宫**：与本宫相对的宫位（隔六位），形成对照与补充
3. **三方**：与本宫形成三合关系的两宫（隔四位），构成联动

例如看命宫，对宫是迁移宫，三方是财帛宫与官禄宫。命宫性格的展开，离不开迁移（外在表现）、财帛（物质取向）、官禄（社会实现）的协同。三方四正的具体联动逻辑详见后篇。

## 实践

### 十二宫的解盘顺序

实际解盘时，通常按「先命身、后三方、再六亲、最后杂曜」的顺序：

1. 先看命宫与身宫（先天格局与后天行运）
2. 再看命宫三方四正（财官迁）定大格局
3. 看六亲宫（兄夫子父）定关系底色
4. 看疾厄、田宅、福德定身心与根基

这个顺序的逻辑是：先定大格局（命身+三方四正），再看具体领域。如果命宫三方四正格局已经偏低，单看某个六亲宫再好，整体也难大富贵；反之命身格局高，即使某个六亲宫稍弱，整体也不会太差。

### 十二宫与人生阶段的对应

十二宫虽然按空间排列，但在时间维度上也有大致的对应：

- 命宫、父母宫：早年（出身与童年）
- 兄弟宫、夫妻宫：青年（同辈与婚恋）
- 子女宫、财帛宫：中年（家庭与事业基础）
- 官禄宫、田宅宫：中晚年（成就与产业）
- 福德宫、疾厄宫：晚年（精神与身体）

这种对应并非严格的「某宫管某岁」，而是大致的人生阶段侧重。配合大限（十年一限）看，每段大限所入宫位，恰好是该阶段人生课题的焦点。

### 各宫心性映射

十二宫不仅是命运预测的「位置」，也可作为认识自己的镜子：

- **命宫**：你天生的性格底色——认清自己是什么样的人
- **兄弟宫**：你与同辈的相处模式——你如何与平辈协作
- **夫妻宫**：你的亲密关系模式——你容易被什么样的人吸引
- **子女宫**：你对晚辈的态度——你是否善于传承与培养
- **财帛宫**：你的金钱观——你靠什么方式获取财富
- **疾厄宫**：你的身体薄弱处——你需注意的健康隐患
- **迁移宫**：你在外面的状态——你是否适合外出发展
- **交友宫**：你的朋友圈——你吸引什么样的朋友
- **官禄宫**：你的事业取向——你做什么事最有成就感
- **田宅宫**：你的安居方式——你对家庭与不动产的态度
- **福德宫**：你的内心世界——你精神上是否安宁
- **父母宫**：你与长辈的关系——你的出身与荫庇

把十二宫当成十二面镜子，每一面照见自己的一个侧面。命盘不是宿命判决书，而是一份「自我说明书」——让你看清自己的天赋与短板、关系模式与心性倾向，再决定如何运用与调整。

### 自省问题清单

- 我最在意的宫位是哪一个？为什么？
- 我命宫的星曜与对宫迁移宫形成什么样的对照？是否一致？
- 我财帛宫与官禄宫是否同向？反映我谋生方式与事业取向是否一致？
- 我福德宫与命宫关系如何？反映我表面性格与内心世界是否一致？
- 我六亲宫（兄夫子父）的整体格局如何？反映我与至亲关系的整体底色？
- 我十二宫中哪一宫最弱？这一宫对应的人生领域，我是否在现实中也有对应短板？
