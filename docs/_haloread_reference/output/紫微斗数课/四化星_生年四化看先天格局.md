## 概念

### 生年四化的地位：先天格局的底色

5-1 已建立四化的三层定义——生年四化、宫干飞化、大限四化。本篇专讲第一层：**生年四化看先天格局**。

生年四化是依**生年干**起出的四化，落在命盘的四个宫位，一生位置固定。它的地位是「先天格局底色」——描述此生整体的能量走向，是其他两层四化（宫干飞化、大限四化）的舞台背景。

打个比方：生年四化是「这一辈子的总体气象」，宫干飞化是「宫位之间的局部互动」，大限四化是「这十年的能量走向」。后者都在前者的舞台上演出——生年化忌入夫妻宫的人，一辈子婚姻都是「先天课题」，大限再逢化忌入夫妻，那十年的婚姻压力会更大；但若生年化禄入财帛，纵然大限逢忌入财帛，整体财气仍是有底子的。

### 化禄所落=先天所得，化忌所落=先天课题

生年四化四颗，落在四个宫位，各主不同：

- **生年化禄所落宫位 = 先天所得**。此生最易得之缘、最易顺遂之领域。该宫所主之事，对你而言是「甜区」——你容易在这里感到满足，也容易在这里依赖。
- **生年化权所落宫位 = 先天主权**。此生最易掌控之事、最易出力之领域。该宫所主之事，对你而言是「权力欲所在」——你容易在这里想要主导。
- **生年化科所落宫位 = 先天名望**。此生最易得名之领域。该宫所主之事，对你而言是「贵气所在」——你容易在这里得到贵人提携或名声。
- **生年化忌所落宫位 = 先天课题**。此生最大缺憾与执念所在。该宫所主之事，对你而言是「执着点」——你容易在这里摔跤、放不下，但也是你此生最大的成长功课。

四化四颗，对应四类宫位。看生年四化，就是看此生整体的能量地图——哪里是甜区、哪里是权力区、哪里是贵气区、哪里是课题区。

### 化禄 ≠ 禄存（再次强调）

5-1 已强调过，本篇再强调一次：**生年化禄所落宫位 ≠ 禄存所在宫位**。

- 生年化禄依生年干起，是动态飞星，描述「先天所得之缘」。
- 禄存依生年支安，是静态辅星，描述「先天财禄之星」。

两者安法不同、性质不同，但可同宫为「叠禄」——化禄与禄存同入一宫，是「财上加财」的大富格之一。看生年四化时，不要把化禄与禄存混为一谈。

## 原理

### 生年化禄落十二宫的先天所得

生年化禄落不同宫位，先天所得的领域不同：

- **化禄入命宫**：先天所得在自身——主性格开朗、一生多顺遂之缘。
- **化禄入兄弟宫**：先天所得在兄弟——主兄弟有助力、易得平辈之财。
- **化禄入夫妻宫**：先天所得在婚姻——主姻缘顺遂、配偶有财。
- **化禄入子女宫**：先天所得在子女——主子女有出息、子女带来财。
- **化禄入财帛宫**：先天所得在财——主一生财气顺、求财易得。
- **化禄入疾厄宫**：先天所得在身体——主身体底子好、病易愈（但疾厄化禄亦主「贪图享受」，须看主星）。
- **化禄入迁移宫**：先天所得在外出——主外出得利、远方有财。
- **化禄入交友宫**（旧称奴仆宫）：先天所得在朋友——主朋友有助力、得朋友之财。
- **化禄入事业宫**（旧称官禄宫）：先天所得在事业——主事业顺遂、事业得财。
- **化禄入田宅宫**：先天所得在田宅——主家有祖产、置产顺遂。
- **化禄入福德宫**：先天所得在福——主一生福厚、享受多。
- **化禄入父母宫**：先天所得在父母——主父母有助力、得长辈之财。

注意：化禄入某宫不等于「该宫一定发财或顺利」——化禄是缘起，能不能把握看化权与化忌。例如化禄入财帛但化忌入事业，主「财气顺但事业有压力」，财易得但事业难成，最终财也聚不住。

### 生年化权落十二宫的先天主权

生年化权落不同宫位，先天主权的领域不同：

- **化权入命宫**：主权在自身——主性格强势、能自主。
- **化权入事业宫**：主权在事业——主事业能掌控、易升迁。
- **化权入财帛宫**：主权在财——主理财有方、能掌控财。
- **化权入夫妻宫**：主权在婚姻——主配偶能干，但亦主自己想掌控婚姻，易生摩擦。
- **化权入子女宫**：主权在子女教育——主对子女管教严，子女有成就但压力大。
- **化权入迁移宫**：主权在外出——主外出能掌控局面。

化权是「掌控」之气，落某宫主「在该宫所主之事上想要主导」。但化权过重反主「控制欲过强」——化权入夫妻易管配偶、化权入子女易压子女、化权入交友易控制朋友。这是化权的双刃性。

### 生年化科落十二宫的先天名望

生年化科落不同宫位，先天名望的领域不同：

- **化科入命宫**：主一生有名气、文采。
- **化科入事业宫**：主事业有名、易升迁得名。
- **化科入财帛宫**：主财有名（但不一定多）——财来得清白、有名望之财。
- **化科入夫妻宫**：主配偶有文名、姻缘得贵人介绍。
- **化科入父母宫**：主父母有文名、得长辈教化。

化科是「名望与贵人」之气，落某宫主「在该宫所主之事上易得名」。化科亦主「逢凶化吉」——化科入命者，遇煞星能减轻，逢凶化吉的概率高于常人。

### 生年化忌落十二宫的先天课题

生年化忌落不同宫位，先天课题的领域不同。本节按中州派视角列出，飞星派视角另见后文：

- **化忌入命宫**：先天课题在自身——主性格执念、自我纠结。一生最大的课题是「如何与自己相处」。
- **化忌入兄弟宫**：先天课题在兄弟——主兄弟有摩擦、易因兄弟生烦恼。
- **化忌入夫妻宫**：先天课题在婚姻——主姻缘有缺憾、易执着于感情。一生最大的课题是「如何与配偶相处」。
- **化忌入子女宫**：先天课题在子女——主子女有波折、易因子女生烦恼。
- **化忌入财帛宫**：先天课题在财——主求财多波折、易因财生烦恼。一生最大的课题是「如何与财相处」。
- **化忌入疾厄宫**：先天课题在身体——主身体易出问题、易因病生烦恼。
- **化忌入迁移宫**：先天课题在外出——主外出多波折、易在外生烦恼。
- **化忌入交友宫**：先天课题在朋友——主朋友有摩擦、易因朋友生烦恼。
- **化忌入事业宫**：先天课题在事业——主事业多波折、易因事业生烦恼。
- **化忌入田宅宫**：先天课题在田宅——主家有变故、易因家生烦恼。
- **化忌入福德宫**：先天课题在福——主福薄、易心生烦恼。一生最大的课题是「如何与自己的内心相处」。
- **化忌入父母宫**：先天课题在父母——主父母有摩擦、易因父母生烦恼。

注意：化忌入某宫不等于「该宫一定坏」——化忌是「缺憾与执念」，也是「动力源」。化忌入事业的人，事业上多波折，但也因此练出更强的能力，许多事业有成者都有化忌入事业的格局——化忌的执着转化为事业的专注。这是化忌的两面性，不可妖魔化（参考慧心斋主《如何推算命运：紫微斗数与四化星》）。

## 实践

### 中州派 vs 飞星派对生年化忌的不同看法

5-1 已给出两派对化忌解读的对照表，本节专门讲两派对**生年化忌**的不同看法。

**中州派看生年化忌**：重在看「化忌所落宫位+所化主星的庙旺」。同样是化忌入夫妻：

- 太阳庙旺化忌入夫妻：太阳庙旺则忌轻，主「姻缘有小损，但不至于大破」。
- 太阳落陷化忌入夫妻：太阳落陷则忌重，主「姻缘有大损，易分离」。

中州派较少做忌星追踪，重星性组合——化忌所化主星的庙旺决定了忌的轻重（参考王亭之《中州派紫微斗数深造讲义》）。

**飞星派/北派看生年化忌**：重在「忌星追踪」与「入库出库」。许铨仁系统特别强调：

- 生年化忌所落宫位是「果报位」——此生因缘最终收束于此。
- 忌星追踪：从生年忌追到宫干飞忌再到大限忌，看因缘链的流动。例如生年忌入夫妻，再追夫妻宫干飞忌入哪宫，看「姻缘的因缘流向何处」。
- 入库出库：忌入辰戌丑未四库宫主「收束」（能量聚而不出，课题深埋难化解），忌出库主「释放」（能量外流，课题可通过外缘化解）。

许铨仁系统重忌的入库出库，是北派四化的特色（参考许铨仁《紫微斗数命理学正解（一）》）。

两派对生年化忌的解读差异很大，初学者须先选定一派入门，不可两种解读混用——这是术语纪律明文要求。

### 生年四化的组合判读

生年四化四颗落四宫，组合起来看整体格局底色。常见组合：

- **化禄与化忌同入一宫**：该宫既是「先天所得」又是「先天课题」——主该宫所主之事「得中有缺、缺中有得」，是此生最纠结的领域。例如化禄化忌同入夫妻，主「姻缘既有所得又有大缺憾」，感情上易「爱恨交织」。
- **化禄与化权同入一宫**：该宫既是「先天所得」又是「先天主权」——主该宫所主之事「机会加执行」，是此生最易成就的领域。例如化禄化权同入事业，主「事业有机会又能掌控」，事业易大成。
- **化禄与化科同入一宫**：该宫既是「先天所得」又是「先天名望」——主该宫所主之事「名利双收」。
- **化忌与化权同入一宫**：该宫既是「先天课题」又是「先天主权」——主该宫所主之事「压力加掌控」，是此生最想掌控却又最易摔跤的领域。例如化忌化权同入财帛，主「财有压力又想掌控财」，易因财生烦恼但财也有动力。
- **化忌单独入某宫**：该宫是此生最大课题所在，但也是最大动力源——化忌的执着若用对方向，反成大成就。例如化忌入事业的人，事业上多波折，但也因此练出更强的能力。

### 一个示例命盘的生年四化判读

假设某命主生年干为甲，甲干四化为「廉贞化禄、破军化权、武曲化科、太阳化忌」。生年四化所落宫位如下（假设）：

- 廉贞化禄入事业宫：先天所得在事业——事业上有先天机会，做事易得顺遂之缘。
- 破军化权入财帛宫：先天主权在财——理财有方，能掌控财。
- 武曲化科入命宫：先天名望在自身——一生有名气、有贵人。
- 太阳化忌入夫妻宫：先天课题在婚姻——姻缘有缺憾，一生最大课题是「如何与配偶相处」。

整体格局底色：事业有机会、财能掌控、自身有名望，但婚姻是最大课题。这是该命主此生整体的能量地图——前三个宫位是「甜区与权力区」，夫妻宫是「课题区」。具体大限走到哪一宫、流年逢什么四化，会在第六章展开。

### 心性映射：生年化忌所落宫位是此生最大课题

生年四化里，最值得自省的是**化忌所落宫位**——它提示你此生最大的执念与课题。

化忌入夫妻、事业、财帛的人，分别要在「亲密关系」「事业成就」「财」三件事上看清自己的执着点——是怕失去？是怕失败？是怕不够？看清之后，才能从执着中转化为动力，而非被执念困住。

这是紫微斗数「照见自己」最深的一层——生年化忌提示的不是「你会怎样」，而是「你此生最容易在哪条路上摔跤」。具体如何用化忌照见人生课题并加以修心，见第 8 章「心性镜子」（8-2）。

## 速查自测

### 生年四化所落宫位速查

| 化星 | 所落宫位主 | 性质 |
|---|---|---|
| 生年化禄 | 先天所得 | 甜区、易满足、易依赖 |
| 生年化权 | 先天主权 | 权力欲、易掌控、易控制 |
| 生年化科 | 先天名望 | 贵气、易得名、逢凶化吉 |
| 生年化忌 | 先天课题 | 执念、易摔跤、最大功课 |

### 化忌入十二宫的先天课题速查

- 命宫：自我纠结。
- 兄弟宫：兄弟摩擦。
- 夫妻宫：姻缘缺憾。
- 子女宫：子女波折。
- 财帛宫：求财烦恼。
- 疾厄宫：身体问题。
- 迁移宫：外出波折。
- 交友宫：朋友摩擦。
- 事业宫：事业压力。
- 田宅宫：家事变故。
- 福德宫：内心烦恼。
- 父母宫：父母摩擦。

### 中州派 vs 飞星派对生年化忌速查

- 中州派：看庙旺星性，星曜庙旺则忌轻，落陷则忌重。
- 飞星派/北派：忌星追踪看因缘流向，重入库出库（许铨仁系统）。

### 自省问题清单

1. 我命盘生年四化所落宫位分别是什么？这些宫位是不是真的对应我「最易满足/最想掌控/最求名望/最执着」的领域？
2. 化禄所落宫位：我有没有在这个领域过度依赖？化禄是甜区，但也是依赖区。
3. 化权所落宫位：我有没有在这个领域过度控制？化权是权力欲，过重则成控制欲。
4. 化科所落宫位：我有没有在这个领域过度追求名望？化科是贵气，但也是虚荣心所在。
5. 化忌所落宫位：我此生最大的课题是什么？我有没有看清自己的执着点？是怕什么？还是想要什么？
6. 我学的是哪一派？中州派的「看庙旺星性」还是飞星派的「忌星追踪」？两派解读差异大，不可混用。

### 字数与术语自检

- 本篇共涉及四化 4 颗（化禄/化权/化科/化忌），与「四化」声称一致。
- 化禄与禄存已明确区分（化禄动态、禄存静态），未混。
- 化忌与自化忌已明确区分（化忌动态因缘、自化忌自寻烦恼不持久），未混称（本篇只讲生年化忌，自化忌留待 5-3）。
- 生年四化与宫干飞化、大限四化已分层，未混（本篇只讲生年四化，后两层留待 5-3、5-4、5-5）。
- 中州派 vs 飞星派对生年化忌的解读已标注流派出处（王亭之 vs 许铨仁）。
- 「大限」为紫微术语，未用「大运」。
- 本篇只讲技法层面「生年四化所落宫位看先天格局底色」，自省应用层面留待第 8 章 8-2「用四化照见人生课题」，未越界。
