## 概念

### 天相：印星的辅佐与制衡

天相阳水，天府星系中的印星。水主智而阳水主显，故天相主印信、辅佐、显达。天相的形象，是一位执掌印信的宰相——辅佐君主、出纳王命、制衡百官，但也容易随环境而变、缺乏主见。

天相是十四主星中最"中庸"的一颗。天相坐命的人，庙旺者主有印信、辅佐有力、事业显达；落陷者主随波逐流、缺乏主见、印信失。天相的"印"，既是权力的象征，也是制衡的工具——印得其所则成辅佐，印失其所则成傀儡。

### 天梁：荫星的清高与孤

天梁阳土，天府星系中的荫星。土主信而阳土主刚，故天梁主荫庇、清高、孤高。天梁的形象，是一位清高的长者——荫庇后辈、清高自持、孤高不群，但也容易孤芳自赏、难与人合作。

天梁是十四主星中最"清"的一颗。天梁坐命的人，庙旺者主有荫庇、清高有名、寿元长；落陷者主孤高自赏、难与人合作、荫庇减。天梁的"荫"，既是庇护之力，也是孤高的根源——荫得其所则成清高有名，荫失其所则成孤芳自赏。

### 七杀：将星的刚猛与变动

七杀阴金，天府星系中的将星。金主义而阴金主刚烈，故七杀主刚猛、决断、变动。七杀的形象，是一位冲锋陷阵的将军——刚猛、果决、变动不居，但也容易孤克、刑伤、难安顿。

七杀是十四主星中最"刚猛"的一颗。七杀坐命的人，庙旺者主刚猛有成、决断力强、事业有成；落陷者主孤克刑伤、变动不安、难安顿。七杀的"将"，既是决断之力，也是变动的根源——将得其所则成刚猛有成，将失其所则成孤克刑伤。

### 破军：耗星的破立与颠覆

破军阴水，天府星系中的耗星。水主智而阴水主耗，故破军主破立、颠覆、消耗。破军的形象，是一位破旧立新的改革者——破立有功、颠覆旧局、消耗旧物，但也容易大起大落、难持久。

破军是十四主星中最"破"的一颗。破军坐命的人，庙旺者主破立有成、改革有力、事业有成；落陷者主大起大落、难持久、消耗过多。破军的"耗"，既是破立之力，也是消耗的根源——耗得其所则成破立有成，耗失其所则成大起大落。

## 原理

### 天相庙旺与落陷

天相的庙旺，主要在丑、未、卯、酉等宫支。庙旺的天相，主有印信、辅佐有力、事业显达、为人忠厚；落陷的天相（如在巳、亥等宫），主随波逐流、缺乏主见、印信失。

天相的解读，有一条重要规则——"逢府看相，逢相看府"。天相与天府永远在命盘上相对（隔六宫），天府为库主，天相为印星，二者互为表里。看天相必看天府，看天府必看天相——天府庙旺则天相亦有根，天府落陷则天相亦失据。

### 天梁庙旺与落陷

天梁的庙旺，主要在子、午、卯、酉等宫支。庙旺的天梁，主有荫庇、清高有名、寿元长、为人正直；落陷的天梁（如在巳、亥等宫），主孤高自赏、难与人合作、荫庇减。

天梁的"荫"，是其最核心的性质。天梁为荫星，主荫庇后辈、逢凶化吉。但天梁的荫，常带"孤"的代价——荫庇别人的人，自己也容易孤高。这是天梁解读中的辩证法：荫与孤，是一体两面。

### 七杀庙旺与落陷

七杀的庙旺，主要在寅、申、巳、亥等宫支。庙旺的七杀，主刚猛有成、决断力强、事业有成、有领导力；落陷的七杀（如在卯、酉等宫），主孤克刑伤、变动不安、难安顿。

七杀的"将"，本主变动与刚猛，故七杀的庙陷差异相对较小——无论庙陷，七杀都主变动。差异在于：庙旺的变动是有成的变动，落陷的变动是难安顿的变动。这是七杀与破军的共性——两星本主变动与破立，庙陷只决定变动的"成败"，不改变变动的"性质"。

### 破军庙旺与落陷

破军的庙旺，主要在子、午等宫支。庙旺的破军，主破立有成、改革有力、事业有成、有开创力；落陷的破军（如在巳、亥等宫），主大起大落、难持久、消耗过多。

破军的"耗"，本主破立与消耗，故破军的庙陷差异也相对较小——无论庙陷，破军都主破立。差异在于：庙旺的破立是有成的破立，落陷的破立是大起大落的破立。

七杀与破军的关系，是天府星系中的重要张力。七杀主刚猛之变，破军主破立之耗——七杀是"将"的变动（有方向），破军是"耗"的变动（无定形）。二者皆与天府（库主）相对，构成"库—将—耗"的三角动态。

## 实践

### 天相的"逢府看相，逢相看府"

天相的解读，最关键是"逢府看相，逢相看府"：

- 天相庙旺 + 天府庙旺：相印相生，主印信稳固、辅佐有力。
- 天相庙旺 + 天府落陷：相有根而库无据，主辅佐有力但财库不稳。
- 天相落陷 + 天府庙旺：相无据而库有根，主财库稳固但印信失。
- 天相落陷 + 天府落陷：相库皆失，主随波逐流、财印两失。

这是天相解读中的基本规则。读者解盘时，切忌只看天相不看天府——天相的吉凶，很大程度上取决于天府的状态。

### 天梁的荫与孤的辩证

天梁的解读，要在"荫"与"孤"之间找平衡：

- 天梁庙旺 + 化禄：荫庇有财，主清高有福。
- 天梁庙旺 + 化权：荫庇有威，主清高有权。
- 天梁庙旺 + 化科：荫庇有名，主清高有声望。
- 天梁庙旺 + 左辅右弼：荫有辅佐，主清高不孤。
- 天梁落陷 + 煞星：孤高自赏，主孤芳自赏、难与人合作。
- 天梁 + 太阳（三合）：驱暗为明，主清高有显达之机。

天梁的"荫"与"孤"，是一体两面——荫庇别人的人，自己也容易孤高。这是天梁解读的辩证法。读者解盘时，要同时看到荫与孤的两面，切忌只看一面。

### 七杀与破军的将耗对照

七杀与破军的对照，是天府星系解读中的重要视角：

| 维度 | 七杀（将） | 破军（耗） |
|---|---|---|
| 五行 | 阴金 | 阴水 |
| 性质 | 刚猛、决断、有方向 | 破立、消耗、无定形 |
| 变动方式 | 将的变动（有方向） | 耗的变动（无定形） |
| 庙陷差异 | 较小（变动性质不变） | 较小（破立性质不变） |
| 与天府关系 | 库有将护 | 库受耗破 |

七杀与破军的共性是"变动"，差异是"变动的方向性"——七杀有方向，破军无定形。这是中州派解读两星的基本视角。

实战中，七杀与破军的解读，更重与辅煞的搭配，而非庙陷：

- 七杀 + 左辅右弼：将得辅佐，主刚猛有成。
- 七杀 + 擎羊陀罗：将受刑伤，主孤克刑伤。
- 破军 + 化禄：耗中有财，主破立有成。
- 破军 + 化忌：耗失其所，主大起大落。
