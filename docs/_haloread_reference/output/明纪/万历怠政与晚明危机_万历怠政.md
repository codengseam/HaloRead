# 明纪·万历怠政

万历十年（1582），张居正死了。二十岁的皇帝朱翊钧松了口气——那个天天盯他读书、管他花钱、连夜里玩个灯火都要训他的张先生，终于没了。

松完这口气，报复就来了。

## 一、不是懒

人们后来说万历懒，说他二十多年不上朝，是明朝最怠惰的皇帝。这话只说对了一半。万历不是懒，万历是赌气。一个赌气的皇帝，比一个懒的皇帝可怕得多——懒是没力气干，赌气是有力气偏不干。

要看懂这份赌气，得从张居正活着的时候说起。

朱翊钧十岁即位，前十年是张居正辅政。张居正是明朝最能干的首辅，也是个最严的老师。小皇帝读书，张居正坐在旁边盯着；背不出《大学》，要挨训；想花钱修个灯火，张先生说"祖宗成法，不可轻费"。一个十岁的孩子，被这么管了十年，怕是怕的，恨也是恨的。可他不敢恨——张先生手握大权，连他亲娘李太后都站张先生那边。李太后吓唬小皇帝的话是："使张先生闻，奈何！"——让张先生知道了，看你怎么办！

一个皇帝，被一个大臣管得这么死，这在中国历史上是头一遭。张居正不是权臣，可比权臣还权臣——他用"考成法"整顿吏治，用"一条鞭法"改革赋税，清丈田亩，太仓银库从入不敷出攒到够支十年。明朝中后期最像样的日子，就是这十年。

可张居正一死，风向就变了。

万历十二年（1584），神宗下诏抄张居正家。抄家的太监赶到江陵，把张府大门封死，等开门时里头已经饿死十几口人。张居正的长子张敬修被拷打自缢，三子张懋修投井未死。抄出黄金万两、白银十余万两——神宗觉得被欺骗了：你张先生天天教训朕节俭，自己却攒了这么多钱？

这份"被欺骗感"，是神宗一辈子的心结。他从此不信治国，不信清流，也不信文官。他觉得所谓治国，不过是张居正那套说教；所谓清流，不过是一帮嘴上仁义、手里捞钱的伪君子。这份不信，是他后来赌气赌到死的根。

可神宗不是糊涂人。亲政头几年，他还像模像样——上朝、批章、见大臣。宁夏哱拜叛乱，他调李如松去平；朝鲜抗倭，他派宋应昌、邢玠去打；播州杨应龙反，他用李化龙去剿。三大征全赢，调度有方，用人得当。这说明神宗不是没本事。

那他后来为什么什么都不干了？

## 二、什么都明白

国本之争，是神宗赌气的真正起点。

他想立郑贵妃生的三子朱常洵当太子。文官集团死保"立嫡立长"的祖制，非要他立皇长子朱常洛。两边僵了十五年。神宗贬姜应麟、贬孙如法、贬雒于仁，贬了一个来十个，贬了十个来一百个。文官是打不怕的——你贬他，他成名；你廷杖他，他成烈士。最后他拗不过，万历二十九年立了朱常洛，可心里那口气咽不下去。

从这一年起，他不上朝了。

不上朝到什么程度？奏疏送上去了，留中不发。留中不是不批，是连看都不看，往那一搁。大臣的奏疏石沉大海，等上几个月没回音是常事。万历三十一年一年间，内阁收到奏疏五百余道，神宗批的不到十分之一。剩下的全留中。

更绝的是不补官。明朝官员升迁、调任、致仕、丁忧，缺了就得补。可神宗不补。万历三十年，吏部尚书缺了，不补；六部侍郎缺了一片，不补；都察院左都御史缺了，不补。六科给事中该有五十八人，万历三十年只剩不到十人。地方上更惨——知府缺了三四十个，知县缺了近百个。有些县衙里头，知县、县丞、主簿全空，政事没人管，百姓打官司都没处打。

大臣熬不住，纷纷上疏求去。可神宗连辞职都不批。你求去，他留中；你再求去，他还留中。有的官员等不及，干脆"挂冠而去"——把官帽往衙门一挂，自己走人。万历三十七年（1609），内阁只剩叶向高一个人，叶向高连上几十道疏求增补阁臣，神宗一概不理。叶向高一个人干五个人的活，干到吐血。

神宗什么都明白。他明白矿税之弊——他对太监说过"朕非不知矿税之弊"；他明白缺官之害——叶向高上疏说"政事废弛"，他没反驳；他明白立储之争会动摇国本——最后他还是立了常洛。可他明白归明白，做归不做。

孟森在《明史讲义》里有一句话，比任何骂他的话都狠："神宗非昏庸之主，乃怠惰之主。昏庸者不知事，怠惰者知事而不为。知而不为，其祸甚于不知。"

一个糊涂皇帝，你劝他，他懂了就做。可一个明白皇帝，你劝他，他说"朕知道"，可还是不做。这份"知道却不做"，堵死了所有劝谏的路。神宗不是不会治国，是不信治国，也是不愿治国——他跟文官赌气赌了一辈子，赌到最后，他什么都不想干了。

可他什么都不干，不等于什么都不干。有一件事他干得最起劲——敛财。

## 三、留中的讽刺

要说清神宗的赌气，得说一样东西——留中。

留中是神宗的武器。皇帝不上朝、不批章、不见大臣，看似是什么都不干，其实是用"什么都不干"来对抗文官。你们不是爱管吗？你们不是爱谏吗？那朕什么都不干，看你们怎么办。

这份消极抵抗，是皇权对文官系统的软刀子。神宗不能正面废除文官系统——祖制在他头上，他废不了。可他能"什么都不干"，让文官也干不成。你上疏，朕不看；你请补官，朕不补；你求去，朕不批。一份奏疏从内阁票拟，送到御前，搁在那里烂掉——制度的信息流断了。一个知县缺了三年不补——制度的执行层空了。一个阁臣求去求了三年没回音——制度的退出机制堵了。

一个制度，信息流断、执行层空、退出机制堵，这个制度就死了。名义上还在，实际上不转了。

黄仁宇在《万历十五年》里看透了这一层。他说万历朝的瘫痪不是皇帝一个人懒，是整个制度僵死了——皇帝想做的事做不了（立常洵为太子），文官想做的事做不成（请立常洛、罢矿税）。两边僵持，谁也动不了谁。皇帝用"不作为"对抗文官，文官用"祖制"对抗皇帝，制度本身却瘫痪了。这份瘫痪，是明朝政治体制的总崩溃——不是某个人坏，是整个制度转不动了。

这份"转不动"，是专制体制的死穴。明朝没有独立于皇权的立法机构，没有独立于皇权的司法机构。皇帝不批，没人能逼他批；皇帝违法，没人能判他违法。东汉桓帝、灵帝怠政，宦官专权，毁掉的是东汉的官僚系统；唐后期皇帝怠政，藩镇割据，毁掉的是唐朝的中央集权。神宗的留中，毁掉的是明朝的整个文官体制。这份毁，是消极抵抗的代价——伤敌八百，自损一千。

可神宗不在乎。他赌气赌到这份上，已经不管明朝死活了。他只管一样东西——钱。

## 四、不上朝不是放手，是攥得更紧

万历二十四年（1596），神宗开始派太监到各地开矿、收税。这就是"矿税之弊"。

起因是宫里头花钱如流水。三大征打了七八年，军费花了一千多万两；修三大殿又几百万两；养皇族宗藩又是一笔大开销。国库早空了。户部不肯给钱，神宗就自己想办法——派太监到各地开矿收税，钱直接进内帑，不过户部。

这一派，就派出了祸害。

派到山东的是陈增，派到临清的是马堂，派到辽东的是高淮，派到陕西的是梁永，派到云南的是杨荣。这些太监带着一帮爪牙，到了地方就敲诈勒索。开矿开不出矿，就逼百姓按户摊派"矿税"；收税没有名目，就见什么收什么——鸡鸭鱼肉、布匹绸缎、嫁娶丧葬，样样都要抽。富户被诬指"盗矿"，不交钱就下狱；地方官敢阻拦，就参一本撤职。

临清最先炸了。万历二十七年，马堂在临清设税卡，过路商贩一律抽税，连小贩挑担子进城都要交钱。临清是运河大码头，商贾云集，这一抽，把市面抽死了。手工业主王朝佐带头反抗，聚众数千人围了税监衙门，放火烧门。马堂的爪牙开枪杀人，王朝佐挺身而出，被官府抓去杀了。王朝佐临刑，临清百姓"罢市相送"，沿街哭声震天。

武昌也炸了。万历二十九年，税监陈奉在湖广横征暴敛，掘人坟墓、淫人妻女，无恶不作。武昌百姓聚众数万，把陈奉的衙门围了，放火烧了。陈奉翻墙逃跑，他的爪牙被百姓打死二十多个。神宗不但不惩办陈奉，反而把上疏弹劾陈奉的冯应京下了诏狱。

最惨的是云南。万历三十四年，税监杨荣在云南搜刮日甚，杖毙百姓、勒索官绅，连云南知府都被他打了一顿。云南百姓忍无可忍，万人起事，攻入杨荣府第，把杨荣和他的爪牙二百余人全部杀死，焚尸扬灰。神宗接到奏报，气得几天不吃饭，可云南天高皇帝远，最后也只能不了了之。

谷应泰在《明史纪事本末》里有一句话，戳中了要害："非阉人好为暴，实主上欲聚敛。"——不是太监天生爱施暴，是皇帝要敛财。太监之所以敢横行，是因为背后有皇帝撑腰；皇帝之所以要聚敛，是因为内帑空虚。可内帑空虚的根子，又是皇帝花钱无度。这是一个恶性循环——皇帝花钱→内帑空→派太监搜刮→太监横行→民变→军费镇压→国库更空→皇帝更缺钱。

这份敛财毁国，比穷兵黩武更隐蔽。穷兵黩武是明着花钱，看得见——汉武帝打匈奴，国库空了，大家都看得见。可敛财是暗着掏——皇帝派太监搜刮，钱进内帑，国库没多一分，百姓少了一分。汉武帝打匈奴打空了国库，可他至少打出了疆土；神宗搜刮掏空了民间，可他什么都没打出来——钱进了内帑，花了修陵、养宗藩、养太监，没一样是利国的。

大臣们一拨接一拨上疏，请罢矿税。万历二十八年，吏部尚书李戴率九卿合疏极谏，神宗留中。万历三十一年，礼部侍郎郭正域上疏，神宗留中。内阁首辅沈一贯也谏，神宗还是留中。沈一贯有一回劝得神宗松了口，答应罢。沈一贯大喜，立刻草诏。可诏书还没发，神宗反悔了——派太监到沈一贯府上，把诏书追回来。沈一贯捧着追回的诏书，老泪纵横。

沈一贯哭什么？他哭的是拔不掉。他看清了矿税是毒瘤，可他拔不掉。神宗反悔，他没办法。这份没办法，是首辅的限——首辅再大，大不过皇帝。

## 五、明亡于万历

万历四十八年（1620）七月，神宗朱翊钧驾崩，年五十八岁。在位四十八年，是明朝在位最久的皇帝。遗诏里头，他终于答应罢一切矿税——可这份遗诏，来得太晚了。

张廷玉等修《明史》，在《神宗本纪》末尾下了这样一段判语："神宗冲龄践阼，江陵秉政，综核名实，国势几于富强。继乃因循牵制，晏处深宫，君臣否隔，政事怠废。至乃矿税繁兴，竖珰四出，毒流海内，民不聊生。论者谓明之亡，实亡于神宗，岂不谅欤！"

——论者谓明之亡，实亡于神宗。

这句话分量极重。张廷玉等是清朝史官，修《明史》时距明亡不过几十年。他们没把明亡归咎于崇祯（亡国之君），而是归咎于万历——因为崇祯朝的烂摊子，根子都在万历朝埋下。万历怠政二十多年，矿税搜刮民穷财尽，缺官不补政事荒废，党争起于国本之争——这些病根，崇祯接手时已经病入膏肓。崇祯再勤政，也救不回来了。

1956年到1958年，考古工作者发掘了定陵——神宗的陵墓。打开棺椁一看，神宗的右腿蜷曲，骨骼变形，是个明显的腿疾患者。有人说这印证了他不上朝有身体原因。可定陵还出土了大量金银珠宝、丝绸织锦，说明神宗在宫里头日子过得不差。他不上朝，可他在后宫享乐、敛财、修陵，一样没少。腿疾让他不便上朝，可不便上朝不等于不能批奏章。他不批、不见，是懒得批、懒得见——更是赌气赌到不愿批、不愿见。

定陵的考古发现，给了神宗一个"身体原因"的台阶，可这个台阶挡不住"怠政"的定性。一个皇帝腿脚不便，可以坐着批奏章，可以召见大臣议事。他不干，不是不能干，是不愿干。

神宗这一辈子，最让人脊背发凉的不是他干了什么，是他什么都不干。一个皇帝，二十多年不上朝，奏疏堆积如山不看，官员缺了一半不补，民变四起不理，辽东努尔哈赤崛起不顾——他把"什么都不干"当成武器，跟文官赌气赌了一辈子。可他赌掉的不是文官的脸面，是明朝的国运。

他死后仅仅二十四年，李自成攻入北京，明朝灭亡。

崇祯在煤山歪脖子树上吊死的时候，大概想不到，把他逼到那一步的根子，是六十年前他爷爷赌的那口气。神宗赌气赌掉了国运，崇祯勤政也救不回来。这份"救不回来"，才是万历怠政最残酷的代价——一个皇帝的赌气，要了一个王朝的命。

---

## 结语

神宗什么都明白，可什么都不做——明白却不做，比糊涂更毁制度；赌气二十年，把张居正攒的家底败光，把明朝的命数耗尽。
