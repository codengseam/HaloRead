# 明纪·国本之争

万历九年（1581）那一夜的事，神宗朱翊钧后来大概一辈子没忘。

那天他到慈宁宫给生母李太后请安。太后不在，端水进来的，是一个姓王的宫女。少年天子一时兴起，临幸了她。按规矩，皇帝临幸谁要赏信物记档，可神宗没给。他嫌丢人——王氏是个宫女，出身太低。他想赖掉。

可赖不掉。王氏怀了孕。肚子一天天大起来，李太后看出来了。太后问神宗，神宗不认。太后让人取来《内起居注》——皇帝每天的行止都记着呢，白纸黑字。神宗没法再赖，低头认了。李太后倒没发火，只说："吾老矣，犹未有孙。果男者，宗社福也。"——我老了，还没抱上孙子。若是男孩，那是宗社的福气。

万历十年八月，王氏生下一个男孩，就是皇长子朱常洛。母以子贵，王氏被册为恭妃。可这一册，是冷宫的开始。

## 一、赢了气头，输了国本

神宗心里头压根没有这对母子。他宠爱的是另一个人——郑贵妃。

郑贵妃是万历十年选入宫的，大兴人。她不是最漂亮的，可她有一样旁人没有的东西——她敢跟神宗"闹"。别的妃嫔见了皇帝规规矩矩，郑贵妃不一样，她会撒娇，会顶嘴，会跟皇帝开玩笑。神宗在朝堂上被文官压得喘不过气，到了后宫，只有郑贵妃这里能松一口气。这一松，松出了一辈子的偏爱。

万历十四年（1586）正月，郑贵妃生下皇三子朱常洵。神宗大喜，立刻晋郑贵妃为皇贵妃。这一晋，朝臣炸了。

为什么炸？皇长子朱常洛已经五岁了，他母亲王恭妃还只是个恭妃；皇三子刚出生，母亲就晋了皇贵妃——这分明是偏爱。再往深里想，皇贵妃位在诸妃之上，仅次于皇后，将来若要立太子，神宗想立谁，岂不是秃子头上的虱子？

内阁首辅申时行率先上疏，请册立皇长子为太子。神宗的回复很硬——"元子尚幼，待二三年后举行"。孩子还小，过两年再说。

这一拖，就是十五年。

神宗为什么偏要立福王？人们后来说他是被郑贵妃迷了心窍。这话只对了一半。神宗宠爱郑贵妃是真，可他想立福王，更深的根子是赌气——他要证明自己是皇帝，自己说了算。文官越是死保"立嫡立长"的祖制，他越是要逆着来。这是皇权对文官系统的较劲。一个皇帝，连立个太子都立不成，这皇帝当得有什么意思？神宗偏要立，立不成，他就拖。拖，是他唯一的武器。

可这份赌气，赌的不是文官的脸面，是国运。

## 二、贬不完

朝臣不依不饶。户科给事中姜应麟上疏："贵妃虽贤，所生为次；恭妃虽微，所生为长。今晋封贵妃而恭妃不与，是轻重倒置。"神宗大怒："册封乃朕意，岂容臣下置喙！"把姜应麟贬到广昌县当典史。

姜应麟一贬，反倒成了榜样。言官们前赴后继。孙如法、沈璟、孙维城……贬了一批又来一批。神宗发现，这帮文官是打不怕的——你贬他，他成名；你廷杖他，他成烈士。越打越上。

这份"贬不完"，是文官集团的力量。神宗可以贬姜应麟、贬孙如法、贬雒于仁，可他贬不完——贬了一个来十个，贬了十个来一百个。明代自土木堡之变后，文官集团崛起，内阁权力日重。到了万历朝，文官集团已经形成一套完整的制衡皇权的机制——言官上疏、内阁封驳、廷议公决。神宗想立福王，不是他一个人说了算，得经过内阁拟旨、六科给事中审核、廷议通过。文官集团卡在每一个环节上，神宗寸步难行。

到了万历二十年，神宗想了个"妙招"——三王并封。皇长子、皇三子、皇五子（后来夭折），三个一起封王，都不立太子。等将来皇后生了嫡子再说。这叫"待嫡"。首辅王锡爵接到旨意，头都大了。他夹在中间两头为难，想了个折中：并封可以，但得约定年限，过期就立长子。可文官不买账。翰林院编修王就学、给事中史孟麟围住王锡爵，当面质问："皇长子已十一岁，岂有并封之理？"王锡爵被问得汗流浃背，回头再劝神宗。神宗拗不过，三王并封之事作罢。

王锡爵后来无奈道："并封之事，吾亦知其不可，然上意如此，吾何以处？"——首辅这个位子，在国本之争里就是个受气包，两头不讨好。

万历十七年十二月，大理寺评事雒于仁上了一道《酒色财气四箴疏》，专挑神宗的毛病——酒、色、财、气四样，一样一样数落。神宗气得七窍生烟，要杀雒于仁。内阁申时行劝住，说杀了反倒坐实了疏里的话。最后雒于仁被革职为民。

这一桩桩一件件，把神宗和文官集团推到了对立面。神宗的招数就一个字——拖。他不上朝，不见大臣，不批奏章，把奏疏"留中"——压着不处理。文官上疏如石沉大海，急得跳脚。可神宗就是不动。

## 三、拖的不是储君，是规矩

神宗的"拖"，不是无能，是武器。

嫡长子继承制是中国古代皇统继承的铁律。《春秋公羊传》说："立嫡以长不以贤，立子以贵不以长。"这条规矩从周朝定下来，两千多年没大变。为什么？因为"贤"没法衡量，你说这个贤，我说这个贤，争起来没完；"长"是客观的，谁先出生谁就是。立嫡立长，是为了避免争议。

可到了神宗这里，皇后无子，皇长子是宫女所生，皇三子是贵妃所生——按"立子以贵"，郑贵妃比王恭妃贵，是不是该立福王？这就是神宗"待嫡"借口的由来：他说要等皇后生子，立嫡子。可皇后生不出，"待嫡"就是无限期拖延。文官看穿了这一点，所以坚持"无嫡立长"。

文官集团死守"立长"，不是因为他们不喜欢福王，是因为他们知道——制度一旦被突破，后果不堪设想。一旦开了"立贤"的口子，每个皇子都可以说自己贤，争起来没完。司马光在《资治通鉴》里讲过："立嫡以长，正也；立庶以贤，权也。权不可常用，正不可暂失。"文官集团守的就是这个"正"。

神宗看不清这个边界，非要硬闯，结果闯进了怠政的死胡同。夏燮在《明通鉴》里点出一个要害：神宗每次拖延立储，理由都不同——一会儿说"元子尚幼"，一会儿说"待嫡"，一会儿说"三王并封"。这些理由都是借口，神宗的真实想法只有一个——想立福王。可他又不敢明说，因为祖制和舆论都不允许。这种"想做又不敢做"的状态，把神宗逼进了怠政的死胡同。

到了万历二十九年，皇长子朱常洛已经十九岁了。十九岁还没读书——神宗不让他出阁读书。出阁读书是皇子的规格，出阁了就等于承认他是储君的预备。这年八月，内阁沈一贯再请册立。神宗终于松口——大概他也拖不下去了，慈圣李太后还在世，老太太一直催；再加上福王朱常洵已经十五岁，按规矩也该就藩了。

万历二十九年十月，神宗册立皇长子朱常洛为皇太子，册封皇三子朱常洵为福王。

神宗赢了——太子没换成；可他输了——把万历朝的政治拖垮了。

## 四、查不出的荒唐

可事情没完。按规矩，福王封了王就该就藩洛阳。可神宗舍不得，拖着不让福王走。朝臣又急了——福王不走，是不是皇帝还想换太子？这一拖又是十三年。万历四十二年（1614），在首辅叶向高的反复力争下，福王终于就藩洛阳。临行前，神宗拉着儿子的手哭了。这一哭，是父亲的心；可这一拖，是皇帝的病。

更可怕的事还在后头。

万历三十一年（1603），京城流传一本匿名小册子《续忧危竑议》，说郑贵妃勾结大臣要废太子立福王。这就是"妖书案"。查来查去查不出主使，最后抓了个叫皦生光的生员顶罪凌迟。朝臣都知道皦生光是个替死鬼，可谁也不敢深究。

万历四十三年（1615）五月初四，更骇人的事来了。一个叫张差的男子，手持枣木棍，闯进太子朱常洛住的慈庆宫，打伤守门太监李鉴，直扑前殿。案子审到后来，牵涉到郑贵妃宫中的两个太监——庞保、刘成。这就是"梃击案"，明末三大案之一。

神宗不愿深究。郑贵妃哭哭啼啼去找太子，太子也怕事闹大，下旨说张差是疯癫之人速速处决。最后张差被凌迟，庞保、刘成在内廷秘密处死，案子草草了结。可天下人都明白——这背后一定有人。

梃击案最让人脊背发凉的，不是案子本身，是案子背后的那股暗流。一个手持木棍的平民，怎么能闯进太子的寝宫？宫禁森严，怎么就让他进去了？牵涉到郑贵妃宫里的太监，这背后是谁的主使？查不下去了——查下去就要动郑贵妃，动了郑贵妃就要动神宗的心头肉。神宗不让查，太子也不敢让查。这案子就这么"草草了结"。

可"草草了结"四个字，比任何判决都恐怖。它说明皇权可以压下真相，说明正义在皇权面前一文不值，说明太子的命——也就是国本——随时可能被人动手，而动手的人查不出来。这份"查不出来"，比梃击案本身更让人害怕。

## 五、冷暴力

国本之争，从万历十四年拖到梃击案，前后近三十年。这场仗里头，最惨的不是神宗，不是文官，是王恭妃和朱常洛这对母子。

神宗对王恭妃的冷落，是这场仗里最阴毒的一笔。王恭妃被幽禁景阳宫，十几年见不到儿子。朱常洛出阁读书前，几乎没受过正经教育。神宗不是不知道这样不对，他是故意冷落——他心里头想立的是福王。

李太后有一次直接问神宗："何以不立长子？"神宗脱口而出："彼都人之子。"——他是宫女生的。李太后大怒，因为李太后自己也是宫女出身。太后说："尔亦都人之子！"——你也是宫女生的！神宗吓得跪在地上，不敢抬头。

这一跪，是神宗的软肋。他知道理亏，可心里头不甘。所以他选择拖——用不上朝、不批奏来对抗文官。可他对抗的不只是文官，他还在折磨王恭妃和朱常洛。

王恭妃被幽禁景阳宫，病重时朱常洛去探望，宫门紧锁，朱常洛砸开锁才进去。母子相见，王恭妃已经看不见了——她哭瞎了眼。她摸着儿子的脸说："儿长大如此，我死何恨。"——儿子长这么大了，我死也值了。说完就咽了气。

这一幕，是国本之争里最惨的镜头。神宗不是不知道王恭妃的苦，他知道，可他不管。他宠爱郑贵妃，冷落王恭妃——这是人之常情；可他利用皇权，让王恭妃生不如死——这是皇权的恶。

人性里的偏爱，一旦加上权力的加持，就会变成制度性的恶。神宗不是暴君，他没杀王恭妃，没废太子；可他用冷暴力，让王恭妃幽禁至死，让朱常洛提心吊胆三十年。这种冷暴力，比暴君的屠刀更阴毒——它不见血，可它杀人。

朱常洛的苦命，到他当皇帝也没结束。万历四十八年（1620），神宗驾崩，朱常洛即位，是为光宗。可他只当了一个月皇帝，就因为"红丸案"死了。一个月的皇帝，一辈子的憋屈。《明史·光宗本纪》赞语里说他"潜德久彰，而享国日浅"——潜德久彰，可当皇帝的日子太短。这八个字，道尽了朱常洛一辈子的憋屈：当太子时提心吊胆，当皇帝时一个月就死了。国本之争的代价，最后是朱常洛这个苦命人来承受的。

## 六、明之亡实亡于神宗

国本之争这场仗，打的是储位，争的是制度，毁的是国运。

谷应泰在《明史纪事本末·争国本》末尾评论，说神宗"爱欲其生，恶欲其死"，宠爱郑贵妃便要立其子，厌恶王恭妃便冷落其子，这是"私情"压过了"公义"。可他又指出，神宗终究没敢废太子，是因为"祖制不可违，众论不可拂"。谷应泰认为，国本之争里文官集团前赴后继，"批鳞者接踵，廷杖者相望"，虽是"忠愤所激"，但也让神宗彻底寒了心，从此"君臣否隔"。这个评价，既肯定了文官的气节，也指出了这场对抗的政治代价——君臣两伤。

《明史·神宗本纪》赞语里，张廷玉等把话说到了底："晏处深宫，君臣否隔"，"纲纪凌夷，大臣争于廷，群臣竞于下"，"故论者谓明之亡，实亡于神宗"。

——明之亡，实亡于神宗。

国本之争不只是立储纠纷，它让神宗彻底放弃了治国，让明朝的政治机器停转，这才有后来的辽东危局、民变四起、最终亡国。

更深一层，国本之争还催生了晚明党争。围绕立储，朝臣分成两派——坚持立长子的"国本派"和倾向福王的"暧昧派"。这种分裂后来演变成齐党、楚党、浙党、东林党之争。顾宪成在东林书院讲学，以"国本"为旗帜——这本身是好事，士人关心国事。可一旦"关心国事"变成了"党同伐异"，麻烦就来了。国本之争里那些前赴后继的清流之士，恐怕想不到，他们用命争来的"国本"，最后竟成了党争的祭品。

国本之争这场仗，没有赢家。神宗赌气拖了三十年，拖垮了明朝的政治；文官死守祖制，守住了太子，可也守出了党争；王恭妃幽禁至死，朱常洛提心吊胆一辈子，当了一个月皇帝就死了。这场仗里头，每个人都觉得自己对——神宗觉得自己是皇帝说了算，文官觉得祖制不可违，郑贵妃觉得替儿子争个太子是人之常情。可每个人都对的结果，是把明朝拖进了深渊。

正义一旦变成党派的标签，正义就变质了。制度一旦被皇权的赌气突破，制度就死了。国本之争，皇帝拖住了太子，文官守住了祖制——可三十年的内耗，把万历朝拖成了"明之亡实亡于神宗"的死局。

---

## 结语

国本之争，皇帝拖住了太子，文官守住了祖制——可三十年的内耗，把万历朝拖成了"明之亡实亡于神宗"的死局；冷暴力比屠刀更阴毒，它不见血，可它杀人。
