# 明纪·鄱阳湖大战

元末争天下，有几场仗是绕不过去的。鄱阳湖大战，是其中最大的一场。

这一仗，前后打了三十多天。朱元璋以二十万对陈友谅号称六十万，以小船对楼船，最后一把火烧掉了陈友谅的舰队，也烧掉了陈友谅的命。战后朱元璋叹了一口气："友谅亡，天下不难定也。"可这一仗的来龙去脉，得从陈友谅这个人说起——他是个踩着别人尸骨爬上去的人，最后也死在这条路上。

## 一、踩着尸骨称帝

陈友谅是沔阳人，家里本是渔户。元末渔民地位低，被人看不起。陈友谅往上爬的心思，比谁都狠——这份狠，是被"看不起"三个字逼出来的。他先在县里当个小吏，嫌没出息。恰逢徐寿辉起兵，红巾军打到沔阳，陈友谅投了过去。

徐寿辉是个布贩子，被彭莹玉、倪文俊等人推为首领，建国号"天完"——这国号起得怪，"天完"二字压在"大元"上头，是要压倒大元的意思。可徐寿辉本人没什么本事，撑台面的是倪文俊。倪文俊是徐寿辉的太师，权倾一时。陈友谅投到倪文俊麾下，一路做到元帅。

可倪文俊自己先出了事。倪文俊想杀徐寿辉自立，事败，逃到黄州找陈友谅——陈友谅是倪文俊的部下。陈友谅二话不说，把倪文俊杀了，收了他的兵。这一手，是陈友谅起家的本钱。杀自己的恩主，眼都不眨。

杀了倪文俊，陈友谅的势力越来越大。他攻下安庆、池州、龙兴（南昌），把江西大部收入囊中。可徐寿辉还在。陈友谅嫌徐寿辉碍事，先把徐寿辉挟持到江州，自己以汉王自居。至正二十年（1360），陈友谅干脆把徐寿辉骗到采石矶，让人击杀于舟中，自立为帝，国号汉。

杀主自立这四个字，是陈友谅一辈子的烙印。本事在于，他敢下手，敢夺权；病在于，"杀主"两个字，让他的人心散了。陈友谅的部下，有不少人是徐寿辉的旧部。陈友谅杀了徐寿辉，这些人表面服，心里不服。这烙印，后来要了他的命。

陈友谅称帝那年，正是朱元璋在应天立足未稳的时候。陈友谅兵强，水军尤其厉害——楼船高数丈，外面包着铁皮，船舱里能跑马，号称六十万大军。他带着这支水军顺流东下，要一举端掉朱元璋。

## 二、龙湾的退潮

朱元璋这边慌了。有人主张投降，有人主张退守钟山，只有刘基一言不发。朱元璋把刘基叫到里屋问，刘基说：先斩主张投降和逃跑的人，再想办法。

刘基这句话，比任何计策都管用。一个谋士在最慌的时候不说怎么打，先说"斩"——斩的不是人，是乱了的心。朱元璋定下心来，才能想出招。

朱元璋用的招是诱敌。他让康茂才写信给陈友谅诈降，约陈友谅到江东桥接应。陈友谅信了——康茂才跟陈友谅是旧识。陈友谅带兵到了龙湾，朱元璋早已设伏。龙湾是个口袋地形，陈友谅的楼船开进来，正赶上退潮，大船搁浅，朱元璋伏兵四起，陈友谅大败，丢了两万多人，狼狈逃回江州。

龙湾一败，陈友谅憋了一口气。这口气憋了三年。

可这口气没把他憋清醒，倒把他憋得更骄。一个强人吃了亏，不是去想自己哪里错了，是去想下次怎么用更大的力气砸回去——这是强人的通病。陈友谅憋了三年，憋出来的不是反省，是更大的楼船、更多的兵、更狠的报复。

三年后，他倾国而来。

## 三、赢了八十五天，输了后半辈子

至正二十三年（1363），陈友谅倾国而来。他造了更大的楼船，高数丈，漆成红色，船舱三层，外裹铁皮，号称六十万，围攻洪都（今南昌）。洪都守将是谁？朱元璋的侄子朱文正。

朱文正是朱元璋大哥朱重四的儿子。朱元璋起兵时，朱文正来投，朱元璋问他想做什么官，朱文正说："叔父成大业，何患不富贵？爵赏先私亲，何以服众？"这话听着懂事，可朱文正骨子里是个不安分的人——一个能把"爵赏先私亲何以服众"说得这么漂亮的人，心里多半是委屈：我比谁都该赏，可我得装着不争。

陈友谅围洪都，朱文正死守。陈友谅的楼船太高，城墙又矮，陈友谅本以为一鼓可下。可朱文正硬是守了八十五天。城墙被攻破，朱文正一边修一边打；陈友谅攻抚州门，城墙破三十余丈，朱文正亲自带兵反击，把缺口堵上。八十五天，陈友谅顿兵坚城之下，进退两难。

这时朱元璋在安丰救韩林儿、刘福通，脱不开身。等朱元璋腾出手来，率二十万大军南下救援，陈友谅这才撤了洪都之围，东出鄱阳湖迎战。

没有朱文正死守洪都八十五天，就没有鄱阳湖的胜利。这八十五天给朱元璋争取了救援的时间，也耗掉了陈友谅的锐气。一个被堵在坚城下八十五天的强人，锐气早磨掉了一半——可陈友谅不觉得自己磨掉了，他还觉得自己是来报仇的。

可朱文正这个人，结局让人唏嘘。按理说，死守洪都这么大的功劳，该重重封赏。可朱元璋是朱文正的叔父，朱元璋觉得自家人不用太客气，封赏没给到位。朱文正心里不服，私下跟张士诚联络，要叛变。朱元璋发觉，把朱文正抓回来，免官软禁。朱文正后来死在软禁中。

一个死守洪都八十五天的功臣，最后落得这么个下场。朱文正的悲剧，是人性的悲剧——功劳太大，赏赐不够，怨气一上来，人就走了歪路。这跟陈友谅的死法是两种：陈友谅死于太强太骄，朱文正死于太委屈太不甘。一个被自己的强压死，一个被自己的委屈拖死。乱世里，人死法各有不同，可根子都在一个"心"字上——心不正，再大的功也救不了自己。

两军在鄱阳湖相遇。

## 四、小船对楼船

陈友谅的舰队，是真正的巨舰。楼船高数丈，连锁为阵，绵延数十里，旌旗蔽日。朱元璋的船小，跟陈友谅的楼船一比，像舢板对航母。头几天打下来，朱元璋吃了大亏——徐达的船被火攻烧了，朱元璋自己的座船也差点被张定边冲到跟前。

张定边是陈友谅的太尉，骁勇无双。鄱阳湖一战，张定边单舰冲阵，直扑朱元璋的座船。朱元璋这边慌了，船搁浅，张定边逼到跟前。要不是常遇春一箭射中张定边，朱元璋的命可能就交代在鄱阳湖里了。

这一箭，救了朱元璋，也救了大明。常遇春的勇，跟张定边的勇，是一个分量——一个冲阵要主公的命，一个一箭救主公的命。鄱阳湖上两个最勇的人，一个输了，一个赢了。可这一箭也透出朱元璋的底：他能活下来，不是靠自己比陈友谅更能打，是靠身边的人比陈友谅身边的人更肯替他拼命。张定边替陈友谅拼命，可陈友谅的部下不是个个都肯——杀主自立那一刻，肯替他拼命的人就少了一半。

可陈友谅犯了一个致命的错。他把楼船连锁。

这跟赤壁之战曹操的失误如出一辙。曹操在赤壁犯过的错，陈友谅在鄱阳湖又犯了一遍。陈友谅不是不知道赤壁的教训，是有他的苦衷：楼船太大，单船作战转向不便，跟朱元璋的小船对阵反而吃亏；连锁起来，可以形成稳定的作战平台，楼船上的士兵互相支援，小船冲不进来。可连锁的代价，就是一旦着火，火势蔓延，想散都散不开。陈友谅赌的是连锁的稳定性，朱元璋赌的是火攻的破坏力。历史有时就这么爱重复——不是人不知道教训，是人总觉得自己不一样，觉得自己能赌赢前人赌输的那一把。

更要命的是鄱阳湖的地理。鄱阳湖是中国最大的淡水湖，南通赣江，北连长江，湖面宽阔处烟波浩渺，可湖中有浅滩、有岛屿、有狭窄水道。陈友谅的楼船在湖面宽阔处占优势，可一旦进了浅水区、狭窄水道，吃水深、转向慢的楼船就成了活靶子。大船有大船的威风，也有大船的死穴——它离不开深水阔面。陈友谅把六十万大军塞进楼船，又把楼船连锁，等于把自己的命全押在"湖面够宽、风向往对"这两件事上。可湖面不会一直宽，风向不会一直顺。把命押在天时地利上的人，天时地利一变，命就没了。

## 五、一支流矢的讽刺

朱元璋备下了火船。船里装满火药、芦苇、鱼油，船头扎着草人披着盔甲，远远看去像一支押阵的水军。

那天，东北风起。朱元璋下令点火。火船借着风势，直冲陈友谅的连锁舰队。火船一撞上楼船，火势借着风，瞬间蔓延。陈友谅的楼船连锁，想散都散不开。一时间，火光冲天，湖水皆赤，陈友谅的舰队烧死、溺死无数。

陈友谅撑不住了。他从船舱里探出头来看战况——这一探头，一支流矢飞来，正中头颅。陈友谅当场毙命。

一支流矢。一个称帝三年的皇帝，死于一支不知道从哪里飞来的箭。陈友谅一辈子踩着别人的尸骨往上爬，杀倪文俊，杀徐寿辉，最后自己也成了一具尸骨，飘在烧红的湖面上。这结局，比任何讽刺都狠。

陈友谅一死，汉军大乱。太尉张定边护着陈友谅的次子陈理，趁夜突围，逃回武昌。陈友谅的舰队，就此覆灭。张定边这一手，是败军之将里最有人味的一笔——故主已死，大势已去，他完全可以投降保命。可张定边不降，护着故主的儿子，杀出重围。这份忠，跟陈友谅的"杀主"成了一个最刺眼的对照：主公杀主上位，太尉却为主公的儿子拼死突围。陈友谅一辈子不信人，可到底有一个人信他。

朱元璋站在船头，看着鄱阳湖上的火光，叹了一句："友谅亡，天下不难定也。"

这话不夸张。陈友谅一死，元末群雄里头号对手就没了，朱元璋的天下，算是真正立住了。可这一仗，朱元璋赢得不轻松——头几天吃亏，座船搁浅，差点被张定边要了命。他能赢，靠的是三样：天时、人和、自己沉得住气。东北风是天时，朱元璋把人为的部分做到极致——火船、火药、芦苇、草人，只等风来。风来了，一把火烧了陈友谅；风不来，火攻就废了。这是"谋事在人，成事在天"。可"成事在天"不等于听天由命——朱元璋把人为做到极致，然后让天时来成全。人和是更狠的：陈友谅的部下各怀鬼胎，朱元璋的部下肯拼命。火攻那把火烧的是楼船，可真正烧垮陈友谅的，是早散了的人心。

陈友谅输的不是船，是人。这话张廷玉他们看出来了，商传也看出来了。陈友谅用权术驭下，能压一时，压不了一世。杀主自立，是权术的极致，也是人心散尽的开始。陈友谅的政权，是强人政权，靠陈友谅个人的威权和武力维持。陈友谅一死，政权就散了。这跟朱元璋的政权不一样——朱元璋的政权，有制度，有人才，有根基。陈友谅的政权，只有陈友谅一个人。一个人撑起来的政权，人一倒，政权就塌。

陈友谅一辈子，输在一个"骄"字上。龙湾之败，是骄；围洪都顿兵坚城八十五天，是骄；鄱阳湖把楼船连锁，更是骄。人在弱的时候容易清醒，在强的时候容易糊涂。陈友谅兵最强的时候，恰恰是他最糊涂的时候。这跟曹操赤壁之败是一个道理——曹操统一北方后志得意满，赤壁一战烧掉了统一天下的梦；陈友谅兵强马壮时不可一世，鄱阳湖一战烧掉了称帝的命。《老子》说"祸兮福之所倚，福兮祸之所伏"。陈友谅的强，是他的福；陈友谅的骄，是他的祸。福祸相依，强极生骄，骄极生败。

大有大难，小有小利。陈友谅的楼船大，朱元璋的小船小。大船稳，可不灵活；小船晃，可转向快。陈友谅把楼船连锁，是想用大压小；朱元璋用火攻，是用小破大。这道理，放到今天做事也一样：最难的事往往是关键的事，可最大的家当也最容易变成最大的包袱——船大难掉头，楼船越大，烧起来越没法救。

## 结语

一把火烧了楼船，也烧出了大明的底子——可烧掉陈友谅的不是那把火，是他自己。杀主自立那一刻，人心就散了；人心一散，六十万楼船也撑不起一个人心。鄱阳湖的火，烧的不过是早就该倒的空架子。陈友谅这辈子，从杀倪文俊到中矢而亡，每一步都踩在别人的尸骨上，最后自己也成了那具尸骨——一个不信人的人，死在没人替他挡一支箭的船上。
