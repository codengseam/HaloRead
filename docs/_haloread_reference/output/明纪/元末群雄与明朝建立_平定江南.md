# 明纪·平定江南

灭张士诚这一仗，是朱元璋拿下江南的关键一仗。陈友谅一死，天下大势已定；可张士诚还盘在苏松，方国珍还守在海上，江南一日不平，朱元璋就一日腾不出手北伐。

可要先打张士诚，陈友谅就会从背后捅刀。朱元璋选了一条反常识的路——先打硬的，再打富的。

## 一、富地养软人

先说张士诚。张士诚是泰州白驹场人，小名九四，跟弟弟士义、士德一块儿贩私盐。私盐这行当，是刀尖上舔血——官府抓得紧，同行抢得凶，能干这行的，没一个是软骨头。张士诚贩盐贩了几年，渐渐聚起一帮弟兄。至正十三年（1353），张士诚带着十八个兄弟起兵，攻下泰州，接着拿下高邮，建国大周，自称诚王，改元天祐。

张士诚刚称王，元朝就派大军来剿。至正十四年，丞相脱脱亲率百万大军围高邮。这百万是虚数，可几十万总有。高邮一座孤城，眼看守不住。可就在这节骨眼上，元顺帝一道诏书，把脱脱罢了。脱脱一走，元军溃散，张士诚绝处逢生。

这一仗，是张士诚的命。脱脱要是没被罢，高邮城破，张士诚就完了。可元朝自己拆自己的台，张士诚捡了一条命。从此张士诚在淮东站稳了脚。

守高邮那阵子，张士诚是真硬。一座孤城，几十万大军围着，他守了几个月没投降。这份硬气是真的。可张士诚的硬气，是用在守势上的硬气，不是用在进取上的硬气。

至正十六年，张士诚渡江南下，一举攻下平江——就是苏州。接着常州、湖州、杭州、松江、嘉兴，浙西富庶之地，全落到张士诚手里。张士诚把都城从高邮迁到平江，自此据有江南膏腴之地。

可张士诚一据了苏松，人就变了。

苏松这地方，自古富庶，丝绸、漕粮、盐利，堆金积玉。张士诚本是盐贩出身，一下子坐拥金山，慢慢就耽于安乐。他弟弟张士信当丞相，朝政一塌糊涂。张士信爱蓄姬妾，造楼船，日日宴饮，政事全丢给手下。手下人贪赃枉法，张士诚也不管。

《明史·张士诚传》说他"外迟重寡言，似有器量，而实无远图"——外表稳重话不多，看着像有度量，其实没有长远打算。又说"为人宽柔，好施与"，可"政令弛废，将帅偃蹇"——为人宽柔，好施舍，可政令松弛，将帅骄横。宽柔本是好事，可宽柔到政令弛废，就是软。仁是有原则的宽，软是没原则的宽。张士诚的宽柔，过了线，成了软。

富地养软人。这话搁在别家身上未必准，搁在张士诚身上字字见血。一个盐贩子，靠着十八个弟兄打下高邮，靠着高邮之战侥幸活下来，一进苏松，骨头就软了。富地是金山，也是陷阱——据了富地，人有退路，有退路就不拼命。张士诚据了苏松，日日宴饮，因为他觉得有苏松在手，够了。可天下是争来的，不是守来的。守富地的人，遇到争天下的人，必败。

## 二、先打会动的

至正二十三年，张士诚自立为吴王。巧的是，朱元璋这一年也自称吴王。一个在苏州，一个在应天（南京），一东一西，史称"东吴""西吴"。两个吴王，迟早要碰。

可朱元璋没急着碰张士诚。他要先碰陈友谅。

陈友谅在上游，张士诚在下游。陈友谅强，张士诚富。先打谁？朱元璋跟刘基商量，刘基说：张士诚自守虏，不足虑；陈友谅据上游，名号不正，先灭陈友谅，张士诚势孤，可传檄而定。

刘基这话，点出张士诚的病——"自守虏"。自守虏就是只图自保、不思进取的人。这样的人，你打别人时他不会偷袭，你打他时他守不住。

朱元璋定下"先陈后张"，这是反常识的。常人的算盘是：先打弱的，再打强的；先打富的，再打穷的。张士诚富而软，陈友谅强而狠，照常理该先打张士诚。可朱元璋偏偏先打陈友谅。

为什么？因为陈友谅会动，张士诚不会动。

陈友谅是"会动的硬汉"。他据上游，随时能顺江而下打应天。你先去打张士诚，陈友谅一定从背后捅你一刀——那时候你前有张士诚，后有陈友谅，腹背受敌，必亡。张士诚是"不会动的富翁"。他只图自保，你打陈友谅，他不会动；他只会坐在苏州看戏，等陈友谅灭了，再轮到他发愁——可到那时候，发愁也晚了。

先打会动的，再打不会动的——这不是兵力的事，是看清谁会动、谁不会动。这眼光，跟常遇春"直捣大都"的武将思维是两路。武将算的是一仗之功，统帅算的是全局之胜。朱元璋是统帅，所以他先打硬的，把会动的按住，再回头收拾不会动的。

## 三、火海里烧出来的上游

至正二十三年七月，鄱阳湖。

陈友谅倾巢而出，号称六十万，楼船高数丈，旌旗蔽日。朱元璋以少敌多，在鄱阳湖上跟陈友谅死磕了三十六天。这一仗，是元末最惨烈的水战。

湖面上是火。陈友谅的楼船连成片，朱元璋用火攻——七条小船装满芦苇、火药，借风冲进敌阵，火借风势，把陈友谅的舰队烧成一片火海。水上的火比陆上的火更狠——船烧了，人往水里跳，可水里头是火油，跳下去也是死。那三十六天，鄱阳湖的水是红的，是黑的，是糊的——红的血，黑的灰，糊的是烧化了的船板和人肉。

陈友谅从船舱探头出来看战况，一支流矢贯穿了他的眼睛，当场死。陈友谅一死，汉军溃。朱元璋赢了。

这一仗，把朱元璋的上游烧安稳了。陈友谅是会动的硬汉，硬汉死了，上游没人能顺江而下捅应天的刀。朱元璋这才腾出手，回头收拾张士诚。

朱元璋后来回忆鄱阳湖，说"友谅亡，天下不难定"。这话不全对——友谅亡，只是西边定了，东边还有张士诚。可这话也对——友谅一亡，张士诚就成了砧板上的肉。一个不会动的人，等会动的人都被灭了，他自己也动不了了。

转过年来，至正二十四年，朱元璋开始收拾张士诚。

## 四、围一年的城

至正二十六年（1366），朱元璋发布《平周檄》，声讨张士诚八条大罪。这八罪，从张士诚起兵反元说起，一直数到他降元又叛、袭杀朱军、据地不朝。檄文里有一句最要紧——"张士诚一盐徒耳，窃据淮海，僭称王号"。这话听着是骂，其实是定调：你张士诚不过是个盐贩子，没有正统名分。

檄文一发，徐达、常遇春率二十万大军东征。朱元璋的打法，跟北伐是一个路数——先剪枝叶，再取根本。先取淮东——泰州、通州、高邮、淮安，把张士诚在江北的地盘全端了。张士诚的老巢高邮，就是这一仗丢的。

接着下湖杭。徐达攻湖州，常遇春攻杭州。湖州、杭州一丢，平江就成了孤城。张士诚的两臂被砍断，只剩一座苏州城。

平江之战，是这一仗的硬骨头。张士诚据城死守，徐达围城筑长围，从至正二十六年十一月围到至正二十七年九月，围了将近一年。

围城是苦差事。城里的粮食吃光了，张士诚的弟弟张士信在城楼上被炮石打死。张士信死的时候，还在城楼上摆酒——这个当丞相的，造楼船、蓄姬妾，打仗时还在城楼上宴饮，最后被炮石砸死。张士诚痛哭。可这时候哭有什么用？弟弟是他自己纵的，政事是他自己丢的，苏州的金山是他自己坐吃山空的。

张士诚两次突围，都没成功。城里的百姓饿得啃树皮，张士诚守着的是一座快死的城。

可张士诚还在守。这份守，跟当年守高邮的硬气，是一脉相承的。张士诚这个人，开头硬，中间软，结尾又硬。中间那段软，把开头和结尾的硬都葬送了——可结尾那段硬，倒让人没法只骂他软。一个在苏州听戏听到城破的人，城破后又上吊两回，这硬气来得晚，可来得真。

## 五、要脸的死了，不要脸的活了

至正二十七年九月初八，平江城破。张士诚上吊，被部下救下，没死成。被俘后押到应天，朱元璋问他话，他闭口不答。朱元璋气得要杀他，张士诚又上吊，这次还是没死成。最后朱元璋让人把他杀了，一说赐自尽。

张士诚一灭，方国珍就慌了。

方国珍是黄岩人，今浙江台州。至正八年（1348），方国珍被仇家告发，一怒之下杀了仇家，跟兄弟国璋、国瑛、国珉逃入海中，聚众数千，专劫元朝的海运粮船。元朝拿他没办法，招安他，授他官。方国珍降了又叛，叛了又降，前后反复好几次。

方国珍据有庆元（今宁波）、温州、台州，地盘不大，可卡在海运要道上。朱元璋崛起后，方国珍一会儿向朱元璋示好，一会儿又向元朝表忠，首鼠两端。一边向朱元璋称臣，一边又接受元朝的官职，还跟张士诚暗通款曲。方国珍心里打的算盘是：朱元璋跟张士诚谁赢还不一定，我先两头下注。

《明史·方国珍传》说他"首鼠两端，怀二心"——像老鼠出洞，左顾右盼，怀着二心。方国珍起兵最早，可一直没成大器，根子就在投机。投机的人，看眼前不看长远；算小账不算大账。

朱元璋灭张士诚时，方国珍还在观望。等平江一破，方国珍知道大势已去，想逃入海中。朱元璋派人写信给他，话说得很直：你逃到海里也没用，我水师能追你。方国珍无奈，只好投降。

至此，江南基本平定。朱元璋腾出手来，第二年——洪武元年（1368），就发兵北伐，直捣大都。

可这江南一平，平出个让人心里发堵的对照：张士诚硬到底，死了；方国珍软到底，活了。方国珍降了朱元璋，倒是得了善终——朱元璋授他广西行省左丞，洪武七年病死，算是元末群雄里少有的好结局。

要脸的死了，不要脸的活了，这账怎么算？历史不按脸算，按成败算。可这话只说了一半。张士诚的死，不是死在"要脸"，是死在"中间软"——他要是一直硬，像守高邮那样守到死，未必输得这么难看；他要是一直软，像方国珍那样两头下注到头，未必不得善终。偏偏他开头硬、中间软、结尾又硬，这三段不接茬，把命搭进去了。方国珍活得久，不是因为他软，是因为他从头软到尾，软得一致——一个一致的人，比一个反复的人，容易活。

## 六、苏松养不出雄心

苏松的金山银海，养得出盐贩子的富贵，养不出争天下的雄心。

这话不是骂张士诚，是骂苏松。或者说，是骂"富地"本身。富地这东西，能养人，也能毁人。会争的人据富地，如虎添翼；不会争的人据富地，如羊入笼。张士诚是羊入笼——他进了苏州，就再也没出来过。

朱元璋跟张士诚一比，差别就出来了。朱元璋也是穷出身，可朱元璋据应天后，节俭、勤政、用人才。张士诚据苏松后，奢纵、怠政、用亲信。两个人都是底层起家，一个越打越硬，一个越坐越软。根子在哪？在志向。朱元璋要的是天下，张士诚要的是富贵。要天下的人，富贵是手段；要富贵的人，富贵是终点。

这条规律，历史上有印证。三国时刘表据荆州，荆州富庶，刘表宽柔，可刘表只图自守，不思进取，最后荆州被曹操夺了。张士诚、刘表，都是据了富地、宽柔自守、最后败亡的人。富地是双刃剑，割得了敌人，也割得了自己。

方国珍是另一路的死法——他不是死于据富地，是死于没立身之本。投机的人，没有立场；没有立场，就没有根基。方国珍据庆元温台，地盘不大，可位置要紧。要是方国珍有立场，据海自强，未必不能成一方诸侯。可方国珍只想投机，不想立身，最后地盘守不住，只能降。

三国时的吕布，反复无常，投丁原杀丁原，投董卓杀董卓，投袁绍叛袁绍，最后被曹操擒杀。吕布的反复，跟方国珍的投机，是一路。反复的人，没有立场；没有立场，就没有盟友。方国珍比吕布幸运，他降了朱元璋，得了善终。可方国珍的善终，是侥幸，不是必然——朱元璋要不是忙着北伐，顾不上收拾方国珍，方国珍未必能善终。

投机的人，算的是小账；算小账的人，最后算空自己。方国珍算了一辈子，最后算到自己头上，只能投降。这账，算得再精，也是空。

朱元璋赢张士诚、收方国珍，靠的不是兵多，是看清了两个人的病——一个病在富地养软，一个病在投机算空。会动的人他先按住，不会动的人他后收拾；要脸的他逼到死，不要脸的他收到活。这一按一收，江南就平了。

---

## 结语

苏松的金山银海，养得出盐贩子的富贵，养不出争天下的雄心；海上的反复投机，算得了一时的小账，算不出一世的大账。朱元璋平江南，平的不是张士诚和方国珍的兵，是这两个人各自的病。
