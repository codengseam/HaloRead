# 明纪·袁崇焕之死

崇祯三年（1630）八月十六日，北京西市刑场。

一辆囚车缓缓推来。车上押着的人，曾是兵部尚书、蓟辽督师，曾手握尚方宝剑，曾率关宁铁骑千里入援京城，曾在广渠门外与后金军血战。可今天，他身穿囚衣，五花大绑，跪在刑场上。

他叫袁崇焕。罪名是"通虏谋叛"。

凌迟。千刀万剐。"寸磔于市"。

## 一、被自己人吃掉的统帅

凌迟是中国最酷的刑。一刀一刀地割，割满三千六百刀才许死。可这一天的西市，最惨的不是凌迟本身，是北京百姓的反应。

京城百姓恨袁崇焕入骨。他们信了——袁崇焕通敌，引后金军到北京城下。他们信得咬牙切齿。所以袁崇焕身上割下来的每一块肉，都有人抢着买。"刽子手割一块，百姓争买一块，开膛取肠，百姓群起抢之，以钱争买，和烧酒生啮，血流齿颊间。"——张岱记下了这一幕。

一个守过辽东、保过京城的统帅，被自己誓死保卫的百姓，一口一口吃掉了。

百姓愚昧吗？愚昧。可这份愚昧，不是天生的，是被喂出来的。他们听见城外的炮声，看见烧杀的火光，他们要一个交代——是谁放后金军进来的？总得有人。朝廷说是袁崇焕，那就是袁崇焕。百姓不需要真相，百姓需要泄恨的对象。袁崇焕就是那个对象。

这是这一场凌迟最残酷的地方——不是刀残酷，是嘴残酷。刀割的是肉，嘴嚼的是忠魂。一个为国家挡了十几年刀的人，最后被国家喂了百姓的嘴。

事情怎么会走到这一步？得从宁远说起。

## 二、宁远那一句"义当死守"

天启六年（1626）正月，努尔哈赤率十三万后金大军，直扑宁远。

宁远是明朝在辽西的最后屏障。守将就是袁崇焕。袁崇焕，广东东莞人，万历四十七年（1619）进士，文官出身，自请守辽。这个人，身材瘦削，说话带南方口音，可骨子里有一股狠劲。

明朝的文官，多数怕辽东——苦寒之地，战火之地，去了就可能回不来。袁崇焕自请守辽，这份胆气，朝中没几个人有。

努尔哈赤打了一辈子仗，从十三副遗甲起兵，统一女真，破抚顺，败杨镐于萨尔浒，下辽阳、沈阳，一路势如破竹。明朝在辽东的城池，丢了一座又一座。袁崇焕前任经略高第，吓得要把关外军队全撤回山海关内。袁崇焕不撤。他带一万兵，守孤城宁远。

努尔哈赤到了宁远城下，写信劝降。袁崇焕回信，只有一句——"义当死守，岂有降理"。

这一仗，袁崇焕用红夷大炮轰后金军。后金军攻城不下，努尔哈赤本人也被炮火所伤，被迫退兵。这是努尔哈赤起兵四十多年来，第一次攻城不下。八个月后，努尔哈赤病死（一说即死于宁远炮伤）。

宁远一战，袁崇焕名震天下。朝中有人感叹："奴贼猖獗，辽东非袁崇焕不能守。"

天启七年（1627），努尔哈赤的继承者皇太极再攻宁远、锦州。袁崇焕又守住了。这一仗叫"宁锦大捷"。可袁崇焕没等来升赏——因为他跟魏忠贤不和。魏忠贤让党羽弹劾袁崇焕，袁崇焕被迫辞职回家。

一个守住辽东的人，被阉党赶回了家。这是天启朝的荒唐。可这份荒唐，到崇祯朝会变本加厉——崇祯朝不是赶他回家，是把他送上刑场。

## 三、五年复辽的谎

崇祯即位，铲除魏忠贤，起复袁崇焕。

崇祯元年（1628）四月，崇祯在平台召见袁崇焕，任命他为兵部尚书、蓟辽督师，赐尚方宝剑。崇祯问：辽东事，几年可复？

袁崇焕脱口而出："计五年，全辽可复。"

崇祯大喜。一个想做事的皇帝，遇到一个敢承诺的统帅——这是崇祯朝开局最好的画面。可这画面是假的。

袁崇焕出来后，同僚问他：五年复辽，有把握吗？袁崇焕苦笑——"聊慰上意耳"。不过是哄皇帝开心罢了。

这句话，后来要了他的命。

袁崇焕为什么敢说这个谎？因为他知道崇祯想听。一个刚铲除阉党、急于求成的年轻皇帝，最想听的就是"几年可复"。袁崇焕给他画了一个饼，崇祯接了，还当真了。

可崇祯当真了，袁崇焕就得兑现。五年过去，辽东没复——不光没复，后金军还打到北京城下了。崇祯觉得被骗了。一个被臣子骗了的皇帝，比一个被敌人打败的皇帝，更恨这个臣子。

袁崇焕的错，不在说大话。说大话是明末官场的通病。他的错，在于不知道崇祯是会记账的皇帝。崇祯把"五年复辽"四个字，一笔一笔记在心里。记到合适的时候，一笔一笔算回来。

可袁崇焕还不知道。他正忙着做另一件大事——杀毛文龙。

## 四、擅杀的越权

崇祯二年（1629）六月，袁崇焕做了一件大事——杀毛文龙。

毛文龙据守皮岛（在今朝鲜椴岛一带），牵制后金后方。毛文龙是浙江仁和人，早年投军辽东。后金攻辽东，毛文龙带二百人逃到皮岛，招抚辽东逃民，渐渐聚起几万兵。皮岛在后金后方，毛文龙时不时袭扰后金，牵制了后金不少兵力。

毛文龙的功劳，是真的。可毛文龙的毛病，也是真的。他虚报兵额——号称有兵数万，实数不过两三万；他侵吞军饷——朝廷每年给皮岛百万军饷，大半进了他私囊；他不听调遣——朝廷让他打后金，他阳奉阴违；他私通后金——有书信往来，虽然未必真降，但已是大忌。

袁崇焕杀毛文龙，以阅兵为名，到双岛，列毛文龙十二大罪，用尚方宝剑将其斩杀。

杀毛文龙，袁崇焕没请示崇祯。

这件事，崇祯记下了。

毛文龙该不该杀？该。毛文龙虚报兵额、侵吞军饷、私通后金，不杀不足以肃军纪。可袁崇焕的错，不在杀，在"擅杀"。毛文龙再有问题，也该请示朝廷处置。袁崇焕用尚方宝剑杀毛文龙，是越权。毛文龙是总兵、挂将军印，袁崇焕是督师，督师杀总兵，于制不合。

程序错了，结果再对也是错。这是政治的残酷——程序比结果重要，至少在君主眼里如此。崇祯后来问罪袁崇焕，头一条就是"擅杀大帅"。

更要命的是杀毛文龙的代价。毛文龙的部将孔有德、耿仲明、尚可喜，后来先后降清，成了清朝的"三顺王"，带去了水师和火器，反过来打明朝。杀一个毛文龙，赔上了三员叛将、一支水师、一整套火器——这份代价，比袁崇焕想象的大得多。

袁崇焕杀了毛文龙，崇祯表面准了，心里头却记下了四个字：不受控制。

一个手握重兵、敢擅杀总兵的督师，皇帝怎么放心？袁崇焕不知道，他的死，从杀毛文龙那一天就开始倒计时了。

## 五、反间计戳中的疑心

崇祯二年（1629）十月，皇太极做了一件出乎所有人意料的事——他不打宁远，不打锦州，绕道蒙古，从喜峰口破关入塞。后金军一路南下，直逼北京。这就是"己巳之变"。

袁崇焕苦心经营的宁锦防线，被绕过去了。

袁崇焕闻讯，率关宁铁骑千里入援。九千骑兵，日夜兼程，比后金军早一步赶到北京城下。在广渠门、左安门，袁崇焕与后金军血战，击退皇太极。

一个守辽的统帅，千里回援，在北京城下击退敌人——这本是天大的功劳。可这份功劳，到了崇祯眼里，变成了疑窦。

为什么后金军能绕到北京？是不是袁崇焕放他们进来的？为什么袁崇焕到了北京城下，不主动出击，反而请求入城休整？——他要进城干什么？

皇太极看准了这一点，使出反间计。他故意让被俘的明朝太监，听到后金营中"袁崇焕与我有密约"的对话，又故意放这个太监逃回北京。太监把"密约"之事报告崇祯。

这一手，技术上并不复杂——蒋干盗书那一套，三国时周瑜就用过。可为什么崇祯会上当？

因为他早就疑袁崇焕了。

杀毛文龙、请求入城休整、五年复辽的承诺没兑现——这些事一件件攒在崇祯心里。反间计不过是最后一根稻草。崇祯不是被骗的，是"愿意被骗"——他早就想动袁崇焕了。反间计给了他一个名分，一个借口。

反间计不是"骗"，是"利用"。利用对方已有的疑心，把它放大、坐实。崇祯若不疑袁崇焕，反间计根本没用——他会想：袁崇焕若真通敌，何必千里入援？可崇祯已经疑了，反间计一来，所有的疑都"坐实"了。一个被疑的人，做什么都像通敌；一个被信的人，做什么都像尽忠。疑心这东西，一旦生了，就再也消不下去。

孙子说："上兵伐谋，其次伐交。"皇太极的反间计，是"伐交"的极致——不战而让敌人自毁长城。这一手，比攻城略地高明得多。

崇祯三年（1630）八月初，崇祯在平台召对。袁崇焕以为是要议军情，没想到崇祯劈头就问："尔擅杀大帅，以兵临城，复潜通虏贼，欲为城下之盟，何也？"

袁崇焕答不上来。崇祯当场下令逮捕，下锦衣卫狱。

半年后，崇祯三年八月十六日，袁崇焕被凌迟处死。兄弟妻子流徙三千里，籍没家产。

## 六、自毁长城

袁崇焕死了。死在他誓死保卫的京城，死在他誓死保卫的百姓嘴里。

整个朝廷，看着袁崇焕被凌迟，没人敢救。兵科给事中钱家修上疏为袁崇焕鸣冤，被崇祯斥责。大学士周延儒、温体仁，明知袁崇焕冤枉，不敢说话。温体仁甚至密疏请杀袁崇焕——他未必真信袁崇焕通敌，可他乐见东林党推举的人倒台。党争之下，袁崇焕的案子，已经不是"通敌不通敌"的问题，是"东林党的人该不该倒"的问题。

一个冤案，要杀一个人，需要的不是一个昏君，是一群沉默的同谋。崇祯是刀，温体仁是磨刀的，党争是磨刀石，百姓是看客。袁崇焕的死，是这群人一起完成的。

谷应泰在《明史纪事本末》里专列"袁崇焕之死"一卷。谷应泰的笔法，重在写"自毁长城"。他写到崇祯听信反间、逮袁崇焕、凌迟处死，笔锋冷峻。谷应泰特别感慨：袁崇焕一死，辽东再无人能守，明朝自毁长城。杀一个袁崇焕，毁的不只是一员统帅，是整个辽东防线，是明朝最后的希望。

这份"自毁长城"，历史反复印证。战国时赵国杀李牧，三个月后赵亡——李牧是赵国最后能挡秦军的将领，杀了他，赵国再无屏障。南朝宋杀檀道济，檀道济临死前怒喝："乃复坏汝万里之长城！"南宋杀岳飞，金人弹冠相庆。明朝杀袁崇焕，皇太极最大的障碍除掉了。

为什么自毁长城？因为多疑。长城是挡外敌的，可多疑的君主，觉得长城也是威胁——手握重兵的统帅，谁知道会不会反？崇祯杀袁崇焕，不是不想要长城，是觉得长城"不可信"。可长城拆了，外敌就进来了。

张廷玉等人在《明史·袁崇焕传》末尾，对袁崇焕有一段评语："崇焕智虽疏，差有胆略，初议五年复辽，不过漫语。然杀文龙以肃军令，亦云过矣。身膺督师，手握重兵，不度德量力，而以一身当天下之怨，是以速祸。"

张廷玉等人的见识在于，他们既指出袁崇焕的过失（擅杀、夸口），也承认他的胆略；既不为他翻案，也不一棍子打死。这种持平之论，比崇祯的"通虏谋叛"公允得多。

崇祯的判语，要等到一百五十二年后才被推翻。乾隆四十七年（1782），乾隆帝下旨，为袁崇焕平反。旨意大意是："袁崇焕督师蓟辽，虽与我朝为难，但尚能忠于所事，明季诸臣，多如崇焕之能忠于所事者，朕甚悯之。"

乾隆的评价，从对手的角度，承认了袁崇焕的忠勇。一个皇帝，能为敌国之臣平反，这份气度，崇祯是没有的。崇祯到死都信袁崇焕通敌，乾隆隔着两朝隔着一百五十二年，却看清楚了袁崇焕的忠。这份讽刺，比凌迟还狠——杀他的，是他保卫的朝廷；为他平反的，是他死战的敌国。

袁崇焕的悲剧，不只是他一个人的悲剧。他是崇祯朝"自毁长城"的第一块多米诺骨牌。他一死，孙承宗被罢，卢象升战死巨鹿，洪承畴降清松山——明朝再无能与后金抗衡的统帅。崇祯不是没人，是有而不能用，用了又不信，不信就毁掉。袁崇焕、孙传庭、卢象升、洪承畴，一个个能臣，都被崇祯折腾掉了。

一个勤政的皇帝，亲手把能臣一个个送上死路——这就是崇祯。他越勤政，越把所有事抓在手里；越抓在手里，越看谁都不放心；越不放心，越要杀；越杀，越没人敢任事；越没人敢任事，明朝亡得越快。

袁崇焕寸磔于市的那一天，崇祯以为自己除掉了一个叛徒。他不知道，他除掉的是明朝最后一根能撑住辽东的柱子。柱子一倒，屋顶就塌了。十四年后，李自成进北京，崇祯在煤山自缢。他临死还说"臣皆亡国之臣"——可这些"亡国之臣"，哪一个不是他亲手毁掉的？

袁崇焕死了，死得比谁都惨。可最惨的不是他的死，是他至死都不知道自己为什么必须死。他以为自己是死于反间计，其实他是死于崇祯的疑心；他以为自己是死于通敌的罪名，其实他是死于"不受控制"四个字。一个守辽的孤臣，死于他不懂得皇帝的心——皇帝不怕你打败仗，皇帝怕你打赢了还不听话。

袁崇焕守住了宁远，守住了锦州，守住了北京城。可他守不住自己的命。一个为明朝挡了十几年刀的人，最后被明朝的刀一刀一刀剐了。这份荒唐，比任何戏文都离奇，也比任何戏文都真。

---

## 结语

崇祯自毁长城，袁崇焕寸磔于市——一个勤政多疑的皇帝，亲手把最后一个能守辽东的人，凌迟在了他誓死保卫的京城，喂了百姓的嘴。

- 《明史·温体仁传》：温体仁密疏请杀袁崇焕、明末党争背景、周延儒温体仁不敢为袁崇焕说话。
- 《明史·食货志》：明末财政破产、辽饷加征、军饷拖欠。
- 《明通鉴》：宁远之战、宁锦之战、己巳之变、袁崇焕入援北京城下之战、平台召对被捕、钱家修上疏鸣冤被斥。
- 谷应泰《明史纪事本末·袁崇焕之死》：自毁长城、袁崇焕一死辽东再无人能守、杀一个袁崇焕毁的是整个辽东防线。
- 《清史稿·太宗本纪》：皇太极绕道蒙古破喜峰口入关、反间计让被俘太监听到袁崇焕密约放回报告崇祯。
- 《清太祖武皇帝实录》：努尔哈赤宁远受炮伤后退兵不久病死。
- 《清高宗实录》：乾隆四十七年为袁崇焕平反、"袁崇焕督师蓟辽虽与我朝为难但尚能忠于所事明季诸臣多如崇焕之能忠于所事者朕甚悯之"。
- 张岱《石匮书后集》：北京百姓争食袁崇焕肉、"刽子手割一块百姓争买一块开膛取肠百姓群起抢之以钱争买和烧酒生啮血流齿颊间"。
- 《孙子兵法·谋攻篇》："上兵伐谋其次伐交"、知己知彼百战不殆。
- 《史记·廉颇蔺相如列传》：赵国杀李牧三月后赵亡。
- 《宋书·檀道济传》：檀道济临死"乃复坏汝万里之长城"。
