# 明纪·洪武开国

洪武元年，正月初四，应天府。

这一天，朱元璋在南郊祭天即位。国号大明，建元洪武，以应天为南京。从皇觉寺的小和尚，到濠州投军的小卒，到应天称吴王，到大明开国皇帝——朱元璋走了十六年。即位诏书里有一句："定有天下之号曰大明，建元洪武。"简简单单一句话，可这句话背后，是一个新王朝要立规矩了。

立规矩，从一个要过饭的人最怕的事开始——怕。

## 一、怕的不是乱，是亡国

朱元璋怕的不是敌人，是"乱"。他见过元朝怎么乱的，见过元末怎么乱的。他要立的每一项规矩，都是怕乱。

先定都。朱元璋定都应天，不是一开始就定死的。应天是他起家的根本，可应天偏在江南，对北方控制力弱。朱元璋心里清楚：自古中原王朝定都，多在长安、洛阳、汴梁，那是天下之中。应天据长江，财赋便利，可偏安一隅的影子，总也抹不掉。

所以洪武元年，朱元璋一边称帝，一边亲自去考察汴梁。汴梁是北宋旧都，地处中原，漕运方便。朱元璋看了又看，最后还是摇头——汴梁无险可守，一马平川，北方铁骑一来就到。后来又议过洛阳、长安，甚至北边的元旧都，议来议去，到底还是定在应天。洪武十一年，改称京师。

这一议，议了十来年。朱元璋的纠结，不是没道理。定都是大事，定在哪里，关系到王朝的命脉。可他最后选应天，有个实在的理由：应天是他起家的根本，根基在此，人心在此，财赋在此。长安、洛阳虽是形胜之地，可久经战乱，残破不堪；汴梁无险可守。朱元璋是务实的人，他要的不是虚名上的"天下之中"，是实实在在能撑住王朝的根。

一个要过饭的人，连定都都讲实在、不讲虚名。这底色，从他少年时就定了。

都定了，下来搭班子。搭班子，朱元璋心里贯穿一个字：防。

## 二、把权收上来

中央这一块，朱元璋沿了元朝的旧制——中书省、六部。中书省是宰相办公的地方，统领六部。朱元璋任命李善长为左丞相，徐达为右丞相。左为尊，李善长是大明开国第一任真宰相。李善长是定远人，少读书，有智计，习法家言，明习故事——法家出身的人，长于制度，严于法度。朱元璋立国之初，制度多出李善长之手。

可朱元璋心里贯穿一个字：防。防宰相专权，防武将造反，防地方割据，防外戚干政，防宦官弄权。他把历史上能想到的隐患，一个个都用制度堵上。

地方这一块，朱元璋动了刀子。元朝地方设行中书省，简称行省。行省权力大，军政财一把抓。元末天下大乱，行省长官各据一方，称王称帝，就是这个制度的弊端。朱元璋自己就是从行省割据里杀出来的，他太知道行省权力大的害处。所以洪武九年，朱元璋废掉行省，改设三司：承宣布政使司，管民政财政，长官叫布政使；提刑按察使司，管刑名按劾，长官叫按察使；都指挥使司，管军政，长官叫都指挥使。三司并立，互不统属，各直接对中央负责。这一刀切下去，地方再没有能跟中央抗衡的势力。三权分立，谁也成不了割据的根。

军事这一块，朱元璋设五军都督府——中军、左军、右军、前军、后军，分领天下卫所。可五军都督府只管军政，不管调兵；调兵权在兵部。将不专兵，兵不私将——这是朱元璋防武将造反的法子。

这一套分权，是从元末的乱里学来的。朱元璋见过元朝权臣专擅，见过元末武将割据，见过陈友谅杀主自立，见过张士诚宽柔致败。朱元璋把这些教训，一条条刻进制度里。可朱元璋防得太狠，也防出了毛病。宰相后来干脆废了，皇帝一个人忙不过来，再后来的皇帝干脆不上朝，大权落到宦官手里；武将分权，将不专兵，打起仗来指挥不灵，后来倭寇、女真，明军越打越弱。朱元璋防住了他这一代的隐患，却防不出后代的隐患——他防得越严，制度越僵，后人越没腾挪的余地。

军事上还有一招，更绝。

## 三、百万兵不费一粒米

朱元璋立了卫所制。"自京师达于郡县，皆立卫所。"卫所是明朝的基层军事单位，一卫大约五千六百人，下辖五个千户所。军户世袭，父死子继，平时屯田自养，战时出征。朱元璋有一句名言："吾养兵百万，不费百姓一粒米。"——这话说的就是卫所屯田。

这制度不是朱元璋凭空发明的，它的老祖宗是北朝以来的府兵制。府兵制是西魏宇文泰立的，兵农合一，寓兵于农。唐朝前期用的就是府兵制，后来府兵制崩了，改成募兵制。朱元璋的卫所制，跟府兵制一脉相承——军户世袭，屯田自养，兵农合一。朱元璋学的是府兵制的好处：兵不费百姓一粒米。

可府兵制为什么崩？因为军户地位下降，逃亡日多。卫所制后来也崩了——军户逃亡，卫所空虚，到明朝中后期，卫所名存实亡，只能靠募兵。戚继光的戚家军、李成梁的辽东铁骑，都是募兵，不是卫所。府兵制撑了百来年，卫所制也撑了百来年，根子都在军户地位和土地兼并。

《明史·兵志》对卫所制有一段评价："立法之意，本为深远"，可后来败坏，"非立法之弊，乃行法之弊"——立法的本意深远，可后来败坏，不是立法的毛病，是执行的毛病。这段评价，点出制度兴衰的一个关键——制度再好，执行一坏，全盘皆输。卫所制立法时是好制度，可军户地位下降、土地被兼并、军官贪污，卫所就空了。

兵不费百姓一粒米，可百姓还得被管住。

## 四、重典与黄册

律法这一块，朱元璋从吴元年（1367）就开始编《大明律》。这部律法编了三十多年，到洪武三十年才颁定。《大明律》之外，朱元璋还亲自编了《大诰》三编——拿真实案例，配以严刑峻法，教化吏民。贪污的、害民的、渎职的，一一列出来，刑罚之酷，让人胆寒。朱元璋还下令，家家户户都得有一本《大诰》，犯法的人若家里有《大诰》，罪减一等。

户籍赋役这一块，朱元璋立了黄册和鱼鳞图册。黄册记户口，十年一造；鱼鳞图册记土地，按田亩图形登记。里甲制——一百一十户为一里，里长、甲首轮流当，催办赋役。这套制度，把百姓牢牢钉在土地上，跑不掉，也躲不开。

礼制这一块，朱元璋下了一道诏："复衣冠如唐制。"——禁止胡服、胡语、胡姓，恢复汉家衣冠。元朝统治近百年，汉人受蒙古风俗影响不少，朱元璋要用衣冠礼乐，把汉家的根找回来。学校科举这一块，洪武三年开科取士。可朱元璋用了几年，觉得科举取来的人只会写文章，不会办事，洪武六年停了科举，改用荐举。到洪武十七年，才又恢复科举，定为三年一考，从此成定制。

这套制度，对官吏猛，对百姓也猛。重典靠的是恐惧，恐惧靠的是执法的人。朱元璋在世时，吏治确实清明——没人敢贪。可朱元璋一走，吏治就坏了。重典能治一时，治不了一世。为什么？因为重典靠恐惧维持，恐惧一散，秩序就塌。韩非子早说过："法之不行，自上犯之。"——法之所以行不通，是从上头开始破坏的。法靠人执，人靠上头管。上头严，法就行；上头松，法就废。

可朱元璋立这套制度，底子是要饭皇帝的细与狭。一个要过饭的人，最知道贪官可恨、灾荒可怕、百姓苦，所以他轻徭薄赋、兴修水利、设预备仓、荒年赈济——这些政策，都是一个要过饭的皇帝才能想得这么细。孟森先生说他"以宽猛相济为体，以重典绳下为用"——对百姓宽，对官吏猛。可他也有要饭皇帝的狭：重农抑商，把人钉在土地上，不许流动；他设计的社会，是个大村庄，每个人有固定户籍、固定职业、固定住处。这是农民的想象力——他眼里的好日子，就是人人有地种、有饭吃，至于工商业、海外贸易、思想自由，他理解不了，也不想要。一个人的长处和短处，往往同源。朱元璋的务实和狭隘，都来自那三年游方。

制度立了，还有最后一桩——自家人的事。

## 五、自家人守边

朱元璋有二十六个儿子。洪武三年起，朱元璋开始封王——长子朱标立为皇太子，其余诸子封为藩王，镇守边塞。燕王朱棣守北平，晋王朱㭎守太原，秦王朱樉守西安，宁王朱权守大宁。这些塞王，手握重兵，镇守北边，防的是蒙古卷土重来。

朱元璋分封诸王，是想用自家人守自家天下。学的是西周和汉初。西周分封，巩固了周朝；汉初分封，却闹出七国之乱；晋朝司马炎大封宗室，结果八王之乱，西晋亡国。朱元璋忘了教训——自家人封了王，有了兵，迟早要争皇位。汉有七国，晋有八王，明有靖难，都是一回事。

这是家天下的逻辑：用人防人，用亲防疏。用文官防武将，用宦官防文官，用藩王防外姓，最后用自家人防所有人。可家天下的逻辑，有一个解不开的矛盾——自家人也是人，是人就有私心。藩王是自家人，可藩王也想当皇帝。用自家人防外人，自家人比外人更危险——因为自家人离皇位更近。贾谊在《治安策》里论汉初分封，说"疏者必危，亲者必乱"——关系疏远的诸侯必然危险，关系亲近的诸侯必然作乱。这话放在明朝，一样准。燕王朱棣是建文帝的亲叔叔，亲得不能再亲，可亲叔叔抢了亲侄子的皇位。

顾炎武在《日知录》里论历代分封，以为封建之制到后世已行不通——分封在古代行得通，是因为古代地广人稀、交通不便，中央管不到边远地方；可到了后世，中央集权已经成熟，分封只会制造割据和内乱。顾炎武这层见识，放在明朝分封上看，正中要害。

朱元璋在世能压住藩王，朱元璋一走，藩王就压不住了。建文帝削藩，燕王朱棣反了，这就是靖难之役的根。朱元璋为太子除刺，除的是功臣；可他没想到，最大的"刺"，是他自己封的藩王。除刺这件事，得从马皇后和太子说起。

## 六、为太子除刺

马皇后是朱元璋的结发妻子，郭子兴的养女。朱元璋起兵时，马皇后跟着他出生入死。朱元璋脾气暴，动不动杀人，马皇后常常劝阻。她劝朱元璋"定天下以不杀人为本"。有一回朱元璋发怒要杀人，马皇后劝不住，就把那人的妻小藏起来，等朱元璋气消了再放出来。马皇后最让人感念的一件事，是洪武十五年她病重时，不肯服药。她说：我吃药要是没用，皇上一定会怪罪太医，太医是要掉脑袋的。我宁可自己死，也不能连累太医。这份心，是真正的仁。

马皇后洪武十五年去世，朱元璋痛哭，从此不再立皇后。马皇后在世时，是朱元璋的"压舱石"——朱元璋要杀人，她劝；朱元璋要发火，她抚；朱元璋身边的功臣，多亏她保下了不少。马皇后一走，朱元璋身边再没人能劝得住他，杀心更重。马皇后的死，是朱元璋晚年性格变化的一个转折点。一个能劝住他的人没了，他砌的那道墙，就再也没人能从里头松动一块砖。

太子朱标，是朱元璋的嫡长子，洪武元年立为皇太子。朱元璋对朱标寄予厚望，请宋濂等大儒教他读书，让李善长、徐达等功臣辅佐他理政。朱标性情宽厚，跟朱元璋的严猛正好相反。朱元璋杀人，朱标常常劝阻。有一回朱元璋要大开杀戒，朱标进谏。朱元璋不说话，第二天把一根长满刺的荆杖扔在地上，让朱标捡起来。朱标面有难色。朱元璋说：我杀这些人，是替你把刺除掉。你拿着就顺手了。

这根刺杖的故事，道出了朱元璋杀功臣的心思——为太子除刺。李善长也死在这套逻辑里。李善长拜左丞相，封韩国公，岁禄四千石——这是开国功臣里头一份。可李善长到底没逃过朱元璋的猜忌。洪武二十三年，胡惟庸案发，李善长被牵连，全家被杀，年七十七。朱元璋杀李善长，不是李善长犯了多大的罪，是朱元璋要为太子扫清障碍。开国功臣威望高、势力大，朱元璋在世能压住，朱元璋一走，太子压不住。所以朱元璋趁自己还在，把功臣一个个收拾了。这份心思，残酷，可在朱元璋看来，是为了家天下的安稳。

可朱标命薄，洪武二十五年病逝，年仅三十七。朱标一死，朱元璋立朱标之子朱允炆为皇太孙。可朱允炆年轻威望浅，压不住那些叔叔——尤其是燕王朱棣。朱元璋为朱标除刺，除的是功臣；可他没想到，最大的"刺"，是他自己封的藩王。墙砌好了，刺除完了，太子却没了——一个为儿子除了一辈子刺的父亲，最后把皇位留给了一个压不住叔叔的孙子。

制度是教训的结晶，可制度也会成为新的教训。朱元璋立的每一项制度，都是从历史的教训里来的。废行省，是元末割据的教训；分五军都督府，是唐末藩镇的教训；废宰相，是历代权臣的教训；立卫所，是府兵制崩坏的教训。朱元璋把能想到的教训，都刻进了制度里。可他没想到的是，他立的制度，后来也成了教训。废了宰相，皇帝忙不过来，宦官乘机弄权——这是明朝中后期最大的祸患。分封诸王，藩王造反——靖难之役就来了。卫所制崩了，募兵制兴起，军费暴涨——明朝后期的财政危机，根子也在这里。制度防住了旧的隐患，却制造了新的隐患。制度是死的，人是活的，靠"防"永远防不全。

## 结语

朱元璋用一辈子的心计，给朱家砌了一道密不透风的墙。墙防宰相、防武将、防地方、防外戚、防宦官，把秦汉以来的中央集权推到了极致。可墙挡得住外人，挡不住墙里的人心——他防了一辈子的隐患几乎都防住了，没防住的那些，把明朝送上了末路。马皇后一走，再没人劝得住他；太子朱标一死，他除的刺白除了，最大的刺是他自己封的藩王。一个要过饭的人，砌了一辈子墙，最后才发现：墙里墙外，都是人心——而人心，是任何制度都防不住、也除不掉的那根刺。

- 《明史·李善长传》：李善长定远人、少读书有智计习法家言明习故事、拜左丞相封韩国公岁禄四千石、洪武二十三年因胡惟庸案赐死年七十七。
- 《明史·后妃传》：马皇后劝谏"定天下以不杀人为本"、病重不肯服药护太医、洪武十五年崩、朱元璋不再立后。
- 《明史·兴宗孝康皇帝传》：朱标嫡长子洪武元年立皇太子、洪武二十五年薨年三十七、朱允炆立为皇太孙。
- 《明史·诸王传》：分封诸王、塞王守边、燕王朱棣守北平、晋王守太原、秦王守西安、宁王守大宁。
- 《名山藏》：朱元璋掷刺杖喻杀功臣为太子除刺。
- 贾谊《治安策》："疏者必危，亲者必乱。"
- 《韩非子·有度》："法之不行，自上犯之。"
- 顾炎武《日知录》：论历代分封之不可行于后世。
- 孟森《明史讲义》：宽猛相济为体、重典绳下为用。
- 商传《明太祖朱元璋》：马皇后为朱元璋晚年性格转折点、家天下制度设计。
