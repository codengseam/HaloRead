# 三顾茅庐

## 一、讲事情

建安十二年，公元207年，隆冬。

刘备四十七岁了，窝在新野这个小地方，已经六年。

六年前，建安六年，刘备在汝南被曹操打败，走投无路，南奔荆州投刘表。刘表收留了他，给他增兵，让他屯驻新野。新野是荆州的北大门，刘表的算盘是让刘备替他挡曹操。

刘备感谢刘表收留，可心里憋屈。他这辈子，跟曹操打了多少年，败多胜少。先后投过公孙瓒、陶谦、吕布、曹操、袁绍，每次都是寄人篱下，每次都被赶得鸡飞狗跳。好不容易有了徐州，又丢了。现在四十多岁的人了，还在给人看大门。

刘表对刘备，表面客气，心里防着。刘表这个人，"外宽内忌"，跟袁绍一个毛病。他给刘备兵，让他守新野，可不让刘备发展势力。刘备在新野六年，无所事事，有一次跟刘表喝酒，中途去上厕所，回来一看，自己大腿上长了赘肉，忍不住掉眼泪。刘表问怎么了，刘备说:"日月蹉跎，老将至矣，而功业不建，是以悲耳。"

——日子一天天过去，人老了，可功业还没建起来，所以悲伤。

这就是"髀肉之叹"的典故。一个快五十岁的人，看着自己大腿上长出的肥肉，知道自己虚度了光阴。这种滋味，不好受。

可就在这年，转机来了。

刘备在新野，招揽了一些人才。其中有个叫徐庶的，颍川人，是个有本事的谋士。刘备很器重他。徐庶跟刘备处了一阵子，看出刘备是个能成事的人，可也看出刘备缺一样东西:一个能定大计的战略家。

徐庶对刘备说:"诸葛孔明者，卧龙也，将军岂愿见之乎?"

——有个人叫诸葛孔明，号称卧龙，将军愿不愿意见见?

刘备说:"君与俱来。"——你带他一起来吧。

徐庶摇头:"此人可就见，不可屈致也。将军宜枉驾顾之。"

——这个人，只能您去见他，不能叫他来见您。将军得亲自去请。

刘备愣了一下。他这辈子，招揽人才不少，可从来都是别人来投他，还没他亲自去请的。可徐庶的话，他信。徐庶是个实在人，不会夸大其词。能让徐庶这么推崇的人，一定不简单。

于是刘备决定:去。

《三国志·诸葛亮传》只用了八个字记载这件事:"由是先主遂诣亮，凡三往，乃见。"

——于是刘备去拜访诸葛亮，去了三次，才见到。

八个字，干巴巴的，可这背后的分量，重得很。

刘备当时是什么身份?左将军、宜城亭侯、豫州牧、皇叔(虽然"皇叔"这个称呼主要见于演义，但刘备确实是汉室宗亲)。一个有这堆头衔的人，去请一个二十七岁的年轻人，去了三次才见到。这要放在那个讲究身份的年代，是极其罕见的。

第一次去，没见到。诸葛亮不在家。刘备留了话，走了。

第二次去，又没见到。还是不在。刘备又留了话，走了。

第三次，见到了。

演义里把这三顾写得极其热闹:第一次是大雪天，刘备顶风冒雪去，诸葛亮外出;第二次是春暖花开，刘备再去，诸葛亮又外出，只见到他弟弟诸葛均，刘备留书;第三次，刘备斋戒沐浴，选了个吉日再去，诸葛亮在家，可正在睡觉，刘备站在台阶下等，等了半天，诸葛亮才醒。这才有隆中对策。

这些细节，史书没记载。《三国志》只说"凡三往，乃见"。演义的加工，是为了突出刘备的诚意和诸葛亮的身价。风雪、等待、睡觉，这些桥段都是为了一个目的:让读者觉得，这个诸葛亮，值得刘备这么请。

可史实上，三顾茅庐这件事本身，就是真的。不光《三国志》有记载，诸葛亮自己后来在《出师表》里也写了:"臣本布衣，躬耕于南阳，苟全性命于乱世，不求闻达于诸侯。先帝不以臣卑鄙，猥自枉屈，三顾臣于草庐之中，咨臣以当世之事，由是感激，遂许先帝以驱驰。"

——我本来是个平民，在南阳种地，只想在乱世里保全性命，不想出名。先帝不嫌我地位低微，委屈自己，三次到草庐里来看我，问我天下大事。我因此感激，答应为先帝效力。

这是诸葛亮自己说的。三顾茅庐，铁证如山。

那么，刘备第三次去，见到了诸葛亮，两人谈了什么?

这就是著名的"隆中对策"。诸葛亮给刘备分析天下大势，提出了一套完整的战略规划。这套规划的核心是:跨有荆、益，外结孙权，内修政理，待天下有变，两路北伐，复兴汉室。

刘备听完，如醍醐灌顶。他这辈子，从来没听过这么清楚的战略。以前他打仗，靠的是勇气和义气，没有长远规划，所以败多胜少。现在，诸葛亮给他画了一张地图，告诉他该往哪儿走，该怎么走。

刘备跟关羽、张飞说:"孤之有孔明，犹鱼之有水也。"

——我有了孔明，就像鱼有了水一样。

关羽、张飞不太高兴。刘备跟他们解释:"孤之有孔明，犹鱼之有水也。愿诸君勿复言。"——你们别再说了。

从此，诸葛亮成了刘备的首席谋士，开始了长达二十七年的君臣合作。

## 二、讲人物

三顾茅庐这事儿，主角是刘备和诸葛亮，可这两个人，反差极大。

先说刘备。

刘备这个人，三国里最被低估的一个。演义里把他写成个爱哭的软柿子，动不动就哭，好像除了哭什么都不会。这是天大的误解。

刘备的本事，在"韧性"二字。他这辈子，败了多少次?数不清。先后被吕布偷袭，被曹操追杀，老婆孩子丢了四五回。可他每次败了，都能爬起来。这种韧性，三国里没几个人有。

刘备的另一个本事，是"得人心"。他走到哪儿，老百姓都愿意跟着他。曹操南下，刘备从新野撤退，十几万老百姓跟着他走，一天才走十几里。有人劝他扔了百姓快跑，刘备说:"夫济大事必以人为本，今人归吾，吾何忍弃去?"

——做大事必须以人为本，现在百姓归附我，我怎么忍心抛弃他们?

这话，不管是真心还是作秀，至少他这么说了，也这么做了。在那个视百姓如草芥的年代，这已经难能可贵。

刘备三顾茅庐，本身就是一个"得人心"的范例。他四十七岁，诸葛亮二十七岁;他是左将军、豫州牧，诸葛亮是一介布衣。可他愿意放下身段，去请一个比自己小二十岁的年轻人。这种诚意，不是装出来的。

为什么刘备这么渴求人才?因为他太清楚自己的短板了。他有关羽、张飞、赵云这样的猛将，可没有一流的谋士。他打仗靠勇气，不靠智谋，所以败多胜少。他需要一个人，能帮他看清天下大势，制定长远战略。这个人，就是诸葛亮。

再说诸葛亮。

诸葛亮，字孔明，琅琊阳都人。他出身不算太差，父亲诸葛珪做过泰山郡丞。可诸葛亮少年丧父，跟着叔父诸葛玄生活。诸葛玄后来去豫章当太守，又丢了官，带着诸葛亮兄弟投奔荆州刘表。诸葛玄死后，诸葛亮兄弟在南阳隆中安家，躬耕陇亩。

诸葛亮这个人，年轻时就有大志。他常以管仲、乐毅自比——管仲是齐桓公的相国，帮齐国称霸;乐毅是燕国的上将，连下齐国七十余城。诸葛亮以这两人自比，说明他的志向不是当个谋士，而是当辅佐霸业的相国。

可当时的人，不太认可他。只有崔州平、徐庶这几个朋友，跟他交好，相信他的才能。

诸葛亮在隆中，不光种地，还读书、观察天下大势。他跟荆州的名士司马徽、庞德公等人有交往。司马徽号称"水镜先生"，庞德公称诸葛亮为"卧龙"。这些名士的推崇，为诸葛亮积累了名声。

诸葛亮选择刘备，是有讲究的。当时天下，曹操已拥百万之众，挟天子以令诸侯，手下谋士如云，诸葛亮去了未必能出头;孙权据有江东，已历三世，张昭、周瑜、鲁肃都是重臣，诸葛亮去了也只能排后面。只有刘备，有雄心无谋士，有猛将无战略，正是诸葛亮施展抱负的最佳平台。

所以，三顾茅庐，不只是刘备请诸葛亮，也是诸葛亮选刘备。这是一次双向选择。

诸葛亮等刘备，等了多久?他在隆中躬耕了十年。这十年，他没闲着，一直在观察、在思考、在准备。等刘备来的时候，他已经把天下大势想透了，所以一见面就能拿出"隆中对"这样完整的战略。

这就是诸葛亮的过人之处:不鸣则已，一鸣惊人。十年磨一剑，出鞘即锋芒。

再说徐庶。

徐庶这个人，在三顾茅庐里是个关键角色。没有他推荐，刘备未必知道诸葛亮。徐庶推荐诸葛亮，有几点值得说。

第一，徐庶推荐诸葛亮，是"让贤"。徐庶当时是刘备的首席谋士，他推荐诸葛亮，等于把自己的位置让出去。这需要胸怀。后来徐庶因为母亲被曹操掳走，不得不离开刘备，临走前又推荐了诸葛亮一次，说"诸葛孔明者，卧龙也"。这等于告诉刘备:我走了，但有个比我强的人，你一定要请他。

第二，徐庶说"此人可就见，不可屈致也"。这话有两层意思:一是诸葛亮这个人有脾气，不能随便叫来;二是诸葛亮值这个价，你得亲自去请。徐庶了解诸葛亮，知道诸葛亮不是那种主动求官的人，得老板亲自去请。这等于给刘备打了预防针:别嫌麻烦，这趟值。

演义里把徐庶的离开写得极其悲壮——徐庶走马荐诸葛，临别时向刘备推荐诸葛亮。史实上，徐庶推荐诸葛亮是在他离开之前还是之后，记载不太明确。但徐庶推荐诸葛亮这件事，是史实。

## 三、讲背景

三顾茅庐这件事，放在建安十二年的背景下，才能看出它的分量。

先说刘备的处境。

建安十二年，曹操已经基本统一北方。前一年，曹操北征乌丸，平定了袁氏残余。这年八月，曹操从柳城班师，北方再无后顾之忧。下一步，曹操的目标一定是南方——荆州、江东。

刘备屯在新野，正对着曹操的南下路线。一旦曹操南下，刘备首当其冲。可刘备手里只有几千兵，挡不住曹操。他需要战略，需要人才，需要出路。

刘表呢?刘表坐拥荆州，带甲十余万，可他"外宽内忌，好谋无决"，跟袁绍一个毛病。刘表年纪大了，儿子又不成器(刘琦、刘琮争位)，荆州内部也不稳。刘备看出荆州迟早要出事，可他不知道该怎么办。

这就是刘备三顾茅庐的背景:前有强敌(曹操)，后有猜忌(刘表)，自己兵少将寡，没有出路。在这种绝境下，徐庶推荐了诸葛亮，刘备像抓住了救命稻草。

再说荆州的人才环境。

荆州在刘表治下，相对安定，吸引了不少北方流亡的士人。诸葛亮、徐庶、崔州平、孟建、石韬，都是从北方来的。刘表本人也喜欢文学，养了一批文人。可刘表不会用人——这些人才在他眼皮底下，他一个都没重用。

诸葛亮在隆中，跟这些士人有交往。他们经常讨论天下大势，形成了一个人脉网络。司马徽、庞德公这些荆州名士，对诸葛亮推崇备至。这个网络，为诸葛亮积累了名声，也为刘备找到诸葛亮提供了线索。

还有一层背景:汉末的"士人风气"。

东汉末年，士人讲究"清高"。有本事的人，往往不愿意主动求官，要等老板来请。这叫"养望"——积累名声，等明主来请。诸葛亮在隆中躬耕，常自比管仲乐毅，这就是在"养望"。

这种风气，跟当时的选官制度有关。察举制下，名声是当官的资本。名声越大，被举荐的机会越多。所以士人们刻意制造名声，要么隐居不仕(显得清高)，要么结交名士(互相吹捧)。诸葛亮走的就是这条路——隐居隆中，结交名士，等明主来请。

可诸葛亮跟一般的"养望"之士不同。他不是光等，他在准备。十年的隆中生活，他把天下大势研究透了，把战略想清楚了。所以刘备一来，他就能拿出"隆中对"。这不是临时应付，是十年磨一剑的结果。

最后说年龄差。

刘备四十七岁，诸葛亮二十七岁。这个年龄差，在当时是两代人。刘备愿意放下身段，三次去请一个比自己小二十岁的年轻人，这需要多大的诚意和远见?

在那个讲究资历、讲究辈分的年代，这几乎是不可想象的。袁绍用人，讲究门第;刘表用人，讲究资历。只有刘备，愿意去请一个二十七岁的布衣。这份诚意，是诸葛亮"由是感激，遂许先帝以驱驰"的根本原因。

## 四、讲道理

三顾茅庐这件事，名家有不少点评。

**陈寿的"评曰"**

陈寿在《三国志·诸葛亮传》末尾评道:"诸葛亮之为相国也，抚百姓，示仪轨，约官职，从权制，开诚心，布公道;尽忠益时者虽仇必赏，犯法怠慢者虽亲必罚，服罪输情者虽重必释，游辞巧饰者虽轻必戮;善无微而不赏，恶无纤而不贬;庶务精练，物理其本，循名责实，虚伪不齿;终于邦域之内，咸畏而爱之，刑政虽峻而无怨者，以其用心平而劝戒明也。可谓识治之良才，管、萧之亚匹矣。"

陈寿这段话，评价的是诸葛亮为相的表现，但根子在隆中对策。诸葛亮能成为"识治之良才，管、萧之亚匹"(管仲、萧何第二)，跟他在隆中十年的准备分不开。陈寿把诸葛亮跟管仲、萧何并列，评价极高——这正是诸葛亮年轻时自比管仲乐毅的应验。

**关于刘备的识人**

裴松之在注《三国志》时，对刘备三顾茅庐一事没有直接评论，但他引《魏略》记载了一个细节:刘备初见诸葛亮时，诸葛亮"陈所以三分之计"，刘备"曰'善'"。裴松之对此事的考证是，《魏略》的记载跟《三国志》有出入，但三顾茅庐本身是可信的。

**关于"鱼水之交"**

刘备说"孤之有孔明，犹鱼之有水也"，这句话后来成了君臣关系的典范。后世评价刘备与诸葛亮的关系，常用"鱼水"二字。唐代杜甫写诗说"三顾频烦天下计，两朝开济老臣心"，就是赞颂这段君臣际遇。

杜甫的《蜀相》:"丞相祠堂何处寻，锦官城外柏森森。映阶碧草自春色，隔叶黄鹂空好音。三顾频烦天下计，两朝开济老臣心。出师未捷身先死，长使英雄泪满襟。"

——"三顾频烦天下计"，说的就是三顾茅庐和隆中对策。杜甫用"频烦"二字，写出了刘备三次拜访的诚意。这是后世文人对三顾茅庐最经典的吟咏。

## 五、问道悟道

三顾茅庐这件事，能悟出几条道理。

**第一条:诚意，是打动人才的唯一通行证。**

刘备请诸葛亮，去了三次。第一次没见到，第二次又没见到，第三次才见到。换个人，可能去一次没见到就算了——你诸葛亮架子大，我不请了。可刘备去了三次。

为什么?因为他知道诸葛亮值这个价。一个能帮他看清天下大势、制定长远战略的人，别说去三次，去十次也值。

这个道理，古今通用。萧何月下追韩信，刘邦筑坛拜将，都是诚意。真正的人才，不会因为你开高工资就来，他要的是被尊重、被信任、被需要的感觉。刘备三顾茅庐，给的就是这种感觉。所以诸葛亮"由是感激，遂许先帝以驱驰"——一辈子为刘备卖命。

现代企业招人，也是一样。真正的人才，不是你用钱能买来的。你得让他觉得，你是真心需要他，真心尊重他。这份诚意，比任何待遇都管用。

**第二条:准备，是抓住机会的前提。**

诸葛亮在隆中十年，看似无所事事，其实在准备。他观察天下大势，研究各方势力，思考战略规划。所以刘备一来，他就能拿出"隆中对"。

机会只留给有准备的人。如果诸葛亮在隆中光种地不读书，刘备来了他也说不出个所以然。正因为他在隆中十年磨一剑，所以一出手就惊艳。

这个道理，古今通用。姜子牙在渭水钓鱼，等了多久?八十岁才遇到周文王。可他等的那些年，没闲着，在研究兵法、研究治国。所以文王一问，他就能拿出整套方略。机会来了，你准备好了，才能抓住;没准备，机会来了也抓不住。

**第三条:双向选择，才是最好的合作。**

三顾茅庐，表面看是刘备选诸葛亮，其实也是诸葛亮选刘备。诸葛亮在隆中十年，不是没机会出山。曹操、孙权都在招人，可他没去。为什么?因为他要选一个能让他施展抱负的平台。

曹操手下谋士如云，诸葛亮去了排不上号;孙权手下有张昭、周瑜，诸葛亮去了也只能当配角。只有刘备，有雄心无谋士，有猛将无战略，正是诸葛亮需要的平台。

所以，三顾茅庐是双向选择:刘备选了诸葛亮，诸葛亮也选了刘备。这种双向选择，才是最好的合作。强扭的瓜不甜，单方面的选择，往往走不远。

**第四条:年龄和资历，不是衡量人才的唯一标准。**

刘备四十七岁，诸葛亮二十七岁。刘备愿意听一个比自己小二十岁的年轻人的意见，这需要眼光。

人最容易犯的错，就是"以资历论英雄"。觉得年纪大的、资历深的，就一定厉害;年纪轻的、资历浅的，就一定不行。可真正的人才，往往不在资历里。甘罗十二岁为丞相，霍去病十七岁封冠军侯，诸葛亮二十七岁定三分天下。年龄从来不是衡量人才的标准，能力和见识才是。

## 六、演义与史实

三顾茅庐是《三国演义》的重头戏，演义加了不少虚构细节，这里对照一下。

**三顾茅庐本身**:史实。《三国志·诸葛亮传》明确记载"凡三往，乃见"，诸葛亮《出师表》也自述"三顾臣于草庐之中"。

**风雪访隆中**:演义虚构。史书没记载第一次拜访的天气。演义写刘备顶风冒雪去，是为了突出诚意。

**诸葛亮睡觉，刘备立等**:演义虚构。史书没记载这个细节。演义写诸葛亮午睡、刘备站在台阶下等，是为了突出诸葛亮的身价和刘备的恭敬。

**童子、诸葛均、黄承彦等人物**:演义加工。史书没记载刘备拜访时遇到这些人的细节。演义加这些人物，是为了增加戏剧性和层次感。

**隆中对策**:史实。《三国志·诸葛亮传》详细记载了诸葛亮的分析。演义基本照搬，但加了诸葛亮"未出茅庐，已知三分天下"的评语，这是文学加工。

**徐庶走马荐诸葛**:部分史实，部分演义。徐庶推荐诸葛亮是史实，但"走马荐诸葛"(临走时骑马回来推荐)的细节是演义加工。史实上，徐庶推荐诸葛亮在前，离开刘备在后，时间线跟演义略有不同。

**刘备"鱼水"之喻**:史实。《三国志·诸葛亮传》记载刘备对关羽、张飞说"孤之有孔明，犹鱼之有水也"。

## 结语

三顾茅庐，请的不只是一个人，是一张通往天下的地图。
