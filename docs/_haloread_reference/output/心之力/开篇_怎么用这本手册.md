# 怎么用这本手册

## 一、入戏

老陈收到这本手册的时候，正处在创业第三年的低谷。他翻开第一章，看完心力的四项公式，觉得有道理；翻到第二章，看完意义感那段，又觉得有道理；翻到第三章，看完 Goggins 那段，热血上头，决定凌晨四点起床、冷水冲淋、戒糖戒碳水。三天后他崩了——凌晨四点起，下午开会打瞌睡，被合伙人当场点名；冷水冲淋完没擦干，感冒发烧两天；戒糖到第四天，半夜偷吃了一整盒巧克力，吃完又自责。

他给我发消息说："你这本手册是不是也跟其他自律书一样，看了爽三天，第四天崩回去？"

我没急着回他。我让他先把手册合上，做一件事——拿出一张纸，写下他这三天到底在做什么。他写完后自己笑了：三天里，他压根没看第四章"避坑"那段，没看"践行"里"选一件自愿难受的小事，连做七天"那句，更没做能量审计。他把这本手册当励志书读了，没当训练手册用。

这本手册的用法，跟励志书不一样。这一章专门讲怎么用。

## 二、破题

先把话说清楚：这本手册不是给你读完就放下的，是给你边读边做、做完回看的。

读完不做的手册，跟没读一样；做了一段不回看的训练，跟瞎练一样。心力这个东西，跟肌肉一个道理——你读再多健身理论，不去举铁，肌肉不会长；你举了铁不记录重量不复盘动作，进步也是随机波动。所以这本手册的用法，关键在三个动作：读、做、复盘。三个动作循环起来，手册才在你手里活起来。

我把这本手册拆成五个板块：源头、蓄积、修复、中国心法、践行。每个板块不是孤立的章节，是循环的环节。源头讲心力从哪里来，蓄积讲怎么把它练厚，修复讲心力漏了怎么补，中国心法讲王阳明、曾国藩、老子、毛泽东各自怎么养这份力，践行讲日课与逆境重启怎么落地。五个板块合起来，是一个"知—练—修—养—行"的闭环。

源头四章是这本手册的引擎。意义感、自愿承受、神经适应性、浩然之气——这四项是相乘关系，任意一项归零，整张公式就归零。读懂源头，你才知道自己心力漏在哪一项：是引擎熄火（意义感没了），还是燃料断供（自愿承受没了），还是回路磨损（神经适应性没了），还是底盘松动（浩然之气散了）。诊断先于训练，训练先于修复。

蓄积章是这本手册的健身房。心力可以被磨损，就可以被重塑——这一章把意志力、自我效能感、心流、多巴胺管理、体能管理一项一项拆开讲，告诉你怎么把心力像练肌肉一样练厚。这里有几句提醒：意志力像肌肉这个说法在心理学界有争议，2016 年一项大规模重复实验没能复现自我损耗效应，所以这一章不会让你"硬撑意志力"，而是给你更稳健的规律训练与生理底座方案。

修复章是这本手册的医务室。心力会漏——长期高压会泄漏，重大挫败会破洞，过度训练会内耗。这一章讲怎么识别漏点、怎么止损、怎么补能量。老子那句"为道日损"——《道德经·第四十八章》，是这一章的总纲：减少妄作、减少外耗，本身就是蓄能。

中国心法章是这本手册的根系。王阳明的致良知、曾国藩的拙诚、老子的柔弱省能、毛泽东的战略定力，四位中国大家各自给心力提供一个中国侧的注脚。这些注脚远不止装饰，它给你一套与西方神经科学互补的话语——神经科学告诉你"怎么做"，中国心法告诉你"为什么这样做能长久"。

践行章是这本手册的操典。日课设计、逆境重启，所有动作在这里汇总成一份可每日执行的清单。Jocko 那句"Discipline equals freedom"——纪律即自由，是这一章的口号。但口号归口号，行动归行动；行动是凌晨闹钟响了你按掉、不是按延后；是计划列在纸上、不是停在脑子里。

## 三、方法论

读、做、复盘三个动作怎么循环起来？我给你三条具体路径。

**第一条：诊断路径——先做能量审计。**

翻开手册第一件事，不是从第一章开始读，是先给自己做一次能量审计。拿出一张纸，分成两栏：左栏写"本周让我有能量的事"，右栏写"本周让我没能量的事"。每栏写五到十条，越具体越好。

写完后看三件事：第一，左栏里有没有"意义感"在做支撑——比如"帮同事解决了一个难题"是有意义的，"刷完一部剧"是即时刺激；第二，右栏里有没有可以砍掉的——无效会议、无效社交、无效刷手机；第三，左右两栏的比例——如果右栏比左栏多一倍以上，你的心力已经处在透支状态，得先修复再练。

能量审计不是做一次就够的。这本手册建议你每两周做一次，连续做三个月，你会看见自己能量曲线的变化规律——什么时候高、什么时候低、什么人或事让你漏得最厉害。规律一旦被你看见，调整就有了抓手。

**第二条：阅读路径——按板块读，不按顺序读。**

这本手册十九章，不必从第一章顺着读到第十九章。按你的能量审计结果选起点：

- 如果诊断出"意义感没了"——从源头章的"意义感是心力的起点"开始读；
- 如果诊断出"自愿承受没了"——从源头章的"自愿承受痛苦是心力的燃料"开始读；
- 如果诊断出"神经回路磨损"——从源头章的"神经回路决定心力上限"开始读；
- 如果诊断出"底盘松动"——从源头章的"浩然之气是中国的心力观"开始读，再延伸到中国心法章；
- 如果诊断出"已经在透支"——直接跳到修复章，先止损再练。

每个板块读完，配套做一件践行动作再往下走。读完意义感那章，写"为何"清单；读完自愿承受那章，选一件"自愿难受"小事连做七天；读完神经回路那章，做"努力即奖励"实验两周。读与做不分离，读了就做，做完才往下读。

**第三条：复盘路径——日课+周复盘+月迭代。**

读完源头四章、做完配套动作之后，你的手册就该进入日课阶段了。这本手册会给你一份日课模板，骨架很简单：

- 早晨：一件"自愿难受"的小事 + 五分钟"为何"清单回看；
- 白天：一次"努力即奖励"的专注时段（推荐四十分钟，参考心流研究的"挑战与技能匹配"原则）；
- 晚上：三分钟"集义"账户记录——今天我做了哪件该做的事；
- 周日：十分钟能量审计 + 一周回看。

日课并非排得越满越好。Jocko 那种凌晨四点半起床、冷水冲淋、跑步、写字的密集日课，是退役海豹指挥官的版本，普通人照搬会崩。曾国藩的日课十二条也有同样问题——他那是咸丰同治年间士大夫的版本，你照搬"夜不出门"这种条目，今天连工作都做不到。日课要按你的生活节律定制，本手册的"日课设计"章会给你一套定制方法。

每周日做一次复盘：本周日课做到几成？没做到的那几项是为什么？下周怎么调？每月做一次迭代：哪几项日课可以加难度？哪几项要砍掉？这套日课+周复盘+月迭代的节律，是班杜拉自我效能感理论的工程化落地——靠连续的小成功积累掌握性经验，不靠一次大目标失败后崩溃。

## 四、避坑

这本手册的用法，最容易掉进三个坑。

**坑一：当成励志书读，读完不做。**

这是最常见的坑。手册里讲弗兰克尔、讲 Goggins、讲曾国藩，读起来都热血上头，可读完不写"为何"清单、不选"自愿难受"小事、不做"努力即奖励"实验，热血三天就退了。励志书是给人打鸡血的，训练手册是给人练肌肉的。前者读完爽，后者做完变。这本手册是后者，请按后者用。

**坑二：照搬英雄日课，不按自己节律定制。**

读到 Goggins 凌晨四点跑步、Jocko 凌晨四点半起床冷水冲淋、曾国藩日课十二条，很容易直接抄过来。可他们是退役海豹、是咸同年间的士大夫，你是一个上有老下有小、白天上班晚上带娃的普通人。照搬英雄日课，第三天就崩。本手册的"日课设计"章会反复强调：日课要按你的生活节律定制，不照搬英雄版本。Goggins 自己也讲过，他的"Accountability Mirror"（问责镜）与其说是让你贴满便利贴，不如说是让你每天早起直面自己的丑陋真相——形式可以变，内核要保住。

**坑三：只练不修，越练越漏。**

有些人日课排得密密麻麻，三个月后人崩了，比不练之前还糟。问题出在只练不修——只往里蓄能，不修漏点。老子讲"为道日损"——《道德经·第四十八章》，损就是修。你的能量漏在无效社交上，砍掉社交比加日课管用；你的能量漏在刷手机上，关掉推送比早起管用。修在练前，练在修后，这本手册的修复章会专门讲怎么修漏。

## 五、践行

这一章不动用太多新动作，先把开篇那五件事接上，再加上两件手册专属动作。

**第一，今天做一次能量审计。** 拿出一张纸，分两栏写"让我有能量的事"和"让我没能量的事"，各五到十条。写完看比例，比例失衡的，先进入修复章。

**第二，按诊断结果选起点章。** 能量审计做完，按本章节给的"阅读路径"选一本手册的起点章节。不必从第一章顺着读。

**第三，配套做一件践行动作。** 读完每章，配套做一件该章给的践行动作再往下读。读与做不分离。

**第四，给自己定一份两周日课。** 不照搬英雄版本，按你的生活节律定一份最简日课——早晨一件"自愿难受"小事、白天一次"努力即奖励"专注时段、晚上三分钟"集义"记录。两周后做第一次复盘。

**第五，准备好一份"复盘日历"。** 不必买专门的复盘本，手机备忘录就行。每周日十分钟，记录本周日课做到几成、下周怎么调。每月一次迭代，记录哪些日课可以加难度、哪些要砍掉。

**第六，建立"中国心法侧栏"。** 准备一页纸，把你读中国心法章时最受触动的几句记下来——王阳明"破心中贼易"、曾国藩"打脱牙和血吞"、老子"自胜者强"、毛泽东"持久战三阶段"。这些话远不止装饰，是你低谷时的弹药库。

**第七，给自己留一个"恢复期"。** 不要把日课排到没有恢复空间。心力训练跟举铁一样，肌肉是在休息时长出来的，神经回路是在睡眠里固化的。每周至少留一天做"减法日"——少做几件日课，多睡一小时。

这套用法你先按七天试一遍。七天后回看，你大概会发现自己已经走过一遍"读—做—复盘"的小循环。把这套小循环跑顺，后面十九章就是把它放大、做厚、做稳。
