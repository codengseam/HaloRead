# 随卦——随时而动

## 一、讲事情

易经第十七卦，叫随卦。

卦象是震下兑上——底下震卦，震为雷；上头兑卦，兑为泽。雷在下，泽在上，雷动入泽中，泽随雷而动。这卦象本身就在讲一件事——随。

"随"是什么意思？随就是跟从，就是顺时顺势而动。可这一卦的"随"，重在随时而动——顺时、顺势、随宜，不拘一格。

卦辞只有六个字——"元亨，利贞，无咎。"

元亨，是大通。利贞，是利于守正。无咎，是没有过错。这六个字分量重——随到好处，能大通，能守正，能无咎。可前提是"随时"——顺时顺势而动，不是乱随。

要说这卦配什么故事，最贴的，莫过于汉光武帝刘秀因时制宜。

刘秀，字文叔，南阳蔡阳人，汉高祖刘邦九世孙。可到他那一代，早成了没落宗室，在家乡种地过日子。地皇三年（22年），天下大乱，王莽新朝摇摇欲坠。刘秀的哥哥刘縯（字伯升）性情刚猛，倾家荡产结交豪杰，在舂陵起兵。刘秀跟着哥哥，骑着一头牛就上了战场——买不起马。

这是刘秀的起点——势单力薄，连匹马都没有。

可刘秀的能耐，从昆阳一战就显出来了。

更始元年（23年），王莽派大司空王邑、大司徒王寻，率大军四十二万围昆阳。城里汉军才八九千人，吓得诸将都想散伙。刘秀站出来，一番话把人心稳住。又趁夜率少数骑兵突围，到定陵、郾城搬来救兵。决战那天，刘秀率三千敢死队直冲王莽中军大营。王邑、王寻轻敌，只带万余人迎战，结果大败，王寻被杀。城里城外夹击，王莽四十万大军一夜崩溃。这一战，把王莽新朝的脊梁打断了。

可这一胜，给刘秀招来了杀身之祸。

更始帝刘玄猜忌刘縯，找了个借口，把刘縯杀了。

消息传到刘秀耳朵里，他正在外头带兵。换作旁人，要么起兵报仇，要么痛哭失态。刘秀两样都没做。他立刻赶回宛城，向更始帝谢罪——谢的是"哥哥有罪"的罪。吃饭睡觉言笑如常，不办丧事，不见旧部，不露一丝悲色。手下人想吊唁，他都不接见。

这是刘秀的"随"——形势如此，他随得住。哥哥刚死，他若硬来，必死无疑。他忍了，忍得连眼泪都不敢流一滴。更始帝反倒不好意思下手了——这么恭顺的人，杀他做什么？

后来更始帝派刘秀去河北宣慰。这一派，是把刘秀送出了笼子。

河北那会儿乱成一锅粥——铜马、大彤、高湖、重连，几十股义军各占一方。刘秀不硬打，因势利导。打得过的，打；打不过的，招；招过来的，安抚。铜马军几十万人，被他一一收编。手下怕铜马军反复，刘秀敢一个人骑马进铜马营巡视，铜马军人心归附，从此关西叫他"铜马帝"。

这是刘秀的"随"——顺势而为，不拘一格。能收则收，能抚则抚，因敌之势，成己之功。

建武元年（25年）六月，刘秀在鄗城（今河北柏乡）称帝，建元建武。这一年他三十一岁。

称帝之后，刘秀没像刘邦那样大杀功臣，也没像项羽那样逞威天下。他做的最多的事，是"与民休息"——释放奴婢，减免赋税，精简官吏，裁并郡县。打了十几年仗，天下疲敝，他知道这时候该"随"什么——随民心，随民力，让百姓喘口气。

刘秀这一辈子，每一步都"随时而动"——起兵时势单力薄，他随得住，骑牛上阵；昆阳之战敌众我寡，他随得住，以少胜多；更始帝猜忌，他随得住，韬光养晦；河北经营，他随得住，因势利导；称帝之后，他随得住，与民休息。

这就是随卦之德。

再看六爻，每一爻都在讲"随"的不同姿态。

初九："官有渝，贞吉。出门交有功。"——官，是馆舍；渝，是变。职位有变，守正则吉。出门与人交往，则有功。这一爻讲随的开局——职位变了，别慌，守正就好；别窝着，出门去交，就有功。

六二："系小子，失丈夫。"——系，是拴住。拴住小子，丢了丈夫。这一爻是反面——取舍不当。小子是弱小之辈，丈夫是刚正之人。随了小的，就丢了大的。随要看对象，不能随错。

六三："系丈夫，失小子。随有求得，利居贞。"——拴住丈夫，丢了小子。随而有求能得，利于守正而居。这一爻跟六二反着——随了大丈夫，丢了小子，是随对了。可随对了还不够，还得守正，才稳。

九四："随有获，贞凶。有孚在道，以明，何咎。"——随而有获，守正防凶。有诚信在道，以明示人，何咎之有。这一爻是随卦的险关——随而有获，容易招忌，所以贞凶。可心里有诚，行在道上，明明白白给人看，就无咎。

九五："孚于嘉，吉。"——孚，是诚信；嘉，是善。诚信于善，吉。这一爻是随卦的顶峰——随到好处，是诚信于善。九五居尊位，以诚随善，天下随之，大吉。

上六："拘系之，乃从维之。王用亨于西山。"——拘系之，从而维系之。王用此道，亨于西山。这一爻是收尾——随到极处，被人拘系捆绑，却仍从而维系，是随的极致。王用此道祭祀西山，亨通。

六爻合起来看，随卦讲的就是一件事——随要看对象、看时机、看正不正。随对了，元亨利贞无咎；随错了，系小子失丈夫；随得有获，要防凶；随到诚信于善，才大吉。

## 二、讲人物

刘秀这个人，史书上写得明白——"性勤于稼穑"，年轻时是个老实种地的。哥哥刘縯爱交游，笑他是高祖的二哥刘仲，没出息。可就是这个"没出息"的刘秀，最后成了光武皇帝。

刘秀的本性是什么？是能屈能伸。

昆阳之战，是刘秀的"伸"。城下四十万敌军，城里八九千兵，诸将吓破了胆。刘秀偏要冲出去搬救兵，又率三千敢死队直冲中军——这是何等的刚决。可这刚决，是逼出来的，是时势所迫，不得不伸。

更始帝杀刘縯，是刘秀的"屈"。哥哥尸骨未寒，他不能哭，不能服丧，不能见旧部，还得去给杀兄的人谢罪。这一屈，屈得比死还难。可刘秀屈住了——饮食言笑如常，不露一丝破绽。手下人想吊唁，他闭门不见。这不是冷血，是大忍。忍住一时之痛，换来一国之命。

刘秀的能屈能伸，跟张良的能忍不一样。张良的忍，是圯上敬履那一跪，跪出《太公兵法》。刘秀的忍，是杀兄之仇压在心头，还得笑着去谢罪——这忍，比张良那一跪重得多。

刘秀的另一个本性，是因时制宜。

河北经营那段，最能看出这一点。他到河北时，兵微将寡，王郎又在邯郸称帝，到处追杀他。刘秀不硬拼，因势利导——能联合的联合，能收编的收编，能安抚的安抚。铜马军几十万人，被他一一收编。手下怕铜马军反复，刘秀敢一个人进营安抚。这是因时制宜——时不同，势不同，随的法子也不同。

刘秀还有一个本性，是宽厚仁柔。

称帝之后，他不杀功臣。高祖刘邦杀了韩信、彭越、英布；光武的"云台二十八将"，大多善终。功臣里有不服的，他容着；有骄纵的，他忍着。寇恂、冯异、耿弇、岑彭，一个个都得了善终。

更难得的是他对百姓的柔。建武二年（26年），他下诏释放奴婢——百姓有嫁妻卖子的，愿意让他们回家，听凭自便，谁敢拘执，依法论罪。又减田租，到建武六年（30年），把十税一恢复到三十税一。打了十几年仗，他先想到的是让百姓喘口气。

这"柔"，不是软弱，是柔道。刘秀自己说过："吾理天下，亦欲以柔道行之。"——我治天下，也要用柔道来行。这话，正是随卦"刚来而下柔，动而说"的注脚——以刚下柔，以柔应刚，动而能悦，这就是随。

## 三、讲背景

随卦为什么取象于雷入泽中？

这得从卦象说起。

震下兑上。震为雷，兑为泽。雷在下，泽在上。雷是动的，泽是静的。雷动入泽中，泽随雷而动——这叫"泽中有雷"。

可这"随"，不是泽跟着雷跑，是雷入泽中，泽因雷而动。雷动于下，泽应于上，一动一应，这就是随。

古人观察自然，发现雷入地中，是秋分之后的事。秋分雷藏，春分雷发。雷藏的时候，万物也跟着收敛；雷发的时候，万物也跟着萌动。万物随雷而作息，这就是"随时"。

所以《象传》说："泽中有雷，随；君子以向晦入宴息。"——泽中有雷，这就是随卦。君子效法此象，到了天晚就入室安息。

这话听着简单，可道理深。向晦入宴息，就是该歇就歇。天黑了，雷都藏了，你还折腾什么？该动则动，该息则息，这就是随时。现代人最缺的就是这个——天黑了不睡，该歇了不歇，违背时序，哪能无咎？

随卦的"神"，就在这一个"时"字。

随在不同场景里，表象不同，内核不变。

因时制宜是随。刘秀起兵时势单力薄，骑牛上阵；称帝后天下疲敝，与民休息。时不同，随的法子也不同。势单力薄时随的是"勇"，天下疲敝时随的是"仁"。可不管随什么，根子都是顺时顺势而动。

顺势而为是随。河北经营，刘秀不硬拼，因势利导，收编铜马。势在敌强我弱，硬拼必败，顺势收编，反败为胜。这是随势。

从善如流是随。刘秀听冯异之策定河北，听耿纯之谏称帝，听伏湛之言安百姓。善言能听，善策能用，这是随善。

可随的内核不变——都是随时而动。要知时，要择善，要守正。不知时，随个什么？不择善，随错了人？不守正，随成盲从了？

## 四、讲道理

孔子《易传》怎么讲随卦？

《象传》说："泽中有雷，随；君子以向晦入宴息。"

——泽中有雷，这就是随卦。君子效法此象，到了天晚就入室安息。

这话点出随卦的根——随时。雷入泽中，是秋雷收藏之象。君子效法此象，作息有时——天黑了就歇，天亮了就起。这是随最朴素的样子，也是最根本的样子。连作息都做不到随时，谈什么大事？

《彖传》说："随，刚来而下柔，动而说，随。大亨贞，无咎，而天下随时，随时之义大矣哉！"

——随卦，刚来而下于柔，动而能悦，这就是随。大通而守正，无咎，天下都随之而动，随时的意义大得很啊！

这段话分量重。"刚来而下柔"——刚爻下来居于柔爻之下，是尊就卑、上就下。"动而说"——震为动，兑为悦，动而能悦。刚下柔，动而悦，这就是随。这话点出随卦的妙处——尊肯就卑，卑才肯从尊。刘秀称帝后不杀功臣、与民休息，正是"刚来而下柔"——尊肯就卑，天下才肯随之。

"天下随时"——天下都随之而动。这话把随从个人提升到天下。一个人随时是小随，天下随时是大随。可大随从小随来——君子先做到向晦入宴息，做到因时制宜，天下才随之而动。

程颐《周易程氏传》论随卦，讲得透。

程颐大意谓，随之道可以致大通，然必利于正，乃得无咎。（大意据《周易程氏传·随》）

——随的道理，可以让人大通，可必须利于守正，才能无咎。

程颐一针见血——随不是无原则的跟从，是守正而随。丢了正，随就成盲从了。刘秀随更始帝，是守正而随——更始帝在位，他守臣子之正；可更始帝无道，他就不随到底，转去河北自立。这就是"利贞"——利于正，正则随，不正则不随。

朱熹《周易本义》论随卦，讲得简练。

朱熹大意谓，随，从也；以刚下柔，动而悦，故为随。（大意据《周易本义·随》）

——随，就是从；刚下来居于柔之下，动而能悦，所以叫随。

朱熹的读法，把卦象和卦义打通——刚下柔，是象；以刚下柔而后能随，是义。尊肯就卑，卑才肯从尊；上肯就下，下才肯从上。这是随卦的进退之道。刘秀敢一个人进铜马营，是"以刚下柔"——尊肯就卑，铜马军才肯从他。

## 五、问道悟道

随卦给现代人的启示，一句话——随的真义在随时，不在跟风。

人这一辈子，谁没"随"过？

职场上，新领导来了，你随不随？随。可怎么随？看领导是善是恶，看政策是正是邪。善而正的，从善如流，跟上去；恶而邪的，守正不阿，别跟着栽。这就是"系丈夫，失小子"——随对的，丢错的。

家庭里，老人老了，你随不随？随。可怎么随？老人对的，听；老人错的，劝。劝不动，守着自己的正，别硬顶。这就是"随有求得，利居贞"——随而有求能得，可要守正才稳。

生意场上，风口来了，你随不随？随。可怎么随？看风是真是假，看势是长是短。真风长势，顺势而上；假风短势，守正不跟。这就是"随有获，贞凶"——随而有获，容易招忌，要守正防凶。

可随卦最深的智慧，在"知时"。

刘秀这一辈子，每一步都"随时而动"——起兵时随勇，昆阳时随决，更始时随忍，河北时随势，称帝后随仁。时不同，随的法子也不同。可根子不变——知时识势，择善而从，守正不阿。

知时，是随的前提。不知时，随个什么？时在勇，你随忍；时在忍，你随勇——都随错了。怎么知时？看势——势在敌强我弱，时在忍；势在敌弱我强，时在冲。刘秀昆阳敢冲，是看出王莽军外强中干；更始时敢忍，是看出自己势单力薄。势看准了，时才看得准。

择善，是随的方向。随错了对象，比不随还糟。六二"系小子，失丈夫"，就是随错了——拴住小的，丢了大的。怎么择善？看人——人是善的，随；人是恶的，别随。刘秀听冯异、听耿纯、听伏湛，都是随善；他没随更始帝到底，是因为更始帝不善。

守正，是随的底线。随不是无原则的跟从，是守正而随。九四"随有获，贞凶"——随而有获，容易招忌，所以要守正防凶。怎么守正？心里有诚，行在道上，明明白白给人看。刘秀收编铜马军后，敢一个人进营安抚，是"有孚在道，以明"——有诚信在道，明明白白给人看，所以无咎。

知时识势，择善而从，守正不阿——这就是随卦的开智之处。

老子说"上善若水"，水善利万物而不争，处众人之所恶，故几于道。水之随，是随形——方则方，圆则圆，可水的性不变。随卦之随，也是这个道理——随时而动，随势而行，可正不可丢。丢了正，随就成盲从了。

## 结语

随时而动，择善守正——知时识势者，方能元亨利贞无咎。
