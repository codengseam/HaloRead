# 小过宜下——小者过而亨

## 一、讲事情

小过这一卦，上震下艮，震为雷，艮为山，是"山上有雷"。雷在山上滚过，声势不小，可到底不是惊天动地那种——雷过山头，响一阵就过去了，过的是小事，不是大事。卦名"小过"，说的就是这层意思：小者过，过而亨。

卦辞怎么讲？

"小过，亨，利贞。可小事，不可大事。飞鸟遗之音，不宜上宜下，大吉。"

小过这一卦，亨通，利于守正。可以做小事，不可以做大事。飞鸟飞过，留下余音——不宜往上飞，宜往下落，这样大吉。

这话听着玄，其实接地气。飞鸟高飞到头了，再往上飞就力竭；往下落一落，反而能歇能蓄。做事也是一样——势头到了，别硬撑着往上冲，往下沉一沉，把小事做扎实，反而能成大事。

彖传里头一句最要紧：

"小过，小者过而亨也。"

——小过，是小者过而能亨。过以利贞，是顺着时势走。柔爻得中，所以小事吉；刚爻失位不中，所以不可大事。有飞鸟之象，飞鸟遗之音，不宜上宜下，大吉——上逆而下顺。

爻辞里头，一头一尾两个"飞鸟"最显眼。初六"飞鸟以凶"——飞鸟硬要往上飞，凶。上六"飞鸟离之，凶，是谓灾眚"——飞鸟飞到头了还要往上冲，凶，是灾祸。两个"飞鸟"都点出"不宜上"三个字。中间几爻，六二"过其祖遇其妣，不及其君遇其臣，无咎"——别往上攀，往下遇，反而无咎；九三"弗过防之，从或戕之，凶"——该防的不防，跟着别人瞎冲，凶；九四"无咎，弗过遇之，往厉必戒"——不过头而遇，往前有危险，得戒着；六五"密云不雨，自我西郊，公弋取彼在穴"——云密而不雨，从西郊起，公在穴中取鸟，是蓄势待发之象。一卦六爻，处处讲的都是"宜下不宜上"。

这话怎么落到地上？讲一个故事你就明白了。

征和二年，公元前91年，长安城里头出了大事。

汉武帝晚年多疑，丞相公孙贺之子被人告发用巫蛊诅咒武帝。武帝大怒，命人彻查。一查就牵连开来，闹成一场大狱——这就是历史上有名的"巫蛊之祸"。

太子刘据被江充陷害，说太子宫里挖出木人。太子恐惧，发兵诛江充，结果被扣上谋反的帽子。丞相刘屈氂带兵与太子战于长安，太子兵败，逃到湖县，被人围捕，自杀。太子妃、三子一女，一并遇害。

太子这一脉，几乎被杀绝了。

可太子还有一个孙子，刚生下来几个月，叫刘病已——后来改名刘询。这孩子也被牵连入狱，关在长安的郡邸狱里头。一个襁褓里的婴儿，关在牢里，眼看活不下去。

这时候，有一个人站了出来。

这人叫丙吉，字少卿，鲁国人，当时做廷尉监，奉诏治巫蛊之狱。他看着这孩子，心里头不忍——太子是无辜的，这孩子更是无辜。他悄悄找了两个女囚，一个叫胡组，一个叫郭征卿，让她们乳养这孩子，自己拿俸禄贴补衣食。

狱里头条件苦，孩子几次病重，丙吉都让乳母悉心照料，硬是把这孩子从鬼门关拉了回来。

这一关，就是五年。

后元二年，公元前87年，武帝病重。有善望气者说长安狱里有天子气。武帝大惊，派人到各狱中把囚犯不论轻重一律杀掉。使者夜里到郡邸狱，丙吉闭门不开，说："皇曾孙在此。他人无辜死者犹不可，况亲曾孙乎！"——皇曾孙在这里。别人无辜被杀都不行，何况是亲曾孙！

使者等到天亮，进不去，回去禀报。武帝一听，恍然若悟，说："天使之也。"——这是天意啊。于是大赦天下。

刘病已这才出了狱。

可出了狱，往哪儿去？他祖父太子刘据一案还没平反，皇室不认他。丙吉把他送到祖母史家——史良娣的娘家——养了一阵。后来武帝临终遗诏，让刘病已入掖庭养于掖庭，归宗正管理。

在掖庭里，刘病已遇到了第二个贵人——掖庭令张贺。张贺本是太子刘据的旧属，感念故主，对这个故主之孙格外照应。他自掏腰包供刘病已读书，又请东海人澓中翁教他《论语》《孝经》。

刘病已这孩子，性子跟一般皇室子弟不一样。他不甘心只在掖庭里读书，常偷偷溜出宫去，到诸陵、三辅一带游走。诸陵是历代先帝的陵邑，三辅是京畿一带。他跟市井里头的人混在一起，斗鸡走马，结交三教九流。

这一混，混出本事来了。

他亲眼看见闾里奸邪怎么作恶，亲耳听见百姓怎么议论官吏，亲身感受吏治的得失。这些事，宫里头长大的皇帝一辈子也看不见，他全看见了。

元平元年，公元前74年，昭帝无嗣驾崩。霍光先立昌邑王刘贺，二十七天废掉。然后选来选去，选中了刘病已——这一年他十八岁。

一个曾经系狱的婴儿，一个在掖庭长大的少年，一个混迹闾里的青年，就这么登上了皇帝的宝座，是为汉宣帝。

宣帝即位后，做的事跟"小过"这一卦，对得上号。

他知道自己是怎么上来的——不是凭门第，不是凭军功，是凭微时那一段经历。所以他不端皇帝架子，躬行节俭，减省用度。他考核郡守县令，亲自过问，赏罚分明。临卦讲过他考核郡守那一套，这里不展开。他综核名实，最恨的就是虚文浮饰——做表面文章的，他一眼就看穿。

为什么他看得这么准？因为他自己就在底层待过，知道那些花架子是怎么糊弄人的。

史家称他在位时"中兴"，跟昭帝并称"昭宣中兴"。

## 二、讲人物

**汉宣帝刘询：从狱中婴儿到中兴之主**

刘询这个人，最难得的是"知民疾苦"四个字。

这四个字，不是从书里读来的，是从他自己的经历里熬出来的。他生下来就坐牢，五岁才出狱；出狱后寄人篱下，先在史家，后在掖庭；少年时混迹闾里，跟三教九流打交道。他见过最苦的人，也见过最坏的事。

这样的人当了皇帝，跟那些生在深宫、长在妇人之手的皇帝不一样。他不会轻信官吏的奏报——他自己就见过官吏怎么糊弄人。他不会轻举妄动——他自己就在刀尖上活过。

《汉书·宣帝纪》说他"躬亲政务，综核名实"，又说他"信赏必罚"。这四个字，是他对付官僚那一套的法子。他不看你嘴上说得多么好听，他看你做出来的事对得上对不上。对得上，赏；对不上，罚。这一手，逼得那些郡守县令不敢再糊弄。

可他也不是一味严苛。他有一段话，说得极有人情味——"庶民所以安其田里而亡叹息愁恨之心者，政平讼理也。"大意据《汉书·宣帝纪》：百姓能在田里安生、没有叹息愁恨，靠的是政令公平、诉讼得理。这话不是坐在龙椅上想出来的，是他自己当百姓时盼过的。

他还做了一件大事——为祖父戾太子刘据平反，为曾祖卫皇后追谥。可他做得很克制，没有大张旗鼓翻案，只是按礼制追谥，把该正的名分正过来。这种"过乎恭"的分寸感，正是小过这一卦讲的——过一点，但不过头。

**丙吉：不居功的真长者**

丙吉这个人，是宣帝一朝最值得说的人物。

他保全了刘病已，可他从来不提这事。宣帝即位后好几年，都不知道自己当年是丙吉救的。后来是宫里一个老婢女上书，说自己对宣帝有保育之功，奏章里头提到丙吉，宣帝才知道这段往事。

宣帝大受感动，要封丙吉。丙吉为人谦退，并不以此自矜。后来才受封博阳侯。

丙吉做丞相时，有个故事特别有名。他出门，路上看见一群人打架，死了人，他不过问；又走，看见一头牛喘，他停下来仔细问。随从奇怪——人命关天的事不问，一头牛喘倒问？丙吉说，百姓打架死伤，是长安令、京兆尹的事，宰相不该在道上过问。可牛喘这种事，关乎时气失节，三公掌调和阴阳，这是我的职责，所以我要问。

这就是"丙吉问牛"。丙吉的智慧，跟小过这一卦对得上——他知道自己该管什么，不该管什么。小事让别人管，大事自己抓。这正是"可小事不可大事"。

丙吉病重时，宣帝亲自去看他，问他身后谁可托付。丙吉推荐了几个人，其中一个是杜延年。宣帝一一记下。丙吉死后，宣帝照着他推荐的人用，个个称职。

丙吉一辈子，最难得的是"不居功"三个字。救了皇帝的命，他不提；做了丞相，他不揽权。这种"行过乎恭"的功夫，正是小过这一卦的精髓。

## 三、讲背景

小过这一卦，得放在它前后的卦序里看。

它前面是中孚，中孚讲诚信感物。诚信立起来了，下面就该"小过"——把诚信落到小事上，一点一点地过。它后面是既济，既济是已经成了。从诚信到小过到既济，是一条路——信在心里，过在小事，成在终局。

那为什么是"小过"，不是"大过"？

《易经》里头有一卦叫大过，是泽风大过，讲的是栋梁之材承受重压，要"独立不惧，遁世无闷"，是大丈夫担当大事的卦。小过不一样。小过是雷山小过，雷在山上滚，声势不小，可到底过的是小事——飞鸟遗之音，响一阵就过去了。

卦象里头藏着讲究。小过六爻，二阳居三四位，四阴在外。阴多阳少，柔盛刚衰。彖传讲"柔得中，是以小事吉也；刚失位而不中，是以不可大事也"——柔爻得中位，所以小事吉；刚爻失位不居中，所以不可大事。眼下这个局面，柔的、小的、退一步的能成；刚的、大的、硬冲的不行。

这跟汉宣帝的处境对得上。

他即位时，霍光还在。霍光是大将军，秉政多年，权倾朝野。宣帝一个十八岁的少年，坐在龙椅上，能跟霍光硬碰硬吗？不能。他得"小过"——把姿态放低，把小事做扎实，等时机。

果然，宣帝即位后头六年，凡事都让霍光做主，自己装得恭恭敬敬。霍光要还政，他辞让不受。霍光死了，他才亲政。亲政之后，他也没大动干戈，而是一点一点地削霍氏的权，最后霍氏谋反被诛，他才彻底掌权。

这一手，就是"小过宜下"——不宜上，宜下；不可大事，可小事。

象传讲得更具体：

"山上有雷，小过；君子以行过乎恭，丧过乎哀，用过乎俭。"

山上有雷，是小过之象。君子看了这卦，就懂得：行止要过于恭敬，丧祭要过于哀戚，用度要过于节俭。三个"过"字，都是"过一点"——不是过分，是稍稍过头。恭一点，哀一点，俭一点，宁可往下走，不要往上冲。

宣帝躬行节俭，正是"用过乎俭"；他对霍光恭恭敬敬，正是"行过乎恭"；他追谥祖父祖母，礼数周全而不张扬，正是"丧过乎哀"。三样全占。

## 四、讲道理

小过这一卦，历代注家说得最多的是"宜下"两个字。

**孔子《易传·彖传》：小者过而亨**

彖传开篇一句定调："小过，小者过而亨也。"——小过，是小者过而能亨。

眼下不是大者当令的时候，是小者得势。小者过——小的人、小的事、小的动作——能过、能成。过以利贞，是顺着时势走。柔得中，所以小事吉；刚失位不中，所以不可大事。

彖传又点出"飞鸟遗之音，不宜上宜下，大吉，上逆而下顺也"。飞鸟飞到头了，再往上飞是逆势，往下落是顺势。顺势则吉，逆势则凶。

孔子这一段，把小过的核心讲透了——不是不能过，是要"小过"；不是不能动，是要"顺势而动"。

**孔子《易传·象传》：行过乎恭，丧过乎哀，用过乎俭**

象传给君子指了三条具体的路：

"君子以行过乎恭，丧过乎哀，用过乎俭。"

行止要过于恭敬，丧祭要过于哀戚，用度要过于节俭。三个"过"字，是"稍稍过头"——不是过分，是宁可多一分。恭一分、哀一分、俭一分，都是往下沉，不是往上冲。

这三条，跟卦象对得上。山上有雷，雷声虽响，到底过山而去——君子看了，懂得势头过了就收，宁可往下沉一沉，把恭、哀、俭三样做到家。

**王弼《周易注》：可小事不可大事，宜下不宜上**

王弼注小过，最重"宜下"二字。他说："小过者，小者过也。可小事不可大事，宜下不宜上。"

王弼是魏晋人，他注《易》以义理为主，不重象数。他看小过，看的是"时"——眼下是什么时势，就该走什么路。眼下是小者得势的时势，那就走小者的路，做小事，往下走。硬要做大事、往上冲，就是逆时，逆时则凶。

这一注，把彖传"上逆而下顺"讲明白了——上逆，是逆时；下顺，是顺时。

**程颐《周易程氏传》：行过乎恭，乃为善过**

程颐讲小过，最重"善过"二字。他说："小过之道，过于小也。行过乎恭，乃为善过。"

程颐是北宋大儒，他看小过，看的是"过"的分寸。过有善过，有不善过。善过是"行过乎恭"——恭一点、敬一点、俭一点，这是好的过。不善过是"弗过防之"——该防的不防，反而过头，这是坏的过。

程颐这一注，把象传三个"过"字提了一层——不是什么"过"都好，是"善过"才好。恭、哀、俭，是善过；骄、奢、纵，是不善过。同样是过，方向不同，吉凶不同。

**朱熹《周易本义》：宜下不宜上，乃为吉**

朱熹注小过，最简明。他说："小过，小者过也。宜下不宜上，乃为吉。"

朱熹是南宋人，他注《易》以程颐为底子，又参以象数。他看小过，看的是"位"——小过六爻，二阳居三四位，四阴在外。阳居阴位，刚而居柔，所以宜下不宜上。往下走，柔能容刚；往上走，刚必折。

朱熹这一注，把彖传"柔得中""刚失位"讲清楚了——为什么宜下？因为柔得中，刚失位。柔得中，往下走能成；刚失位，往上冲必折。

## 五、问道悟道

小过这一卦，往深了想，藏着几条经得起琢磨的规律。

**势弱时，宜下不宜上。**

飞鸟飞到头了，再往上飞就力竭。这时候往下落一落，蓄一蓄，反而能再飞。做事也是一样——势头到了头，别硬撑着往上冲，往下沉一沉，把小事做扎实，反而能成大事。

汉宣帝即位时，霍光在位。他要是硬碰硬，跟霍光争权，下场不会比昌邑王刘贺好——刘贺二十七天就被废了。宣帝不争，让霍光做主，自己装得恭恭敬敬。这一装就是六年。六年里头，他把小事做扎实——考察吏治、了解民情、积累人脉。等霍光一死，他才动手，一举把霍氏拿下。

这就是"可小事不可大事"——大事不能做，做小事；小事做扎实了，大事自然成。

这道理，跟老子讲的"曲则全，枉则直"是一个路子。曲一点，反而能全；枉一点，反而能直。硬挺着，反而折了。小过讲的就是这个"曲"——不是窝囊，是顺势。

**知民疾苦，方能治民。**

汉宣帝为什么能"综核名实"？因为他自己就在底层待过。他见过官吏怎么糊弄人，见过百姓怎么受苦，见过制度怎么走样。这些事，宫里头长大的皇帝看不见，他全看见过。

《汉书·宣帝纪》说他"具知闾里奸邪，吏治得失"。这十个字，是他当皇帝的本钱。不是凭门第，不是凭军功，是凭这一段微时经历。

这道理，跟孟子讲的"民为贵，社稷次之，君为轻"是一个路子。民贵君轻，不是嘴上说说，是得真知道民在哪儿、苦在哪儿。不知道民的苦，谈什么民贵？

宣帝知道。所以他考核郡守，亲自过问；他减省用度，躬行节俭；他平反冤狱，追谥祖父。每一件，都是"小过"——做一点，过一点，不张扬，不冒进。可一点一点做下来，就做成了"中兴"。

**善过与不善过，分寸在心。**

程颐讲"行过乎恭，乃为善过"。同样是"过"，恭是善过，骄是不善过；俭是善过，奢是不善过；哀是善过，纵是不善过。

这分寸，不在外头，在心里头。恭一分是善过，恭十分就是谄媚；俭一分是善过，俭十分就是吝啬；哀一分是善过，哀十分就是伤身。过与不过，差之毫厘，失之千里。

汉宣帝的过，过在分寸上。他对霍光恭，恭到让霍光放心，可没恭到让霍光以为他软；他对祖父追谥，谥到把名分正过来，可没谥到大动干戈翻案；他躬行节俭，俭到减省用度，可没俭到苦行僧。每一件，都过一点，不过头。这就是"小过"的功夫——过乎恭，不过乎谄；过乎哀，不过乎伤；过乎俭，不过乎吝。

这功夫，难就难在分寸。过了头，就是大过；不过，就是无过。小过之难，正在"小"字上——不大不小，刚刚好。

---

## 结语

小过这一卦，教的是势弱时的活法——别硬冲，往下沉一沉，把小事做扎实，把恭、哀、俭三样做到家，时机一到，自然能成。
