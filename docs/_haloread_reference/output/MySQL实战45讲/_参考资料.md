## 九、引用使用规范（BUG-032 教训）

1. **正文行文**：用作者/书名/机构名内联说明，如「林晓斌在《MySQL 实战 45 讲》第 04 讲中指出」「《高性能 MySQL》第 5 章给出的实践建议是」。
2. **不使用方括号编号引用**：禁止 `[BOOK-1]`、`[OFFICIAL-1]`、`[VIDEO-1]` 等编号占位，读者看不懂。
3. **每篇文末**：必须有 `## 本章参考资料` 小节，按出现顺序去重列出本章引用的信源（用作者/书名/机构名 + 章节/讲次），不单列书名。
4. **数字事实**：B+ 树复杂度、MySQL 默认隔离级别、InnoDB 页大小（默认 16KB）、Buffer Pool 默认大小（128MB）等数字必须准确，跨版本差异须标注（如 MySQL 5.7 vs 8.0）。
5. **命令可核验**：所有 SQL/Shell 命令必须可在 MySQL 8.0 运行，参数默认值与版本差异须标注。
