# 史记·汉纪·张骞通西域

## 讲事情

这故事得从一只饮器说起。

汉武帝即位不久，从投降的匈奴人口里听到一桩旧事：匈奴老上单于当年打败了月氏，把月氏王的脑袋砍下来，做成了饮器——喝酒用的。月氏人恨得牙痒，可打不过，只好往西跑，跑到了天涯海角。武帝一听，眼睛亮了：月氏恨匈奴，朕也恨匈奴，要是能派人找到月氏，约他们从西边夹击匈奴，匈奴不就腹背受敌？

可问题是，月氏跑到哪儿去了？没人知道。要去，得穿过匈奴的地盘，再穿过西域，那是九死一生的路。

武帝下诏募使者。满朝文武，没人敢应。这时有个郎官站了出来，叫张骞，汉中人。

建元二年，公元前139年，张骞带着堂邑父和一百多号人，从陇西出发了。堂邑父是匈奴人，本名甘父，是张骞家的胡奴，善射，做向导。

一出陇西，就是匈奴的地盘。张骞一行没走多远，就被匈奴骑兵逮了个正着。

军臣单于看了张骞的文书，冷笑："月氏在吾北，汉何以得往使？吾欲使越，汉肯听吾乎？"——月氏在我北边，汉朝凭什么能派人去？我要是派人去南越，你汉家肯答应？

张骞无话可说。他被扣下了。

这一扣，就是十年。

匈奴人对张骞不坏。给他娶了匈奴妻子，生了儿子。可张骞心里那根弦，始终没断——他随身带着的那根汉节，那根代表汉朝使节身份的竹杖，他一天都没丢。节旄掉光了，光秃秃一根杆子，他攥在手里，像攥着自己的命。

十年里，匈奴人让他放羊放马，看管得渐渐松了。张骞心里记着使命：往西，往西，找到月氏。

机会来了。张骞带着堂邑父和几个随从，趁夜往西逃。

逃出去，是茫茫戈壁。走了几十天，干粮吃完了，水也喝光了。堂邑父善射，射些飞鸟走兽，张骞他们生吃充饥。塔克拉玛干沙漠北缘，那地方连匈奴都不敢深入，张骞硬是走过去了。

终于，他们到了大宛。

大宛在今天的费尔干纳盆地，出产汗血宝马。大宛王早就听说东方有个汉朝，富得流油，正愁没路通商。一见张骞，大宛王高兴坏了。张骞说："我要去月氏，请大王派人送我。我回汉后，汉家必以厚礼相报。"大宛王答应了，派了向导和翻译，把张骞送到康居。

康居在今天的哈萨克斯坦南部，是个游牧大国。康居王又派人把张骞送到大月氏。

大月氏这时候在哪儿？在妫水以北。妫水，就是今天的阿姆河。大月氏被匈奴打败后，一路西迁，过了大宛，过了康居，最后在妫水以北定居下来。这片地方土地肥沃，少有寇盗，大月氏人过得舒舒服服。

张骞找到大月氏王，说明来意：汉家天子派我来，约大王共击匈奴，报当年灭王之仇。

可大月氏王的反应，让张骞凉了半截。

大月氏王说：当年是当年，现在是现在。我们在这儿过得挺好，离汉朝万里之遥，何必再去招惹匈奴？再说了，当年匈奴杀的是先王，如今先王已死，新王在位，这仇，也就淡了。

张骞劝了一年多，大月氏王就是不松口。大月氏"殊无报胡之心"——根本没有报仇的心。

张骞没办法，只好往回走。

他想从羌人地盘绕回去，避开匈奴。可羌人也归附了匈奴，张骞又被匈奴逮住了。

这一扣，又是一年多。

转机来了。军臣单于死了，匈奴内乱，太子於单跟单于的弟弟伊稚斜争位，打得不可开交。张骞趁乱，带着胡妻和堂邑父，又一次逃了出来。

元朔三年，公元前126年，张骞回到长安。

他出发时，是一百多人。回来时，只有他和堂邑父，两个人。

前后十三年。

武帝看着这个衣衫褴褛、满面风霜的中年人，看着他手里那根光秃秃的汉节，半天说不出话。

张骞把西域的见闻，一五一十讲给武帝听：大宛出汗血马，康居游牧，大月氏在妫水北，大夏在大月氏南，安息更大，有几百个城市——那边不产丝绸漆器，不会铸钱币铜器。还有身毒，就是天竺，在大夏东南数千里。

武帝听得入了迷。原来西边还有这么多国家，这么多奇物，这么大一片天地！

武帝拜张骞为太中大夫。

后来，张骞跟着卫青出击匈奴。他曾在西域十几年，知道哪儿有水草，给大军当向导，立了功，封博望侯。博望，"广博瞻望"的意思。

可张骞最大的功，不在封侯。元狩四年，公元前119年，武帝拜张骞为中郎将，第二次出使西域。这回他带了三百人，每人两匹马，牛羊数万，金币丝帛无数。他到了乌孙，又遣副使分赴大宛、康居、大夏、安息。

安息，就是帕提亚帝国，当时西方跟汉朝并列的两大帝国之一。安息王派了两万骑兵到边境迎接汉使。汉使把丝绸、铜器献给安息王，安息王回赠鸵鸟蛋和魔术师。

西域诸国，从此始通于汉。

司马迁给这件事下了两个字——"凿空"。

凿，是开凿；空，是虚空。凿空，就是在虚空里开出一条路来。前人没走过的路，张骞走出来了。前人不知道的世界，张骞带回来了。

## 讲人物

**张骞：十三年持节不失**

张骞这个人，最了不起的不是聪明，是韧。

一百多人出发，两个人回来，十三年。这十三年里，他被匈奴扣了十年多，娶了胡妻，生了儿子，匈奴人给他田给他畜，让他放牧为生。换作旁人，早认命了——有妻有子有田有畜，比在长安当个郎官强多了。可张骞不认。他心里只有一个念头：往西，找到月氏；回来，复命。

这念头，靠什么撑着？靠那根汉节。

汉节是汉朝使节的信物，一根竹竿上挂牦牛尾。张骞攥着这根节，十年没撒手。节旄掉光了，杆子光秃秃的，他还是攥着。这根节，是他对汉朝的承诺，也是他对自己的交代。

司马迁写张骞，用了八个字："为人强力，宽大信人。"——性子强韧有力，待人宽厚有信。"蛮夷爱之"——匈奴人、西域人都喜欢他。这八个字，是张骞能活下来的本钱。强力，是身体扛得住；宽大信人，是人缘处得好。在西域那种地方，光靠蛮力活不下来，得靠人缘。张骞能跟匈奴人处，能跟大宛人处，能跟康居人处，能跟大月氏人处——这份本事，比武艺管用。

张骞的忠诚，不是愚忠。他没死节，没殉国，他活着回来了。活着，比死难。死了，一了百了；活着，得忍十年，得逃两次，得走万里戈壁，得咽下所有屈辱。张骞选了活，因为他心里有使命。使命没完成，不能死。

后来张骞封博望侯，又因从李广击匈奴失期，当斩，赎为庶人。可武帝离不开他，又起用他出使西域。张骞不抱怨，不摆谱，接着干。这份心态，是真正的职业使节。

**汉武帝：开眼看世界的天子**

武帝这个人，雄才大略，可他最大的本事，是敢想。

闻匈奴破月氏以其头为饮器，他就想到联合月氏夹击匈奴。这想法，在当时看是天方夜谭——月氏在哪儿都不知道，怎么联合？可武帝就敢想，敢想就敢做，下诏募使。

张骞走了十三年，杳无音信。换作别的皇帝，早忘了这事。可武帝没忘。张骞回来，武帝不光听他讲，还信他讲，照他讲的去办——通西域，联乌孙，击匈奴。

武帝的远见，在于他看到了西域的战略价值。张骞说大宛出汗血马，武帝就想要——不是为了玩，是为了改良战马，打匈奴。张骞说西域诸国兵弱而富，武帝就想到通商——断匈奴右臂，把西域拉到汉朝这边来。

武帝不是完人。他好大喜功，穷兵黩武，晚年下轮台罪己诏。可在通西域这件事上，他的眼光是超前的。他看到了别人看不到的世界，做了别人做不到的事。一个皇帝，能有这份开眼看世界的胸襟，不容易。

**堂邑父：忠仆与向导**

堂邑父这个名字，知道的人不多。可没有他，张骞走不出戈壁。

堂邑父是匈奴人，本名甘父，是张骞家的胡奴。他善射，懂草原，懂沙漠，懂怎么在绝境里找水找食。张骞一行被匈奴扣了十年，逃出来走戈壁，干粮吃完了，是堂邑父射飞鸟走兽，让张骞他们生吃充饥，才活下来。

堂邑父的忠诚，是另一种忠诚。他不是汉人，可他跟着张骞，从长安出发，到西域，再回长安，十三年没离开。张骞被扣，他在；张骞逃亡，他在；张骞复命，他在。一百多人出发，两个人回来，一个是张骞，一个就是堂邑父。

史书里没给堂邑父立传，他的事迹只在《大宛列传》里带了几笔。可这个人，是张骞通西域的另一半。没有张骞的韧，走不完这十三年；没有堂邑父的射，走不出那片戈壁。使节的功，有向导的一半。

## 讲背景

张骞通西域这事，得放在汉匈战争的大棋局里看。

汉初以来，匈奴是汉朝最大的外患。白登之围后，汉朝实行和亲，送公主送财物，换一时安宁。可匈奴收了礼，照样南下抢掠，"岁入盗边，杀略人民甚众"。文帝景帝忍了几十年，到武帝即位，国库充盈，士马精强，武帝不想再忍了。

元光二年，公元前133年，武帝用王恢之计，设马邑之谋，想诱歼匈奴主力。事泄，匈奴撤退，汉匈彻底翻脸。从此，汉朝主动出击，卫青、霍去病连年出塞。

可匈奴不是孤立的。匈奴在西边，控制着西域——今天的新疆和中亚一带。西域有三十六国，大多臣服匈奴，给匈奴提供粮食、马匹、向导。匈奴从西域得到补给，就能跟汉朝长期对抗。

武帝要彻底解决匈奴，就得断其右臂——把西域从匈奴手里夺过来。怎么夺？先得知道西域是什么样，有哪些国家，能不能拉拢。这就是张骞出使的战略背景。

再说月氏西迁。月氏本是敦煌祁连山一带的游牧民族，强盛时控弦十余万。可匈奴冒顿单于崛起，先破东胡，再破月氏。月氏王被杀，头被做成饮器。月氏残部西逃，过了天山，过了费尔干纳，最后在妫水以北定居，就是大月氏。还有一小部分留在祁连山，跟羌人混居，叫小月氏。

大月氏西迁，是世界史上的大事。他们到了中亚，灭了当地的大夏（巴克特里亚王国），后来建立了贵霜帝国。贵霜帝国成为跟汉朝、罗马、安息并列的四大帝国之一。这一切的源头，就是匈奴破月氏那一场仗。

再说西域的地理。西域，狭义指今天的新疆塔里木盆地，广义包括中亚。塔里木盆地四周是雪山，中间是沙漠，沙漠边缘有一串绿洲城邦——楼兰、车师、焉耆、龟兹、疏勒、于阗。这些城邦靠雪山融水生存，绿洲之间隔着沙漠，所以西域城邦小而分散，没法统一，容易被强权控制。

匈奴控制西域，靠的是军事威慑和傀儡统治。匈奴设"僮仆都尉"，常驻西域，征收赋税。西域诸国苦匈奴久矣，可没人带头反抗。张骞来了，给了他们另一个选择——可以投靠汉朝。

再说丝绸之路的开辟意义。张骞通西域之前，东西方的交流是零星的、间接的。中原的丝绸，经过中亚商人层层转手，最终到达罗马，可汉朝和罗马互相不知道对方的存在。张骞通西域后，汉朝和安息建立了直接联系，丝绸之路正式开通。中国的丝绸、漆器、铸铁技术西传；西方的葡萄、苜蓿、胡麻、汗血马东传。更重要的是，佛教也是经由西域传入中国——那是后来的事，可路是张骞开的。

张骞凿空，不只是开了一条商路，是开了一扇文明交流的窗。这扇窗，影响了此后两千年的世界。

## 讲道理

张骞通西域，历代名家多有评论。

**司马迁**在《史记·大宛列传》里，给张骞的壮举下了两个字——"凿空"。原文是："然张骞凿空，其后使往者皆称博望侯，以为质于外国，外国由此信之。"——张骞凿空之后，后来出使西域的人都自称博望侯，作为人质取信于外国，外国因此信任他们。

"凿空"二字，分量极重。凿，是开凿；空，是虚空。前人没走过的路，张骞在虚空里开凿出来。这不是一般的出使，是开辟前所未有的通道。司马迁用这两个字，是把张骞跟开天辟地的人物并列。太史公惜字如金，可这两个字，他给得毫不犹豫。

司马迁还在《大宛列传》开头写了一句："大宛之迹，见自张骞。"——大宛这个地方的事迹，是从张骞开始才为汉人所知。整部《大宛列传》，几乎就是张骞的见闻录。司马迁把一个使节的见闻，写成了一篇列传，这本身就是最高的评价。

司马迁又写张骞的性格："骞为人强力，宽大信人，蛮夷爱之。"——这十三个字，是太史公对张骞为人的定评。强力是韧，宽大信人是德，蛮夷爱之是效。有韧有德有效，这是使节最高的境界。司马迁写人，向来一字褒贬，这八个字给得郑重。

**班固**在《汉书·张骞李广利传》末尾的赞里写道："孝武之世，图制匈奴，患其兼从西国，结党南羌，乃表河西，列四郡，开玉门，通西域，以断匈奴右臂。隔绝南羌、月氏，匈奴单于失援，遁逃远去，幕南无王庭。遭值文、景玄默，养民五世，天下殷富，财力有余，士马强盛，故能睹犀布、玳瑁则建珠崖七郡，感枸酱、竹杖则开牂柯、越巂，闻天马、蒲陶则通大宛、安息。自是之后，明珠、文甲、通犀、翠羽之珍盈于后宫，蒲梢、龙文、鱼目、汗血之马充于黄门，巨象、师子、猛犬、大雀之群食于外囿。殊方异物，四面而至。"——《汉书·张骞李广利传》。

班固这段话，把武帝通西域的来龙去脉讲得清清楚楚。武帝要制匈奴，怕匈奴联合西域和南羌，于是开辟河西，设立四郡，开玉门关，通西域，断匈奴右臂。文景之治养民五世，天下殷富，士马强盛，这才有力气开疆拓土。通西域后，殊方异物四面而至，明珠、文甲、通犀、翠羽填满后宫，汗血马充盈黄门，巨象、狮子、猛犬、大雀养在苑囿。

班固的评语，是肯定里有保留。他肯定武帝通西域的功绩——断匈奴右臂，幕南无王庭；可他也点出，这事靠的是"遭值文、景玄默，养民五世，天下殷富，财力有余"。言下之意，通西域是国力撑起来的，不是武帝一人之功。文景攒的家底，武帝才有资格挥霍。班固的公允，在于他既看到功，也看到功背后的本钱。

这两位史家，一前一后，一个说"凿空"，一个说"断匈奴右臂"，角度不同，分量都重。司马迁看到的是开创，班固看到的是战略。一个使节，能让两位大史家这样下笔，千古少有。

## 问道悟道

张骞通西域这一段，藏着几条让人开眼的规律。

**第一，坚韧不拔，是成大事的底色。**

张骞十三年，被扣十年多，逃两次，走万里戈壁，一百多人只剩两个。这要是没有一股子韧劲，早就死在匈奴的帐篷里了，或者老死在草原上，没人记得他。

韧，不是硬撑。硬撑是跟自己较劲，撑不住就崩。韧是弯而不折——匈奴给他娶妻，他娶了；给他田畜，他收了；让他放牧，他放了。可他心里那根弦没断，那根汉节没丢。他在匈奴帐篷里过日子，可他不是匈奴人，他是汉使。这份"身在曹营心在汉"的定力，比死节难得多。

历史上成大事的人，都有这股韧。越王勾践卧薪尝胆二十年，忍到夫差放松警惕，一举灭吴。苏武出使匈奴，被扣十九年，持节牧羊，节旄尽落，跟张骞一个模样。司马迁受宫刑，忍辱著书，写成《史记》。这世上，能忍的人，比能冲的人走得远。冲一时容易，忍一世难。张骞的十三年，是忍出来的。

可韧也有条件。韧不是盲目的忍耐，是带着目标的忍耐。张骞忍，因为他要找到月氏，要复命。目标在，韧就有根。目标没了，韧就成了认命。张骞的韧，是有根的韧——那根汉节，就是他的根。

**第二，地理大发现的先驱。**

张骞通西域，比哥伦布发现新大陆早了一千六百多年。哥伦布是带着船队、带着地图、带着王室的全力支持出海。张骞呢？一百多人，两匹马，一根节，往未知里走。

张骞之前，汉人不知道西域有什么。匈奴以北是未知，西域以西是未知。张骞走出去，带回来一个世界——大宛、康居、大月氏、大夏、安息、身毒，这些名字第一次出现在中国史书里。司马迁写《大宛列传》，材料几乎全是张骞带回来的。一个人，撑起了一部列传。

地理大发现的意义，不在于发现了多少地方，在于打开了人的眼界。张骞之前，汉人以为天下就是中原加匈奴加南越。张骞回来一说，武帝才知道西边还有安息，还有身毒，还有一个广阔的世界。这眼界一开，汉朝的战略格局就变了——不再只是南北对抗，而是东西联通。

这规律，古今皆然。张骞通西域，汉朝的视野从东亚扩展到中亚。郑和下西洋，明朝的视野从东亚扩展到印度洋。哥伦布发现新大陆，欧洲的视野扩展到全球。每一次地理大发现，都伴随着文明的跃升。闭关锁国的文明，注定落后；开眼看世界的文明，才能进步。

可地理大发现也有代价。张骞通西域，带来了贸易和文化交流，也带来了战争——汉武帝为了大宛的汗血马，发兵数万远征，死伤过半。哥伦布发现新大陆，开启了殖民时代，美洲原住民几乎被灭绝。地理大发现是双刃剑，开了一扇窗，也开了一道口子。可无论如何，发现本身，是进步的前提。没发现，连选择的资格都没有。

**第三，文明交流，是历史进步的引擎。**

张骞凿空之后，丝绸之路正式开通。这条路，走的不只是丝绸，是文明。

中国的丝绸、漆器、铸铁技术、井渠法，经由这条路传到西方。西方的葡萄、苜蓿、胡麻、胡桃、胡荽、石榴、汗血马，经由这条路传到中国。更重要的是，佛教经由这条路传入中国，深刻影响了中国两千年。

文明交流的意义，在于互相激发。中国有了葡萄、苜蓿、汗血马，汉朝的农业和军事都上了台阶。西方有了丝绸、铸铁，罗马的贵族穿上了中国丝绸。佛教传入中国，跟儒道融合，催生了禅宗、理学。这些交流，不是一方占另一方便宜，是互相成就。

这规律，历史上有无数例子。唐朝开放包容，长安城里住着波斯人、阿拉伯人、日本人、新罗人，唐朝是当时世界上最辉煌的文明。明朝海禁，清朝闭关锁国，中国就渐渐落后了。日本明治维新，主动学习西方，一跃成为强国。文明交流，是进步的引擎；闭关锁国，是衰落的开始。

可文明交流也有讲究。交流不是全盘接受，是有选择的吸收。张骞带回来汗血马，可没带回来匈奴的掠夺习性。唐朝吸收外来文化，可没丢掉自己的根。交流要有自信——我吸收你的好东西，可我不变成你。这份自信，来自自身的强大。汉朝强大，才敢通西域；唐朝强大，才敢开放。弱小的文明，一交流就被同化，甚至被消灭。

张骞凿空，开的不只是一条路，是一种姿态——开眼看世界，主动走出去，把外面的好东西带回来，把自己的好东西送出去。这种姿态，是文明进步的根本动力。

## 结语

一根光秃秃的汉节，走出了一条丝绸之路——张骞用十三年告诉后人：世界很大，得自己走出去看。
