# 始封西垂

## 一、讲事情

秦人最早的故事，得从一个吞鸟蛋的女子说起。

颛顼帝的后代有个叫女修的，正在织布，一只玄鸟飞过，掉下一枚卵。女修把卵吞了，竟怀了孕，生下大业。这故事听着玄，可殷商的祖先契，也是母亲简狄吞玄鸟卵生的——"天命玄鸟，降而生商"。嬴姓和子姓（商）的起源神话，都跟玄鸟有关，说明这两个部族同源，都是东方以鸟为图腾的古老部族。——《史记·秦本纪》

大业娶了少典氏的女儿女华，生下大费。大费就是伯益。大禹治水，伯益跟着干，负责给鸟兽喂食，鸟兽多驯服。舜帝看大费能干，赐他姓嬴，还把本族的女子嫁给他。伯益的后人，从此以嬴为姓。——《史记·秦本纪》

嬴姓这族人，世代有个本事——会赶车。夏朝末年，大费的后人费昌，看夏桀无道，弃夏归商，给商汤当御手。鸣条一战，商汤败夏桀，费昌赶的车。商朝中叶，大费的玄孙孟戏、中衍，给商帝太戊当御手，太戊把女儿嫁给了中衍。嬴姓在商朝"世有功，多以显贵，遂为诸侯"——世代有功，多显贵，成了诸侯。——《史记·秦本纪》

到了周朝，嬴姓出了个造父。造父给周穆王赶车，穆王西巡狩，乐而忘归，造父驾车日行千里，载着穆王赶回来平乱。穆王封造父于赵城，嬴姓从此有了赵氏。后来三家分晋的赵国，就是造父的后人。造父的远房侄孙非子，就是秦人的直系祖先。——《史记·秦本纪》

非子住在犬丘。犬丘在今甘肃天水一带，是周朝西边的边地。非子不爱别的，就爱马和牲畜，"好马及畜，善养息之"——喜欢马和牲畜，善于饲养繁育。——《史记·秦本纪》。犬丘那地方，水草丰美，正适合放牧。非子养的马，膘肥体壮，繁殖得快，名声传到了周孝王耳朵里。

周孝王是周朝第八位天子。当时周朝跟西戎的关系紧张，马匹是战略物资——打仗要马，戍边要马，传信也要马。周孝王听说非子会养马，把他召到汧渭之间——汧水和渭水交汇的地方，让他专门给王室养马。

非子到了汧渭之间，大展身手。"马大蕃息"——马匹大量繁殖。——《史记·秦本纪》。周孝王高兴，想立非子为大骆的嫡嗣，继承大骆这一支。

可这事有人反对。申侯出来拦。申侯的女儿嫁给了大骆，生的儿子叫成。要是非子当了嫡嗣，申侯的外孙成就没份了。申侯跟周孝王说了一段话，大意是：从前骊山之女嫁给我先祖，生了女儿，嫁给戎胥轩，生了中潏。中潏因为母亲的缘故，归附周朝，保西垂，西垂因此太平。现在我女儿嫁给大骆，生了成。嬴姓和申姓世代通婚，西戎才服。您要立非子，断了成的嫡嗣，西戎恐怕要反。——《史记·秦本纪》

申侯这番话，软中带硬。明面上讲的是两家通婚的好处，暗里是威胁——你立非子，西戎就反。周孝王听了，权衡利弊，没立非子为嫡嗣。

可周孝王没亏待非子。他搬出伯益来做说辞——"昔伯翳为舜主畜，畜多息，故有土，赐姓嬴氏"——从前伯益给舜管牲畜，牲畜繁殖多，所以有封土，赐姓嬴。又说"今其后世亦为朕息马，朕其分土为附庸"——现在他的后代也给我养马，我分给他土地，做附庸。——《史记·秦本纪》

周孝王把秦地封给非子，让他做附庸，号称秦嬴，恢复嬴氏的祭祀。非子从此有了"秦"这个名号，秦人这支部族，正式登上历史舞台。这一年，大约是公元前905年。

非子受封，只是附庸。附庸是什么？比诸侯低一等，没有资格朝见天子，只是依附于某个诸侯。非子的封地秦，在今甘肃清水一带，方圆不过几十里，是块小地方。可就是这块小地方，成了后来秦国的根基。

非子之后，秦嬴生秦侯，秦侯在位十年。秦侯生公伯，公伯在位三年。公伯生秦仲。秦仲即位第三年，出大事了。

这一年，周厉王无道，国人暴动，周厉王逃到彘。诸侯叛周，西戎也反了。西戎不仅反，还灭了犬丘大骆之族。犬丘是非子原来住的地方，大骆是非子的父亲。西戎把大骆这一族灭了，等于把秦人的老巢端了。——《史记·秦本纪》

周宣王即位后，想起秦人。周宣王是周厉王的儿子，周朝中兴之主。宣王即位，以秦仲为大夫，命他诛西戎。秦仲从附庸升为大夫，有了正式的官职，也有了伐戎的使命。

秦仲伐戎，打了二十年。秦仲立二十三年，死在戎人手里。秦仲为周朝戍边，战死沙场。——《史记·秦本纪》

秦仲死后，周宣王没忘秦人。秦仲有五个儿子，周宣王把他们召来，给了七千兵，让他们去打西戎。五兄弟带着七千兵，跟西戎死战，"破之"——打败了西戎。——《史记·秦本纪》

周宣王于是把秦仲的后人重新封了一遍，把原来大骆的犬丘之地也还给他们，封为西垂大夫。秦人从此有了两块地——秦地和犬丘，官职也升为西垂大夫，专门镇守西边。

一个养马的部落，经过四代人，从附庸升到大夫，在西垂荒原上扎下了根。

## 二、讲人物

**秦非子：靠本事吃饭的养马人**

非子这个人，史书没记他什么豪言壮语，只记了他一个本事——会养马。"好马及畜，善养息之"，八个字，就是非子的全部画像。可这八个字，是秦人起家的根本。

非子的本事，不是凭空来的。嬴姓这族人，世代跟牲畜打交道。伯益给舜"调驯鸟兽"，是非子的远祖；造父善御，是非子的族亲。嬴姓从东方迁到西方，一路上靠的就是畜牧和御车的本事。非子住在犬丘，那地方水草好，正适合放牧。非子把祖传的本事用上了，养的马膘肥体壮，繁殖得快。

非子能受封，靠的是真本事，不是靠关系。周孝王召他养马，是因为"马大蕃息"——马匹大量繁殖，这是实打实的成绩。周孝王封他，也是因为这份成绩。非子没靠打仗，没靠钻营，靠的是一手养马的好本事。

可非子也有他的局限。他只是附庸，不是诸侯。他想当大骆的嫡嗣，被申侯一拦，就没当成。非子在这场政治博弈里，是被动的一方。周孝王封他秦地，与其说是抬举非子，不如说是周孝王需要一个人在西边养马戍边。非子的本事被周天子用上了，可非子的地位，还是低。

非子的后人，秦仲，比非子更进一步。秦仲不只是养马，还当了大夫，带兵打仗。秦仲打了二十年，死在戎人手里。秦仲的儿子们，接着打，打败了西戎。从非子到秦仲的儿子们，秦人从养马的附庸，变成了戍边的大夫。这份转变，是拿命换来的。

**周孝王：识人用人的天子**

周孝王是周朝第八位天子，是周穆王的儿子。周孝王能识人，他听说非子会养马，就把非子召到汧渭之间，专门养马。这份识人，不简单。一个边地的牧马人，天子能听说他的本事，还能召来用，说明周孝王是真在意马政。

周孝王用非子，也变通。他想立非子为大骆的嫡嗣，被申侯一拦，没立成。可他没亏待非子，封非子为附庸，邑之秦。这份变通，是政治智慧。周孝王既没得罪申侯，又奖赏了非子，还把秦地这块边地交给非子镇守。一石三鸟。

周孝王封非子时说那番话，也讲究。他把非子跟伯益类比——"昔伯翳为舜主畜，畜多息，故有土"——给非子的封赏找了祖宗的依据。这份讲究，是给非子撑门面，也是给嬴姓正名。

周孝王这人，还有一段故事。他是周穆王的儿子，按理说王位轮不到他。周穆王传位周共王，共王传位周懿王。周懿王死后，王位被周孝王夺了——孝王是懿王的叔父。孝王即位，是周朝王位继承的一次变故。孝王自己就是靠变通上位的，他封非子时的变通，跟他自己的经历有关。一个靠变通上位的天子，用变通的方式用人，顺理成章。

**申侯：保外孙的老丈人**

申侯在这场博弈里，是关键人物。他反对立非子为嫡嗣，明面上讲的是两家通婚的好处，暗里是保自己外孙成的地位。

申侯那番话，说得有水平。他没直接说"我外孙不能没份"，而是讲了一段历史——从前骊山之女嫁给我先祖，生女嫁戎胥轩，生中潏，中潏归周保西垂。现在我女儿嫁大骆，生成。嬴姓和申姓世代通婚，西戎才服。申侯把自家的婚事跟西垂的安危绑在一起，让周孝王不得不掂量。

申侯的话里，还藏着威胁——你立非子，西戎就反。这威胁不是空话。申姓跟西戎有渊源，申侯的先祖跟戎人通婚，申侯在西戎里有关系。申侯要是真挑动西戎反，周朝西边就不得安宁。周孝王听了申侯的话，没立非子，是权衡利弊的结果。

申侯这个人，是西周晚期典型的外戚。西周的外戚，靠嫁女儿跟王室联姻，靠外孙的地位保自家的势力。申侯嫁女儿给大骆，生的儿子成，是大骆的嫡嗣。成在，申侯的势力就在。非子要是当了嫡嗣，成就没份，申侯的势力就受影响。申侯保成，就是保自己。

申侯这族人，在西周晚期很关键。后来周幽王废申后和太子宜臼，申侯联合犬戎攻周，杀了周幽王，西周灭亡。——《史记·周本纪》。申侯这族人，能调动戎人，能左右王室，势力不小。非子养马这场博弈里，申侯一开口，周孝王就得听，可见申侯的分量。

**秦仲：战死沙场的戍边人**

秦仲是非子的曾孙，秦人第四代首领。秦仲即位第三年，周厉王无道，西戎反，灭了犬丘大骆之族。秦人的老巢被端，秦仲的压力可想而知。

周宣王即位，以秦仲为大夫，命他诛西戎。秦仲从附庸升为大夫，有了正式官职，也有了伐戎的使命。秦仲伐戎，打了二十年。秦仲立二十三年，死在戎人手里。

秦仲这二十年，不好过。秦人当时地小兵少，西戎势力大。秦仲以一附庸之族，对抗西戎，能打二十年，已经不容易。秦仲最后战死，是拿命履行了戍边的职责。

秦仲的死，没白死。周宣王召秦仲的五个儿子，给七千兵，让他们接着打。五兄弟打败了西戎，周宣王封他们为西垂大夫。秦仲的死，换来了秦人地位的提升——从附庸到大夫，从一块地到两块地。秦仲用命给秦人铺了路。

秦仲这人，史书没记他什么话，只记了他的死。可他的死，是秦人崛起的关键一环。没有秦仲的二十年苦战，没有秦仲的战死，周宣王不会给秦仲的儿子们七千兵，秦人也不会升为西垂大夫。秦仲是秦人从养马部落向戍边诸侯转变的转折点。

## 三、讲背景

秦非子养马受封这事，得放在西周的马政、附庸制度、西戎威胁和周王室衰弱的大背景里看。

先说马政。西周时，马是战略物资。打仗要马——西周的战车，一辆四马；戍边要马——骑兵侦察要马；传信要马——邮驿要马。周朝有专门管马的官，叫"牧人""圉人"。周王室的马，主要养在汧渭之间——汧水和渭水交汇的地方，那地方水草丰美，适合放牧。周孝王召非子到汧渭之间养马，是把非子放到了周朝最重要的马场。

周孝王为什么这么在意马？跟当时的局势有关。周穆王时，周朝西巡狩，跟西戎有过接触。周共王时，周朝跟犬戎打过仗。西戎是周朝西边的大患，打西戎要马，戍西边要马。周孝王召非子养马，是为了对付西戎。非子养的马，是周朝西边戍防的战略物资。

再说附庸制度。西周的封建，分几等。最高是诸侯，有封国，有朝见天子的资格。其次是附庸，附庸于某个诸侯，没有朝见天子的资格，地也小。非子受封，是附庸，不是诸侯。这说明周孝王对非子的封赏，是有限度的——给块地养马戍边可以，给诸侯的名分不行。

周孝王封非子为附庸，也有讲究。附庸的地位低，可附庸有附庸的好处——附庸直接依附王室，不经过诸侯。非子是周王室的附庸，直接听命于周天子，不受诸侯管辖。这份直接依附，让秦人在西边有了相对独立的地位。后来秦人能发展壮大，跟这份独立地位有关。

西戎的威胁，是秦人崛起的最大背景。西戎是周朝西边的游牧部族，分布在今甘肃、陕西西北部。西戎跟周朝的关系时好时坏——好的时候通婚贸易，坏的时候打仗抢掠。周朝西边的安宁，靠的就是跟西戎的关系。

西戎里有一支叫犬戎，跟周朝的关系特别复杂。犬戎跟申侯有渊源——申侯的先祖跟戎人通婚。犬戎后来攻破镐京，灭了西周，是周朝的大患。非子住在犬丘，犬丘就是犬戎活动的地方。非子养马，秦人戍边，都是在犬戎的眼皮底下。

周厉王无道，是西戎反的直接原因。周厉王是周朝第十位天子，他任用荣夷公专利，把山林川泽的收益收归王室，断了百姓的生路。国人暴动，周厉王逃到彘，周朝出现"共和行政"。——《史记·周本纪》。诸侯叛周，西戎也反。西戎反了，灭了犬丘大骆之族。秦人的老巢被端，是周厉王无道的连带后果。

周宣王中兴，是秦人地位提升的契机。周宣王即位，整顿朝政，征伐四夷。周宣王对西戎用兵，需要西边有人戍守。秦人世代跟西戎打交道，是周朝西边最好的戍守力量。周宣王以秦仲为大夫，命他伐戎，是借秦人之力戍西边。秦仲战死，周宣王又给秦仲的儿子们七千兵，让他们接着打。周宣王用秦人，是"以夷制夷"的思路——用熟悉西戎的秦人，对付西戎。

周宣王给秦仲五子七千兵，这数字有讲究。七千兵，在西周晚期是不小的数目。周宣王肯给秦仲五子七千兵，说明周宣王是真把秦人当戍边的主力。这七千兵，是秦人后来打败西戎的关键。没有这七千兵，秦仲五子打不过西戎。

秦人从附庸升为西垂大夫，也是周王室衰弱的反映。西周晚期，周王室衰弱，西边的诸侯靠不住，周天子只能扶持秦人这样的边地部族戍边。秦人的崛起，是周王室衰弱的产物——周天子需要人戍边，秦人需要地盘和名分，双方一拍即合。后来秦人能发展成秦国，最后统一六国，根子就在这时候扎下。

## 四、讲道理

司马迁在《史记·秦本纪》里记秦非子养马受封，没单独写一段"太史公曰"专门评非子。可司马迁记事的笔法，本身就有评价。

司马迁把秦的起源，从女修吞玄鸟卵开始写，一路写到非子养马受封，再写到秦仲战死、秦仲五子破戎。这条线，写得详细。司马迁的用意，是把秦这个统一六国的大帝国的根，追到一个养马的小部落。秦的祖先，给舜管过牲畜，给商汤赶过车，给周穆王驾过车，到非子这一代，给周孝王养马。一代代都是伺候人的活，可就是这份伺候人的活，养出了后来的秦帝国。司马迁的笔法是：英雄不问出处，帝国也有卑微的起源。

司马迁在《史记·秦本纪》末尾的太史公曰里，梳理了嬴姓的谱系——"秦之先为嬴姓。其后分封，以国为姓"——又点出"秦以其先造父封赵城，为赵氏"。这是讲嬴姓的来路和分支，不是直接评非子养马。可司马迁把嬴姓的谱系梳理得这么清楚，本身就是在给秦找根——秦不是凭空冒出来的，秦的根在嬴姓，在伯益，在非子。一个统一六国的帝国，根子能追到舜帝时代的养畜人，这是司马迁想让人看到的。

贾谊《过秦论》论秦的兴起，开篇讲"秦人拱手而取西河之外"，是从秦孝公讲起，没直接论非子养马。可贾谊论秦的根基，说"秦地被山带河以为固，四塞之国也"——秦地有山有河作为屏障，是四塞之国。——《过秦论》。贾谊看秦的地理优势，这块地理优势的起点，就是非子受封的秦地。非子在汧渭之间养马，秦地在今甘肃清水一带，正是后来秦国关中根据地的西端。贾谊论秦的地理，根子能追到非子受封的那块地。

## 五、问道悟道

秦非子养马受封这事，藏着两条让人开眼的规律。

**小本领做成大事业。**

非子靠一手养马的本事，拿到第一块封邑。养马这事，听着不起眼，可非子把养马养到了"马大蕃息"的程度，被周天子看上，封了附庸。一个养马的人，成了秦人的始祖。后来秦人从附庸到大夫，从大夫到诸侯，从诸侯到霸主，从霸主到帝王。这条路的起点，就是非子养马。

历史上靠小本领起家的，不止非子。周朝的祖先弃，擅长种庄稼，被舜帝封为后稷，后来周人靠农业起家，灭了商朝。——《史记·周本纪》。周的起源，跟秦的起源一样，都是靠一手实在的本事。刘邦起家时是个亭长，管十里地的治安，可就是这个小亭长，最后成了汉高祖。——《史记·高祖本纪》。朱元璋起家时是个和尚，要过饭，可就是这个要饭的和尚，最后成了明太祖。

这些人有个共同点：起点低，本事实在。非子养马，弃种庄稼，刘邦当亭长，朱元璋当和尚，起点都不高。可他们都有一手实在的本事，把本事用到了该用的地方。非子养马养到了周天子都看上的程度，弃种庄稼种到了被舜帝封为后稷的程度。本事不在大小，在精；机会不在早晚，在遇。非子遇上了周孝王，弃遇上了舜帝，刘邦遇上了秦末大乱，朱元璋遇上了元末大乱。本事和机会一碰，大事就成了。

**边患反而成就了戍边的人。**

秦人能崛起，靠的是西戎的威胁。没有西戎的威胁，周天子不会在西边养一支戍边的力量。非子养马，是因为打西戎要马；秦仲伐戎，是因为西戎反了；秦仲五子破戎，是因为周宣王要借秦人之力对付西戎。秦人每一步崛起，都跟西戎的威胁有关。

这条规律，历史上反复出现。周朝的祖先古公亶父，因为戎狄的侵逼，从豳迁到岐下，在岐下发展起来，最后灭了商朝。——《史记·周本纪》。周人的崛起，跟戎狄的侵逼有关。晋朝的刘渊，是匈奴人，被晋朝留在并州戍边，可刘渊借着戍边的机会壮大，最后灭了西晋。——《晋书·刘元海载记》。唐朝的安禄山，是胡人，被唐朝留在幽州戍边，可安禄山借着戍边的机会壮大，最后造反，差点灭了唐朝。——《旧唐书·安禄山传》。

这些事例，跟秦人的崛起是一个路子：边患威胁，朝廷扶持边地的人戍守；戍守的人借着朝廷的支持壮大；壮大之后，反过来威胁朝廷。秦人壮大了，最后灭了周朝（秦昭王灭周）；刘渊壮大了，灭了西晋；安禄山壮大了，差点灭了唐朝。

可秦人跟刘渊、安禄山不一样。秦人是周朝的附庸，世代受周朝封赏，对周朝有认同。秦人戍边，是替周朝戍边，不是借机割据。秦人从附庸到大夫，从大夫到诸侯，从诸侯到帝王，是循序渐进，不是一步登天。秦人用了六百多年，从非子养马到秦始皇统一六国。这份慢功夫，是秦人跟刘渊、安禄山的区别。

边患成就戍边的人，这条规律有正反两面。正面：边患逼出戍边的人，戍边的人保了朝廷。反面：戍边的人壮大，反过来威胁朝廷。秦人走的是正面——戍边保周，最后取周而代之。刘渊、安禄山走的是反面——戍边壮大，直接造反。同一条规律，走法不同，结局不同。

## 结语

一匹马换一块地，一块地长出一个帝国。
