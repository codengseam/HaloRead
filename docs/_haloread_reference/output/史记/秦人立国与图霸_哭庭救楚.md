# 哭庭救楚

## 一、讲事情

这一章的故事，得从一桩灭门惨案说起。

楚平王有个太子叫建，太子建有两个老师，一个叫伍奢，一个叫费无忌。费无忌是个小人，太子建不喜欢他，他却得楚平王的宠。楚平王给太子建娶媳妇，娶的是秦国的姑娘。费无忌一看这秦女长得美，跑去跟楚平王说："这姑娘太漂亮了，大王您自己留下吧。"楚平王居然真就自己纳了。——《史记·伍子胥列传》。

费无忌干完这事，心里发虚：太子建将来要是即位，能饶了我？他就开始在楚平王面前说太子的坏话，说太子怨气冲天，要造反。楚平王信了，要拿太子建问罪。太子建闻风逃到宋国。费无忌又跟楚平王说："伍奢是太子的老师，不杀伍奢，太子必有后患。"楚平王把伍奢下了狱。——《史记·伍子胥列传》。

伍奢有两个儿子，大的叫伍尚，小的叫伍子胥。费无忌知道伍子胥厉害，跟楚平王说："把伍奢关着没用，他两个儿子在外头，尤其那个伍子胥，文武双全，不一起杀了，将来必为楚患。不如骗他们回来，说回来就放父亲，不回来就杀父亲。"楚平王派人去召伍尚、伍子胥。

伍尚对弟弟说："楚王召我们，是要把我们父子一锅端。可父亲叫我们回去，我们怎能不回？我回去，跟父亲一起死。你跑，将来替我们报仇。"——"楚之召我兄弟，非欲以生我父也，恐有脱者后生患，故以父为质，诈召二子。二子到，则父子俱死。何益父之死？往而令雠不得报耳。不如奔他国，借力以雪父之耻。俱灭，无为也。"——《史记·伍子胥列传》。

伍尚回了郢都，跟父亲一起被杀。伍子胥拉弓搭箭，对着使者，使者不敢上前，伍子胥逃了。——《史记·伍子胥列传》。

这一逃，逃出了后面所有的事。

伍子胥一路往吴国跑。楚平王下令悬赏捉拿，画影图形，各处关卡都贴着伍子胥的像。到了昭关——这是楚吴交界的险要关口，盘查得紧，伍子胥差点没脱身。——《史记·伍子胥列传》。出了昭关，一条大江拦路。江上有个渔父，认出了他，划船送他过江。伍子胥解下佩剑："这剑值百金，送您。"渔父说："楚国之法，得伍胥者赐粟五万石，爵执珪——楚国的法令，抓到你给五万石粟，还封执珪之爵。我要是贪财，还等你这把百金的剑？"——《史记·伍子胥列传》。

伍子胥到了吴国，正赶上吴王僚和公子光争权。公子光想夺位，伍子胥看出来了，给他荐了一个刺客——专诸。公子光让专诸把短剑藏在烤鱼里，刺杀了吴王僚，自立为王，就是吴王阖闾。——《史记·吴太伯世家》《史记·刺客列传》。

伍子胥帮阖闾夺了位，阖闾也给了伍子胥机会。阖闾用孙武为将，伍子胥为副，练吴军。吴国在东南悄悄崛起。

楚平王死了，楚昭王即位。楚昭王是个少年，楚国朝政被费无忌一党把持，内政混乱。伍子胥等的就是这一天。

前506年，吴王阖闾亲率吴军伐楚。孙武为将，伍子胥、伯嚭为副。吴军沿淮河西进，舍舟登陆，千里奔袭。楚军仓促迎战，五战五败。吴军攻入郢都，楚昭王仓皇出奔，逃到随国。——《史记·楚世家》《左传·定公四年》。

伍子胥进了郢都，没去找楚昭王，先找楚平王的墓。楚平王死了十年了，伍子胥把他从坟里挖出来，抄起鞭子，抽了三百下。——"及吴兵入郢，伍子胥求昭王。既不得，乃掘楚平王墓，出其尸，鞭之三百，然后已。"——《史记·伍子胥列传》。

三百鞭，鞭的是死人，泄的是活人十几年的恨。

伍子胥有个好朋友，叫申包胥。当年伍子胥逃亡时，申包胥放他走的。如今伍子胥鞭了楚平王的尸，申包胥躲在山里，派人给伍子胥传话："子之报雠，其以甚乎！吾闻之，人众者胜天，天定亦能破人。今子故平王之臣，亲北面而事之，今于僇尸，此岂其无天道之极乎！"——你报仇报得太过分了！我听说，人多能胜天，天定也能破人。你本是平王的臣子，亲自北面事奉他，如今侮辱他的尸首，这难道不是违背天道到了极点吗！——《史记·伍子胥列传》。

伍子胥回了句话，成了千古名言："为我谢申包胥曰，吾日莫途远，吾故倒行而逆施之。"——替我谢谢申包胥，我日子不多了，路还远，所以我只好倒行逆施了。——《史记·伍子胥列传》。

"日暮途远，倒行逆施"——这八个字，是一个复仇者最后的自白。他知道自己做得过分，可他不在乎了。

申包胥见劝不住伍子胥，可楚国还得救。救楚国，找谁？找秦国。

为什么找秦国？因为秦楚是姻亲。楚昭王的母亲是秦女——当年楚平王纳的太子建未婚妻，就是秦国姑娘，后来生了楚昭王。楚昭王身上流着秦国的血。——《史记·楚世家》。

申包胥日夜兼程，跑到秦国，求见秦哀公，说："吴为封豕长蛇，以荐食上国——吴国像大猪大蛇一样蚕食上国，楚国的祸，也是秦国的祸。楚国要是亡了，秦国的西部也不得安宁。请大王出兵救楚。"——《左传·定公四年》。

秦哀公听了，不冷不热地说："你先去馆舍休息，寡人将图之——寡人考虑考虑。"——《左传·定公四年》。

申包胥一听，这是推辞。他不走。他立在秦庭的墙根下，开始哭。日夜不绝声，勺饮不入口。一天，两天，三天……七天。秦哀公为之赋《无衣》，申包胥顿首而坐。——《左传·定公四年》。

《无衣》是《诗经·秦风》里的一首：

"岂曰无衣？与子同袍。王于兴师，修我戈矛。与子同仇！"——谁说没衣服？我跟你穿一件战袍。大王要出兵，修好我的戈矛，跟你同仇敌忾！——《诗经·秦风·无衣》。

秦哀公赋这首诗，就是答应了。申包胥哭了七天七夜，把秦国哭动了。

前505年，秦哀公派子蒲、子虎率车五百乘救楚。——《左传·定公五年》。

五百乘，在当时是一支不小的兵力。秦军入楚，和楚军残部合兵。秦楚联军在沂地大败吴军，又在军祥打败吴军。吴军腹背受敌，加上吴王阖闾的弟弟夫概王在国内造反，阖闾不得不撤兵回国。楚昭王复入郢都。——《史记·楚世家》《左传·定公五年》。

楚国复国了。申包胥立了大功。楚昭王要封赏他，申包胥逃赏，隐居不见。——"申包胥逃赏"——《左传》。

他说："吾以君之存楚也，以赏之，是卖楚也——我哭秦庭是为了存楚国，要是领了赏，就成了卖楚国之祸换富贵的人了。"——《左传》。

## 二、讲人物

**伍子胥：复仇者的执念与悲剧**

伍子胥这辈子，就活在一个"仇"字上。

父兄被杀，他没死，他要报仇。逃亡路上，过关卡，乞饭食，受尽屈辱。到了吴国，帮阖闾夺位，练吴军，等了十几年，终于等到了破楚的那一天。

伍子胥的复杂，在于他既是英雄，又是"恶人"。破楚鞭尸，是英雄的复仇；可鞭一个死了十年的人，掘墓出尸，抽三百鞭——这又是恶人的做派。司马迁看得很准："弃小义，雪大耻，名垂于后世"——抛弃小义，洗雪大耻，名垂后世。——《史记·伍子胥列传》。可司马迁也叹："怨毒之于人甚矣！"——怨毒这种东西，对人影响太大了！——《史记·伍子胥列传》。

伍子胥的结局也不好。后来吴王夫差听信伯嚭的谗言，赐伍子胥死。伍子胥临死说："抉吾眼悬吴东门之上，以观越寇之入灭吴也——把我的眼睛挖下来，挂在吴国东门上，我要看着越国军队进来灭吴。"——《史记·伍子胥列传》。伍子胥一生复仇，最后自己也死于别人的谗言和君王的猜忌——他成了另一个"伍奢"。

伍子胥这个人，最让人唏嘘的是：他太懂复仇，可他不懂放下。他对楚平王的恨，烧了一辈子；他对吴王夫差的忠，也烧了一辈子。恨和忠，都是火，烧完别人烧自己。

**申包胥：忠臣的极致**

申包胥和伍子胥是朋友，可两个人走了截然相反的路。

伍子胥报仇，申包胥救国。伍子胥鞭尸，申包胥哭庭。伍子胥倒行逆施，申包胥以命相搏。

申包胥的可贵，在于他"不劝而救"。他劝过伍子胥，伍子胥不听，他没再纠缠。他转身就去秦国搬救兵。他知道，劝一个复仇者收手是白费口舌，可救一个国家还来得及。

哭秦庭七日，是申包胥最震撼的一幕。七天七夜不吃饭，日夜哭不绝声。这不是作秀，是真拿命在换。秦哀公一开始不想出兵——秦国跟吴国无冤无仇，凭什么替楚国打仗？申包胥用命逼秦国表态。这跟后来蔺相如完璧归赵时面对强秦"怒发上冲冠"是一个路子——弱者面对强者，唯一的筹码就是不要命。——《史记·廉颇蔺相如列传》。

申包胥最让人佩服的，是逃赏。楚国复国，他功劳最大，封赏是题中应有之义。可他跑了。他说，我救楚国是为了楚国，不是为了自己。要是领了赏，就成了"卖楚国之祸换富贵"的人。这份清醒，千古少有。多少人立了功就等着封赏，申包胥立了这么大的功，却躲得远远的。

《左传》记申包胥，没有一句"君子曰"评价。可《左传》记了他的话，记了他的哭，记了他的逃赏——这些事实本身就是评价。一个忠臣该有的样子，申包胥全占了。

**秦哀公：被哭醒的秦君**

秦哀公在《史记》里着墨不多，可这一仗，是他这辈子最关键的一笔。

秦哀公一开始不想管。楚国的事，跟秦国有什么关系？申包胥来求救，他说"寡人将图之"——我考虑考虑。这是外交辞令，翻译过来就是"不想管"。

可申包胥哭了七天七夜，秦哀公扛不住了。他赋了《无衣》——"岂曰无衣？与子同袍"——这是答应出兵了。

秦哀公为什么改主意？一方面是被申包胥的忠义打动，另一方面也是算清了账：吴国要是灭了楚国，下一个就是秦国。楚国是秦国和吴国之间的缓冲，楚国亡了，秦国直接面对吴国的兵锋。秦哀公不是被哭糊涂了，是被哭醒了。

**楚昭王：少年流亡之君**

楚昭王在这出戏里，是个被动的主角。

他爹楚平王造的孽，他来还。少年即位，吴军打进来，他仓皇出奔，逃到随国。随国是个小国，可收留了他。吴军追到随国，对随人说："周之子孙在汉川者，楚实尽之——周朝的子孙封在汉水流域的，楚国都灭了，你们凭什么还留着？"随人没交出楚昭王。——《左传·定公四年》。

楚昭王复国后，倒是长进了。他记住了逃亡的苦，治国比他父亲强。后来楚国能从灭国的边缘缓过来，楚昭王是关键。可这一章里，他是个被救的角色——救他的是申包胥，是秦哀公，是五百乘秦军。

## 三、讲背景

这一仗，得放在三重背景里看：楚国的内乱、吴国的崛起、秦楚的姻亲。

先说楚国的内乱。楚平王杀伍奢，根子在费无忌。费无忌为什么能得宠？因为楚平王自己就糊涂。楚平王纳了儿媳，这是乱伦；听信谗言杀太子杀伍奢，这是昏聩。楚国从楚庄王称霸之后，几代国君一代不如一代，到楚平王这里，内政已经烂了。伍子胥鞭尸，鞭的是楚平王，可这烂摊子是楚国自己造成的。

楚国的问题，还在于贵族专权。楚国的大族——屈、景、昭三家，把持朝政。费无忌能得逞，就是因为楚国的权力制衡出了问题。一个佞臣能害死太子太傅一家，说明楚国的制度已经失灵。

再说吴国的崛起。吴国本是东南蛮夷，中原诸侯不拿它当回事。可到了阖闾这一代，吴国突然强了。为什么？因为阖闾用了两个人——孙武和伍子胥。孙武是齐国人，写了《孙子兵法》，练吴军；伍子胥是楚国人，恨楚国入骨，一心要报仇。两个"外来人才"，把吴国推上了霸主的位子。这跟秦穆公用百里奚、由余是一个路子——人才不问出身，谁用谁强。

吴国能五战入郢，还靠一个地利——吴国在长江下游，楚国在长江中游。吴军沿长江溯流而上，水路便利。楚国的主力在北方防晋国，南方空虚。吴军从楚国最薄弱的方向打进来，一举破都。

三说秦楚的姻亲。秦国为什么肯出兵？除了申包胥哭得狠，还有一个硬邦邦的理由——秦楚是姻亲。楚昭王的母亲是秦女——当年楚平王纳的太子建未婚妻，就是秦国姑娘，后来生了楚昭王。楚昭王身上流着秦国的血。楚国亡了，秦国的外孙怎么办？秦国在楚国的政治影响力怎么办？申包胥找秦国，找对了人。

秦楚联姻不是一天两天了。从秦穆公时代起，秦楚就通婚。穆公帮晋文公回国，那是秦晋联姻；可秦楚联姻更早、更持久。春秋时秦晋联姻叫"秦晋之好"，可秦楚联姻才是真正的"老亲"。

四说春秋末年的格局。前506年，正是晋国衰落、吴国崛起、楚国动荡的时候。晋国国内六卿专权，自顾不暇；楚国被吴国打趴下；吴国一时风光无两。秦国偏居西陲，本来没人拿它当回事。可这一仗，秦国一出手，吴国就得撤兵——这说明什么？说明秦国已经是华夏秩序里的一块砝码。天平往哪边倾斜，秦国加在哪边，哪边就赢。

## 四、讲道理

这一章，真正有明确点评的名家，首推司马迁。

司马迁在《史记·伍子胥列传》末尾有一段"太史公曰"，对伍子胥有直接评价："怨毒之于人甚矣！王者尚不能行之于臣下，况同列乎！向令伍子胥从奢俱死，何异蝼蚁。弃小义，雪大耻，名垂于后世，悲夫！方子胥窘于江上，道乞食，志岂尝须臾忘郢邪？故隐忍就功名，非烈丈夫孰能致此哉？"——怨毒对人影响太大了！君王尚且不能对臣下滥用，何况同辈！假使伍子胥跟伍奢一起死了，跟蝼蚁有什么区别。抛弃小义，洗雪大耻，名垂后世，可悲啊！伍子胥困在江上，路上讨饭，心里何尝有一刻忘记郢都？所以隐忍成就功名，不是烈性丈夫谁能做到！——《史记·伍子胥列传》。

司马迁这段话，有两层意思。一层是叹"怨毒"之烈——怨气这种东西，能把人烧一辈子，连君王都不能随便对臣下结怨，何况平辈。楚平王杀伍奢，结的就是这个怨；伍子胥鞭尸，还的就是这个怨。另一层是赞伍子胥的"隐忍"——他要是为了"小义"（孝道里"父死子随"那一套）跟伍奢一起死了，那就是蝼蚁；他抛弃小义，忍辱负重，终于雪了大耻，这才是"烈丈夫"。

司马迁为什么这么懂伍子胥？因为他自己也是忍辱负重的人。他替李陵说话受了宫刑，没死，写《史记》。伍子胥父兄被杀没死，报仇。两个人都是"弃小义，就大业"的人。司马迁评伍子胥，有他自己的影子在里头。

关于申包胥，古代名家明确点评的不多。《左传》记了他的事，没下"君子曰"的断语。但《左传》记了一个细节：申包胥逃赏，"曰：吾以君之存楚也，以赏之，是卖楚也"——《左传》。这个细节本身就是一种评价——一个人立了救国的大功，却把功劳推掉，因为他觉得救国不是为了领赏。《左传》不评，却用事实说话。

关于秦哀公赋《无衣》，《毛诗序》对《无衣》有一句注："《无衣》，刺用兵也。秦人刺其君好攻战，亟用兵，而不与民同欲焉。"——《毛诗序》。可这个说法跟秦哀公救楚的语境对不上——秦哀公赋《无衣》是答应出兵救楚，不是"好攻战"。所以《毛诗序》的说法，更可能是针对秦国后来的穷兵黩武，而非这首诗在定公四年这场外交里的本意。秦哀公赋《无衣》，取的是"与子同袍""与子同仇"的意思——秦国跟楚国同穿一件战袍，同仇敌忾。

## 五、问道悟道

哭庭救楚这出戏，藏着三条让人开眼的规律。

**第一条：复仇的边界——日暮途远，倒行逆施。**

伍子胥鞭尸三百，申包胥责备他"无天道之极"。伍子胥回了一句"日暮途远，倒行逆施"——我知道过分了，可我等不及了。

复仇这件事，最难的不是下决心，是收手。伍子胥收不住，因为他恨了太久。一个人恨了十几年，一旦有了报仇的机会，理智是拦不住的。可复仇过了头，就变成了另一种恶。伍子胥鞭尸，鞭的是死人，可这举动本身，已经跟楚平王杀他父兄一样——都是恃强凌弱，只不过一个欺负活人，一个欺负死人。

历史上能跟伍子胥比的，是战国时的赵襄子。智伯瑶水灌晋阳，赵襄子联合韩魏灭了智伯，把智伯的头骨漆成酒杯。——《史记·刺客列传》。豫让要替智伯报仇，几次刺杀赵襄子不成，最后赵襄子让他击衣报仇，豫让剑击其衣然后自杀。赵襄子对智伯的恨，也到了辱尸的地步。可赵襄子是"胜者辱尸"，伍子胥是"复仇辱尸"——一个是炫耀，一个是泄愤，都是恨过了头。

伍子胥的悲剧，在于他活成了一个"复仇机器"。机器没有刹车，只能一直转到散架。他后来被吴王夫差赐死，临死要"抉眼悬门"看越人灭吴——到死都在恨，到死都在"倒行逆施"。一个被恨烧了一辈子的人，最后也被别人的恨和猜忌烧死——这大概是天道的另一种"定"。

**第二条：忠诚的极致形式——以命换命。**

申包胥哭秦庭七日，是"以命换命"的极致。他不是哭两声做样子，是七天七夜不吃饭，日夜不绝声。这是拿命在赌——要么秦国出兵，要么我死在秦庭。

这种"以命相搏"的忠诚，在中国历史上有一条脉络。比申包胥早的，有商朝末年的伯夷叔齐，不食周粟，饿死首阳山。——《史记·伯夷列传》。比申包胥晚的，有豫让漆身吞炭刺赵襄子。——《史记·刺客列传》。再往后，有诸葛亮"鞠躬尽瘁，死而后已"。——诸葛亮《后出师表》。

这些人有一个共同点：他们把"忠"看得比"命"重。伯夷叔齐忠于商朝，饿死；申包胥忠于楚国，哭到快死；豫让忠于智伯，毁容自残；诸葛亮忠于蜀汉，累死军中。这份"忠"，不是愚忠——伯夷叔齐不是忠于纣王，是忠于自己的原则；申包胥不是忠于楚平王，是忠于楚国这个"社稷"。

申包胥的忠，还有一个层次——他忠的是"国"，不是"君"。楚平王杀伍奢，是昏君；楚昭王逃亡，是弱主。可楚国这个"社稷"不能亡。申包胥救的不是楚昭王一个人，是楚国几百年的基业。这份"忠社稷不忠君王"的清醒，比单纯的"忠君"高了一个层次。逃赏那句话说得明白——他存的是楚国，不是给自己捞资本。

**第三条：秦国已是华夏秩序的砝码。**

这一仗最深远的意义，不在楚国复国，在秦国出手。

前506年，吴国破楚，风光无两。可秦国一出手，五百乘战车，吴国就得撤兵。这说明什么？说明秦国已经是华夏棋盘上的一颗重子。楚国是棋盘上的大国，吴国是新兴的霸主，可秦国一加进来，天平就倾斜了。

秦国从穆公之后，沉寂了一百多年。康公、共公、桓公、景公、哀公，几代国君没什么大作为，中原诸侯都快忘了秦国了。可这一仗，秦国提醒了所有人：我还在这儿。楚国亡了，下一个就是秦国——秦哀公看穿了这一层，所以才出兵。这不是义气，是战略。

秦国这次出兵，还埋下了一颗种子——秦国开始介入中原事务。穆公时代秦国想东进被晋国挡住，转向西戎；如今晋国衰落，楚国被打趴，中原出现权力真空，秦国趁机插了一脚。这一脚不大，可它是个信号：秦国不再只盯着西戎，它开始往东看了。

两百多年后，秦始皇统一六国。可秦国"东望"的起点，不是商鞅变法，是这一次——申包胥哭来的五百乘战车。穆公往西看，哀公往东看，孝公往东打，始皇往东收。秦国的东进，是一条一百多年的长线，哭庭救楚是这条线上的第一个结。

---

## 结语

三百鞭抽碎了旧恨，七日哭哭来了救兵——这一仗，秦国第一次成了华夏天平上的砝码。
