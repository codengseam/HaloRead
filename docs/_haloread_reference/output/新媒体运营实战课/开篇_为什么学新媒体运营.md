# 流量重构了生意，也重构了个人：为什么人人都要懂新媒体运营

## 入戏：一条短视频把一家濒临倒闭的工厂救活了

2024 年，浙江一家做了十二年外贸的袜子厂订单腰斩，仓库里压着三百万双库存。老板娘五十多岁，被儿子拉去拍了一条抖音——她坐在车间里，一只一只把袜子翻过来给镜头看针脚，用方言说「我们家袜子二十年没换过机器，就是不肯偷工」。那条视频没投一分钱 Dou+，三天跑了七百万播放，库存两个月清空，还接到了四十多个海外华人的批发私信。同一年，北京一家开了八年的实体书店在商场里贴出「即将闭店」，老板发了一条朋友圈告别，被朋友转到小红书，评论区涌进上千条「求别关」，但书店还是关了——因为那阵热度没有转化成任何可持续的生意。

这两件事发生在同一年、同一个国家、同一套平台规则下。区别只有一个：袜子厂老板娘无意中踩中了「流量如何被分配、信任如何被积累、价值如何被看见」这三件事，书店老板没有。这不是运气问题，是认知问题。流量重构了生意，也重构了个人——这件事已经发生五年了，但很多人还没意识到。

## 破题：新媒体运营是数字时代的生存元技能

本篇要讲的「道」只有一句：新媒体运营不再是某个岗位、某种职业，而是数字时代人人都需要的生存元技能——无论你做不做自媒体，你都得懂注意力如何流动、信任如何建立、价值如何被看见。

黄有璨在《运营之光》里把中国互联网划成三段：概念驱动时代（1995—2004），谁先把一个新概念搬上线谁赢；产品驱动时代（2005—），产品同质化之前谁的产品好谁赢；运营驱动时代（至今—未来），产品都差不多之后，谁更懂用户、谁更会连接用户谁赢。大意据《运营之光》。我们正处在他说的第三个时代。产品同质化的意思不只是「两个 App 功能一样」，而是「两个火锅店、两本理财书、两个健身教练提供的东西都差不多」。当供给端高度同质，决胜的就变成了「谁能把价值送到对的人眼前，并让对方相信」——这恰恰是新媒体运营在做的事。

黎万强在《参与感》里讲过一句话：「新营销的第一步，让自己的公司成为自媒体」。原文已核实。这句话写在 2014 年，十二年过去，它已经从「公司的选择」变成了「个人的必答题」。一个律师不懂新媒体，案源就只能靠老客户转介绍；一个手工艺人不懂新媒体，作品就只能压在仓库里；一个求职者不懂新媒体，简历就只能躺在 HR 的邮箱里。关键不在「你要不要做网红」，关键在「你的价值能不能被看见」。不被看见的价值，在这个时代等于不存在。

罗振宇提出过「国民总时间」的概念——大意据得到 App。意思是每个人每天只有 24 小时，扣掉吃饭睡觉通勤，留给内容的时间是有限的，而且不再增长。所有平台、所有创作者、所有 App 都在抢这同一块不再变大的蛋糕。这带来一个直接后果：流量不是自然资源，是竞争性资源。你不主动去抢，你的潜在用户就被别人抢走。所以学新媒体运营，本质是学「在注意力稀缺的时代，如何让自己的价值被分配到」。

刘润在《底层逻辑》里给过一个公式：底层逻辑 + 环境变量 = 方法论。大意据《底层逻辑》。新媒体运营就是这样一个东西——它的根本（流量、信任、价值三件事）三十年没变，但它的环境变量（平台规则、算法权重、用户习惯）每年都在变。学新媒体运营，学的是把不变的规律套到变化的环境里去的那套手艺。

## 方法论：从认知到行动的三步转弯

第一步：把「做不做自媒体」换成「我的价值被多少人看见」。

很多人对新媒体运营的抵触，来自一个误解——以为学运营就是学当网红。黎万强在《参与感》里讲小米做自媒体的逻辑：不是为了涨粉，是为了「让用户参与进来」。大意据《参与感》。把这句话平移到个人身上：你做内容不是为了当网红，是为了让「你提供的价值」有一个被看见的入口。

可执行的动作是给自己做一次「价值盘点」。拿出一张纸，写三行：① 我比周围 80% 的人更懂什么；② 我做过什么别人愿意为之付费的事；③ 这两样如果只让我用一条 60 秒视频讲清楚，我会怎么讲。三行写不出来，说明你还没想清楚自己的价值——这恰恰是大多数人「想做自媒体却做不起来」的根本原因。粥左罗在《打造个人IP》里讲「认知差是内容硬通货」，大意据《学会写作》——你的认知水平高于读者，才能征服读者。盘点价值，就是先把「认知差」找出来。

第二步：把「打广告」换成「种内容」。

过去做生意的逻辑是「我有产品，我要打广告让人知道」。今天做生意的逻辑是「我先持续输出有价值的内容，吸引一群认同我的人，再让他们看到我的产品」。这两条路径的区别是：前者花钱买曝光，曝光一停流量就断；后者用内容换信任，内容沉淀下来流量持续来。黎万强在《参与感》里讲「做自媒体而非做广告」，原文已核实，指的就是这件事。

可执行的动作者是给自己定一个「最小内容承诺」——每周一条图文或一条短视频，连续做三个月。内容不必精致，但必须满足两个条件：① 围绕你盘点出的价值；② 对读者有具体用处（哪怕是「如何挑一双不臭脚的袜子」这种小事）。三个月后回头看，哪条数据好、哪条没人看，规律就出来了。董宇辉在东方甄选最初开播的半年，直播间只有几十个人，但他坚持用「讲书+讲产品」的方式做内容，等算法识别到这批内容的高完播率，流量就来了。内容是因，流量是果，顺序不能反。

第三步：把「等流量」换成「懂分配」。

平台规则就是新商业法则。过去开店靠地段——好的地段自带流量；今天起号靠算法——算法决定了你的内容被推给谁。理解这一点，你就不会把「做不起来」归因于「平台针对我」，而是会去问「我有没有读懂这个平台的分配规则」。

可执行的动作者是选定一个主战场，把它的规则吃透。抖音重前 3 秒留存和完播率，小红书重首图点击和收藏率，B 站重三连和弹幕密度，视频号重社交裂变，公众号重打开率和转发率——这些权重以各平台官方现行规则为准，且每年都在调整。先选一个，把它当下的规则研究到能复述给朋友听，比同时铺五个平台但每个都半懂要有效十倍。半佛仙人在公众号和 B 站都做到了头部，但他的起家是先把公众号的图文长文做到极致，再迁移到 B 站，不是一开始就五平台齐发。

## 避坑：学新媒体运营的三个典型弯路

第一个坑：把「学工具」当成「学运营」。

很多人花几千块报课学「如何用剪映」「如何用 CapCut 做转场」「如何用 ChatGPT 写文案」，学完还是做不起来。问题出在：工具是「形」，运营是「道」。会用剪映不会让你做出爆款，就像会拿笔不会让你写出好文章。黎万强在《参与感》里讲雷军定的七字诀「专注、极致、口碑、快」，原文已核实——这四个字里没有一个跟工具有关，全是认知和动作。先学道，再学形，顺序反了一定走弯路。

第二个坑：盯着粉丝数，忽略信任数。

新手最容易陷入的执念是「什么时候能涨到一万粉」。但凯文·凯利在「1000 True Fans」里算过那笔账：1000 个「愿意购买你任何作品」的铁粉，每人每年 100 美元，就够谋生。大意据《1000 True Fans》。一个 10 万粉但没人付费的账号，远不如一个 2000 粉但每次发售都有 300 人下单的账号值钱。董宇辉 3900 万粉的价值，不在于那个数字，在于这批粉对他「内容第一」调性的深度信任。盯着粉丝数做运营，会逼着你去做泛流量；盯着信任数做运营，才会逼着你去做精准人群。

第三个坑：把「做不起来」归因于「平台不行」。

「公众号红利没了」「抖音内卷太严重」「小红书被品牌占领了」——这些话每年都有人讲，每年也都有人在被讲「不行」的平台上从头做起来。吕白在《从零开始做内容》里讲过一句反常识的话：「爆款都是重复的，昨天火的选题明天还会火」。大意据《从零开始做内容》。意思是平台规则会变，但人性不变，能击中人性的内容结构永远有效。把做不起来的原因推给平台，就等于放弃了「我能不能更懂规则、更懂用户」这个唯一的改进抓手。罗振宇十年做 60 秒语音，从公众号时代做到视频时代，平台换了好几轮，他的内容底盘没换——因为底盘是信任，不是平台。

## 践行：本周可以做的四件事

一、做一次价值盘点。拿出一张纸，写完那三行：我比 80% 的人更懂什么、我做过的付费事项、60 秒怎么讲清楚。写不出来就再想一周，不要急着发内容。

二、选定一个主战场。不是五个平台齐发，是先选一个。选的标准是「你的目标用户在哪里活跃」——卖母婴用品选小红书，卖知识产品选公众号或视频号，卖实体低价货选抖音。选定之后，去这个平台的官方创作者学院（抖音巨量学、小红书创作者学院、B 站创作学院、微信公开课）把它当下的规则文档读完一遍。

三、定一个「最小内容承诺」。每周一条，连续三个月。内容围绕你盘点出的价值，对读者有具体用处。三个月后复盘数据，找出哪条跑得好、为什么。

四、把「粉丝数」从你的 KPI 里划掉，换成"本周有多少人和我产生了深度互动（评论超过 10 字、私信咨询、转发给朋友）"。这个数字比粉丝数诚实十倍。

学新媒体运营，学的不是当网红，是当数字时代的明白人——明白注意力怎么流动、信任怎么积累、价值怎么被看见。这三件事搞懂了，你做不做自媒体都受益一辈子。

## 参考来源

- 黄有璨《运营之光：我的互联网运营方法论与自白》：互联网三时代论（概念驱动→产品驱动→运营驱动）；运营的本质是连接产品与用户的价值纽带；做运营就像做人，长期主义与真诚是高阶运营的底色。
- 黎万强《参与感：小米口碑营销内部手册》：新营销的第一步，让自己的公司成为自媒体；互联网七字诀「专注、极致、口碑、快」；先做忠诚度再做知名度；做自媒体而非做广告。
- 罗振宇（得到 App / 罗辑思维）：国民总时间概念——时间是内容的终极稀缺资源，内容竞争本质是用户时间的零和博弈。
- 刘润《底层逻辑：看清这个世界的底牌》：底层逻辑 + 环境变量 = 方法论；长期主义与复利思维。
- 粥左罗《打造个人IP》《学会写作》：流量 = 时间 + 注意力；个人 IP = 信任资产 + 影响力；认知差是内容硬通货。
- 凯文·凯利《1000 True Fans》（2008 年博客原文，2016 年重写版收录进蒂姆·费里斯《巨人的工具》）：1000 个铁粉定理；铁粉数学；直接关系优于中介。
- 吕白《从零开始做内容》《小红书爆款规律》：爆款都是重复的；传播本质九要素。
- 半佛仙人（2025 新榜内容节演讲、新榜专访）：围绕「目标—成本—收益—风险」四维度做内容决策；从 0 到 60 万粉一年。
