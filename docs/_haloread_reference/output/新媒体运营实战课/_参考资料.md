## 使用建议

- 如需理解运营底层逻辑，可从黄有璨《运营之光》+乔纳·伯杰《疯传》+凯文·凯利1000铁粉理论入手
- 如需掌握某个平台，首推对应平台官方学院（巨量学/小红书商业学院/B站创作学院等）
- 如需理解爆款内容，吕白系列+STEPPS法则是核心
- 如需变现闭环，李撒欢《短视频变现》+各平台官方商单平台规则
- 如需AI赋能，余智鹏《DeepSeek帮你玩赚全媒体运营》+各平台AI内容标注规则
- 平台规则迭代快，凡涉及具体数字/阈值/政策，均以平台官方现行规则为准
