# 经历不经过反思只是经历

## 一、入戏：做了很多事，没长出多少本事

很多人一年的工作经历很丰富——做了几个项目、读了几十本书、试了几种新方法、上了几门课。可一年下来问自己"我比一年前多了什么本事"，常常答不上来。经历堆了一摞，本事没长几寸。

差距出在哪？出在经历和本事之间，缺了一道反思。经历本身不自动转化为能力，它只是给你提供了原材料；要把原材料加工成能力，得有一道工序叫复盘。中国学习者成甲在《好好学习》《好好思考》两本书里，把这道工序拆成了可操作的方法。这一篇讲怎么把反思做成日常，并用费曼加复盘的双循环，给整个抗遗忘体系收尾。

## 二、破题：反思是追问假设，不是记流水账

成甲提出了一个概念叫"临界知识"——大意是少数能跨领域复用的规律，比如复利、概率、二八、临界点，它们的杠杆远大于具体技巧。学知识要优先抓临界知识。这个判断和上一篇讲的迁移是同一个方向：能跨领域复用的，才是值得花力气打磨的。

但更关键的是他对反思的定义。成甲强调，反思日记不是记流水账，而是追问三件事：今天哪些假设被验证了或被推翻了？我用了什么思维模型？哪里可以做得更好？这三问把反思从"今天发生了什么"挪到"今天我的假设对不对"——前者是日记，后者才是反思。

为什么要追问假设？因为人做事不是凭空做的，背后都有一套"我这么做能成"的隐含判断。这套判断对不对，单看结果往往看不出来——你成功了，可能是假设对，也可能是运气好；你失败了，可能是假设错，也可能是执行问题。只有把假设摆出来、和结果对照、再调整，下一次的判断才会更准。反思的本质，是给你的假设做调试。

把这条和本专栏前面几篇连起来看：元认知让你看见假懂，卡片笔记让你记下真懂，知识系统让真懂互相连接，迁移让真懂飞到新领域。但所有这些，都需要一个持续校验的机制——否则你写在卡片上的"真懂"可能本身就是新的假懂，你迁移出去的原理可能本来就站不住。反思就是那个持续校验的机制。

## 三、方法论：费曼加复盘的双循环

把反思做成日常，可以用两个循环叠加。

第一个是费曼循环：学—讲—发现盲点—再学。每当你产生"我懂了"的感觉，别立刻放过它，用费曼技巧兜底——选一个概念，用大白话讲给一个不懂的人听，找出讲不清的地方，回到材料简化语言再讲。讲得清，说明这次是真懂；讲不清，就是新发现的假懂，回到材料去补。这是对"懂没懂"的校验。本专栏第二章已经讲过费曼法，这里它在抗遗忘里的角色，是给每一次"懂了"的感觉当质检员。

第二个是复盘循环：做—复盘—发现假设偏差—再做。每当你做完一件事，别立刻翻篇，用三问兜底：今天我用了什么假设？这些假设被验证了还是被推翻了？哪里可以做得更好？把答案简短写下来，第二天做下一件事时回头看一遍。这是对"对不对"的校验。

两个循环叠加，覆盖了学习的两个易错点。费曼循环治假懂——你以为懂了，讲不出来就露馅；复盘循环治盲动——你以为做对了，假设没验证就靠运气。前者校验知识，后者校验行动。每天用一次费曼、每天用一次复盘，等于给你的学习和做事各装一道质检工序。

具体怎么落地？给每天留两段十分钟。第一段在学完新东西后，挑一个概念用大白话讲一遍（讲不出来就回材料补）；第二段在睡前，写三句话回答复盘三问。两段加起来不到二十分钟，但它们会把"我以为我懂了"和"我以为我做对了"这两种最容易骗自己的感觉，分别按在地上检验一遍。

## 四、避坑：别把反思做成表演

反思也有自己的坑，最常见的有三种。

第一种是"流水账式反思"。把今天做了什么列一遍，再加一句"今天很有收获"或"明天继续努力"，就当反思完了。这种日记写一年也不会长本事，因为它没追问假设。判断标准很简单：你写的反思里，有没有至少一个被检验过的具体假设？没有，就是流水账。

第二种是"付费知识陷阱"。成甲在书里专门提醒：买课不等于学会，收藏不等于掌握。有的人每天听两节付费课、剪藏五篇好文章，反思时却只记"今天又学到了很多"。知识必须经过自己的加工与产出才算内化——你能不能用自己的话写出来、讲出来、用出来？不能，那就是把"接触"当成了"掌握"，是能力错觉的变体。

第三种是"反思的自我感动"。有的人反思写得文采飞扬、感情充沛，写完很满足，但下一次做事还是老样子。反思的目的是改变下一次的判断和行动，不是制造今天的好心情。判断标准是：明天遇到类似情境，你会因为今天的反思做出哪个不同的动作？想不出来，反思就没真正落地。

## 五、践行：把这一章收束成一个日常

一，从今晚开始，睡前用三句话写复盘：今天我用了什么假设？这个假设被验证了还是被推翻了？明天哪里可以做得更好？写在卡片或笔记里，给自己留一条可回看的痕迹。

二，每周挑一次"我懂了"的感觉，用费曼四步兜底。挑一个最近自认为搞明白的概念，讲给一个不懂的人听（哪怕是讲给空气），讲不下去的地方就是下周该补的盲点。

三，每两周回看一次这两周的复盘和费曼记录，问自己一个问题：哪些假设反复被推翻？哪些盲点反复出现？反复出现的，就是你的临界问题，值得用本专栏前面所有方法集中攻克——元认知看清它、卡片笔记记下它、知识系统连接它、迁移把它送出去。

四，把这一章的五篇当作一个完整的回路来用：第一篇教你识别假懂，第二篇教你把真懂写下来，第三篇教真懂彼此连接，第四篇教真懂飞到新领域，第五篇教这一切被持续校验。任何一篇缺位，回路都会漏气。

抗遗忘不是一次性的努力，是一套每天转一圈的循环。你不需要每天都做对所有事，但你需要每天让费曼和复盘各转一次，让假懂和盲动各被逮一次。一年下来，被你逮住的次数，就是你真正长出的本事。至此，从第二章费曼法的"学—教"两步，到这里加上复盘的"复盘"一步，"学—教—复盘"的闭环才算完整——这正是本专栏要送给你的一套终身学习操作系统。
