# 一顶帽子一个角度

## 一、入戏：三个人朝三个方向使劲

开会讨论一个方案，A 一开口讲优点，B 立刻接缺点，C 说再想想，三个人同时朝三个方向使劲，谁也听不进谁。半小时过去，优点没说透，缺点也没列全，剩下一肚子气。爱德华·德博诺（Edward de Bono）把这种场面诊断为"对抗式思考"——每个人选一个立场，互相消耗。

他在 1985 年的《六顶思考帽》里给了一个换法：别让所有人同时朝不同方向想，让所有人同时朝同一个方向想，然后一起换方向。一顶帽子一个角度，同一时刻全员同色。这听起来简单，妙处在于把"对抗"换成了"平行"。

## 二、破题：平行思维与六色角色

德博诺的核心概念是平行思维（parallel thinking）。传统讨论里，A 主张优点、B 主张缺点，双方各守一边互相消耗；平行思维让所有人先一起戴白帽看事实，再一起戴红帽说感受，再一起戴黑帽挑风险——同一时刻全员同色，每种模式都能充分展开，不必互相打架。

六顶帽子是六种角色，也是六种思维模式：

白帽，事实与数据。这顶帽子下只问"我们已知什么、还缺什么信息"，不带判断。

红帽，情感与直觉。这顶帽子下允许说"我就是觉得不对劲"，无需给理由。情感被允许出场，才不会在暗处干扰判断。

黑帽，批判与风险。这顶帽子下专挑毛病、估风险、找问题点。批判被收编到一个时段里，避免随时插话。

黄帽，乐观与价值。这顶帽子下专找好处、估收益、看机会。乐观也被收编到一个时段里，和批判对等。

绿帽，创造与替代方案。这顶帽子下提新想法、找替代路、做发散。黑帽挑出的毛病，到绿帽这里想办法绕开。

蓝帽，元控制。这顶帽子管流程本身——戴帽顺序、时间分配、收尾总结。蓝帽是"想思考这件事"的思考。

德博诺还有一条关键判断：大意据《六顶思考帽》，思考最大的敌人并非没想法，而是想法彼此打架；分离它们才能让每一种都被听见。人没法同时既乐观又批判，强制分离，每种模式才能跑满。试过同时既夸又骂一件事吗？脑子里两种声音打架，结果哪边都没想透。分离式思考把这个内耗拆开——乐观时专心乐观，批判时专心批判，每种模式都有它专属的几分钟。

## 三、方法论：蓝白红黄黑绿蓝走一遍

把六顶帽用起来，先掌握流程。一次完整的"戴帽"通常这样走：蓝帽开场，定议题和顺序；白帽先过事实，把已知和未知摆清；红帽让直觉出场，记录第一感受；黄帽找价值，把好处说透；黑帽挑风险，把问题列全；绿帽想替代方案，针对黑帽提出的问题找路；蓝帽收尾，总结决定和下一步。

每顶帽子戴几分钟即可，德博诺强调时间盒——避免无休止讨论。红帽尤其要短，三十秒到一分钟，直觉说完就过，别让它演成长篇辩护。

举个会议的例子。团队在讨论要不要上线一个新功能。蓝帽开场定议题和顺序（白—红—黄—黑—绿—蓝）。白帽三分钟：列出已知数据——用户量、开发成本、竞品情况，以及还缺的市场调研。红帽一分钟：每个人说一句第一感觉，"我兴奋"或"我不踏实"，不解释。黄帽三分钟：全员找好处——能带来什么用户、什么收入、什么口碑。黑帽三分钟：全员挑风险——技术风险、合规风险、与现有功能冲突。绿帽三分钟：针对黑帽提的风险想替代或缓解。蓝帽收尾：综合决定和下一步。整场不到二十分钟，该说的都说了，没人内耗。

戴帽顺序并非固定。德博诺建议多数场合白帽在前（先摆事实），但若团队情绪激烈，可先戴红帽让情绪出场再继续；若方案已基本成型、只想验风险，可直奔黑帽。蓝帽的责任就是根据情境定顺序，别机械套模板。

这套路也可以一个人用，做自我决策。比如要不要接一个新项目：白帽列出已知条件（时间、报酬、要求）；红帽说第一直觉（是兴奋还是发怵）；黄帽列收益（能学到什么、能认识谁）；黑帽列风险（占多少时间、会不会拖垮别的事）；绿帽想替代（能不能只接一部分、能不能换个时间）；蓝帽综合做决定。一个人也能把六种模式都跑一遍，比脑子里一锅粥强。

关键纪律是"同一时刻同色"。最容易破纪律的，是黄帽正说着好处，黑帽忍不住插嘴"但是……"。这种插嘴一出现，平行就退回对抗。蓝帽的责任就是守住纪律——谁违规，提醒谁换回去。

## 四、避坑：黑帽独大红帽被嫌

第一个坑，把六顶帽简化成"六种性格"。有人把黑帽当"悲观型人格"、黄帽当"乐观型人格"，开会时让"悲观的人"专戴黑帽、"乐观的人"专戴黄帽。这把工具用拧了。六顶帽是思维模式，每个人都该六色都能戴；固定角色等于放弃了平行思维的意义——又退回各守一隅。

第二个坑，黑帽独大。许多团队讨论一上来就黑帽，且黑帽时间最长，结果方案没说两句就被挑死。德博诺的设计里黄帽和黑帽是对等的，甚至很多场合建议先黄后黑——先看见价值，再评估风险，避免一棍子打死。

第三个坑，红帽被嫌弃。理性主义团队常觉得红帽"不专业"，跳过它。但直觉往往承载着没被说出来的经验判断，跳过红帽，它就在暗处影响黑帽黄帽的发挥。给红帽一分钟，让它光明正大出场，反而能净化后面的讨论。

第四个坑，蓝帽缺席。没有蓝帽管流程，戴帽顺序乱、时间超、收不了尾，六顶帽退化成自由发言。蓝帽不一定非得是领导，但必须有人担任，负责开场定序、中途守纪律、结尾总结。

## 五、践行：独自走一遍六顶帽

挑一个你最近要做的小决定（比如要不要报一门课、要不要换一种复习方法），独自走一遍六顶帽。找张纸分六栏，每栏写一种颜色，按蓝—白—红—黄—黑—绿—蓝的顺序走。

白帽那栏只写事实，不写判断；红帽那栏写第一感受，一两句即可；黄帽那栏至少写出三个好处；黑帽那栏至少写出三个风险；绿帽那栏针对每个风险想一个替代或缓解；最后蓝帽那栏写下你的决定和下一步动作。

走完一遍如果还决定不了，多半是白帽信息不够——你列不出足够事实。这时别在红黄黑绿里打转，退回白帽去补信息。蓝帽的价值就在这里：它能在卡住时诊断"卡在哪一帽"，然后对症下药。

走完你会发现自己更清楚了——这并非因为你得到了新信息，而是因为你把脑子里本来就在打架的几种声音分了类、排了队、各跑了一遍。德博诺这套的妙处正在于此：它不替你做决定；该被听见的声音都被听见了，决定自然就浮现。一顶帽子一个角度，把对抗换成平行，争辩就变成了协作。
