## 五、值得肯定的引用克制实践

1. **主动纠误**：复利「世界第八大奇迹」托名爱因斯坦（复利篇）、明托《金字塔原理》首版 1985 非 1973（MECE 篇、金字塔篇）、锤子钉子归属 Kaplan 1964/Maslow 1966 而非芒格（开篇、多元格栅篇、五种陷阱篇）、「我知道我一无所知」金句为西塞罗/第欧根尼转述非柏拉图原话（批判性思维篇）——均给出 Quote Investigator 或三方核验来源。
2. **史料层累交代**：道德经三版本对勘（反者道之动篇）、王阳明「致良知是学问大头脑」疑似后人概括声明不直引（致良知篇）、confirmation bias 术语从 Wason 1960 到 Nickerson 1998 的演变（五种陷阱篇）、DBTL 上溯 PDCA/Shewhart 环（DBTL 篇）。
3. **现代案例可考性**：马斯克电池案例（$600→$80/kWh，Kevin Rose 访谈）注明 SpaceX 2% 为媒体推算非亲口断言；贝索斯 2001 年请柯林斯到西雅图出自柯林斯《转动飞轮》引言亲述；马克斯 2015 年与芒格午饭出自 Oaktree 备忘录。
4. **古文直引格式规范**：所有古文直引均 ≤20 字，出处普遍精确到篇/章/条（如《传习录·上·徐爱录》第5条、《道德经·第四十章》王弼通行本、《中庸·第六章》朱熹分章）。
5. **大意据标注到位的范例**：逆向思维篇「如果我知道我会死哪里……」标「大意据芒格演讲转述」并交代原句为民间谚语；能力圈篇芒格意旨标「大意据《穷查理宝典》多处转述」。
