2. **奥卡姆剃刀术语首次使用年份**：文中标注 1852 年 Sir William Hamilton 首次使用「Ockham's Razor」。多源指向 Hamilton 著《Discussions on Philosophy** (1852)，但部分词源考证给出 1846 年或更早版本，建议在重版时进一步核对 OED 词源条目。

3. **巴菲特 99% 以上财富在 50 岁后获得**：文中已标注「基于公开财富数据」。媒体常用数据为 95%-99% 区间，精确数字因年份与汇率口径不同有出入。当前表述「99% 以上」与 Forbes 实时数据基本吻合（巴菲特 50 岁时财富约 1 亿美元，现财富约 1000 亿美元），属可接受范围。

4. **明托《金字塔原理》首版具体年份**：本专栏标注 1985 年（首版由 Pitman Publishing 出版），这是学界与麦肯锡校友网络公认年份。1973 年是明托离开麦肯锡的年份，常被误记为出版年。本专栏已正确处理。

5. **SpaceX 火箭原材料占售价 2% 上下**：文中已明确标注「媒体基于马斯克'原材料拆解'思路的二手推算，非马斯克本人精确表述，不同来源间数字有出入」，处理恰当，无需进一步动作。

6. **「如果我知道我会死在哪里，我就永远不去那个地方」**：文中标注「大意据芒格演讲转述，原句为民间谚语，芒格系引用者，精确出处未在原典逐字定位」，处理恰当。

## 六、亮点与好做法

本专栏在真实性维度有几处值得沉淀的好做法，建议在后续专栏中复用：

1. **误传黑名单主动澄清**：7 条误传全部正确处理，且大多在正文显式澄清（而非仅在文末注释）。如复利-爱因斯坦在正文第 25 行专门用一段讲清误传源流，菲茨杰拉德《崩溃》在正文明确写「不是很多人误传的《了不起的盖茨比》」。

2. **二手转述显式标注「大意据」**：所有非逐字原话的引文均标注「大意据《XX》」或「大意据《XX》转述」，与逐字原文严格区分。

3. **学界争议处加注**：如苏格拉底「我知道我一无所知」金句标注「后世转述，非柏拉图原话逐字」、波普尔对马克思/弗洛伊德的划界判定标注「学界拉卡托斯、库恩等人有争议」、达里奥未将方法命名为「贝叶斯」标注「本文系按思路类比」。

4. **王阳明引文精确到年份与篇目**：如「破山中贼易，破心中贼难」精确到《王阳明全集·卷四·文录一·与杨仕德薛尚谦书》明正德十三年正月（1518 年），「致良知」立为学问宗旨精确到正德十六年（1521 年），龙场顿悟精确到 1508 年。

5. **「致良知是学问大头脑，是圣人教人第一义」**：在致良知篇文末主动声明「措辞疑似后人概括，未在《传习录》原文逐字核到，本篇不作为直引使用」——这种「不确定就不直引」的姿态值得推广。

6. **奥卡姆剃刀溯源**：准确澄清「如无必要，勿增实体」并非奥卡姆原话（实为 1639 年约翰·庞奇首提），「奥卡姆剃刀」名称更晚（1852 年 Hamilton 首用）——这两层澄清在科普读物中极为少见，本专栏处理到位。

7. **MECE 缩写归属**：准确引明托麦肯锡校友访谈自述「it was Aristotle, but I was the first one to abbreviate it and apply it to analyzing groups of ideas」——既承认思想源头（亚里士多德），又肯定明托的操作化首创，处理精细。
