# 感光度：可用上限与降噪的真实策略

> 本篇覆盖 ISO 值（ISO）（感光度）的物理原理、90D 可用上限实测、降噪策略与 Auto ISO 设置；不覆盖曝光补偿与白平衡的搭配，那些在后续篇章展开。

## 一、入戏

第一次进演唱会场馆，光线暗，自动白平衡还压不住舞台灯的色偏，相机把 ISO 顶到了 12800。回家一看，照片全是彩色噪点，肤色发紫发绿，整组照片沦为"能看但没法精修"的废片。

第二次去学乖了，手动把 ISO 锁在 800。结果快门被压到 1/15 秒，歌手一动整个人糊成虚影。第三次再调，把 ISO 设 Auto 上限 6400，最低快门 1/250 秒，才算勉强出片。

这是 ISO 的本质矛盾：开高了画质崩，开低了画面糊，中间地带又窄又难找。新人最容易掉进两个坑——要么死守"ISO 100 出片最干净"的教条，宁可糊也不愿提一档；要么彻底放手"反正后期降噪"，结果后期怎么救都救不回来。

90D 是 3250 万像素的 APS-C 画幅机身，原生 ISO 100-25600，可扩展 H:51200。这台机身的 ISO 上限和降噪策略，需要单独讲清楚——它和全画幅、和早期 APS-C 都不一样。

## 二、破题

ISO（International Organization for Standardization，相机中特指传感器感光度）在数码时代的含义和胶片时代已经不同。胶片时代 ISO 是胶片本身的属性，买一卷就固定一种；数码时代 ISO 是传感器信号放大器的增益——传感器本身只有一种基准感光度（通常对应最低 ISO），其他 ISO 都是把这个信号放大得到的。

放大信号的同时也放大了噪声。这就是为什么高 ISO 会带来噪点：不是传感器变得更敏感了，而是把原本就存在的底噪放大了。

ISO 影响三件事：

第一是曝光量。每提高一档 ISO，传感器输出信号增益提高 2 倍，等效于增加一档曝光。

第二是动态范围。增益越高，传感器能容纳的最亮与最暗差距越小。ISO 100 时 90D 动态范围约 13.5 档（DxOMark），ISO 3200 时降到约 10.2 档，到 ISO 6400 时只剩约 8 档。这意味着高 ISO 不只是噪点增多，更是亮部更易过曝、暗部更易死黑。

第三是色深和色彩准确度。高 ISO 下色彩采样精度下降，肤色容易出现偏色，这是后期最难救的部分——和噪点不一样，噪点靠 AI 还能压下去，色偏很难还原。

理解这三点，你才能跳出"ISO 越低越好"的简化教条。真正的判断是：在保证快门和景深的前提下，ISO 越低越好；不能保证时，宁可提高 ISO 损失画质，也不要靠过慢的快门让画面糊掉——糊掉的废片，降噪救不回来。

## 三、方法论

### 90D 可用 ISO 实测上限

90D 在不同 ISO 档位下的实测表现如下（基于多源交叉验证）：

- ISO 100-400：画质优秀，噪点极低，色彩与色调过渡细腻。这是风光、人像精修的舒适区。
- ISO 1600：可用档位上限，阴影保留良好。这个档位下拍的 RAW 格式（RAW）（原始数据格式）文件，后期还能拉回 2 档阴影。
- ISO 3200：噪点开始明显，动态范围降至约 10.2 档。需后期降噪，但主体细节仍可保留。
- ISO 6400：RAW 仍可恢复，但需较激进的降噪，细节受损。这是弱光纪实和暗光人像的极限区。
- ISO 12800-25600：应急使用，细节损失明显。一般只用于"必须拍到"的新闻、纪实场合。
- H:51200：扩展档，画质崩坏，颜色几乎无法还原，仅作救命用。

把这套实测换算成实战决策：

- 白天户外风光、人像精修：ISO 100-200，不动。
- 阴天、室内自然光：ISO 400-800。
- 室内灯光、傍晚户外：ISO 1600。
- 室内演出、夜晚街景：ISO 3200，必要时上 6400。
- 暗光抓拍、演唱会：ISO 6400 上限，超出靠堆栈或 AI 降噪。

### 降噪策略：先预防，再后期

90D 的高 ISO 降噪有三个层次，从前期到后期：

**前期预防——关闭相机内高 ISO 降噪**。相机内的高 ISO 降噪默认是"标准"，它会直接涂抹 RAW 之外的细节，让 JPEG 看起来"干净"但实际损失锐度。RAW 拍摄者必须关闭，让降噪完全在后期做，否则等于双重损失。

**前期预防——关闭长曝降噪**。长曝降噪会让相机在每张长曝后拍一张等时长黑帧减噪，等于把曝光时间翻倍，且不利于堆栈。星空摄影时一定关闭。

**前期预防——堆栈降噪**。同一构图连续拍 10-20 张，后期用 Sequator（Windows 免费）、Starry Landscape Stacker（Mac）或 Photoshop "统计-平均值"对齐堆叠。原理是噪声是随机的，多张平均后噪声互相抵消，信号保留。信噪比提升约 √n 倍——10 张堆栈降噪约 3 倍，相当于把 ISO 6400 的画面拉回到 ISO 800 的水准。

**后期处理——AI 降噪**。Adobe Lightroom Classic 内置 AI Denoise、Topaz DeNoise AI、DxO PureRAW（DeepPRIME）可以在单张基础上显著降噪，效果接近甚至超过堆栈。代价是计算量大，一张照片处理 30 秒到 2 分钟。

**后期处理——传统降噪**。Lightroom 的"降噪"滑块（明度降噪 + 颜色降噪）是最常用的，但要注意：明度降噪高于 40 就开始糊细节，颜色降噪高于 50 会出现色块。配合蒙版，只对暗部和低频区域降噪，保留主体锐度。

### Auto ISO：让相机管 ISO，你管快门和光圈

90D 支持 Auto ISO，可设上限和最低快门。这是被严重低估的功能——很多人觉得 Auto ISO 是"懒人模式"，实际上它是 90D 在动态场景下最聪明的设置。

Auto ISO 的三个关键设置：

第一，设上限。90D 可设 100-25600 全范围，但实战中应该锁到 3200 或 6400——超过你愿意接受的画质底线，宁可欠曝后期拉。

第二，设最低快门。这是 Auto ISO 的灵魂。设到 1/125 秒，相机就会在快门掉到 1/125 之前一直保持低 ISO，掉到 1/125 之后才提 ISO。等于让相机在你画的红线内自动平衡。

第三，理解"最低快门"的实际含义。它不是绝对下限，而是相机优先保持的下限。当 ISO 已经到上限还曝光不足时，相机仍会继续放慢快门——这时 Av 模式下你会在取景器里看到快门值闪烁警告，提示你"已经到极限了"。

### 别人不会告诉你的点：90D 没有"神奇 ISO"，100 就是基准

网上流传一种说法："某相机有个'最干净 ISO'，比基准 ISO 还干净"。新手听了到处找这个神奇档位，结果反而没用好基础。

90D 的情况很明确：基准 ISO 是 100，画质最好。原生整数档（100/200/400/800/1600...）是传感器的原生增益档，画质最佳；中间档（125/160/250/320...）是通过数字增益在整数档之间插值出来的，画质略低于相邻整数档，不存在"中间档比基准档更干净"的情况。90D 也没有 ISO 50 扩展档——L:50 在 90D 上不存在，别去找。

90D 原生 ISO 范围 100-25600，扩展档只有 H:51200 一档。新手常犯的错是花时间找"神奇 ISO"，其实该花时间练的是曝光判断与快门优先级——把找"神奇 ISO"的精力省下来，去练曝光和构图，比纠结 100 还是 125 更干净有用得多。

## 四、避坑

### 反面案例一：死守 ISO 100 拍室内孩子

拍室内孩子的家人合影，坚持"画质至上"，ISO 锁 100。结果快门被压到 1/15 秒，孩子一动整个糊。后期怎么救都救不回来——糊掉的画面降噪压不回去。

正确的做法是：宁可 ISO 1600 略有噪点，也要保证快门 1/250 秒。噪点靠 AI 降噪能压回 60%，糊掉的画面压不回 1%。

### 反面案例二：开相机内"高 ISO 降噪 = 强"

进演唱会场馆前把"高 ISO 降噪"调到"强"，以为这样 JPEG 直出更干净。回家发现照片锐度严重下降，连头发丝都被抹平。RAW 文件反而没受影响——但 90D 默认会基于降噪后的数据进行预览，让 RAW 后期判断也失准。

正确做法：RAW 拍摄者关闭相机内高 ISO 降噪，JPEG 拍摄者设"弱"或"标准"，"强"档几乎从来不用。

### 反面案例三：星空摄影开长曝降噪

拍星空时长曝 20 秒，开了长曝降噪，结果相机每张要拍 40 秒（20 秒曝光 + 20 秒黑帧）。一晚原本能拍 200 张素材，只能拍 100 张，堆栈降噪的素材量减半。更糟的是，黑帧降噪打乱了拍摄节奏，星点位移明显。

正确做法：星空摄影一律关闭长曝降噪，改用堆栈降噪或后期 AI 降噪。

### 反面案例四：Auto ISO 上限设到 25600

把 Auto ISO 上限设到机身最大 25600，以为"给相机最大自由度"。结果在室内灯光下相机真的把 ISO 顶到 25600，照片全是彩色噪点，几乎不可用。

正确做法：上限锁到 3200 或 6400。超过这个值，宁可让相机放慢快门，至少还能后期靠堆栈补救。

## 五、践行

1. 用同一场景、同一构图、同一光线依次拍 ISO 100、400、800、1600、3200、6400、12800 七张，回家 100% 放大比较噪点、动态范围、色偏，找出你能接受的 ISO 上限。
2. 测试你的后期降噪流程：把 ISO 6400 那张分别用 Lightroom 传统降噪（明度 40、颜色 50）和 AI Denoise 处理，比较哪种保留细节更多。
3. 试一次堆栈降噪：夜晚固定三脚架，拍同一星空 20 张，用 Sequator 或 Photoshop "统计-平均值"堆叠，对比单张和堆栈结果的信噪比差异。
4. 设一次完整的 Auto ISO 配置：上限 6400、最低快门 1/125 秒，在光线变化的室内走一圈，看相机如何自动平衡 ISO 和快门。
5. 拍一组 ISO 100 和 ISO 160 的对比：在同一光线、同一构图下各拍 5 张，100% 放大比较暗部噪点，验证"原生 ISO"和"中间 ISO"的差异。
