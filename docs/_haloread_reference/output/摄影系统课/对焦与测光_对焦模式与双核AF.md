# 对焦模式与双核 AF：从单次到伺服的判断

> 本篇覆盖 One-Shot、AI Servo、AI Focus 三种对焦模式的判断逻辑、全像素双核 CMOS AF 原理与 Live View 眼部检测 AF 的适用边界；不覆盖对焦区域选择和追焦命中率细节，那是下一篇的主题。

## 一、入戏

第一次拍孩子跑步，老老实实按快门，回来一看十张九糊。以为是镜头跑焦，拿对焦测试卡一测，单次对焦准得不行。再回去拍孩子，这次盯紧取景器，发现按快门那一瞬间孩子已经跑出对焦平面——单次对焦锁了，但锁住的是按下快门前那一瞬间的位置，孩子已经不在那里了。

这是新人最容易踩的对焦坑：用错了模式。单次对焦（One-Shot）适合静止主体，连续对焦（AI Servo）适合运动主体，但 90D 选项里还有第三种 AI Focus，相机自动在两者之间切换——很多人觉得"自动最省心"，结果在复杂场景里被相机来回切换，命中率反而最低。

佳能 EOS 90D 有两套对焦系统：OVF（OVF）（光学取景器）下用 45 点全十字型相位对焦，Live View 下用全像素双核 CMOS AF。这两套系统的原理、性能、适用场景完全不同，搞不清楚就会一直抱怨"对焦不准"，其实是用错了系统。

## 二、破题

对焦模式只决定一件事：合焦后要不要继续跟随。

**One-Shot 单次自动对焦**：半按快门合焦后锁定，直到全按快门或松开手指。适合静止主体——风光、产品、合影、摆拍人像。优点是合焦精度高，对焦亮度范围 EV -3 至 18（中央点 f/2.8，ISO 100），暗光下也能合焦；缺点是合焦后主体一动就废。

**AI Servo 人工智能伺服自动对焦**：半按快门后持续对焦，跟随主体移动。适合运动主体——体育、儿童、动物、车流。优点是动态追焦；缺点是合焦判定靠预测算法，命中率不是 100%。

**AI Focus 人工智能自动对焦**：相机在 One-Shot 和 AI Servo 之间自动切换。理论上"两全其美"，实际是"两头不靠"。相机判断主体动还是静靠的是合焦平面变化幅度，反应有几帧延迟，常常在主体已经动起来时才切到 Servo，已经晚了。

判断逻辑很简单：

- 你能确定主体不动 → One-Shot
- 你能确定主体在动 → AI Servo
- 你不确定主体会不会动 → AI Focus（少数场景可用）

但实战中"不确定"的场景几乎不存在——拍孩子你肯定知道孩子在动，拍合影你肯定知道大家站着。所以 90% 的场景，AI Focus 都是错误选择。

## 三、方法论

### OVF 与 Live View：两套对焦系统怎么选

90D 有两套完全独立的对焦系统：

**OVF 对焦系统**：45 点全十字型相位对焦。45 个点全部是 f/5.6 十字型，其中 27 个支持 f/8（9 个十字型），中央点是 f/2.8 + f/5.6 双十字型。这套系统的优势是响应快、追焦稳、OVF 取景无延迟；缺点是对焦覆盖率只有取景器中央区域，边缘没有对焦点，拍偏构图时只能先对焦再构图，会引入焦平面位移误差。

**Live View 对焦系统**：全像素双核 CMOS AF（DPAF），最大 5481 个对焦位置，覆盖率水平约 88% × 垂直约 100%，对焦亮度范围 EV -5 至 18（f/1.2，中央点，One-Shot，ISO 100）。这套系统的优势是覆盖广、可触屏选点、支持眼部检测 AF；缺点是 Live View 取景有延迟、追焦不如 OVF 稳定、连拍速度从 10 fps 降到 7 fps。

选择标准：

- 拍体育、动物、街拍、需要快速反应 → OVF + AI Servo + 10 fps
- 拍人像、产品、需要精确选点 → Live View + DPAF + 触屏
- 拍视频 → Live View + DPAF（视频只能用 Live View）

### 全像素双核 CMOS AF 原理

DPAF 是 90D 在 Live View 下对焦又快又准的关键。原理是这样的：

90D 的传感器上每个有效像素都由两个独立的光电二极管组成，分别位于像素的左右两侧。这两个二极管同时收集光线，但因为有微小的左右位置差，它们看到的画面有微小的相位差——和 OVF 的相位对焦是同一个原理。

合焦时，相机同时读出这两个二极管的信号，计算相位差，就能瞬间判断焦前还是焦后、差多少。整个过程比传统对比度检测（contrast detect）快得多，几乎和 OVF 相位对焦一样快。

合焦完成后，两个二极管的信号合并成一个完整的像素信号用于成像。所以 DPAF 不会损失像素质量，每个像素都既能对焦又能成像。

这套系统的实战含义是：90D 在 Live View 下的对焦速度远快于早期单反的 Live View，几乎可以替代 OVF 用于静态拍摄。但有一个例外——AI Servo 追焦时，DPAF 仍然不如 OVF 稳定，因为传感器读出速度比反光板相位检测慢，对快速横移的主体反应会有几帧延迟。

### Live View 眼部检测 AF

90D 在 Live View 下支持眼部检测 AF，One-Shot 和 Servo 模式都支持。这个功能对人像摄影是革命性的：

- 触屏点击主体脸部 → 相机自动追踪到眼睛 → 锁定眼部合焦
- 主体移动 → 相机持续跟踪眼部
- 不需要手动移动对焦点，不需要先对焦再构图

实战场景：

- 拍户外人像，模特在场景里走动，眼部检测持续跟踪
- 拍儿童抓拍，孩子乱动也能保证眼部合焦
- 拍会议合影，多张连拍保证每个人眼部都清楚

但眼部检测有限制：

- 主体眼部要清晰可见，戴口罩、墨镜、侧脸都可能识别失败
- Live View 取景有延迟，快速运动的主体对焦比 OVF 慢
- 弱光下识别率下降，ISO 6400 以上经常丢眼部

### AF 微调：解决镜头跑焦

90D 支持 AF 微调（AF Microadjustment），±20 步，可调全部镜头统一偏移，也可调最多 40 支镜头按序列号单独记忆。这是解决"镜头跑焦"的关键功能。

镜头跑焦的典型表现：用最大光圈拍人像，鼻子清楚眼睛糊，或者反过来。原因是镜头机械装配有公差，相位对焦模块和实际成像平面有几微米到几十微米的偏差。

AF 微调步骤：

1. 三脚架固定相机，对焦测试卡（或一把刻度尺）45 度斜放。
2. 用最大光圈、最近对焦距离拍一张，看合焦点偏前还是偏后。
3. 进入菜单"AF 微调"，调 +5 或 -5，再拍一张对比。
4. 反复调整直到合焦点落在标尺目标位置。
5. 保存为该镜头的微调值，相机会按序列号记住。

但要注意：

- AF 微调只对 OVF 相位对焦有效，Live View DPAF 不需要也不影响。
- 变焦镜头广角端和长焦端可能跑焦方向不同，要分别测试。
- 微调过一次后，镜头装到其他 90D 机身上可能要重新调。

### 别人不会告诉你的点：OVF 中央双十字的隐藏代价

90D 的 OVF 中央点是 f/2.8 + f/5.6 双十字型，比其他 44 点多了一组 f/2.8 横向检测线。这意味着中央点在使用 f/2.8 或更大光圈镜头时，精度比其他点高得多。

但很多新手不知道的是：这个双十字只在 f/2.8 及以上光圈下激活。如果你用 f/4 的镜头（如 EF 50mm f/1.4 收到 f/4，或 EF 24-105mm f/4L），中央点退化为单十字，精度和其他点一样。要用满双十字，镜头最大光圈必须 ≥ f/2.8。

实战含义：拍需要最高对焦精度的场景（如商业人像、产品微距），尽量用 f/2.8 或更大光圈镜头，并且把对焦点选在中央，先对焦再构图。不要用 f/4 镜头还要追求"中央双十字的精度"——那时候中央点和其他点没区别。

## 四、避坑

### 反面案例一：所有场景都用 AI Focus

第一次拍孩子跑步用 AI Focus，相机判定"主体静态"用 One-Shot，孩子一动就废；判定"主体动态"切 Servo，但切换有延迟，已经丢了。整组照片对焦命中率只有 30%。

正确做法：拍孩子明确用 AI Servo，不要"自动切换"。

### 反面案例二：拍合影用 Live View 眼部检测

拍三人合影，开了 Live View 眼部检测，结果相机只对到中间那个人，左右两人虚化。原因是眼部检测一次只跟踪一只眼睛。

正确做法：多人合影改用 OVF 单次对焦 + 单点对焦，对焦到中间人物，f/8 保证景深。眼部检测适合单人。

### 反面案例三：用 AF 微调后镜头换机身忘了重测

在 90D 上给 EF 50mm f/1.4 调了 -5 微调，镜头借给朋友在 80D 上用，全跑焦。原因是不同机身的相位对焦模块装配公差不同，微调值不通用。

正确做法：每支镜头在每台机身上单独测试一次微调。镜头出借、归还、上维修保养后都要重测。

### 反面案例四：拍视频用 OVF 设置

拍视频时还用 OVF 模式下设置的 AI Servo + 大区域 AF，结果视频里对焦频繁拉风箱。原因是视频必须用 Live View + DPAF，OVF 相位对焦在反光板抬起时根本不工作。

正确做法：拍视频切到视频模式，对焦设置独立保存为视频用的 C1/C2/C3 自定义模式。

## 五、践行

1. 同一静止主体（花瓶、玩偶）用 One-Shot 和 AI Servo 各拍 10 张，比较合焦精度。你会发现 One-Shot 在静止场景下精度更高。
2. 找一个跑步的家人或朋友，分别用 AI Focus、AI Servo + 单点、AI Servo + 大区域三种方式各拍 20 张，统计命中率，验证"AI Focus 是最差选择"。
3. 拍 5 张人像，分别用 OVF 中央单点和 Live View 眼部检测，比较构图自由度和合焦精度。
4. 用三脚架 + 对焦测试卡，给你的常用镜头做一次 AF 微调，记录微调值贴在镜头盖上。
5. 拍一组视频：用 Live View + DPAF + 眼部检测拍一段 30 秒人像，看眼部跟踪是否连续。
