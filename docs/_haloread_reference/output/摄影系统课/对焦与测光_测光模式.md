# 测光模式：评价到点测光的判断边界

> 本篇覆盖评价测光、局部测光、点测光、中央加权四种测光模式的原理与适用边界，以及 90D 的 216 区测光传感器；不覆盖曝光补偿与 AEB 包围（AEB）（自动包围曝光）的具体操作，那是下一篇的主题。

## 一、入戏

第一次拍逆光人像，用评价测光，相机看整张画面：天空占大部分，亮得很，于是把曝光压暗，结果人成了一个黑剪影。第二次学乖了用点测光，对准人脸测光，结果人脸清楚了，但天空白成一片完全过曝。

测光模式决定相机"按什么来曝光"。同样的画面，不同测光模式给出的曝光值能差 2-3 档。新人最常踩的坑是"全程用评价测光"，结果是大部分场景都凑合，但任何逆光、舞台、雪地、沙滩场景全都翻车。

90D 在 OVF（OVF）（光学取景器）下用 22 万像素 RGB+IR 测光传感器，216 区（18×12 矩阵）划分画面，比早期单反的 63 区精细得多。但精细的传感器不会自动解决"按什么测光"的问题——模式选错，再精细的传感器也救不回。

## 二、破题

测光（Metering）是相机测量画面亮度、给出建议曝光值的过程。90D 用一个独立的 22 万像素 RGB+IR 测光传感器（OVF 模式下），把画面分成 216 个小区域分别测量，然后根据所选测光模式给出综合曝光值。

测光模式只决定一件事：怎么把 216 个区域的亮度合成一个曝光建议。

**评价测光（Evaluative Metering）**：把 216 个区域全部加权考虑，并对当前对焦点附近加权更高。逻辑是"假设主体在对焦点附近，整张画面要平衡"。优点是大多数场景都凑合，是默认模式；缺点是高反差场景（逆光、雪地、舞台聚光）会被大面积亮区或暗区误导。

**局部测光（Partial Metering）**：只测量画面中央约 6.5% 的圆形区域。逻辑是"主体在画面中央，背景不重要"。适合主体在中央且背景反差大的场景，比如逆光人像主体占中央、舞台聚光下的演员。

**点测光（Spot Metering）**：只测量画面中央约 2.0% 的小圆点。逻辑是"我要精确控制这块小区域的曝光"。适合精确控制主体关键部位（如人脸、白雪、黑色西装）的曝光。点测光是手动曝光的最好朋友，但也最容易翻车——点偏一点，曝光就差几档。

**中央加权平均测光（Center-Weighted Metering）**：测量整张画面，但中央区域加权更重。逻辑是"主体大概率在中央，但整张画面也要看"。这是胶片时代的标准模式，现在主要用于拍合影、街拍这类主体在中央但场景不太极端的情况。

### 测光范围的物理含义

90D OVF 模式的测光范围是 EV 1-20（50mm f/1.4，ISO 100），Live View 模式是 EV -2 至 20。

EV 1 大约相当于"很暗的室内，只能勉强看到东西"，EV 20 是"正午阳光直射雪地"。EV -2（Live View 专属）大约是"月光下的景色"——比 EV 1 还暗 3 档。

实战含义：在 EV 1 以下的极端暗光下，OVF 测光会失效，相机给不出准确曝光值，必须切到 Live View 测光，或手动估算曝光。这种情况在拍星空、深夜街景时常见。

## 三、方法论

### 评价测光：默认但不是万能

评价测光是 90D 的默认模式，也是 80% 场景下最省心的选择。它的逻辑是"对当前对焦点附近加权更高"，意味着：

- 你对焦点选在哪里，相机就认为主体在哪里，曝光偏向那里
- 整张画面的其他区域也参与计算，但权重低

这个模式适合：

- 风光（光比均匀，主体分散）
- 街拍（主体不确定，整张画面都要看）
- 顺光人像（光比不大，主体在中央）
- 大部分日常场景

但评价测光在以下场景会失准：

- 逆光人像（背景大面积天空，相机被亮区压暗）
- 雪地、沙滩（大面积白色，相机误以为过曝而压暗，结果主体欠曝）
- 舞台聚光（主体小而亮，背景大面积黑，相机被暗区误导而提亮，结果主体过曝）
- 黑色西装、黑色汽车（大面积黑色，相机误以为欠曝而提亮，结果主体过曝）

### 局部测光：主体的"中间地带"

局部测光只看画面中央 6.5% 的区域，相当于把对焦点周围一小块画面单独测光。逻辑是"主体在中央，整张画面其他部分忽略"。

适用场景：

- 逆光人像主体占中央（背景天空大面积，主体在中央，评价测光会被天空误导，局部测光只看主体）
- 舞台聚光下的演员（演员占画面中央，背景黑）
- 拍花、拍特写（主体占中央，背景虚化或杂乱）

局部测光的使用要领是：先把主体放在画面中央半按快门测光，按住 AE Lock 锁定曝光，再移动相机构图。这样能保证测光准确又构图自由。

### 点测光：精确控制的两面性

点测光只看画面中央 2.0% 的小点，是最精确也是最容易翻车的测光模式。

点测光的真正用法是"测灰卡"——对一块 18% 中性灰卡测光，相机给出的曝光值就是把这个灰卡拍成中性灰的曝光。如果你知道主体的反光率（白雪约 +2 EV，黑色西装约 -2 EV，人脸约 +1 EV），就能精确控制每个区域的曝光。

但点测光的陷阱是：

- 测点偏离 1 度，曝光就差几档
- 主体反光率非中性灰时，相机仍按 18% 灰曝光，主体会被拍成中性灰（白雪变灰雪，黑色西装变灰西装）

实战推荐：

- 拍雪景：点测光对准白雪，曝光补偿 +1.5 到 +2 EV（白加黑减）
- 拍黑色西装：点测光对准西装，曝光补偿 -1.5 到 -2 EV
- 拍逆光人像：点测光对准人脸，曝光补偿 +0.5 到 +1 EV
- 拍舞台演员：点测光对准演员脸部，曝光补偿看光线情况

### 中央加权平均测光：胶片时代的遗产

中央加权是胶片时代的标准测光模式，把整张画面加权但中央权重更高。这个模式现在主要用于：

- 拍合影（主体在中央但场景需要平衡）
- 拍街拍（主体大概率在中央但场景复杂）
- 老法师的习惯（用惯了不想换）

90D 上中央加权的作用已经被"评价测光 + 对焦点加权"覆盖了大部分，所以这个模式实际上是"评价测光的简化版"，新场景几乎不需要。

### Live View 测光的差异

90D 在 Live View 下用 384 区测光，区域比 OVF 的 216 区更细。但更重要的是覆盖范围不同：

- Live View 局部测光约 4.5%（OVF 是 6.5%）
- Live View 点测光约 2.6%（OVF 是 2.0%）
- Live View 测光范围 EV -2 至 20（比 OVF 多 3 档暗光）

这意味着 Live View 在暗光下测光更准，且点测光覆盖更大、更容易命中主体。但 Live View 取景有延迟，运动场景不适合。

实战推荐：拍风光、人像精修用 Live View + 触屏选点测光，精确度高；拍体育、动物用 OVF + 评价测光，反应快。

### AE Lock：测光后构图的关键

90D 的 AE Lock（曝光锁定）功能在 One-Shot AF + 评价测光下完成对焦时自动锁定，创意区模式可手动锁定（按机背 * 键）。

AE Lock 的实战用法是"先测光、锁定、再构图"：

1. 把测光模式设为局部或点测光
2. 把主体放在画面中央半按快门测光
3. 按住 AE Lock 锁定曝光值（取景器显示 * 标志）
4. 移动相机构图
5. 完成对焦，全按快门

这种拍法在逆光、舞台、雪地等极端场景下能保证曝光准确，比依赖曝光补偿更可控。

### 别人不会告诉你的点：评价测光的对焦点加权是双刃剑

90D 评价测光的核心机制是"对焦点附近加权更高"。这个机制在大部分场景下是优势——你对焦在哪里，相机就认为主体在哪里，曝光偏向那里。

但这个机制在以下场景反而是陷阱：

**对焦点偏离主体关键区域**。拍人像时对焦点常常选在眼睛附近，但眼睛周围可能是黑头发、黑衣服，评价测光会被这块暗区误导而提亮，结果整张画面过曝，人脸高光溢出。

**多对焦点场景**。AI Servo + 大区域追焦时，对焦点可能在画面里跳动，评价测光的加权区域也跟着跳，导致连续几张曝光值差异很大。

实战推荐：

- 拍静态人像精修：改用点测光 + AE Lock，对准脸部测光锁定，再构图
- 拍动态追焦：用 M 模式手动设曝光值，避免测光随对焦点跳动

## 四、避坑

### 反面案例一：拍逆光人像用评价测光

逆光拍人像，相机看整张画面，天空占大部分且亮，于是把曝光压暗，人成了黑剪影。

正确做法：用局部测光或点测光，对准人脸测光 + 曝光补偿 +1 EV。

### 反面案例二：拍雪景不补偿

拍雪山雪地，相机被大面积白色误导，认为过曝而压暗，雪变成灰雪。

正确做法：用点测光对准白雪 + 曝光补偿 +1.5 到 +2 EV（白加黑减）。

### 反面案例三：点测光点偏一点

拍舞台演员用点测光，本来想测脸部，结果点偏到了演员黑色衣服上，相机以为欠曝而提亮，演员脸部过曝死白。

正确做法：点测光要精确瞄准关键部位。90D 点测光范围只有 2.0%，要确保测点完全在目标上，必要时用 Live View 放大确认。

### 反面案例四：拍星空用 OVF 评价测光

拍星空时用 OVF + 评价测光，结果画面全是黑的，相机给出的曝光值是 1 秒 ISO 25600，星点稀疏且噪点爆炸。

正确做法：星空摄影 EV 1 以下，OVF 评价测光失效。切到 Live View 测光（EV -2 范围）或直接 M 模式手动设曝光（ISO 3200、10 秒、最大光圈）。

## 五、践行

1. 找一个逆光人像场景，分别用评价测光、局部测光、点测光各拍 3 张，比较三种模式下人脸和天空的曝光差异。
2. 找一片雪地或白墙场景，用评价测光拍一张，再点测光 + +2 EV 拍一张，比较哪种雪更白。
3. 练习 AE Lock 流程：把测光模式设局部测光，对准主体测光，按 * 锁定，再移动相机构图，全按快门。拍 10 张体会这种"先测后构"的节奏。
4. 找一个高反差场景（如逆光下的人物半身），分别用 OVF 评价测光和 Live View 点测光拍 5 张，比较哪种更准确。
5. 拍一组夜景街灯，看 90D 在 EV 1 以下暗光的测光表现：用 OVF 评价测光和 Live View 评价测光各拍 3 张，比较哪种更接近肉眼感受。
