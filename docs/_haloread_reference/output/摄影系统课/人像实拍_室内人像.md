# 室内人像：窗户光与高感弱光策略

> 本篇覆盖室内人像的窗户光运用、家居场景布光、低光高感策略、EF-S 18-135 旅游镜室内限制；不覆盖影棚多灯布光方案（商业变现与影棚人像篇另述）与闪光灯离机引闪细节（闪光灯与人造光 篇展开）。

## 一、入戏

新手对室内人像的普遍误解是「室内光线稳定，比户外好拍」。恰恰相反——室内光线比户外复杂得多。户外的光是太阳一个光源，方向与强度可预测；室内的光是窗户光、室内灯、反射光多个光源混合，每个房间的光都不一样，每移动一米光就变了。室内人像的真正难点是「读懂这个房间的光」，然后决定机位、参数、是否补光。

但室内有一个户外没有的优势——窗户光。北向窗户的散射光被摄影界公认为「最经典的人像光」，是伦勃朗光的原型——17 世纪荷兰画家伦勃朗就是用画室北窗的散射光画出了那些经典的肖像。窗户光的好处是柔、有方向、可控——光从一侧来，主体另一侧有阴影，立体感天然形成；纱帘可以再柔化，反光板可以补阴影侧，整套方案不需要任何闪光灯。一只 90D + 一只 EF 50mm f/1.8 STM + 一块反光板，在任何一个有窗户的房间里，都能拍出与棚拍接近的质感。

本篇要讲的，是窗户光的运用方法、家居场景的布光思路、90D 在低光下的高感策略（ISO 1600 可用 / 3200 噪点 / 6400 可恢复）、EF-S 18-135 旅游镜在室内的真实限制，以及如何在不开闪光灯的前提下，把家里的客厅、卧室、书房拍出棚拍感。

## 二、破题

室内人像的光线问题集中在三个变量：光的强度、方向、色温。三个变量都受窗户影响。

**强度：决定 ISO 与快门。** 室内自然光比户外弱 4-6 档——晴天户外 ISO 100、1/200 秒；室内靠窗 1-2 米距离，同样 ISO 100 需要快门 1/15-1/30 秒，远到 3-4 米距离可能需要 1/4 秒。这意味着室内手持拍摄必须拉高 ISO 或用大光圈镜头。90D 在 ISO 1600 噪点可控是可用档位上限，ISO 3200 噪点开始明显需后期降噪，ISO 6400 RAW 仍可恢复但需较激进降噪细节受损，ISO 12800-25600 应急使用细节损失明显。

**方向：决定立体感。** 窗户光的方向由窗户与主体的相对位置决定。侧窗光（主体侧对窗户 45°）是最美的人像光——光从一侧打来，主体另一侧有阴影，立体感天然形成，脸颊鼻梁一侧有亮、一侧有暗，这种光就是伦勃朗光的雏形。顺窗光（主体背对窗户，相机面对窗户）是逆光——头发与肩膀有轮廓光，脸部欠曝需补光。逆窗光（主体面对窗户，相机背对窗户）是顺光——脸部均匀照亮但缺立体感，是新手最常见的失败模式。

**色温：决定肤色与氛围。** 窗户光的色温受三因素影响——室外天气（晴天约 5500K、阴天约 6500K、黄昏约 3000K）、窗户朝向（北窗光稳定偏冷、南窗光偏暖变化大、东窗上午光、西窗下午光）、室内反射（白墙反射准确、黄墙偏暖、绿墙偏冷）。室内灯的色温与窗户光不同——白炽灯约 2700K-3200K 暖黄、LED 灯约 3000K-5000K 多变、节能灯约 4000K-5000K。两种光混合时画面会有色温分裂——窗户光侧脸偏蓝白，室内灯侧脸偏黄，需要关掉室内灯只用窗户光，或者后期分色温校正。

**EF-S 18-135 旅游镜的室内限制。** 旅游一镜走天下在户外表现优秀，在室内有明显限制——f/3.5-5.6 的最大光圈太小，室内弱光下要拉高 ISO 到 3200-6400 才能用 1/125 秒快门，画质受损明显。室内人像建议优先用 EF 50mm f/1.8 STM（等效 80mm，f/1.8 大光圈）—— ISO 400 仍能用 1/125 秒，画质优秀。50mm 在室内空间稍紧（80mm 等效焦段要退 2-3 米拍半身），但画质与虚化优势远大于 18-135 的便利性。

记住一个判断口诀：**窗户决定方向，光强决定 ISO，色温决定肤色，镜头决定画质。** 四者顺序不可颠倒——窗户方向不对，其他都白搭。

## 三、方法论

### 窗户光的四种基本布光模式

窗户光按主体与窗户的相对位置分四种模式，每一种有不同的视觉效果。

**侧窗光（45° 侧光）：最美的人像光。** 主体侧对窗户 45°，相机在窗户与主体之间。光从一侧打来，主体另一侧脸颊有阴影，鼻梁在脸中央形成明暗分界，立体感天然形成。这是伦勃朗光的原型——17 世纪荷兰画家伦勃朗用画室北窗的这种光画出《夜巡》等经典肖像。阴影侧可用反光板补光，控制光比在 3:1 到 4:1 之间——这是商业人像最常用的光比。

**逆窗光（逆光）：发丝光与轮廓光。** 主体背对窗户，相机面对窗户。光从主体背后打来，头发与肩膀形成金色或银色轮廓光，脸部处于阴影。这种光的氛围感最强——发丝发光的画面很有「电影感」。但脸部欠曝需要补光——反光板补阴影侧，或者用闪光灯 HSS 高速同步补光主体，或者直接拍成剪影（剪影与环境人像 篇展开）。

**顺窗光（顺光）：均匀但缺立体感。** 主体面对窗户，相机背对窗户。光从相机方向打来，脸部均匀照亮但没有阴影，立体感弱，画面偏平。这种光是新手最常见的失败模式——拍出来的人像没有立体感，像证件照。除非特殊需要（如拍证件照、翻拍），不建议用顺窗光。

**远窗光（散射光）：柔而弱。** 主体远离窗户 3-5 米，光经过多次反射后变成柔光，强度大幅下降。这种光适合拍柔光人像——女性、儿童肖像的柔光感最好。但远窗光强度弱，需要拉高 ISO 或用大光圈镜头。

### 北窗光与南窗光的差异

摄影界公认「北窗光是最经典的人像光」，原因是北向窗户（北半球）的阳光是天空散射光，没有直射阳光——光柔、稳定、一天里色温与强度变化小。北窗光适合任何时段拍摄，是室内人像的首选。

南窗光（北半球）的阳光有直射——上午到下午都有阳光射入，强度与方向变化大。南窗光的优点是强度高，可以用低 ISO 拍摄；缺点是变化大，每 30 分钟光的方向与强度都变，且正午可能有直射阳光射入，形成硬光斑——硬光斑下人像会形成局部过曝与硬黑阴影，破坏画面。南窗光的解决方法是用纱帘柔化——纱帘把直射阳光变成柔光，效果接近北窗光但强度更高。

如果家中只有南窗，建议上午 9-11 点或下午 14-16 点拍摄——这两段时间太阳角度斜，阳光经纱帘柔化后光质最柔。正午 11-14 点的南窗阳光太硬，避开。

### 家居场景的布光思路

家中不同房间的光线特征不同，每个房间适合拍不同的人像风格。

**客厅：环境人像与半身人像。** 客厅空间最大，适合拍半身到全身环境人像。客厅通常有大窗户或落地窗，侧窗光充足。家具（沙发、椅子、茶几）可作道具——模特坐沙发上侧对窗户，是最经典的家居人像构图。客厅的挑战是杂物多——电视、装饰、玩具容易入画，需要清理或用大光圈虚化。

**卧室：亲密感与柔光。** 卧室空间小但氛围亲密，适合拍情绪人像——模特在床上、靠墙、看窗外。卧室的窗户光通常从一侧来，是天然的侧窗光。床单、枕头、被子可作道具与色块。卧室的挑战是床占空间大，机位受限，建议用 35mm 等效 56mm 或 50mm 等效 80mm 拍半身特写。

**书房：知识感与情绪。** 书房有书架、书桌、台灯，适合拍「她在阅读」的环境人像——模特靠书架、翻书、写字。书房的台灯可作为补光——台灯色温暖（约 2700K-3200K），与窗户冷光形成色温对比，但要注意色温分裂——台灯侧脸偏黄、窗户侧脸偏蓝，可以关掉台灯只用窗户光，或者后期分色温校正。

**厨房：生活感与质感。** 厨房有水龙头、瓷砖、餐具，适合拍「她在做饭」的生活人像——模特切菜、洗碗、煮咖啡。厨房的挑战是空间小、光线弱（多数厨房无大窗），需要拉高 ISO 到 1600-3200 或用大光圈镜头。

### 90D 在低光下的高感策略

90D 是 APS-C 画幅 32.5MP，像素密度高，高感弱的真实表现（来自器材参数表）：

| ISO | 评价 |
|---|---|
| 100-400 | 画质优秀，噪点极低 |
| 1600 | 可用档位上限，阴影保留良好 |
| 3200 | 噪点开始明显，需后期降噪；动态范围降至约 10.2 档 |
| 6400 | RAW 仍可恢复，但需较激进降噪，细节受损 |
| 12800-25600 | 应急使用，细节损失明显 |

基于这个表，室内人像的高感策略是：

**优先保 ISO 100-400**：用大光圈镜头（EF 50mm f/1.8 STM）在窗户光充足处拍摄，ISO 100-400 画质优秀。这是室内人像的首选档位。

**ISO 1600 是上限**：当光线弱（如远离窗户 3 米、阴天、傍晚），用 ISO 1600 仍能保持画质，是「可用档位上限」。

**ISO 3200 谨慎使用**：噪点开始明显，需后期降噪。建议同时拍 RAW 格式，后期用 Lightroom 降噪或 AI 降噪（Lightroom AI Denoise、Topaz DeNoise）。

**ISO 6400 仅 RAW 应急**：噪点明显但 RAW 仍可恢复，需较激进降噪。这种档位下 JPEG 直出基本不可用，必须 RAW 后期。

**ISO 12800-25600 不建议**：细节损失明显，只能应急——如重要瞬间抓拍，画质次要。室内人像不建议用这个档位。

90D 的动态范围在 ISO 100 约 13.5 档（DxOMark），是最佳画质档位。室内尽量保 ISO 100-400，画质才优秀。

### EF-S 18-135 旅游镜的室内限制与替代方案

EF-S 18-135mm f/3.5-5.6 IS USM 在户外的旅游一镜走天下优势，在室内变成明显劣势——f/3.5-5.6 的最大光圈太小，室内弱光下要拉高 ISO 才能用 1/125 秒快门。

室内人像的替代方案：

**首选 EF 50mm f/1.8 STM**：等效 80mm，f/1.8 大光圈，室内 ISO 400 仍能用 1/125 秒。这只镜头约 1200-1600 元，是室内人像的性价比之王。缺点是定焦，要靠走动改变构图，室内空间紧时退不开。

**进阶 EF 35mm f/2 IS USM 或 EF-S 24mm f/2.8 STM**：等效 56mm 或 38mm，室内空间紧时退得开。35mm f/2 比 50mm f/1.8 慢一档，但有 IS 防抖，弱光下可用 1/15 秒仍清晰。

**EF 85mm f/1.8 USM**：等效 136mm，f/1.8 大光圈，室内空间足够时（如大客厅）可用，虚化更强。但 136mm 在小房间退不开，是空间充裕时的进阶选择。

**EF-S 18-135 仅作广角端使用**：在客厅等大空间用 18mm 等效 29mm 拍环境人像（带场景），光圈 f/3.5 在 ISO 1600 下能用 1/60 秒（IS 防抖救活）。但半身人像与特写不要用 18-135，画质与虚化都不够。

### 别人不会告诉你的点：室内人像的「白卡校正法」

资深人像摄影师有一个心法——室内人像最大的后期难题不是噪点，是色温分裂。窗户光（约 5500K-6500K）与室内灯（约 2700K-4000K）色温不同，两种光混合时画面会出现色温分裂——窗户侧脸偏蓝白，室内灯侧脸偏黄。后期整体调色无论怎么调，都会让一侧肤色不自然。

解决方法是「白卡校正法」——拍摄前在主体位置放一张白卡（或灰卡），拍一张白卡照作为白平衡参考。后期在 Lightroom 中用白平衡吸管点击白卡，得到这个混合光的准确色温。但白卡只能解决整体色温，不能解决色温分裂。

色温分裂的真正解决方法是「关掉室内灯」——只用窗户光，色温统一。如果室内灯必须开（如氛围需要），就在后期用 Lightroom 的「色调分离」面板——把高光（窗户侧）偏冷一点，阴影（室内灯侧）偏暖一点，模拟两种光的色温差异，让画面看起来「故意」有色温分裂，而不是「失控」的色温分裂。

记住一个判断口诀：**色温分裂是室内人像的头号难题，关灯是首选，白卡是基础，色调分离是补救。**

## 四、避坑

**反面案例一：用顺窗光拍人像。** 顺窗光让脸部均匀照亮但缺立体感，拍出来像证件照。改用侧窗光 45°，主体侧对窗户，立体感天然形成。

**反面案例二：正午直射阳光射入室内。** 正午阳光从南窗射入，形成硬光斑——光斑下过曝，光斑外欠曝，画面有强烈的明暗对比。用纱帘柔化，或避开正午时段。

**反面案例三：高 ISO 直出 JPEG。** ISO 3200 以上 JPEG 直出噪点明显，画质受损。室内弱光建议同时拍 RAW，后期用 Lightroom 降噪或 AI 降噪。

**反面案例四：用 EF-S 18-135 拍室内特写。** 18-135 的 f/3.5-5.6 在室内弱光下要拉高 ISO 到 3200-6400 才能用 1/125 秒，画质受损明显。室内特写优先用 50mm f/1.8 或 35mm f/2 大光圈镜头。

**反面案例五：室内灯与窗户光混用不校正色温。** 两种光色温不同，画面出现色温分裂——窗户侧脸偏蓝白，室内灯侧脸偏黄。优先关掉室内灯只用窗户光；如果必须开室内灯，用白卡校正 + 后期色调分离。

**反面案例六：反光板太近补光过度。** 反光板离主体太近（小于 0.5 米）会补光过度，阴影侧比亮侧还亮，画面失去立体感。反光板距离主体 1-1.5 米，补光强度约 1-1.5 档，保持光比 3:1 到 4:1。

## 五、践行

1. 在家中找一面有大窗户的墙（最好是北窗或南窗加纱帘），让模特侧对窗户 45°，用 90D 配 EF 50mm f/1.8 STM 拍一组侧窗光人像，光圈 f/2.8、ISO 400、快门 1/125 秒，对比不同反光板距离（0.5 米、1 米、1.5 米）下光比的差异。
2. 用同一种参数（f/2.8、ISO 400）分别拍「侧窗光、逆窗光、顺窗光」三组人像，对比三种光位下的立体感与氛围差异，找出你最喜欢的光位。
3. 关掉室内灯只用窗户光拍一组人像，与开着室内灯混用拍一组人像，对比色温分裂问题。后期用 Lightroom 的色调分离面板，把混用组的色温分裂调整为「故意」的色调分裂。
4. 在低光场景（如远离窗户 3 米的卧室）用 ISO 1600、3200、6400 三档拍同一组人像，RAW 格式记录，后期在 Lightroom 中分别用默认降噪与 AI Denoise 降噪，对比三档 ISO 的可用性边界。
5. 在家中三个不同房间（客厅、卧室、书房）各拍一组环境人像，注意每个房间的窗户方向与色温差异，整理一份「我家每个房间的光特征表」作为后续拍摄的参考。
