### 2. 二分搜索（[[P02_二分搜索专题]]）

**适用场景**：有序数组、单调函数、搜索边界

**经典模板**：
```python
left, right = 0, len(nums) - 1
while left <= right:
    mid = left + (right - left) // 2
    if nums[mid] == target:
        return mid
    elif nums[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
```

### 8. BFS/DFS（[[P08_BFS_DFS专题]]）

**适用场景**：图的遍历、最短路径、连通性判断

**BFS 模板**（层序遍历）：
```python
from collections import deque
queue = deque([start_node])
while queue:
    node = queue.popleft()
    for neighbor in get_neighbors(node):
        if not visited[neighbor]:
            visited[neighbor] = True
            queue.append(neighbor)
```

**DFS 模板**（递归）：
```python
def dfs(node):
    if node is None:
        return
    visited[node] = True
    for neighbor in get_neighbors(node):
        if not visited[neighbor]:
            dfs(neighbor)
```

**典型题目**：
- 岛屿数量（BFS/DFS）
- 腐烂的橘子（BFS）
- 二叉树的右视图（BFS）
- 太平洋大西洋水流问题（DFS）
