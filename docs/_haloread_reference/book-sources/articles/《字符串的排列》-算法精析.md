## 题目信息

**题目名称**：字符串的排列

**LeetCode链接**：[字符串的排列](https://leetcode.cn/problems/zi-fu-chuan-de-pai-lie-lcof/)

**难度**：Medium

**描述**：输入一个字符串，打印出该字符串中字符的所有排列。你可以以任意顺序返回这个字符串数组，但里面不能有重复元素。



## 🧊 知识点"破冰"讲解

**回溯法 (Backtracking)**。

*   想象你有 `n` 个空位，手里有 `n` 张牌。
*   第一个空位可以放任意一张牌（比如选了 'a'）。
*   第二个空位只能放手里剩下的牌。
*   ...
*   当所有空位都填满，就得到一个结果。
*   **去重技巧**：如果手里的牌有两张 'a'，为了避免产生重复的结果，我们规定：**"同样的牌，在同一个位置上，只能被选一次"**。通常通过排序后 `if nums[i] == nums[i-1]` 来判断。



## ❓ "为什么"的解释

**Q：为什么要选右上角（或左下角）？左上角不行吗？**

**A：** 问得好！
*   如果你站在 **左上角**（最小值）：
    *   往右走是变大，往下走也是变大。
    *   如果 `Current < Target`，你是该往右还是往下？**不知道啊！** 两个方向都可能藏着目标，这就没法"排除法"了。
*   如果你站在 **右上角**：
    *   往左是变小，往下是变大。
    *   方向是 **互斥** 的，这就给了我们明确的指引。**这就是选它的根本原因！**
