**题目名称**：左旋转字符串

**描述**：字符串的左旋转操作是把字符串前面的若干个字符转移到字符串的尾部。

## 🛒 题目"人话"翻译

`"abcdefg", k=2` -> `"cdefgab"`。

把头上的帽子摘下来戴到屁股上。

## 🧊 知识点"破冰"讲解

**字符串切片 (Slicing)**。

*   `s[:n]`：前 n 个字符。
*   `s[n:]`：从第 n 个开始到最后的字符。
*   拼起来：`s[n:] + s[:n]`。

## 💡 思路"推导"过程

这道题非常简单，关键在于理解字符串切片的操作。

**第一阶段：暴力解法**
*   创建一个新字符串，先把后半部分加进去，再把前半部分加进去。
*   时间复杂度 O(N)，空间复杂度 O(N)。

**第二阶段：Python 切片**
*   利用 Python 的切片特性，一行代码搞定。
*   `s[n:]` 获取后半部分，`s[:n]` 获取前半部分。
*   拼接起来即可。

## ❓ "为什么"的解释

**Q：为什么切片操作这么简单？**

**A：** Python 的字符串切片操作非常高效，底层实现是 C 语言级别的内存拷贝，所以虽然看起来简单，但性能很好。
