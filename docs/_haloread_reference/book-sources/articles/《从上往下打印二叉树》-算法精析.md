## 题目信息

**题目名称**：从上往下打印二叉树

**LeetCode链接**：[Binary Tree Level Order Traversal](https://leetcode.cn/problems/binary-tree-level-order-traversal/)

**难度**：Medium

**描述**：从上到下打印出二叉树的每个节点，同一层的节点按照从左到右的顺序打印。



## 🧊 知识点"破冰"讲解

**BFS (广度优先搜索)**。

*   需要用到 **队列 (Queue)**（先进先出）。
*   **套路**：
    1.  把根节点扔进队列。
    2.  队列不空就循环：
        *   拿出一个节点，打印它。
        *   如果它有左孩子，扔进队列。
        *   如果它有右孩子，扔进队列。



## ❓ "为什么"的解释

**Q：为什么要用队列而不是栈？**

**A：** 
*   队列是先进先出（FIFO），保证了先入队的节点先被访问。
*   这样就能保证同一层的节点按从左到右的顺序被访问。
*   如果用栈（后进先出），就会变成深度优先遍历。

**Q：时间复杂度是多少？**

**A：** 
*   每个节点被访问一次，时间复杂度是 O(N)。
*   空间复杂度是 O(N)，因为队列中最多存储一层的节点。
