## 题目信息

**题目信息**：LeetCode 34. Find First and Last Position of Element in Sorted Array / 剑指 Offer 53 - I

*   **原题链接**：[在排序数组中查找数字 I](https://leetcode.cn/problems/zai-pai-xu-shu-zu-zhong-cha-zhao-shu-zi-lcof/)
*   **难度**：Easy
*   **描述**：统计一个数字在排序数组中出现的次数。

### 🛒 题目 "人话" 翻译

**"查户口"**。

*   一个排好序的队伍里,有多少个叫 "张三" 的人？
*   因为是排好序的,所以所有的 "张三" 肯定都挨在一起。
*   你只需要找到 **第一个** "张三" 和 **最后一个** "张三",数数中间有多少人就行。



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：线性扫描** 🐢
*   **想法**：遍历数组,统计目标数字出现的次数。
*   **问题**：时间复杂度 O(N),没有利用数组有序的特性。

**第二阶段：二分查找边界** 🐇
*   **想法**：用二分查找找到目标数字的左右边界。
*   **关键点**：
    *   找左边界：当找到目标时,继续往左找,看前面还有没有。
    *   找右边界：当找到目标时,继续往右找,看后面还有没有。
*   **公式**：`count = right - left + 1`

**第三阶段：优化实现** ✨
*   **想法**：写一个辅助函数,可以找左边界或右边界。
*   **参数**：`is_left` 标志位,表示找左边界还是右边界。



### 🚗 代码 "带逛"

```python
def search(nums, target):
    """
    统计目标数字出现的次数
    """
    # 辅助函数：找边界
    def find_boundary(is_left):
        left, right = 0, len(nums) - 1
        boundary = -1
        
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                boundary = mid
                # 如果找左边界,就往左边缩,看前面还有没有
                if is_left:
                    right = mid - 1
                # 如果找右边界,就往右边缩,看后面还有没有
                else:
                    left = mid + 1
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
        return boundary

    left_idx = find_boundary(True)
    # 如果左边界都没找到,说明这个数压根不存在
    if left_idx == -1:
        return 0
        
    right_idx = find_boundary(False)
    
    return right_idx - left_idx + 1
```
