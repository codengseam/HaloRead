## 题目信息

**题目名称**：两个链表的第一个公共节点

**LeetCode链接**：[两个链表的第一个公共节点](https://leetcode.cn/problems/liang-ge-lian-biao-de-di-yi-ge-gong-gong-jie-dian-lcof/)

**难度**：Easy

**描述**：输入两个链表，找出它们的第一个公共节点。



## 🧊 知识点"破冰"讲解

**双指针 (Two Pointers)** 的浪漫解法。

*   **问题**：A 的路程可能比 B 长，如果同时出发，A 还没到路口，B 已经跑过了。
*   **解法**：
    *   A 走完自己的路，就去走 B 的路。
    *   B 走完自己的路，就去走 A 的路。
    *   **原理**：`len(A) + len(B) == len(B) + len(A)`。
    *   只要两人速度一样，最终一定会在终点（或者路口）相遇。



## ❓ "为什么"的解释

**Q：为什么要选右上角（或左下角）？左上角不行吗？**

**A：** 问得好！
*   如果你站在 **左上角**（最小值）：
    *   往右走是变大，往下走也是变大。
    *   如果 `Current < Target`，你是该往右还是往下？**不知道啊！** 两个方向都可能藏着目标，这就没法"排除法"了。
*   如果你站在 **右上角**：
    *   往左是变小，往下是变大。
    *   方向是 **互斥** 的，这就给了我们明确的指引。**这就是选它的根本原因！**
