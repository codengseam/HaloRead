## 题目信息

**题目信息**：LeetCode 剑指 Offer 62. 圆圈中最后剩下的数字

*   **原题链接**：[圆圈中最后剩下的数字](https://leetcode.cn/problems/yuan-quan-zhong-zui-hou-sheng-xia-de-shu-zi-lcof/)
*   **难度**：Easy
*   **描述**：0,1,···,n-1这n个数字排成一个圆圈,从数字0开始,每次从这个圆圈里删除第m个数字。求出这个圆圈里剩下的最后一个数字。

### 🛒 题目 "人话" 翻译

**"约瑟夫环" (Josephus Problem)**。

*   杀人游戏。数到 m 的人出局。
*   最后活下来的是几号？



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：模拟法** 🐢
*   **想法**：用数组或链表模拟整个过程。
*   **问题**：时间复杂度 O(N^2),会超时。

**第二阶段：数学推导** 🐇
*   **想法**：从最后一步倒推。
*   **推导**：
    *   当只剩 1 个人时,索引是 0
    *   当剩 2 个人时,索引是 `(0 + m) % 2`
    *   当剩 3 个人时,索引是 `((0 + m) % 2 + m) % 3`
    *   ...
    *   当剩 n 个人时,索引是 `f(n, m)`
*   **公式**：`f(n, m) = (f(n-1, m) + m) % n`

**第三阶段：迭代实现** ✨
*   **想法**：用循环代替递归,避免栈溢出。
*   **步骤**：
    1.  初始化 `ans = 0`
    2.  从 2 遍历到 n,每次更新 `ans = (ans + m) % i`
    3.  返回 `ans`



### 🚗 代码 "带逛"

```python
def lastRemaining(n, m):
    # 最终剩下的那个数,在只剩 1 个人时,索引是 0
    ans = 0
    
    # 倒推回去：从 2 个人开始,一直推到 n 个人
    for i in range(2, n + 1):
        ans = (ans + m) % i
        
    return ans
```
