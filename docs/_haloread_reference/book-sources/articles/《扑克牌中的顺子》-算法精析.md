## 题目信息

**题目名称**：扑克牌中的顺子

**LeetCode链接**：[扑克牌中的顺子](https://leetcode.cn/problems/bu-ke-pai-zhong-de-shun-zi-lcof/)

**难度**：Easy

**描述**：从若干副扑克牌中随机抽 5 张牌，判断是不是一个顺子。

## 🛒 题目"人话"翻译

**"癞子斗地主"**。

*   给你 5 张牌，问能不能组成顺子。
*   0 (大小王) 是 **"万能牌"**，可以填补任何空缺。
*   如果有对子（除了 0 以外的重复牌），那肯定不是顺子。

## 🧊 知识点"破冰"讲解

**排序 + 计数**。

1.  **排序**：先把牌理顺。
2.  **数 0**：看看有几张万能牌。
3.  **找空缺**：看看相邻两张牌之间缺了几个数（比如 1 和 4 之间缺了 2, 3，缺 2 个）。
4.  **判断**：如果空缺的总数 <= 万能牌的数量，那就赢了！
5.  **特判**：如果发现对子（比如两个 3），直接输。

## 💡 思路"推导"过程

这道题的关键在于如何判断能否组成顺子。

**第一阶段：暴力解法**
*   枚举所有可能的排列组合，看是否有顺子。
*   太复杂了。

**第二阶段：排序 + 空缺统计**
*   先排序，然后统计空缺的数量。
*   如果空缺数量 <= 大小王的数量，就可以组成顺子。
*   如果有重复牌（除了大小王），直接返回 False。

## ❓ "为什么"的解释

**Q：为什么最大牌减最小牌要小于 5？**

**A：** 因为如果是顺子，5 张牌的最大值和最小值之差应该小于 5（因为连续的 5 个数，最大值 - 最小值 = 4）。如果差值大于等于 5，说明中间有空缺，而且空缺太大，即使有大小王也填不满。

## 🚗 代码"带逛"

```python
def isStraight(nums):
    # 1. 排序
    nums.sort()
    
    joker = 0
    for i in range(4):
        # 统计大小王数量
        if nums[i] == 0:
            joker += 1
        # 如果有对子（非0），直接 False
        elif nums[i] == nums[i+1]:
            return False
            
    # 最大牌 - 最小牌 < 5，则一定能组成顺子
    return nums[4] - nums[joker] < 5
```
