## 题目信息

**题目信息**：LeetCode 153. Find Minimum in Rotated Sorted Array / 剑指 Offer 11

*   **原题链接**：[Find Minimum in Rotated Sorted Array](https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array/)
*   **难度**：Medium
*   **描述**：把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。输入一个递增排序的数组的一个旋转，输出旋转数组的最小元素。

### 🛒 题目 "人话" 翻译

想象你有一把**卷尺**，上面的数字是 1, 2, 3, 4, 5。有人把这把卷尺在某个位置剪断了，比如在 3 和 4 之间，然后把后面那段拼到了前面。变成了 `[4, 5, 1, 2, 3]`。

**你的任务**：就是找到那个"断点"后的第一个数字（也就是最小值 1）。
