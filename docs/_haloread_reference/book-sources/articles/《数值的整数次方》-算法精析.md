## 题目信息

**题目信息**：LeetCode 50. Pow(x, n) / 剑指 Offer 16

*   **原题链接**：[Pow(x, n)](https://leetcode.cn/problems/powx-n/)

*   **描述**：实现 pow(x, n) ，即计算 x 的 n 次幂函数。

### 🛒 题目 "人话" 翻译

计算 `x` 的 `n` 次方。
比如 `2.0` 的 `10` 次方。



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：暴力循环** 🐢
*   **想法**：不就是乘 n 次吗？我写个循环！
*   **行动**：`result = 1; for i in range(n): result *= x`
*   **结局**：能用,但是太慢了！如果 n 是 10^9,你要循环 10 亿次。

**第二阶段：快速幂** 🐇
*   **想法**：能不能减少乘法次数？
*   **观察**：
    *   `x^10 = (x^5)^2`
    *   `x^5 = (x^2)^2 * x`
    *   `x^2 = (x^1)^2`
*   **规律**：
    *   如果指数是偶数：`x^n = (x^(n/2))^2`
    *   如果指数是奇数：`x^n = (x^(n//2))^2 * x`
*   **结论**：这样每次指数减半,乘法次数从 O(N) 降到 O(log N)！

**第三阶段：处理负指数** ✨
*   **问题**：如果 n 是负数怎么办？
*   **解法**：先算正指数,最后取倒数。`x^(-n) = 1 / x^n`



### 🚗 代码 "带逛"

```python
def power(base, exponent):
    """
    数值的整数次方
    """
    # 1. 处理底数为 0 的特殊情况
    if abs(base - 0.0) < 0.0000001 and exponent < 0:
        raise ZeroDivisionError

    # 2. 处理负指数：先算正的,最后取倒数
    ret = power_value(base, abs(exponent))
    if exponent < 0:
        return 1.0 / ret
    else:
        return ret

def power_value(base, exponent):
    # 递归结束条件
    if exponent == 0:
        return 1
    if exponent == 1:
        return base
        
    # 递归：求一半次方的结果
    # exponent >> 1 就是除以 2
    ret = power_value(base, exponent >> 1)
    
    # 结果平方
    ret *= ret
    
    # 如果指数是奇数,还要再乘一次底数
    # exponent & 1 == 1 判断奇偶
    if exponent & 1 == 1:
        ret *= base
    return ret
```
