## 题目信息

**题目名称**：连续子数组的最大和

**LeetCode链接**：[Maximum Subarray](https://leetcode.cn/problems/maximum-subarray/)

**难度**：Easy

**描述**：输入一个整型数组，数组中的一个或连续多个整数组成一个子数组。求所有子数组的和的最大值。



## 🧊 知识点"破冰"讲解

**动态规划 (DP)**。

*   `dp[i]`：以第 `i` 个数结尾的子数组的最大和。
*   如果 `dp[i-1]` > 0：那加上它肯定更好。`dp[i] = dp[i-1] + nums[i]`。
*   如果 `dp[i-1]` <= 0：那加上它只会变差（或不变）。干脆不要前面的了，只要我自己。`dp[i] = nums[i]`。



## ❓ "为什么"的解释

**Q：为什么要选右上角（或左下角）？左上角不行吗？**

**A：** 问得好！
*   如果你站在 **左上角**（最小值）：
    *   往右走是变大，往下走也是变大。
    *   如果 `Current < Target`，你是该往右还是往下？**不知道啊！** 两个方向都可能藏着目标，这就没法"排除法"了。
*   如果你站在 **右上角**：
    *   往左是变小，往下是变大。
    *   方向是 **互斥** 的，这就给了我们明确的指引。**这就是选它的根本原因！**
