## 题目信息

**题目信息**：LeetCode 236. Lowest Common Ancestor of a Binary Tree / 剑指 Offer 68 - II

*   **原题链接**：[二叉树的最近公共祖先](https://leetcode.cn/problems/er-cha-shu-de-zui-jin-gong-gong-zu-xian-lcof/)
*   **难度**：Easy
*   **描述**：给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

### 🛒 题目 "人话" 翻译

**"寻根问祖"**。

*   你（p）和你表弟（q）都在同一个家族族谱（树）里。
*   往上找,你们俩共同的爷爷或者太爷爷是谁？
*   我们要找 **辈分最低**（离你们最近）的那位祖先。



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：暴力记录路径** 🐢
*   **想法**：找到从根到 p 的路径,找到从根到 q 的路径。
*   **问题**：需要额外的空间存储路径,而且需要两次遍历。

**第二阶段：递归查找** 🐇
*   **想法**：用递归在树中查找 p 和 q。
*   **关键点**：
    *   如果当前节点是 p 或 q,返回当前节点。
    *   如果左子树找到了,右子树也找到了,当前节点就是最近公共祖先。
    *   如果只有一边找到了,返回找到的那一边。

**第三阶段：优化实现** ✨
*   **想法**：简化递归逻辑。
*   **步骤**：
    1.  如果当前节点为空,返回 None
    2.  如果当前节点是 p 或 q,返回当前节点
    3.  递归查找左子树和右子树
    4.  根据左右子树的返回值判断



### 🚗 代码 "带逛"

这里展示两种思路。一种是通用的递归法（最常用）,一种是记录路径法（文件中的思路）。

```python
# 方法一：递归法（推荐）
def lowestCommonAncestor(root, p, q):
    # 1. 递归尽头：如果空了,或者找到了 p 或 q,直接返回
    if not root or root == p or root == q:
        return root
        
    # 2. 去左右子树找
    left = lowestCommonAncestor(root.left, p, q)
    right = lowestCommonAncestor(root.right, p, q)
    
    # 3. 汇总结果
    # 如果左右都找到了,说明当前节点就是分岔口（最近公共祖先）
    if left and right:
        return root
    # 否则,谁找到了就返回谁
    return left if left else right
```
