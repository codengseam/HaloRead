## 题目信息

**题目信息**：LeetCode 151. Reverse Words in a String / 剑指 Offer 58 - I

*   **原题链接**：[翻转单词顺序](https://leetcode.cn/problems/fan-zhuan-dan-ci-shun-xu-lcof/)
*   **难度**：Medium
*   **描述**：输入一个英文句子,翻转句子中单词的顺序,但单词内字符的顺序不变。

### 🛒 题目 "人话" 翻译

`"I am a student."` -> `"student. a am I"`



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：暴力翻转** 🐢
*   **想法**：把整个字符串翻转,然后逐个单词再翻转回来。
*   **问题**：实现起来比较麻烦,需要处理空格和边界。

**第二阶段：Python 黑魔法** 🐇
*   **想法**：利用 Python 的字符串处理函数。
*   **步骤**：
    1.  `split()` 切分字符串
    2.  `[::-1]` 翻转列表
    3.  `' '.join()` 连接字符串
*   **优点**：代码简洁,效率高。

**第三阶段：手动实现** ✨
*   **想法**：如果面试官不让用内置函数,可以手动实现。
*   **步骤**：
    1.  去除首尾空格
    2.  从后往前遍历,识别单词
    3.  把单词加入结果列表



### 🚗 代码 "带逛"

```python
def reverseWords(s):
    # split() 默认会去除多余的空格,非常方便
    return " ".join(s.split()[::-1])
```
