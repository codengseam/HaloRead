## 题目信息

**题目信息**：剑指 Offer 21. 调整数组顺序使奇数位于偶数前面

*   **原题链接**：[调整数组顺序使奇数位于偶数前面](https://leetcode.cn/problems/diao-zheng-shu-zu-shun-xu-shi-qi-shu-wei-yu-ou-shu-qian-mian-lcof/)
*   **难度**：Easy
*   **描述**：输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有奇数位于数组的前半部分，所有偶数位于数组的后半部分。

### 🛒 题目 "人话" 翻译

就像军训排队。
教官喊："男的站左边，女的站右边！"
你需要指挥大家迅速交换位置。

### 🧊 知识点 "破冰" 讲解

**双指针法 (Two Pointers)**。
*   一个指针 `left` 从头找，找偶数（因为偶数不该待在左边）。
*   一个指针 `right` 从尾找，找奇数（因为奇数不该待在右边）。
*   两个人都找到了？**交换！**

### 🚗 代码 "带逛"

```python
def reorder(nums, func):
    """
    调整数组顺序，支持自定义判断条件
    :param nums: 数组
    :param func: 判断函数（返回True放右边，False放左边）
    """
    left, right = 0, len(nums) - 1
    while left < right:
        # left 向右走，直到遇到一个偶数（不满足条件的）
        while left < right and not func(nums[left]):
            left += 1
        # right 向左走，直到遇到一个奇数（不满足条件的）
        while left < right and func(nums[right]):
            right -= 1
        # 两人都停下来了，交换
        if left < right:
            nums[left], nums[right] = nums[right], nums[left]

def is_even(num):
    # 判断是否为偶数
    return (num & 1) == 0
```
