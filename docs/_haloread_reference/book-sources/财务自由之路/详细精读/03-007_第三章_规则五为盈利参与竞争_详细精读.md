# 第三章：新规则 - 03-007 规则五：为盈利参与竞争，经历冒险

## 一、核心观点提炼

1. **风险与回报正相关**：在当今时代，经历冒险的人会得到更多回报，带有冒险精神的投资者能获得12%甚至更高的收益，企业家能够发财。

2. **稳定与自由不可兼得**：我们必须在自由和稳定之间做出抉择，那些寻求稳定的人最终找到的是恐惧，用生活质量换取稳定的幻象。

3. **真正的风险在于不改变**：变化对于那些不理解并因此不做改变的人来说才是真正有风险的，成功的企业家和投资者只做显而易见正确的事情。

## 二、关键概念解析

| 概念 | 定义 | 应用场景 |
|------|------|----------|
| 风险意识 | 在信息时代，主动承担经过评估的风险以获取更高回报的思维方式 | 投资决策、创业选择、职业转型 |
| 稳定vs自由 | 两种互斥的人生选择：追求稳定意味着放弃自由和成长机会，追求自由则需要接受不确定性 | 职业规划、生活方式选择、财务决策 |
| 冒险悖论 | 成功者看似冒险，实则只做显而易见正确的事；真正的风险来自于对变化的无知和抗拒 | 商业决策、投资策略、个人成长 |
| 小步与大步 | 慎重走好每一小步可以完成很多事情，但实现成功需要勇气迈出一大步 | 目标设定、行动计划、突破瓶颈 |

## 三、可执行的动作建议

### 立即执行（今天）
- 回顾过去5年，列出自己经历过的"大冒险"，评估从中获得的成长和回报
- 识别当前生活中一个因回避风险而停滞不前的领域
- 写下自己追求"稳定"所付出的真实代价（生活质量、机会成本、内心恐惧）

### 短期行动（本周内）
- 选择一个自己一直想做但不敢尝试的事情，制定具体的行动计划
- 学习一位成功企业家或投资者的思维方式，理解他们如何评估"风险"
- 在投资、职业或个人成长中选择一个领域，主动承担一次经过计算的风险

### 中期行动（本月内）
- 建立个人的风险评估框架，区分"冒失"与"经过计算的冒险"
- 找到一位愿意冒险的榜样，研究其决策逻辑和行为模式
- 制定一个"迈出一大步"的计划，设定明确的时间节点和成功标准

## 四、误区警示

1. **混淆冒险与冒失**：真正的成功者不是盲目冒险，而是只做显而易见正确的事情。冒险不等于鲁莽，而是基于对变化的理解和把握。

2. **追求稳定的幻象**：许多人以为稳定就是安全，实际上稳定往往伴随着恐惧和不安。稳定是一种代价高昂的选择，它消耗的是生活质量和个人成长的机会。

3. **只走小步的陷阱**：虽然慎重走好每一小步很重要，但如果永远不敢迈出一大步，就无法实现真正的突破和成功。小悬崖峭壁之间的来回跨越毫无意义。

4. **回避风险等于回避成长**：缺乏继续接受教育的准备、回避风险，是大多数人无法赚到本应赚到的东西的主要原因。

## 五、启发性思考问题

1. 在过去5年中，你回避了哪些风险？这些回避让你失去了什么机会？

2. 你目前追求的"稳定"是否真的是稳定，还是只是恐惧的伪装？

3. 如果要在自由和稳定之间做出选择，你会选择哪一个？为什么？

4. 你生命中有哪些里程碑式的成功是在经历风险之后才实现的？

5. 你如何区分"经过计算的冒险"和"冒失的行为"？你的判断标准是什么？

## 六、金句摘录

> 那些经历冒险的人，无论如何都会得到更多的回报。

> 现如今，我们必须抉择，自己想要自由还是稳定。两者不可兼得。那些寻求稳定的人找到的是恐惧。

> 恐惧扭曲了事物本来的意义，使我们感知异常。

> 巨人常常会绊倒，而爬虫不会，因为它们所做的一切只有深挖和爬行。

> 实际上没有什么是免费的。安稳也并非如此。为了得到稳定，我们放弃的是生活质量。

> 许多人用充满奇遇和美好的令人羡慕的生活换取稳定的幻象，即充满恐惧和不安的生活。

> 我们有能力慎重地走好每一小步完成很多事情，甚至一切事情。但是，我们需要勇气迈出一大步来实现成功。

> 旧规则：躲避风险。风险带来的只有不必要的危险。人们参与竞争就应该不输。
> 新规则：竞争是为了获利。去经历风险吧。
