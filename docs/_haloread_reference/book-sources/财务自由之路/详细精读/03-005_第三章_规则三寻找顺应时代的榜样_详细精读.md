# 第三章：新规则 - 03-005 规则三：寻找顺应时代的榜样

## 一、核心观点提炼

- **观点一**：榜样随时代变化，找稳定工作不再是年轻人的梦想，选择榜样应顺应时代趋势而非固守旧观念
- **观点二**：从事与自己能力和热情相匹配的工作比单纯追求高收入更重要，这是获得财富的关键
- **观点三**：新规则强调向那些"以此为乐"的人看齐，而非仅仅追求职位稳定

## 二、关键概念解析

| 概念 | 定义 | 应用场景 |
|------|------|----------|
| 顺应时代的榜样 | 指在当下社会环境中获得成功、具有影响力且能被年轻人认同的成功人士，如明星、运动员、企业家等 | 思考自己想要成为什么样的人，选择能够激励自己的现代榜样 |
| 能力与热情匹配 | 指所从事的工作既符合自身能力水平，又能带来内在热情和满足感 | 职业选择时评估"我擅长什么"与"我热爱什么"的交集 |
| 旧规则 | 找一个稳定的"职位"，如成为医生或律师，勤奋可以使人实现目标 | 传统职业路径，侧重稳定性和社会地位 |
| 新规则 | 向那些拥有一份工作并且以此为乐的人看齐，思考自己的能力和爱好，以此发展事业 | 现代职业发展，侧重个人满足感和自我实现 |

## 三、可执行的动作建议

### 立即执行（今天）

- 列出3-5个你目前认为是"成功人士"的人，分析他们为什么吸引你
- 写下你目前的工作中，哪些部分让你感到满足，哪些让你感到枯燥

### 短期行动（本周内）

- 与2-3位在不同领域工作的朋友交流，了解他们选择职业的考量
- 思考"能力"与"热情"两个维度，列出你在哪些领域既有能力又有热情

### 中期行动（本月内）

- 进行职业探索，尝试了解1-2个你感兴趣但从未深入了解的领域
- 寻找在你渴望领域的榜样，研究他们的成长路径和关键决策
- 评估当前工作是否能让你"以此为乐"，如果不能，开始规划转型

## 四、误区警示

1. **固守旧规则**：认为只有成为医生、律师等传统稳定职业才算成功，忽视新时代下多元化的成功路径
2. **只看收入不看热情**：选择工作时只关注薪资高低，忽视与自身能力、热情的匹配度，最终导致职业倦怠
3. **把"安稳"当作唯一目标**：将"稳定的工作"作为职业选择的唯一标准，而非思考这份工作能否带来满足感
4. **盲目模仿榜样**：简单模仿成功人士的外在行为，而不理解他们成功背后的能力基础和热情驱动

## 五、启发性思考问题

1. 你现在的榜样是谁？这个人为什么成为你的榜样？他的成功路径是否适合你？
2. 你目前从事的工作，是否让你感到"以此为乐"？如果不能，问题出在能力不匹配还是热情不在？
3. 你认为自己最大的能力优势是什么？最让你充满热情的事情是什么？这两者的交集在哪里？
4. 如果不考虑金钱和社会期待，你真正想做的是什么？
5. 你愿意为了从事自己热爱的工作，付出多长时间的努力和准备？

## 六、金句摘录

> "工作的意义并不在于根据一份报酬高低来找一份活计。一份工作必须先符合自己的能力和热情，这一点尤为重要。"

> "如果我们做一些自己喜欢并且与自己能力相符的工作，那么收入会滚滚而来。"

> "如果我们做的事情不能使我们得到极大的满足，那么一切技巧都不会对我们的成功起到什么作用。"

> "思考一下您的能力和爱好之所在，然后以此来发展自己的事业。"
