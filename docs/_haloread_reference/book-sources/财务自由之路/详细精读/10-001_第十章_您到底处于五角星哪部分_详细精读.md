# 第十章：做企业家，不做接管者 - 10-001 您到底处于这个五角星的哪一部分呢

## 一、核心观点提炼

1. **身份选择的必要性**：企业家和雇员是两种截然不同的角色，必须做出明确选择，不可能也不应该两者兼顾。

2. **"受雇的企业家"陷阱**：试图同时扮演雇主和雇员角色的人，最终会陷入两边的任务都无法很好完成的困境。

3. **企业家的正确位置**：企业家不应该在企业里面工作，而应该凌驾企业之上管理企业，这样才能成功经营公司并获得高质量生活。

## 二、关键概念解析

| 概念 | 定义 | 应用场景 |
|------|------|----------|
| 受雇的企业家 | 试图同时兼顾雇主身份和雇员工作的人，实际上是自我雇佣的"打工者" | 自由职业者、小企业主、个体经营者需要自我诊断是否陷入此状态 |
| 企业家vs雇员 | 企业家位于五角星左半部分，负责管理和战略；雇员位于右半部分，负责执行和操作 | 在职业规划时明确自己的定位，选择适合的发展路径 |
| 凌驾企业之上 | 企业家应该站在更高维度管理企业，而非陷入日常事务中 | 企业主需要建立系统、授权他人，而非事必躬亲 |
| 仓鼠之轮 | 形容那些看似忙碌却原地打转、无法取得真正进步的状态 | 当企业效益仅够维持生存时，需要反思是否陷入此困境 |

## 三、可执行的动作建议

### 立即执行（今天）
- 在纸上画出五角星图，标注自己当前所处的位置
- 诚实回答：我是在"管理企业"还是在"企业里工作"？
- 列出自己目前承担的所有角色和任务，区分哪些是企业家任务，哪些是雇员任务

### 短期行动（本周内）
- 统计过去一周的工作时间分配，计算花在"管理"和"执行"上的时间比例
- 识别3项可以授权或外包的日常事务性工作
- 反思：如果完全脱离日常运营，企业能运转多久？

### 中期行动（本月内）
- 制定清晰的职责分离计划，明确哪些工作必须由企业家角色完成
- 建立至少一个能够替代自己执行日常工作的系统或人员
- 设定"脱离执行层"的时间节点和可量化目标

## 四、误区警示

1. **误区：可以同时做好企业家和雇员**
   真相：两种角色对思维方式、任务重点、结果期待都有本质差异，同时兼顾会导致两边都做不好。

2. **误区：企业小就必须事必躬亲**
   真相：企业规模越小，越需要企业家站在更高维度思考战略，否则永远无法突破"仅够维持生存"的状态。

3. **误区：不工作就是不负责**
   真相：企业家最大的责任是建立能够自动运转的系统，而非亲自完成每一项工作。

## 五、启发性思考问题

1. 如果我明天开始完全不参与日常运营，我的企业能撑多久？一周？一个月？还是立刻崩溃？

2. 我目前的工作时间中，有多少比例是在"做正确的事"（战略规划、系统建设），有多少是在"把事情做对"（日常执行）？

3. 我选择成为企业家是因为真正想要创造价值，还是仅仅因为不想被别人雇佣？这个动机是否影响了我对企业家角色的理解？

## 六、金句摘录

> 作为雇主，您不应该在企业里面工作，而是应该凌驾企业之上管理企业。

> 很多公司所创造的效益仅仅能够维持公司生存：您只是在仓鼠之轮上做着无用功。

> 只要您在公司里面工作，您就不可能像管理您的公司那样，成功经营公司，也不可能赚那么多钱，也从来不会取得高质量的生活。
