# 第一章：仓鼠之轮 - 01-001 赚钱的五种方式

## 一、核心观点提炼

1. **赚钱只有五种方式**：雇员(A)、自由职业者(F)、投资者(I)、企业家(U)、专家(E)，这五种方式构成了一个完整的五角星图。

2. **学校教育存在盲区**：学校几乎只培养右半部分的雇员和自由职业者，却不教授如何成为投资者、企业家或专家。

3. **左侧领域收益更高**：投资者、企业家和专家的收入远高于雇员和自由职业者，因为他们的思维方式不同，遵循的游戏规则也不同。

## 二、关键概念解析

| 概念 | 定义 | 应用场景 |
|------|------|----------|
| 五角星图 | 将五种赚钱方式可视化呈现的模型，分为左右两部分 | 用于定位自己当前所处的收入领域，规划职业发展方向 |
| 左半部分领域 | 投资者(I)、企业家(U)、专家(E)三个领域 | 适合追求高收益、愿意承担风险、具备独立思考能力的人 |
| 右半部分领域 | 雇员(A)、自由职业者(F)两个领域 | 适合追求稳定、擅长执行而非决策的人，也是学校教育的主要培养方向 |
| 雇员/工人(A) | 为他人工作，出卖时间换取固定收入的人 | 适合风险厌恶型人格，需要稳定收入来源 |
| 自由职业者(F) | 独立工作，专业技能变现，但仍在用时间换钱 | 医生、律师、咨询师等专业人士 |
| 投资者(I) | 用钱生钱，通过投资获得被动收入 | 有一定资本积累后，通过理财实现财富增值 |
| 企业家(U) | 创建系统，让他人为自己工作，实现收入与时间解绑 | 创业者、企业所有者，月收入平均是雇员的5倍 |
| 专家(E) | 在特定领域拥有顶尖专业能力，享有声望和高收入 | 行业顶尖顾问、意见领袖、专业领域权威 |

## 三、可执行的动作建议

### 立即执行（今天）
- 在五角星图中标注自己当前所处的位置，诚实评估自己属于哪个领域
- 列出自己目前的主要收入来源，判断是主动收入还是被动收入
- 回忆学校教育经历，记录学校教给我们的赚钱技能有哪些

### 短期行动（本周内）
- 分析自己所在领域的工作者与左侧领域工作者的收入差距，查找具体数据
- 识别自己性格中的强项和弱项：是更擅长执行还是决策？是更喜欢稳定还是挑战？
- 找到一位左侧领域（投资者/企业家/专家）的成功人士作为学习榜样

### 中期行动（本月内）
- 深入了解左侧三个领域中的某一个，阅读相关书籍或参加课程
- 制定一个从当前领域向目标领域过渡的初步计划
- 开始培养左侧领域所需的思维方式：从"如何完成任务"转向"如何创造价值"

## 四、误区警示

1. **误以为学校教育足够**：学校只教我们成为好员工，却不教我们如何成为投资者、企业家或专家。如果只依赖学校教育，我们很难突破收入天花板。

2. **盲目跟随大众选择**：很多人选择某个领域仅仅因为"大家都是这么做的"，而不是基于对自己强项、弱项、风险承受能力和人生目标的深入分析。

3. **认为存在"最佳领域"**：每个领域都有优势和弊端，不存在绝对的最佳选择。关键在于找到与自己性格、能力和目标相匹配的领域。

4. **忽视思维方式的差异**：左侧领域的人赚得更多，根本原因在于他们有完全不同的想法，遵循完全不同的游戏规则。仅模仿行为而不改变思维，无法真正实现领域跨越。

## 五、启发性思考问题

1. 如果不受任何限制，你最想成为五角星图中哪个领域的人？是什么阻止了你？

2. 你目前所处的领域，是你主动选择的结果，还是被动接受的结果？如果是被动接受，是谁或什么因素影响了你的选择？

3. 德国半数家庭的收入仅占总货币供应量的4.5%，这个数据对你有什么启发？你认为造成这种分配格局的根本原因是什么？

4. 学校教育为什么只培养右半部分领域的人才？这种教育模式对社会有什么影响？对个人又有什么影响？

5. "他们有着完全不同的想法，并且遵从完全不同的游戏规则进行游戏"——你能举出具体的例子说明左侧领域和右侧领域的思维方式有哪些不同吗？

## 六、金句摘录

> 其实您能够赚钱的方式只有五种。

> 学校几乎就是为位于右半部分的领域而准备的。我们学习如何成为一名好的职员或者自由职业者。但是我们丝毫不了解如何成为一位成功的投资者、企业家或者专家。

> 位于左半部分领域使人赚更多的钱。拿企业家举例，他们每月可支配的资金平均是雇员或者工人的5倍。

> 并不存在哪个"最佳领域"。每个领域都有其优势，同时也存在弊端。最终选择了哪一领域，取决于您的强项和弱项，取决于您对风险的准备，取决于您的性格和目标。

> 重要的是，您下意识地对五角星图中的一个领域进行选择。不是说您仅仅因为"大家都是这么做的"，就终生致力于某个对您来说并非最佳选择的领域。

> 为什么一小部分人赚得的远多于其他人呢？答案就是他们有着完全不同的想法，并且遵从完全不同的游戏规则进行游戏。
