# 第六章 如何找到适合自己的工作 - 精读笔记

## 一、核心观点

### 1. 找工作的错误观念
- 只有28%的人在工作中找到乐趣
- 大多数人关注劳务市场，忽略自身能力和兴趣
- 劳务市场不应决定我们的命运，内心才是决定者

### 2. 失业者择业四部曲
1. **选择有帮助意义的态度**：把解雇看成机会的开始
2. **不要马上找新工作**：拿出1-3周回答基本问题
3. **像没有失业一样工作**：早8:30到晚6:00，保持纪律
4. **为下一个100天制定计划**：10-20天思考，80-90天行动

### 3. 找工作的最佳方法
**最不靠谱的方法**：
- 网络求职
- 盲目发求职信
- 回复招聘广告
- 向劳动局寻求帮助

**最有效的方法**：
- 询问朋友和熟人（40%成功率）
- 直接给公司打电话（70%成功率）
- 亲自到公司拜访（100天内必定找到）

### 4. 面试技巧
- 不要过多谈论自己
- 谈话内容涉及未来
- 不要过早谈论权利
- 展示求知欲
- 请求无偿工作几天
- 不要在谈话时递交申请材料
- 每次面试后寄感谢信



## 三、动作建议

### 立即执行
- [ ] 创建认知日志
- [ ] 回答自我探索问题（见下方）
- [ ] 列出乐趣清单和能力清单

### 短期行动（1-3个月）
- [ ] 找到乐趣与能力的交集
- [ ] 列出可能合适的公司名单
- [ ] 每天至少拜访4家公司

### 中期行动（3-12个月）
- [ ] 拜访200家公司
- [ ] 建立人脉网络
- [ ] 向每家公司询问推荐

### 自我探索问题
1. 如果没有失败，我会做什么？
2. 世界所有人中，哪份工作是我最热爱的？
3. 什么能给我带来乐趣？（至少50件）
4. 我擅长什么？（50-100件）
5. 我有哪些聪明才智？（11种类型）
6. 我10次最重要的成功是什么？
7. 如果还有6个月活着，我想做什么？
8. 我什么时候激发超长能力？



## 五、思考问题

1. **理想工作**：如果没有任何限制，我想从事什么样的工作？

2. **能力匹配**：我的核心能力是什么？哪些工作需要这些能力？

3. **乐趣来源**：什么事情让我废寝忘食？我能从中赚钱吗？

4. **求职策略**：我是否使用了最有效的求职方法？

5. **价值展示**：我如何向雇主展示我的独特价值？
