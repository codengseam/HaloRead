# 第十一章 先服务 后收益 - 精读笔记

## 一、核心观点

### 1. 服务与收益的关系
- 必须先播种，才有收成
- 不能先赚钱再提供服务
- 收入是别人对您的感谢

### 2. 服务的五个领域（服务五角星）
```
          任务
           △
          /  \
    家庭和朋友 /  \ 机制
        /      \
       /        \
穷人和病人---------同事和顾客
```

### 3. 五个领域的服务内容

| 领域 | 服务内容 | 核心要点 |
|------|----------|----------|
| 同事和顾客 | 帮助、满足、使开心 | 收入是别人的感谢 |
| 任务 | 从属于比自我更大的事情 | 暂时抛开自己 |
| 家庭和朋友 | 用时间表达爱、支持、分享知识 | 优质时间而非数量 |
| 穷人和病人 | 帮助、照顾、陪伴 | 博爱精神 |
| 机制 | 改善、优化、传承 | 创造持久价值 |

### 4. 博爱的意义
- 自由和平等是反义词，互相排斥
- 博爱是克服这对矛盾的关键
- 贫穷并非穷人关心的问题，是我们的责任



## 三、动作建议

### 立即执行
- [ ] 思考如何更好地服务同事和顾客
- [ ] 确定能给予家人和朋友的优质时间
- [ ] 考虑是否能够捐出收入的10%

### 短期行动（1-3个月）
- [ ] 找到帮助穷人和病人的方式
- [ ] 改善自己所在的工作机制
- [ ] 从属于一项比自己更大的任务

### 中期行动（3-12个月）
- [ ] 建立持续的服务习惯
- [ ] 参与公益项目
- [ ] 创造持久的社会价值



## 五、思考问题

1. **服务意识**：我是否在为他人提供服务？我的收入是否来自别人的感谢？

2. **时间分配**：我是否给家人和朋友足够的优质时间？

3. **社会责任**：我是否关注穷人和病人？我是否在帮助他人？

4. **机制贡献**：我是否在改善我所在的机制？我是否创造持久价值？

5. **捐赠意愿**：我是否愿意捐出收入的10%？我是否相信给予会带来更多？
