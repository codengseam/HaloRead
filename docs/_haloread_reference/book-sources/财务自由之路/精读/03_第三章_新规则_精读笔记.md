# 第三章 新规则 - 精读笔记

## 一、核心观点

### 信息时代的12条新规则

| 规则 | 旧规则 | 新规则 |
|------|--------|--------|
| 规则一 | 工作一次赚一次钱 | 一份工作多次盈利 |
| 规则二 | 出售生产力 | 出售好想法 |
| 规则三 | 找稳定职位的榜样 | 向热爱工作的人看齐 |
| 规则四 | 公司和国家关心我们 | 我们必须关心自己 |
| 规则五 | 躲避风险 | 为盈利参与竞争，经历冒险 |
| 规则六 | 错误是不好的 | 错误是对未来的投资 |
| 规则七 | 学习在学校进行 | 终身学习，不断成长 |
| 规则八 | 连续性是好的 | 有时需完全重新定位 |
| 规则九 | 赚钱是辛苦的 | 赚钱是一项游戏 |
| 规则十 | 克服弱点 | 发展强项 |
| 规则十一 | 追求完美个性 | 解决弱点或转化为强项 |
| 规则十二 | 工作只为赚钱 | 工作包含学习+定位+日常工作 |

### 规则详解

**规则一：并非一次，而是多次**
- 投资者从1欧元重复赚钱
- 作家、作曲家、发明家从版权获利
- 企业家从盈利中获益
- 关键：用想法换金钱，而非用时间换金钱

**规则四：关心自己的安全感**
- 公司不再为员工承担终身责任
- 国家福利并非白送
- 安全感来自自身能力和转变准备

**规则六：错误是好事**
- 错误的6种反应：说谎、否认、辩解、推卸责任、放弃、学习
- 只有"学习"是正确的反应
- 成功之路总是要经历错误

**规则七：不断学习，不断成长（K.L.U.W.）**
- 读专业书籍
- 记录日志
- 参加研讨会
- 寻找榜样

**规则十二：工作、学习和定位是一体**
- 理想时间分配：70%日常工作 + 10%学习 + 20%定位
- 或理想分配：1/3日常工作 + 1/3学习 + 1/3定位

### 中期行动（3-12个月）
- [ ] 每年做些能多次赚钱的事情
- [ ] 每年参加3-4次研讨会
- [ ] 找到并研究榜样
- [ ] 发展一项特别才能



## 五、思考问题

1. **规则对比**：我目前遵循的是旧规则还是新规则？哪些规则我需要改变？

2. **多次收入**：我目前的工作是否只能获得一次收入？我能创造哪些多次收入来源？

3. **强项识别**：我的核心强项是什么？我是否在发展它？

4. **学习习惯**：我是否在持续学习？我的学习方式是什么？

5. **时间分配**：我的工作时间如何分配？是否包含学习和定位时间？
