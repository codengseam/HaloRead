## 题目信息

**题目信息**：LeetCode 104. Maximum Depth of Binary Tree / 剑指 Offer 55 - I

*   **原题链接**：[二叉树的深度](https://leetcode.cn/problems/er-cha-shu-de-shen-du-lcof/)
*   **难度**：Easy
*   **描述**：输入一棵二叉树的根节点,求该树的深度。

### 🛒 题目 "人话" 翻译

**"量身高"**。

*   这就好比量一个家族几代同堂。
*   树的深度 = `max(左孩子的深度, 右孩子的深度) + 1 (我自己)`。



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：观察规律** 🐢
*   **想法**：树的深度就是从根到最远叶子节点的层数。
*   **观察**：每个节点的深度 = max(左子树深度, 右子树深度) + 1。

**第二阶段：递归实现** 🐇
*   **想法**：用递归计算每个节点的深度。
*   **步骤**：
    1.  如果节点为空,返回 0
    2.  递归计算左子树深度
    3.  递归计算右子树深度
    4.  返回 max(左深度, 右深度) + 1

**第三阶段：优化实现** ✨
*   **想法**：可以用 BFS（层序遍历）来计算深度。
*   **方法**：每遍历一层,深度加 1。



### 🚗 代码 "带逛"

```python
def maxDepth(root):
    """
    计算二叉树的深度
    """
    # 1. 如果是空树,深度为 0
    if not root:
        return 0
        
    # 2. 递归计算左右子树的深度
    left_depth = maxDepth(root.left)
    right_depth = maxDepth(root.right)
    
    # 3. 取最大值 + 1
    return max(left_depth, right_depth) + 1
```
