## 题目信息

**题目名称**：数组中数字出现的次数

**LeetCode链接**：[数组中数字出现的次数](https://leetcode.cn/problems/shu-zu-zhong-shu-zi-chu-xian-de-ci-shu-lcof/)

**难度**：Medium

**描述**：一个整型数组 `nums` 里除两个数字之外，其他数字都出现了两次。请写程序找出这两个只出现一次的数字。

## 🛒 题目"人话"翻译

**"找不同"**。

*   大家都是成双成对的，只有两个人是单身狗。
*   怎么把这两个单身狗揪出来？

## 🧊 知识点"破冰"讲解

**异或 (XOR)** 的分组技巧。

1.  **第一步**：把所有数字异或一遍。
    *   成对的数字异或为 0，消掉了。
    *   剩下的结果是 `a ^ b`（a 和 b 是那两个单身狗）。
2.  **第二步**：`a ^ b` 肯定不为 0（因为 a != b）。
    *   说明二进制里至少有一位是 1。
    *   这就意味着，在这一位上，a 和 b 是不一样的（一个是 0，一个是 1）。
3.  **第三步**：利用这一位，把所有数字分成两组。
    *   第一组：这一位是 0 的。
    *   第二组：这一位是 1 的。
    *   这样，a 和 b 就被强行分开了！而且其他的成对数字，肯定也会被分到同一组里（因为它们一样）。
4.  **第四步**：分别对两组再异或一次，分别得到 a 和 b。

## 💡 思路"推导"过程

这道题的核心在于如何将两个只出现一次的数字分开。如果只有一个数字出现一次，其他都出现两次，直接全部异或就能得到答案。但现在有两个，我们需要想办法把它们分开。

关键观察：两个不同的数字在二进制表示中，至少有一位是不同的。我们可以找到这一位，用它来分组。

## ❓ "为什么"的解释

**Q：为什么要找到最低位的1？**

**A：** 因为这一位能区分两个不同的数字。如果某一位是1，说明在这一位上，两个数字一个是0，一个是1。这样我们就可以把所有数字按照这一位是否为1分成两组，每组中只有一个单身狗，其他都是成对的。

## 🚗 代码"带逛"

```python
def singleNumbers(nums):
    # 1. 全员异或
    ret = 0
    for n in nums:
        ret ^= n
        
    # 2. 找到区分度（最低位的 1）
    # ret & (-ret) 是一个位运算 trick，能快速找到最低位的 1
    div = 1
    while div & ret == 0:
        div <<= 1
        
    # 3. 分组异或
    a, b = 0, 0
    for n in nums:
        if n & div:
            a ^= n
        else:
            b ^= n
            
    return [a, b]
```
