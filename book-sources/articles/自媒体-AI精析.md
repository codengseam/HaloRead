> *   **核心定义**：自媒体不是“什么都发”，而是**在特定赛道（如育儿、职场、兴趣）建立可识别、可持续的内容标签**，从而吸引精准用户。
> *   **关键知识点1：赛道选择**
>     *   **定义与示例**：选择“自身有优势（专业/经历）”或“有强烈兴趣（能持续输出）”的领域。例如视频中的“育儿绘本故事”，精准对标“家长”人群。
>     *   **常见错误**：选择过于宽泛的赛道（如“生活分享”），导致用户画像模糊，变现困难。
>     *   **30秒实操**：花30分钟，在抖音/小红书搜索3个你感兴趣的领域，分析其头部账号的**内容形式、更新频率、变现方式**，判断自己能否模仿并坚持。
> *   **关键知识点2：爆款内容三段式公式**
>     *   **定义与示例**：**痛点/好奇点（开头3秒）+ 解决方案/干货展示（中间）+ 行动号召/价值升华（结尾）**。如视频开头：“两年前我月薪6000...（痛点），今天我分享4个技能...（方案），建议狠狠收藏...（号召）”。
>     *   **30秒实操**：为你选定的赛道，用上述公式写一个15秒短视频的文案框架。
> 
> **🔹 技能二：AI技能（核心是“工作流设计”与“提示词工程”）**
> *   **核心定义**：AI技能不是学会所有工具，而是**用少数工具串联成一个高效的工作流**，将创意快速产品化。
> *   **关键知识点1：AI辅助内容生产工作流**
>     *   **定义与示例**：视频中演示了从“文案→图片→视频”的完整流水线：**ChatGPT（生成文案和分镜）→ 绘图AI（如Midjourney/小豆生成图片）→ 视频合成AI（如小梦/剪映自动成片）**。
>     *   **30秒实操**：用ChatGPT或文心一言，输入：“请为3-6岁儿童生成一个关于‘分享’的简短绘本故事大纲，包含5个场景描述。”
> *   **关键知识点2：有效提示词（Prompt）三要素**
>     *   **定义与示例**：**角色（你是什么）+ 任务（要做什么）+ 格式/要求（输出成什么样）**。例如：“假设你是一位资深育儿专家（角色），请将上文故事改写为5个分镜头脚本（任务），每个脚本包含场景描述和旁白（格式）。”
>     *   **常见错误**：指令模糊，如“写个故事”，AI输出结果不可控。

> *   **关键知识点1：Excel数据透视表（80%场景的核心）**
>     *   **定义与示例**：用于快速分类汇总。例如，销售数据中，拖拽“产品”到行，“销售额”到值，1秒得出各产品总销售额。
>     *   **30秒实操**：打开任意有分类和数字的数据表，使用【插入→数据透视表】，尝试将不同字段拖入“行”和“值”区域，观察汇总结果变化。
> *   **关键知识点2：AI表格工具（ChatExcel类）的对话式分析**
>     *   **定义与示例**：用自然语言命令AI处理表格。如：“将C列日期按月份分组，并计算每个月的销售总额”。
>     *   **常见错误**：问题描述不清。应具体到列名和计算方式。

> **🔹 技能四：信息获取能力（核心是“信源质量”与“批判思维”）**

> *   **关键知识点2：信息处理FACT法则**
