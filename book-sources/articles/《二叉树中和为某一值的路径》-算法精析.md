## 题目信息

**题目信息**：LeetCode 113. Path Sum II / 剑指 Offer 34

*   **原题链接**：[Path Sum II](https://leetcode.cn/problems/path-sum-ii/)
*   **难度**：Medium
*   **描述**：给你二叉树的根节点 `root` 和一个整数目标和 `targetSum` ,找出所有 **从根节点到叶子节点** 路径总和等于给定目标和的路径。

### 🛒 题目 "人话" 翻译

**"寻宝路线"**。

*   你从山顶出发。
*   每走一步捡起一个金币。
*   要求：必须一直走到山脚（叶子节点）,并且手里的金币总数刚好等于 `target`。
*   把所有符合要求的路线都画下来。



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：暴力遍历** 🐢
*   **想法**：遍历所有从根到叶子的路径,计算每条路径的和。
*   **问题**：如何记录路径？如何回溯？

**第二阶段：DFS + 记录路径** 🐇
*   **想法**：用 DFS 遍历树,用一个列表记录当前路径。
*   **关键点**：
    1.  每到一个节点,就把它的值加入路径。
    2.  如果到达叶子节点,检查路径和是否等于 target。
    3.  如果等于,就把这条路径加入结果。
    4.  回溯时,要把当前节点从路径中移除。

**第三阶段：优化实现** ✨
*   **想法**：用递归实现 DFS。
    *   参数：当前节点、剩余目标值、当前路径
    *   终止条件：到达叶子节点且剩余值为 0
    *   递归：左右子树
    *   回溯：移除当前节点



### 🚗 代码 "带逛"

```python
def pathSum(root, target):
    res, path = [], []
    
    def recur(node, tar):
        if not node: return
        
        # 1. 做选择：把当前节点加入路径,目标值减去当前值
        path.append(node.val)
        tar -= node.val
        
        # 2. 判断是否到达叶子节点且满足目标
        if tar == 0 and not node.left and not node.right:
            res.append(list(path)) # 注意要拷贝一份 path,因为 path 后面会变
        
        # 3. 继续向下找
        recur(node.left, tar)
        recur(node.right, tar)
        
        # 4. 撤销选择（回溯）：退回上一步之前,要把当前节点吐出来
        path.pop()

    recur(root, target)
    return res
```
