## 题目信息

**题目信息**：LeetCode 509. Fibonacci Number / 剑指 Offer 10- I

*   **原题链接**：[Fibonacci Number](https://leetcode.cn/problems/fibonacci-number/)
*   **难度**：Easy
*   **描述**：写一个函数，输入 n ，求斐波那契（Fibonacci）数列的第 n 项。斐波那契数列的定义如下：F(0) = 0, F(1) = 1, F(N) = F(N - 1) + F(N - 2).

### 🛒 题目 "人话" 翻译

这就是著名的 **"兔子繁殖"** 问题。
*   第 0 个月有 0 对兔子。
*   第 1 个月有 1 对兔子。
*   从第 2 个月开始，每一对兔子每个月都会生出一对新兔子（假设兔子永远不死且一出生就有生育能力）。
*   问第 N 个月有多少对兔子？

简单说就是：**今天的数 = 昨天的数 + 前天的数**。

### 🧊 知识点 "破冰" 讲解

这道题是 **动态规划 (DP)** 的入门题，也是 **递归** 的反面教材。

*   **递归法**：`f(n) = f(n-1) + f(n-2)`。
    *   **问题**：这会造成大量的重复计算！比如算 `f(5)` 要算 `f(4)` 和 `f(3)`，算 `f(4)` 又要算 `f(3)` 和 `f(2)`。`f(3)` 被重复算了好几次。
    *   时间复杂度是恐怖的 O(2^N)。

*   **迭代法 (DP)**：
    *   既然只要知道前两个数就能算下一个，那我们只存这两个数不就行了？
    *   `a` (前天), `b` (昨天) -> `a+b` (今天)。
    *   然后往前挪一天：`a` 变成昨天的 `b`，`b` 变成今天的 `a+b`。

### 🚗 代码 "带逛"

这里展示一个 Python 特有的 **生成器 (Generator)** 写法，虽然题目只需要第 N 个，但如果面试官让你打印前 N 个，这个写法非常优雅且节省内存。

```python
def fib(num):
    """
    斐波那契数列生成器
    :param num: 生成的数量
    """
    a, b = 0, 1
    for i in range(num):
        yield b # 每次调用吐出一个数
        a, b = b, a + b

# 如果只需要第 n 个数：
def fib_nth(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```
