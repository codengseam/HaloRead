## 题目信息

**题目信息**：剑指 Offer 33. 二叉搜索树的后序遍历序列

*   **原题链接**：[二叉搜索树的后序遍历序列](https://leetcode.cn/problems/er-cha-sou-suo-shu-de-hou-xu-bian-li-xu-lie-lcof/)
*   **难度**：Medium
*   **描述**：输入一个整数数组,判断该数组是不是某二叉搜索树的后序遍历结果。

### 🛒 题目 "人话" 翻译

给你一串数字,问你：这串数字能不能摆成一棵 **二叉搜索树 (BST)**,而且是按 **"左 -> 右 -> 根"** 的顺序读出来的？

**BST 的铁律**：左子树全比根小,右子树全比根大。



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：观察规律** 🐢
*   **想法**：后序遍历是"左右根",所以最后一个肯定是根。
*   **观察**：BST 的特性是"左小右大"。
*   **推导**：如果能找到左右子树的分界线,就能递归验证。

**第二阶段：寻找分界线** 🐇
*   **想法**：从左往右扫描,找到第一个比根大的数。
*   **关键点**：这个位置之前都应该比根小（左子树）,之后都应该比根大（右子树）。
*   **验证**：如果右子树里有比根小的数,说明不是 BST。

**第三阶段：递归验证** ✨
*   **想法**：把问题分解成子问题。
    *   验证左子树是不是 BST
    *   验证右子树是不是 BST
*   **终止条件**：如果子数组长度 <= 1,肯定是 BST。



### 🚗 代码 "带逛"

```python
def verifyPostorder(postorder):
    """
    验证是否为 BST 的后序遍历
    """
    def recur(i, j):
        # i, j 是当前子数组的左右边界
        if i >= j: return True
        
        # 1. 找到根节点（最后一个元素）
        root_val = postorder[j]
        
        # 2. 划分左子树
        p = i
        while postorder[p] < root_val:
            p += 1
        # 此时 p 指向第一个属于右子树的节点（或者根节点）
        m = p 
        
        # 3. 检查右子树（必须全大于 root）
        while postorder[p] > root_val:
            p += 1
            
        # 4. 如果 p 没有走到 j,说明右子树里混进了小叛徒
        # 并且递归检查左子树 [i, m-1] 和右子树 [m, j-1]
        return p == j and recur(i, m - 1) and recur(m, j - 1)

    return recur(0, len(postorder) - 1)
```
