## 题目信息

**题目信息**：LeetCode 426. Convert Binary Search Tree to Sorted Doubly Linked List / 剑指 Offer 36

*   **原题链接**：[Convert Binary Search Tree to Sorted Doubly Linked List](https://leetcode.cn/problems/convert-binary-search-tree-to-sorted-doubly-linked-list/)
*   **难度**：Medium
*   **描述**：输入一棵二叉搜索树,将该二叉搜索树转换成一个排序的循环双向链表。要求不能创建任何新的节点,只能调整树中节点指针的指向。

### 🛒 题目 "人话" 翻译

**"变形金刚"**。

*   给你一棵树（BST）,让你把它"压扁"成一条链子。
*   **左指针** 变成 **前驱 (prev)**。
*   **右指针** 变成 **后继 (next)**。
*   顺序要是排好序的（从小到大）。
*   最后头尾还要连起来（循环）。



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：观察特性** 🐢
*   **想法**：BST 的中序遍历是有序的。
*   **观察**：双向链表也是有序的。
*   **推导**：可以在中序遍历的过程中,把节点连接起来。

**第二阶段：连接节点** 🐇
*   **想法**：用两个指针 `pre` 和 `cur`。
    *   `pre` 指向前一个访问的节点
    *   `cur` 指向当前访问的节点
*   **连接**：
    *   `pre.right = cur`（前驱的下一个是当前）
    *   `cur.left = pre`（当前的上一个是前驱）

**第三阶段：处理头尾** ✨
*   **问题**：如何找到头节点（最小的）？如何连接头尾？
*   **解法**：
    *   头节点：第一个访问的节点（`pre` 为 None 时）
    *   尾节点：最后一个访问的节点（遍历结束时的 `pre`）
    *   连接：`head.left = tail`, `tail.right = head`



### 🚗 代码 "带逛"

```python
def treeToDoublyList(root):
    if not root: return None
    
    # pre 用来记录上一个遍历到的节点
    # head 用来记录链表的头节点（最小的那个）
    pre, head = None, None
    
    def dfs(cur):
        nonlocal pre, head
        if not cur: return
        
        # 1. 递归左子树
        dfs(cur.left)
        
        # 2. 处理当前节点
        if pre:
            # 如果有前任,互相连接
            pre.right = cur
            cur.left = pre
        else:
            # 如果没有前任,说明我是第一个（头节点）
            head = cur
            
        # 更新前任为自己,方便下一个节点连我
        pre = cur
        
        # 3. 递归右子树
        dfs(cur.right)
    
    dfs(root)
    
    # 4. 处理头尾连接（循环链表）
    head.left = pre
    pre.right = head
    
    return head
```
