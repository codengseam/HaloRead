## 题目信息

*   **原题链接**：[Number of 1 Bits](https://leetcode.cn/problems/number-of-1-bits/)
*   **难度**：Easy
*   **描述**：编写一个函数，输入是一个无符号整数，返回其二进制表达式中数字位数为 '1' 的个数。

### 🛒 题目 "人话" 翻译

给你一个整数，比如 9。
*   9 的二进制是 `1001`。
*   里面有两个 `1`。
*   输出 2。

就是数数二进制里有多少个 1。

### 🧊 知识点 "破冰" 讲解

这道题考的是 **位运算 (Bit Manipulation)** 的一个神技：`n & (n-1)`。

*   **原理**：
    *   `n`：二进制表示。
    *   `n-1`：会把 `n` 最右边的那个 `1` 变成 `0`，并且把这个 `1` 右边的所有 `0` 变成 `1`。
    *   **结论**：`n & (n-1)` 的作用是 **"消除 n 的二进制表示中最后一个 1"**。

*   **比喻**：就像拆砖墙，每次只拆最下面的一块砖，拆了几次墙倒了（变成0），就说明原来有几块砖。

### 🚗 代码 "带逛"

```python
def num_of_1(n):
    """
    计算二进制中 1 的个数
    """
    ret = 0
    while n:
        # 只要 n 还不为 0，说明里面还有 1
        ret += 1
        # 消除掉最后一个 1
        n = n & n-1
    return ret
```
