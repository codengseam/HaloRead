## 题目信息

**题目信息**：LeetCode 剑指 Offer 67. 把字符串转换成整数

*   **原题链接**：[把字符串转换成整数](https://leetcode.cn/problems/ba-zi-fu-chuan-zhuan-huan-cheng-zheng-shu-lcof/)
*   **难度**：Medium
*   **描述**：写一个函数 StrToInt,实现把字符串转换成整数这个功能。不能使用 atoi 或者其他类似的库函数。

### 🛒 题目 "人话" 翻译

**"挑食的转换器"**。

*   给你一串乱七八糟的字符,比如 `"   -42 with words"`。
*   你要把它变成整数 `-42`。
*   **规则**：
    1.  丢掉开头的空格。
    2.  看看有没有正负号。
    3.  开始读数字,读到非数字就停。
    4.  如果读出来的数太大了（超过 int 范围）,就截断。



### 💡 思路 "推导" 过程

我们来模拟一下小白的思考过程：

**第一阶段：简单转换** 🐢
*   **想法**：遍历字符串,把每个字符转成数字,然后拼接起来。
*   **问题**：没有处理空格、正负号、溢出等边界情况。

**第二阶段：处理边界** 🐇
*   **想法**：逐步处理各种边界情况。
*   **步骤**：
    1.  去掉首尾空格
    2.  判断正负号
    3.  逐个读取数字字符
    4.  遇到非数字字符停止
    5.  处理溢出

**第三阶段：溢出检测** ✨
*   **关键点**：在乘 10 之前检查是否会溢出。
*   **方法**：
    *   如果 `res > INT_MAX // 10`,那么 `res * 10` 一定会溢出。
    *   如果 `res == INT_MAX // 10`,需要检查最后一位数字是否超过 7（对于正数）或 8（对于负数）。



### 🚗 代码 "带逛"

```python
def str_to_int(s):
    # 1. 去掉首尾空格
    s = s.strip()
    if not s: return 0
    
    # 2. 处理正负号
    res, i, sign = 0, 0, 1
    int_max, int_min = 2**31 - 1, -2**31
    boundary = int_max // 10
    
    if s[0] == '-':
        sign = -1
        i = 1
    elif s[0] == '+':
        i = 1
        
    # 3. 开始读数字
    for j in range(i, len(s)):
        # 遇到非数字,直接结束
        if not '0' <= s[j] <= '9':
            break
            
        # 4. 处理溢出（在乘 10 之前检查）
        if res > boundary or (res == boundary and s[j] > '7'):
            return int_max if sign == 1 else int_min
            
        # 字符转数字：ord(c) - ord('0')
        res = res * 10 + (ord(s[j]) - ord('0'))
        
    return sign * res
```
