## 题目信息

**题目信息**：剑指 Offer 17. 打印从1到最大的n位数

*   **原题链接**：[打印从1到最大的n位数](https://leetcode.cn/problems/da-yin-cong-1dao-zui-da-de-nwei-shu-lcof/)
*   **难度**：Easy
*   **描述**：输入数字 n，按顺序打印出从 1 到最大的 n 位十进制数。比如输入 3，则打印出 1、2、3 一直到最大的 3 位数 999。

### 🛒 题目 "人话" 翻译

这就好比汽车的 **里程表**。
给你一个 n 位的里程表，让你把它从 00...01 跑到 99...99。

### 🧊 知识点 "破冰" 讲解

*   **面试陷阱**：这道题在 C++ / Java 中是考 **大数越界** 问题。因为 n 可能很大，大到 long long 都存不下。这时候需要用 **字符串** 来模拟数字加法。
*   **Python 优势**：Python 的整数是自动扩容的，不会溢出（除非内存爆了）。所以直接打印在 Python 里是合法的，但在面试时最好提一下"如果是在 C++ 里，我需要用字符串模拟"。

### 🚗 代码 "带逛"

```python
def print_max_n(n):
    """
    打印从 1 到最大的 n 位数
    """
    # 10 的 n 次方减 1 就是最大的 n 位数
    # 比如 n=3, 10^3 - 1 = 999
    for i in range(1, 10 ** n):
        print(i)
```
