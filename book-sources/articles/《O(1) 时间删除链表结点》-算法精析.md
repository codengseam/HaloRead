## 题目信息

**题目信息**：剑指 Offer 18. 删除链表的节点

*   **原题链接**：[删除链表的节点](https://leetcode.cn/problems/shan-chu-lian-biao-de-jie-dian-lcof/)
*   **难度**：Easy
*   **描述**：给定单向链表的头指针和一个要删除的节点指针，定义一个函数在 O(1) 时间内删除该节点。

### 🛒 题目 "人话" 翻译

你排在队伍中间，突然想走。
*   **常规做法**：找到你前面的人，让他不再指着你，而是指着你后面的人。但这需要从头找，因为你不知道前面是谁（单向链表）。
*   **O(1) 做法**：你把你后面那个人的衣服扒下来穿上，变成他的样子，然后把他踢出去。
    *   **效果**：虽然走的是"肉体上的他"，但"精神上的你"（值）已经被替换掉了。在外界看来，就是"你"消失了。

### 🚗 代码 "带逛"

```python
def delete_node(link, node):
    """
    O(1) 时间删除链表结点
    :param link: 头节点
    :param node: 要删除的节点对象
    """
    # 1. 如果要删的是尾节点，没办法，只能从头找前驱
    # 因为后面没人了，没法玩"变身"把戏
    if node.next is None:
        while link:
            if link.next == node:
                link.next = None
            link = link.next
    # 2. 如果是中间节点
    else:
        # 把下一个节点的值拿过来（变身）
        node.val = node.next.val
        # 拿到下下个节点的指针
        n_node = node.next
        # 跳过下一个节点（踢人）
        node.next = n_node.next
        # 物理删除（可选）
        del n_node
```
