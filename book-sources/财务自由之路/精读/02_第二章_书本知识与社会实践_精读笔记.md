# 第二章 书本知识与社会实践 - 精读笔记

## 一、核心观点

### 1. 思考方式决定命运
- 正是习惯了的思考方式使我们变成今天这样
- 同样的思考方式不会使我们变成想成为的样子
- 五角星图左半部分和右半部分的人有着完全不同的思考方式

### 2. 两种教育的差异
| 来源 | 父母/学校 | 教练 |
|------|-----------|------|
| 知识来源 | 书本知识 | 社会实践 |
| 对待错误 | 不能犯错 | 犯错是好事 |
| 对待风险 | 不要冒险 | 不冒险一事无成 |
| 对待安全感 | 找安稳工作 | 利用能力创造安全感 |
| 对待金钱 | 金钱不重要 | 金钱是选择的可能性 |
| 对待权利 | 我有权利 | 首先看到义务 |

### 3. 是什么-做什么-有什么
**错误顺序**：想要拥有 → 做事情 → 成为某人
**正确顺序**：成为某人 → 做事情 → 拥有

- 首先通过克制学习成为怎样的人
- 然后去做正确的事情
- 最后自然会拥有梦寐以求的事物

### 4. 破产vs贫穷
- **破产**：一种暂时的状态
- **贫穷**：持续的、一种思维方式
- 关键区别：是否相信自己能够改变



## 三、动作建议

### 立即执行
- [ ] 审视自己的思考方式更接近哪一领域
- [ ] 识别自己从父母/学校继承的限制性信念
- [ ] 开始用"我怎样才能..."替代"我不能..."

### 短期行动（1-3个月）
- [ ] 寻找一位成功领域的榜样/教练
- [ ] 开始学习五角星图左半部分领域的知识
- [ ] 建立错误学习日志

### 中期行动（3-12个月）
- [ ] 系统学习投资、创业或专家定位知识
- [ ] 从实践中学习，接受犯错
- [ ] 建立新的思维习惯



## 五、思考问题

1. **思考方式评估**：我的思考方式更接近五角星图的哪一边？这种思考方式能帮我实现目标吗？

2. **信念审视**：我从父母和学校继承了哪些限制性信念？哪些需要改变？

3. **学习方式**：我主要通过书本学习还是实践学习？我是否害怕犯错？

4. **安全感来源**：我的安全感来自哪里？是外界还是自身能力？

5. **责任意识**：我更多关注"我的权利"还是"我的义务"？
