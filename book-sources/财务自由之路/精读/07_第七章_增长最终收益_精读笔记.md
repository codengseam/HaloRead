# 第七章 增长最终收益 - 精读笔记

## 一、核心观点

### 1. 五条思想基础定理
1. **每个人都是独立的**：不是为老板工作，而是为自己工作
2. **您得到的总是您赚到的东西**：想赚更多，就要拥有更多
3. **了解游戏规则才能取胜**：不了解规则的人处于劣势
4. **对于收入没有任何规则**：所有规则都是人为的，可以改变
5. **不是得到加薪，而是赚到加薪**：取决于您的不可替代性

### 2. 收入衡量的三个准则
1. 老板如何评估您的价值
2. 您如何评估自己的价值
3. 您的谈判技巧

### 3. 加薪15条规则
| 规则 | 德文首字母 | 核心要点 |
|------|-----------|----------|
| 1. 做到最好 | G | 总是做到最好，没有不重要的任务 |
| 2. 创造收入的活动 | E | 80%收入来自20%活动，优先完成 |
| 3. 乐于助人 | H | 帮助同事和客户，成为受欢迎的人 |
| 4. 热爱所做 | A | 热爱每一项工作，做到最好 |
| 5. 学习成长 | L | 终身学习，不断成长 |
| 6. 游戏心态 | T | 把工作视为游戏，轻松对待 |
| 7. 发展强项 | S | 通往顶级收入之路是发展强项 |
| 8. 提高自信 | E | 自信力决定收入 |
| 9. 全神贯注 | R | 同一时间只做一件事 |
| 10. 高度紧迫 | H | 尽可能快地完成工作 |
| 11. 不公开反对 | Ö | 沉默比争论更聪明 |
| 12. 心胸开阔 | H | 永远不要因小事动怒 |
| 13. 承担责任 | U | 做雄鹰，不做鸭子 |
| 14. 展示优势 | N | 永远不要表达怀疑 |
| 15. 要求加薪 | G | 您必须主动争取 |

### 4. 成功日志
- 每天记录5件成功完成的事
- 每月研究一次，找出用到的能力
- 建立自信力的关键工具



## 三、动作建议

### 立即执行
- [ ] 开始记录成功日志（每天5项）
- [ ] 识别20%创造收入的活动
- [ ] 制定加薪计划

### 短期行动（1-3个月）
- [ ] 每天练习一条加薪规则
- [ ] 制作三份清单（为何值得加薪、为公司做了什么、未来能做什么）
- [ ] 检测市场价值

### 中期行动（3-12个月）
- [ ] 与决策人约定会谈
- [ ] 进行加薪谈判
- [ ] 实现至少20%加薪

### 加薪谈判四步
1. 解释为什么喜欢为公司工作，表达感谢
2. 问老板如何评价你的工作，对表扬表达谢意
3. 补充突出优点和能力，指出未来能为公司做什么
4. 引到加薪话题，请老板给建议（不要先提具体数字）



## 五、思考问题

1. **价值评估**：我的工作对公司有多大价值？我是否充分展示了这个价值？

2. **关键活动**：哪20%的活动创造了我80%的收入？我是否优先完成它们？

3. **雄鹰vs鸭子**：我更像雄鹰还是鸭子？我如何变得更像雄鹰？

4. **自信力**：我的自信力如何？我是否记录了自己的成功？

5. **加薪准备**：我是否准备好要求加薪？我有足够的理由吗？
