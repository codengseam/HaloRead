# 第九章 将自己定义成专家 - 精读笔记

## 一、核心观点

### 1. 专家的高收入逻辑
- 专注于500欧元大钞，而非捡起20张5欧元钞票
- 通往高收入的途径是将自己定义成专家
- 专家兼具专业知识和经济头脑

### 2. 10条定位原则

| 原则 | 核心要点 | 说明 |
|------|----------|------|
| 1. 不求更好，但求不同 | 质量是前提，不同才是关键 | 找到与众不同的特点 |
| 2. 光做到卓越还不够，还要做到非凡 | 卓越不够，必须超凡脱俗 | 赢者得到一切 |
| 3. 争做第一人 | 宁做鸡头，不做凤尾 | 小领域更容易成为第一 |
| 4. 创造新的定位类型 | 如果不能成为第一，就创造新类别 | 艾米丽·艾尔哈特的例子 |
| 5. 宁可精细，不要宽泛 | 越精尖越好 | 钉子比石头更容易击中木板 |
| 6. 选择基本需求，不要特殊套路 | 基于基本需求定位 | 不依赖特定技术或方式 |
| 7. 选择属于我的目标群体 | 精准定义目标客户 | 了解客户的一切 |
| 8. 为他人解决一个问题 | 找到最紧急的问题 | 问题越大，专家越值钱 |
| 9. 把您做的事情说出来 | 公开宣传自己的专长 | 只有名气决定经济价值 |
| 10. 您来确定价格 | 与众不同才能定价 | 否则只能参与价格战 |

### 3. 创意的4个阶段
1. **准备阶段**：收集信息，提出问题
2. **对比阶段**：借鉴别人的主意
3. **酝酿阶段**：大脑持续思考
4. **恍然大悟阶段**：突然冒出想法

### 4. 专家定位的流程
```
找到乐趣所在 + 发现天赋能力 → 确定与众不同之处 → 选择精细领域 → 解决紧急问题 → 公开宣传 → 确定价格
```



## 三、动作建议

### 立即执行
- [ ] 每天拿出15分钟思考在哪方面能成为第一人
- [ ] 列出自己的乐趣和能力清单
- [ ] 找到乐趣与能力的交集

### 短期行动（1-3个月）
- [ ] 每天至少1小时用于定位练习
- [ ] 创建创意日志
- [ ] 研究成功人士的定位策略

### 中期行动（3-12个月）
- [ ] 确定自己的精细定位领域
- [ ] 开始公开宣传自己的专长
- [ ] 建立专家形象（写作、演讲等）

### 创意收集方法
- 记录想找到答案的问题
- 收集别人的成功主意
- 到领域最优秀的人那里学习
- 入睡前思考问题
- 随身携带纸笔记录灵感



## 五、思考问题

1. **独特性**：我有什么是别人没有的？我如何与众不同？

2. **第一人**：我能在哪个小领域成为第一人？

3. **精细定位**：我的领域是否足够精细？是否太宽泛？

4. **解决问题**：我的目标群体有什么最紧急的问题？我如何解决？

5. **公开宣传**：我是否让公众知道我的专长？我如何宣传自己？
