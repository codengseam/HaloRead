# 第一章 仓鼠之轮 - 精读笔记

## 一、核心观点

### 1. 赚钱是一项游戏
- 赚钱是一项人人都能参与并获利的游戏
- 大部分人不了解游戏规则，因循守旧，原地踏步
- 收入高低不取决于聪明程度或努力程度，而取决于是否了解游戏规则

### 2. 五种赚钱方式（五角星图）
```
        专家(E)
          △
         /  \
投资者(I)/    \企业家(U)
       /      \
      /        \
职员(A)---------自由职业者(F)
```

**左半部分（高收入领域）**：投资者、企业家、专家
**右半部分（传统领域）**：职员、自由职业者

### 3. 仓鼠之轮概念
- 许多人像仓鼠在轮子里奔跑，使尽全力却原地踏步
- 收入提高，开支同步增长，负债日益增加
- 四种仓鼠之轮：职员/自由职业者、投资者、企业家

### 4. 三种赚更多钱的方式
1. 在一个领域提高收入（一年内可提高20%）
2. 在一个领域工作，同时从另一个领域获取额外收益
3. 实现领域飞跃（带来最佳收益）

### 5. 梦想收入设定
- 成功人士不将目标依赖于目前状况
- 他们描绘想要追求的事物，然后思考如何变成那样的人
- 以目标为导向，而非以自己为导向



## 三、动作建议

### 立即执行
- [ ] 写下3年后想达到的月收入目标（纯收入）
- [ ] 分析自己目前处于五角星图的哪个领域
- [ ] 评估自己是否处于仓鼠之轮中

### 短期行动（1-3个月）
- [ ] 确定想要进入的五角星图领域
- [ ] 开始学习该领域的游戏规则
- [ ] 寻找该领域的榜样/教练

### 中期行动（3-12个月）
- [ ] 开辟第二条收入渠道
- [ ] 建立自己的"机制"雏形
- [ ] 每月记录收入日志



## 五、思考问题

1. **自我定位**：我目前处于五角星图的哪个领域？这个领域是否符合我的性格和目标？

2. **仓鼠之轮检测**：我是否处于仓鼠之轮中？我的收入增长是否被开支增长抵消？

3. **梦想收入**：如果不受任何限制，3年后我想达到什么收入？我是否敢于写下这个数字？

4. **领域选择**：我是否愿意为进入更高收入的领域付出代价（风险、学习、改变）？

5. **机制思维**：我目前的工作/事业是否创造了可离开我独立运转的机制？
