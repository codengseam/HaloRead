## 一、玄幻流派核验

### 1.1 定义与边界

- **核实结论**：【版本可查】"玄幻"为中国网络文学原创类型词，无严格西方对应分类
- **定义**：以东方玄学、修炼体系、神话元素为基础，融合西方魔法 / 异界 / 巨兽等元素的幻想小说大类。区别于"仙侠"的内核在于：玄幻**不限定东方修真体系**，可包含斗气、魔法、武魂、血脉、异火等多元力量体系；仙侠则以道教修真为底色
- **边界辨析**：
  - 玄幻 vs 仙侠：玄幻力量体系多元且可自创（如斗气、武魂），仙侠遵循"炼气→筑基→结丹→元婴"等道教修真阶梯
  - 玄幻 vs 奇幻：奇幻多偏向西式魔法 / 剑与魔法，玄幻是中文圈本土生长的混合类型
- **流派起源**：黄易《大唐双龙传》被视为现代玄幻奠基，但真正网络化、流派化是 2003 年起点中文网崛起之后
- **撰写建议**：正文写「玄幻是中国网络文学最具代表性的本土大类，力量体系多元（斗气 / 武魂 / 血脉 / 异火皆可），与仙侠的核心区别在于不绑定道教修真阶梯」。

### 1.5 核心套路

- **力量阶梯**：明确等级体系（斗气 / 武魂 / 血脉 / 修炼境界），等级即战力即爽点
- **金手指载体**：异火榜、武魂觉醒、血脉传承、神秘玉佩 / 戒指老爷爷（《斗破苍穹》药老）
- **逆袭叙事**：天才陨落 → 废柴 → 觉醒金手指 → 步步登天
- **榜单驱动**：异火榜、神兵榜、强者榜——可收集式金手指制造长线追更动力
- **世界观扩张**：从家族 → 学院 → 帝国 → 大陆 → 星空 → 宇宙（"地图升级"是玄幻核心节奏）

### 1.6 读者画像与爽点设计

- **读者画像**：据公开报道，玄幻读者以 18-35 岁男性为主，学生与初入职场群体占比高，付费意愿强，是起点中文网男频第一大流量池
- **爽点设计**：
  - **打脸爽**：废柴身份被嘲讽 → 实力碾压打脸 → 围观者震惊
  - **收集爽**：异火 / 神兵 / 武魂逐一收服，制造"集齐"期待
  - **升级爽**：等级突破带来战力跃升，章节末尾留突破悬念
  - **称霸爽**：从家族称霸 → 大陆称霸 → 宇宙称霸的层级递进
- **数据硬约束**：不编造具体万订数字；可写"据公开报道，《斗破苍穹》长期位列起点总点击 / 推荐榜前列，是起点玄幻类现象级作品"

## 二、都市流派核验

### 2.1 定义与边界

- **核实结论**：【版本可查】"都市"为起点中文网主流分类之一
- **定义**：以现代都市为背景，围绕职场、商战、神豪、修真归来、异能、医道、兵王等元素展开的网络小说。区别于玄幻的核心在于：都市**背景为当代现实世界**（即便加入异能 / 修真元素，地理与社会结构仍是现代都市）
- **子流派**：都市神豪流、都市修真流、都市医道流、都市兵王流、都市异能流、都市娱乐明星流
- **边界辨析**：都市 + 异能 ≠ 玄幻——都市异能仍以现代都市为社会底座，玄幻则构建异世界
- **撰写建议**：正文写「都市流以当代现实社会为底座，子流派包括神豪 / 医道 / 兵王 / 修真归来 / 娱乐明星等，与玄幻的核心区别是社会背景是否为现代现实」。

### 2.2 代表作核验：会说话的肘子《大王饶命》《夜的命名术》

- **核实结论**：【核实】作者本名、生年、籍贯、作品数据均经多源证实

- **流派归属**：都市 + 异能 + 群像 + 赛博朋克元素（《夜的命名术》融合表世界 / 里世界设定）
- **权威出处**：起点中文网作家专区
- **撰写建议**：正文写「会说话的肘子（本名任禾）代表作《大王饶命》《夜的命名术》，以都市异能为底座融合赛博朋克群像叙事，是 2020 年后都市流现象级作家」。

- **核实结论**：【版本可查】作者笔名、作品名、连载时间均经多源证实，但精确字数与起止日期多源数据不一
- **作者**：卖报小郎君，起点中文网白金作家（笔名，本名公开报道较少，正文引用建议只写笔名）

### 2.4 核心套路

- **金钱爽点**：神豪流以"花钱"为核心爽点（系统要求花钱才能升级）
- **职业爽点**：医道流妙手回春、兵王流拳打恶霸、娱乐流打造巨星

### 2.5 读者画像与爽点设计

- **读者画像**：据公开报道，都市流读者年龄层略高于玄幻（25-40 岁男性为主），覆盖上班族 / 蓝领群体，对现实社会压力有共鸣，番茄免费平台都市类目流量极大
- **爽点设计**：
  - **现实代入爽**：主角处境贴近读者（穷、被歧视、被甩），代入感强
  - **金钱碾压爽**：神豪流直接以花钱解决一切问题
  - **地位翻转爽**：从被看不起到被人仰望
  - **情感爽**：前女友后悔、女总裁倒贴等情感补偿叙事
- **数据硬约束**：不编造订阅数字；可写"据公开报道，《大奉打更人》长期占据起点月票榜前列，是 2020-2022 年都市探案流现象级作品"

## 三、仙侠流派核验

### 3.1 定义与边界

- **核实结论**：【版本可查】"仙侠"为中国网络文学原创类型词，源自传统武侠 + 神话
- **定义**：以道教修真为底色，围绕"长生求道"展开的幻想小说。区别于玄幻的核心在于：仙侠**严格遵循道教修真体系**（炼气 / 筑基 / 结丹 / 元婴 / 化神 / 炼虚 / 合体 / 大乘 / 真仙），世界观以"修真界"为主
- **边界辨析**：
  - 仙侠 vs 玄幻：仙侠力量体系绑定道教修真阶梯，玄幻力量体系多元可自创
  - 仙侠 vs 修真：业内常将二者混用，严格区分时"修真"是更宽泛的概念（含现代修真），"仙侠"特指传统修真界背景
- **流派起源**：还珠楼主《蜀山剑侠传》被视为仙侠文学鼻祖；网络仙侠流派化自忘语《凡人修仙传》开始
- **撰写建议**：正文写「仙侠以道教修真为底色，力量体系遵循炼气→筑基→结丹→元婴等修真阶梯，与玄幻的核心区别在于是否绑定道教修真体系」。

### 3.2 代表作核验：忘语《凡人修仙传》（凡人流开山）

- **核实结论**：【核实】作者笔名、作品名、连载时间、字数均经中国作家网 / 起点作家专区交叉证实
- **作者**：忘语，起点中文网白金作家（笔名）
- **作品数据**：
  - 《凡人修仙传》：2008-02 至 2013-09，约 771 万字，起点中文网，已完结
- **流派归属**：**凡人流开山之作**——以普通凡人韩立为主角，无显赫背景、无逆天天赋，靠谨慎、算计、机缘步步修真
- **等级体系核验**（多源一致）：炼气 → 筑基 → 结丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 真仙
- **权威出处**：中国作家网网络文学栏目、起点中文网
- **撰写建议**：正文写「忘语《凡人修仙传》（2008-2013，约 771 万字）开创凡人流，主角韩立以普通凡人身份靠谨慎与机缘步步修真，等级体系为炼气→筑基→结丹→元婴→化神→炼虚→合体→大乘→真仙」。

### 3.4 代表作核验：烽火戏诸侯《剑来》《雪中悍刀行》

- **长生追求**：与天争寿，逆天改命——核心母题

### 3.6 读者画像与爽点设计

- **读者画像**：据公开报道，仙侠读者年龄层略长于玄幻（25-40 岁男性为主），偏好长篇慢热、世界观严密的作品，对文笔有要求
- **爽点设计**：
  - **境界突破爽**：每次境界突破带来战力质变与地位提升
  - **机缘夺宝爽**：秘境探宝、抢夺天材地宝
  - **智斗爽**：凡人流以谨慎算计为核心爽点（区别于无脑碾压）
  - **长生爽**：从凡人到真仙的生命跃迁
- **数据硬约束**：不编造订阅数字；可写"据公开报道，《凡人修仙传》是起点仙侠类最具影响力的开山作品之一"

## 四、系统流派核验

### 4.1 定义与边界

- **核实结论**：【版本可查】"系统流"为网络文学原生概念，以"系统"作为金手指载体
- **定义**：主角获得一个"系统"（具象化的游戏化界面 / 任务发布者），通过完成任务获得奖励、升级能力。系统是金手指的具象化与自动化
- **边界辨析**：
  - 系统流 vs 修炼流：系统是金手指载体，修炼流是力量体系；二者可叠加（系统 + 修炼）
  - 系统流 vs 无限流：无限流以"主神空间"为核心（如《无限恐怖》），主神空间可视为"类系统"，但严格意义上无限流是独立流派
- **流派起源**：**【争议】** 见 4.2
- **撰写建议**：正文写「系统流以'系统'作为金手指的具象化载体，通过任务—奖励机制驱动主角升级，是网络文学原生金手指模式」。

### 4.2 系统流开山归属争议核验

- **核实结论**：【争议】系统流开山作品存在多源说法，无定论
- **多方观点**：
  - **观点 A（最早可考）**：2007 年起点书号 136613《魔兽世界之现实世界的法师》——最早出现"系统"界面的可考作品之一
  - **观点 B（带动潮流）**：2011 年 8 月《修真门派掌门人》——带动系统流大规模流行

- **撰写建议**：开山归属必须标注"争议"，列多方观点，不得武断归一。

## 五、重生流派核验

### 5.1 定义与边界

- **核实结论**：【版本可查】"重生流"为网络文学原生概念
- **定义**：主角死亡后重生回过去某个时间点，带着前世记忆重新生活。区别于穿越的核心在于：重生是**回到自己经历过的时间**，穿越是去到陌生时空
- **边界辨析**：
  - 重生 vs 穿越：重生 = 回到自己过去；穿越 = 去到陌生时空（异界 / 古代 / 平行世界）
  - 重生 vs 回档：回档是死后回到存档点（受系统控制），重生是主动带着记忆回到过去
- **流派起源**：**【核实】** 周行文《重生传说》（2004-2005，起点中文网）被视为都市重生小说滥觞之作
- **撰写建议**：正文写「重生流以主角带着记忆回到过去为核心，与穿越的区别是回到自己经历过的时间；周行文《重生传说》（2004-2005）是都市重生小说滥觞之作」。

### 5.2 重生流开山归属核验

- **核实结论**：【核实】周行文《重生传说》经中国作家网确认为"都市重生小说滥觞之作"
- **作品数据**：
  - 《重生传说》：2004-2005，起点中文网，已完结（具体字数多源不一，建议只写连载时间）
- **权威出处**：中国作家网网络文学栏目
- **撰写建议**：正文可写「周行文《重生传说》（2004-2005，起点中文网）被中国作家网认定为都市重生小说滥觞之作」。

### 5.3 代表作核验：月关《回到明朝当王爷》

- **核实结论**：【版本可查】作者本名、作品名均经多源证实，但连载起止时间多源不一
- **作者全名**：月关，本名**魏立军**，起点中文网白金作家
- **作品数据**：
  - 《回到明朝当王爷》：起点中文网，已完结
  - **时间冲突标注**：多源数据不一——一源为 2006-11-08 至 2008-01-17，另一源为 2007-03-12 至 2010-12-25。正文引用建议只写"2006-2008 前后"或"约 2007 年连载"
- **流派归属**：历史穿越 + 重生（严格意义上是穿越到古代成为王爷，业内常归"历史穿越 / 重生"）
- **撰写建议**：正文写「月关（本名魏立军）《回到明朝当王爷》（约 2007 年前后连载）是历史穿越 / 重生流代表作」。注意精确日期多源不一。

### 5.4 代表作核验：荣小荣《逍遥小书生》《如意小郎君》

- **核实结论**：【核实】作者笔名、作品数据均经多源证实
- **作者**：荣小荣，阅文集团大神作家（笔名）
- **作品数据**：
  - 《逍遥小书生》：2016-10-08 至 2018-05-24，约 287 万字，起点中文网，已完结
  - 《如意小郎君》：2018-05-24 至 2019-08-26，约 247 万字，起点中文网，已完结

### 5.5 核心套路

- **弥补遗憾**：前世错过的人 / 事，今生重新把握
- **降维打击**：带着现代知识回到古代 / 过去，形成降维优势
- **蝴蝶效应**：改变过去引发连锁反应
- **情感补偿**：前世的遗憾今世弥补，是重生流核心情感驱动

  - **先知爽**：知道未来走势，逢凶化吉
  - **降维爽**：现代知识降维打击古代 / 过去
  - **情感补偿爽**：前世错过的人今世重逢
- **数据硬约束**：不编造订阅数字；可写"据公开报道，重生流是男频长盛不衰的金手指模式"

## 六、历史与科幻流派核验

### 6.1 定义与边界

- **核实结论**：【版本可查】"历史"与"科幻"为起点中文网主流分类
- **定义**：
  - **历史流**：以中国古代历史为背景，包含架空历史、历史穿越、历史同人等子流派
  - **科幻流**：以科学幻想为核心，包含硬科幻、软科幻、末世、星际、赛博朋克等子流派
- **边界辨析**：
  - 历史穿越 vs 重生：历史穿越是去到古代（异时空），重生是回到自己过去
  - 科幻 vs 玄幻：科幻以科学原理为底色（即便软科幻也讲逻辑），玄幻以玄学为底色
- **撰写建议**：正文写「历史流以中国古代为背景，科幻流以科学幻想为核心；历史穿越与重生的区别在于是否回到自己经历过的时间」。

### 6.2 代表作核验：远瞳《异常生物见闻录》《深海余烬》

- **核实结论**：【核实】作者本名、生年、作品数据、获奖信息均经多源证实
- **作者全名**：远瞳，本名**高俊夫**，1988-12-26 生，起点中文网白金作家
- **作品数据**：
  - 《异常生物见闻录》：2014 年首发，约 525 万字，起点中文网，已完结
  - 《深海余烬》：2022-07-10 至 2024-05-02，约 259 万字，起点中文网，已完结
- **获奖信息核验**：《深海余烬》获**第 33 届中国科幻银河奖最佳网络科幻小说奖**——此为可查证权威奖项
- **流派归属**：科幻 + 末世 + 星际 + 克苏鲁元素（远瞳被称为"网文科幻第一人"，此说法流传广，正文引用建议加"业内俗称"）
- **权威出处**：起点中文网、中国科幻银河奖官方公告
- **撰写建议**：正文写「远瞳（本名高俊夫）代表作《异常生物见闻录》《深海余烬》（2022-2024，约 259 万字），《深海余烬》获第 33 届中国科幻银河奖最佳网络科幻小说奖」。

### 6.3 历史流代表作补充

- **月关《回到明朝当王爷》**：见 5.3 节，历史穿越 / 重生流代表作
- **荣小荣《逍遥小书生》《如意小郎君》**：见 5.4 节，历史穿越轻松向代表作
- **撰写建议**：历史流与重生流有交叉，月关、荣小荣作品常同时归"历史穿越 / 重生"两类。

### 6.4 核心套路

**历史流**：
- 现代人穿越古代，带着现代知识降维打击
- 改变历史走向（架空历史）
- 朝堂权谋、军事争霸、科技种田
- 与历史名人互动

**科幻流**：
- 末世生存（丧尸 / 异变 / 资源匮乏）
- 星际争霸（舰队 / 文明对决）
- 赛博朋克（虚拟现实 / 阶级对立）
- 克苏鲁（宇宙恐怖 / 不可名状）
- 硬科幻（科学原理推演）

### 6.5 读者画像与爽点设计

- **读者画像**：据公开报道，历史流读者偏好有历史知识的群体（30-50 岁男性为主），科幻流读者年龄层较广（20-40 岁男性），对世界观严密性有要求
- **爽点设计**：
  - **历史流爽点**：改变历史走向、降维打击古人、与历史名人称兄道弟
  - **科幻流爽点**：世界观揭秘、文明对决、科技碾压、宇宙探索
- **数据硬约束**：不编造订阅数字；可写"据公开报道，远瞳是起点科幻类最具代表性的白金作家"

## 九、男频题材篇 6 章写作大纲建议

### 第 1 章 · 玄幻流：力量阶梯与逆袭叙事

- **核心**：力量体系多元 + 等级即战力即爽点 + 地图升级节奏
- **案例**：辰东《神墓》（太古战争流开山）/ 天蚕土豆《斗破苍穹》（异火榜 + 斗气体系）/ 我吃西红柿《吞噬星空》（玄幻 + 科幻融合）
- **撰写要点**：
  - 讲清玄幻与仙侠的边界（力量体系是否绑定道教修真）
  - 拆解"废柴逆袭"叙事模型
  - 讲清"地图升级"节奏（家族→学院→帝国→大陆→星空→宇宙）
  - 引用《斗破苍穹》等级体系作为力量阶梯设计范例
- **避免**：不得将《斗罗大陆》归入系统流（武魂系统是力量体系）

### 第 2 章 · 都市流：现实代入与金手指补偿

- **核心**：现代都市背景 + 现实社会关系爽点 + 多子流派（神豪 / 医道 / 兵王 / 修真归来 / 娱乐）
- **案例**：会说话的肘子《夜的命名术》（都市 + 异能 + 赛博朋克）/ 卖报小郎君《大奉打更人》（探案 + 修真 + 朝堂）
- **撰写要点**：
  - 讲清都市流与现实社会压力的共鸣机制
  - 拆解"前女友后悔""岳父刮目相看"等社会关系爽点设计
  - 讲清神豪流"花钱即升级"的反向金手指逻辑
  - 番茄免费平台都市类目流量分析
- **避免**：《大奉打更人》精确起止日期多源不一，避免写死

### 第 3 章 · 仙侠流：修真阶梯与凡人逆袭

- **核心**：道教修真底色 + 严格境界划分 + 长生求道母题
- **案例**：忘语《凡人修仙传》（凡人流开山）/ 耳根《仙逆》/ 烽火戏诸侯《剑来》（连载中）
- **撰写要点**：
  - 讲清仙侠与玄幻的边界（是否绑定道教修真体系）
  - 拆解凡人流"谨慎算计"vs 玄幻"无脑碾压"的爽点差异
  - 引用《凡人修仙传》9 阶修真体系作为境界设计范例
  - 讲清"宗门体系""资源争夺""心魔道心"等核心母题
- **避免**：不得将凡人流开山归给其他作品；"三巨头"说法需加"业内俗称"

### 第 4 章 · 系统流：任务奖励与机械降神

- **核心**：系统作为金手指具象化 + 任务—奖励机制 + 数据化反馈
- **案例**：横扫天涯《天道图书馆》（系统 + 师道流）
- **撰写要点**：
  - 讲清系统流开山**争议**（不得武断归一），列多方观点
  - 拆解签到流、抽奖流、商城流等子流派
  - 讲清系统作为"机械降神"在关键时刻的救命作用
  - 番茄免费平台系统流流量分析
- **避免**：不得将《斗罗大陆》归入系统流（武魂系统是力量体系，非任务—奖励系统）

### 第 5 章 · 重生流：记忆金手指与遗憾补偿

- **核心**：带着记忆回到过去 + 弥补遗憾 + 降维打击
- **案例**：周行文《重生传说》（重生流滥觞）/ 月关《回到明朝当王爷》（历史穿越 / 重生）/ 荣小荣《逍遥小书生》（轻松向）
- **撰写要点**：
  - 讲清重生与穿越的边界（是否回到自己经历过的时间）
  - 拆解"前世记忆"作为金手指的设计逻辑
  - 讲清"情感补偿"作为重生流核心情感驱动
  - 讲清重生流读者画像（有社会阅历群体为主）
- **避免**：月关《回到明朝当王爷》精确日期多源不一，避免写死

### 第 6 章 · 历史与科幻流：降维打击与世界观揭秘

- **核心**：历史流（现代人穿越古代降维打击）+ 科幻流（世界观揭秘 + 文明对决）
- **案例**：远瞳《深海余烬》（获第 33 届中国科幻银河奖）/ 远瞳《异常生物见闻录》
- **撰写要点**：
  - 讲清历史穿越与重生的边界
  - 讲清科幻与玄幻的边界（科学原理 vs 玄学）
  - 拆解远瞳"克苏鲁 + 末世 + 星际"融合世界观
  - 引用《深海余烬》获银河奖作为网文科幻标杆案例
- **避免**："远瞳是网文科幻第一人"说法需加"业内俗称"

## 十一、未决事项与后续建议

1. **系统流开山归属** —— 存在多源说法（2007 起点最早可考 / 2011《修真门派掌门人》带动潮流 / 2007《无限恐怖》开创类系统流），正文必须标注"争议"并列多方观点，不得武断归一。
2. **签到流开山归属** —— 多本早期作品并存，无定论。正文建议只讲签到流作为系统流子流派的特征，不讲开山。
3. **《吞噬星空》字数** —— 多源数据不一（342 万 / 478 万 / 582.6 万），正文引用建议写区间值或仅注连载时间。
4. **《大奉打更人》连载时间** —— 多源数据不一（2019-12 至 2022-06 vs 2020-03 至 2021-08），正文引用建议只写"约 2020-2022"。
5. **《回到明朝当王爷》连载时间** —— 多源数据不一（2006-11 至 2008-01 vs 2007-03 至 2010-12），正文引用建议只写"约 2007 年前后"。
6. **"起点三巨头""网文科幻第一人"等俗称** —— 流传广但无权威出处，正文引用必须加"业内俗称"，不得作为定论断言。
7. **订阅 / 万订数据** —— 不编造具体数字。如需讲商业表现，用"据公开报道长期位列起点月票榜 / 总点击榜前列"等模糊表述。
