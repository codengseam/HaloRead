# 抖音心理科普短视频课 · 写作提示词与执行方案存档

> 本文件为专栏写作过程使用的提示词、执行方案与验收记录存档，不进入专栏展示（下划线前缀文件被 check_book_structure.py 豁免、被 build_site.py 排除出站点）。保留用于复盘与迭代。

## 一、专栏定位

- 主题：抖音心理科普短视频创作全链路（从0到持续输出）
- 读者：想用抖音做心理科普的普通人（不一定是心理学专业出身）
- 核心目标：让普通人照着这个专栏能从0跑通"定位→算法→脚本→剪辑→心理内容内核→运营→合规→变现"全链路，做出既爆款又合规的心理科普短视频
- archetype：modern（现代方法论桶）
- 输出目录：`/workspace/output/抖音心理科普短视频课/`
- 输出：62 篇正文 + 4 个辅助文件（_meta.yaml / _心理学理论速查.md / _参考资料.md / _提示词.md）

## 二、执行流程（已落地）

1. **多专家团评估方案**：用 Task 工具启动 3 个 subagent（架构师/内容专家/规则）并行评审 13 模块大纲，整合意见后定稿。关键修订：篇数52→62（合规伦理3→6篇、内容内核5→7篇、素材库6→12篇）、chapter_sort连续1-13、内容线前移（素材库+案例移到拍摄剪辑之前）、爆款案例换血（半佛仙人→替换为反面案例）、新建 _心理学理论速查.md 基石

3. **写资产文件**：创建 _心理学理论速查.md（理论基石）+ _参考资料.md（资料存档）+ _meta.yaml（13 chapters 映射）+ _提示词.md（执行方案存档）
4. **并行写作**：用 Task 工具启动 8 个 subagent 分两波并行写作（第一波 4 个写 31 篇基础模块 + 第二波 4 个写 31 篇应用模块）

6. **三方评审 + 修复**：3 个评审 subagent 五维度打分，修复硬伤后复跑质检

## 三、写作硬约束（写入每个写作 subagent 的 query）

1. 目录：`/workspace/output/抖音心理科普短视频课/`
2. 文件命名：两级路径 `{chapter}_{event}.md`，中文，下划线 `_` 分隔，禁止用 `-`，禁止子目录
3. 每篇必须有完整 frontmatter（title/book/chapter/event/sort/chapter_sort/created_at/source_agents 八字段）
4. archetype: modern（现代方法论桶）
5. 正文中文，每篇 2500-4000 字
6. 5 段结构固定段名 + 中文序号 + 冒号 + 副标题：`## 一、入戏：{灵魂短语}` … `## 五、践行：{灵魂短语}`；五段内分小节一律 `###`，禁止再起 `##`
7. 文末必须有 `## 参考来源` 标题（**不是"参考资料""参考文献""本章参考资料"**，content_quality.py:267-272 只认"参考来源"四字）
8. 禁内联跳转：禁用"（见XX篇）""（详见XX章）""（见上文/前文）"等一切带"见/详见/参见+篇/章"的括号跳转；提及他篇用自然语句"（XX 篇展开）"
9. 禁用"底层逻辑""底层操作系统"（0 容忍，出现即报）；"底层"二字单独可用但建议替换为"机制/原理/根本"
10. 术语词序铁律："英文+中文"词序（如「CBT 认知行为疗法」而非「认知行为CBT」），首次出现用全角括号隔断「中文全称（缩写）」，之后用中文
11. 子标题/H1 用"名词：洞察短语"格式（如「黄金3秒：开场的注意力争夺战」），禁裸2字词
12. 每篇开头一句边界声明（blockquote）：`> 本篇覆盖 X，不覆盖 Y（XX 篇展开）`
13. 文末加 `> 信息截止：2026-07`
14. 引号强制用「」，禁用 ASCII 双引号包中文
15. 「不是X而是Y」句式每篇 ≤3 处，改写多样化（与其说X不如说Y / 关键不在X而在Y / X是表象Y是内核）
16. 行内引用每千字 ≤3 处，同一出处连续出现只保留一处，普通叙述句不挂行内出处
17. 引用真实性：只引用 `_心理学理论速查.md` + `_参考资料.md` 中已 WebSearch 核验的条目，不杜撰书籍版本/作者/出版社/年份/量表数据；名人语录存疑不引
18. 跨篇协调：金句/案例/理论只在归属篇完整展开，其余篇一笔带过（见跨篇协调表）
19. 心理科普合规边界：不诊断、不处方、不承诺治愈；涉及自杀/自伤/家暴/虐待等高危话题必须给出求助渠道（全国统一心理援助热线 12356 / 北京心理危机研究与干预中心 010-82951332 / 希望24热线 400-161-9995 / 共青团12355）
20. 每批次撰写后必须 Glob/LS 核对实际产出文件数 = 预期（subagent 输出丢失先例）

## 九、关键纠正（调研发现）

1. **《让创意更有黏性》作者**：用户资料误作"希瑟·布斯、乔纳森·奇克"，实际为**奇普·希思+丹·希思（Chip & Dan Heath）兄弟**。希瑟·布斯是美国社会活动家，乔纳森·奇克是爱丁堡大学精神科教授，均与本书无关。已写入 _心理学理论速查.md 锁定
2. **求助热线**：用户原描述"全国心理援助热线 12320-5"为信息混淆。实际为**12356**（国家卫健委2024-12-06设置，2025-05-01全国统一接通）。12320是全国公共卫生公益热线，不是心理援助专线。已写入 _心理学理论速查.md 锁定
3. **半佛仙人领域错配**：金融/商业吐槽非心理科普，已从爆款案例模块替换为"反面案例翻车账号"篇
4. **罗翔定位**：法学普法非心理科普，模块7罗翔篇标注"非心理科普案例，仅作叙事方法论参考"，明确"强观点风格在心理科普中须克制"
