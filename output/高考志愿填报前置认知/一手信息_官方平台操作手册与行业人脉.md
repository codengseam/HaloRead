# 一手信息：官方平台操作手册与行业人脉挖掘

本章是工具书：每个权威平台怎么操作、每个电话怎么打。机构能查到的数据你都能免费查官方最新版，数据以本省考试院为准。

## 一、阳光高考平台操作手册

教育部"阳光高考"信息平台（gaokao.chsi.com.cn）是志愿填报第一入口，四个核心功能：

### 1. 招生章程查询路径

- 路径：首页 → "招生章程"频道 → 输入院校名 → 查看2026版章程PDF
- 关键词检索：章程PDF内搜"单科要求""体检""专业级差""录取规则""调剂"五词逐条核对
- 注意：必须看2026版，往年版作废；章程是法律性文件，高于第三方

### 2. 专业目录查询

- 路径：首页 → "专业库" → 按学科门类/专业类/专业名称三级检索
- 每个专业可查：门类、代码、培养目标、核心课程、就业方向、开设院校、近三年就业率
- 2026新增38专业标注"新增"可单独筛选（据教育部《普通高等学校本科专业目录（2026年）》）

### 3. 就业质量报告查询

- 路径：首页 → "高校就业质量报告" → 选院校 → 查看年度报告
- 重点看：各专业就业率、深造率、薪资中位数、用人单位TOP10、地域
- 注意：院校自报，看连续3年趋势，单一年份可能美化

### 4. 同分去向查询（阳光志愿系统）

- 路径：阳光高考平台 → "阳光志愿"信息服务系统
- 功能：输入分数/位次/选科，给同分考生往年去向、AI助手"智慧小招"、21项生涯测评
- 注意：AI推荐仅供参考，必须用本省一分一段表+院校章程交叉复核

## 二、各省考试院数据下载统一步骤

各省考试院官网是本省数据唯一权威源。31省入口见资料汇编章，或百度搜"XX省教育考试院"认准 gov.cn。通用路径：

### 1. 一分一段表（最核心）

- 路径：省考试院官网 → "高考"频道 → "成绩查询"或"分数段统计表" → 下载PDF/Excel
- 用途：分数换算成全省位次，定位院校准绳
- 注意：物理组/历史组、首选科目不同分别查；艺术/体育单独列表

### 2. 投档线下载

- 路径：省考试院官网 → "高考"频道 → "投档线"或"录取分数线" → 按批次（本科提前/本科批/专科批）下载
- 用途：核对目标院校往年投档分/位次
- 注意：看"投档线"不是"录取线"——前者进档门槛，后者最终分，可能不同

### 3. 征集志愿通知查询

- 路径：省考试院官网 → "高考"频道 → "征集志愿"或"志愿征集"专栏
- 用途：本科批/专科批未完成计划院校公布征集志愿，滑档兜底
- 注意：征集窗口极短（通常1天），提前盯紧；计划表注明剩余计划、分数要求

### 4. 政策性加分查询

- 路径：省考试院官网 → "高考"频道 → "政策规定"或"加分政策" → 查看2026年加分项目及分值
- 用途：确认是否符合加分条件（烈士子女、少数民族、退役士兵等）
- 注意：加分各省不同且逐年收紧；只适用本省投档，部分院校不认

## 三、院校官网挖掘手册

院校官网是源头，三个必挖文档：

### 1. 招生章程PDF

- 路径：院校官网 → "招生就业"或"本专科招生" → "招生章程" → 下载2026 PDF
- 比阳光高考更早发布，含附件（分专业计划、选科表）

### 2. 就业质量报告

- 路径：院校官网 → "招生就业"或"就业指导中心" → "就业质量报告" → 年度PDF
- 重点看四表：各学院就业率、各专业深造率、薪资分布、用人单位TOP10
- 进阶：对比同档次院校报告，看"性价比"

### 3. 学科评估结果

- 路径：中国学位与研究生教育信息网（CDGDC）→ "学科评估" → 查询结果（A+/A/A-/B+）
- 用途：判断学科实力，比"985/211/双一流"更细
- 注意：第五轮评估部分未公开，第四轮仍可参考

## 四、招生办电话话术模板

打电话是最被低估的获取方式，招生办老师掌握的信息比任何第三方都准（陈红雨团队观点）。建议工作日上午9:30-11:30、下午14:30-16:30打，避开周一上午和周五下午。

### 1. 自我介绍模板

> "老师您好，我是2026届XX省考生（物理组/历史组），分数XX，位次XX，选科XX。我对贵校XX专业很感兴趣，有几个问题想请教，占用您两分钟。"

### 2. 必问5个问题

1. "我这个位次报贵校XX专业录取概率多少？往年这个专业的录取位次区间？"
2. "贵校XX专业有没有单科要求？比如数学或英语最低分？"
3. "贵校录取规则是分数清、专业清还是专业级差？级差是多少？"
4. "进档后服从调剂会不会退档？"
5. "XX专业入学后能不能转专业？比例和条件是什么？"

### 3. 怎么记录

打一个电话记一张卡：院校、专业、接电话老师工号/姓名、5问题答复、是否明确、备注。多院校对比一目了然。

### 4. 不会答的问题怎么办

老师不答概率（合规要求），换问"往年录取位次区间"自己推算；说"以章程为准"，把章程条款再问一遍；态度敷衍，换时间再打或打院系教学秘书（院系官网可查）。

### 5. 交叉验证

同一问题问招生办+在校学长+查章程原文，三方一致才可信。仅一方说，存疑。

## 五、行业一手信息获取

### 1. 部委官网报告检索

| 部委 | 检索路径 | 能查到什么 |
|---|---|---|
| 工信部（miit.gov.cn） | "政策文件""产业政策""白皮书" | 制造业十大领域、工业软件、集成电路报告 |
| 发改委（ndrc.gov.cn） | "政策发布""新闻发布会" | 十五五产业政策、低空经济人才缺口 |
| 人社部（mohrss.gov.cn） | "新职业""就业景气报告" | 72个新职业目录、行业就业景气度 |
| 统计局（stats.gov.cn） | "统计公报""行业数据" | 各行业从业人员、平均工资 |

### 2. 行业协会数据

| 协会 | 能查到什么 |
|---|---|
| 中国半导体行业协会 | 集成电路人才报告、产销数据 |
| 中国汽车工业协会 | 新能源汽车产销、人才需求 |
| 中国低空经济联盟 | 低空经济规模、企业名录 |
| 中国机床工具工业协会 | 机床产销、五轴国产化率 |

### 3. 上市公司年报看行业前景

- 路径：巨潮资讯网（cninfo.com.cn）或交易所官网 → 搜公司 → "定期报告" → 年度报告
- 披露：年报4月30日前披露上年年报，半年报8月底前
- 重点看四页："管理层讨论与分析"、"主营业务分析"、"员工情况"、"研发投入"
- 举例：半导体看中芯国际、韦尔股份、北方华创；电池看宁德、比亚迪

## 六、人脉信息获取

数据是死的，人是活的——在校学长一句话胜过十份报告。

### 1. 校友会/校友群

- 路径：目标院校官网 → "校友会"或"校友办" → 查各地校友会；或通过高中老师联系本校考上的学长
- 提问方向：学习体验、就业去向、转专业难度、考研氛围

### 2. 学长学姐提问话术

> "学长/学姐好，我是今年高考生，想报贵校XX专业。能请教三个问题吗：1）这个专业平时课业压力大不大，最难的是哪门课？2）你们这一届毕业大概都去了哪儿，考研的多还是就业的多？3）如果重新选，你还会选这个专业吗，为什么？"

三个问题覆盖"学习体验+就业去向+真实评价"。

### 3. 行业从业者联系

| 平台 | 适合找谁 | 怎么用 |
|---|---|---|
| 脉脉 | 互联网、半导体、新能源从业者 | 搜目标公司+岗位，看职场帖，私信请教 |
| LinkedIn（领英） | 外企、金融、咨询从业者 | 搜职位，看职业路径，礼貌connect |
| 在行 | 各行业专家付费咨询 | 搜"高考志愿""职业规划"或目标行业，约行家 |
| 知乎 | 各专业/行业从业者 | 搜"XX专业就读体验""XX行业真实情况"，注意辨别软文 |

提问原则：先做功课再问，别问百度能查的；真诚说明身份；尊重时间，一次1—3个问题。

## 七、实习与跟岗：用一周时间验证专业选择

### 1. 跟岗体验

通过家长人脉联系行业从业者，跟岗1-3天，观察：日常工作是否如孩子想象、强度能否接受、对专业的真实评价（非招生话术）。

### 2. 院校开放日与夏令营

关注目标院校"开放日""夏令营"通知，部分985/211暑假有体验营，可进实验室、听课。

### 3. 暑期实习项目

"实习僧"等平台有高中生暑期项目，多为付费但能体验真实场景，优先目标专业。

## 写在最后

一份官方章程、一张一分一段表、一通招生办电话，胜过机构话术、网上鸡汤、邻居经验。本手册每条路径都免费、公开、权威，就看你愿不愿意花时间查。
