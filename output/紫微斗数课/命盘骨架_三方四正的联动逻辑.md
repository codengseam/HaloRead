## 概念

### 三方四正的定义

「三方四正」是紫微斗数解盘的核心结构概念，指以本宫为中心、与其形成特定几何关系的四个宫位的合称：

- **本宫**：所要观察的那个宫位
- **三合两宫**（即「三方」）：与本宫相隔四位（120 度）的两个宫位，构成等边三角
- **对宫**（即「四正」之一）：与本宫相隔六位（180 度）的宫位

合起来，本宫 + 三合两宫 + 对宫 = 四个宫位，即「三方四正」。

### 术语纪律：三方四正 ≠ 邻宫 ≠ 夹宫

初学者常把三方四正与邻宫、夹宫混淆，必须严格区分：

- **三方四正**：本宫 + 三合两宫 + 对宫（共四宫，含本宫）
- **邻宫**：与本宫左右相邻的两个宫位（隔一位）
- **夹宫**：本宫左右两邻宫同时坐星，从两侧「夹」住本宫

三者是完全不同的几何关系。三方四正是「120 度 + 180 度」的远程联动；邻宫是「相邻」的近程影响；夹宫是「两侧夹击」的特殊结构。混用会导致解盘逻辑错乱。

### 流派视角声明

本篇采三合派/中州派视角，重在星曜组合与三方四正格局判断。飞星派重四化飞星联动，三方四正退居次要，飞星派的具体方法另见第 5、7 章。同一张命盘，三合派与飞星派看重的「联动关系」并不相同——三合派看星曜在三方四正上的组合，飞星派看四化在宫位间的飞移。本篇先讲三合/中州视角的三方四正。

## 原理

### 三合的构成

地支三合是三方四正的底层基础：

- 申子辰合水（北方三合）
- 寅午戌合火（南方三合）
- 巳酉丑合金（西方三合）
- 亥卯未合木（东方三合）

宫位三合依地支三合而定。十二宫每宫都有固定的三方两宫：

| 本宫 | 三方两宫 | 对宫 | 三方四正合参 |
|---|---|---|---|
| 命宫 | 财帛、官禄 | 迁移 | 命、财、官、迁 |
| 兄弟宫 | 疾厄、田宅 | 交友 | 兄、疾、田、交 |
| 夫妻宫 | 迁移、福德 | 官禄 | 夫、迁、福、官 |
| 子女宫 | 交友、父母 | 田宅 | 子、交、父、田 |
| 财帛宫 | 命、官禄 | 福德 | 财、命、官、福 |
| 疾厄宫 | 兄弟、田宅 | 父母 | 疾、兄、田、父 |
| 迁移宫 | 夫妻、福德 | 命宫 | 迁、夫、福、命 |
| 交友宫 | 子女、父母 | 兄弟 | 交、子、父、兄 |
| 官禄宫 | 命、财帛 | 夫妻 | 官、命、财、夫 |
| 田宅宫 | 兄弟、疾厄 | 子女 | 田、兄、疾、子 |
| 福德宫 | 夫妻、迁移 | 财帛 | 福、夫、迁、财 |
| 父母宫 | 子女、交友 | 疾厄 | 父、子、交、疾 |

每宫的三方四正配对是固定的，不因命宫位置不同而改变。

### 三方四正的作用机制

三方四正之所以重要，是因为任何一宫的吉凶都不是孤立的，必须看四宫合参：

1. **本宫坐星定基调**：本宫所坐主星决定该宫领域的基本性质
2. **三合两宫提供资源或牵制**：三合两宫的星曜会增强或削弱本宫
3. **对宫形成对照与补充**：对宫是本宫的「另一面」，提供对照视角
4. **四宫合参断一事**：四宫星曜是否同向、是否补强、是否削弱，共同决定该宫领域的实际格局

例如看夫妻宫，不能只看夫妻宫坐什么星，还要看：

- 夫妻宫本星定感情基调
- 迁移宫（三方）反映外在交往环境
- 福德宫（三方）反映精神世界对感情的影响
- 官禄宫（对宫）反映事业与感情的对照

四宫合参，才能看清感情的全貌。

### 三方四正的权重

在三合派/中州派中，三方四正的权重很高：

- 本宫坐星占约 50% 权重
- 对宫坐星占约 30% 权重（对宫影响仅次于本宫）
- 三合两宫共占约 20% 权重

对宫之所以权重高，是因为它直接「照」本宫——对宫的星曜性质会渗透到本宫。例如命宫空宫（无主星）时，直接借迁移宫的主星安星，称为「借星安宫」（注：借星力量减半，流派间对此有细节差异）。

## 实践

### 命宫三方四正的判读

命宫三方四正 = 命宫 + 财帛宫 + 官禄宫 + 迁移宫。这四宫合参定人生大格局：

- **命宫主星**：定性格底色
- **财帛宫主星**：定谋生方式
- **官禄宫主星**：定事业取向
- **迁移宫主星**：定外在表现

四宫星曜是否同向、是否互相补强、是否互相削弱，决定人生格局高低。例如：

- 命宫紫微、官禄宫武曲、财帛宫天府、迁移宫七杀——帝王格局，四宫皆强星，互相补强
- 命宫天同、官禄宫天机、财帛宫太阴、迁移宫巨门——四宫皆文星，偏文职格局

### 借星安宫的特殊情况

当本宫无主星（空宫）时，三合派采用「借星安宫」：借对宫主星安入本宫，作为本宫的「虚拟主星」参看。

例如命宫空宫，迁移宫坐紫微天府，则命宫借紫微天府作为虚拟主星，按紫微天府的性质解读命宫。

注：借星力量减半（约为本宫有主星时的 50%），且流派间对借星的细节有差异——中州派承认借星但权重降低，部分飞星派则不重借星而重宫干飞化。本篇采中州派视角。

### 其他宫位三方四正的判读要点

除命宫外，其他宫位也各有其三方四正的重点：

- **夫妻宫三方四正**（夫迁福官）：感情 + 外在环境 + 精神世界 + 事业对照，四宫合参看婚恋全貌
- **官禄宫三方四正**（官命财夫）：事业 + 性格 + 谋生 + 感情，四宫合参看事业与人生的协调度
- **财帛宫三方四正**（财命官福）：财富 + 性格 + 事业 + 精神，四宫合参看财富与福分的匹配度
- **疾厄宫三方四正**（疾兄田父）：身体 + 平辈 + 家底 + 长辈，四宫合参看健康与家底的关系

每个宫位都不是孤立断事，都必须放在三方四正的网络中看。

### 心性映射

三方四正不仅是技法，也是认识自己的结构：

- **命宫主星**：你是什么样的人
- **财帛宫主星**：你靠什么谋生
- **官禄宫主星**：你的事业舞台
- **迁移宫主星**：你在外面的样子

四宫联读，你才能看清自己「性格—谋生—事业—外在表现」是否一致。许多人的内在性格与外在表现不一致（命迁星性冲突），或事业取向与谋生方式不一致（官财星性冲突），三方四正能把这种不一致照出来。

## 速查自测

### 三方四正速查口诀

- 三方四正 = 本宫 + 三合两宫 + 对宫
- 三方 ≠ 邻宫（邻宫是相邻一位）
- 三方 ≠ 夹宫（夹宫是两邻宫夹本宫）
- 对宫权重仅次于本宫（约占 30%）
- 空宫借对宫主星，称「借星安宫」（力量减半）

### 自省问题清单

- 我命宫三方四正（命财官迁）四宫主星是否同向？反映我「性格—谋生—事业—外在」是否一致？
- 我夫妻宫三方四正（夫迁福官）反映我的感情与事业、精神、外在环境是什么关系？
- 我哪一宫是空宫？借星安宫后，该宫领域的性质如何？
- 我命宫与迁移宫（对宫）星性是否冲突？反映我内在与外在是否一致？
- 三方四正的视角下，我最强的格局在哪一宫？最弱的又在哪一宫？
