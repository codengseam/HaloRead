## 概念

### 大限是什么

紫微斗数把人的一生切成若干个十年段落，每一段叫一个**大限**（注意：紫微斗数全程用「大限」，不用八字术语「大运」）。一个大限管十年运程，从命宫起，阳男阴女顺行、阴男阳女逆行，依次走到兄弟宫、夫妻宫……每一个大限对应一个宫位，那个宫位就叫这一大限的「大限命宫」。

大限命宫定下来后，整张命盘的十二宫也要相应平移——大限命宫所在宫位，就是这一大限的「命宫」；它的对宫是大限迁移，三合两宫是大限财帛、大限事业……这种宫位随大限整体平移的现象，叫**飞宫**（5-3 已区分过：飞宫是大限/流年宫位飞移，与四化飞星是两件事，本篇下面会再用到这层区分）。

大限起算年龄由**五行局**决定。水二局从二岁起、木三局从三岁起、金四局从四岁起、土五局从五岁起、火六局从六岁起。每隔十年换一个宫位。例如水二局命主，2-11 岁大限命宫在命宫，12-21 岁大限命宫在兄弟宫（或夫妻宫，看顺逆），22-31 岁再走下一个，依此类推。

### 大限四化：动态的十年引动

大限除了把宫位平移，还会起一组**大限四化**——由大限命宫的天干起四化，飞入平移后的十二宫。这一组四化是这一大限十年里的「动态引动力」，与生年四化（先天静态）和宫干飞化（一生互动）属于不同层次。

三层四化叠在一张盘上，结构会变得复杂。但读法其实有规律：

- **生年四化**是底色，管一生。
- **大限四化**是这一十年的高光，管十年。
- **宫干飞化**是十二宫之间的互动，一生都在起作用。

大限四化所落的宫位，是这十年里被引动的领域；大限四化所化的星，是这十年里被激活的能量。读一个大限，先把大限四化所落的四个宫标出来，再与生年四化对照，就能看出这十年的「主线」。

### 太岁入卦（紫云派技法）

紫云派（台湾三合派代表紫云先生所创）有一套独特的大限技法，叫**太岁入卦**。所谓太岁入卦，是把流年太岁（流年地支所对应的宫位）作为「入卦」点，看大限宫位与太岁宫位之间的星曜与四化互动，进而推断这一年的吉凶。

太岁入卦的思路与中州派、飞星派都不同：中州派重三方四正与生年四化；飞星派重宫干飞化与忌星追踪；紫云派的太岁入卦则把「流年太岁」作为额外的引动点，与大限宫位形成「双宫联动」。

需要明确标注：**太岁入卦是紫云派（三合派一支）的特有技法**，中州派、飞星派并不通用。看盘时如用太岁入卦，须标注「依紫云派」。本篇在实践段会演示一次太岁入卦的简化用法，让读者有个直观印象。

## 原理

### 大限宫位的读法

进入某个大限后，第一步是看大限命宫坐什么星、什么宫位。例如命主 32-41 岁大限命宫走到财帛宫原宫位——这十年的「我」就从财帛宫原宫位的星曜读起。如果财帛宫原坐紫微天府，那这十年就是「紫府同宫的大限」，主这十年有领导力、有财库、有尊贵气场。

大限命宫之外，还要看大限三方四正（大限命宫 + 大限财帛 + 大限事业 + 大限迁移）的星曜组合。这与本命三方四正的读法一致，只是换成大限平移后的宫位。

大限命宫的对宫——大限迁移——尤其要紧。大限迁移代表这一十年「向外发展」的能量，与出国、远行、换环境、对外形象相关。大限命迁线被煞星冲破，主这十年外出多波折；有吉星会照，主外缘顺。

### 大限四化的读法

大限命宫的天干起四化，飞入大限平移后的十二宫。读法分两层：

**第一层：大限四化所落的宫**。这一组四化所落的四个宫，就是这十年被引动的四个领域。例如大限化禄入大限财帛、大限化权入大限事业、大限化科入大限夫妻、大限化忌入大限命——这十年的主线就是「进财、事业掌权、感情平顺、自我消耗」。

**第二层：大限四化与本命宫位的对照**。大限平移后的宫位，对应本命的某个宫位。例如大限命宫走到本命财帛宫，那大限化禄所入的大限财帛，可能就是本命的某个宫位。这种「大限宫位 → 本命宫位」的对照，可以看出十年里被引动的本命领域。读大限要练出这种「双层宫位」的视角。

### 大限四化与生年四化的叠加

大限四化与生年四化叠加，有几种典型模式：

- **大限禄入生年忌所在宫**：这一十年给先天执念送来化解，是「十年解忌」的好时段。
- **大限忌入生年忌所在宫**：这一十年加重了先天执念，是「忌上加忌」的难时段。
- **大限禄入生年禄所在宫**：这一十年给先天福报加福，是「福上加福」的顺时段。
- **大限忌入生年禄所在宫**：这一十年给先天福报添忧，是「福中带忧」的警示时段。
- **大限忌冲本命命宫**：大限忌所落宫位的对宫是本命命宫，主这一十年对命主冲击大。
- **大限禄入本命命宫**：这一十年给命主送禄，主顺。

这六种模式是叠加判读的基础。看一个大限，先把这六种模式查一遍，十年的主线就清楚了。

### 太岁入卦的简化用法

紫云派太岁入卦的完整操作较繁，本篇只演示简化用法。假设命主在某大限内的某流年，流年地支为午——午宫就是这一年的太岁入卦点。看大限命宫与午宫之间的星曜与四化互动：

- 大限命宫与午宫是不是本命三方四正的关系？若是，主这一年与大限主线同向。
- 大限四化有没有飞入午宫？若有，主这一年太岁宫位被引动。
- 午宫本命坐什么星？这颗星在这一年有什么象意？

太岁入卦的关键，是把「流年太岁」作为额外的引动点，与大限主线叠加。紫云派用这一技法推断具体流年的事件，常与大限四化配合使用。中州派、飞星派不通用此法，看流年多用流年四化（6-4 内容）。

## 实践

### 一个大限的完整读法

下面用一个简化盘例演示一个大限的完整读法。假设命主 32-41 岁大限，大限命宫走到本命事业宫原宫位。

**第一步：大限命宫与三方四正**。大限命宫在本命事业宫，原坐天机太阴（机阴同度）。大限三方四正：大限命宫机阴、大限财帛、大限事业、大限迁移。这十年的「我」读机阴——天机主思虑、太阴主细腻，主这十年心思细密、注重规划。

**第二步：大限四化所落宫**。假设大限命宫天干为乙，乙令天机化禄、天梁化权、紫微化科、太阴化忌。大限四化所落：机禄在大限命（自化禄）、梁权在大限福德（天梁坐福德）、紫科在大限夫妻（紫微坐夫妻）、阴忌在大限命（自化忌）。注意机阴同宫，乙令机化禄、阴化忌，等于大限命宫同时自化禄与自化忌。

**第三步：大限四化与本命宫位对照**。大限命宫在本命事业宫，所以这十年的「自我实现」与本命事业宫深度相关。大限自化禄解事业宫的先天格局，主这十年事业上有自我滋养；大限自化忌又消耗事业宫，主这十年事业上也自我消耗。禄忌同宫相战，主这十年事业起落反复。

**第四步：与生年四化叠加**。假设生年忌在本命夫妻宫，大限化科入大限夫妻——夫妻宫这一十年被大限化科所引动，主这十年婚姻有平顺名声上的助力，是「科解忌」的一种弱化。

**第五步：大限忌冲检查**。大限化忌在大限命（自化），不冲对宫。但若大限忌所落宫位对宫是本命命宫，主这一十年对本命命主冲击大，要专门提防。

**第六步：太岁入卦（如采紫云派）**。若命主 35 岁流年太岁在子，看大限命宫（在本命事业宫）与子宫（本命某宫）的星曜与四化互动，作为这一年的细化推断。

**串联结论**：这十年（32-41 岁）心思细密、重规划（机阴大限），事业上起落反复（大限自化禄+自化忌相战），婚姻上有名声平顺的助力（大限科入夫解生年忌）。化解方向：事业上不要因一时得失而自我消耗（自化忌），把心力放在长线规划（机阴之长）；婚姻上借助大限科的助力化解先天执念。

### 大限引动格局

大限还会引动本命的格局。例如本命有「火贪格」但一直未爆发，进入某个大限后，大限命宫走到贪狼所在宫位，且大限四化化禄入贪狼——这就引动了火贪格的爆发能量，主这十年有突发的机遇。

反之，本命有「府相朝垣」的稳健格局，进入某个大限后大限忌冲本命命宫，则稳健格局在这十年被压制，主这十年波折。

格局不是静态的——它在哪个大限被引动、哪个大限被压制，决定了它的实现时段。这一点把 6-1 的格局与 6-3 的大限串了起来：**格局是底色，大限是时段，两者叠加才看出起伏**。

### 大限读法的几个注意点

实战中读大限，几个常见注意点：

- **大限不是流年**。大限管十年，是「时段」；流年管一年，是「节点」。本篇讲时段，6-4 讲节点。
- **大限四化不是生年四化**。大限四化是十年动态引动，与生年四化（先天）层次不同。两者叠加时不能混淆层次。
- **大限命宫不等于本命命宫**。大限命宫是十年「临时命宫」，本命命宫是一生「根本命宫」。两者所坐星曜可能完全不同。
- **大限忌所落宫位最要紧**。与 5-4 的忌星追踪一样，大限忌的来去是十年的核心执念所在。
- **大限引动要看双向**。大限四化飞入本命某宫，是「大限引动本命」；本命某宫的星曜在大限命宫三方四正里，是「本命参与大限」。两个方向都要看。

## 速查自测

### 大限六步读法

1. 看大限命宫与三方四正（坐什么星）。
2. 看大限四化所落宫（引动哪些领域）。
3. 看大限四化与本命宫位对照（双层宫位）。
4. 与生年四化叠加（六种模式查一遍）。
5. 查大限忌冲（是否冲本命命宫）。
6. 太岁入卦（如采紫云派，作为流年细化）。

### 大限四化与生年四化叠加六模式

| 模式 | 象意 |
|---|---|
| 大限禄入生年忌宫 | 十年解忌 |
| 大限忌入生年忌宫 | 忌上加忌 |
| 大限禄入生年禄宫 | 福上加福 |
| 大限忌入生年禄宫 | 福中带忧 |
| 大限忌冲本命命宫 | 十年冲击大 |
| 大限禄入本命命宫 | 十年顺 |

### 自省问题清单

- 我能不能分清「大限」「大运」「流年」？紫微用大限，禁大运；大限管十年，流年管一年。
- 我命盘当前在哪个大限？大限命宫走到本命哪个宫位？
- 当前大限的四化所落哪四个宫？这十年的主线是什么？
- 当前大限有没有「禄解忌」或「忌加忌」？是化解时段还是执念加重时段？
- 我命盘上的格局，在哪个大限被引动、哪个大限被压制？
- 太岁入卦是哪一派的技法？我用不用这一法，标注清楚了没有？
