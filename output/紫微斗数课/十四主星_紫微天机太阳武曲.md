## 概念

### 紫微：帝星的尊贵与孤独

紫微阴土，北斗主星，十四主星之首。古诀称"紫微为帝座"，主尊贵、领导、信实。紫微的形象，是一位端坐朝堂的帝王——既要有威仪，也要有辅佐；威仪而无辅佐，则成孤君；辅佐而无威仪，则成傀儡。

紫微的性格底色，是"自尊"二字。紫微坐命的人，无论庙旺落陷，都自带一种不容轻慢的气场。庙旺者，自尊化为领导力与号召力；落陷者，自尊化为面子与固执。理解紫微，要从这股"自尊"的流向入手。

### 天机：谋臣的灵动与浮动

天机阴木，紫微星系中的谋臣。木主仁，故天机主人智谋、思虑、变动。天机的形象，是帝王身边的军师——机智、敏捷、善谋划，但也容易想太多、变动不居。

天机星性最活泼，最忌固定。天机坐命的人，思维跳跃、兴趣广泛、适应力强，但也容易三分钟热度、缺乏定性。天机的"机"字，既是机智之机，也是机动之机——动得其所则成智囊，动失其所则成浮萍。

### 太阳：贵星的光明与消耗

太阳阳火，紫微星系中的贵星。火主礼，故太阳主光明、博施、外向。太阳的形象，是高悬天空的日轮——普照万物、不求回报，但也因此持续消耗、易陷劳碌。

太阳的核心性质是"博施"。太阳坐命的人，庙旺者主慷慨大方、乐于助人、事业显达；落陷者主付出多而回报少、劳碌奔波。太阳还有"男主父、女主夫"的层次：男命看太阳即看自己与父亲，女命看太阳即看丈夫与父亲。

### 武曲：财星的刚毅与孤克

武曲阴金，紫微星系中的财星。金主义，故武曲主刚毅、决断、财权。武曲的形象，是执掌财权的将军——刚直、果决、善理财，但也容易孤克、寡情、过于刚硬。

武曲是十四主星中最大的财星，但其财是"刚毅之财"——靠决断与执行得来，不是靠柔取巧取得。武曲坐命的人，庙旺者主事业有成、善于理财、刚毅果决；落陷者主孤克寡情、财来财去、人际关系紧张。

## 原理

### 紫微庙旺与落陷

紫微的庙旺，主要在午、巳、未等宫支。庙旺的紫微，主尊贵显达、领导有方、受人敬重；落陷的紫微（如在子、丑等宫），主孤傲、固执、面子重、难与人合作。

紫微的吉凶，很大程度上取决于辅佐星曜。古诀云"紫微无辅为孤君"，紫微喜左辅右弼同宫，谓之"帝有辅佐"；紫微无辅而遇煞，谓之"孤君在野"，主孤立无援、自视甚高而难成事。这是紫微解读中最关键的一条。

### 天机庙旺与落陷

天机的庙旺，主要在卯、辰、寅等宫支。庙旺的天机，主智谋过人、思维敏捷、善于谋划、应变力强；落陷的天机（如在酉、戌等宫），主浮动不安、想太多、缺乏定性、做事反复。

天机最忌与煞星同宫。天机与擎羊同宫，主"机谋反成阴谋"，思虑化为算计；天机与陀罗同宫，主"思虑拖延"，决断力不足；天机与火星同宫，主"急躁多变"，行事莽撞。天机喜与左辅右弼、文昌文曲同宫，主智谋得以施展。

### 太阳庙旺与落陷

太阳的庙旺，以午宫为最——"日丽中天"，主光明磊落、声名显赫、博爱无私、事业显达。此外卯、辰、巳、未、申等宫亦为庙旺。落陷的太阳，主要在申酉戌亥等宫——其中申酉为"日落西山"，主劳碌奔波、付出难有回报。

太阳的庙陷落差，是十四主星中最为显著的。同一颗太阳，在午与在申，吉凶几乎完全相反。这是中州派解读太阳的第一条铁则：**先看庙陷，再论星性**。

太阳还有男女命差异：男命太阳庙旺主自身显达、父运佳；女命太阳庙旺主夫运佳、自身亦显。男命太阳落陷主自身劳碌、父缘薄；女命太阳落陷主夫运薄、婚姻多波折。

### 武曲庙旺与落陷

武曲的庙旺，主要在辰、戌、丑、未等宫支（土金之地，土生金旺）。庙旺的武曲，主刚毅果决、事业有成、财源稳固、执行力强；落陷的武曲（如在午、巳等火地，火克金），主孤克寡情、财来财去、人际紧张。

武曲最忌与煞星同宫。武曲与擎羊同宫，主"刑伤"，易有外伤、手术、官非；武曲与陀罗同宫，主"拖延刑克"，财来迟而多阻；武曲与火星同宫，主"寡宿"，孤克之性更甚。武曲喜与天府、天相同宫，谓之"财印相生"，主财权有印信、稳健可托。

## 实践

### 紫微与辅煞的搭配

实战解盘看紫微，最关键是看辅佐与煞星：

- 紫微 + 左辅右弼 + 天魁天钺：帝有辅佐、贵人扶持，主大贵之格。
- 紫微 + 文昌文曲：帝有文才，主文贵。
- 紫微 + 擎羊陀罗：孤君在野，主孤立无援。
- 紫微 + 火星铃星：帝受煞扰，主暴起暴落。
- 紫微无辅无煞：孤君，主清高自赏、难以成大事。

### 天机的智谋与浮动解盘

天机的解盘，要看其"机"是用于智还是用于浮：

- 天机庙旺 + 文昌文曲：智谋得文才辅佐，主学业有成、文思敏捷。
- 天机庙旺 + 左辅右弼：智谋得贵人辅佐，主事业有成、谋略可行。
- 天机落陷 + 擎羊：智谋化为算计，主心机重、人际关系紧张。
- 天机落陷 + 陀罗：思虑拖延，主决断力不足、错失良机。
- 天机 + 太阴（同宫或对照）：机月同梁格的基础，主稳健而有谋。

### 太阳的男女命差异

太阳的解读，男女命差异显著：

**男命**：太阳即自身（命宫坐太阳时）与父亲。太阳庙旺，主自身显达、父运佳、事业光明；太阳落陷，主自身劳碌、父缘薄、付出多而回报少。

**女命**：太阳即丈夫与父亲。太阳庙旺，主夫运佳、丈夫有成就、自身亦显贵；太阳落陷，主夫运薄、婚姻多波折、独自承担家务。

这是太阳解读中最重要的性别差异。读者解盘时切忌男女命一概而论——同一颗太阳在午，男命主自身显达，女命主夫显达，断语完全不同。

### 武曲的财与克的平衡

武曲的解读，要在"财"与"克"之间找平衡：

- 武曲庙旺 + 天府/天相：财印相生，主财权稳固、事业有成。
- 武曲庙旺 + 化禄：财源广进，主大富。
- 武曲落陷 + 化忌：财来财去、刑伤孤克，主财破人伤。
- 武曲 + 七杀/破军：将相组合，主刚猛有为之格，但也主大起大落。
- 武曲 + 贪狼：武贪格，主中年发迹，但早年多波折。

武曲的"克"，并非全凶——克得其所，则成决断与执行；克失其所，则成孤克与刑伤。这是武曲解读的辩证法。
