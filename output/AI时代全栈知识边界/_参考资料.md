## 一、各领域知识掌握深度与 AI 辅助建议(总览表)

| 技术领域 | 建议掌握核心程度 | 必须掌握的核心知识点(AI 难以替代) | 可放心交给 AI/理解原理即可 | 典型 AI 辅助工具/场景 |
| :--- | :--- | :--- | :--- | :--- |
| **Python** | 精通(核心语言) | 基本语法、数据类型、控制流、函数、面向对象编程、常用库、异常处理、文件操作、GIL、生成器、asyncio、装饰器 | 繁琐样板代码、特定库生僻 API、复杂正则表达式、简单脚本生成 | GitHub Copilot、Cursor、ChatGPT |
| **HTML/CSS/JavaScript** | 熟练应用 | HTML 语义化、CSS 盒模型、Flexbox/Grid、JS 异步编程、DOM 操作、Vue/React 核心概念、原型链、事件循环、闭包、this 指向 | 复杂 CSS 选择器、兼容性写法、简单 JS 交互、UI 组件初步代码、动画 | v0.dev、Copilot |
| **计算机网络** | 理解核心原理 | OSI/TCP-IP 模型、HTTP/HTTPS、DNS、三次握手四次挥手、跨域、状态码、TCP/UDP 区别、网络调试 | 复杂抓包分析、底层协议细节、socket API 细节记忆 | AI 解释网络异常、生成请求脚本 |
| **数据库(SQL & NoSQL)** | 精通应用与设计 | SQL(CRUD/复杂查询/索引优化)、事务 ACID、MySQL/PostgreSQL 原理、Redis 适用场景、B+树、MVCC、隔离级别 | 复杂 SQL 调优、底层存储引擎细节、分布式数据库运维 | AI 生成复杂 SQL、解释慢查询日志 |
| **操作系统** | 理解核心概念 | 进程/线程/协程、内存管理、文件系统、I/O 模型、权限管理、Linux 命令、容器原理 | 内核源码级细节、极底层系统调用、系统调优极端参数 | AI 解释进程状态、生成 Shell 脚本 |
| **数据结构与算法** | 掌握核心与思想 | 基本结构(数组/链表/栈/队列/树/图/哈希)、基本算法(排序/查找/递归/分治)、复杂度分析、贪心/DP 思想 | 手写复杂算法极致优化、冷门算法细节、竞赛级技巧 | AI 解释思路、生成基础代码 |
| **中间件** | 理解应用与原理 | 消息队列(RabbitMQ/Kafka)使用与场景(解耦/异步/削峰)、Redis 数据结构与策略、ES 查询原理 | 中间件源码级实现、极端场景深度调优 | AI 配置中间件、解释错误日志 |
| **架构与设计** | 具备全局思维 | 分层架构、微服务、事件驱动、SOLID、RESTful/GraphQL、设计模式、可用性可扩展性 | 超大规模复杂架构细节、企业级架构所有演进原因 | AI 生成架构图、分析优劣 |
| **软件工程** | 融入实践 | Git、CI/CD 概念、测试类型、代码审查、项目管理流程 | 复杂构建脚本、高度定制 DevOps 编排 | AI 生成提交信息、编写单测 |
| **运维与部署** | 掌握核心流程 | Linux 操作、Docker 容器化、服务器部署、日志监控、云服务基本使用 | K8s 深度调优、服务网格、大规模集群运维脚本 | AI 生成 Dockerfile、解释部署错误 |
| **安全** | 建立意识与防范 | Web 漏洞(SQL 注入/XSS/CSRF)原理与防范、加密概念(HTTPS/哈希/对称非对称)、身份验证授权、安全编码 | 加密算法数学细节、复杂渗透测试、安全产品配置 | AI 扫描漏洞、生成安全测试用例 |
| **分布式理论** | 理解核心 | CAP/BASE 理论、分布式事务(2PC/TCC/本地消息表)、分布式 ID、幂等性 | 分布式事务完整落地代码、一致性算法编码 | AI 解释理论、设计简单流程 |
| **AI 工程化** | 掌握应用流程 | 大模型 API 调用、Prompt 工程、RAG 流程、LangChain/LlamaIndex、向量数据库、模型微调概念 | 大模型底层训练算法、自研模型架构、推理引擎深度优化 | AI 优化 Prompt、设计 RAG、生成测试数据 |
| **项目管理与业务建模** | 掌握基础 | 需求分析、业务流程梳理、ER 图设计、RESTful 接口规范、简单排期、模块划分 | 业务流程图绘制、ER 图绘制、项目排期表、需求文档撰写 | AI 辅助生成文档 |

### 6.1 Python
**必须掌握**:基础语法、数据类型、面向对象、异常捕获、模块包管理、虚拟环境;常用标准库原理、Python 内存机制、GIL 锁、深浅拷贝、闭包与装饰器;并发模型(线程/进程/协程、GIL 限制、asyncio);项目工程化(依赖管理、版本锁定、日志规范、PEP8、单元测试);框架选型(Django/Flask/FastAPI);性能定位、报错日志解读、AI 生成代码漏洞校验。
**可交给 AI**:复杂正则、工具函数、第三方库 API、爬虫脚本、接口实现、批量文件处理、CRUD 样板、封装工具类。
