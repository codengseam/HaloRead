# 说话之道·英雄之旅

> 本篇覆盖英雄之旅 12 阶段模型及其在长篇叙事与个人品牌中的应用，不覆盖故事为何有效（讲故事为什么真的有效 篇展开）、六大类信任故事（六大类故事框架 篇展开）、单故事七步结构（靶心人公式 篇展开）。

## 一、入戏：乔布斯用三个故事讲完一场英雄之旅

2005 年 6 月，斯坦福大学毕业典礼。乔布斯上台，没讲大道理，先说了一句：「今天我想给你们讲三个故事，就这样，没什么大不了。」

第一个故事是「连接人生点滴」：大学旁听的书法课当时看不出有用，十年后设计第一台 Mac 时全用上了。第二个故事是「爱与失去」：三十岁被自己创办的苹果开除，从巅峰跌到谷底，又作为初学者创办 NeXT 和皮克斯，最终回到苹果。第三个故事是「死亡」：一年前被诊断胰腺癌，医生让他回家安排后事，后来发现是罕见可手术类型，从鬼门关走了一遭。

三个故事讲完，他落下一句「Stay hungry, stay foolish」。

这场演讲只有十五分钟，但十几年过去，它依然是毕业演讲史上被引用最多的之一。为什么一场只讲三个个人小故事的演讲会有这么大的影响力？因为它暗合了一个最深的故事结构——英雄之旅。

三个故事恰好串成英雄之旅的完整弧线：第一个是「日常世界」与「冒险召唤」（书法课埋下 Mac 字体伏笔）；第二个是「严峻考验」与「重生」（被开除的低谷成了创造力最旺盛的时期）；第三个是「接近深穴」与「满载而归」（与死亡擦肩，把看清带回来）。乔布斯未必 consciously 套用英雄之旅，但叙事本能让他讲出了一个符合这个结构的故事。英雄之旅是被几千年叙事验证过、最稳的长篇结构。

## 二、破题：英雄之旅是长篇叙事的金字塔

英雄之旅（hero's journey）由神话学家约瑟夫·坎贝尔（Joseph Campbell）在 1949 年的《千面英雄》（The Hero with a Thousand Faces）中提出。坎贝尔研究世界各地的神话，发现这些看似千差万别的故事共享同一个基本结构——英雄从日常世界出发，经历试炼与转化，最终带着收获归来。他把这个结构命名为「单一神话」（monomyth）。

后来好莱坞编剧顾问克里斯托弗·沃格勒（Christopher Vogler）把坎贝尔的结构简化为 12 个阶段，写成《作家之旅》（The Writer's Journey）一书，成为好莱坞编剧的常用工具。《星球大战》《黑客帝国》《指环王》都明显套用了这 12 阶段。

讲故事领域里，培训师哈迪娅·努里丁（Hadiya Nuriddin）在《故事力法则》（Story Rules）一书中，把英雄之旅模型应用到职场叙事和个人品牌构建，让它落地为普通人讲长篇故事的工具。

为什么说英雄之旅是长篇叙事的金字塔？金字塔原理是职场表达的「最稳结构」（结构篇展开），英雄之旅则是长篇叙事的「最稳结构」。它能承载的篇幅最大——从十分钟演讲到三小时电影，从几十万字小说到半辈子的职业叙事，都能套用。稳定性来自两点：一是完整弧线（出发-试炼-归来），听众跟着走不会迷失；二是内在转化（主角被改变），转化带来意义感，意义感带来记忆。

靶心人公式适合讲一件有冲突的事（短篇、单事件），英雄之旅适合讲一段有转化的旅程（长篇、多事件）。前者解决「讲清楚一件事」，后者解决「讲明白一段路」。

## 三、方法论：12 阶段精简版

坎贝尔-沃格勒的 12 阶段完整版比较学术，下面给你一个职场与个人叙事能直接用的精简版。

### 阶段一：日常世界——交代主角起点

故事开始前，主角生活在平凡的日常世界里。这一阶段交代主角的起点状态——他是谁、在做什么、有什么不满足。

写法要点是「具体」+「不满足」。具体到能让人看见画面（哪个城市、什么工作、生活什么样），不满足到能让人感到他要走。没有不满足的日常世界，故事启动不了——主角没有出门的理由。

乔布斯第一个故事里的「日常世界」是大学旁听书法课的那个学生，看起来漫无目的，但暗藏对美的好奇心。

### 阶段二：冒险召唤——一个事件打破日常

某个事件发生了，召唤主角离开日常世界。这个事件可以是一封信、一通电话、一次偶遇、一个危机、一个机会。

冒险召唤是故事的启动键。写法要点是「外部性」+「不可拒绝」。召唤最好来自外部（公司被收购、亲人病重、机会从天而降），且让主角无法拒绝（不响应就承担明显代价）。这种不可拒绝性，让出发显得自然而非刻意。

### 阶段三：拒绝召唤——主角一开始不想走

接到召唤后，主角的第一反应往往是拒绝。他害怕、他犹豫、他觉得现狀还过得去、他觉得风险太大。

这一阶段让主角显得真实——没有人接到大召唤会毫不犹豫跳进去。拒绝召唤也制造张力，听众会想「他最后去不去」。乔布斯被苹果开除后一度想逃离硅谷——这是他的「拒绝召唤」时刻。

### 阶段四：遇见导师——一个外力推他出门

主角在犹豫时，遇到一个导师或一个外力，把他推过门槛。导师可以是人（前辈、老师、朋友），也可以是一本书、一次谈话、一个事件。

导师的作用是「给勇气」+「给工具」。勇气让他敢出门，工具让他出门后有办法。写法上，导师不必是 Wise Old Man 那种刻板形象，一个一句关键话点醒你的同事、一本关键时刻读到的书、一次让你改变看法的失败，都可以是导师。

### 阶段五：跨越门槛——主角真正出发

主角跨过门槛，进入未知的世界。这一刻是不可逆的——他回不到原来的日常了。

跨越门槛是故事的真正起点，前面四阶段都是铺垫，从这里开始主角才真正进入「冒险」。写法上，门槛要写得「具体」——一个签下的合同、一次辞职、一次搬到新城市、一次正式接受挑战。具体门槛让「出发」有了仪式感。

### 阶段六：试炼、盟友、敌人——在未知世界摸爬

进入未知世界后，主角经历一系列试炼，结交盟友，遭遇敌人。这一阶段是故事的主体部分，篇幅最长。

试炼是小冲突的连续，盟友是帮主角的人，敌人是阻挡主角的人。乔布斯被开除后创办 NeXT 和皮克斯——NeXT 没成功（试炼），皮克斯做出《玩具总动员》（盟友和成果），同时他要面对自己性格带来的麻烦（敌人是他自己）。

### 阶段七：接近深穴——主角逼近核心考验

试炼累积到一定程度，主角接近故事的核心考验——「深穴」。深穴是主角最害怕面对的东西，可能是真正的强敌、内心最深的恐惧、最艰难的抉择。

接近深穴的写法要点是「铺垫恐惧」。要让听众感到主角面对的是什么，并为主角捏一把汗。深穴越可怕，后面的重生越有力量。

### 阶段八：严峻考验——主角进入深穴

主角进入深穴，面对最严峻的考验。这一阶段是故事的高潮，主角可能死亡（神话里的字面死亡，职场故事里的象征性死亡——失败、跌停、崩溃）。

乔布斯讲自己被诊断出胰腺癌的那一刻，就是他的严峻考验——他直面了死亡。这是他三个故事里最深的穴。

### 阶段九：获得奖赏——主角从深穴中带出东西

通过严峻考验后，主角获得奖赏。奖赏可以是物质的（宝剑、宝藏），更常见是认知的——一个领悟、一种力量、一份清醒。

乔布斯从死亡考验里带出的奖赏是：「记住你即将死去，是我所知避免掉入『失去什么』陷阱的最佳方法。」这是认知层面的奖赏。

### 阶段十：归来之路——主角开始往回走

获得奖赏后，主角开始从深穴归来，往日常世界走。但归来之路并不顺利，往往还有最后的考验在等着。

这一阶段在职场叙事里经常被压缩或省略，因为个人故事的「归来」往往跟「重生」融合。但理解它的存在有助于让叙事有完整弧线——主角不是只往深处走，他要走回来。

### 阶段十一：重生——主角在归来途中完成转化

归来途中，主角经历最后一次转化——他不再是出发时的那个人了。重生是英雄之旅的精神内核，主角在外在旅程的同时完成了内在蜕变。

乔布斯讲完死亡故事后，把「向死而生」的清醒带回生活，这就是他的重生。重生不一定是大彻大悟，可以是「从此我看这件事的方式变了」。

### 阶段十二：满载而归——主角带回日常世界

主角带着奖赏和转化，回到日常世界。他不再是出发时的自己，他带回的东西改变了日常世界，也改变了他跟日常世界的关系。

乔布斯的「满载而归」是那句「Stay hungry, stay foolish」——他把三个故事的全部收获凝结成一句劝告，递给毕业生。这句话既是给毕业生的，也是给他自己的，更是整个英雄之旅的落点。

## 四、避坑：12 阶段是地图不是清单

英雄之旅 12 阶段好用，但用错地方比不用更糟。下面是几个最常见的坑。

### 坑一：12 阶段全填，个人故事撑不起

最大的坑是把 12 阶段当清单，个人故事硬塞全 12 阶段。一个五分钟的职业分享，要塞 12 阶段，每阶段只能分到一句话，故事变成提纲。个人故事取 5 到 7 阶段即可——挑最能体现你转化的几个阶段重点讲，其余阶段一笔带过或省略。判断标准是「这阶段对转化有没有贡献」，有则留，无则删。

### 坑二：把生活事件硬拔成「英雄」叙事

第二个坑是把普通的职场经历硬拔成「英雄」叙事。一次普通的项目调动、一次常规的晋升，硬要套英雄之旅，反而显得做作。英雄之旅适合「有真实转化的旅程」——创业、转行、重大失败后的重建、人生方向的转折。普通事件用靶心人公式就够了，不必动用英雄之旅。工具挑场景用，杀鸡别用牛刀。

### 坑三：忽视转化，只讲事件流水账

第三个坑是只讲事件经过，不讲主角转化。英雄之旅的精神内核是转化——主角在旅程中变成了不同的人。如果只讲「我做了什么、又做了什么、最后做了什么」，那叫简历，不叫英雄之旅。讲英雄之旅，必须回答：这段旅程让你变成了什么样的人？回答不出来，故事就没灵魂。

### 坑四：导师阶段硬编

第四个坑是为了凑「遇见导师」阶段，硬编一个导师形象。真实的导师未必是 wise old man，可以是一句点醒你的话、一本改变你看法的书、一次让你清醒的失败。如果实在没有明确的导师阶段，可以省略，让「冒险召唤」直接把主角推过门槛。真实比完整重要。

## 五、践行：英雄之旅精简三件套

把英雄之旅落地，给你三件套：精简模板、今日练习、一句话行动。

### 精简模板：个人英雄之旅 5-7 阶段速填表

挑一段你人生中真实发生转化的旅程（创业、转行、重大失败后重建、人生方向转折），按下表填空。挑 5 到 7 个阶段重点填，其余可省略：

| 阶段 | 是否填 | 我的一句话 | 转化点 |
|---|---|---|---|
| 1. 日常世界 | ☐ | _____（起点状态+不满足） | |
| 2. 冒险召唤 | ☐ | _____（外部触发，不可拒绝） | |
| 3. 拒绝召唤 | 可省 | _____（犹豫、害怕） | |
| 4. 遇见导师 | ☐ | _____（人/书/事，给勇气或工具） | |
| 5. 跨越门槛 | ☐ | _____（不可逆的出发动作） | |
| 6. 试炼盟友敌人 | ☐ | _____（小冲突+帮手+阻挡者） | |
| 7. 接近深穴 | 可省 | _____（逼近核心考验） | |
| 8. 严峻考验 | ☐ | _____（最深的穴，象征性死亡） | ★ 转化关键点 |
| 9. 获得奖赏 | ☐ | _____（认知或物质收获） | |
| 10. 归来之路 | 可省 | _____（往回走的最后考验） | |
| 11. 重生 | ☐ | _____（从此我变成了什么样的人） | ★ 转化关键点 |
| 12. 满载而归 | ☐ | _____（带回日常世界的落点） | |

填表的关键在第 8、11 两个转化点——严峻考验和重生。这两点填不出来，整段旅程就没灵魂。

### 今日练习：用英雄之旅重写你的职业转折

挑一次你人生中真实的职业转折（第一次跳槽、转行、创业、被裁后重建），用英雄之旅 5 到 7 阶段重写一遍。要求：

第一步，写下转折前的「日常世界」——你在做什么，有什么不甘。第二步，写下「冒险召唤」——什么事件让你决定动。第三步，写下「跨越门槛」——你做出了哪个不可逆的动作。第四步，写下「严峻考验」——这段旅程里你最深的穴是什么，你经历了什么象征性的「死亡」。第五步，写下「重生」——这段旅程让你变成了什么样的人。第六步，写下「满载而归」——你带回日常世界的是什么（一句领悟、一种能力、一份清醒）。

六步写完后，找一个朋友，用五到八分钟讲给他听。讲完问他两个问题：「这个故事里你最有感触的是哪个时刻？」「听完你对这个人的印象是什么？」第一个问题检验转化点是否成立，第二个问题检验个人品牌是否传达。两个回答都让你满意，这段职业转折的英雄之旅叙事就成了。

### 一句话行动

下次有人问你的职业故事或人生转折（面试、自我介绍、个人品牌场合），用英雄之旅 5 到 7 阶段在脑子里过一遍——把事件串成转化，把转化落成印象，你的故事就不再是简历，而是有弧线的旅程。
