> 本篇拆解奥巴马「Yes We Can」演讲的排比节奏三招与练习，不覆盖演讲开场结构体系（结构篇展开），也不覆盖个人故事库的完整搭建（故事篇展开）。

## 一、入戏：二〇〇八年格兰特公园的那个夜晚

二〇〇八年十一月四日深夜，芝加哥格兰特公园。几十万人挤在草坪和路灯之间，有人在哭，有人在笑。那一晚，巴拉克·奥巴马当选美国第四十四任总统，也是历史上第一位非裔总统。据历史档案与媒体留存的影像，他走上临时搭起的玻璃背板讲台，身后是一排排美国国旗。他没有挥手致意，停了几秒，等人群的呼喊自己落下去，才开始讲话。

那场胜选演讲里，他反复说了三个词——「Yes We Can」。据演讲实录的常见译法，大意是：是的，我们能。每一次他说出这三个词，前面都跟着一个具体的群体——林肯说我们能，废奴主义者说我们能，妇女和抗争者说我们能，工人和移民说我们能。一遍一遍，节奏越来越重，人群的喊声一次比一次高。讲到「这是我们的时刻」时，全场几乎失控。

奥巴马的控场能力来自反复打磨。他早年讲话偏学术，二〇〇四年民主党全国代表大会那场 keynote 演讲让他一举成名，也显出他对节奏的敏感。到了二〇〇八年，胜选夜那段排比被反复引用、被写进歌词。一段排比能做到这种程度，靠的是他对节奏与信念之间关系的精准拿捏。

## 二、破题：奥巴马用节奏把信念种进每个人心里

### 区别：演讲传递信息，节奏种下信念

普通的胜选演讲传递的是信息——感谢支持者，承诺施政，展望未来，听众听完就散场。奥巴马那晚传递的是信念——这件事是我们一起做成的、未来也一起做的归属感。差别在于节奏。信息靠内容传递，信念靠节奏种植。同一个意思，平铺直叙地说，听众点头；用排比一层层推上去地说，听众心里发烫。节奏的作用，是让一句话从听见升级到记住，再升级到相信。

### 难点：节奏不能空转，必须配信念

会排比不等于有节奏，有节奏也不等于种得下信念。排比如果只是同一句话重复三次，听着像喊口号；节奏如果只是声音高低起伏，听着像表演。真正能种下信念的节奏，每一步都比上一步更接近信念的核心——从事实到决心，从决心到归属。奥巴马的排比之所以有效，是因为每一句 Yes We Can 前面都垫着一个具体的群体和一段具体的历史，节奏往上走的同时，内容也在往下扎。节奏是外壳，信念是内核，两者咬合才有力量。

### 收束：用节奏把我变成我们

节奏走到顶点，要把听众带到一个地方——从他能转向我们能。奥巴马全篇最高明的一步，是把一场个人胜利重新定义为一场集体胜利。Yes We Can 里的那个 We 是关键，排比每一次重复都在强化这个我们。听众喊到最后，已经分不清是在支持一位总统，还是在确认自己是某个更大共同体的一员。节奏的终极作用，是让说话者退后，让信念上前。

## 三、方法论：奥巴马式表达三招

### 排比造势：三个 Yes We Can 层层垫高

第一招是排比造势。奥巴马没有把 Yes We Can 当口号喊三次，他把它当作一个不断被历史垫高的支点。每段排比的结构都是：先讲一个具体的人或群体（林肯、废奴主义者、妇女、移民、工人），再讲他们面对的困难，最后用一句 Yes We Can 收尾。听众听到的是一段历史在被反复唤醒，同一个词只是载体——林肯时代我们能，废奴时代我们能，民权时代我们能，所以今天我们也能。

造势的关键是垫。垫，指的是每句排比之前先给一个具体的场景或人物，让 Yes We Can 有了根，成了回声。空洞的排比只会让人觉得在给自己壮胆；垫了场景的排比才会让人觉得在呼应一段共同的历史。

可执行动作：选一个你想传达的信念（比如这个团队能成事、这条路走得通），为它写三段排比，每段前面垫一个具体的人、事或场景。三段的垫料要有时间或难度的递进——从更早、更难的那个，到当下、稍轻的那个。写完读出声，听三段之间是不是在往上走。如果三段听着一样高，说明垫料没有递进，回去重选。

### 节奏推进：从低处往高处走

第二招是节奏推进。奥巴马整篇演讲的节奏曲线是经过设计的——开头低沉克制，承认胜利来之不易；中段逐渐升温，铺排历史；到 Yes We Can 那段推到顶峰；顶峰之后再用「这是我们的时刻」收束。从低到高，再到一个更深的高，有起有伏。这种节奏曲线让听众的情绪跟着走——开头被压住，中段被带着跑，顶峰被点燃，收束时被沉下来。

推进的关键是留白。奥巴马在每个排比之间都留了停顿，让上一句的回声在听众心里响一会儿。停顿是节奏的灵魂——没有停顿，排比成了机关枪；有停顿，排比成了鼓点。普通人最缺停顿的勇气，总怕一停就被认为卡壳，于是把所有话连成一片。奥巴马敢停，因为他知道停顿是给信念留出扎根的时间。

可执行动作：把刚才写的三段排比读三遍。第一遍正常读，第二遍在每段之间停一秒，第三遍在 Yes We Can 收尾词之前停两秒。感受三种读法的差别——停顿越敢给，信念感越重。把停顿位置标在稿子上，下次当众讲时按标记停。

### 信念收束：把胜利重新定义为我们的时刻

第三招是信念收束。排比造势到顶峰，需要一个收束把节奏落下来，同时把信念钉死。奥巴马选的收束是「这是我们的时刻」——一句话同时做了三件事：把节奏从顶峰落到深沉的低点；把个人胜利重新定义为集体时刻；把听众的视线从今晚的胜利拉向未来的责任。一个好的收束，让听众离场时带走一种身份——我是这个时刻里的人。

收束的关键是身份转换。排比让听众跟着喊我们能，那是跟着说；收束让听众心里出现这是我们的时刻，那是变成自己的话。跟着说，是情绪被点燃；变成自己的话，是信念被种下。前者散场就散了，后者会跟着听众回家。奥巴马用我们这个词把责任也塞给了听众——这是我们的时刻意味着接下来也是我们的事。信念不是被给出来的，是被这样分担出去的。

可执行动作：为你那段排比写一句收束。收束要满足两个条件——一是把节奏从高落到低，让听众心里沉一下；二是用一个我们，把信念从你身上转到听众身上。写完读一遍，看收束是否让你心里从激动变成踏实——激动是情绪，踏实是信念，后者才是收束要的效果。

## 四、避坑：排比节奏的两个雷区

### 排比变空洞：同一句话重复三次

第一个雷区是排比变空洞。很多人学了排比之后，把同一句话重复三次就以为是在用奥巴马的招——我们能行、我们能行、我们能行。这不是排比，这是复读。奥巴马的排比每层都有不同的垫料（不同的人、不同的时代、不同的困难），同一句 Yes We Can 在不同垫料上呈现不同意味。空洞的排比只会让人觉得你词穷，越喊越虚。

避免空洞的方法是每层换垫料。排比的句子可以同一个，但每层前面铺垫的场景、人物、困难必须不同，而且要有递进——从更难的到稍轻的，或从更远的到更近的。递进让排比有了方向，听众跟着走，情绪才会上扬。没有递进的排比是水平的，喊几遍就累了；有递进的排比是斜向上的，越喊越有劲。

### 节奏变表演：声音起伏盖过内容

第二个雷区是节奏变表演。有些人学了节奏推进后，开始刻意压低声音、突然抬高、刻意停顿，把演讲变成朗诵比赛。听众的反应是他在演。奥巴马的节奏之所以自然，是因为他的停顿和起伏为内容服务——内容到了关键处，节奏自然起伏。节奏是从内容里长出来的，不是贴在内容外面的装饰。

避免表演化的方法是先写对再练声。先把稿子写到位——哪里该停、哪里该重，由内容决定。稿子定了，再照着练声，让声音服从内容。如果稿子本身没有节奏感，靠声音硬撑的起伏都是表演。节奏的最高境界，是听众记住你说的话，而不是你怎么说的。

## 五、践行：排比节奏三件套

### 排比节奏模板：垫料—排比—收束三段

下次要在一个场合传达一个信念（团队动员、家庭动员、公开演讲都算），按这三段走。

第一段（垫料）：回想一个比现在更难的时刻，那时候具体的人或群体面对具体的困难，他们做了什么。用一段不超过六十字的场景垫底，给排比一个根。

第二段（排比）：在场景一我们做到了什么，在场景二我们做到了什么，在场景三我们做到了什么。所以——我们能（你的核心信念）。三句排比，每句配一个具体场景，场景要有递进，最后用核心信念收口。

第三段（收束）：这不是一个人的事，这是一个共同体的时刻。从某个时间点开始，这件事就是我们所有人的事。把节奏从高落到低，用我们把信念从你身上转到听众身上。

三段走完，听众先被历史垫住，再被排比推高，最后被收束沉下。垫料、排比、收束，缺垫料排比悬空，缺排比节奏起不来，缺收束信念种不下。

### 今日练习：为一个信念写三段排比

选一个你真心相信的信念——关于你自己的（我配得上这份工作），关于你团队的（我们能熬过这个冬天），或关于你家庭的（这个家还能更好）。为它写三段排比，每段前面垫一个具体的场景或人。三段的垫料要有递进——从更早或更难的，到当下或稍轻的。写完在每段排比之间标出停顿位置，在收尾信念词之前标一个更长的停顿。然后读出声，读三遍——第一遍找内容，第二遍找节奏，第三遍找信念感。如果第三遍读完你心里发烫又踏实，说明这篇排比成了。

### 一句话行动：开口前先垫一层场景

下次要用排比表达信念之前，先在心里问自己——我这句话前面垫了什么场景？如果什么都没垫，先别开口，回去找一个具体的人或事垫上。垫了再排比，排比才有根；不垫就排比，排比就是口号。奥巴马的 Yes We Can 之所以被记住，是因为这五个字前面垫了一整段美国历史。
