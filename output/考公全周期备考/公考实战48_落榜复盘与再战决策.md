# 第48章：如果这次没上岸，复盘转向与再出发

查到成绩那一刻，差两分没进面，或者面试被刷了。老哥不跟你说"加油""失败是成功之母"这种话——没用，还恶心人。这一章讲的是落榜后怎么把情绪按住、把账算清、把路选对。

先讲个反常识的事：每年落榜的考生里，真正适合再战的不到四成。剩下六成里，有一部分是分数差距太大、年龄不占优、经济扛不住，再战也是炮灰；还有一部分其实有更好的出路，只是被"非公务员不可"的执念困住了。所以这一章不是教你"再战必胜"，是教你"该不该再战、怎么再战、再战之外还有什么"。

落榜这件事，本质是一个决策问题，不是情绪问题。情绪只是决策的干扰项。你得先把情绪的生理机制搞清楚，再用决策框架把账算明白，最后才谈再战或者转向。

## 概念

### 学情把脉：落榜后的两种废

落榜后常见两种状态。一种是"瘫型"——查完分往床上一躺，三天不起，外卖盒子堆一地，手机不想看，群消息全是未读。一种是"躁型"——立刻发誓"明年必上岸"，当晚就报了两万块的全协议班，第二天辞职，第三天把朋友圈签名改成"再战"。两种状态都是应激反应，都不是理性决策。瘫型把时间耗在情绪里，躁型把决策做在最冲动的时候。

### 本质透视：决策问题，不是情绪问题

落榜的本质是"一个目标阶段性没达成"，接下来要做的是"下一步怎么办"的决策。情绪是决策的噪声，不是决策的依据。这章的逻辑链是：先处理生理（让大脑降温）→再处理情绪（把噪声降下来）→再做决策（用框架算账）→最后执行（再战或转向）。把顺序搞反了，比如在最难受的时候做辞职决定，三个月后大概率后悔。

## 原理

### 落榜后48小时：先处理生理，再处理情绪

落榜的难受不是"心态不好"，是生理反应。大脑收到"重大目标受挫"的信号，皮质醇和肾上腺素飙升，你会出现心跳加速、失眠、食欲紊乱、注意力涣散。这是哺乳动物的应激反应，跟意志力无关，跟坚强无关。

所以48小时内的第一件事不是"振作起来"，是让身体先降温。一个可执行的行为指标：**连续两个晚上保证6小时以上睡眠**。做不到就别逼自己复盘，先恢复睡眠。失眠超过72小时的，建议去医院，别硬扛。

48小时情绪急救SOP：

| 时间 | 动作 | 目的 |
|---|---|---|
| 0-2小时 | 离开查分页面，关掉考公群和朋友圈，物理隔离信息源 | 切断刺激 |
| 2-8小时 | 找一个不考公的朋友或家人，把情绪说出来，不寻求建议只求倾诉 | 情绪外化 |
| 8-24小时 | 吃一顿热饭，洗个热水澡，睡一觉，不做任何决定 | 生理恢复 |
| 24-48小时 | 在纸上写下"现在最想做的三件事"，逐条问自己"这是情绪驱动还是理性驱动" | 决策防火墙 |

**要不要立刻告诉家人？** 看家庭支持度。支持型家庭当天说，焦虑型家庭缓一周说，索取型家庭（会借机施压要钱的）不说也行。判断标准：你预测家人知道后的反应，是"帮你减压"还是"给你加压"。加压的，缓一缓。

**是否辞职改在职？** 别在48小时内做这个决定。辞职是重大财务决策，应激状态下做的辞职决定，三个月后大概率后悔。给自己两周冷静期，再按后面的5维评估表算账。

**避免冲动决策红线（三条）：**
1. 48小时内不辞职、不分手、不搬家、不报班。
2. 48小时内不在朋友圈发"再也不考了"或"明年必上岸"。
3. 48小时内不联系前任、不买大件、不做任何超过5000元的消费。

### 再战5维评估表：算清再战的账

落榜后第二周，情绪稳定了，用这张表给自己打分。每项0-2分，满分10分。7分以上建议再战，4-6分建议谨慎再战或转向，3分以下建议转向。

| 维度 | 0分 | 1分 | 2分 | 自评 |
|---|---|---|---|---|
| 分数差距 | 差进面分数线15分以上 | 差5-15分 | 差5分以内或进面被刷 | |
| 年龄 | 30岁以上且首次考 | 28-30岁 | 28岁以下 | |
| 经济 | 无存款且需全职备考 | 有3-6个月生活费 | 有6个月以上生活费或可在职考 | |
| 家庭 | 强烈反对或经济不支持 | 中立观望 | 支持再战 | |
| 机会成本 | 放弃现有工作损失大、行业上升期 | 一般 | 现状无发展、再战机会成本低 | |

这张表的逻辑是：分数差距小、年龄有优势、经济有保障、家庭支持、机会成本低——这五条占得越多，再战越值得。反过来五条都不占，再战就是把时间和钱往水里扔。

### 复盘上次失败：四个维度逐条挖

很多人"复盘"就是"我下次努力点"，这叫情绪复盘，不叫理性复盘。真正的复盘要落到具体行为上。

**知识盲区。** 把上次每个模块的正确率列出来，找出低于60%的模块。不是"我数量关系差"，是"我数量关系的工程问题正确率只有30%，这是盲区"。盲区要具体到题型。

**时间投入。** 算一下你过去一年真正用于备考的有效小时数。注意是"有效"——刷手机看考公资讯不算，看视频不走心不算。一个量化方法：拿出备考期间的日历，标注每天真正学习的小时数加总。多数人会发现自己以为的"全职备考一年"其实有效学习时间不到600小时。

**心态。** 考试时有没有因为紧张而失误？有没有因为前面模块超时导致后面乱蒙？心态问题要落到"哪个环节崩了"，不是笼统的"心态不好"。

**选岗。** 你报的岗位竞争比多少？有没有更合适的岗位你当时没考虑？选岗失误是落榜的高频原因，很多人分数不低，但报了"三不限"千人岗，神仙也救不了。

### 换地区/换岗/换备考策略决策矩阵

复盘完，决定再战的，要决策三件事：换不换地区、换不换岗位类型、换不换备考策略。

| 决策项 | 维持 | 更换 |
|---|---|---|
| 地区 | 当地有岗位且竞争比可接受 | 当地全是三不限或竞争比超500:1，考虑跨省或下沉基层 |
| 岗位 | 上次岗位类型适合自己且有机会 | 上次岗位类型连续两年无突破，换赛道（如综合类转执法类） |
| 备考策略 | 上次有明确短板且已找到方法 | 上次方法没找到瓶颈，换老师或换学习模式 |

### 再战的隐性成本核算公式

再战不是免费的。算一笔账：

**再战总成本 = 直接成本 + 隐性成本**
- 直接成本 = 报班费 + 资料费 + 考试报名费 + 差旅费（约0.5-2万）
- 隐性成本 = （再战期间放弃的工资收入）+（同龄人晋升的差距）+（心理消耗）

举个例子：在职考生年薪8万，辞职再战一年，直接成本1万，隐性成本8万（工资）+1万（晋升延迟）= 10万。这一年再战如果上岸了，公务员年收入假设6万，看似亏了，但长期看是赚的。如果再战还没上岸，这一年净亏10万加一年时间。

把这个数算出来，你才不会稀里糊涂地"再战"。

## 实践

### 多元出路：公务员不是唯一选项

老哥见过太多考生被"非公务员不可"困住，结果错过了更适合自己的路。体制内的入口不止一个：

**事业编。** 事业编考试比公务员考试频次高、竞争比相对低，待遇差距在缩小。很多事业单位的工作内容和公务员接近，是"曲线进体制"的好选择。

**选调生。** 应届生或基层服务项目人员才能报，竞争比低于普通国考，但要求党员、学生干部等条件。符合条件的优先冲。

**国企。** 央企和地方国企的校招、社招，待遇可能高于公务员，工作稳定性也好。烟草、电网、银行系统的国企，是仅次于公务员的"铁饭碗"。

**基层项目。** 三支一扶、西部计划、特岗教师，服务期满后有定向招录公务员的政策，是"先基层后上岸"的通道。

**考研再战。** 如果本科学历是短板（很多好岗位要研究生），考研提升学历后再考公，是长期策略。

### 再战实操清单

1. 落榜后两周内：完成5维评估表 + 4维度复盘，白纸黑字写下来。
2. 决定再战后的第一周：制定新一轮备考计划，明确每天有效学习时长（建议在职4小时/天，全职8小时/天）。
3. 每月底：用真题模考一次，对比上次同期的分数，确认是否在进步。三个月没进步的，停下来重新评估方法。
4. 考前两个月：进入冲刺模式，停止新课，专注真题和错题。

## 速查/自测

### 模板 H｜决策型（ch48 落榜再战必用）

这是5维评估表的决策版，配红线和默认选项：

**5维自评得分：____/10**

- 得分≥7 → 默认选项：再战，按再战实操清单执行。
- 得分4-6 → 默认选项：谨慎再战。建议在职考，同时投递事业编/国企作为保底。
- 得分≤3 → 默认选项：转向。优先事业编、国企、基层项目，公务员作为副线。

**决策红线：**
- 连续两次再战都没进面 → 第三次必须换赛道或转向。
- 经济储备低于3个月生活费 → 不建议全职再战。
- 年龄超35岁（公务员报考上限）→ 转向事业编或国企。

### 在职生速查卡

- 落榜48小时内：不辞职、不报班、不发朋友圈。
- 第二周：完成5维评估 + 4维度复盘，每周日晚复盘一次进度。
- 决定再战：维持现工作，每天有效学习4小时（早1.5+午0.5+晚2）。
- 同步动作：每月投递1-2个事业编或国企岗位作为保底。

### 在校生速查卡

- 落榜48小时内：联系就业指导中心，了解应届生身份保留政策。
- 第二周：完成5维评估 + 复盘，决策"再战考研"还是"再战考公"还是"先就业"。
- 伪努力预警：再战不等于"换个教室继续刷题"。如果上次的方法没让你上岸，照搬只会再输一次。先复盘方法，再谈努力。
- 应届身份珍贵：能冲选调的别错过，能走校招国企的别等社招。

### 自测题

1. 落榜后48小时内应避免哪三类冲动决策？
2. 再战5维评估表中，分数差距维度2分的标准是什么？
3. 再战隐性成本核算公式包括哪两项？
4. 公务员之外，体制内入口至少有哪几条？

答案：1. 不辞职、不分手/搬家、不报班、不超5000元消费（任答三类）；2. 差5分以内或进面被刷；3. 直接成本 + 隐性成本；4. 事业编、选调生、国企、基层项目、考研再战。

一句话小结：落榜不是终点，也不是必须再战的起点——先算清账，再决定路。情绪按住，决策做对，路自然就有了。
