## 一、整体协作框架提示词

启用多个 agent 组成专家团理解需求，并使用 skills 和 checklist 规范执行，用多个 agent 并行提速，完成后启用专家团检查并修复完成，采用 loop agent 的思维来开发优化这个项目。

**核心目标**：让普通人系统掌握形象管理（从头到脚、由内而外），把形象作为"非语言沟通能力"的底层逻辑讲透，并落地为可执行的系统——不只是"穿得体面"，而是理解形象如何放大个人价值。面向普通人，侧重男性视角（参考资料均聚焦男士形象管理），但底层原理通用。

**关键约束**：
- 主题模块和引用内容仅供参考，需启用多专家团评估方案后并行执行
- 找到更全面的资料和引用，直接输出相关的几十篇 md 文档
- 最终生成的提示词、引用资料需要保留，但不能放在专栏里展示（`_` 前缀文件）
- 单章引用放文章末尾，全部引用单独列一个 md（`_参考资料.md`）
- 其余专栏内容直接展示出来
- 完成严格质检

## 四、正文撰写硬约束（写入每个撰写 subagent 的 query）

### 4.1 frontmatter 契约（8 字段，缺一不可）

```yaml
---
title: <标题，与预编排表一致，不含章节号>
book: 形象管理课
chapter: <开篇|道|形|术|场|健|避坑|践行|终极>
event: <事件名，与文件名一致>
created_at: 2026-07-02
source_agents:
  - image-style-expert
sort: <章内序号，与预编排表一致>
chapter_sort: <章序号，与预编排表一致>
---
```

- `chapter` 必须与文件名章节部分一致（check_book_structure.py 强制校验，P1）
- `event` 必须与文件名事件部分一致（P1）
- 同章内 `sort` 从 1 起严格递增不重复（P2）
- 同章内 `chapter_sort` 必须一致（P1）

### 4.2 五段式结构（modern 桶，带中文序号 + 冒号 + 副标题）

```markdown
# <标题>

## 一、入戏：<副标题>

<场景化代入，现代生活场景，300-500 字>

## 二、破题：<副标题>

<点出本章要解决的核心问题与底层逻辑，400-600 字>

## 三、方法论：<副标题>

<核心方法，分点阐述，1500-2500 字，本章主体>

## 四、避坑：<副标题>

<常见误区 3-5 个，每个 150-300 字>

## 五、践行：<副标题>

<可执行的动作清单或思考路径，300-500 字>

## 参考来源

- <条目1>
- <条目2>
```

**关键**：
- 五段标题严格用 `## 一、入戏：` / `## 二、破题：` / `## 三、方法论：` / `## 四、避坑：` / `## 五、践行：`，带中文序号和冒号，副标题自拟
- 文末标题必须是 `## 参考来源`（**不是「本章参考资料」**——中国礼仪课先例不合规，已被 loop_log 标记，content_quality 校验的是 `## 参考来源`）
- 每篇正文 ≥ 2000 字

### 4.3 引用与真实性

- 只引用 `_research_*.md` 和 `_参考资料.md` 中已 WebSearch 核验的条目，不杜撰书籍版本/作者/出版社/年份/数字
- 名人语录必须可查，存疑的不引
- 行内引用密度：每千字「——《XX》」式引用 ≤ 3 处
- 同一出处连续出现只保留一处
- 普通叙述句/白话改写不挂行内出处
- 单章引用放文末「## 参考来源」，每条注明书名/篇名及对应内容
- 禁止内联跳转（详见第 X 章 / 见上文 / 见前文 / 详见下章等），提及另一章用自然语句

### 4.4 现代术语处理（全角括号隔断法）

MODERN_ENGLISH_WHITELIST 未覆盖的术语（如 TPO / SPF / PA / BMI / EPOC / O 型腿等），首次出现用全角括号隔断「中文（英文）」，首次定义后正文用中文。

### 4.5 语言风格

- 禁止 AI 套路句式（我们可以看到 / 综上所述 / 这告诉我们 / 值得注意的是 / 总而言之等）
- 禁止现代学科术语硬套（底层逻辑 / 坐标系 / 博弈论等，必要术语除外）
- 无中英混杂（专有名词除外，且遵循 4.4 隔断法）
- 段尾升华不超过 2 处，且只在破题/践行/终极
- 男性视角侧重，但底层原理通用；不刻意贬低女性形象管理

### 4.6 章节边界（防重复）

- 「形」组只讲诊断与认知（怎么判断体型/色彩季型/风格/体态问题）
- 「健」组只讲训练与执行（怎么矫正体态/怎么练/怎么吃）
- 「术」组讲通用工具（西装/衬衫/领带等单品的通用法则）
- 「场」组讲场景应用（具体场合怎么穿，不重复「术」的通用法则）
- 「避坑」组只讲跨章节系统性误区，不重复单章已讲过的雷区
- 讲人物/案例引用前文事件时一笔带过，不展开

### 4.7 误传清单引用

各 subagent 撰写时必须读取对应 `_research_*.md` 的「WebSearch 误传核验记录」，确保不写入已核验为误传的说法。重点误传（必须规避）：
- 色彩季型奠基人是 Bernice Kentner（1979），Carole Jackson（1980）是普及者，不可混淆
- 黑色西装非商务正装标配，商务正装首选藏蓝/炭灰
- 西装扣子法则 S/A/N 原始指向三粒扣
- 局部减脂不成立，仰卧起坐不减肚子
- 深蹲膝盖可超脚尖，强行限制伤腰
- 冷水/收敛水不能物理收缩毛孔
- 防晒不分男女，男士同样需要
- Mehrabian 7-38-55 法则仅限情感态度不一致场景，不可泛化为"沟通 93% 靠非语言"

## 六、质检阶段提示词（阶段 3，4 个 subagent 按维度横切）

4 个质检 subagent 并行，各负责一个维度横切 45 篇：
- _review_authenticity.md：真实性（40 分）—— 核验色彩理论/版型规范/营养数据/礼仪惯例来源准确；引用书籍版本/作者/出版社可查；名人语录可查；规避已核验误传
- _review_readability.md：可读性（30 分）—— 故事感/重复控制/语言风格（AI 套路句/现代术语硬套/中英混杂）
- _review_order.md：顺序（15 分）—— 单章叙事顺序/章节排序/文件命名与章节编号一致
- _review_citation.md：引用克制（15 分）—— 内联引用清理/行内引用密度/文末来源

**合格线**：每篇总分 ≥ 92 且任一维度 ≥ 满分 70%（防维度塌陷）。
**质检口径隔离**：质检 subagent 用独立提示词 + 带锚点评分量表，强制产出"问题清单+扣分明细"，禁止只打分。
**复检闭环**：修复后由非原质检 subagent 抽检 30% 确认问题闭合。
