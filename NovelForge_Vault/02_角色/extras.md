## 三、升格流程

> 当一个龙套需要二次出场且戏份加重时：
> 1. 在 `02_角色/supporting/` 新建独立 md。
> 2. 把本表中该龙套标注「已升格 → 文件路径」。
> 3. 在 `.state/characters/` 新建对应 JSON 状态文件。
> 4. 同步刷新 `00_控制面/master_index.md` 的角色索引（主 Agent 手动维护）。
