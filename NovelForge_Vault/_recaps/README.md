# 前情提要（_recaps）

> 本目录存放每 10 章冻结一份的前情提要，作为「稳定锚点」。
> 当上下文窗口不够装载全部历史正文时，context_composer 优先召回最近一份 recap + 当前卷大纲 + 本章章纲。

---

## 冻结规则

- 每 10 章冻结一份：ch_001-010 → `recap_ch_001_010.md`，ch_011-020 → `recap_ch_011_020.md`。
- 冻结时机：第 10/20/30... 章定稿后，由 `scripts/novelforge/freeze_recap.py` 自动生成。
- 冻结后**禁止修改**，如发现历史错误，新建 `recap_ch_NNN_MMM_v2.md` 而非覆盖。

---
