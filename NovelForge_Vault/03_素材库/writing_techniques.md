## 一、节奏控制

### 1.1 章节内部节奏

- **黄金比例**：前 1/3 铺垫 → 中 1/3 冲突 → 后 1/3 高潮 + 钩子。
- **断章技巧**：高潮未完时切章，留下「主角刀已出鞘，反派头未落」式悬念。
- **字数与节奏**：紧张段落用短句短段，舒缓段落可适当铺陈。

### 1.2 卷级节奏

- 每卷开头 3 章必须建立本卷核心冲突，禁止慢热。
- 每卷中段安排 1 个小高潮，避免平淡。
- 每卷结尾大高潮 + 留下下卷悬念。

## 二、心理-生理映射词库

> 与 `00_控制面/style_guide.md` 附录 A 联动。本节是「扩展词库」，提供具体写法示例。

### 2.1 紧张

- 生理：心跳加速 / 手心出汗 / 呼吸变浅 / 喉咙发紧 / 耳鸣
- 写法示例：「他听见自己的心跳，咚、咚、咚，盖过了外面所有的声音。」

### 2.2 愤怒

- 生理：太阳穴跳动 / 拳头攥紧 / 牙关咬合 / 太阳穴血管凸起 / 呼吸粗重
- 写法示例：「太阳穴一根血管突突地跳，他攥紧的拳头指节发白。」

### 2.3 恐惧

- 生理：后背发凉 / 腿软 / 汗毛倒竖 / 胃部抽搐 / 喉咙发干
- 写法示例：「一股凉意从尾椎窜上后脑，他下意识后退半步，腿却像灌了铅。」

### 2.4 得意

- 生理：嘴角上扬 / 眼神发亮 / 步伐变轻 / 手指轻叩
- 写法示例：「他嘴角不自觉地上扬，脚步也轻了几分，连袖口的褶皱都显得顺眼。」

### 2.5 犹豫

- 生理：手指敲击 / 目光游移 / 咬嘴唇 / 脚尖转方向
- 写法示例：「他的手指无意识地在桌沿敲了三下，目光从门口移到窗外，又移回来。」

## 三、对话技巧

### 3.1 潜台词

- 角色不直接说，让读者通过动作猜。
- 例：「师弟最近修炼可还顺利？」（潜台词：我知道你卡在瓶颈了）

### 3.2 身份语言

- 长辈用古语，少年用短句，商贾用利益术语。
- 详见 `00_控制面/style_guide.md` 附录 B。

### 3.3 节奏停顿

- 重要台词前留动作或环境描写，制造停顿。
- 例：他端起茶盏，吹了吹浮沫，才慢悠悠开口：「这事儿，怕是没那么简单。」

## 四、场景描写

### 4.1 五感锚点

- 每场戏至少一个：视觉/听觉/嗅觉/触觉/味觉。
- 修仙场景常用：灵气浓度（触）、剑鸣（听）、丹香（嗅）。

### 4.2 动静对比

- 静景写完后用一个动作打破，制造张力。
- 例：月色如水，庭院寂静。忽然，一片瓦片轻轻落下。

## 五、伏笔技巧

### 5.1 埋伏笔三原则

1. **自然**：伏笔必须像正常描写，不能突兀。
2. **可忆**：埋的时候要让读者「下意识记住」，回收时才能想起。
3. **可校验**：每个伏笔登记到 `04_大纲与脉络/hooks_registry.json`。

### 5.2 回收节奏

- short 伏笔：1-5 章内回收。
- long 伏笔：跨卷回收，但每 10 章必须提示一次（status → hinted）。
- core 伏笔：全书回收，每卷开头/结尾必须提示。
