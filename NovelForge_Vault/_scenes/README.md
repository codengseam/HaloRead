# 关键场景存档（_scenes）

> 本目录存放「需要 L1 召回的关键场景」全文，普通流水章不入库。
> 仅当某场景满足以下条件之一时才存档：
> 1. 后续章节需要精确召回细节（如关键对话、伏笔场景）。
> 2. 场景信息密度高，靠摘要不足以支撑后续写作。
> 3. 标志性转折（金手指觉醒、第一次打脸、卷末高潮）。

---

## 命名规范

`ch_NNN_角色_关键词.md`

示例：
- `ch_001_主角_觉醒金手指.md`
- `ch_007_主角_识破师弟.md`
- `ch_020_反派_血煞老祖登场.md`

---

## 存档模板

```markdown
# 场景存档：ch_00N · 角色 · 关键词

## 场景信息
- 章号：ch_00N
- 卷号：vol_NN
- 出场角色：____
- 场景定位：____（在原章中的位置，如「ch_007 中段」）

## 场景全文
（粘贴正文片段）

## 召回说明
> 后续章节需要召回本场景时的提示，例如：
> - 主角在此章透露了真实身份，后续不能与不知道的人设冲突。
> - 反派在此章立下了「三月之约」，ch_0NN 必须兑现。

## 关联伏笔
- H-0NN：____
```
